屏幕截图之旅：新Hotmail第4波

微软正在将数以百万计的Hotmail帐户更新到运行速度更快且包含比以前更多功能的最新版本。我们的帐户最近更新过，因此我们将浏览一下新功能。

Windows Live Hotmail是首批免费的网络邮件服务之一，当时它被称为Hotmail，是HTML的翻版。近年来，微软将其在线服务重新命名为Windows Live品牌，所以现在Hotmail通常与@hotmail.com或更新的@live.com电子邮件地址一起使用。你甚至可以在自己的域名上使用Hotmail和其他Live服务。*尽管Hotmail以前很受欢迎，但它已经失去了许多用户，特别是技术人员，被谷歌广受欢迎的Gmail服务抢走。--新Hotmail。那么让我们来看看新的Hotmail有没有提供服务。

了解一下新功能

如果你最近没有访问过Hotmail，你可能会注意到登录屏幕略有变化，使用了更大的Metro风格的字体和干净的设计。

Hotmail的新安全功能之一是新的一次性登录代码。您可以请求一个代码，该代码将通过短信发送到您的手机。这是从公共计算机登录Hotmail的一个很好的方式，而不会将您的密码泄露给上面的恶意应用程序。

登录后，您会注意到浏览器窗口或选项卡顶部有一个新的Hotmail徽标。如果您仍然看到旧的Windows Live徽标，则您仍在使用旧版本，因此您必须等待您的帐户在接下来的几周内更新。

新的收件箱看起来与以前的版本基本相同，但周围都有一些调整。

省时功能

一开始，您会注意到收件箱中有新的查看选项；您可以选择仅查看地址簿中未读的邮件或来自联系人的邮件。您甚至可以选择社交更新，仅查看来自Twitter、Facebook和其他网络的邮件；这是快速清除收件箱中所有社交网络垃圾邮件的好方法。

另一个节省时间的功能是新的扫描工具。在Hotmail页面顶部选择此功能可以快速管理发件人的所有电子邮件。

您可以让Hotmail自动将联系人中的所有电子邮件移动到特定文件夹中，甚至自动删除联系人中的邮件。这是我们看到的设置收件箱规则的最简单方法之一，而且效果很好。

Hotmail在您左侧的文件夹下面包含新的快速视图。在这里，您可以轻松查看已标记的邮件，或包含照片或办公文档的邮件。您甚至可以单击发货更新来查看您可能收到的所有发货通知电子邮件。

Hotmail的搜索功能也得到了改进，用户只需开始打字，Hotmail就会建议替代选项，还可以使用标准搜索查询来搜索来自特定发件人的电子邮件等。

在新Hotmail中阅读电子邮件

Hotmail的阅读窗格重新设计了漂亮实用的页眉。*单击页眉底部将其展开。

现在，您可以查看有关此电子邮件的所有信息，包括发件人的电子邮件地址和邮件的其他收件人。

如果电子邮件包含附件，您将在标题正下方看到有关文件的一些快速信息。_单击下载链接下载文件，或者如果电子邮件包含Office文件，请单击图标在Office Web Apps中将其打开。-即使您没有安装Office，这也是阅读Word文档和预览PowerPoint演示文稿的快捷方式。

如果Hotmail检测到该电子邮件来自邮件列表，它会在邮件顶部放置一个取消订阅链接，以帮助您停止接收不需要的电子邮件。

在标题上，您会注意到收到邮件的时间下方有一个新的操作按钮。*单击此按钮可快速回复、删除、转发或执行更多操作。*您还可以单击时间旁边的标志来标记邮件以供稍后查看。

我们最喜欢的新功能是对话视图。“您现在可以按对话查看您的电子邮件，这与Outlook 2010中的对话视图非常相似。”要激活它，请单击阅读窗格最右侧的按钮，然后选择对话。

现在，您会注意到电子邮件主题旁边有一个数字，显示该对话中有多少条消息。

阅读邮件时，您将在邮件上方和下方的对话中看到其他邮件。您可以阅读预览，或单击邮件在同一阅读窗格中查看完整的电子邮件。

撰写电子邮件

无论您是回复电子邮件还是撰写新邮件，新的Hotmail都包括一些工具来简化您的工作。您可以通过Office Web Apps添加Office文档，并在SkyDrive上随相册发送几乎无限数量的图片。或者，选择From Bing选项卡将各种Web内容添加到您的邮件中。

您可以直接从侧边栏添加Bing Search中的剪贴画、图片甚至视频。只需在上面选择您想要的选项，然后搜索要添加到您的消息中的内容。

广告

当你阅读电子邮件时，Hotmail确实会在右侧包含横幅广告，但有一些新的工具可以帮助你管理横幅广告。如果广告妨碍了你阅读电子邮件，你可以点击右下角的X将其关闭。请注意，当你切换到另一封邮件时，广告会再次打开，但这可以让阅读特别宽的电子邮件变得更容易。

或者，如果你发现广告有攻击性，点击广告反馈链接，让Hotmail知道你对该广告的想法。这将帮助他们让广告对每个人都不那么烦人。“这与Facebook中的广告非常相似，我们认为这在帮助人们对他们展示的广告感到舒服方面大有裨益。”

信差 / 通信员 / <生化>信使 / 先驱

Hotmail和其他新的Windows Live应用程序现在与Messenger Web完全集成。*您可以直接在Hotmail中与好友聊天。*如果您在Office Web应用程序中打开附件或查看日历，Messenger将与您一起进行，以便您可以继续对话。

您可以从Hotmail顶部的Messenger选项卡中查看您的联机联系人。单击联系人可开始与他们进行即时消息会话。

您还可以从电子邮件标题查看您的朋友的联机状态。单击开始与他们聊天，或者如果您将鼠标悬停在您自己的联系信息上，您可以单击更改您的个人资料图片。

更多功能即将推出

Hotmail有一些在不久的将来会添加的更多新的、很棒的功能，包括全会话SSL加密和ActiveSync，可以将您的Hotmail电子邮件、联系人和日历与流行的移动设备同步。

结束 / 结尾 / 结论 / 推论

无论您是Hotmail的长期用户还是正在考虑从当前的电子邮件提供商切换到Hotmail，Wave 4更新使Hotmail成为比以往任何时候都更好的电子邮件服务。我们喜欢新的对话视图，但也很感激您可以在不喜欢的情况下将其关闭。新的附件选项使共享图片、链接等变得更容易。社交网络集成到联系人中，让你很容易与所有的朋友保持联系。还有一些领域Gmail更容易使用；例如，我们更喜欢Gmail的消息内回复；只需点击一下，你就可以回复并发送了。不过，还是试试新的Hotmail吧；我们认为你可能会发现一些喜欢的东西！

请记住：新的Wave 4 Hotmail尚未在所有帐户上激活，因此如果您没有看到新功能，请耐心等待；您的帐户应该很快就会更新。
