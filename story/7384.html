Microsoft使用Windows 8.1 System Builder许可来误导消费者

如果您从Amazon、Newegg或其他在线零售商购买了Windows8.1的“System Builder”OEM副本，则可能违反了Windows许可协议。这意味着从技术上讲，你有一个“非正版”的Windows副本。

微软在这里误导了消费者。这在过去是一个问题，所以微软修复了Windows8的许可问题。但是，出人意料的是，他们又回到了Windows8.1的惯用伎俩。

编者按：我们联系了微软公共关系部，要求澄清这里的许可问题，但他们没有做出回应。考虑到Windows10将于明天发布，我们不得不假设他们将在未来进行许可改革。但这个许可问题现在对任何制造计算机的人都很重要，所以请继续阅读所有细节。

原始设备制造商许可证在Windows7之前还可以，但后来…

如果你曾经在网上购买过Windows7，你可能会购买“OEM”或“System Builder”版本，这比标准的零售版本便宜得多。当你搜索Windows时，它会出现在亚马逊和Newegg的顶部。人气统计数据显示，大多数人购买了System Builder版本。

但实际上你是不允许购买System Builder副本的！执照上的小字说你不能把它用于自己的个人用途。取而代之的是，OEM System Builder的副本只面向那些制造计算机然后再销售的人。

在Windows7之前，为自己的PC购买OEM System Builder许可证完全可以。事实上，这种更便宜的许可证也有合法的缺点--例如，没有微软的直接支持，Windows的副本被捆绑在一台PC上。

在Windows7中，微软改变了流行的Windows System Builder/OEM许可证。普通人不再被允许使用它来制造自己的PC，但微软继续像卖蛋糕一样向这些人出售它们。2009年，ZDNet的Ed Bott出色地记录了这个问题。阅读“在您自己的PC上使用OEM Windows可以吗？不要问微软“更多的背景信息。

只需看看下面的屏幕截图-Windows 7家庭高级系统构建器版(Windows 7 Home Premium System Builder Edition)是Amazon.com上最畅销的操作系统产品，该版本“旨在预装在新PC上以供转售”。微软确切地知道它是如何登上第一的位置的-因为普通电脑用户是买账的。

Windows8拥有“个人使用许可证”

当然，Windows8也有它的问题。但微软在Windows8上的授权是正确的-他们看到Windows7 OEM System Builder的授权情况是疯狂的。为了解决这个问题，他们在Windows8系统构建器许可证中增加了“个人使用许可证”。这意味着你可以购买Windows 8 System Builder许可证，并将其安装在你正在构建的新PC上。“这很好，因为人们无论如何都在这么做。”真的，他们只是在撤销他们为Windows7所做的一项荒谬的许可改变。

正如微软的Windows8许可指南所说：

当时，Windows8的许可证变更被广泛报道。而且，有一段时间，一切都解决了。

Windows 8.1将所有内容改回

Windows 8.1被认为是一个全新的操作系统，它已经签署了新的许可协议。个人使用许可已从Windows 8.1系统生成器许可证中删除。听说Windows 8解决了此问题后，您可能已经购买了Windows 8.1的System Builder副本，但很抱歉！您使用的是Microsoft眼中的非正版Windows。这一点在微软的“Windows System Builder个人使用许可”页面上写得非常清楚：

当时，微软宣布，他们将从典型的消费者零售渠道--电脑商店和亚马逊(Amazon)和新浪(Newegg)等在线零售商--下架System Builder的Windows副本。事实上，媒体报道说，微软正在扼杀Windows的“系统生成器”副本，这样每个人都可以只购买标准的零售副本。2013年9月，TechCrunch报告称：

然而，一年多过去了，当你去为你的新电脑购买最新版本的Windows时，你仍然可以在亚马逊和Newegg的顶部找到System Builder的Windows 8.1副本。很明显，人们都在买这些东西。事实上，System Builder版的Windows8.1 Pro现在比Amazon.com上的标准零售版Windows8.1 Pro便宜46美元。很明显，人们购买这些系统构建商许可证是为了个人使用。亚马逊(Amazon)和新浪(Newegg)等网站上的许可信息并没有明确表示“你不能为自己的个人电脑购买本产品”--如果微软真的要执行它们的许可协议，就应该这么说。

当您搜索Windows8.1时，Amazon会显示消息“如果您是系统构建商，Amazon提供Windows OEM产品。否则，请选购我们的Windows 8.1产品。“。但是拜托，我们都知道普通人都在买System Builder版本。这条信息也故意含糊不清--您可能认为自己是系统构建商，因为您正在构建自己的PC。你不是-至少根据Windows8.1的条款，尽管你是Windows8许可证下的系统构建器。在此之前，Windows7并不认为你是系统构建者，而Windows Vista和以前的Windows版本则认为你是系统构建者--根据微软的网站，尽管许可协议本身含糊不清。

呀！谁能跟踪这些东西？

想要合法的驾照吗？那些要额外收费。

听着，这是一个完全荒谬的情况。如果你是一个普通的PC发烧友，你可能会购买Windows的System Builder版本，因为它更便宜，而且可以在Amazon和Newegg上显示。如果你对此有所保留，当你听说Windows8的许可变化时，他们可能已经澄清了。微软有了钱，你得到了一份Windows副本，然后事情就脱轨了。尽管你从合法零售商那里购买了正版Windows，微软也拿到了钱，但微软却可以说你违反了Windows许可协议。

如果您真的阅读了许可协议，您可以选择。你可以通过购买大多数人正在购买的相同的Windows副本来故意违反许可证-微软在允许和不允许之间来回选择。或者，您可以尝试“做正确的事情”并遵循许可协议。想做正确的事吗？这会让你付出额外的代价--“做正确的事”是一种优质产品。

这是价格细分--你只想要一份能正常工作的Windows吗？给你，专业版130美元。是否要遵守许可协议并拥有正确许可的Windows Professional副本？那要额外付46美元，谢谢您的惠顾。

当然，据我们所知，微软实际上并没有执行这一规定。但他们确实会对企业进行软件许可审计。如果你使用的是商务版Windows8.1，为了安全起见，你最好把多余的钱花在零售版上--这也是他们所指望的。

希望微软没有回应我们，因为他们知道这场许可惨败是站不住脚的。他们可能已经在下一版本的Windows中解决了此问题。(或者这可能是一厢情愿的想法。)

如果他们没有修复System Builder许可证，微软需要停止玩游戏。不想让人们购买Windows的System Builder版本？然后停止在亚马逊和新浪上销售它们，或者至少在上面贴上一条很大的“你可能不被允许购买”的免责声明。想继续销售Windows的廉价System Builder副本吗？然后更改许可协议，再次允许个人使用。简单！
