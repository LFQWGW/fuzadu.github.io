如何安装和设置Windows Home Server

您是否曾希望能够从一台中央服务器访问家中所有计算机上的所有重要文件、音乐、照片等？今天，我们将介绍Windows Home Server，并向您展示如何在网络上的其他计算机上安装和配置它。

Windows Home Server(WHS)可满足您的家庭需求，也可用于家庭或小型办公室。它允许您将重要文档和数字媒体文件集中在一个盒子上，并使您可以从网络上的其他计算机轻松访问。它有能力备份10台计算机，并在需要时恢复它们。它充当介质服务器、备份解决方案、数据恢复、文档管理，并允许您从任何有Web连接的地方访问所需内容。使用个性化的网站地址，您可以安全地下载和上传数据文件。

您可以购买已经安装了WHS的服务器机器，也可以创建自己的WHS(这会更加极客和有趣)。最酷的是，您可以重新调整旧台式机的用途以运行Windows Home Server。一台拥有1 GHz Pentium III、512MB RAM和80 GB硬盘的机器将作为最低要求工作，但如您所知，最低要求和推荐要求是完全不同的。您可以从Microsoft的WHS PDF指南(链接如下)查看最低和推荐的系统要求。

Windows Home Server控制台

若要将网络上的计算机连接到服务器，您需要安装可单独下载的Windows Home Server连接器。您需要将WHS连接器安装到网络上要与服务器连接的每台计算机上。它将您的计算机连接到WHS，允许夜间自动备份，监视计算机网络运行状况，并使您能够从计算机远程管理服务器。

如果你不想浪费一张CD来安装它，你可以使用一个免费的程序，如Virtual CloneDrive，挂载ISO并将文件传输到闪存驱动器。

在XP上安装连接器之前，您需要安装Microsoft.NET Framework2.0，如果需要，它将指导您完成此过程。

打开WHS控制台时，系统会提示您登录服务器。这里还有一些其他功能，比如记住密码、密码提示和重置控制台。

登录后，您可以配置和管理服务器及其不同功能。我们将在以后的文章中更详细地介绍WHS控制台。

如果您需要对服务器进行更详细的更改，另一个好的选择是使用远程桌面，这比将显示器、键盘和鼠标连接到远程桌面更容易。

下载更新

现在安装已完成，我们需要添加来自Microsoft的所有最新更新。单击开始\所有程序，然后单击Windows更新。这将为您提供最新的服务器安全更新和Power Pack更新。Power Pack是一些更新，可添加新功能并修复Windows Home Server的已知问题。

结束 / 结尾 / 结论 / 推论

本指南将帮助您开始使用新的家庭服务器。如果你想集中你的数字娱乐，并从任何运行XP或更高版本的机器上访问它，它在很多很酷的方式下都会派上用场。或者，如果您有一个家庭或小型办公室，并且希望能够从任何有网络连接的地方访问您的工作。如果您是高级用户，有几个很好的理由让您在家里拥有自己的服务器。在接下来的一年里，我们将为您提供关于如何设置和使用您的家庭服务器的更详细的教程。如果你碰巧有一台不用的台式机，并且可以专门用作服务器，他们会提供30天的免费试用，这样你就可以自己试用了。如果你喜欢它，你可以购买一个完整的许可证，我在网上看到过，价格在99美元左右，或者更便宜一些，这取决于你去哪里看。
