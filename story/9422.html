如何查找任何设备的IP地址、MAC地址和其他网络连接详细信息

每台联网设备-计算机、智能手机、平板电脑、智能家居小工具等-都有一个IP地址和一个唯一的MAC地址，可以在您的网络上识别它。下面是如何找到你可能随处可见的所有设备上的信息。

在本教程中，我们将讨论如何在您的本地网络中查找设备的IP地址，通常称为专用IP地址。您的本地网络可能使用路由器连接到Internet。该路由器还将有一个公共IP地址-一个在公共互联网上标识它的地址。要查找您的公共IP地址，您可能需要登录到路由器的管理页面。

Windows 10

在Windows 10上，您可以比在以前版本的Windows上更快地找到此信息。如果您是通过Wi-Fi连接的，请单击任务栏最右侧系统托盘中的Wi-Fi图标，然后单击“网络设置”链接。

在“设置”窗口中，单击“高级选项”。(您也可以通过打开设置应用程序并导航到网络&互联网>Wi-Fi来访问此窗口。)。向下滚动，您将在“属性”部分看到此信息。

如果您使用的是有线连接，请转到“设置”>“网络和互联网”>“以太网”。在右侧，您将看到列出了您的连接。单击您想要的。

稍微向下滚动到“属性”部分，你就会找到你想要的信息。

Windows 7、8、8.1和10

您可以通过其他方式在以前版本的Windows上找到此信息-旧方法在Windows10上仍然有效。

转到“控制面板”>“网络和共享”(或Windows7中的“网络和互联网”)，然后单击“更改适配器设置”链接。

右键单击需要其信息的连接，然后从上下文菜单中选择“状态”。

在“Ethernet Status”(以太网状态)窗口中，单击“Details”(详细信息)按钮。

“Network Connection Details”(网络连接详细信息)窗口将显示您需要的信息。请注意，MAC地址被列为“物理地址”。

您还可以通过打开命令提示符并运行以下命令在任何版本的Windows中查找此信息：

ipconfig

MacOS X

如果你是通过Wi-Fi连接的，在MacOS X中查找这些信息的最快方法是按住“Option”键，然后单击屏幕顶部菜单栏上的Wi-Fi图标。使用Option键，还可以快速访问Mac OS X中其他地方的状态信息。

你会在“IP地址”旁边看到你的Mac的IP地址。此处的其他详细信息将显示有关您的无线网络和路由器IP地址的信息。

无论您的连接是无线的还是有线的，您都可以通过打开Apple菜单，然后转到系统首选项>网络来查找此信息。选择您的网络连接，然后单击“高级”。您可以在“TCP/IP”选项卡上找到IP地址信息，在“Hardware”选项卡上找到MAC地址。

iPhone和iPad

要在运行苹果iOS的iPhone、iPad或iPod Touch上查找这些信息，请首先进入“设置”>“Wi-Fi”。轻触任何无线网络连接右侧的“i”图标。您将在此处看到IP地址和其他网络详细信息。

要查找您的MAC地址，请转到设置>常规>关于。稍微向下滚动一点，你会看到你的MAC地址被列为“Wi-Fi地址”。

（科幻小说用语）机器人

在Android上，你可以在设置应用程序中找到这些信息。从屏幕顶部向下拉并轻触齿轮图标，或打开您的应用程序抽屉并轻触“设置”应用程序图标即可将其打开。

点击无线和网络下的“Wi-Fi”选项，点击菜单按钮，然后点击“高级”以打开高级Wi-Fi屏幕。您会发现IP地址和MAC地址显示在此页面的底部。

就像在Android上一样，这些选项的位置可能会略有不同，具体取决于您的制造商如何定制您的设备。上述过程是在运行Android 6.0棉花糖的Nexus7上执行的。

ChromeOS

在Chromebook、Chromebox或任何其他运行Chrome OS的设备上，您可以在设置屏幕上找到此信息。

点击屏幕右下角的状态区，点击弹出式列表中的“Connected to[Wi-Fi Network Name]”选项，然后点击你所连接的网络名称。你也可以通过点击Chrome中的菜单按钮，选择“设置”，然后点击你所连接的Wi-Fi网络的名称来进入。

您可以在“连接”选项卡上找到IP地址信息，在“网络”选项卡上找到MAC地址。

<计算机>Linux 操作系

在现代Linux系统上，应该可以很容易地从状态或通知区域访问此信息。查找网络图标，单击它，然后选择“连接信息”。您将在此处看到IP地址和其他信息-MAC地址被列为“硬件地址”。

至少，在大多数Linux发行版现在使用的NetworkManager中是这样的。

如果您只能访问终端，请运行以下命令。忽略“lo”接口，它是本地环回接口。在下面的屏幕截图中，“eth0”是要查看的网络连接。

ifconfig

从游戏机到机顶盒，这一过程在其他设备上也是类似的。您应该能够打开设置屏幕并查找显示此信息的“状态”屏幕、可能在某处显示网络连接详细信息的“网络”屏幕或“关于”屏幕上的信息列表。如果找不到这些详细信息，请对您的特定设备执行网络搜索。
