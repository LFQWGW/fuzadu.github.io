如何在Ubuntu上安装KVM和创建虚拟机

如果您使用的是Linux，则不需要VirtualBox或VMware来创建虚拟机。您可以使用KVM-基于内核的虚拟机-在虚拟机中运行Windows和Linux。

您可以直接使用KVM，也可以将其与其他命令行工具配合使用，但是使用过其他虚拟机程序的人会对图形化Virtual Machine Manager(Virt-Manager)应用程序最为熟悉。

创建虚拟机

一旦安装了KVM，使用它的最简单方式就是使用Virtual Machine Manager应用程序。你会在你的仪表盘里找到的。

单击工具栏上的Create New Virtual Machine按钮，Virtual Machine Manager将引导您选择安装方法、配置虚拟机的虚拟硬件以及安装所选的来宾操作系统。

如果您使用过VirtualBox、VMware或其他虚拟机应用程序，您会熟悉该过程。您可以从光盘、ISO映像甚至网络位置安装。

要为虚拟机分配超过2 GB的内存，您需要64位Linux内核。运行32位内核的系统最多可以为虚拟机分配2 GB的RAM。

默认情况下，KVM为您提供类似NAT的桥接网络-您的虚拟机不会作为自己的设备出现在网络上，但它将通过主机操作系统访问网络。如果您在虚拟机中运行服务器软件，并且想要从网络上的其他设备访问它，则必须调整网络设置。

选择安装方法后，Virt-Manager将在窗口中引导来宾操作系统。像在物理机上一样安装来宾操作系统。

管理虚拟机

Virtual Machine Manager窗口显示已安装虚拟机的列表。在窗口中右键单击虚拟机以执行操作，包括启动、关闭、克隆或迁移它们。

您可以通过单击虚拟机窗口中的I形工具栏图标来查看有关虚拟机的信息并配置其虚拟硬件。
