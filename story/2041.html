在Windows Web平台上运行WordPress和其他Web应用程序

您是否希望在您的电脑上运行WordPress或其他Web应用程序，以便轻松测试和设计网站？下面我们将介绍如何在您的计算机上只需几个快速步骤即可获得最新的Web应用程序。

今天的许多Web应用程序，如WordPress、MediaWiki等，都是开源的，可以从任何一台甚至只有一个简单的本地Web服务器的计算机上免费运行。*它们通常很难安装在您的计算机上，因为它们需要许多依赖项，如PHP和MySQL。微软已经努力使这一点变得更容易，发布了Windows Web平台安装程序。这使得您只需单击几下就可以在Windows中安装许多流行的Web应用程序和免费工具。

这里我们将介绍如何安装WordPress和免费的Visual Web Developer 2010速成版，以便使用Web平台安装程序编辑Web代码。但是，如果您更愿意安装不同的Web应用程序或工具，请随意选择它们，因为安装一般都是相似的。

开始

前往Microsoft的Web开发站点并下载Web平台安装程序(链接如下)。*这将非常快速地下载，因为它只是一个小型加载程序。*当您运行此加载程序时，它将下载Web平台安装程序文件。Web平台安装程序可在XP、Vista和Windows 7以及相关版本的Windows Server上运行。

片刻之后，Web平台安装程序将打开并加载有关最新Web产品的信息。

现在您可以选择要安装的产品。您可以快速选择多个类别(如Web服务器、数据库等)的推荐产品。

或者，单击类别下的自定义

并准确选择要安装的内容。请注意，您计算机上已安装的项目将呈灰色显示。

我们希望安装Visual Web Developer 2010速成版，因此请选择“工具”下的“自定义”，然后选择“Visual Web Developer 2010速成版”。

或者，要获得更多预设选项，请选择窗口底部的选项。

您可以选择将多媒体、开发人员和企业工具添加到列表中，也可以从提要添加新的预设列表。

选择要安装的特定Web应用程序

我们想安装WordPress，所以不要选择预设，而是选择左侧的Web应用程序选项卡。现在，您可以根据类别从各种应用程序中进行选择，也可以按从A到Z、最受欢迎或最高评级的列表一起查看它们。

单击要安装的应用程序旁边的复选框将其选中，或单击“i”以了解详细信息。

这是WordPress的更多信息窗格。*如果您准备安装它，请单击该复选框。

现在，如果您愿意，您可以返回并将更多Web应用或工具添加到安装列表中。*Web平台安装程序会自动查找并选择MySQL等必备应用，因此您不需要为查找它们而烦恼。

一旦您选择了要安装的所有内容，请单击窗口底部的Install(安装)按钮。

Web平台安装程序现在将向您显示所选的所有内容，包括它自动选择的组件。请注意，我们只选择了安装WordPress和Visual Web Developer 2010速成版，但它也自动选择了MySQL和PHP。*单击我接受继续。

在安装开始之前输入MySQL的管理员密码。

现在，Web平台安装程序将接管您的所有Web应用程序，自动下载、安装和配置您的所有Web应用程序。它还将激活您的计算机上可能需要的可选Windows组件。*这可能需要几分钟时间，具体取决于您选择的组件和您的网速。

设置您的测试站点

安装完成后，系统将要求您输入有关您的站点的一些信息。您可以简单地接受默认值或输入您自己的选择，然后单击“继续”。

现在，您需要为您的Web应用程序输入一些信息。*安装WordPress时，您需要选择一个数据库并输入管理用户名和密码。*可能还会要求您输入额外的信息以提高安全性，但对于仅限本地的测试站点，这不是必需的。*完成后，请单击“继续”。

在它完成你的Web应用的安装过程中，你还需要等待几分钟。好的是，一旦安装完成，它们就可以只需最少的配置就可以运行了。

您就完成了！安装程序会告诉您它安装的所有内容，以及是否有任何问题。在我们的测试中，Visual Web Developer 2010速成版未能成功安装。通常问题可能出在下载上，因此单击“完成”，然后重新选择未安装并再次运行安装程序的应用程序。

现在您可以从PC上运行wordpress了。单击启动wordpress链接或在浏览器中输入http://localhost:80/wordpress即可开始。

您只需在WordPress上再做一点设置即可使其运行。*在浏览器中打开WordPress页面后，请输入博客名称和电子邮件地址，然后单击“安装WordPress”。

几秒钟后，您应该会看到成功！使用您的用户名和临时密码登录页面。请复制密码，然后单击“登录”。

输入admin作为用户名，粘贴随机生成的密码，然后单击Log in。

WordPress将提醒您更改默认密码。/单击“是，带我转到我的个人资料页”链接以执行此操作。

输入您更容易记住的内容，然后单击更新配置文件。

现在你就可以在Windows上享受新的WordPress安装了。你可以添加插件和主题，以及使用普通WordPress站点所能做的一切。*这是从localhost运行的仪表板。

这是默认运行的博客。

设置Visual Web Developer 2010学习版

如前所述，Visual Web Developer 2010学习版在第一次尝试时未正确安装，但在第二次无缝安装。*安装后，照常从[开始]菜单启动。*第一次运行时可能需要几分钟才能加载，因为它正在完成安装。

您可能会注意到，加载程序时显示的闪屏显示仅用于评估目的。这是因为您仍需要注册程序。

您有30天的时间来注册该程序，但让我们继续注册以避免此步骤。-单击菜单栏中的帮助，然后选择注册产品。

在弹出窗口中单击在线获取注册密钥。

您需要使用您的Windows Live ID登录，然后填写快速表单。

完成后，复制显示的注册密钥并将其粘贴到Visual Web developer中的注册对话框中。

现在，您已经注册了一个完全符合标准和智能感知的免费Web开发程序，可以帮助您更智能、更快速地工作。而且，它与您本地的Web应用程序配合得很好，因此您可以使用这个简单的安装程序从桌面创建、调整和部署所有内容！

安装更多应用程序

您可以随时在将来再次运行Web平台安装程序，并根据需要添加更多应用程序。安装过程会在[开始]菜单中添加一个指向安装程序的链接；只需运行它，然后对您的新选择重复上述步骤即可。

此外，如果需要，您还可以从安装程序清理安装过程中下载的安装文件。1单击窗口底部的选项链接，然后向下滚动并选择删除安装程序缓存文件夹。

不幸的是，卸载应用程序并不那么容易。如果您希望卸载Web平台安装程序以及随其安装的所有内容，则需要单独卸载每个项目。要查看所有一起安装的内容，一个简单的方法是按日期对卸载程序中的条目进行排序。*在我们的情况下，我们还在同一天安装了其他一些应用程序，但更容易看到哪些是一起安装的。

或者，如果你不喜欢使用“程序和功能”来卸载它们，可以试试像Revo Uninstaller Pro这样的程序。

结束 / 结尾 / 结论 / 推论

无论您是全职Web开发人员，还是只是喜欢测试最新的Web应用程序，Web平台安装程序都能让您的计算机快速轻松地加载最新的组件。事实上，安装这些工具及其所有依赖项比安装许多标准的盒装程序要容易得多。
