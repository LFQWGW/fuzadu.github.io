如何在MacOS中启用按键重复

任何使用Mac的人都知道，输入特殊字符真的很容易-你只需按住一个字母即可。如果你想和你的朋友玛娜一起玩精灵宝可梦GO，那就太棒了，但如果你真的很想现在就玩，那就不是那么好了。

相关：如何在OS X中通过两次按键输入特殊字符

通常，在MacOS上，当你按住一个键时，它会显示一个弹出窗口，允许你选择一个特殊字符(如果有任何分配给该特定键的字符)。

如果钥匙后面没有任何特殊字符，而您按住它，则不会发生任何事情。

您可以重复一些键，如空格键、退格键和箭头键，但仅此而已。那么，你怎么解决这个问题呢？

你可以把它切换回更传统的学校行为-按住钥匙就可以重复-但这并不太方便。你真的不能两种选择都有，因为你必须跳过的圈套让它成为一种这样或那样的交易。

你必须在终端中使用一个命令，这看起来似乎很容易，但为了使其生效，你必须重新启动计算机。想把它改回去吗？输入命令，然后再次重新启动。

为此，请首先从Utilities(实用程序)>Applications(应用程序)文件夹中打开终端。

在终端打开的情况下，现在输入以下命令并按Return键。

defaults write -g ApplePressAndHoldEnabled -bool false

接下来，重新启动计算机，您现在应该能够重复所有字符。

不过，在继续之前，您可能不想确保调整重复速度以符合您的喜好。为此，请先打开系统首选项，然后单击打开“键盘”。

应用的控件是键重复和延迟到重复。

第一个选项将让您指定按键重复的速度。请注意，如果它处于禁用状态，则关键点根本不会重复。

“延迟到重复”选项允许您决定按住某个键多长时间后才开始重复。

如果要重新启用特殊字符，则需要在终端中再次输入命令，将最后一部分更改为“true”，然后再次重新启动计算机。

defaults write -g ApplePressAndHoldEnabled -bool true

诚然，这不是在重复字符和特殊字符之间切换的最方便的方式。虽然你可能不会经常使用其中的一个，这看起来一定是个大问题，但有足够多的Mac用户在问同样的问题。

如果苹果只需在键盘首选项面板中添加一个复选框，让你根据需要启用或禁用特殊字符，那会方便得多。

不幸的是，从粗略的谷歌搜索来看，这是一个至少从OS X Lion开始人们就一直在关注的问题，所以很明显，苹果认为这是不需要解决的问题。
