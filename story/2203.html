确保Microsoft Security Essentials已更新定义文件

如果您是Microsoft Security Essentials的新用户，您可能不清楚如何手动或自动更新定义文件。在这里，我们将向您展示一些方法，您可以使用它们来确保MSE始终是最新的。

如果MSE需要更新，它将在任务栏中显示为橙色感叹号图标，让您知道需要注意。

手动更新MSE 1.0和2.0 Beta定义

有几种方法可以手动更新MSE 1.0和2.0测试版。首先，您可以打开它并转到更新选项卡，然后单击更新按钮。

另一种方法是使用Windows更新。从“开始”菜单或任务栏(如果已将其固定在任务栏中)打开Windows Update。

如果您每天手动检查Windows更新，很可能总是会有新的定义更新。

您也可以前往Microsoft恶意软件防护中心，下载最新的定义文件并手动安装。

然后以管理员身份运行mpam-fe.exe，您就会得到更新。

MSE 1.0和2.0测试版的自动更新

要确保在扫描之前获得最新的恶意软件定义文件，请转到设置选项卡，然后转到计划扫描。确保在选择运行排定扫描之前检查最新的病毒和间谍软件定义。

这将确保MSE在扫描之前下载最新的定义。

如果您将Windows Update设置为每天自动下载和安装更新，请确保并选中给我推荐的更新，方式与我接收重要更新的方式相同。这样，您每天都能得到最新的定义文件。

MSE更新实用程序

如果您不想启用Windows更新的自动安装，有一个名为MSE更新实用程序的第三方应用程序可以提供帮助。只需下载并安装它，接受默认设置即可。

安装后，它将在任务栏中运行，并确保您拥有最新的定义文件，具体取决于您如何安排它。

如果要确保始终处于最新状态，请将更新间隔设置为“每天”，然后选中“启动时运行”。这将适用于MSE V1.0，但不适用于新的MSE Beta 2.0。

结束 / 结尾 / 结论 / 推论

无论您是手动还是自动操作，如果您是Microsoft Security Essentials的新手，这些快速提示将有助于确保MSE始终是最新的。如果您还没有试用过，请查看我们的MSE2.0Beta屏幕截图之旅。
