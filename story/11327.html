如何在Ubuntu Linux上安装最新版本的Flash

在2012年放弃Flash for Linux后，Adobe在2016年重启了Firefox和Linux上其他浏览器的Flash插件，但Ubuntu默认情况下仍会安装旧版本的Flash，除非你不厌其烦地安装新版本。

最新版本的Flash比旧版本更安全，提供了额外的攻击保护。它也应该更加稳定。Chrome和Chromium“PPAPI”版本的插件还包括更多针对网站的功能，比如硬件3D加速和对网络视频的DRM支持。这绝对值得拥有…。只需要几个额外的步骤就可以得到。

过时的选项：在安装Ubuntu的同时安装Flash

Ubuntu在安装时会提供“为图形和Wi-Fi硬件、Flash、MP3和其他媒体安装第三方软件”复选框。

但是，此选项并不理想。它安装在Flash的最旧版本-11.2版中。在撰写本文时，可以使用以下说明获得Flash版本23。

它还只安装Flash的NPAPI版本。它没有安装适用于Chromium的PPAPI版本。如果你在Ubuntu上使用Flash，你需要按照下面的说明安装最新版本。

简单的选择：我只需使用Google Chrome

如果你使用谷歌Chrome，你根本不需要安装Flash。Google Chrome使用自己的捆绑Flash插件(不要与Chromium混淆，Chromium不是随Flash一起提供的)。它与您的浏览器一起保持更新，确保您始终拥有最新版本。不幸的是，因为它不是一个系统范围的插件，所以没有其他浏览器可以使用Chrome的捆绑版本的Flash。

要在Ubuntu上安装Google Chrome，请转到Google Chrome下载页面，下载适用于Ubuntu系统的.deb文件，双击它，然后告诉Ubuntu安装它。

安装适用于Firefox、Chromium和其他浏览器的最新Flash插件

Canonical提供了更新版本的Flash插件包。此软件包包含Firefox所需的NPAPI插件和Chromium所需的PPAPI插件，因此它应该可以在系统上的所有浏览器中启用Flash支持。

首先，打开软件和更新工具。在仪表盘中搜索“软件”，然后单击“软件和更新”图标启动它。

单击“Other Software”(其他软件)选项卡并激活“Canonical Partners”存储库(如果尚未启用)。

在需要密码时提供密码，然后单击“关闭”按钮。如果存储库旁边已经有一个复选框，则不需要在此执行任何操作-只需单击“关闭”按钮即可。

您将被告知需要下载有关可用软件包的新信息。单击“重新加载”按钮，等待下载完成。

一旦完成，就可以安装最新版本的Flash了。你可以在Ubuntu软件应用程序中搜索“Adobe Flash”来做到这一点，但是我们发现终端更可靠一些。打开终端窗口并运行以下命令以安装Adobe Flash插件：

sudo apt install adobe-flashplugin

输入您的密码，并在系统提示您同意安装时键入“y”。

现在将安装Flash插件。您需要重新启动Firefox、Chromium或任何其他Web浏览器，然后它才能看到Flash插件并使用它。

此过程还会安装Adobe Flash Player首选项工具，类似于Flash在Windows和MacOS上提供的工具。要找到它，请在Dash中搜索“Flash”并启动它。您可以使用它来配置Flash插件设置。
