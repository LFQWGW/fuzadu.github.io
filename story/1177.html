使用Opera Unite将您的计算机转变为文件、音乐和Web服务器

想要通过Web与人们共享文档、照片和其他文件吗？现在您可以使用Opera Unite，这是Opera 10浏览器中包含的一项新服务。今天我们来看看Unite的新功能及其提供的功能。

根据他们的网站，“Opera Unite是一种协作技术，它使用Opera桌面浏览器内的紧凑型服务器来共享数据和服务。”本质上，这项服务是把你的电脑变成一个服务器，在那里你可以分享文档和照片、流式播放音乐、创建聊天室、托管你自己的网站等等。你可以让文件向公众开放，或者使用受密码保护的邀请链接，真正酷的是，访问它的人可以使用他们想要的任何浏览器。

就任 / 安顿 / 安装 / 就职

您要做的第一件事是安装Opera 10，在撰写本文时我们使用的是Beta 3 Build 1708。“Unite功能仍处于开发的Alpha阶段，需要与Opera Labs站点分开安装(我们包含了下面的链接)。”事实上，在其开发的这个阶段，您可以只安装Unite功能，但作为一个好奇的极客，我想看看它将如何与新版本的浏览器一起运行会是最好的。

要开始使用Unite，系统将提示您创建一个Opera帐户。

自定义设置并根据您将从何处使用自定义帐户来创建自定义帐户。

高级设置允许您指定上传速度、默认端口和全局可见性。

要完成安装，请选择用于保存共享文档和文件的文件夹。

你可以从侧边栏访问它，然后单击Unite选项卡。在这里，你可以打开和关闭该服务，并且会有一个可以添加的其他服务的列表，如冰箱、媒体播放器、Opera Messenger、照片共享、休息室和Web服务器。

统一要素

文件共享使您可以轻松共享文件和其他文档。

冰箱是一个整洁的地方，你邀请的其他人可以在你的电脑上保存笔记。

Media Player可以让您在任何有Internet连接的地方收听您的整个音乐收藏。

照片共享允许您与朋友和家人共享照片，而不必先将照片上传到单独的服务。

每项附加服务或功能都需要快速安装…。例如，我们在这里安装Opera Messenger。

Opera Unite主页将有一个正在运行的服务列表，您可以在这里找到朋友可以访问该网站的URL。

您可能希望为您运行的每项服务设置单独的目录，以确保工作有序、安全。例如，您不希望每个人都访问My Documents文件夹中的个人信息。

这是一个访问另一个用户的Unite页面的示例，他们在他们的文件共享页面上共享一本电子书，您可以在浏览器中查看文档或将文件下载到您的硬盘上。

结束 / 结尾 / 结论 / 推论

Opera Unite是一个非常酷的功能，将被添加到新的Opera浏览器中。它允许任何人在某种程度上成为自己的网站管理员，并使其变得极其简单。但有一件事引起了人们对这项服务的担忧，那就是它的安全级别。根据他们的网站和服务条款，“Opera Unite”在“安全沙箱”中运行，只有您选择的文件或文件夹才能访问；其他任何人都无法访问您计算机上的其他文件或文件夹。您授予对指定文件的访问权限，例如您想要共享的数据目录，而不允许访问指定区域之外的任何内容。“此外，他们不会保留您共享哪些文件的日志，也不要求对其拥有任何权利。”不共享不需要的数据基本上由您决定。

Opera在科技界肯定不是很受欢迎，但你可能想重新看看它，看看它的最新版本，作为Internet Explorer的另一个替代品。
