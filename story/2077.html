屏幕截图之旅：了解新的Office Web Apps

Office2010最常被谈论的新功能之一就是Web Apps。在这里，我们将带您参观Web Apps服务，以及您可以期待的内容。我们涵盖了亮点和一些需要改进的领域。

Office Web Apps是Office桌面应用的一个很好的对应物。它们可以很好地呈现文档，让您可以使用熟悉的工具快速编辑重要文档并进行协作。Office Web Apps主要在Office.live.com上运行，您可以通过浏览器和Office 2010集成来访问它们。Office Web Apps还可以由企业通过SharePoint 2010或托管的Microsoft Online Services在内部使用。*如果您是Facebook用户，您可以在新的Docs.com上与朋友协作，该Docs.com允许您在最适合您的任何地方使用Office Online。

这里我们将介绍Office Live上的Office Web Apps，因为它们与其他Windows Live服务(如Hotmail和SkyDrive)集成在一起。

Office Live入门

如前所述，Office Live是Office Web Apps的主要位置。请访问Office Live站点(下面的链接)，然后使用您的Windows Live ID登录或创建一个新ID(如果您没有Windows Live ID)。

欢迎使用您的Office Live仪表板…。第一次登录时，您会注意到界面相当稀疏。但是，如果您一直使用SkyDrive共享和保存文档，您会在新的Office Live仪表板中看到它们。

您可以单击右侧突出的应用程序图标之一，立即开始创建新文档、电子表格或演示文稿。

或者，您可以从从计算机快速添加文件开始。Office Web Apps支持Word、Excel、PowerPoint和OneNote文件，因此只需单击添加文件链接即可开始。

您上载的文档存储在SkyDrive上，因此请在您的SkyDrive中选择一个文件夹或创建一个新文件夹来存储您正在上载的文档。*如果您要与世界共享您的文档，您也可以将它们保存在公用文件夹中。

将您的文档拖到上载页面的拖放方框中，或单击链接选择要上载的文档。

您的文档将自动上载，如果需要，您可以在等待期间添加更多文档。当文档全部上载完毕后，请单击“继续”。

现在，您的新文件将可以直接从云中使用。*将鼠标悬停在所需文件上并从可用选项中选择。*您可以单击文档名称联机查看，选择“在浏览器中编辑”以在Office Web App中打开该文件类型的文件，或单击“更多”链接查看更多选项。

联机编辑文件后，可以单击版本历史记录链接查看该文件的以前版本。

在这里，我们在右侧预览了文档的当前版本，并可以从左侧的菜单中选择较旧的版本。这在与其他人协作处理文件时非常有用，因为即使有人删除了文件中的数据，您也可以确保永远不会丢失重要信息。

创建新文档

如果您希望创建新文件，请单击页面顶部附近的“新建”链接，然后选择所需的文件类型。

您还可以从任何Windows Live Web应用程序(如Hotmail、SkyDrive等)访问Office文档。将鼠标悬停在顶部的Office链接上，然后选择要创建的文件类型，或选择最近使用的文档。

快速浏览Office Web Apps

Office Web Apps的外观和工作方式与Office 2010相似，但功能较少。在我们的测试中，我们发现该功能集足以满足大多数快速编辑需求。*文件在我们的PC上呈现方式与Office 2010中的一样，这比其他一些在线办公应用程序有了很大的改进。*OneNote Web App非常适合协作，因为您可以在新的Messenger Web应用程序中同时实时编辑文件和聊天，所有这些都在同一个页面上。

Word Web App

Word可能是个人电脑上使用最多的Office应用程序，网络版给了它一些很好的功能。你不能用它做所有的事情，但它比写字板或其他轻量级文档编辑器有更多的功能。它的文档在网上呈现得很好，包括字体、样式、文本框和页脚，甚至复杂的脚本语言。

您可以通过单击工具栏上的“在浏览器中编辑”按钮直接在浏览器中编辑文档。*从“文件”菜单，您可以打印文档、下载文档或与他人共享文档。

Word Web App不包括Word的所有标准功能，但它确实包括许多重要功能，包括帮助样式列表。它还包括多种字体可供使用。*某些文档功能(如文本框)可能无法联机编辑，多栏文档可能在编辑器中显示为单栏。任何无法联机编辑的功能都将保留，因此它们在Word的联机查看器桌面版本中仍会正确显示。

但这并不是说你不能在Word Web App中创建复杂的文档。“主页”选项卡提供了许多你习惯使用的工具，包括多种语言的标题样式和拼写检查。

“插入”选项卡允许您添加表，或浏览以包括桌面版本中提供的Office剪贴库中的剪贴画。

如果您已插入图像，则会出现“图片”选项卡。在此，您可以更改图像的替代文本，以及放大、缩小或缩放图像。

插入表格同样会生成额外的功能区选项卡，您可以在其中更改表格的布局。这些额外选项卡中的这些额外功能增加了Web应用程序的可用性，但提供的选项仍比桌面应用程序少。

如果需要在文档中查找文本，请返回到阅读模式…。编辑模式不包括查找工具。它包含在阅读模式中，工作方式与桌面版本类似。

Excel Web App

当涉及到电子表格时，数字、公式和数据是最重要的，Excel Web App在这些方面做得很好。*它只包括基本工具，不允许您插入图形或其他高级图形。*它可以呈现图形，包括Excel 2010中的新迷你图功能。*如果您在电子表格中编辑数据，它将自动重新计算图形并进行相应更新。

Excel Web App仍然是数字和公式的行家里手，用户开始输入公式，它会自动打开智能感知风格的弹出窗口，里面有可用的公式，就像桌面上的Excel一样。输入公式时，可以像往常一样点击选择单元格。

您还可以轻松地对数据进行排序、筛选和格式化。

它允许多人在同一电子表格上进行实时协作，您可以通过底部的链接查看当前正在编辑的用户。与OneNote一样，您也可以使用Messenger直接与其他用户聊天。

PowerPoint Web应用程序

PowerPoint Web应用程序起初看起来并不像其他Office Web应用程序那样功能丰富，但它的功能出人意料地强大。它包括PowerPoint 2010中的各种模板，并允许您创建和编辑SmartArt图表。幻灯片呈现方式与桌面上PowerPoint中的效果相同，但高级动画(如PowerPoint 2010中包含的动画)将被标准的淡入效果所取代。您无法向联机创建的新演示文稿添加过渡效果。与其他应用程序一样，上载文件中不支持的功能将保留在中，因此它的呈现方式与您在桌面上的PowerPoint中打开时相同。

您可以在默认查看器中预览演示文稿。您可以从底部的菜单跳到特定幻灯片，或从工具栏中的按钮开始放映幻灯片。

这将打开全窗口演示文稿。*在大多数浏览器中按F11键，您将能够全屏查看和演示PowerPoint。*您可以使用移动鼠标时出现的箭头按钮或键盘上的箭头键移动到下一张幻灯片。*并且在缓存演示文稿后，它会非常快速地呈现幻灯片。

您可以联机编辑现有演示文稿或创建新演示文稿。*创建新演示文稿时，您可以从演示文稿的各种模板中进行选择。*一旦选择了主题，您就无法更改它。*您也不能像脱机那样编辑主题或更改配色方案，但由于模板种类繁多，因此不会破坏交易。

您现在可以在幻灯片中添加和编辑文本。请注意，您不能插入新的艺术字，但如果您的演示文稿在标题中包含艺术字，则它将自动应用于文本。

此外，字体颜色工具将显示您的主题中包含的颜色，这很不错。

您也可以从“开始”工具栏插入新幻灯片。“如果您插入新幻灯片，您可以从各种标准版式中选择，包括桌面版本中的标准版式。

即使演示文稿是从您的计算机上载的，PowerPoint Web App也会将演示文稿的主题应用到新幻灯片

“插入”选项卡看起来很稀疏，但“SmartArt”按钮原来是一个隐藏的宝石。“您可以从数十个SmartArt图表中进行选择，然后将它们直接插入到演示文稿中。

当您编辑时，功能区上将出现一个新的SmartArt选项卡。这令人惊讶地包括了SmartArt工具在完整PowerPoint中的几乎所有功能，包括光滑的3D样式。

遗憾的是，PowerPoint Web App不包含任何动画或过渡工具，因此您无法将其添加到在线演示文稿中。但我们对Web App中包含的主题和SmartArt的质量感到惊讶。

OneNote Web应用程序

如上所述，OneNote Web App可能是功能最齐全的。*它非常适合协作，并且允许多个用户同时编辑。*如果您已从计算机上载笔记本或将其与OneNote 2010同步，您的笔记本将包含几乎所有插入桌面应用程序中的功能和内容。

与其他网络应用程序一样，在线编辑器本身的功能比桌面编辑器少，但它比Evernote等其他笔记应用程序有更多的功能。*功能区选项卡具有与Word类似的功能，只是增加了标记按钮。

“插入”选项卡允许您输入新的页面和分区以及表格、图片和剪贴画。

由于OneNote是围绕协作而设计的，因此您可以使用视图选项卡来查看特定分区的作者。这将使用该分区的编辑器标记笔记本的每个部分。

我们发现的问题

虽然我们发现Office Web Apps非常有用，而且比我们预期的要好，但我们确实遇到了一些问题。*首先，如果您上载使用Office 2010预发布版本创建的文档，您可能无法对其进行编辑。*您可以很好地查看它们，但如果您尝试编辑它们，您可能会看到类似下面的提示。*如果您看到这种情况，请尝试在Office App的已发布版本(包括带有兼容包的Office 2003)中打开文档。*将文档另存为包含新文件的新文件

Web应用程序中不支持某些功能，例如Excel中的3D图形。请注意，不支持的功能将保留在文件本身中，并且在桌面应用程序中打开时仍会正确显示。

虽然OneNote和Excel Web应用程序支持共同创作，但Word和PowerPoint Web应用程序不支持。如果您尝试在另一个人同时编辑Word或PowerPoint文档，则可能会提示您Web应用程序无法打开该文档。

最后，一些用户可能会遇到的一个问题是，功能区在编辑区域中占用了大量空间。对于小屏幕的用户，或者浏览器中打开了许多工具栏的用户来说，这可能是一个问题。但就像在桌面版本中一样，您可以最小化功能区。或者只需双击任一功能区选项卡，或单击右上角的箭头按钮，功能区将最小化，只显示选项卡标签。

结束 / 结尾 / 结论 / 推论

虽然还有其他的在线办公套件，如Google Docs和Zoho，但MS Office Web Apps值得一看。“我们发现它与竞争对手的服务提供的功能相比非常有竞争力。当使用未安装标准Office的计算机时，它是标准Office的有用替代品。所有Web应用程序都可以很好地呈现Office中的文档，甚至可以让您编辑文档的大部分部分。

无论您在大多数流行浏览器上使用Windows、Linux还是Mac，您都将确保能够访问来自Microsoft的优秀Office应用程序。*如果您将所有重要文件保存在您的在线帐户中，您可以从任何地方访问这些文件，并与全球各地的同事进行协作。
