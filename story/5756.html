15个系统工具，您无需再在Windows上安装

Windows包含许多广泛使用的系统实用程序的自己版本。Windows8增加了各种新的实用程序，但其中许多实用程序也可以在Windows7上使用。

如果您在设置新的Windows安装后所做的第一件事是安装所有您喜欢的实用程序，这可以加快速度。更好的是，您遇到的每台Windows计算机上都有这些实用程序。

<计算机>抗计算机病毒的

Windows8包括一个名为Windows Defender的防病毒程序，因此所有Windows用户最终都将在他们的电脑上安装防病毒程序。Windows8不再麻烦用户在设置计算机后立即下载和安装杀毒软件。

Windows8中的Windows Defender本质上是具有新名称的Microsoft Security Essentials。如果您使用的是Windows7，您可以安装免费的Microsoft安全软件包。如果您使用的是Windows 8，则无需安装第三方防病毒软件。

防火墙 / 同 Chinese wall

如果您仍在使用第三方防火墙，您应该知道这并不是完全必要的。Windows的内置防火墙在默认情况下起到了同样的作用，即阻止未经请求的传入流量，并阻止访问公共Wi-Fi网络上的敏感网络服务，如网络文件共享。

喜欢调整他们的应用程序和管理允许连接到互联网的应用程序的用户会更喜欢第三方防火墙，但每个使用Windows XP SP2和更高版本的Windows用户都有坚固的防火墙。

安全套间

除了防病毒和防火墙功能外，Internet安全套件还包括反网络钓鱼、删除Cookie和其他安全功能。这些功能都不是必需的-您的浏览器包括内置的网络钓鱼保护，如果您希望删除Cookie，则可以在关闭它时自动删除Cookie。您不需要安装凌乱、臃肿的安全套件来保护您的系统。

在Windows8上尤其如此，它包括额外的安全功能，比如SmartScreen功能，它可以在运行所有应用程序之前扫描它们的可信性。

分区管理器

对于Windows上的基本分区管理，您不需要任何第三方工具。只需使用Windows附带的磁盘管理应用程序即可缩小和放大现有分区、创建新分区并对其进行格式化。尽管更高级的分区移动功能可能需要第三方工具，但此工具包含的功能对于大多数基本分区操作来说已经绰绰有余。

Windows 8上的存储空间功能甚至可以将多个驱动器上的分区合并为一个大的逻辑分区。

ISO和IMG文件装载

如果您需要装载ISO或IMG文件才能将其作为虚拟磁盘进行访问，则不再需要安装第三方工具。Windows8的文件资源管理器包括集成的光盘映像安装工具。需要挂载其他类型的光盘映像的用户或使用Windows 7的用户仍需要安装用于挂载光盘映像文件的实用程序。

光盘刻录

从Windows7开始，Windows集成了对将数据刻录到光盘、擦除可重写光盘，甚至直接将ISO映像文件刻录到光盘的支持。不需要第三方光盘刻录套件。如果您仍然需要刻录音频CD，您甚至可以在Windows Media Player中执行此操作。

PC清洁应用程序

PC清洁应用程序一般都是骗局，特别是那些想让你花钱“让你的电脑感觉像新的”的骗局。如果您确实希望删除临时文件并通常释放计算机上的存储空间，您可以使用Windows附带的免费磁盘清理工具。如果您确实需要第三方PC清洁工具，只需下载CCleaner-跳过所有付费PC清洁应用程序。

启动管理器

Windows 8在Windows任务管理器中内置了一个新的启动管理器。它允许您查看和控制在启动时随计算机启动的应用程序，最终提供一种简单而明显的方式来控制这一点。在以前的Windows版本中，用户必须使用隐藏的msconfig工具，或者使用内置在CCleaner中的启动管理器。

多个监视器实用程序

Windows 8具有跨多台显示器显示桌面任务栏的功能。此功能之前需要第三方多个监视器工具。虽然您可能仍需要第三方工具来高级调整多个显示器设置，但扩展任务栏不再需要这些工具。

文件复制

TeraCopy是Windows7和更早版本的Windows的流行文件复制替代产品。它允许您暂停文件复制操作，并以更智能的方式处理错误，而不是暂停操作并等待输入。

如果您升级到Windows 8，您会发现这些功能现在已经集成到文件资源管理器(以前称为Windows资源管理器)中，为所有用户提供了更强大、更智能的复制和移动文件的方式。

高级任务管理器

Process Explorer是Windows任务管理器的强大替代品。它显示更多信息，包括显示流程之间关系的分层树视图。这对于Google Chrome等生成大量进程的应用程序尤其有用。

在Windows 8上，新的任务管理器现在包含更多信息，并提供分层信息，使您可以轻松查看应用程序的进程，因此依赖Process Explorer实现此特定功能的用户将不再需要安装它。

系统信息实用程序

Windows包括各种系统信息实用程序。系统信息工具显示有关系统内硬件的信息，而资源监视器应用程序显示有关计算机资源使用情况的信息-您可以查看总的资源使用情况或深入查看哪些应用程序正在与哪些网络地址通信，或者哪些应用程序使用的磁盘输入/输出资源最多。

PDF查看器

Windows8包括PDF查看器，允许您在不安装Adobe Reader等程序的情况下查看PDF。不幸的是，Windows Reader是一款现代化的应用程序，不能很好地与桌面集成，尽管默认情况下所有PDF都可以从桌面打开。

不过，如果你使用Google Chrome或Mozilla Firefox，你的浏览器就有一个集成的PDF阅读器。您可以将PDF文件与Chrome或Firefox相关联，以便在您的浏览器中打开它们，如果您只需要查看没有任何高级功能的简单PDF文件，则无需使用第三方PDF阅读器。

虚拟机

Windows 8包括Hyper-V，无需安装VirtualBox或VMware等第三方虚拟机解决方案即可创建和使用虚拟机。我们已经介绍了如何使用Hyper-V在虚拟机中安装Ubuntu，而无需安装任何额外的软件。

备份工具

除非您有相当高级的备份需求，否则您可能会发现Windows为您提供了足够强大的备份工具。Windows 7有自己的备份和恢复功能，而Windows 8添加了文件历史记录，其工作原理类似于苹果的Time Machine。这两种工具都是开始备份本地文件的便捷方式，无需安装-或者更糟糕的是，购买第三方备份解决方案-的麻烦。

在Windows 7上，甚至在Windows 8上，这一功能仍然存在，你可以使用Windows 7的备份工具来创建整个操作系统的完整映像，将来可以从该映像进行恢复。

这里的大多数工具都不是最强大的选项，因此许多用户可能仍然更喜欢安装第三方软件并获得更多功能。然而，微软一直在缓慢地将重要软件添加到Windows中，每个版本都需要较少的必要第三方系统实用程序。
