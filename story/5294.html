如何使用桌面浏览器访问移动网站

许多网站为智能手机、iPad和其他移动设备提供特定的界面。无论您是需要测试移动网站，还是只是好奇地想看看它们是什么样子，您都可以在桌面浏览器中访问它们。

您可以通过更改浏览器的用户代理来做到这一点-我们之前已经解释过什么是浏览器用户代理。使用iPad用户代理访问网站可能还会迫使它为您提供HTML5视频，如果您试图避免使用Flash，这是很好的选择。

用户代理切换器扩展

我们已经介绍了如何在不安装任何其他软件的情况下更改浏览器的用户代理。但是，为了便于使用，您可能希望安装一个浏览器扩展，使您可以快速轻松地更改浏览器的用户代理。

您需要安装适用于Chrome的User-Agent Switcher或适用于Firefox的User Agent Switcher，具体取决于您使用的浏览器。对于Internet Explorer，您可能希望尝试UAPick User-Agent Switcher附加组件。

设置移动用户代理

要更改用户代理，请在浏览器工具栏上找到User Agent Switcher Extension图标，单击该图标，然后在列表中选择移动用户代理。

(安装后，您可能需要将User Agent Switcher图标添加到Firefox的工具栏中。为此，请右键单击工具栏，选择自定义，然后将User Agent Switcher图标拖放到Firefox的工具栏上。)

刷新你当前所在的页面(点击工具栏上的刷新图标或者直接按F5)，你就会看到它的移动版本。您可以访问其他网站，只要您的用户代理设置为移动用户代理，您就可以看到它们的移动版本。

完成后，选择Default User Agent(默认用户代理)选项。

该过程在其他扩展中类似。即使您使用的是浏览器的内置用户代理切换器，这也应该是一个相当简单的过程。

其他用户代理

一些用户代理切换器没有提供全面的用户代理列表。例如，Firefox的user Agent Switcher扩展不包括允许您将用户代理设置为iPad的选项。

您可以通过单击User Agent Switcher图标并选择Edit User Agents来下载其他用户代理。单击要导入的用户代理的下载列表链接，您将能够下载和导入更全面的用户代理列表。

如果您更愿意手动输入用户代理字符串，您可以在诸如Mobile Browser ID(User-Agent)Strings网站之类的网站上找到它们。
