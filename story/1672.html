如何使用Wine在Ubuntu 9.10中安装Spotify

Spotify可以说是在线提供的最好的流媒体音乐服务。Spotify可以在Windows和Mac上使用。不过，通过使用Wine，你也可以安装Spotify在Ubuntu上运行。

注意：Spotify并非在所有国家或美国都可用。下面我们看一下如何使用Wine1.2在Ubuntu9.10上安装Spotify。

首先，我们需要在Ubuntu上安装Wine。如果您已经有了Wine，您可以很容易地跳过这一步。要在Ubuntu9.10上安装Wine，首先进入System\Administration\Synaptic Package Manager打开Synaptic Package Manager。

系统将提示您输入密码。输入您的管理员密码，这将打开Synaptic Package Manager。

选择包装葡萄酒1.2并单击Apply。这将在您的Ubuntu上安装Wine，它位于Applications\Wine下。

现在您已经安装了Wine。您只需下载Windows版本的Spotify并将其安装在您的Wine上即可。为此，请下载Spotify(windows版本)。下载后，导航到Download文件夹，右键单击Spotify安装程序，然后单击使用Wine Windows Program Loader打开。

这将在您的Ubuntu机器上安装Spotify。

现在，如果你住在美国以外的地方，你可以在你的Ubuntu机器上享受Spotify。另外，请务必查看我们对Spotify的评论。
