防病毒软件会让你的电脑变慢吗？也许你应该用免责条款

当你使用Windows PC时，用防病毒解决方案保护你的计算机是理所当然的，但不幸的是，它同时会减慢你的速度。下面是如何提高你的表现，至少是一点点。

我们不会坐在这里告诉你不要安装杀毒软件，因为那是不负责任的。我们今天要做的是解释如何使用写入繁重的操作来排除某些文件夹，从而在不将您自己置于额外危险的情况下提高您的PC速度。

注意：在开始排除任何文件之前，请记住，更改任何默认安全设置都可能有风险，您可能应该关闭浏览器选项卡并逃跑。或者把文章打印出来烧掉。还有，图片是由xkcd提供的。

您应该排除哪些文件？

一般的想法是，如果您有一些经常写入硬盘的应用程序，您可能应该排除它们正在写入或读取的文件夹，只要这些应用程序是可信和安全的。

例如，如果您使用的虚拟机在相当恒定的基础上从硬盘读取和写入，则应确保防病毒应用程序不会扫描这些文件和文件夹。下面是一些你可以考虑排除的例子：

虚拟机目录：如果您使用的是VMware或VirtualBox，则应确保排除这些位置。这实际上是促使本文撰写这篇文章的原因，也可能是我们提到的事情带来的唯一显著的性能提升。 Subversion/TortoiseSVN文件夹：您有没有尝试过对源代码控制项目进行大型签出，但失败了？它很可能与您的防病毒应用程序冲突。这件事发生在我个人身上。 个人照片/视频文件夹：您有用数码相机拍摄的大量照片或视频吗？只要您只将此位置用于从SD卡复制的文件，就没有理由在进行照片编辑时扫描它并降低您的PC速度。 合法音乐文件夹：如果你从可疑来源下载音乐，这不适用。如果你翻录了自己的CD，或者从亚马逊等合法的地方下载，那么你可以安全地排除你的音乐文件夹。 Windows Update文件夹：这实际上直接来自Microsoft知识库文章-你会注意到他们不推荐这样做，因为他们不能这样做，以防有人为了起诉他们而编写特殊的病毒，但同样的原则也适用。

其他场景

每个人的PC上都有几乎无限多的应用程序和场景，所以很难准确地说出在您的PC上运行哪些应用程序-但有一种方法可以让您自己使用进程监视器，这是微软Sysinterals提供的一个很棒的工具。

只需打开process Monitor，然后“取消选中”工具栏右侧的所有小图标，只选中“显示文件系统活动”。此时，您将看到列表中项目的加载和加载，以及对文件系统的每次访问。

您可以使用这些数据来确定哪些应用程序在不断地读写硬盘，然后根据这些文件的安全性选择是否排除它们。

不排除文件类型，排除文件夹

当您排除文件名或文件扩展名时，您是在告诉防病毒软件完全忽略系统中任何位置的这些文件，这可能会导致安全问题。更好的办法是排除您知道安全的特定文件夹，如您的虚拟机文件夹。

始终从Internet扫描文件

我首先要说的是，这应该不用说…。不管怎么说，似乎总是这样说的，…。但你应该绝对确保扫描来自互联网上任何地方的任何文件，特别是当这些文件来自Torrent或其他类似的病毒泛滥的来源时。

所有防病毒应用程序的工作方式都不同

接下来要提到的是，并不是每个防病毒应用程序都会以相同的方式工作-例如，默认情况下，AVG只扫描一组特定的文件扩展名，而不扫描没有扩展名的文件。如果不扫描这些扩展，就无法判断(至少在没有基准测试的情况下)排除文件夹是否会造成性能差异。

然而，其他一些防病毒应用程序并不局限于特定的文件类型，因此您需要深入研究特定应用程序的设置。

从Microsoft Security Essentials中排除文件

Microsoft Security Essentials使排除文件变得非常简单-只需进入“设置”，选择左侧的“排除的文件和位置”，然后将文件夹添加到右侧的列表中。

您可能会注意到，MSE无论如何都不会使您的系统变慢很多。

从AVG Anti-Virus中排除文件

正如我们前面提到的，如果你进入“工具”->“高级设置”…，AVG的工作原理会稍有不同

然后进入Resident Shield->Advanced Settings查看当前正在扫描的文件类型列表。您将注意到，AVG总是扫描没有扩展名的文件，这通常不会造成问题，但根据您使用的应用程序的不同，可能会出现问题。

您可以切换到Resident Shield->Excluded Items，将其添加到文件夹或特定文件中，以便从扫描中排除。

我们不会涵盖所有其他防病毒应用程序，但它们的工作原理大致相同。此外，我们更喜欢Microsoft Security Essentials。
