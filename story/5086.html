正如你可以想象的那样，使用闪存驱动器比从DVD安装Windows更快，但它仍然不能很好地扩展。为了尝试并消除可伸缩性问题，我们还可以从普通网络共享安装Windows。

虽然这一开始听起来可能是一个可行的选择，但它肯定不是没有缺陷的。您知道，为了访问网络上的安装文件，您仍然需要从包含WinPE(Windows预安装环境)的DVD或USB引导每台PC，然后您可以使用命令行连接到网络共享并开始安装。然而，大规模部署Windows7中最常用的方法是一种称为WDS(Windows部署服务)的技术。要使用WDS，您必须具备以下条件：

运行Windows Server 2008或更高版本的服务器。 您的环境需要具有Active Directory、DHCP和DNS。 您的客户端必须支持PXE Boot(因为安装将通过网络完成)。

如果您已经具备了所有这些条件，那么与普通网络部署相比，您将获得巨大的优势，例如：

组播

在典型的网络安装期间(不使用WDS时)，客户端将首先将3.5 GB的Windows安装文件下载到每台PC，然后再执行实际安装。这种方法既繁琐又浪费带宽。当使用多播而不是每台PC从服务器提取自己的文件时，服务器将使用一对多的方法将文件复制到您的客户端，从而节省大量带宽。

更新

由于源文件存储在单个存储库中，因此您只需更新WDS服务器上的映像。这样，您执行的任何新安装都将自动安装最新的Service Pack。如果您的环境中有新的硬件，您还可以滑入驱动程序。

自动化 / 自动操作

虽然使用普通应答文件和网络共享可以实现部分自动化，但通过使用Windows AIK(自动安装工具包)、SIM(系统映像管理器)和SCCM(系统中心配置管理器)等工具以及Windows部署服务，您可以实现完全自动化。请记住，无论您选择哪种方法安装Windows(包括WDS)，您都需要更改PC上的BIOS设置才能从正确的硬件启动。

清洁安装

当我们谈到全新安装时，它通常意味着硬盘驱动器要么是空的，要么是要格式化的，并且以前的操作系统将会丢失。这与保留文件和设置的升级不同。从Windows 7开始，可以在已包含操作系统的驱动器上进行全新安装。在这种情况下，系统会警告您旧的Windows安装将被移到硬盘驱动器根目录下的Windows.old文件夹中。需要全新安装的一个示例是当您想要更改操作系统的体系结构时。例如，如果要将Windows的x86(32位)副本更改为x64(64位)副本。

双引导

我们在How-to Geek这里是虚拟化的忠实粉丝，然而这是一个相当新的概念。在快速启动虚拟机的日子之前，如果我们想要尝试新的操作系统，或者上帝保佑我们必须支持传统软件，我们将不得不双引导我们的计算机。简而言之，这意味着我们必须同时运行两个截然不同的操作系统。

如今，几乎没有任何令人信服的理由需要双启动。但是，就考试目标而言，您需要知道在资源受限的情况下可能会进行双引导。您还应该注意，这至少需要您的硬盘上有两个分区。你也可以选择两个单独的驱动器，这个规则的一个例外是新的“从VHD引导”功能，它允许你在不对硬盘分区的情况下运行双引导Windows7机器。最后，请记住，在双引导时，先安装旧操作系统总是更容易，这样Windows会自动为您设置引导菜单。

注意：由于这是一次微软考试，我们将坚持使用微软软件，但您应该注意到，没有什么可以阻止您同时启动Windows和Linux。

选择正确的Windows 7版本

Windows价格昂贵已不是什么秘密，在企业环境中更是如此。节省许可成本的关键是选择正确的版本。让我们来看看其中的一些不同之处。

Windows 7简易版

虽然你不能去当地的电脑商店买一份初学者版，但你确实需要知道它的考试目标。

可用性：OEM。 缺少功能：Aero、DVD播放器、Windows Media Center、IIS、Internet连接共享、域成员资格、EFS、Applocker、DirectAccess、BitLocker、远程桌面主机、BrancheCache。 其他限制：2 GB内存限制、仅1个物理CPU、5个已安装的应用程序、仅限x86

Windows7Home Basic

家庭普通版具有与入门版相同的功能，只是应用程序有限制。它还允许在x86安装上使用高达4 GB的内存，在x64安装上支持8 GB内存。

Windows 7家庭高级版

Windows7家庭高级版提供了学生或标准家庭用户在PC上需要的所有功能。

可用性：OEM和零售业。 缺少功能：域成员资格、EFS、Applocker、DirectAccess、BitLocker、远程桌面主机、BrancheCache。 其他限制：4 GB内存(X86)或16 GB内存(X64)，2个物理CPU

Windows 7专业人员

针对小型企业主，Windows7专业版大大提高了系统的内存限制。

可用性：OEM和零售业。 缺少功能：Applocker、DirectAccess、BitLocker、BrancheCache。 其他限制：4 GB内存(X86)或192 GB内存(X64)，2个物理CPU

Windows 7企业

你也不能从商店购买Windows7企业版，因为它只能通过批量许可才能买到。

可用性：仅限批量许可客户。 附加：从VHD启动。 其他限制：4 GB内存(X86)或192 GB内存(X64)，2个物理CPU

Windows 7极限

旗舰版面向极客和发烧友，包含企业版的所有功能。它也可以在零售店购买。

可用性：OEM和零售业。 附加：从VHD启动。 其他限制：4 GB内存(X86)或192 GB内存(X64)，2个物理CPU
