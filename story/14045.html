如何备份您的东西并切换到新的Mac

换掉你的Mac的时候到了。一旦你决定了一个新的型号，支付了现金，打开了昂贵的新玩具的盒子，就是时候开始做生意并转移你的数据了。

您需要的所有工具都与MacOS捆绑在一起(或多或少使用一两根电缆，或者一个外置驱动器)。而且，如果你买了一台新的Mac，苹果会免费转移你的数据。

苹果将免费迁移您的数据

如果你不想自己转移数据，苹果的员工可以帮你转移。你只要带上你的旧Mac电脑就可以了。苹果过去对这项服务收费99美元，但现在，任何购买新Mac的人都可以免费使用。

对于MacBook和Mac迷你机型来说，这很有意义。不过，如果你有一台旧的iMac或Mac Pro，你最好在家里传输数据。

您的选择

苹果在MacOS中有一个名为Migration Assistant的工具，可以帮助您进行两端的数据传输。在Migration Assistant中，您可以通过以下三种方式之一传输数据：

Mac到Mac，通过本地网络(或机器之间的本地无线连接)。 使用目标磁盘模式和电缆的Mac到Mac。 从存储在外部驱动器上的Time Machine备份还原。

如果您想要使用第一个选项，有几件事需要牢记。如果两台机器都运行MacOS Sierra或更高版本，那么您可以通过本地Wi-Fi连接进行无线传输。此连接直接在两台机器之间进行，因此不需要两台机器连接到同一网络。

如果您的Mac较旧，并且只支持OS X El Capitan，则必须确保两台计算机都连接到同一网络。你可以通过Wi-Fi实现这一点，或者为了获得更好的效果，可以使用以太网线。迁移助手允许你从运行OS X Snow Leopard 10.6.8或更高版本的旧Mac电脑传输数据。

不确定您的Mac运行的是哪个MacOS或OS X版本？单击屏幕左上角的Apple徽标，然后选择关于此Mac。您应该会看到概述选项卡上列出的当前操作系统的名称和版本号。

选项1：通过网络实现Mac到Mac

传输Mac内容最简单的方法之一(不需要电缆或外部驱动器)是通过网络连接。

尽管简单，但这是将您的数据迁移到新Mac的最慢方式。如果您有大量数据(超过200 GB)要传输，您应该计划几个小时才能完成。你甚至可能要等上一夜。

给您带来优势的一件事是如果您使用有线网络连接。有线传输不仅更可靠，而且速度更快。“有线”连接意味着您的新旧计算机都通过以太网电缆连接到路由器。通过这种方式传输不需要互联网连接，因此您甚至可以临时重新定位您的网络设备来加快传输速度。

按照以下步骤通过网络将您的数据从Mac传输到Mac：

在您的旧Mac上，启动Migration Assistant并单击“继续”。 出现提示时，选择“转到另一台Mac”，然后单击“继续”。 在您的新Mac上，启动Migration Assistant并单击“继续”。 出现提示时，选择“从Mac、Time Machine Backup或启动盘”，然后单击“继续”。 出现提示时选择Mac图标，然后单击“继续”。 注意安全代码，并检查它是否与您的旧Mac(如果提供)上的安全代码匹配。 选择要从旧Mac传输的信息，然后单击“继续”。

现在，您只需等待传输完成即可。如果您转移到共享相同用户帐户的Mac，系统将要求您重命名或替换新Mac上的用户帐户。如果你选择更换账户，所有数据都将被删除，但由于Mac是新的，这应该无关紧要。

选项2：通过电缆实现Mac到Mac

目标磁盘模式是一种比通过网络传输驱动器内容更快的方法。如果您用高速电缆将旧Mac直接连接到新Mac，则可以显著缩短整体传输时间。

目标磁盘模式适用于Thunderbolt3、USB 3.0或更高版本、Thunderbolt2和FireWire。尽管支持USB 3.0，但如果您想通过USB Type-A连接，则必须使用2012 Mac或更高版本。您的新Mac将需要使用USB Type-C接口。

以下是一些需要牢记的事情：

你可以用苹果迅雷3(USB-C)电缆连接迅雷3到迅雷3，或者迅雷3到USB 3.0 Type-C。 如果您从迅雷2接口复制，您可以使用带有迅雷2转迅雷3(USB-C)适配器的标准迅雷2电缆。 要从USB Type-A复制到USB Type-C，请确保使用支持USB 3.0或更高版本的电缆(USB 2.0不起作用)。 您不能使用为MacBook充电时使用的USB Type-C电缆。

您必须在目标磁盘模式下重新启动旧Mac才能正常工作。这样做时，它会将旧Mac的驱动器挂载到新Mac上，因此它会显示为外部卷。如果您已使用FileVault Encryption保护驱动器，则必须在挂载时键入密码才能对其进行解密。

按照以下步骤通过电缆将数据从Mac传输到Mac：

将传输电缆从旧Mac连接到新Mac。 打开您的旧Mac电脑，并在其启动时按住“T”键。如果已经打开，请转到“系统首选项”>“启动盘”，然后单击“目标盘模式”。等待它重新启动。 在您的新Mac上，等待旧Mac的驱动器出现。如果出现提示，请键入您的FileVault密码。如果你看不到你的旧Mac，启动Disk Utility并检查侧边栏。出现卷时选择该卷，单击“文件”(File)>“装载”(Mount)，然后键入您的FileVault密码。 挂载驱动器后，启动Migration Assistant，然后单击“继续”。 出现提示时，选择“从Mac、Time Machine Backup或启动盘”，然后单击“继续”。 单击相关启动盘的图标。 选择要传输到新Mac的数据，然后单击“继续”。

不能让你的旧Mac出现吗？确保使用正确的电缆或适配器连接两台机器。为了在迅雷2或3连接上获得最佳效果，请仅使用Apple品牌的电缆。

传输时间最终取决于您使用的电缆。有源铜制迅雷3电缆价格昂贵，但提供高达40 Gbps(最高。每秒5 GB的速度)。迅雷2电缆提供的速度大约是这个数字的一半(20 Gbps)，而USB 3.1和3.0分别管理10 Gbps和5 Gbps。

传输完成后，像弹出其他驱动器一样弹出旧Mac的驱动器。

方法3：从时光机备份

如果您使用Time Machine将Mac备份到外部驱动器，则此最后一种数据传输方法非常理想。您只需将备份驱动器连接到新Mac，然后使用Migration Assistant导入数据即可。在继续之前，请确保在旧Mac上创建最新的Time Machine备份。

此方法通常也比使用网络连接更快-特别是当您的驱动器支持USB 3.0或更高版本时。

按照以下步骤从Time Machine备份传输数据：

在您的新Mac上，启动Migration Assistant，然后单击“继续”。 出现提示时，选择“从Mac、Time Machine Backup或启动盘”，然后单击“继续”。 出现提示时选择Time Machine图标，然后单击“继续”。 当您看到可用备份列表时，请选择一个(您可能需要最新的备份)。 选择要从旧Mac传输的数据，然后单击“继续”。

别忘了安全地弹出你的时光机驱动器。如果您打算使用此驱动器备份您的新Mac，则需要再次设置Time Machine。

大数据迁移

你不必使用移民助手，但它让生活变得容易得多。不过，您也可以在目标磁盘模式下连接您的旧Mac，然后手动复制您想要保留的任何文件。或者，您也可以通过“系统偏好设置”>“共享”使整个Mac驱动器在网络上可用。

现在是决定如何处理你的旧Mac的时候了。你可以把它当作一个联网的时光机驱动器，擦除你的驱动器并从头开始安装MacOS，或者把它卖了，然后收回你花在升级上的一些钱。
