用于Chromebook和PC的30多种基于Web的传统桌面应用替代方案

无论你是使用Chromebook，还是只对从安装的桌面应用程序切换到基于浏览器的应用程序感兴趣，基于网络的软件都可以取代人们在电脑上使用的许多程序-而且往往可以改进它们。

使用基于网络的应用程序的一个好处是，它们将在你的设备上进行本地同步-例如，到你的智能手机上。您的数据将始终备份，并可从您的计算机访问。

办公室 / 官职 / 好事 / 政府机关

除非您是要求苛刻的Office用户，并且使用Microsoft Office桌面版本中的宏和其他高级功能，否则您可能会对基于Web的办公套件同样满意，或者甚至更满意，因为基于Web的办公套件是免费的。

Google Docs允许您使用内置到Google Drive中的文档、工作表和幻灯片应用程序打开和创建文档、电子表格和演示文稿。Microsoft的Office Web Apps允许您使用基于Web的Word、Excel、PowerPoint和OneNote版本，其界面类似于桌面应用。

我们在过去已经更详细地介绍了这两个办公套件-阅读我们对Google Docs和Office Web Apps的概述以获取更多信息。最棒的是，两者都是完全免费的。

生产力 / 生产率 / 繁殖力 / 生产能力

其他生产力应用程序也以基于Web的形式提供：

日历：Google Calendar提供了一个功能齐全的在线日历解决方案，具有重复活动、可用性安排和许多其他功能。微软最近还将其基于网络的日历从乏味的旧Live Calendar更新为Windows8风格的日历应用程序。两者都是可靠的选择。谷歌日历是更成熟的应用程序，但如果你与微软的生态系统捆绑在一起，微软的日历可能是理想的-例如，它可以与Windows 8日历应用程序同步，而谷歌日历不能再与Windows 8同步。 任务：Google在Gmail和Google Calendar中内置了自己的简单任务管理器。它的功能不是很全面，但如果你需要一些简单的东西，可以从你的电子邮件和日历中访问，它可以做到这一点。任务管理器是一种相当常见的应用程序类型，还有许多其他类型的应用程序可用-包括Remember the Milk、Any.DO等。它们都可以同步到你的智能手机上，这样你就可以随时随地访问它们。 备注：毫无疑问，Evernote是最受欢迎的在线笔记应用。Evernote不仅仅用于笔记-它可以捕获和存储图像、文档和其他类型的文件。Evernote也可以通过Windows桌面应用程序访问，所以你不必在浏览器中使用它。还有许多其他的笔记应用程序，从Minimal Simplenote到Google的新Google Keep笔记解决方案，不一而足。微软的OneNote也可以通过Office Web Apps以Web应用的形式提供。

传播 / 交流 / 通信 / 联系方法

不出所料，通信应用程序在浏览器中运行得非常好--无论您是想要收发电子邮件、发送即时消息，还是甚至是进行语音和视频聊天。

电子邮件：三个最常见的基于Web的电子邮件系统是Google的Gmail、微软的Outlook.com(以前的Hotmail)和Yahoo！邮件。Gmail曾经是无可争辩的赢家，但微软正与Outlook.com展开更激烈的竞争。就连雅虎现在也在认真改进雅虎！邮件和其他网络应用程序。我们更喜欢Gmail，但Outlook.com肯定是一个竞争者-如果你使用微软的服务，或者仍然使用旧的@hotmail.com地址作为你的主电子邮件地址，甚至可能会更好。无论您选择哪种解决方案，您都可以转发来自旧帐户的邮件，并使用Gmail或Outlook.com将它们合并到一个电子邮件收件箱中。 通讯：许多网站都内置了通讯功能，Gmail内置了Google talk，Outlook.com内置了Skype和MSN，Facebook的网站内置了Facebook聊天。如果你还在使用其他即时通讯网络，比如aim和yahoo，或者只是想让你所有的朋友都聚集在一个地方聊天，我们推荐imo.im作为多协议桌面即时通讯程序(如pigin和trillian)的强大的基于网络的替代方案。 语音和视频聊天：您可以通过Google Talk或Google Hangout通过Gmail或Google+进行语音和视频聊天。如果你想使用Skype，你可以使用Outlook.com内置的Skype功能。然而，Skype需要一个插件，所以它不能在Chromebook上运行--但它可以在Windows PC上运行。 社交网络：当然，你不需要为社交网站开发新的网络应用程序--不管怎样，每个人都在使用这些网站的网络应用程序版本。这是一个很好的提醒，提醒我们Web应用程序是多么普遍-我们都在使用Web应用程序，从社交网络到网上银行和购物，从社交网络到网上银行和购物，我们都没有把它们当作Web应用程序。

演奏音乐

当谈到在线音乐服务时，我们的选择太多了。无论您是想将收藏的MP3文件上传到云端并获得随时随地收听的能力，还是想订阅让您可以收听1700多万首歌曲的自助音乐收听服务，还是想使用类似收音机的流媒体服务来选择为您播放什么，您都有各种各样的选择。

音乐寄存柜：流行的基于网络的音乐寄存柜包括亚马逊的Cloud Player和Google Music(苹果的iCloud不能通过浏览器访问)。这些服务将扫描你的电脑中的音乐，并让你通过网站和移动应用程序访问该音乐的副本。如果他们还没有那首音乐的拷贝，他们会帮你上传的。然后，您可以在浏览器中从任何位置访问您的音乐收藏，而不必担心万一您的音乐收藏的硬盘出现故障时备份它。亚马逊和谷歌音乐都会向你出售歌曲，并允许你将它们存储在你的在线音乐储存柜中，以及下载它们。这些服务是免费的-如果你想购买一首歌曲或存储大量歌曲，你必须付费。 随心所欲的聆听：如果你想完全放弃维护音乐收藏，可以使用Spotify、Rdio或MOG等音乐流媒体服务。其中许多服务提供一些免费套餐，而付费套餐每月收费5-10美元。你可以访问超过1000万首歌曲(例如，Spotify上的1700多万首)，你可以在浏览器或移动应用程序中聆听，而不必担心购买或下载音乐-只需搜索和播放即可。 流媒体电台：Pandora允许你根据你喜欢的歌曲和艺术家建立自己的流媒体电台。您可以跳过歌曲，但Pandora会为您选择播放列表。Slaacker还提供流媒体广播服务，其中包括各种专业电台以及您可以制作的定制电台。TuneIn Radio是最像传统收音机的--它允许你收听基于网络的流媒体电台，就像你过去在Winamp里面播放的那种。TuneIn Radio还可以让您访问许多在线流媒体的地面广播电台。

观看视频

基于网络的视频服务也有所改善，逐渐减少了对BitTorrent客户端和本地视频播放器或iTunes等桌面媒体商店的需求。不幸的是，这些服务大多主要在美国运作。

Netflix、Hulu和Amazon Instant Video是最受欢迎、选择最多的流媒体视频服务。你还可以经常在合适的电视网网站上观看各种最近的电视剧集。许多网站都允许你在线租赁电影--甚至YouTube也允许你购买和租赁电影和电视节目。

在线视频流媒体服务有点复杂-与音乐服务不同，你可能必须使用几个服务才能获得所有你想看的视频。许多服务也仅限于美国-Netflix并非如此，但在其他地方的目录非常有限-让世界其他地方的用户运气不佳。

更多应用程序

以下是您可能希望在浏览器中执行的其他一些常见任务：

图像编辑：如果你已经在Google+Photos(前身为Picasa)这样的服务中存储了你的照片，该服务可以自动从你的Android设备上传照片，你很可能会在你的图像存储服务中找到一个基本的照片编辑器。如果你在寻找更强大的功能，你应该试一试Pixlr-它不是Photoshop，但对于基于浏览器的图像编辑器来说，它的功能出人意料地强大。 视频编辑：信不信由你，即使你有Chromebook，你也可以在浏览器中录制和编辑视频。YouTube包括一个基本的视频编辑器，如果你将视频上传到YouTube，它就可以工作，而WeVideo可能是最强大的基于浏览器的视频编辑器。 会计：如果你使用会计程序来跟踪你的财务状况，你就不一定再需要Quicken了。Intuit旗下的Mint.com是一个财务管理网站，它将自动跟踪您的银行账户、信用卡和投资余额，在一个地方显示您的所有余额和交易。Intuit还开发了Quicken、QuickBooks和其他流行的程序。您可以制定预算、对事务处理进行分类和查看报告。Mint.com很方便，因为它会自动为您提取大量数据，消除了繁忙的工作。

这当然不是你可能想要在浏览器中做的所有事情的详尽列表，也不会列出你可以用来完成这些任务的每一个web应用程序。无论您是使用Chromebook还是对Windows PC上的Web应用感兴趣，我们都尝试包含一些最常见的任务和最高质量的Web应用来帮助您入门。
