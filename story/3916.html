如何从命令提示符更改计算机的IP地址

使用控制面板更改PC上的IP地址很容易，但您知道也可以从命令提示符执行此操作吗？

使用控制面板界面更改您的IP地址并不困难，但需要在多个不同的窗口和对话框中单击。不过，如果您是命令提示符的粉丝，则可以使用netsh命令更快地完成此操作，这只是Windows中内置的最好的网络实用程序之一。

netsh命令允许您在Windows中配置网络连接的几乎所有方面。要使用它，您需要使用管理权限打开命令提示符。在Windows 10或8.1中，右键单击[开始]菜单(或按键盘上的[Windows+X])，然后选择“Command Prompt(Admin)”(命令提示符(管理))。在以前版本的Windows中，在“开始”中搜索“命令提示符”，然后右键单击结果并选择“以管理员身份运行”。

查看您的网络信息

在更改IP地址和相关信息之前，您需要找到要更改的接口的网络全名。要执行此操作，请键入以下命令：

netsh interface ipv4 show config

向下滚动，直到看到您要查找的界面。在我们的示例中，我们将修改Wi-Fi接口，该接口在我们的机器上仅命名为“Wi-Fi”。您还将看到Windows分配给接口的其他默认名称，例如“Local Area Connection”、“Local Area Connection*2”和“Ethernet”。只要找到你要找的那个，并记下确切的名字就行了。您还可以将名称复制并粘贴到记事本，然后稍后再粘贴回命令提示符，以简化操作。

更改您的IP地址、子网掩码和默认网关

有了接口名称，您就可以更改IP地址、子网掩码和网关了。为此，您将使用以下语法发出命令：

netsh interface ipv4 set address name="YOUR INTERFACE NAME" static IP_ADDRESS SUBNET_MASK GATEWAY

因此，例如，您的命令可能如下所示：

netsh interface ipv4 set address name="Wi-Fi" static 192.168.3.8 255.255.255.0 192.168.3.1

这里的信息会被你想用的任何东西取代。在我们的示例中，该命令执行以下操作：

使用接口名称“Wi-Fi” 将IP地址设置为192.168.3.1。 将子网掩码设置为255.255.255.0。 将默认网关设置为192.168.3.1

如果您正在使用静态IP地址，但希望切换到使用DHCP服务器(例如您的路由器)自动分配的IP地址，则可以改用以下命令：

netsh interface ipv4 set address name=”YOUR INTERFACE NAME” source=dhcp

更改您的DNS设置

您还可以使用netsh命令更改网络接口使用的DNS服务器。第三方DNS服务器(如Google Public DNS和OpenDNS)可能比您的ISP提供的DNS服务器更快、更可靠。无论您更改DNS服务器的原因是什么，您都可以在路由器上进行更改，以便影响从路由器获取信息的所有设备，也可以在单个设备上进行更改。如果您只想更改一台PC的DNS服务器，使用netsh命令很容易。

您需要使用该命令两次：一次用于设置主DNS服务器，另一次用于设置辅助或备份DNS服务器。要设置主DNS服务器，请使用以下语法：

netsh interface ipv4 set dns name="YOUR INTERFACE NAME" static DNS_SERVER

因此，例如，您的命令可能如下所示(其中我们将其设置为Google的主公共DNS服务器8.8.8.8)：

netsh interface ipv4 set dns name="Wi-Fi" static 8.8.8.8

要设置辅助DNS服务器，您将使用非常类似的命令：

netsh interface ipv4 set dns name="YOUR INTERFACE NAME" static DNS_SERVER index=2

因此，继续我们的示例，您可以将您的辅助DNS设置为Google Public DNS辅助服务器，即8.8.4.4：

netsh interface ipv4 set dns name="Wi-Fi" static 8.8.4.4 index=2

就像IP地址一样，您也可以更改它，以便网络接口自动从DHCP服务器获取其DNS设置。只需使用以下命令：

netsh interface ipv4 set dnsservers name"YOUR INTERFACE NAME" source=dhcp

现在你就知道了。无论您是更喜欢在命令提示符下键入，还是只想给您的同事留下深刻印象，现在您都知道更改IP地址设置所需的所有命令行魔力了。
