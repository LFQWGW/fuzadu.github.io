安全计算：Windows Live OneCare

当Windows Live OneCare第一次问世并处于早期阶段时，许多人认为它还有很多不足之处，但它在开发过程中已经走过了很长的路，提供了更好、更有用的功能。这款由Microsoft提供的一体式安全套件包括基本安全功能等。

包括改进的防火墙，病毒和间谍软件保护，反身份盗窃，性能调整，以及易于配置的集中备份。它最显著的特点是能够同时控制3台PC。

对于本文，我将在Windows Vista旗舰版64上安装它。我想在其他版本的Vista和XP中，该过程非常相似。*如果您知道其他版本的Windows有任何问题，或者想要分享您使用Windows Live OneCare的经验，请一定要留言，让我们知道！

启动Live OneCare安装过程后，它会立即检查更新。然后，安装程序的其余部分开始。首先，您需要选择适当的语言，阅读软件套件的简要说明，然后单击下一步。

接下来，系统将提示您阅读并接受EULA和使用条款。之后，Windows Live OneCare将下载并安装。使用我的DSL连接，大约需要10分钟才能完成。

加载完所有内容后，需要重新启动。

返回后，您将看到某种欢迎屏幕。在这里，您可以选择步骤1，这将要求您购买许可证(49.95美元)，或者选择有限试用，这是我将选择的。由于这是我第一次使用Windows Live OneCare，所以我想在90天的试用期间测试所有东西。

我马上注意到的另一件事是几个气球和其他消息，指出OneCare认为我的系统缺乏安全性。

在选择之后，我想继续免费试用，微软觉得有必要通过再次询问来确保我对此有信心。

第二次是一种魅力，Windows Live OneCare已经准备就绪，并且已经开始工作。

在启动与Last.fm相关的AIMP Audio Player时，OneCare防火墙问我是否想编程访问互联网，这是我注意到的一个改进(也需要习惯)。“这绝对是对Windows本机防火墙的改进，Windows本地防火墙只阻止传入的连接。”一开始这有点恼人，但如果你允许每个已知的安全程序，这将是最后一次询问你。

您可能记得第一次安装Vista时选择了弹出的下一个设置。“OneCare要选择PC连接到的网络连接类型。”

Windows Live OneCare控制中心是更改设置、运行扫描和关联其他安全任务的位置。

Windows Live OneCare设置允许您配置不同保护组件如何工作的各个方面。

PC调整中的可计划功能之一。这将执行典型的维护任务，如整理硬盘碎片、扫描病毒和检查Microsoft的安全更新。

您可以使用OneCare圈管理多台基于Windows的PC(最多三台)。此外，使用Windows Live ID登录还可以让您从OneCare试用版中获益更多。

运行病毒和间谍软件手动扫描时，您将获得三种扫描类型可供选择：快速扫描、完整扫描或自定义扫描。使用所有扫描，您还可以保留扫描报告的存档。

无论您选择哪种扫描类型，都会显示进度屏幕。如果需要，此屏幕还允许您暂停扫描。

一次成功的健康扫描！

定期会有通知，告诉您有关尚未安装的Microsoft安全更新的信息。这些通知提供了更新的简要详细说明，还允许您连接到Microsoft网站以获取更多信息。

结论：

由于总体上使用Windows Live OneCare，所以我认为它是一个非常好的家用计算机安全解决方案。它的侵入性不是很强，而且问题似乎比第三方一体化安全套件有时遇到的问题要少。安全套件实用程序(如Norton或McAfee)确实会将它们自己深深地锁定在您的操作系统中，可能会有问题。如果我要推荐一个一体化解决方案，那么它将是一个设计为无缝集成的解决方案。如果微软默认将其包含在Windows操作系统中，那就太好了，我希望微软能在默认情况下将其包含在Windows操作系统中，这将是一种很好的解决方案，因为它的设计是无缝集成的。如果微软默认将其包括在Windows操作系统中，那就太好了，也许Windows 7？OneCare可以在任何装有Windows XP Home SP2或更高版本以及所有Vista(32或64位)版本的PC上运行。注*注意XP 64位版本不受支持。
