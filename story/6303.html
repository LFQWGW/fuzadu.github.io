Windows 8.1中您应该知道的前12个PC设置

自从Windows8.1发布以来，情况发生了很大变化。与其说是控制面板，不如说是以Metro为主题的“PC设置”，这无疑承担了越来越多的责任；许多控制面板终于被引入到PC设置中，随着微软推出新版本的Windows 8，我们可能可以预期这一趋势将继续下去。

前段时间，我们发表了一篇关于如何在Windows8中找到控制面板的文章。令我们惊讶或并不意外的是，它获得了相当多的评论。考虑到Windows8用户在浏览新界面时明显感到困惑，我们觉得是时候更新或加强我们原来的范围和重点，解释一下“PC设置”中的一些新功能，以及比较它与老式桌面控制面板可能存在的任何相似之处。

不幸的是，要涵盖所有这些设置需要的时间要比我们花在一篇简单文章上的时间长得多，而且相当多的设置-例如“锁定屏幕”设置-是相当不言而喻的。下面是我们认为你应该知道的12个最重要的Windows8.1电脑设置(没有按重要性排序)，以及在必要的情况下，它们与同等的桌面控制面板相比有何不同。

但首先，快速回顾一下…

在Windows8.1中查找控制面板很容易，在这里我们将使它变得更容易。如果您使用的是键盘/鼠标，请右键单击“Start”按钮或使用“Win键+X”。

在弹出的菜单中，选择“控制面板”：

如果您使用的是触摸屏(在桌面上)，请从屏幕右边缘向左滑动，点击“设置”咒语，然后点击“控制面板”：

“电脑设置”是怎么回事？

我们在前一篇文章中没有介绍的“PC设置”是控制面板的“Metro”版本。这些电脑设置中的许多都是开始屏幕界面独有的，都是为了让平板电脑或触摸屏用户可以轻松操作和配置设备，而不必费力点击桌面上较小的控件，如按钮和滑块。

要使用鼠标打开电脑设置(在开始屏幕上)，请将指针移到屏幕的右上角或右下角，单击“设置”咒语，然后单击底部的“电脑设置”-要使用触摸操作，请从屏幕右边缘向左滑动来激活咒语。

使用键盘导航到PC设置或围绕PC设置非常简单。您可以切换到点击“Win Key+W”并直接搜索您想要的设置，或“Win Key+I”，然后使用向下箭头选择“更改PC设置”并按“Enter”打开。

PC和设备

PC设置分为九个类别，这些类别又细分为相关主题。例如，“PC和设备”分为以下九个设置区域：

如果您的设备安装了Windows 8.1，您应该花时间彻底了解每一个PC设置。请注意，您的系统可能具有其他设备上未显示的设置，反之亦然。例如，如果您有蓝牙适配器，则会出现“蓝牙”设置，就像如果您有触摸屏而不是使用物理键盘，“键入”设置将显示更多选项。

表演 / 展览 / 显示 / 炫耀

“显示”设置是您习惯于在“控制面板->显示->屏幕分辨率”中看到的-当您在桌面上单击鼠标右键并选择“屏幕分辨率”时，您也可以访问它。

虽然Metro版本看起来不错，而且触摸起来很容易使用，但我们更喜欢旧版本。新的显示器设置没有检测到连接到显卡的第三个显示器(屏幕截图中的第一个显示器)，因此需要求助于旧的方法。

器件

“设备”设置类似于“设备和打印机”，不同之处在于“设备”更精简，不会像控制面板版本那样显示多媒体设备。

控制面板的功能与Metro基本相同，不过蓝牙设备在归类为“设备和打印机”时会有自己的设置。

角和边

万岁的“角落和边缘”设置！简而言之，您可以禁用热点和Metro风格的应用程序切换(从左侧边缘滑动)，这非常棒，特别是在台式机上！

这也值得在“任务栏和导航”控制面板上添加一个新的“导航”选项卡(右键单击“任务栏”->“属性”)。

电力和睡眠

与相应的控制面板相比，电源和睡眠设置菜单非常基本。您可以使用此功能对设备的电源方案进行最基本的调整。

但是，如果您真的想调节设备功耗和性能，您仍然需要使用“电源选项”控制面板，然后单击“高级选项”。

帐目

“帐户”设置和“用户帐户”控制面板有相似之处，但微软在PC设置中加入了最基本的功能(添加和删除用户)，同时将其从控制面板中删除。因此，除了最高级的用户帐户管理之外，并不一定要使用控制面板版本。

你的帐户

您可以将您的帐户与Microsoft“断开”(将其设置为本地帐户)，然后更改/创建您的帐户图片。“您的帐户”设置添加了将现有帐户更改为儿童帐户的选项，尽管具有讽刺意味的是，Microsoft家庭安全设置仍然是一个仅限控制面板的选项。

如果您确实想要完善您的帐户，例如添加更多个人身份信息，您将需要登录到Live.com网站上的Microsoft帐户-使用“More Account Settings Online(更多在线帐户设置)”选项。

登录选项

如果把“登录选项”命名为“密码选项”，那么“登录选项”设置会更清晰一些，但是名字里有什么呢？需要注意的重要一点是，您的所有设备安全设置都在这里，因此有人可能会说这是最重要的设置组之一(如果不是这样的话)。

这似乎是您更改密码和添加其他登录选项的唯一方法。它是仅有的具有系统范围影响的PC设置之一，但没有等效的控制面板，因此您想知道它在哪里以及如何使用。

同样值得注意的是“PIN”选项，它允许你通过4位数字快速访问你的设备，以及大肆宣传的“图片密码”，它允许你用在图片上画的手势打开你的设备(在触摸屏上比用鼠标更有效)。

总体而言，除非您有主密码，否则不能使用后两个选项，当然您有主密码，因为每个人的密码都会保护他们的系统，对吗？

其他账户

在这里的“其他帐户”设置中，您可以添加、编辑和删除用户帐户。请注意，当您“编辑”帐户时，可以将帐户更改为三种类型之一：管理员、标准或子帐户。

最后，您可以为分配的访问设置一个帐户，我们认为最好将其描述为Kiosk模式。基本上，如果您希望一个或多个用户使用一个或多个应用程序，您可以使用此设置分配该用户。如果您想要限制孩子的访问权限或在企业中使用平板电脑(如销售点设备)，这很可能很有用。

搜索和应用程序

“搜索和应用程序”中的大部分内容都与开始屏幕(Windows应用商店应用程序)相关，但是某些设置也会影响桌面行为(特别是“默认设置”)。

查找 / 搜寻 / 详查 / 调查

让任何更喜欢谷歌的人(几乎所有人)都很沮丧的是，必应是Windows8.1的首选搜索引擎(它是微软的产品，所以我们不能说它让我们感到惊讶)。这意味着您现在可以选择在Windows搜索中查看Bing结果。

谢天谢地，你可以把它全部关掉，让你的搜索保持在本地，也就是说，你不会把你的搜索词发送到必应的服务器上。我们不确定Bing是否或多长时间存储本地搜索，但我们假设这与您使用Bing.com的策略相同。

此外，如果你想掩盖你的踪迹，你可以清除你的“搜索历史”。

通知

应用程序通知，也就是吐司通知，是从右上角滑出的彩色地铁风格的小条，通知你日历日期和闹钟等等。

这里没有什么复杂的事情，老实说，吐司通知有点烦人。所以你可以一下子把它们都关掉。

如果你发现一些通知很有用，比如日历事件，但你不想在每个Facebook“赞”或收到新邮件时都收到祝酒词通知，你可以在每个应用程序的基础上关闭它们。

这些设置仅适用于Windows应用商店应用，因此你仍需要使用“通知区域图标”控制面板来调整桌面应用程序通知你的方式。

应用程序大小

“应用程序大小”类似于“程序和功能”控制面板，只是它适用于Windows应用商店应用程序。它会显示应用程序的大小，如果它们占用了太多空间，还可以让您快速卸载它们。这也是一种快速、可靠的方式，可以删除系统中预装的所有无用应用程序(只需将其发布)。

缺省值

“默认”设置执行“默认程序”控制面板的操作。第一个屏幕允许您设置最常见的应用程序默认值(Web浏览器、电子邮件、音乐播放器等)，但…

…。在底部还有“按类型选择默认应用”和协议的链接。

请注意，您可以将桌面应用程序设置为默认设置，因此这些设置确实会在系统范围内产生影响。

网状物 / 网状系统 / 人际网 / 网络

总体而言，我们认为Windows网络有所倒退。可以说，你只能通过“网络”设置来做这么多事情，但是同样的，你也可以在“网络和共享中心”控制面板中做同样的事情。

连接

“连接”设置允许您查看和配置现有的网络连接，无论它们是有线、无线还是虚拟专用网络(VPN)。

您还可以设置您的连接的可发现性(用于公用或专用网络)，这意味着您的设备对其他设备(如Microsoft家庭组)或公用网络上的设备显示或保持隐藏状态。如果你在星巴克工作很多，最好禁用“查找设备和内容”，当你在家或在私人网络上时，重新启用它。

最后，您可以将任何连接的属性复制到剪贴板，还可以查看数据使用详细信息(对于移动网络，即计量网络)。

天驱

今天的一切都充满了品牌的机会：苹果有iCloud，谷歌有Drive，微软有SkyDrive(目前)。

SkyDrive很好地集成到了Windows8.1中，就像iCloud可以同步Mac桌面上的各种数据和设置一样，SkyDrive for Windows也是如此。总体而言，SkyDrive对于那些一直梦想拥有真正的漫游配置文件的人来说是一笔巨大的资产。它不是完全完美的(例如，你不能同步桌面应用程序)，但对于希望在多个Windows设备上获得更一致体验的Windows用户来说，这是向前迈出的一大步。

同步设置

“SkyDrive”设置使您可以很好地控制SkyDrive在您的系统中扮演的角色，例如它与您的个人资料使用的每个系统同步了多少设置和文件。

“同步设置”允许在Metro和桌面环境中进行大范围同步-颜色、主题、布局、应用程序历史记录等。“同步设置”中有很多内容，如果你使用Microsoft帐户登录，那么你需要知道所有东西都做了什么。

一方面，同步是一件很棒的事情。另一方面，它在SkyDrive上存储了大量个人信息，这可能会产生意想不到的后果。例如，您可能在台式PC上将任务栏放在顶部，但这可能不适用于您的笔记本电脑。因此，您可能希望关闭任务栏同步。

此外，请密切注意“其他设置”，特别是“Web浏览器”和“密码”，因为它们存储潜在的敏感信息。

最后，“备份设置”允许您将设置备份到SkyDrive，并在需要刷新系统时将其恢复到PC。请注意，您实际上可以禁用同步设置，但仍可以备份您的设置。

控制面板的末尾？

那么，这些新奇的PC设置是否意味着控制面板的终结呢？可能吧。很难想象微软会继续在这些类型的桌面元素上投入时间和编码努力，除非它决定180并完全放弃开始屏幕。在这一点上，值得投入一些时间和精力，更好地掌握新的PC设置，因为这不太可能是我们最后一次看到它们。当然，如果您想个性化和优化您的开始屏幕体验，PC设置是实现这一目标的唯一途径。
