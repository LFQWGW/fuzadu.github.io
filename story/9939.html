Windows10的“可选功能”有哪些功能，以及如何打开或关闭这些功能

Windows 10自带多个“可选”功能，你可以通过“Windows功能”对话框打开或关闭这些功能。这些新功能中有很多是针对企业网络和服务器的，也有一些是对每个人都有用的。以下是对每个功能的用途以及如何打开或关闭它们的解释。

无论您是否启用了Windows 10功能，所有这些功能都会占用您的硬盘空间。但你不应该只启用每一个新功能-这可能会导致安全问题和系统性能降低。仅启用您需要并将实际使用的功能。

如何查看Windows的可选功能，以及如何打开和关闭它们

Windows10不提供从新的设置应用程序管理这些功能的方法。您必须使用控制面板中的旧Windows功能对话框来管理功能。

在此Windows功能对话框中，您可以启用Microsoft的Hyper-V虚拟化工具、Internet信息服务(IIS)Web服务器和其他服务器以及Windows的Linux子系统等功能。您还可以删除对某些默认功能的访问权限-例如，您可以禁用Internet Explorer以在Windows 10中隐藏旧版Web浏览器。此处提供的确切功能取决于您使用的Windows 10版本。

要启动控制面板，请右键单击开始按钮或按键盘上的Windows+X，然后从弹出的菜单中选择“控制面板”。

单击列表中的“程序”，然后在“程序和功能”下选择“打开或关闭Windows功能”。

您还可以使用单个命令快速启动此窗口。要执行此操作，请打开“开始”菜单，键入“optionalFeature”，然后按Enter键。您也可以按Windows键+R打开运行对话框，键入“optionalFeature”，然后按Enter键。

此时将显示可用Windows功能列表。如果某个功能旁边有复选标记，则表示该功能已启用。如果某个功能没有复选标记，则该功能处于禁用状态。

如果您在框中看到一个正方形，则该功能包含多个子功能，并且只有部分子功能处于启用状态。您可以展开该功能以查看其哪些子功能已启用，哪些未启用。

单击“确定”，Windows将应用于您所做的任何更改。根据您启用或禁用的功能，Windows可能需要重新启动计算机才能使更改生效。

您可以完全脱机执行此操作，无需任何Internet连接。这些功能存储在您的计算机上，在您启用它们时不会下载。

那么你应该打开还是关闭什么呢？我们整理了Windows 10专业版上提供的一些重要功能的列表，因为许多最有趣的功能-如Hyper-V虚拟化服务器-都需要Windows 10专业版。如果您使用的是Windows 10家庭版，您只会拥有其中的一部分功能。如果您使用的是Windows10企业版或教育版，您将拥有更多可用的功能。这些只是你可能会遇到的最常见的几个。

.NET Framework 3.5(包括.NET 2.0和3.0)：要运行为这些版本的.NET编写的应用程序，您需要安装此软件。如果应用程序需要，Windows将自动安装它们。 .NET Framework 4.6高级服务：如果需要，这些功能也会自动安装。它们只在运行需要它们的应用程序时才是必需的。 Active Directory轻型目录服务：它提供LDAP(轻型目录访问协议)服务器。它作为Windows服务运行，并为网络上的用户身份验证提供目录。它是完整Active Directory服务器的轻量级替代方案，仅在某些企业网络上有用。 嵌入式Shell启动器：如果要将Windows10的Explorer.exe Shell替换为自定义Shell，则需要此功能。微软的文档建议您在Kiosk模式下使用此功能来设置传统的Windows桌面应用程序。

Hyper-V：这是微软的虚拟化工具。它还包括底层平台和服务，以及用于创建、管理和使用虚拟机的图形化Hyper-V管理器工具。 IE 11：如果您不需要微软的传统网络浏览器，可以完全禁用对Internet Explorer的访问。 Internet信息服务：它提供Microsoft的IIS Web和FTP服务器以及管理这些服务器的工具。 Internet信息服务托管Web核心：这允许应用程序在其自己的进程中使用IIS托管Web服务器。只有在需要运行需要它的应用程序时才需要安装它。 隔离用户模式(Isolated User Mode)：Windows10中的一项新功能，允许应用程序在安全的隔离空间中运行(如果它们被编程这样做的话)。您只需要这个程序，您需要使用请求或请求它。以下是一段包含更多技术细节的视频。 遗留组件(DIrectPlay)：DirectPlay是DirectX的一部分，某些游戏将其用于联网和多人游戏。当您安装需要DIrectPlay的旧游戏时，Windows 10应该会自动安装它。 媒体功能(Windows Media Player)：如果您不需要，可以从此处禁用对Windows Media Player的访问。 Microsoft Message Queue(MSMO)Server：此旧服务通过对消息进行排队而不是立即发送来改善不可靠网络上的通信。*只有当您有专门要求和使用此功能的业务应用程序时，此服务才有用。 Microsoft Print to PDF：如果您愿意，可以从此处禁用Windows10附带的PDF打印机(但它非常有用，我们不知道您为什么要禁用它)。

多点连接器：它允许多点管理器和仪表板应用程序监视和管理您的计算机。它只在公司网络上有用，而且只有在这些网络使用这些管理工具的情况下才有用。 打印和文档服务：默认情况下启用Internet打印客户端和Windows传真和扫描功能。这些功能可以通过网络打印、传真和扫描。您还可以添加对LPD和LPR网络打印协议的支持，尽管这些协议较旧且不常见-只有当您必须连接到需要它们的网络打印机时才需要它们。此处的扫描管理功能用于管理和监控连接到网络的扫描仪。 RAS连接管理器管理工具包(CMAK)：此工具允许您为VPN创建自定义远程访问配置文件。除非您知道需要它来管理网络，否则您不需要它。 远程差分压缩API支持：这提供了比较同步文件的快速算法。与许多其他功能一样，只有当应用程序特别需要它时，它才有用。 RIP侦听器：此服务侦听路由器发送的路由信息协议公告。只有当您的路由器支持RIPv1协议时，它才有用。这在公司网络上可能有用，但在家里不会有用。 简单网络管理协议(SNMP)：这是用于管理路由器、交换机和其他网络设备的旧协议。只有在使用此旧协议的环境中工作时，它才有用。 简单的TCPIP服务(即ECHO、白天等)：这包括一些可选的网络服务。“ECHO”服务可能对某些企业网络上的网络故障排除有潜在的帮助，但在其他情况下，这些服务就没有用处了。 SMB 1.0/CIFS文件共享支持：这允许与旧版本的Windows(从Windows NT 4.0到Windows XP和Windows Server 2003 R2)共享文件和打印机。Linux和Mac操作系统也可能使用较旧的SMB协议进行文件和打印机共享。 Telnet客户端：它提供了一个telnet命令，允许您远程连接到运行telnet服务器的计算机和设备上的命令行界面。Telnet较旧且不安全。这些天您真的不应该通过网络使用telnet，但是当连接到一个古老的设备时，这可能会很有用。 TFTP客户端：它提供一个TFTP命令，允许您使用普通文件传输协议将文件传输到计算机和设备。TFTP也很旧而且不安全，所以你也不应该使用它。但是你可能不得不用一些古老的设备来使用它。 Windows Identity Foundation 3.5：较早的.NET应用程序可能仍需要此功能，但.NET4包含新的身份框架。只有当您需要运行需要它的旧版.NET应用程序时，才需要安装它。 windows PowerShell 2.0：PowerShell是比旧的命令提示符更高级的脚本和命令行环境。默认情况下，它是启用的，但如果您愿意，您可以禁用PowerShell。

Windows Process Activation Service：这与Internet信息服务Web服务器相关。只有在运行需要它的服务器应用程序时才需要它。

Windows Subsystem for Linux：在Windows 10周年纪念更新中，这项服务使您可以在Windows 10上使用Ubuntu Bash shell并运行Linux应用程序。 Windows TIFF iFilter：此功能使Windows索引服务能够分析.TIFF文件并执行光学字符识别(OCR)。它在默认情况下是禁用的，因为这是一个CPU密集型进程。但是，如果您使用大量TIFF文件-例如，如果您定期将纸质文档扫描为TIFF-这可能是一个有用的功能，使您可以更轻松地搜索这些扫描的文档。 工作文件夹客户端：此工具允许您将文件夹从公司网络同步到您的计算机。 XPS服务：这使您可以打印更多XPS文档。Microsoft用Windows Vista创建了这种文档格式，但它从未流行过，所以您最好还是打印成PDF格式。关闭此功能后，XPS打印机将从已安装的打印机列表中消失(不过，您也可以在Devices&Printers窗口中右键单击XPS打印机，然后选择“Remove Device”)。 XPS查看器：此应用程序允许您查看XPS文档。

大多数Windows用户将永远不需要访问此窗口并主动管理这些功能。Windows 10将在必要时自动安装程序所需的功能，不过对于某些功能，您可以很方便地知道在哪里可以打开或关闭它们。如果你曾经没有你认为应该有的功能，这是一个检查的好地方。
