屏幕截图教程：Microsoft Security Essentials 2.0 Beta

新的测试版Microsoft Security Essentials昨天面向有限数量的用户发布。在这里，我们将看看您对How-to Geek最喜欢的PC安全应用程序的新版本有什么期待。

我们已经批准了Microsoft Security Essentials。现在有一个新的测试版，我们今天来看看。它是通过Microsoft Connect发布的，先到先得，额满即止。它也不是在所有国家都可以使用，所以根据你所在的位置，你可能无法访问它。它有32位和64位两个版本。

安装和设置

您需要登录到您的Microsoft Connect帐户，如果您还没有帐户，请注册。然后您可以下载适合您操作系统的版本。

很明显，昨天从MS Connect下载文件时出现了问题，他们建议使用Microsoft文件传输管理器(链接如下)作为解决办法。

按照安装向导进行安装非常简单。如果您已安装Microsoft Security Essentials(MSE)1.0版，则可以升级。

决定是否加入客户体验改善计划…

它将为您提供启用Windows防火墙的服务。如果您使用另一个防火墙应用程序，或者只是不想打开它，您可以取消选中它。

现在，请等待升级进行。所需时间视情况而定。我们在Windows7Home Premium Core2 Duo系统上安装了32位版本，大约花了3分钟。

要完成升级，需要重新启动。

从重启返回后，您可以开始设置它。用户界面有一些新的外观，但使用它基本上与当前版本相同。

如果您没有安装当前版本的MSE，则不需要重新启动，它将提供获取最新定义更新并立即扫描您的系统。

64位版本

在我们的Windows7旗舰机上升级64位版本时，我们确实遇到了一些问题。我们最终用Revo Uninstaller Pro卸载了当前版本，当出现以下错误时，我们重新启动了机器，它安装成功，工作正常。

新特点

它仍然具有你已经习惯的功能，比如计划扫描和实时保护，并且还提供了新的功能。Windows7和Vista的一个有趣的新功能(XP或2003不提供)是您可以启用网络保护。这将检查您的网络流量并阻止任何可疑连接。对于那些拥有家庭或小型企业网络的人来说，这是一个非常好的功能。

此外，它还包括我们在安装过程中查看的防火墙集成。

它与Internet Explorer集成，可防范恶意脚本等基于Web的威胁。

它还包括新的更新保护引擎，可提供更好的检测、更轻松的清理和更好的性能。我们运行了一些虚假的测试病毒文件，检测结果与当前版本一样好，清理工作轻而易举。

在Windows Home Server上运行

如果您在家庭或小型办公室网络上运行Windows Home Server，您会喜欢MSE Beta在WHS版本1和Vail上工作的事实。只需将可执行文件放在共享文件夹中，然后将远程桌面放入您的服务器。或者，如果您正在使用WHS的高级管理控制台，您可以通过该控制台访问您的服务器桌面。然后按上面所示安装即可。

Windows Home Server版本1在Server 2003上运行，因此新的网络保护功能不可用。然而，它应该在WHS Vail测试版中可用，因为它构建在Server2008上。

即使在重新启动后，或者如果您没有登录，它也会在任务栏中运行，因此您不需要将其作为服务安装。
