如何访问Chromebook上的共享文件夹、网络打印机和VPN

将Chromebook连接到Windows网络，您可能会大吃一惊。您的Chromebook无法访问共享文件夹或网络打印机，无论它们是从Windows、Mac还是Linux系统共享的。

Chromebook可以连接到VPN、文件共享和打印机-但前提是这些资源必须以特定方式提供。如果网络资源配置正确，这应该很容易。

访问网络文件共享

相关：如何在网络上的Windows、Mac和Linux PC之间共享文件

Chromebook没有集成对浏览Windows文件共享(称为SMB或CIFS共享)或来自其他操作系统的NFS共享的支持。这看起来可能有点傻，因为MacOSX和Linux都可以访问Windows文件共享-即使是iPad和Android平板电脑也有可以访问标准共享文件夹的应用程序！开发者应该可以创建能够访问此类文件的Chrome应用程序，但这些应用程序尚未创建。

如果你有一个文件服务器，你想让它的文件在Chrome OS系统上可用，你必须让这些文件以Chrome网络浏览器能理解的方式可用。

任何允许你创建可通过标准HTTP或HTTPS访问的Web界面的服务器软件都可以在Chromebook上使用。例如，ownCloud服务器软件通过HTTP Web界面提供文件访问，因此您可以通过Chrome浏览器轻松访问。即使像Apache这样的标准HTTP服务器也可以在浏览器中列出目录的内容并使文件可供下载，尽管您不能上传文件。

Chromebook还集成了浏览FTP站点的支持，因此FTP服务器也可以让Chromebook访问您的文件。您可以使用FileZilla或其他FTP服务器程序设置FTP服务器。只需在Chrome的地址栏中输入FTP服务器的地址，以ftp：//开头即可。(请注意，Chrome OS只有集成的只读FTP支持，所以你必须找到第三方的“FTP web应用程序”或Use Chrome应用程序，比如售价2.99美元的sFTP客户端。)

确保以安全的方式设置任何服务器软件。只允许从您的本地网络而不是通过Internet访问这些服务器，以避免安全问题。

你也可以将你的文件转储到Google Drive中，然后就可以在Chromebook的Files应用程序中访问它们。如果你将它们存储在Dropbox、OneDrive或其他云存储服务中，你可以在该服务的网站上访问它们。如果您想要管理本地文件共享，这显然不是理想的做法。

添加网络打印机

相关：如何在网络上的Windows、Mac和Linux PC之间共享打印机

Chromebook也不支持标准的网络打印机。如果您有一台从Windows、Mac、Linux或Linux共享的打印机，Chrome OS将无法与其通信。你甚至不能使用直接连接到Chromebook的USB打印机，所以这并不令人惊讶。

Chrome OS打印解决方案是Google Cloud Print。许多新的Wi-Fi打印机支持Google Cloud Print以及苹果的AirPrint和其他打印协议。支持Google Cloud Print的打印机无需任何特殊配置即可工作-只需正常配置并使其可用于您的Google帐户即可。单击打印时目标下的更改按钮，然后选择打印机。

如果您的打印机较旧，不支持Google Cloud Print，则需要向Google Cloud Print注册该打印机。

为此，请在连接到打印机的Windows、Mac或Linux计算机上安装Chrome，并将Chrome：//设备插入地址栏。从此处将您的打印机连接到Google帐户。只要电脑开机并且Chrome正在运行，您就可以从Chromebook打印到打印机。

连接到VPN

谷歌的Chrome OS集成了对VPN的支持。请注意，Chrome OS目前只支持使用PSK的IPsec上的L2TP、基于证书的身份验证的IPsec上的L2TP，以及OpenVPN VPN。Chromebook可以连接到Cisco AnyConnect网络，但Cisco ASA设备必须设置为支持IPSec上的L2TP。

这听起来有点冗长，但是如果您知道VPN的详细信息，连接到VPN应该会很容易。

打开设置屏幕，单击Internet连接下的添加连接，然后单击添加专用网络。

在此处输入您的VPN详细信息。如果你不知道这些细节，你可以从你的雇主或其他提供你VPN的人那里得到。

如果您需要安装证书文件，则需要在另一个页面上执行此操作。运营VPN的组织将在您需要时提供这些文件。首先，将它们下载到您的Chromebook中。接下来，在您的地址栏中键入Chrome：//Settings/Certificate，然后按Enter键。您还可以在“设置”页中搜索“证书”来定位此屏幕。

使用“证书”选项卡上的“导入并绑定到设备”按钮导入任何.pfx或.p12文件(用户证书)，使用“授权”选项卡上的“导入”按钮导入任何.crt或.p7b(服务器证书颁发机构)文件。

添加VPN时，您可以从已安装的证书中进行选择。

开发人员模式选项

相关：如何使用Crouton在Chromebook上安装Ubuntu Linux

如果Chrome OS限制太多，也可以选择开发者模式。启用开发者模式，您就可以在Chromebook上安装桌面Linux系统。然后，您可以连接到其他类型的VPN，浏览Windows文件共享，并且无需Google Cloud Print即可打印。典型的Linux发行版支持的任何东西都应该是可能的。

当然，这是更多的工作。Chromebook之所以吸引人，是因为它们很简单，而且提供了一个几乎不需要配置的PC风格的桌面。进入开发人员模式，您必须使用终端命令才能正常工作，并使用更复杂的桌面界面。

开发者模式是为极客设计的。别误会我们的意思：这对极客来说很棒！但是，如果你只是一个典型的Chromebook用户，只想访问共享的网络文件夹，那么你可能应该远离开发者模式。

而且，如果你从学校或雇主那里得到了一台Chromebook，因为它安全且易于管理，如果你启用开发者模式并开始修补内部结构，他们可能不会感到兴奋。

如果您需要访问某个组织(特别是向您提供Chromebook的组织)提供的VPN、文件共享或网络打印机，您应该与该组织联系。要求他们配置服务器软件，以便可以从Chromebook访问重要的网络资源。
