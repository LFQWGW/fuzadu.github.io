不再收取升级费用：使用Google Docs或Office Web Apps代替Microsoft Office

微软最近推出了Office 2013和Office 365，这是一项订阅服务。Office365的月费为9.99美元，一年的费用为99美元，而Office2013的家庭和企业版价格为219.99美元，一次只能在一台PC上使用。

在打开钱包之前，请后退一步，问问自己是否真的需要Microsoft Office。如果您像大多数家庭用户、学生，甚至许多企业用户一样，并不真正需要Microsoft Office。有一些高品质的办公套房可以代替使用。

基于Web的办公套件是免费的

谷歌文档是完全免费的。它可以在你想要的任何设备上使用，甚至提供智能手机和平板电脑应用程序，这样你就可以在移动设备上快速查看你的文档。

微软还提供自己的免费Office Web Apps，与Google Docs竞争。微软的Office Web Apps也是完全免费的，根本不需要订阅费。您可以在浏览器中访问Microsoft的Office Web Apps，就像使用Google Docs一样。

面对微软每次发布新版本Office都要支付数百美元的前景，或者在未来10年花费1000美元订阅Office 365，你应该认真考虑这些免费办公套件中是否有一款可以满足你的需求。

您真的需要完整版Microsoft Office吗？

Microsoft Office的完整桌面版本包含大量功能，其中大部分功能是在Google Docs甚至微软的Office Web Apps中找不到的。如果您正在写一本书，并且需要使用Microsoft Word建立索引，使用Microsoft Access管理复杂的数据库，或者在Microsoft Excel中使用复杂的宏，这些解决方案是不会起作用的。

另一方面，如果您使用办公套件编写标准的Word文档，创建简单的演示文稿，或将电子表格与公式组合在一起，Google Docs或Office Web Apps将为您工作。

Google Docs可以将您的文档导出为Microsoft Office格式和PDF文件。Office Web Apps是由Microsoft创建的，应该与Microsoft Office格式具有出色的兼容性。您甚至可以将您的文档作为网页在线托管，以便于与他人共享。

基于Web的办公套件的优势

除了免费之外，基于Web的办公套件还可以在任何地方使用，并且包含强大的协作功能-这两种办公套件都允许人们通过互联网同时处理同一文档。无需安装桌面软件，因此您可以登录浏览器，开始在任何PC上工作，而无需安装本地办公套件。

无论您使用哪种笔记本电脑和台式机，您的文档都将始终保持同步-您甚至可以从智能手机或平板电脑访问它们。它们存储在网上，因此只有一份主副本，即使您的计算机硬盘死机，您也不会丢失它们。

这些办公套房缺乏功能，这甚至可能是一种优势。Google Docs和Office Web Apps提供了更简单的界面，更容易找到你使用的最重要的功能。

离线支持怎么样？

无论您的ISP网络连接出现问题，还是您在大西洋上空的飞机上，或者您正试图在地铁隧道中完成工作，离线访问您选择的办公套件都是至关重要的。

Google Docs确实提供离线支持，尽管它只适用于Google Chrome。如果您使用Google Chrome，您可以在Google Drive中启用脱机访问。您将能够脱机查看、编辑和创建文档、电子表格、演示文稿和绘图。您的更改将在您下次连接到Internet时同步。

微软的Office Web Apps它们不提供任何离线功能。要脱机编辑文档，您的桌面需要完整的Microsoft Office套件。

Google Docs与Office Web Apps之比较

Google Docs是基于网络的办公套件的先驱。它仍然提供Office Web Apps没有的功能，例如离线创建、编辑和查看文档的能力。它还会在您处理文档时自动保存文档，而Office Web Apps则要求您手动保存文档。如果您的连接出现问题或浏览器崩溃，您的文档在Google Docs中将是安全的，而在Word Web应用程序中可能无法访问。不过，Word Web应用确实有一个“文档恢复”功能，这可能会有所帮助。

微软的Office Web Apps随后紧随其后。如果您习惯了Microsoft Office最新版本中的功能区界面，则其界面会更加熟悉。

Office Web Apps的一个缺点是，它在界面中提供了在Word、Excel或PowerPoint的桌面版本中打开文档的按钮。这些按钮是一个无处不在的提醒，提醒你使用的不是微软首选的办公体验，而Google Docs是谷歌的主办公套件。Office Web Apps试图推动您购买完整的Microsoft Office。

Google Docs可以从Google Drive访问，而Office Web Apps则与微软的SkyDrive网站集成。我们最熟悉的是Google Docs，它在许多方面都是最成熟的选择，具有离线支持、自动保存和其他功能。Office Web Apps仍然缺少这些重要功能，但如果您喜欢Office功能区界面并希望与Microsoft Office文件格式的格式保持最佳兼容性，您可能会选择Office Web Apps。两个都可以试一试。

其他免费替代方案

显然，Google Docs和Office Web Apps并不是Microsoft Office的唯一替代品。有很多选择，但我们会提到两个很好的选择，如果你喜欢免费的本地办公套件，你可以下载到你的电脑上并在本地使用。

LibreOffice：LibreOffice是一个基于OpenOffice的正在积极开发的项目。这是一套功能强大的办公套房，功能齐全。它无法与微软Office的所有功能相匹敌，但它的功能比上面的网络应用程序更完整。然而，这也使得它比上面的Web应用程序更加复杂。 Abiword：Abiword是一款非常简单的文字处理程序，如果您只需编写基本文档，它就能很好地工作。它很小，下载速度很快，而且非常轻巧。

Microsoft Office是办公套件的黄金标准-我们知道这一点，不会争辩。Office是目前功能最强大、功能最丰富的办公套件。但大多数家庭用户和学生-甚至许多商业用户-并不需要黄金标准。他们需要做好基础工作的东西，Google Docs和Office Web Apps都符合条件。作为免费产品，它们对许多人来说是比全额付费微软Office更好的选择。
