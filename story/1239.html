Microsoft Security Essentials是免费的防病毒实用程序

Microsoft Security Essentials是一款免费实用程序，可提供实时保护，以识别和清除PC中的有害病毒和其他恶意软件。今天我们来看看它的表现和它必须提供的保护。

Microsoft Security Essentials(MSE)是免费的反恶意软件保护，取代了以前的“付费”实用程序One Care。它已经离开测试版，并于昨天向公众发布了1.0版。与我们之前向您展示的测试版一样，此版本提供实时保护，并在后台安静运行。它实际上运行起来很像测试版，用户界面也基本相同。

在安装之前，请确保计算机上没有运行其他防病毒或间谍软件应用程序。但是，您不需要关闭Windows Defender，因为MSE会为您处理它。

您必须验证您的Windows副本才能安装和使用Security Essentials。

用户界面基本、简单、易于使用。安装后，它会立即更新病毒定义。在那之后，它会在后台自动更新，所以一旦安装好，就是一个“设置好就忘了它”的过程。

数据库更新后，计算机状态变为绿色。顶部有4个选项卡，允许您控制不同的功能。

在“设置”下，您可以根据恶意软件的威胁级别选择应对其采取的操作。

默认情况下，实时保护处于启用状态，但如果您想要禁用它，则可以禁用它。实时保护对系统资源的占用很小，根本不会减慢其他进程的速度。

它会在通知区域显示一个绿色图标，让您知道您的系统受到保护。

手动扫描时会显示进度，并告诉您所需的时间。根据文件量和硬件性能的不同，系统之间的扫描时间会有所不同。

它的工作方式与大多数其他防病毒应用程序一样，允许您扫描单个文件。

如果检测到威胁，则会在通知区域中弹出红色注意屏幕，您可以根据建议的操作立即清除该威胁，或查找有关该威胁的更多详细信息。

如果您想要更多细节，它们实际上提供了大量有关威胁的信息。当在详细信息屏幕中时，您可以根据需要更改推荐操作，并从那里清除它。

在Out测试期间，我们试图下载受感染的文件，但MSE阻止了它，除非手动选择忽略它，否则不允许继续下载。这非常好，这样就不会有人忽略警告，在屏幕上单击，然后尝试下载并安装应用程序。

为了进行测试，我们在32位版本的Windows7家庭高级版上运行了这个程序，但它也可以在XP和Vista上运行。它对系统资源的占用很少，并且运行安静，不会不断弹出消息，也不会打扰您。不过，在重要的时候，它确实会弹出一条消息。如果发现威胁，它将弹出红色警报，让您知道恶意软件，并决定采取什么行动。这是很好的，有一个免费的反恶意软件实用程序，是易于使用和有效的。虽然微软并不总是提供最好的安全解决方案，但这一次他们似乎做对了。
