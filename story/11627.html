如何从iTunes电影和电视节目中删除DRM

您在iTunes上购买了电视节目或电影。你想在你的Android手机，Plex媒体服务器，或者基本上不是苹果生产的任何东西上观看。为什么不管用呢？

所有iTunes视频购买都被苹果的数字版权管理(DRM)计划Fairplay锁定。这无助于防止盗版，但肯定会让愿意为电影和电视节目付费的人的生活变得恼人。那么，一个想看东西的非海盗应该做什么呢？

如果你想在非苹果设备上观看你的视频，你需要去掉它的数字版权管理(DRM)功能。我们发现了两种方法可以做到这一点，这两种方法都在不损失视频质量的情况下剥离了DRM：

安魂曲是免费的，但它需要一个古老版本的iTunes才能工作。 Tuneskit不是免费的，但它的使用要容易得多，在撰写本文时，它可以与最新版本的iTunes一起使用。

我们将检查这两种方法，您可以决定哪一种更适合您。尽管Tuneskit也提供了Mac版本，但我们将使用Windows Server来学习这些教程。可悲的是，安魂曲在Mac系统身份保护上不容易工作，这使得降级iTunes几乎是不可能的。不过，我们还是推荐在虚拟机上运行Requiem，这样你就可以在Mac上通过在VirtualBox中安装Windows来使用它。

轻松删除DRM：*Tuneskit

到目前为止，转换视频最简单的方法是使用Tuneskit。它可以在Windows和MacOS上运行，会定期更新以与最新版本的iTunes配合使用，而且是无损的。(Tuneskit有一个演示，如果您想先测试一下，可以转换5分钟的视频。)

启动iTunes，并确保当前已下载并授权在您的计算机上播放您要转换的任何视频。

然后，启动Tuneskit。单击“添加文件”开始。

Tuneskit将扫描您的iTunes库；选择要从其中删除DRM的视频。

它们将被添加到您的列表中。点击“转换”，转换过程将开始。

转换可能需要一段时间，具体取决于您的处理器。

该过程完成后，单击“已转换”按钮可浏览已转换的介质。

您的原始iTunes文件保持不变，转换后的文件位于单独的文件夹中。

单击放大镜以在Windows资源管理器中查看文件。

为了确保该过程有效，请尝试在非iTunes视频播放器中播放视频。你应该会发现他们工作得很顺利。

祝贺你!。您已经从iTunes视频中删除了DRM，现在可以随心所欲地观看它。

有一件小事需要注意：Tuneskit的DRM剥离对于视频和5.1音频是无损的，这意味着你不会损失任何质量。但是，如果您的视频也带有立体声AAC音轨-或者只带有立体声AAC音轨-Tuneskit会根据我们的判断转换该音轨，这将导致音频质量的小幅度(对大多数人来说都不明显)损失。因此，如果您真的关心音频质量，请确保您使用的是5.1版杜比数字音轨，而不是2声道AAC音轨-或者使用以下立体声音轨的安魂曲方法。

自由而复杂的方式：安魂曲

Requiem是一个免费的、基于Java的应用程序，能够从iTunes视频中删除Fairplay DRM。问题是：它已经有一段时间没有维护了，只有在2012年发布的iTunes10.7上才能使用。

设置iTunes 10.7有两种方法：

你可以从电脑上完全卸载iTunes，然后安装旧版的iTunes 10.7。如果你花了很多时间来调整你的iTunes设置，并且不想丢失任何东西，那么不推荐使用这种方法。如果你是iPhone或iPad的用户，这也不是一个好主意，因为旧版本的iTunes不能与新版本的iOS同步。 你可以在通常不使用iTunes的电脑上安装iTunes10.7，也可以设置一个专门用于从视频中剥离DRM的虚拟机。从干净开始意味着你需要重新下载并授权任何你想要转换的视频，但它可以防止你破坏正在运行的iTunes安装。

为简单起见，我们建议您在不使用iTunes的机器上安装iTunes10.7，无论是虚拟的还是其他的。只是比能出差错的要少得多。如果这就是您要做的，请随意跳过下面的第一步。

第一步：完全卸载更新版本的iTunes(如有必要)

如果您计划降级当前安装的iTunes，我们还有一些工作要做。在Windows上，进入控制面板>程序和功能，然后卸载“Apple Inc.”生产的所有产品，包括iTunes、Bonjour和Apple Software Update。请确保电脑上没有任何与Apple相关的东西。

在移除所有东西后，旧的iTunes安装可能无法正常工作。我们在测试中没有这个问题，但您的里程数可能会有所不同。如果你有问题，像Revo Uninstaller这样的软件可能会很有帮助。

第二步：安装iTunes 10.7并确保您的视频经过授权

转到iTunes 10.7下载页面。下载运行可执行文件以安装iTunes。请确保禁用自动更新。

然后，随着安装幻灯片中宣传昨天的iPhone和iPad，沉浸在对最近过去的怀旧中。

当安装完成后，你会注意到iTunes在五年内发生了多少变化。你可以去商店下载所有你想要转换的视频。(你不能从现有系统传输这些视频--你需要在iTunes10.7上重新下载视频才能正常工作。)

下载完成后，请确保您的视频确实在iTunes中播放。您可能需要授权您的计算机，在某些情况下还需要重新下载整个视频。

第三步：安装Java Runtime Environment(如有必要)

Requiem需要Java Runtime环境才能运行，因此请转到Java下载页面并下载JRE安装程序。

运行可执行文件以安装Java。

现在已经安装了Java，我们终于可以运行安魂曲了。

第四步：运行安魂曲

假设您的所有下载都已完成，请关闭iTunes。如果您尚未下载安魂曲，请下载。(如果您被重定向到“反水蛭”页面，请尝试右键单击下载链接并将其粘贴到地址栏中。)。该程序以ZIP文件的形式提供，并且是可移植的，因此请打开它并将可执行文件解压缩到您喜欢的任何位置。

快速注意：如果你想保留字幕和其他元数据，你需要在启动安魂曲之前运行mkvtoolnix和CCExtractor。

Run Requiem将扫描您的iTunes目录中是否有任何受保护的文件，然后删除保护。

如果您收到任何错误，请确保您已授权iTunes并且视频可以播放。

该过程完成后，安魂曲将删除您的视频的受保护版本，并将其替换为完全不受保护的版本。

转到Windows Explorer…中的媒体文件夹

…。用iTunes以外的其他东西打开视频，以确保视频正常工作。

祝贺你!。你已经从你的视频中删除了DRM，现在可以用任何你喜欢的媒体播放器播放它。安魂曲的视频和音频转换是完全无损的，所以质量不会有任何损失-无论你使用的是哪一首音轨。

如果你不知道，Tuneskit是目前为止更简单的方法，特别是当你有很多文件要剥离的时候。使用免费方法可能会有太多令人沮丧的事情出错。但是，如果你真的反对花钱，安魂曲可以在一个夹克…工作。只要你愿意忍受这些麻烦。
