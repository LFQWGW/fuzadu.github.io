如何用一个“我的世界”账户玩多人局域网游戏

所以你想和你的家人一起玩“我的世界”，但是你只有一个账号。你将不能在线玩，但只要对配置文件做一些调整，你们应该都可以在家里一起通过网络玩游戏，而不需要额外的账户。

我为什么要这么做呢？

为孩子购买“我的世界”是许多父母感到困惑的一个问题：每个孩子都需要单独的“我的世界”账户吗？答案完全取决于你想让你的孩子用“我的世界”做什么，以及他们的目标是什么。

相关：探索“我的世界”多人服务器

如果你想让你的孩子能够在线玩，这样他们就可以访问不同的“我的世界”社区和服务器，他们想要同时在线玩，那么他们每个人都需要一个单独的优质“我的世界”账户(目前零售价为27美元)。“我的世界”服务器对每个登录进行身份验证，每个用户都需要具有唯一和有效的“我的世界”ID。

然而，如果您的目标是让您所有的孩子(或朋友)在您家的局域网(LAN)上一起玩耍，您不需要多个付费保费帐户来做到这一点。只要有一个拥有高级帐户的用户，您就可以有效地“克隆”该用户，并调整二级用户的配置文件，以允许更多玩家加入本地游戏。

这一调整不允许所有人在线玩，也不会让其他用户合法访问“我的世界”认证或皮肤服务器。这不是破解或盗版利用。然而，它确实有一个缺点：当其他玩家查看时，每个玩家都会有相同的默认“Steve”皮肤。但对于一个家庭来说，这是一种很好的方式，可以廉价地允许兄弟姐妹或他们的朋友快速组织一场“我的世界”局域网派对，而不会在高级许可证上损失数百美元。

话虽如此，如果你确实发现你的家人正在认真使用“我的世界”，并且你为年幼的孩子制作的“克隆”客户端很受欢迎，我们会鼓励你购买一个完整的账户。你的孩子不仅有能力在成千上万台令人敬畏的“我的世界”服务器上玩，并为他们的玩家角色定制皮肤，你还将支持游戏的开发。例如，尽管我的家族99%的《我的世界》游戏是在我们的局域网上内部完成的，但我的家族中的每个人都有自己的账户。

对于准备深入研究的读者，让我们来看看如何在LAN上轻松地运行多个客户端。对于可能已经感到有点不知所措的“我的世界”的新玩家或家长，让我们推荐查看“我的世界”的家长指南，以获得对这款游戏的很好的介绍和它的全部内容，如果需要更深入的了解，请参阅涵盖初学者和高级“我的世界”游戏的由多部分组成的极客学校系列。

您需要什么？

如果你正在阅读这本指南，你可能手头上有你需要的一切。但是，在我们进入操作步骤阶段之前，让我们花一点时间来清楚地概述一下需要什么。

首先，你至少需要一个优质的“我的世界”账户。你将需要在每台你打算用来玩“我的世界”的电脑上至少登录这个高级帐户一次，这样高级帐户才能下载必要的资产。

其次，每个额外的玩家需要一台计算机。这台机器上的“我的世界”播放器配置文件将被半永久性地更改，以允许您使用不冲突的用户名在本地网络上玩游戏。(请注意，您的世界保存或其他游戏数据都不会被删除或面临删除的风险，但如果您想要再次使用常规帐户登录，则需要反向执行此过程。)

最后，如果您想要对辅助玩家的皮肤进行本地更改(这将允许他们看到自己独特的皮肤，但是，因为使用了Mincraft皮肤身份验证，所以不会影响其他人对他们的看法)，您需要创建一个简单的资源包。最后一步是完全可选的，除非你的玩家真的想要一个定制的皮肤(同样，只有他们才能看到)，否则你可以跳过它。在本教程的最后一节中，我们将向您介绍此过程。

如何配置辅助客户端

您需要进行的所有配置更改都将在辅助计算机上完成。在任何情况下，您都不需要对主要的“我的世界”计算机(原始帐户持有人使用的计算机)进行任何更改，因此，在本教程的其余部分中，请继续在您的辅助计算机前坐下。

在我们开始进行配置更改之前，让我们先向您展示如果您尝试在没有进行必要的配置更改的情况下登录会发生什么情况。如果次要玩家登录到主要玩家的Open LAN游戏(即使用主要玩家的帐户)，他们将看到此错误消息：

“我的世界”本质上是说“等等。你不可能是约翰。约翰已经存在了！“。事情就这样结束了。尽管本地局域网游戏不像官方(和第三方服务器)那样完全通过“我的世界”服务器进行身份验证，但是本地游戏仍然尊重这样一个事实，即同一个游戏中不应该有两个完全相同的玩家。如果它真的允许两个相同的玩家加入游戏，毕竟结果将是灾难性的，因为重要的东西，如角色清单和Ender Chest清单都与世界保存文件中玩家的用户名相关联。

为了绕过名字检查，并避免两个同名玩家带来的错误，我们需要-你猜对了-给第二个玩家一个新的名字。为此，我们需要对其中一个“我的世界”配置文件进行简单编辑。

访问“我的世界”配置文件的最简单方法(不需要深入到配置文件夹的内部)就是使用“我的世界”启动器中的便捷快捷方式跳转到那里。

在我们继续之前，您需要启动“我的世界”启动器至少一次，并使用您的主“我的世界”帐户登录，以便如前所述下载必要的资源并准备好玩辅助机器。这个过程很简单，只需登录并单击一次“播放”按钮，就像您要玩普通的“我的世界”游戏一样。

一旦你完成了获取资产的预演，然后再次启动“我的世界”发射器，如上所述。首先，记下右下角的“Welcome，[username]”条目。在这一点上，名称应该是您的优质我的世界帐户的名称。如果你的“我的世界”的用户名是SuperAwesomeMinecraftGuy，它应该说“欢迎，SuperAwesomeMinecraftGuy”。

确认用户名后，单击左下角的“编辑个人资料”按钮。

在配置文件编辑器屏幕中，选择“打开游戏目录”以跳转到我们需要编辑的文件位置。

在游戏目录中，您将看到一个名为“Launcher_profiles.json”的文件，如上面突出显示的那样。在记事本或记事本++等纯文本编辑器中打开文件。

在.json文件中，您将看到如下所示的条目：

{        "displayName": "John",

John，或者“displayName”旁边的任何名字都是“我的世界”官方账号的用户名。将名称编辑为您想要的任何用户名，并保留引号。

{        "displayName": "Angela",

在我们的例子中，我们将“John”更改为“Angela”，这样John和Angela就可以在LAN上一起玩了。正常情况下，如果您登录到远程的“我的世界”服务器，更改显示名称会产生错误，但是，因为本地网络游戏不会根据官方的“我的世界”服务器验证用户名，所以我们可以在这里输入任何我们想要的用户名。

保存文档，关闭配置文件编辑器窗口，然后重新启动“我的世界”启动器以使更改生效。

再次仔细检查右下角。高级“我的世界”账户的用户名现在应该被您编辑过的用户名所取代(在我们的例子中，用户名应该是“Angela”，而且它确实是这样做的)。

要进行测试，请在主计算机上启动一款“我的世界”游戏，加载地图，然后打开地图以供LAN Play使用。反过来，让次要玩家加入现在开放的局域网游戏。

您应该看到，或多或少，与我们在上面的屏幕截图中看到的完全一样：使用新用户名和默认Steve皮肤的次要玩家。你们现在可以自由地在一起玩了！

请记住，如上所述，所有玩家数据都链接到游戏中的用户名。如果“Angela”想要更改她的用户名，她应该首先将她所有的角色清单和Ender箱子里的东西都倾倒在一个安全位置的常规箱子里。

要逆转我们刚刚经历的过程，只需导航回.json文件并将displayName变量更改回原来的状态(高级帐户持有者的用户名)。

如何更改本地外观

正如我们在教程前面提到的，有些玩家可能希望执行一个非常不必要但很有趣的步骤：为辅助玩家添加自定义皮肤。

这一点有一个很大的警告：因为显示的皮肤是由“我的世界”内容服务器管理的，所以任何未经验证的播放器都将始终显示为其他玩家的默认皮肤。这意味着如果我们用这个小把戏把Angela的皮肤换成另一种皮肤，唯一能看到皮肤变化的人就是Angela。

相关：如何使用资源包重塑您的“我的世界”

尽管如此，如果第二玩家真的想要一个自定义的皮肤来截屏或只是为了好玩，给他们这个皮肤是微不足道的。

我们的小头像皮肤小把戏的关键是卑微的“我的世界”资源包。简而言之，资源包允许玩家将游戏中几乎每个对象的纹理或图形与其他纹理互换。虽然这通常是为了改善(或改变)您周围世界的外观，但在我们的例子中，我们可以利用它来替换玩家皮肤。

如果你想了解更多关于资源包的一般知识，从它们的工作原理到哪里可以为你的游戏找到有趣的资源包，一定要查看我们的指南“如何用资源包重塑你的我的世界”来深入了解它们。出于本教程的目的，我们将给您一个速成课程，教您如何制作一个非常简单的资源包来为您的辅助播放器添加新的皮肤。

创建资源包

首先，确保您位于刚刚更改名称的同一台计算机上。其次，使用相同的技巧进入我们在上一节中使用的游戏目录(Launcher->Edit profile Button->Game Dir)来访问游戏目录。在游戏目录中，查找/resource cepacks/文件夹。

在资源包文件夹中，创建一个新文件夹。给它起个合理的名字，比如“单人换肤”或“安吉拉换肤”，这样你以后就能很容易地认出它了(在游戏中也是如此)。打开文件夹并创建新的文本文档。在文本文档内粘贴以下文本：

{ "pack": { "pack_format": 1, "description": "1.8 How-To Geek Skin Change Pack" } }

将文本文档另存为“pack.mcmeta”(确保将文件扩展名从.txt切换为.mcmeta，而不是将其另存为“pack.mcmeta.txt”)。接下来，您需要创建一系列嵌套的文件夹，以模拟My Wift中的实际资源文件夹(因为资源包就是这样工作的)。您需要创建一个“Assets”文件夹，在其中包含一个“Minworld”文件夹，而在该文件夹中又包含一个“Textures”文件夹，其中包含一个“Entity”文件夹，如下所示：

\assets\minecraft\textures\entity

最后，您需要将您想要使用的任何皮肤的.png文件放在该文件夹中，并将其重命名为“steve.png”。在我们的例子中，我们从Minecraftskins.com抓取了这张星球大战沙警的皮肤，将其粘贴到文件夹中，并将其重命名。

加载资源包

接下来，我们需要加载有问题的“我的世界”的副本并应用资源包。按Esc键调出游戏中的菜单，选择Options->Resource Pack，然后从可用的资源包中选择您刚刚创建的资源包。

在上面的屏幕截图中，您可以看到我们专门为本教程创建的“HTG皮肤”包。单击资源包图标(它将变成播放图标)，然后再次单击它以将其转移到“Selected Resource Pack”列中。然后点击“完成”按钮。

资源包中的steve.png文件将替换默认的Steve皮肤，如上所述，您将穿上新的皮肤！同样，我们想要强调的是，只有应用资源包的玩家才能看到更改，但在辅助机器上自定义玩家的外观仍然是一种有趣的方式。

仅此而已：只需一个简单的配置文件调整和一个可选的资源包，您现在就可以在本地网络上使用多台计算机玩“我的世界”了。正如我们在引言中强调的那样，这不是破解在线游戏的一种方式，它确实有其局限性。这个诀窍适用于让弟弟妹妹玩游戏或临时的局域网派对。但是，如果在你的预算中为你家里的每个全职球员购买一份拷贝，我们会鼓励你这样做。
