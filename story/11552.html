如何摆脱AVG的通知和捆绑软件

AVG Antivirus包括浏览器扩展、通知和其他肯定会让您分心的功能。如果你只想要一个杀毒软件在后台默默地保护你的系统，这里有一些如何瘦身的方法。

相关：Windows 10最好的杀毒软件是什么？(Windows Defender是否足够好？)

这个过程是用AVG上的免费版本执行的，名为AVG AntiVirus Free。要了解有关我们推荐的防病毒程序的更多信息，请查看本指南。

你不能禁用的东西

在我们开始之前，请注意AVG的免费版本有一些你无法禁用的内置广告。当您打开AVG应用程序本身时，您将看到安装“PC TuneUp”之类的软件的提示。值得庆幸的是，这些广告仅限于AVG界面本身，所以您只能在打开AVG的控制面板时才能看到它们。

去掉AVG的浏览器扩展

AVG试图安装具有大量权限的“AVG Web TuneUp”浏览器扩展。现代版本的Chrome和Firefox不允许应用程序在未经您许可的情况下安装扩展，但无论如何您可能已经同意安装扩展。

相关：不要使用杀毒软件的浏览器扩展：它们实际上会降低您的安全性

我们建议您不要使用杀毒软件的浏览器扩展，因为它们实际上会降低您上网的安全性。如果您确实安装了浏览器扩展，现在就应该将其卸载。

在Chrome中，单击菜单并选择更多工具>扩展。点击AVG Web Tuneup Up旁边的垃圾桶图标将其删除。

在Firefox中，单击菜单按钮并选择“Add-ons”。点击Extensions选项卡上AVG Web TuneUp右侧的“Remove”按钮。

在Internet Explorer中，单击Internet Explorer工具栏上的齿轮图标，然后选择“管理加载项”。单击“工具栏和扩展”列表中的“AVG Web Tuneup”项，然后单击“Disable”(禁用)。

您也可以选择完全卸载AVG Web TuneUp组件。这个AVG功能需要您启用浏览器扩展，所以如果您卸载这些扩展是没有用的。

要卸载AVG Web TuneUp，请转到控制面板>卸载程序。卸载“AVG Web TuneUp”应用程序。

除了这个组件及其相关的浏览器扩展，AVG没有捆绑任何额外的软件。

禁用AVG的通知

AVG的其余选项位于AVG界面中。要找到它，请左键单击通知区域中的AVG图标，或右键单击它并单击“打开AVG”。AVG图标可能隐藏在系统托盘图标左侧的向上箭头后面。

单击AVG Zen窗口中的“Antivirus”图标以访问您的AVG AntiVirus设置。

单击AVG Antivirus窗口右上角的“Menu”(菜单)按钮，然后选择“Settings”(设置)。

AVG会定期显示弹出通知，包括在后台自动更新其病毒定义时。

如果您只想禁用所有这些通知和弹出窗口，请激活常规窗格中的“静默模式”复选框，然后单击“确定”。

要自定义显示哪些通知，请禁用静默模式复选框并展开“弹出窗口”部分。您可以为信息弹出窗口、更新弹出窗口、警告弹出窗口和警报弹出窗口设置不同的持续时间。例如，您可能希望通过将信息和更新弹出窗口设置为“0”来禁用信息和更新弹出窗口，但保持启用警告和警报弹出窗口。

如果AVG在尝试更新其病毒定义或程序软件时遇到错误，通常会向您发出警告。

如果您不想看到这些通知，可以禁用它们。单击设置窗口左侧的“更新”选项卡，并取消选中“详细信息”下的“发生错误时显示通知框”。

AVG仍将保护您的计算机免受病毒和恶意软件的攻击，但您不必忍受浏览器通知和不必要的通知。
