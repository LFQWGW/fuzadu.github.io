10种快速加速运行速度慢的Mac的方法

Mac电脑的运行速度也可能很慢。如果您的Mac运行速度似乎比应有的速度慢，这里提供的各种提示应该可以帮助您识别和解决问题。与Windows PC一样，Mac运行速度慢的原因有很多。

远离那些承诺“优化”你的Mac并让它运行得更快的程序。一些“清理”程序-比如Mac版的CCleaner-可以删除临时文件并释放空间，但它们不会让你的Mac运行得更快。

查找资源匮乏的进程

相关：当Mac上的应用程序没有响应时如何强制退出

使用活动监视器-有点像Windows上的任务管理器-来查看您正在运行的进程，并发现它们正在吞噬过多的资源。要启动它，请按Command+Space组合键打开Spotlight for Search，键入Activity Monitor，然后按Enter键。

单击“%CPU”标题可按CPU使用情况排序，并查看使用最多CPU的所有正在运行的应用程序和进程。在某些情况下，您可能想要终止使用99%CPU的单个失控应用程序。要强制退出进程，请通过单击将其选中，然后单击工具栏上的X按钮。请确保您没有退出正在做一些重要事情的过程。您始终可以先尝试以正常方式关闭资源耗用的应用程序。

如果这不起作用，请单击“查看”菜单，然后选择“所有进程”以查看您的Mac上运行的所有进程。你也可以点击内存部分-使用大量内存的进程可能会导致Mac速度变慢。也可以试试“磁盘”部分--大量使用磁盘的进程也可能会导致您的Mac速度变慢。

关闭应用程序

MacOSX喜欢让应用程序在被告席上运行。即使单击应用程序窗口上的红色“X”按钮也不会将其关闭-它将保持在后台运行。在大多数情况下，这不是一个大问题。但是，如果您的Mac运行速度很慢，您可以关闭其中一些应用程序。

查找码头上标有点的应用程序，右键单击或按住Ctrl键单击它们的图标，然后选择“退出”。

删除启动程序

相关：Mac OS X：更改登录时自动启动的应用程序

如果您的Mac在您登录后速度较慢，则可能是它的启动程序太多。

要管理启动程序，请单击Apple菜单图标并选择“系统首选项”，打开“系统首选项”窗口。单击“用户和组”图标，选择您当前的用户帐户，然后单击“登录项目”。取消选中登录时不希望启动的任何应用程序。

如果你曾经想让一个程序在你登录Mac时自动启动，可以将它拖放到这个列表中，或者点击列表底部的“+”按钮并添加。

清理发射剂

CleanMyMac3的隐藏功能之一是它可以清理启动代理，这些启动代理是在后台秘密运行并在你不知道的情况下自动启动其他软件的微小助手应用程序。如果您经常运行大量您不记得开始的程序，那么您可能会遇到启动代理问题。

只需下载并运行CleanMyMac3，然后进入扩展->启动代理来彻底清理垃圾。

降低透明度和动画

相关：如何在MacOS中禁用窗口透明度

透明度和动画会给老款Mac电脑的图形硬件带来负担。减少它们可以帮助加快速度-这是值得一试的。

要执行此操作，请打开系统首选项窗口。单击“辅助功能”图标，然后选中“降低透明度”选项以降低透明度。在OS X Yosemite上，此选项可以显著提高一些较旧的Mac电脑的速度。

您可能还想单击“Dock”首选项图标，然后选择“缩放效果”而不是“精灵效果”，这样可以稍微加快窗口最小化动画的速度。

减轻您的Web浏览器的亮度

您的Web浏览器可能就是导致您问题的应用程序。通常的小贴士也适用于Mac--特别是考虑到Google Chrome在MacOSX上的性能特别差。

尽量减少使用的浏览器扩展的数量，一次打开的选项卡更少，以节省内存和CPU资源。

你可能还想试试Mac OS X附带的Safari浏览器，它似乎确实比Chrome性能更好-特别是在电池供电方面。例如，如果你可以不使用Safari，并且不依赖Chrome的某个功能或扩展，那么你可能想认真试一试。

禁用FileVault磁盘加密

相关：如何加密Mac的系统驱动器、可移动设备和单个文件

默认情况下，在Mac OS X Yosemite上启用FileVault磁盘加密。这有助于在Mac文件被盗时保护其安全，防止未经授权访问这些文件。它还可以防止他人在未经您许可的情况下更改您Mac上的密码和登录。

在某些Mac上，这可能会导致Mac启动或登录非常慢。如果是这种情况，您可以进入系统首选项窗口，单击“安全与隐私”图标，单击“FileVault”标题，然后关闭FileVault磁盘加密。

我们建议您保持启用FileVault，除非您的Mac需要很长时间才能启动或登录。

提高查找器的速度

当您打开Finder窗口查看文件时，默认情况下它会打开一个“All My Files”(所有我的文件)视图。如果您的Mac上有非常多的文件，则此视图的加载速度可能会很慢，从而在每次打开新的Finder窗口时都会降低Finder的速度。

您可以通过单击“Finder”菜单并在Finder中选择“Preferences”来防止这种情况发生。在“New Finder Windows Show”下选择您的首选文件夹-例如，您可以让所有Finder窗口自动打开到Downloads文件夹。Finder将不再加载“所有我的文件”视图。

释放磁盘空间

相关：释放Mac硬盘空间的10种方法

与在任何计算机上一样，如果磁盘空间非常小，释放磁盘空间也可以加快速度。要进行检查，请单击Apple菜单，选择“关于这台Mac”，然后查看“存储”标题下的内容。

如果没有足够的空闲空间，你会想要在Mac的内部存储器上释放空间。

重置您的SMC以修复各种问题

相关：如何修复Mac OS X上的“Wi-Fi：No Hardware Installed”(Wi-Fi：未安装硬件)错误

这是一个有用的提示，可以解决Mac上的各种系统问题，尽管许多人-特别是有经验的Windows用户-不会想要尝试这一点。重置系统管理控制器(或SMC)可以解决从性能缓慢到启动问题和Wi-Fi硬件问题的所有问题。这实际上不会擦除任何数据-这就像是对Mac执行较低级别的重启。

按照本文中的说明重置Mac的SMC。

重新安装Mac OS X

相关：如何擦除Mac并从头重新安装MacOS

当所有其他方法都失败时，请尝试重新安装您的操作系统。这对每一台设备都是一个很好的提示-如果你在iPhone或iPad上有问题，这甚至是一个好主意。

如果您的Mac带有机械硬盘，您可能希望尝试将其升级到固态硬盘。这永远是提高任何计算机速度的必经之路。
