Windows 10中的OneNote入门指南

微软已经修改了许多内部应用，以符合Windows10的设计美学和增强的功能，我们在新的OneNote中获得的也没有什么不同。

如果你已经是Evernote桌面应用程序等竞争产品的忠实客户，微软的快速笔记解决方案提供的许多功能应该已经让你感到熟悉了。然而，原始OneNote的长期用户可能仍然需要一些导航新设计的提示，这就是我们来此提供帮助的原因。

值得一提的是，OneNote实际上有两个版本：一个是标准桌面版，是Office的一部分；另一个是新的通用应用，是Windows10的一部分。我们说的是后者。

创建和编辑备注

OneNote是随Windows 10(包括家庭版、专业版和企业版)的库存版本一起默认安装的，只要您更新到最新版本的Microsoft操作系统，OneNote应用图标就会自动出现在[开始]菜单中。要启动OneNote，只需向下滚动菜单，然后单击此处突出显示的“OneNote”磁贴：

如果你是第一次使用OneNote(或任何其他Office 365产品)，你会被提示进入“只读”模式，直到你签约免费试用，或者冒险购买每月6.99美元的订阅，每30天支付6.99美元(“Home”版本9.99美元，允许你在同一个账户上同时存储最多4个用户)。

但是，如果您使用Microsoft帐户而不是本地帐户登录，则只需单击此选项，您就会发现OneNote for Windows 10功能齐全，无需注册任何额外订阅。因此，只需点击“开始使用OneNote”按钮，忽略其中的“只读”部分即可。

验证帐户后，您将进入OneNote，并准备开始创建您自己的笔记。

记事本

进入后，您将在左侧菜单中看到您的第一个“笔记本”，它将根据您用来启动程序的帐户自动命名。

每本笔记本可以包含无限数量的“章节”，这些章节本身可以容纳“页面”，正如你所看到的，这使得每本笔记本作为一个整体的深度几乎是无限的。

要创建新便笺，请从单击“最近的便笺”旁边的加号开始，或者只需单击屏幕中央，您的第一个分区便会自动出现。

管理您的页面

要开始管理您的页面，最好将每个页面都看作是它自己的白板，其中包含许多可用的添加内容和额外功能，可以根据您的特定需求进行修改。正如我们上面提到的，您的分区的总和是组成一个笔记本的部分，而在每个分区内，您可以创建多个页面。

不仅如此，对于每个页面列表，您还可以将其他页面指定为“子页面”，它会将较低级别的页面附加到另一个页面，作为同一部分的一部分。

例如，您可以在这里看到我们在一个部分工作。然后，每个部分都可以被分成几个页面，在本教程中，这些页面相应地被标记为“待办事项列表”、“工作”、“杂货”和“项目委员会”。

要将新页面添加到父部分，我们单击了左栏中的加号，加号表示“+Pages”。然后，每个页面都充当自己的单独可用空间，您或其他有权访问笔记本的用户可以在其中添加或编辑任意数量的内容。

虽然从表面上看，这可能听起来有点不堪重负，但一旦你开始钻研OneNote，所有这些都开始无缝地流动在一起，并形成了一款广泛的笔记应用程序，当涉及到将尽可能多的功能打包到一个程序中时，它会不遗余力地做到这一点。

编辑 / 选编 / 评注 / 剪辑

在设置好页面并准备开始创建自己的笔记后，您会注意到OneNote的工作方式有点像“剪贴簿”，文本、图像甚至绘图都可以拼接在同一个笔记中，在一个地方创建包含各种想法和集思广益的概念的软木板。如果您想在文本上叠加图像，OneNote不会阻止您。需要在某人的购物单上涂鸦一些额外的东西吗？OneNote兼容平板电脑和手写笔，如果您觉得特别有艺术性，甚至可以使用鼠标。

要编辑任何页面，您需要从排列在每个笔记本顶部的四个选项卡中的一个选项卡中进行。

产地 / 家乡 / 避难所

首先是家庭酒吧。在这里，您可以使用标准编辑工具，如创建项目符号列表、更改字体样式(粗体、斜体等)或创建新的缩进。

您也可以在这里看到“标签”菜单，单击此处突出显示的箭头即可找到该菜单：

您可以将标签应用到OneNote中的任何类型的内容(文字、图像、绘图等)，在我们的示例中，您可以看到我们已经用“问题”、“联系”和“地址”贴纸标记了虚拟文本部分，以获得更好的视觉组织提示。

这使得任何分享笔记的人都可以轻而易举地知道他们正在查看的内容属于哪一类，甚至不需要事先阅读。

插入 / 嵌入 / 塞入 / <剖>附着于某部位

如果您想要插入任何类型的外部媒体，无论是图片、新的电子表格表格，甚至是整个文件，“插入”选项卡都可以完成。

插入图片后，您可以选择将其从便笺中剪切、复制、粘贴、删除或使用下面突出显示的菜单修改上传的文件。

添加的任何文本都将被限制在一个框中，该框将根据其中的内容量进行挤压或拉伸。不过，如果您想自己组织，您也可以使用每个独立条目右上角的滑块手动更改大小。

绘制 / 拉 / 拔出 / 汲取

当然，如果没有随心所欲地随意涂鸦的选项，笔记应用程序会是什么样子呢？

绘图选项卡包含一些基本的工具，您可以使用它们来放置一些涂鸦，这些涂鸦可以覆盖在图片上，将人们从一个段落指向下一个段落，或者只是给同一项目的合作者留下快速消息。

绘图选项卡还包含用于高亮显示文本的选项，以防您需要从人群中脱颖而出的条目的特定部分。

视力 / 视线 / 看法 / 风景

最后，我们来看看查看选项卡，它恰好也是可用选项中最轻的。

在这里，你可以做三件事中的一件：改变缩放(放大或缩小)，改变放大程度，或者在你的项目中添加规格线，真正给它带来“大学笔记本”的感觉。简单、干净和有效，您在这里得到的正是您需要的，而不是介于两者之间的任何其他东西。

设置

要进入设置菜单，首先需要单击OneNote窗口左上角的菜单按钮。

从这里开始，“设置”图标将出现在左下角。

单击此按钮，您将进入以下菜单，该菜单将从OneNote主窗口的右侧弹出。

选项

在选项选项卡中，您可以找到更改所有Notes同步到哪个OneDrive帐户的设置。这一点很重要，因为这封电子邮件是您每次进行任何更改时所有笔记本和页面都会自动同步到云的地方。

请确保严格控制此帐户上的所有内容，并将您的隐私配置设置为最大默认值，否则您的笔记将公开提供给从另一台计算机使用该云存储的任何人。

也就是说，也可以选择关闭自动同步过程，在这种情况下，你所做的任何保存都将绑定到本地计算机，并且只有在每次手动保存时才会上传到云中。

最后一个选项和同步是切换到“同步所有文件和图像”。打开时，此设置只会使您从另一台计算机访问的任何笔记在下次启动时加载速度更快，方法是将更大的文件和图片下载到您正在使用的系统，而不是在每次关闭或重新打开文件时从外部服务器重新加载它们。

帐目

若要管理哪些帐户链接到您的OneNote，请打开“设置”菜单中的“帐户”选项卡。

从这里只会弹出一个窗口，其中包含已绑定到您的已注册Office 365帐户的用户列表，以及添加另一个帐户的选项。

如果您确实选择添加另一个帐户，您将只能使用已支付其Office 365订阅费用的企业(工作)或学校电子邮件地址来执行此操作。

独处 / 隐私 / 秘密 / 隐居

设置菜单中的下一个是隐私部分，自从Windows10第一次被删除以来，这个话题已经被证明是许多用户的热点问题。

相关：了解Windows 10的隐私设置

虽然这个设置所监控的大部分内容都是相对无害的(“稳定性问题”和“系统配置”只是听起来像什么)，但“您使用最频繁的功能”统计数据让我们感到毛骨悚然。

这意味着，在您在OneNote(或任何其他关联的Office 365应用程序)中执行的所有操作中，Microsoft都会记录您的鼠标移动位置、您单击的内容以及您使用其软件的任何特定部分的频率。

为了安全起见，不会通过线路不断单击发送这种级别的信息(也为了在笔记本电脑上编辑时节省电池)，建议您始终关闭此开关。

信任中心

在OneNote设置菜单的所有选项中，“信任中心”是目前为止最模糊的。

尽管除了“改进Office”之外，微软还没有详细说明他们想要这些数据做什么，或者他们如何处理这些数据，但我们建议，除非您出于某种原因明确需要这些数据，否则最好将这两个选项都关闭。

共享

现在您的笔记已经完成，是时候与世界上的其他人分享您的创作了。

就像Windows 10文件资源管理器中嵌入的文件共享功能一样，OneNote右上角的“共享”按钮将调出一个菜单，其中包含通过您之前通过Windows应用商店下载的任何应用程序将您创建的内容发送到Web上的选项。

在这种情况下，我们可以通过Windows Mail客户端、Facebook、直接通过Gmail或另一台计算机上的单独OneNote帐户持有人进行共享。

同样，如果您已经向任何单击它的人授予了权限，您也可以将要共享的任何页面或笔记本复制为剪贴板中的直接链接。为此，请找到您要共享的内容，然后右键单击它以调出以下子菜单。

选择“将链接复制到页面”选项，你最终会得到一个类似于下面这样的链接。从链接的在线门户，您可以控制任何有权访问便笺的人的编辑权限，还可以通过公共绘图功能、图像上载和共享文本进行协作。

这个菜单也是你可以找到“Pin to Start”选项的地方，它可以获取任何页面或笔记本，并以可快速点击的快捷方式的形式将整个内容发送到你的“开始”菜单。

OneNote可能不值得微软单独开价使用Office365套件，但它仍然配备了足够的功能和调整，让Evernote或Google Keep等其他免费选项物有所值。这是一种简单的方式来跟踪您的所有日常活动，并使与团队成员的协作和创建项目变得有趣、容易，并且可以从移动设备和桌面进行普遍访问。

您可以在微软在其主页上提供的链接中找到适用于您首选系统的OneNote下载。
