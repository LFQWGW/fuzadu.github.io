将Windows Media Center连接器设置为Windows Home Server

带有Power Pack 3的Windows Home Server的优势之一是它与Windows Media Center的集成。在这里，我们来看看如何安装Windows Media Center连接器以及它必须提供的功能。

安装连接器

安装Windows Home Server(WHS)后，您可能注意到在启动Windows Media Center(WMC)时会弹出一条消息。只需对该消息单击“确定”，除非您愿意，否则不需要立即设置。我们知道可以随时从“开始”菜单进行设置。

下面我们来看看如何在Windows7家庭高级版上安装它。打开“开始”菜单，然后单击“Windows Media Center连接器”，如下所示。

安装向导启动，您可以单击Next…

输入您的WHS…的密码

那太简单了！你都准备好了，…。但是，需要重新启动机器。

从重启返回后，打开WMC，在主菜单中您将看到家庭服务器的条目，其中包括TV Archive和Console View。

电视档案馆

电视档案功能让您轻松移动录制的电视到您的WHS。您可以选择单个节目或将所有录制的节目存档到服务器。

您还可以将压缩副本保存到服务器或公共视频文件夹。您可以从原始分辨率中进行选择，Windows Mobile为320×240，Zune为720×480，每个分辨率均为WMV格式。这样做的好处是您可以录制您最喜欢的节目，并将原始和/或压缩版本存储在服务器上。

控制台视图

另一个不错的功能是控制台视图。它可以让您舒适地坐在沙发上或电脑上查看有关服务器的不同统计数据，无论是哪种情况。

从控制台视图中，您可以看到：

存储。 驱动器。 备份。 共享文件夹。 介质计数。 服务器和网络运行状况。 Windows Home Server硬件信息

现在，当您进入录制的电视文件夹时，您会看到它正在从服务器中提取节目。这节省了本地计算机上的驱动器空间，并免除了手动传输文件的麻烦。

它还可以与vista中的媒体中心以及…配合使用

Windows Media Center连接器添加了用于管理录制的电视文件的附加功能。它可以将它们从本地机器的硬盘上移走，省去了您自己移动它们的麻烦。此外，能够从沙发上查看基本的服务器统计数据也是一个肯定的优势。
