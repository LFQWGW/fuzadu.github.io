11您可能不知道的Ubuntu One功能

虽然Ubuntu One看起来像是Ubuntu独有的文件同步服务，但它不止于此--你可以在Windows、Android、iOS和网络上使用Ubuntu One。Ubuntu One为每个人提供了5 GB的免费存储空间。

Ubuntu one包括在线共享文件或文件夹、将音乐流式传输到智能手机、在所有设备上同步已安装的应用程序等功能。

同步任何文件夹

默认情况下，Ubuntu只同步您主目录中Ubuntu one文件夹内的文件。不过，您可以右键单击任何文件夹，指向Ubuntu one菜单，然后选择Synchronize this older开始同步它。您可以从Ubuntu one应用程序管理您的同步文件夹。

限制带宽

默认情况下，Ubuntu One使用所有可用的带宽进行文件上传和下载，但如果您的连接速度较慢，您可以限制其上传和下载速度。带宽设置位于Ubuntu One窗口的设置窗格中。

在Windows上使用Ubuntu One

Ubuntu one不仅仅在Linux上运行--Ubuntu还提供了一个完全支持文件同步的Windows客户端。Ubuntu one是一个跨平台的文件同步服务--如果你是一个从未使用过Ubuntu的Windows用户，你甚至可以使用它。

共享文件

使用文件管理器或Ubuntu One网站中的右键菜单，您可以在互联网上公开或私下与其他Ubuntu One用户共享文件和文件夹。我们在过去已经介绍过使用Ubuntu one来共享文件。

同步安装的软件

Ubuntu One可以在你的电脑之间同步你从Ubuntu软件中心安装的软件，所以很容易跟踪你安装了哪些软件。我们过去已经介绍过Ubuntu软件中心的同步软件。

使用移动应用程序

Ubuntu one提供适用于Android、iPhone、iPad和iPod Touch的应用程序。有了这款应用，你可以从你的移动设备上随时随地访问你的Ubuntu One文件。

自动从智能手机上传照片

这款移动应用程序还可以自动将你在智能手机上拍摄的照片上传到你的个人Ubuntu one云中，这样你就可以轻松地在所有设备上访问你的照片。

移动音乐流媒体

Ubuntu One还提供适用于Android和iOS的Ubuntu One Music应用程序。这款应用程序允许你从任何地方将你的音乐流式传输到你的移动设备上-你还可以在你的设备上缓存文件以供离线收听。这项功能每月收费3.99美元，包括20 GB的存储空间-前30天你可以免费试用。它还与Ubuntu One的Rhythmbox音乐播放器附带的Ubuntu One音乐商店协同工作-您购买的任何音乐都会立即发送到您的Ubuntu One帐户，并在Ubuntu One Music应用程序中提供。

同步联系人

Ubuntu One可以同步您的联系人并在线存储。目前，你可以在Ubuntu One网站上从Facebook导入联系人，也可以手动添加。以前版本的Ubuntu支持与Evolution电子邮件客户端同步的联系人，但Ubuntu 12.04似乎没有与Thunderbird同步的联系人。

管理浏览器中的文件

如果您的计算机上没有安装Ubuntu one，您可以从Ubuntu one网站访问和管理您的文件。您可以从浏览器下载文件、上载文件或管理现有文件。

管理设备

从Ubuntu One窗口的Devices选项卡或Ubuntu One网站的Devices页面，您可以管理链接到您帐户的计算机和其他设备。
