Windows和Windows Server之间有什么不同？

微软提供Windows的台式机和服务器版本。乍一看，Windows 10和Windows Server 2016看起来很相似，但各自有不同的用途。Windows 10擅长日常使用，而Windows Server管理许多计算机、文件和服务。

Windows 10和Windows Server共享相似的代码

如果你加载Windows 10和Windows Server 2016的干净版本，一开始很容易混淆这两个系统。它们可以具有相同的桌面、相同的开始按钮，甚至相同的任务视图按钮。它们使用相同的内核，可以可行地运行相同的软件。例如，你可以在两者上都安装Google Chrome或Microsoft Office。

但相似之处仅限于此。微软把Windows10设计成一个你坐在前面的桌面，把Windows Server设计成一个服务器(它的名字就在那里)，运行人们通过网络访问的服务。虽然Windows Server确实有桌面选项，但Microsoft建议在没有图形用户界面的情况下安装Windows Server(或删除它)，只留下一个命令行来工作，以减少运行服务器所需的开销。这包括推送选择Nano Server，它会丢弃GUI和本地登录功能，以换取使用比标准Server安装少得多的空间。

Windows Server包括服务器软件

如果启用了GUI，则在Windows Server加载后不久，会启动一个服务器管理器程序，显示两个操作系统之间的第一个明显差异。您可以在此处添加特定于服务器的功能，如Windows部署服务、DHCP服务和Active Directory域服务。这些功能允许将操作系统远程部署到其他计算机、为客户端计算机创建静态IP地址、控制将其他计算机加入域的网络域以及创建域用户。这样的功能在Windows10本身是不可用的，尽管你可以安装第三方软件，如Apache Web服务器。

此外，Windows Server支持SMB Direct等功能，以实现更快的文件共享，更好地支持弹性文件系统，在没有服务器的情况下获得类似功能的唯一方法是将Windows 10 Pro用于工作站。

服务器也是为协同工作而设计的，因此您可能会让一台服务器完成上面的一到两个角色，而另一台服务器承担其他角色来分散工作。

Windows Server支持更高端的硬件

Windows Server还支持功能更强大的硬件。Windows 10专业版的最大内存限制为2 TB，而Windows Server允许的最大内存为24 TB。桌面用户甚至不太可能考虑如此大量的RAM，但服务器可以在通过Hyper-V管理多个用户、计算机和潜在虚拟机之间，充分利用其更大的RAM容量。

Windows10对处理器也有限制。Windows 10家庭版只支持一个物理CPU，而Windows 10专业版支持两个。Server 2016最多支持64个插槽。同样，Windows 10的32位版本只支持32个内核，64位版本支持256个内核，但Windows Server对内核没有限制。

为了更接近这些功能，您必须在工作站上使用Windows10专业版，它支持4个CPU和6 TB的RAM。

Windows Server已锁定

与Windows10的LTSB分支非常相似，Windows Server删除了几个功能。您将找不到Cortana、Microsoft Store、Edge或Timeline。Windows Server仍在使用Internet Explorer，而不是Edge，并且它被锁定以阻碍正常的Web浏览。在下载Google Chrome时，我们必须为所有Google的URL添加例外才能完成下载。Windows Server的额外安全性使其几乎出现在所有通过Internet Explorer访问的网站上。

Windows Server不支持使用Microsoft帐户登录，因此您无法将您的设置从另一台电脑带到它。相反，您需要使用本地帐户或域帐户登录。虽然Windows 10 Home最终获得了暂停更新功能，但Windows Server可以完全通过组策略禁用更新(Windows 10 Enterprise和Windows LTSB也可以)。

Windows 10是熟悉的桌面体验

虽然Windows10缺乏特定于服务器的功能，但它在其他方面弥补了这一不足。Windows10更新来得更快、更频繁，它有一些Windows Server上没有的功能，比如Timeline和Cortana，而且它没有被锁定。安装新软件，特别是从互联网上下载的软件，只需要很少的麻烦就可以通过，而且如果你使用Microsoft帐户登录，你的首选项就会从一台机器带到另一台机器。

此外，Windows10还有其他功能，如手机、渐进式Web应用程序和Windows Linux子系统。其中一些功能依赖于Windows Server无权访问的Microsoft应用商店。

如果您愿意，您可以更改Windows 10以满足您的需求，使其更像Windows 7。

Windows Server也更贵

如果你有Windows7、8或8.1密钥，你可以免费安装Windows10。Windows Server 2016许可证并不容易购买(毕竟它们是面向商业的)，而且价格昂贵。如果您是一家企业，根据您的规模和需要，单个许可证的费用可能在500美元到6200美元之间。大多数购买者转而使用批量许可路线。Windows Server主要面向企业，因此会相应定价。

如果你正在考虑在你的个人电脑上安装Windows操作系统，你的最佳选择是Windows10。你仍然可以使用Windows7、8或8.1键来激活它，而且这些功能都是为家庭用户量身定做的。但是，如果您希望Windows操作系统管理其他计算机，无论是在家中还是在办公室，提供文件服务器或Web服务器，那么Windows服务器是显而易见的选择。
