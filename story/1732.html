立即在您的PC上试用Windows Phone 7

自从微软在2月份的世界移动通信大会上发布了新的Windows Phone 7系列以来，人们对它的期待就一直在积累。现在，由于有了免费的开发工具，你可以在你的PC上获得基本Windows Phone 7系列设备的第一手体验。

Windows Phone 7系列代表着微软在移动领域的巨大变化，将广受赞誉的Zune HD UI带到了一个创新的手机平台上。Windows Mobile经常因落后于其他智能手机平台而受到批评，但微软试图通过即将发布的这个新版本夺回领先地位。“一个平台必须有开发者支持才能有用，所以他们已经发布了全套免费开发工具，这样任何人今天都可以为它制作应用程序。或者，如果你只是想玩Windows Phone 7，你可以使用内置的仿真器来尝试新的Metro UI。”

请注意：这些工具是客户技术预览版，因此只有在您习惯使用预发行软件时才安装它们。

开始

首先，下载Windows Phone开发人员工具CTP(下面的链接)，然后运行安装程序。这将在您的计算机上安装Visual Studio 2010 Express for Windows Phone、Windows Phone Emulator、Silverlight for Windows Phone和XNA 4.0 Game Studio的客户技术预览版(CTP)，所有这些都是必需的，不能单独安装。

出现提示时接受许可协议。

单击“立即安装”以安装您需要的工具。唯一的安装自定义选项是保存文件的位置，因此如果需要，请选择“自定义”。

安装程序现在将自动下载并安装您需要的组件，并且还将根据您的操作系统另外下载32位或64位程序。

大约安装到一半时，系统会提示您重新启动系统。

重新启动计算机后，安装程序将自动恢复，无需进一步输入。

安装完成后，单击“立即运行产品”开始。

在您的PC上运行Windows Phone 7

现在您已经安装了Windows Phone开发人员工具，是时候运行Windows Phone模拟器了。如果在安装完成后单击了“立即运行产品”，Visual Studio 2010 Express for Windows Phone应该已经启动了。

如果没有，只需在开始菜单搜索中输入“visual studio”，然后选择“Microsoft Visual Studio2010速成版Windows Phone”即可。

现在，要运行Windows Phone 7仿真器，我们必须测试一个应用程序，因此，即使您不会编程，我们也可以打开一个手机应用程序模板，然后测试它来运行仿真器，首先单击首页左侧的新建项目。

任何应用程序模板都可以做到这一点，但这里让我们选择“Windows Phone应用程序”，然后单击“确定”。

这是你的新应用程序模板，它已经包含了基本的手机应用程序框架。如果你想开发Windows Phone应用程序，可以从这里开始，但目前我们只想看看Windows Phone 7的运行情况。

因此，要运行仿真程序，请单击菜单中的“调试”，然后选择“开始调试”。

您的新应用程序将在Windows Phone 7系列仿真器中启动。*默认模板提供的功能不多，但它确实显示了在Windows Phone 7中运行的示例应用程序。

探索Windows Phone 7

点击模拟器上的Windows按钮进入主屏幕。注意到类似Zune HD的过渡动画。模拟器只包括Internet Explorer、您的测试应用程序和一些设置。

单击右侧的箭头可查看列表中的可用应用程序。

“设置”允许您在仿真程序中更改主题、区域设置以及日期和时间。*它也有“应用程序设置”窗格，但此窗格当前未填充。

时间设置显示独特的Windows Phone用户界面。

您可以通过按Windows按钮返回主屏幕。这是正在运行的Internet Explorer应用程序，虚拟键盘处于打开状态以输入地址。请注意，此仿真器也可以接受键盘输入，因此您无需单击虚拟键盘即可输入地址。

这是在Windows phone7的IE浏览器中运行的谷歌。

Windows Phone 7支持加速度计，您可以在模拟器中模拟此功能。/单击其中一个旋转按钮可沿该方向旋转屏幕。

在Windows phone7的横向模式下，这是我们在Internet Explorer中最喜欢的网站。
