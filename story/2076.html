初学者极客：OneNote 2010入门

您听说过OneNote吗，但想知道它到底是什么，以及如何使用它？让我们来看看OneNote 2010，看看如何使用它来保持更有条理和更紧密的联系。

获取OneNote 2010

如果您刚刚开始使用Office 2010，您将在安装Office时获得OneNote。默认情况下，OneNote随Office 2010的所有版本(基本简易版除外)一起安装，因此无论您是在家里还是在企业环境中使用Office，您都会拥有它。您可以单独购买OneNote，但我们建议您随Office套件一起购买，因为如果您经常使用2个或更多Office程序，这通常是最经济的方法。

如果您已安装Office 2010，则您的计算机上可能已经安装了OneNote 2010。您可以在Microsoft Office下的[开始]菜单中找到它，或者只需在[开始]菜单搜索中输入OneNote即可快速找到它。

如果您不想购买Office 2010或不使用Windows，您仍然可以在新的Office Web Apps中试用OneNote 2010。*这将在您的浏览器中提供功能受限的OneNote版本。*即使您的计算机上安装了OneNote，这也是一个有用的选项，因此请查看我们的Office Web Apps截图教程，了解有关OneNote Web App的更多信息。

在OneNote中查找您的方式

当您第一次启动OneNote时，您将看到一个示例笔记本，其中包含有关如何使用它的一些信息和示例。

OneNote将您的笔记组织在笔记本中，每个笔记本分为多个分区，可能有多个页面。您的笔记本位于左侧，分区位于顶部作为选项卡，页面列在右侧。

您可以从文件选项卡创建新笔记本。*默认情况下，新笔记本将联机存储在Office Web Apps中，以便您可以从任何位置访问并与其他人共享，但您也可以将笔记本仅存储在您的计算机上。有关详细信息，请查看我们关于如何将OneNote与Office Live同步并随处编辑的文章。或者，如果您要与OneNote 2007用户共享笔记本，请查看我们关于如何以OneNote 2007格式转换和共享笔记本的文章。/或者，如果您想要与OneNote 2007用户共享笔记本，请查看我们关于如何转换和共享OneNote 2007格式的笔记本的文章。/或者，如果您想要与OneNote 2007用户共享笔记本，请查看我们关于如何转换和共享OneNote 2007格式的笔记本的文章。

您可以通过按右侧带星号的选项卡来创建新的笔记本分区。然后，在窗口中间单击以创建新页面，或按右侧的New Page。

您可以在列表中四处拖动页面以按照您的需要对其进行组织，还可以通过将笔记拖到另一个页面下的右侧将其与另一个注释分组。然后，您可以折叠页面组以保持它们的组织和整洁。

做笔记

您可以在页面上的任何位置输入文本、图片等内容。您输入的所有内容都包含在一个容器中，您可以在页面中移动，这使得组织笔记变得很容易。

您可能会注意到，显示的编辑工具并不多，但默认情况下这些工具只是处于隐藏状态。“选择顶部的”主页“、”插入“或其他选项卡可查看功能区中的选项和工具。

如果希望工具始终可用，请双击功能区选项卡或单击退出按钮下的箭头。

笔记不必枯燥乏味；您可以从插入选项卡中插入各种各样的项目，包括文件、视频和屏幕剪辑。*屏幕剪辑工具非常有用；按下按钮或键盘快捷键Win+s可以捕获屏幕上的任何内容并将其添加到笔记本中。

如果您有触摸屏或绘图板，也可以在OneNote中创建墨迹注释。

OneNote也非常适合创建大纲。您可以重新排列项目符号和大纲，就像您可以重新排列笔记本的任何其他部分一样。

由于OneNote旨在帮助您跟上所有数据，因此它与高级搜索集成是很自然的。您可以搜索笔记中的文本，包括图像中的文本。只需在窗口右上角的搜索框中输入您要查找的文本，OneNote就会在您键入时自动开始从所有笔记本中查找结果。

OneNote包含强大的OCR技术，可用于索引和搜索图像中的文本，但您也可以利用该技术从图片中复制文本。请查看我们关于如何使用OneNote OCR任何内容的文章。

将数据发送到OneNote

OneNote使您可以轻松地从各种程序收集笔记。*默认情况下，OneNote将加载到您的系统托盘中，以便您可以快速创建屏幕截图或开始记录新笔记。

OneNote安装了虚拟打印机，因此您可以将任何程序的信息发送到OneNote。您只需像往常一样从您的程序打印，但在打印对话框中选择发送到OneNote 2010。

如果您的Office版本包含Outlook，您还可以直接从功能区轻松地将电子邮件添加到OneNote，这样您就永远不会丢失重要数据。

现在，简化的OneNote窗口将停留在屏幕的右侧，准备好让您草草记下并保存任何内容。

结束 / 结尾 / 结论 / 推论

无论您要保存什么内容，OneNote 2010都能让您轻松快捷地保存笔记、想法、屏幕截图等内容。*因为它现在是所有Office 2010套件的一部分，所以您很有可能尝试一下，看看它是如何为您自己工作的。我们喜欢OneNote中的页面设计，它允许您随时随地移动任何内容，新的Office Live集成可帮助您随时随地保存笔记。
