使用Amahi升级您的Windows Home Server

Windows Home Server(WHS)是市场上最可靠、功能最丰富的网络附加存储设备之一。然而，WHS 2011删除了一些关键功能。如果你想在不丢失功能的情况下进行升级，只需看看Amahi就可以了。

Windows Home Server在2007年发布时受到好评的功能有：

驱动器扩展器：允许多磁盘冗余，并将多个驱动器合并到单个共享空间。 远程访问：从您的网络内外访问服务器。 加载项：向您的WHS添加更多功能，而无需计算复杂的配置

Windows Home Server 2011最近问世，但它去掉了驱动器扩展器功能，需要比其前身快40%的CPU(1.4 GHz)和4倍的RAM(2 GB)。如果您正在寻找升级，但又不想失去您现在拥有的所有功能，开源项目Amahi可能是您完美的解决方案。

Amahi是一个构建在Fedora上的Linux发行版(一个基于Ubuntu的系统正在开发中)，它使得设置家庭NAS变得很容易。除了其他基于Linux/BSD的NAS发行版为您提供的功能之外，Amahi还可以轻松地远程访问自动动态DNS更新、带有Greyhole的驱动器扩展器和功能，以及通过其“不要将其称为应用商店”存储库一键安装加载项。

我们将带您完成在HP Media Smart服务器上使用Express CD安装Amahi的过程。您可以在任何备用计算机上安装Amahi，但是Express CD是为在无头硬件(没有显示器的计算机)上运行而设计的。由于HP MediaSmart服务器已经是一款低功耗、多驱动器的服务器，因此非常适合免费升级。我们的下一篇文章将向您展示设置和使用新的家庭数字助理(HDA)的一些基础知识。

步骤1：收集需求

要在MediaSmart服务器上执行安装，您需要一些东西。

A Amahi帐户：您需要注册帐户，这样您才能获得我们稍后将使用的安装代码。 Express cd iso：从下面的链接下载。 一张空白CD：相信我们，USB安装需要一些破解，在安装2之后，它的工作方式仍然与从物理CD安装不太一样。 备用硬盘：可以小到4 GB，但我们推荐更多，这样您也可以将其用于存储。 备用台式计算机：尽管Express CD是为无头计算机设计的，但初始安装时仍然需要显示器。

第2步：备份您的现有NAS

如果要在现有NAS上安装Amahi，第一步将是备份。如果您有Windows Home Server，请按照我们的指导将您的所有信息备份到外部硬盘。

如果您有不同的NAS平台，您可以查看您的平台是否有内置备份系统，也可以通过网络手动将所有文件复制到备用硬盘。

在开始安装之前，请确保您有一个备份，如果您有足够的备用硬盘，您应该保持现有的NAS完好无损，以防出现严重的问题。

步骤3：安装Amahi

将iso文件刻录到CD后，将备用硬盘驱动器放入要用来执行安装的临时桌面。安装将擦除备用硬盘以及插入系统的任何其它硬盘上的所有内容。因此，请确保您只有您想要插入的硬盘。

图片来自贾斯汀·拉克曼(Justin Ruckman)

插入正确的硬盘后，从安装CD启动系统。前几个步骤只是要求您选择您的语言、键盘布局、时区和root密码，因此我们将跳过这些步骤，并假设您知道要在那里做什么。

在进行初始文件复制和驱动器格式化时，请转到另一台计算机上的amahi.org，然后使用您的用户帐户登录。登录后，您将进入控制面板。单击左侧的“Your HDAS”，然后向下滚动到为您生成的安装代码。

初始驱动器格式化和文件复制完成后，在出现提示时输入安装代码。

机器将重新启动(请确保取出安装CD)，并将您带回另一个屏幕以完成设备配置。完成后，您的新HDA应该会自动重启，并为您带来一个登录屏幕。

可选步骤：EXPRESS CD中当前有一个错误，导致我的机器一遍又一遍地重复上述步骤。要跳过此步骤，请选择转到调试控制台并运行命令ConfigAmahi的选项。系统将完成HDA配置并重新引导到最终可用状态。

步骤4：硬盘移植

如果您不使用HP MediaSmart服务器或希望将HDA保留在刚刚安装的台式机上，则不需要执行以下步骤。仅当您要将HDA移动到新机器时，这些才适用。

机器重新启动后，使用您在安装过程中设置的root用户和密码登录到计算机。

为了允许Amahi更改物理硬件，我们只需要配置机器，使网络在新机器上工作。从终端运行以下命令。

rm/etc/udev/rules.d/70-Persistent-Net-Rules

然后使用以下命令打开eth0配置文件

Nano/etc/sysconfig/network-scripts/ifcfg-eth0

在文本编辑器中，删除以HWADDR开头的行，并将ONBOOT=NO更改为ONBOOT=YES。

下次系统启动时，系统将自动重新配置有线以太网适配器。关闭机器并拔出硬盘。从HP MediaSmart服务器上卸下所有硬盘，然后将新硬盘放入最低的插槽中。

打开系统电源，检查您的路由器以确保设备出现在网络上，然后从另一台计算机打开网页以配置Amahi。
