无线打印说明：AirPrint、Google Cloud Print、iPrint、ePrint等

标准的好处是有太多的标准可供选择。这绝对适用于无线打印机。购买打印机时，您会发现大多数打印机都支持各种不同的打印标准。

这些标准之所以出现，是因为打印-即使是无线打印-可能会非常混乱。它们都旨在使打印变得更容易，但它们都不同，工作方式也不同。

Wi-Fi打印

相关：通过网络或Internet远程打印的4种简单方法

标准Wi-Fi打印本质上是标准有线USB打印的无线形式。与USB打印一样，它需要打印机驱动程序。

支持Wi-Fi的打印机可以连接到您的无线网络，从而可供其他计算机和设备使用。然后，网络上的计算机可以通过网络打印。这也使得在多台计算机之间共享一台打印机变得容易。然而，这个过程仍然可能相当混乱。您仍然需要安装适当的打印机驱动程序，然后计算机才能打印到它。

这可能是一个问题。例如，假设您想要从iPhone、Android手机、iPad或其他类型的平板电脑打印到无线打印机？您应该如何在手机或平板电脑上安装打印机驱动程序？

蓝牙打印

相关：不只是头戴式耳机：蓝牙可以做的5件事

一些打印机支持蓝牙打印，尽管这远没有Wi-Fi打印那么普遍。要使用它，你需要一部集成蓝牙的手机、平板电脑或笔记本电脑。然后，您必须将您的设备和打印机配对。这是通过本地短距离蓝牙连接实现的，因此设备之间必须足够近才能正常工作。

然后，只要你在打印机附近，你就可以通过蓝牙将文档发送到打印机，就像你可以使用蓝牙配对耳机或在附近的设备之间传输文件一样。

蓝牙打印可以，但不方便。许多打印机不包括蓝牙无线电，当它们配备蓝牙无线电时，您需要在设备上启用蓝牙无线电，完成配对过程，并在打印之前离打印机足够近。有了支持Wi-Fi的打印机，同一网络上的任何设备都可以使用打印机，即使它们相距很远或没有蓝牙无线电。

Apple AirPrint

AirPrint是苹果解决打印机驱动程序和蓝牙配对问题的解决方案。支持AirPrint的打印机将宣传为兼容AirPrint。大多数打印机制造商生产与AirPrint兼容的打印机，同时还支持其他无线打印标准。你不必购买专门的苹果设备打印机就可以使用AirPrint。

要使用AirPrint打印机，您只需像典型的Wi-Fi打印机一样将其连接到您的无线网络即可。接下来，您可以拿起您的iPad、iPhone、iPod Touch或Mac电脑，在任何程序中选择打印选项。然后，您将看到本地网络上兼容AirPrint的打印机列表。要打印到打印机，只需从列表中选择打印机即可。

就这样。您不必安装打印机驱动程序或进行配对过程。苹果设备将自动检测同一网络上的AirPrint打印机，无需进一步配置即可打印。

AirPrint非常方便，但最大的缺点是它只支持苹果设备。你不能使用AirPrint从Windows PC或Android设备打印-至少，如果没有可能无法工作的非官方黑客解决方案，就不能打印。幸运的是，兼容AirPrint的打印机通常还支持其他类型的无线打印标准，因此您也可以从非苹果设备打印到它们。

GoogleCloud打印

相关：如何(以及为什么)开始使用Google Cloud Print

Google Cloud Print是谷歌对无线打印混乱局面的解决方案。当您使用支持Google Cloud打印的打印机时，它会像另一台支持Wi-Fi的打印机一样连接到您的无线网络。然后，您可以将您的打印机与一个Google帐户相关联，该帐户通过Internet与您进行通信。

然后，您只需使用您的Google帐户登录，即可从任何设备通过Google Cloud Print打印到您的打印机。Google Cloud Print提供与Android和Chrome的集成，以及苹果iOS的应用程序和与标准Windows打印系统的集成。当您打印到Google Cloud打印打印机时，您的文档将通过Internet发送到Google，Google再将其发送到您的打印机。

这意味着你可以使用Google Cloud Print做一些更高级的事情，比如在互联网上打印，而不需要扰乱端口转发，也不需要通过其他人的谷歌账户轻松地与他们共享你的打印机。

与苹果的AirPrint不同，Google Cloud Print可以在许多不同的操作系统上使用。您甚至可以使用Google Cloud Print连接器使您现有的打印机充当Google Cloud打印打印机，从而允许您从移动设备打印到它。

iPrint、ePrint和其他制造商特定的解决方案

为了不被苹果和谷歌超越，打印机制造商已经创建了自己的无线打印标准。这包括Epson iPrint、HP ePrint等。

这些标准背后的想法是，您可以简单地从移动设备的应用程序商店下载相关的应用程序-例如，Epson iPrint应用程序或HP ePrint应用程序。然后，该应用程序可以通过网络无线打印到制造商的一台打印机。

这些可能是有用的解决方案，特别是如果你有一台支持Wi-Fi的打印机，但它不支持其他标准，如Apple AirPrint或Google Cloud Print。但是，它们只是没有与底层系统集成，可能无法打印您想要打印的每一种类型的文档。

如果你想从iPhone上打印，你可以选择苹果集成的AirPrint或Epson iPrint应用程序，使用AirPrint可能要好得多。

上面有一个名字明显没有出现，那就是微软。如果你有Windows Phone设备，你将不得不使用制造商专用的应用程序，因为微软还没有开发出自己的标准来轻松连接无线打印机。

无线打印空间有点乱七八糟，而且不必要地令人困惑。然而，好的一面是这些标准并不是相互排斥的。您可以轻松获得支持标准Wi-Fi打印、Apple AirPrint、Google Cloud Print和打印机制造商自己的解决方案的新打印机。然后，您可以选择适合您当时使用的任何设备的类型或打印机连接。
