在Android、iPhone和iPad上使用Chrome浏览的10个提示

无论您使用的是台式PC、智能手机还是平板电脑，Chrome都是一款功能强大的浏览器。这些小贴士将帮助你更快地浏览，并在旅途中利用Chrome的功能。

如果你的设备没有附带Chrome for Android，你可以从Google Play上获得它。苹果应用商店提供iPhone和iPad版Chrome。

在选项卡之间滑动

相关：在iPad和iPhone上使用Safari浏览的8个提示和诀窍

Chrome提供了滑动手势，可以在打开的标签之间快速切换。这些手势只适用于Android手机、iPhone和iPad版的Chrome，而不适用于Android平板电脑版本。

在Android手机上，将手指放在Chrome工具栏上的任何位置，然后向左或向右滑动。

在iPhone或iPad上，将手指放在屏幕边缘，向内滑动，即可在打开的选项卡之间移动。此功能取代了iOS上Safari中的“滑动后退或前进”手势。

使用选项卡列表-仅电话

在手机上，触摸Chrome的Tab键即可查看所有打开的标签。轻触选项卡可切换到该选项卡，轻触X按钮可关闭选项卡，或将手指放在打开的选项卡上并向左或向右滑动以将其关闭。您可以通过选择菜单中的关闭所有选项卡来快速关闭所有打开的选项卡。

双击以智能缩放

Chrome有一个“智能缩放”功能，可以让你在网页上的任何地方双击并放大。例如，如果你正在浏览一个桌面网站，而主内容栏很小，双击它，Chrome就会智能地放大到页面的这一部分。双击的列将占据显示器的整个宽度。

请注意，此功能仅适用于为桌面浏览器设计的网站。谷歌在移动优化网站上禁用了智能缩放功能，以加快速度。当然，标准的按键到缩放手势也可以在Chrome中使用。

快速选择菜单选项-仅限Android

在大多数Android应用程序中使用菜单时，您需要轻触菜单按钮，等待菜单出现，然后轻触您想要选择的菜单选项。Chrome提供了一种更快的方法。触摸菜单按钮，向下移动手指，直到位于菜单选项上方，然后将手指从屏幕上抬起。这允许您在单个运动中选择一个菜单选项。

谷歌语音搜索

你可以在Chrome的任何地方轻松访问谷歌语音搜索。在平板电脑上，只需点击地址栏中的麦克风图标即可。在电话上，点击地址栏，点击X，然后点击出现的麦克风图标。您可以在这里进行搜索，说出网站地址，或提出问题。

请求桌面站点

要在Chrome中请求网站的桌面版本，请点击菜单按钮，然后选择请求桌面站点。Chrome将伪装成桌面浏览器。

打开选项卡和浏览器数据同步

相关：如何在任何浏览器中同步浏览器数据并随时随地访问

当您使用Google帐户登录时，Chrome会在您的设备之间同步打开的选项卡和其他浏览器数据。如果你在笔记本电脑、智能手机和平板电脑上使用Chrome，这一功能尤其有用--你会一直在其他设备上看到你打开的标签，这样你就可以轻松地从你停止的地方开始。要查看打开的标签，请访问Chrome的新标签页并点击页面右下角的Other Devices选项，或者点击Menu按钮并选择Other Devices。

启用预加载和带宽减少

Chrome的预加载和带宽缩减功能可以节省时间和带宽。预加载在默认情况下是打开的，它会让Chrome自动获取它认为您下一步要加载的网页。例如，如果你正在阅读一篇长达五页的文章，你会看到第二页的末尾，Chrome会在你点击链接之前获取第三页，所以当你点击链接时，页面的加载速度会更快。默认情况下，该功能仅在Wi-Fi上启用，以避免浪费宝贵的移动数据。

相关：如何在智能手机上浏览网页时减少数据使用量

如果你启用了减少数据使用功能，你加载的网页将通过谷歌的服务器进行路由，并在它们到达你之前进行压缩。这将减少您的带宽使用量-如果您想保存有限数量的移动数据，这一点很重要。

要配置这些功能，请点击菜单按钮，点击设置，然后选择Android上的带宽管理或iPhone或iPad上的带宽。

GoogleCloud打印

相关：无线打印说明：AirPrint、Google Cloud Print、iPrint、ePrint等

Chrome在所有平台上都集成了Google Cloud Print。轻触菜单按钮，然后轻触打印，即可打印到任何支持Google Cloud Print的打印机。如果您没有支持此功能的打印机，您可以在台式计算机上安装Chrome，使任何打印机都可以通过Google Cloud Print访问。

在iPhone和iPad上，Chrome也支持苹果的AirPrint-但Google Cloud Print允许你从智能手机或平板电脑上访问任何打印机。

旗子

相关：如何访问Chrome(和Chromebook)中的实验功能

桌面版的Chrome包含隐藏的实验功能，Android版也是如此。在Chrome的地址栏中键入Chrome：//标志即可访问这些功能。你会在Chrome Beta for Android中找到更多新功能，你还可以通过在地址栏中键入chrome：//chrome-urls来找到更多隐藏的Chrome页面。

请注意，此功能在iPhone和iPad上不可用。Chrome本质上只是Safari在这些平台上的一个不同界面。

除非您知道自己在做什么，否则不应该更改任何这些设置。

如果你在越狱的iPhone或iPad上使用Chrome，你可以使用第三方软件将Chrome设为默认浏览器，并启用仅为Safari预留的快速Nitro JavaScript引擎。
