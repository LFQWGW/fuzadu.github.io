如何在Windows 10上使用内置的Windows Defender Antivirus

Windows 10内置了名为Windows Defender的实时杀毒软件，实际上相当不错。它自动在后台运行，确保所有Windows用户都受到病毒和其他恶意程序的保护。以下是它的工作原理。

从Windows 10的安全创建者更新开始，Windows Defender的用户界面略有变化，并被集成到新的Windows Defender安全中心-该中心还提供对安全相关工具的访问，如家庭保护、防火墙设置、设备性能和运行状况报告以及浏览器安全控件。如果您还没有更新到创建者更新，您应该仍然能够很好地跟随。

什么是Windows Defender？

微软在Windows XP、Vista和7时代提供了一款名为Microsoft Security Essentials的独立杀毒应用程序。在Windows 8中，该产品进行了一些整理，与Windows捆绑在一起，并更名为Windows Defender。如果有点好坏参半的话，那就相当不错了。诚然，其他反病毒应用程序-如BitDefender和Kaspersky-在基准测试中可以保护您免受更多病毒的攻击。

但Windows Defender也有一些优势。到目前为止，它是最没有侵入性的应用程序，只要有可能就在后台处理事情，而不是一直唠叨你。Windows Defender与Web浏览器和其他应用程序的配合也更好-比大多数其他防病毒应用程序更尊重它们的安全和隐私设置。

您可以自行决定使用什么，但Windows Defender不是一个糟糕的选择(几年前它已经克服了大部分问题)。然而，我们建议除了您选择的任何防病毒应用程序外，还可以运行像MalwareBytes这样的反恶意软件应用程序。

利用自动扫描和更新功能

与其他防病毒应用程序一样，Windows Defender会自动在后台运行，在下载文件、从外部驱动器传输文件以及在打开文件之前扫描文件。

相关：为什么防病毒软件隔离病毒而不是删除它们？

您完全不需要考虑Windows Defender。它只会在发现恶意软件时弹出通知您。它甚至不会问你想要如何处理它发现的恶意软件-它只会自动清理东西，并对文件进行隔离。

你偶尔会看到一个通知弹出窗口，让你知道何时执行了扫描，通常你可以通过在Windows 10中打开操作中心来查看上次扫描的详细信息。

如果Windows Defender确实发现了威胁，您还会看到一条通知，告诉您它正在采取行动清除这些威胁-您不需要采取任何行动。

相关：您将无法在Windows 10主页上禁用(或延迟)Windows更新

防病毒定义更新通过Windows Update自动到达，并像安装任何其他系统更新一样进行安装。这些类型的更新不需要重新启动计算机。这样一来，用户就不需要担心Windows Defender的更新，因为这一切都是在后台悄悄自动处理的。

查看您的扫描历史记录和隔离的恶意软件

您可以随时查看Windows Defender的扫描历史记录，如果通知您它已阻止恶意软件，您也可以查看该信息。要启动Windows Defender安全中心，只需点击开始，键入“Defender”，然后选择“Windows Defender安全中心”。

在Windows Defender安全中心窗口中，切换到“Windows Defender”选项卡(盾牌图标)，然后单击“扫描历史记录”链接。

“扫描历史记录”屏幕显示所有当前威胁，以及有关上次扫描的信息。如果您想查看隔离威胁的完整历史记录，只需单击该部分中的“查看完整历史记录”链接即可。

在这里，您可以看到Windows Defender已隔离的所有威胁。要查看有关威胁的更多信息，请单击其右侧的箭头。要查看更多信息，请单击展开特定威胁时显示的“查看详细信息”链接。

在这里您不需要执行任何其他操作，但如果您没有让Windows Defender在发现威胁时将其删除，您将可以在此屏幕上选择执行此操作。您还可以从隔离中恢复该项目，但只有在您绝对确定检测到的恶意软件是高度误报的情况下才应执行此操作。如果你不是绝对的，百分之百的确定，不要让它运行。

执行手动扫描

回到主“Windows Defender”选项卡上，您还可以通过单击“Quick Scan”按钮让Windows Defender运行快速手动扫描。通常情况下，您不需要为此而烦恼，因为Windows Defender提供实时保护，还会执行定期的自动扫描。然而，如果你只是想要安全-也许你只是更新了你的病毒定义-运行快速扫描绝对没有坏处。

您还可以单击该屏幕上的“高级扫描”链接来运行三种不同类型的扫描：

全扫描：因为快速扫描只扫描您的内存和常见位置。完全扫描将检查每个文件和正在运行的程序。这很容易需要一个小时或更长时间，所以当你不打算经常使用电脑时，最好这样做。 自定义扫描：自定义扫描允许您选择要扫描的特定文件夹。您也可以通过右键单击PC上的任何文件夹并从上下文菜单中选择“使用Windows Defender扫描”来执行此操作。 Windows Defender脱机扫描：Windows运行时，某些恶意软件很难删除。当您选择脱机扫描时，Windows会重新启动并在Windows加载到PC之前运行扫描。

配置病毒和威胁防护设置

默认情况下，Windows Defender自动启用实时保护、基于云的保护和示例提交。实时防护可确保Windows Defender通过实时扫描您的系统自动发现恶意软件。如果出于性能原因需要，您可以在短时间内禁用此功能，但Windows Defender将自动重新启用实时保护，以确保您以后的安全。基于云的保护和示例提交允许Windows Defender与Microsoft共享有关威胁和检测到的实际恶意软件文件的信息。

要启用或禁用任何这些设置，请单击主“Windows Defender”选项卡上的“病毒和威胁防护设置”链接。

然后在出现的屏幕上切换设置。

为某些文件夹或文件设置排除项

如果您向下滚动同一“病毒和威胁防护设置”页面的最底部，您还可以设置排除项-您可能不希望Windows Defender扫描的文件、文件夹、文件类型或进程。只需单击“添加或删除排除项”链接即可。

如果反病毒软件通过扫描某个你知道是安全的应用程序来显著降低速度，创建排除项可以再次加快速度。如果您使用虚拟机，您可能希望从扫描过程中排除这些大文件。如果你有一个你知道是安全的大型照片或视频库，你不会真的想要扫描减慢你的编辑速度。

要添加排除项，请单击“添加排除项”按钮，从下拉菜单中选择要添加的排除项类型，然后将Windows Defender指向要排除的任何项。

只是要小心，谨慎而聪明地使用排除。您添加的每个排除项都会略微降低您PC的安全性，因为它们会告诉Windows Defender不要查看某些地方。

如果您安装另一个防病毒软件怎么办？

如果您安装了其他防病毒应用，Windows 10会自动禁用Windows Defender。在安装另一个防病毒应用时，Windows Defender将不会继续执行实时扫描，因此它不会干扰你的其他应用。不过，您仍然可以使用Windows Defender执行手动或脱机扫描，作为首选防病毒应用程序的备份。

如果您卸载了其他防病毒软件，Windows Defender将自动重新启动并接管，从而提供防病毒保护。

然而，请注意，某些反恶意软件应用程序-如MalwareBytes-可以与Windows Defender一起安装，两者都将提供免费的实时保护。

无论您喜欢哪种防病毒产品，未来每一次新的Windows安装都至少会有一个基准内置防病毒保护，这是件好事。虽然它可能并不完美，但Windows Defender确实做得不错，侵入性最小，当与其他安全的计算和浏览实践相结合时，可能就足够了。
