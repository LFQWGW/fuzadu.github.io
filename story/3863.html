如何使用虚拟机同时运行Internet Explorer 7、8和9

如果您开发网站，您需要使用多个版本的不同浏览器来测试您的网站。有多种方法可以在同一台计算机上运行多个版本的IE，但某些版本不能同时运行。

但是，我们将向您介绍一种绕过此限制的方法，该方法允许您在Windows中同时运行Internet Explorer 7、8和9。微软已经创建了一些自定义的Windows VHD文件，允许网页设计者在Internet Explorer7、8和9中免费测试他们的网站。您可以将这些文件导入Microsoft Virtual PC。Windows虚拟机中提供以下版本的Internet Explorer。

Windows Vista中的IE7-IE8和IE9的安装文件也可安装在此虚拟机中。 Windows7中的IE8以及IE9的安装文件-IE9的安装文件也可安装在此虚拟机中。 Windows 7中的IE9

请注意，Windows 7和Vista VHD文件很大，并且拆分到多个文件中。下载您要运行的IE版本的每个虚拟机的所有文件。我们将向您展示如何解压缩这些文件，以便它们创建完整的VHD文件。您还需要下载Microsoft Virtual PC，它实际上是Windows的更新。虚拟机和Virtual PC的下载链接在本文的末尾。要下载Virtual PC，您必须验证Windows。本文末尾提供了下载链接。

注意：如果您在其中一个虚拟机上安装较新版本的IE，它将替换较早版本。如果需要运行所有三个版本，请为每个版本创建一个虚拟机。

要安装Virtual PC，请双击您下载的.msu文件。

此时将显示确认对话框。单击是安装更新。

阅读许可条款，然后单击I Accept继续安装。

将显示安装进度。

更新安装完成后，将显示以下对话框。若要立即重新启动您的电脑，请单击“立即重新启动”。如果要确保在重新启动之前所有内容都已关闭，请单击“关闭”。但是，请确保在使用Virtual PC之前重新启动PC。

要展开IE虚拟机，请双击.exe文件，它是压缩文件的第一部分。在本例中，我们将提取IE8 Windows7虚拟机并将其导入Virtual PC。

如果显示Open File-Security Warning(打开文件-安全警告)对话框，请单击Run(运行)继续安装。

阅读许可协议，然后单击接受以继续提取虚拟机文件。

此时将显示WinRAR自解压存档对话框。接受目标文件夹的默认位置(应该是压缩文件的当前位置)，或者单击“浏览”为生成的虚拟机文件选择其他位置。我们接受了默认位置。单击Install(安装)。

将显示安装进度。

解压文件后，您将看到一个.vhd文件(虚拟机硬盘驱动器)和一个包含虚拟机设置的.vmc文件。

若要打开Virtual PC，请从“开始”菜单上的“Windows Virtual PC”文件夹中选择“Windows Virtual PC”。

Windows资源管理器窗口将打开，显示虚拟机的位置(如果存在)。单击工具栏上的创建虚拟机按钮。如果看不到该按钮，请单击双向右箭头按钮，然后从下拉菜单中选择创建虚拟机。

此时将显示创建虚拟机向导。在名称编辑框中输入虚拟机的名称。接受虚拟机文件的默认位置。它将放置在启动Virtual PC时在Windows资源管理器窗口中打开的Virtual Machines文件夹中。单击“下一步”。

此时将显示指定内存和网络选项屏幕。在编辑框中输入希望虚拟机使用的RAM大小(以MB为单位)。若要将虚拟机连接到您的网络，请选中“使用计算机网络连接”复选框。您很可能需要打开此选项，以便您的虚拟机可以通过您的网络访问Internet。

在Add a Virtual Hard Disk屏幕上，选择Use an Existing Virtual Hard Disk单选按钮，然后单击Browse。

在选择虚拟硬盘对话框中，导航到解压.vhd和.vmc虚拟机文件的文件夹。选择.vhd文件，然后单击打开。

选中Enable Undo Disks(启用撤消磁盘)复选框。

注意：启用撤消磁盘选项允许您返回到首次在Virtual PC中设置虚拟机时存在的初始设置。这很有帮助，因为这些虚拟机中的Windows版本未激活。他们基本上处于审判模式。从Microsoft站点：

我们将在本文后面解释如何撤消更改并将虚拟机重置为初始设置。

单击创建。

您应该会在启动Virtual PC时打开的资源管理器窗口的Virtual Machines文件夹中看到一个.vmcx文件。

若要打开虚拟机，请选择.vmcx文件，然后单击变为可用的“打开”按钮旁边的箭头按钮。从下拉菜单中选择Windows Virtual PC。

在启动期间，虚拟机屏幕上会显示两个用户。每个虚拟机都有两个用户，但似乎只有一个用户可以工作。为每个虚拟机选择以下用户名：

Windows Vista中的IE7：管理员。 Windows 7中的IE8：IEUser。 Windows 7中的IE9：管理员

对每台虚拟机使用密码“Password1”(不带引号)登录Windows。

此时将显示Windows激活对话框，说明激活期已过。单击对话框右下角的Cancel(取消)。

可能会显示Microsoft Security Essentials对话框。单击关闭。

在虚拟机中启动Windows后，单击任务栏上的Internet Explorer图标以打开IE。

IE打开。您可以设置默认主页并查看需要在此版本的IE中测试的任何网页。

您可以通过从“帮助”菜单中选择“关于Internet Explorer”来检查版本。

要关闭虚拟机，请从操作菜单中选择关闭。

在Windows Virtual PC对话框中，从下拉列表中选择关机。如果要将关机设置为默认选项，请选中“将其设为默认值并且不再显示此消息”复选框。单击OK。

虚拟机关闭后，您可以更改其设置。为此，请选择要更改的虚拟机的.vmcx文件。在资源管理器中，单击工具栏上的设置。

如果您在Windows虚拟机中的30天试用期即将结束，则可以将虚拟机重置为初始设置，从而重置试用期。要执行此操作，请在“Windows虚拟PC设置”对话框左侧的列表中选择“撤消磁盘”。确保选中“启用撤消磁盘”复选框，然后单击“放弃更改”。

将显示一个警告对话框。若要继续重置虚拟机设置，请单击“继续”。请记住，您在虚拟机中对Windows系统所做的任何更改都将丢失。

您将返回到设置对话框。单击确定将其关闭。

您可以在IE7虚拟机中安装IE8或IE9。桌面上有图标，可以方便地安装任一版本。但是，请记住，安装IE8或IE9将取代IE7。

在IE8虚拟机中，您将在C：\Internet Explorer Versions目录中找到要安装IE9的文件。

一旦您打开虚拟机一次，登录过程就会稍有不同。下次打开虚拟机时将显示以下对话框。

单击IEUser并在编辑框中输入“Password1”。单击OK。

启用虚拟机的集成功能时将显示一个对话框。

虚拟机将打开，您可以运行IE。
