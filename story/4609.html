如何使用Android SDK在PC上截取Android屏幕

如果你的设备使用的是2.x版的Android，比如姜饼或Froyo，就没有内置的截屏方式。不过，你可以通过将安卓手机连接到电脑并使用谷歌的安卓SDK来截图。

如果你使用的是Android 4.0或更高版本，你可以通过同时按下音量降低和电源按钮来截图。如果您的设备已扎根，您还可以从Google Play安装众多屏幕截图应用程序中的一个。

安装Android SDK

在安装Android软件开发工具包(SDK)之前，您需要从Oracle网站下载并安装Java开发工具包(JDK)。下载并安装32位版本的JDK，即使您使用的是64位版本的Windows-Android SDK似乎也需要32位版本。在甲骨文网站上，32位版本被标识为Windows x86版本。

JDK安装到您的系统后，从Google的Android开发者网站下载Android SDK。

接下来，从开始菜单打开Android SDK管理器(只需搜索“SDK管理器”)。如果您为所有用户安装了Android SDK，则必须右键单击SDK管理器快捷方式并选择Run as Administrator。

选中Android SDK Platform-Tools框并单击Install按钮进行安装-Platform Tools包包含我们需要的adb(Android Debug Bridge)实用程序。如果您看到错误，则SDK管理器可能未以管理员权限运行。

配置驱动程序和USB调试

您需要为您的Android设备安装驱动程序。这些通常可以从您的制造商处获得。例如，如果您使用的是三星设备，请安装Samsung Kies以获取适用于您的Android的驱动程序。驱动程序将随软件包自动安装。

您必须在Android设备上启用USB调试，方法是进入其设置屏幕，点击应用程序，点击开发，然后选中USB调试复选框。

安装驱动程序并启用USB调试后，使用随附的USB电缆将Android手机连接到计算机。您的计算机应该会自动为电话配置所需的驱动程序。

截图

要截图，您必须启动Dalvik Debug Monitor应用程序。导航到您安装Android SDK的文件夹-C：\Program Files(X86)\android\android-sdk(如果您为所有用户安装的话)。启动位于此文件夹的Tools子文件夹中的ddms.bat文件。

通过为Dalvik Debug Monitor创建快捷方式，您可以使其在将来更容易启动-例如，您可以将其拖放到Start按钮以快速创建快捷方式。

一旦启动，您应该会在窗口中看到已连接的Android设备。选择设备，单击Device(设备)菜单，然后选择Screen Capture(屏幕捕获)。

使用窗口顶部的刷新按钮拍摄新的屏幕截图。使用保存按钮将屏幕截图保存到图像文件，或使用复制按钮将其复制到剪贴板。您还可以在使用旋转按钮保存或复制屏幕快照之前旋转它。

如果你想在将来截屏，只需将你的Android手机连接到你的电脑上，启动ddms.bat文件(可能是通过快捷方式)，然后使用屏幕截图工具。
