针对Windows 7和8解释的8个备份工具

Windows上的备份可能会令人困惑。无论您使用的是Windows 7还是Windows 8，您都有相当多的集成备份工具可供考虑。Windows8也做了相当多的改变。

您还可以使用第三方备份软件，无论您是要备份到外部驱动器还是要将文件备份到在线存储。我们不会在这里讨论第三方工具--只讨论那些内置到Windows中的工具。

Windows 7上的备份和还原

相关：极客学校：学习Windows 7-备份和恢复

Windows 7具有自己的备份和还原功能，可让您手动或按计划创建备份。您可以在“控制面板”中的“备份和还原”下找到它。

Windows 8的原始版本仍然包含此工具，并将其命名为Windows 7文件恢复。这使得以前的Windows7用户可以从那些旧的Windows7备份中恢复文件，或者继续使用熟悉的备份工具一段时间。Windows 8.1中删除了Windows 7文件恢复。

系统恢复

相关：系统还原在Windows中的工作原理

Windows7和Windows8上的系统还原功能都是一种自动系统备份功能。它按计划或在您执行某些任务(如安装硬件驱动程序)时创建重要系统和程序文件的备份副本。如果系统文件损坏或计算机软件变得不稳定，您可以使用系统还原从系统还原点还原系统和程序文件。

这不是备份您的个人文件的方法。它更像是一种故障排除功能，使用备份将系统恢复到以前的工作状态。

Windows 7上的早期版本

相关：使用Windows 7的早期版本及时回溯并保存您的文件

Windows7的早期版本功能允许您恢复旧版本的文件-或删除的文件。这些文件可以来自使用Windows 7的备份和还原功能创建的备份，但也可以来自系统还原点。当Windows 7创建系统还原点时，它有时会包含您的个人文件。以前的版本允许您从还原点提取这些个人文件。

这仅适用于Windows 7。在Windows 8上，系统还原不会创建个人文件的备份副本。在Windows 8上删除了以前版本的功能。

文件历史记录

相关：如何使用Windows的文件历史记录备份数据

Windows8用文件历史记录替换了Windows7的备份工具，尽管默认情况下该功能未启用。文件历史记录旨在提供一种在外部驱动器或网络位置创建数据文件备份的简单方法。

文件历史记录取代了Windows7的备份和以前版本的功能。Windows系统还原不会在Windows 8上创建个人文件的副本。这意味着，除非您自己启用文件历史记录，否则无法实际恢复旧版本的文件-默认情况下不会启用该功能。

系统映像备份

相关：如何在Windows 7、8或10中创建系统映像备份

Windows还允许您创建系统映像备份。这些是整个操作系统的备份映像，包括系统文件、安装的程序和个人文件。此功能包括在Windows 7和Windows 8中，但在Windows 8.1的预览版中隐藏。在许多用户投诉之后，它被恢复，并且在Windows 8.1的最终版本中仍然可用-单击“文件历史记录”控制面板上的“系统映像备份”。

存储空间镜像

相关：如何使用Windows 10的存储空间镜像和合并驱动器

Windows8的存储空间功能允许您在软件中设置类似RAID的功能。例如，您可以使用存储空间在镜像配置中设置两个大小相同的硬盘。它们将在Windows中显示为单个驱动器。当您写入此虚拟驱动器时，文件将保存到两个实体驱动器。如果一个驱动器出现故障，您的文件在另一个驱动器上仍可用。

这不是一个很好的长期备份解决方案，但它是一种确保在单个驱动器出现故障时不会丢失重要文件的方法。

Microsoft帐户设置备份

相关：Windows 10中需要Microsoft帐户的所有功能

Windows 8和8.1允许您备份各种系统设置，包括个性化设置、桌面设置和输入设置。如果你使用Microsoft帐户登录，则会自动启用OneDrive设置备份。可以在电脑设置应用中的OneDrive>同步设置下控制此功能。

此功能仅备份几个设置。这实际上更像是一种在设备之间同步设置的方式。

OneDrive云存储

相关：Windows 8.1如何集成SkyDrive Everywhere

自从Windows8发布以来，微软就没有太多谈论文件历史记录。这是因为他们希望人们改用OneDrive。

OneDrive(以前称为SkyDrive)是在Windows 8.1中添加到Windows桌面的。在此保存你的文件，它们将与你的Microsoft帐户捆绑在一起联机存储。然后，您可以登录任何其他计算机、智能手机、平板电脑，甚至通过网络登录并访问您的文件。微软希望普通PC用户使用OneDrive“备份”他们的文件，这样他们就可以在任何设备上使用。

您不必担心所有这些功能。只要选择一种备份策略，以确保您的文件在您的计算机硬盘出现故障时是安全的。无论是集成备份工具还是第三方备份应用程序，请确保备份您的文件。
