如何管理Windows 7备份和还原使用的硬盘空间

与以前版本的Windows相比，Windows 7提供了更好的计算机备份实用程序，但有时可能会占用太多磁盘空间。下面我们来看一下如何通过删除较旧的备份来收回一些空间。

管理备份大小

首先单击开始，在搜索框中输入Backup and Restore，然后按Enter。

然后在“Backup”部分下，单击“Manage Space”。

在“Manage Windows Backup Disk Space”屏幕中，单击“Data file Backup”部分下的“View Backups”。

将显示不同时间段的备份列表。您可以删除较旧的文件，只保留最新的备份。如果您存储了多个备份周期，这将释放大量空间。

您将收到一条消息，确认您要删除该备份。如果删除最新的备份，则可以选择将其删除并再次运行备份，以便所有内容都是最新的。

既然我们已经能够释放一些空间，那么让我们来看看如何只保存最近存储的系统映像。返回到我们之前所在的“管理Windows备份磁盘空间”窗口，这次单击“系统映像”部分下的“更改设置”。

现在，选择“仅保留最新的系统映像并最小化备份使用的空间”，然后单击“确定”。

结束 / 结尾 / 结论 / 推论

备份您的系统非常重要，特别是在您有大量敏感数据和重要文件的情况下。如果允许Windows管理空间，它将自动保存备份周期，但不会占用备份磁盘上超过30%的空间。当它达到30%的范围时，它将开始删除较旧的系统映像。如果您使用Windows Backup and Restore并需要备份磁盘上的更多空间，请使用这些提示。首先通过手动删除较旧的备份来创建更多空间，然后只保存最新的备份。
