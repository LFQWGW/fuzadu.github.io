Windows RT桌面与Windows 8有何不同

Windows RT和Windows 8不是一回事。虽然Windows RT的桌面看起来像Windows8，但Windows RT的桌面非常有限。这一区别不仅对极客很重要，对所有Windows用户都很重要。

我们之前已经解释过Windows RT和Windows 8之间的区别。与Windows RT不同，Windows8包括一个功能齐全的桌面(尽管默认情况下没有开始菜单)。

桌面上没有非Microsoft应用程序

Windows RT上最不和谐的变化是，虽然桌面礼物看起来和Windows8上的桌面一模一样，但你不能在上面安装你自己的软件。您只能在桌面上运行预安装的Microsoft应用程序。

Windows RT在ARM体系结构上运行，而Windows 8和以前版本的Windows使用x86体系结构。Windows桌面应用程序开发人员必须修改他们的应用程序才能在Windows RT上工作。然而，微软不会允许他们这样做。微软正在利用架构变化迫使第三方应用程序开发人员为新的现代环境而不是传统的桌面环境编写应用程序。

这意味着，如果您要在Windows RT系统上使用桌面，您将使用Microsoft Office和Internet Explorer。OpenOffice和Mozilla Firefox不会是选项。如果要在桌面上编辑图片和文本文件，可以使用MS Paint和记事本，但不能安装Paint.NET或Notepad++。相反，微软希望你从Windows应用商店安装第三方应用。这些应用程序在以前称为Metro的现代界面中运行。

预装的Windows桌面应用程序

Windows RT桌面包括Windows附带的各种常见应用程序。一些最重要的应用程序是用于管理文件的Internet Explorer 10和文件资源管理器(以前称为Windows资源管理器)的桌面版本。

大多数其他预装程序(包括记事本和MS Paint)也可用。但是，Windows Media Player不适用于Windows RT-Microsoft希望您在现代界面中使用新的音乐和视频应用程序。

MicrosoftOffice

值得注意的是，Windows RT包括四个Microsoft Office 2013应用程序的预装版本：Word、Excel、PowerPoint和OneNote。

虽然有适用于OneNote的现代版应用程序，但还没有适用于OneNote的现代版Word、Excel或PowerPoint应用程序。如果你购买了像Surface RT这样的Windows平板电脑，你将不得不使用触控不友好的桌面来使用Word、Excel和PowerPoint应用程序。您可以在Office 2013应用程序中启用触摸模式，但桌面本身并不是为触摸输入而设计的。

虽然Windows RT将重点放在新的现代界面上，但微软最重要的生产力应用程序仍然仅限于桌面。目前尚不清楚为什么不包含Word、Excel或PowerPoint的现代版本。一种理论是，对于这些类型的生产力应用，桌面可能比新的现代界面更有用。另一种理论是，微软未能让Office团队加入到新界面的行列中来。也许两者兼而有之。

虽然微软将Windows RT设备定位为平板电脑，让你可以用来完成真正的工作，但Office 2013的内置版本是基于Office Home&Students的。如果你想在Windows RT上使用Office应用程序进行商业(或“创收”)，从技术上讲，你必须向微软支付商业使用权费用。这些Office应用程序也不支持宏或外接程序。

配置您的平板电脑

Windows桌面还包含传统的Windows控制面板，并提供其提供的所有常见设置。虽然Windows8和Windows RT提供了现代风格的PC设置应用程序，但新界面并不包含标准Windows控制面板中的所有选项。

如果要安装语言包，更改Internet Explorer 10的默认搜索引擎，并修改其他不太常用的设置，您需要离开触控友好的现代环境，使用桌面上的控制面板。在与之竞争的平板电脑操作系统上的触控界面中提供了这些设置。

为什么桌面会在那里？

这是无可避免的：桌面在Windows RT上感觉格格不入。全力以赴使用现代应用程序，放弃Windows RT上的桌面，可能会带来更有凝聚力的体验。

桌面仍然有用，主要有两个原因：更改尚未从“现代电脑设置”应用程序更改的设置；使用尚未由Office团队移植到新的“现代”界面的Microsoft Office应用程序。

在Windows RT的未来版本中，预计桌面将完全消失，因为Office应用程序是为现代环境编写的，并且PC设置应用程序中添加了更多设置。微软不允许第三方开发者为Windows RT的桌面开发应用程序，这表明他们不应该在未来的Windows RT版本中依赖桌面。

在购买新的Windows平板电脑或笔记本电脑时，请务必考虑桌面应用程序对您的重要性。Windows8(X86)和Windows RT(ARM)系统在商店里都有类似的桌面，但当你回到家并尝试安装自己的应用程序时，你会注意到不同之处。
