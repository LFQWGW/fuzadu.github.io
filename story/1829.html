使用Ubuntu Live CD安全擦除PC硬盘

您是否曾将PC赠送或出售给其他人，但真的想先彻底擦除硬盘？今天，我们将向您展示如何使用Ubuntu Live CD从您的PC上获取您的个人信息。

当你在Windows、Ubuntu或任何其他操作系统中删除文件时，它实际上并没有破坏存储在硬盘上的数据，它只是将该数据标记为“已删除”。如果您稍后覆盖它，那么数据通常是无法恢复的，但是如果操作系统没有覆盖它，那么您的数据仍然存储在您的硬盘上，任何拥有正确软件的人都可以恢复。

通过安全地删除文件或整个硬盘，您的数据将永远消失。

注：现代硬盘非常复杂，以恢复数据为生的专家也是如此。不能保证本文中介绍的方法会使您的数据完全不可恢复；但是，对于大多数恢复方法和公众随时可用的所有方法，它们都会使您的数据无法恢复。

擦除整个硬盘

如果您要处理一个旧硬盘，或者将其送给其他人，那么您可能想要擦除整个硬盘。可以在硬盘上调用分解，但在现代文件系统上，分解过程可能是可逆的。我们将使用程序擦除来安全地删除硬盘上的所有数据。

与shred不同的是，默认情况下，Ubuntu中不包含擦除，所以我们必须安装它。打开Synaptic Package Manager，方法是单击屏幕左上角的System，然后展开Administration文件夹并单击Synaptic Package Manager。

擦除是宇宙存储库的一部分，默认情况下未启用。我们将通过单击Synaptic package Manager窗口中的Settings>Repositories来启用它。

选中“社区维护的开放源码软件(宇宙)”旁边的复选框。单击关闭。

您需要重新加载Synaptic的软件包列表。单击Synaptic Package Manager主窗口中的重新加载按钮。

重新加载软件包列表后，搜索字段上方的文本将更改为“正在重建搜索索引”。

单击“擦除”标签左侧的复选框，然后选择“标记为安装”。

单击Apply按钮开始安装过程。在弹出的摘要窗口中单击应用按钮。

安装完成后，单击关闭按钮并关闭Synaptic Package Manager窗口。

结束 / 结尾 / 结论 / 推论

如果您的硬盘上有敏感信息(很有可能是这样)，那么在赠送或处置硬盘之前安全地删除敏感文件是个好主意。最安全的删除数据的方法是用锤子敲打几下，但从Ubuntu Live CD上粉碎和擦除是一个很好的选择！

这项技术并不是处理旧电脑中数据的唯一方法，但它只是展示了修复Windows电脑时Linux Live CD的真正用途-你可以重置密码、清除病毒感染、恢复删除的文件，甚至可以从死掉的Windows电脑中恢复文件。如果你不想随身携带光盘，你可以创建一个可引导的Ubuntu闪存驱动器。
