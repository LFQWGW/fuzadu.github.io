通过网络或Internet远程打印的4种简单方法

远程打印不一定很难，无论您是想打印到大厅另一头的打印机，还是要打印到地球另一端的打印机。我们将介绍一些无需直接连接到打印机即可打印的简单方法。

我们在这里将重点放在最简单的选项上。我们不会讨论如何设置Internet打印协议(IPP)或JetDirect并允许其通过防火墙或复杂的Windows网络配置，因为这些选项最适合IT专业人员。

获取无线打印机

即使您仍在打印，也不需要将单独的打印机连接到家里的每台计算机上。许多新打印机都是网络打印机，可以通过Wi-Fi连接到您的网络。连接后，您可以在每台计算机上安装相应的驱动程序软件，所有计算机都可以通过网络打印到该打印机。

与与Windows共享本地打印机不同，您不必让主计算机保持打开状态-只要打印机处于打开状态，您就可以直接打印到它。

这些打印机只允许您通过本地网络打印到它们，所以如果您想要通过Internet打印到它们，您需要一些其他技巧。

在您的本地网络上共享打印机

Windows使您可以轻松地在本地网络上的计算机之间共享打印机。如果您有通过USB连接到计算机的本地打印机，这是理想的选择。设置打印机共享后，打印机的功能几乎就像网络打印机一样。只要打印机连接到的计算机通电，网络上的任何其他授权计算机都可以打印到该计算机。

在Windows 7或Windows 8上执行此操作的最简单方法是使用家庭组功能。只需设置一个家庭组并选中“打印机”选项即可共享您连接的打印机。假设共享打印机的计算机处于联机状态，将您的其他计算机加入家庭组，他们将看到网络打印机出现在他们的可用打印机列表中。

与标准网络打印机一样，这只能在本地网络上运行。您可以在不在同一家庭组上的计算机之间共享打印机，但只使用家庭组会更容易。

使用Google Cloud Print访问远程打印机

Google Cloud Print是谷歌的远程打印解决方案。许多新打印机都内置了对Google Cloud Print的支持。如果打印机不支持Cloud Print，您可以通过Google Chrome中的Google Cloud Print设置Google Cloud Print使其可用。

一旦打印机配置为使用Google Cloud Print，它就会与您的Google帐户相关联。然后，您可以使用您的Google帐户凭据远程访问打印机。您还可以与另一个Google帐户共享您的一台打印机，这样您就可以允许其他人远程打印到您的计算机，就像您通过Google Drive与他们共享文件一样。

直到最近，Google Cloud Print还有点新奇。Google Chrome包含对Cloud Print的支持，您可以在iOS和Android上使用Cloud Print应用程序远程打印到Cloud Print。然而，谷歌最近推出了一项针对Windows桌面的Google Cloud Printer服务。安装它后，Google Cloud Print将出现在标准打印对话框中，因此您可以从Microsoft Office或任何其他桌面应用程序远程打印到Cloud Printers。

对于互联网打印，Google Cloud Print为普通用户提供了最完美的体验和最简单的设置体验。

使用VPN访问远程网络上的打印机

如果您想在离开本地网络时访问标准网络打印机或通过Windows网络共享的打印机，您可以使用虚拟专用网络(VPN)。连接到VPN，您的计算机将创建到远程网络上的VPN服务器的安全通道。您的所有通信都将通过此隧道发送，因此您的计算机将像连接到远程网络一样运行。这意味着本地共享打印机以及其他网络资源(如Windows文件共享)都可以访问。

一旦您的计算机连接到VPN，打印机就可以使用了，您可以打印到它，就像您在同一个本地网络上一样。许多企业网络都设置了VPN，以便其员工可以远程连接到企业网络，因此您可能已经能够使用现有的VPN连接进行此操作。

设置自己的VPN比使用Google Cloud Print更复杂，但这是可以做到的。Windows包括对设置VPN服务器的隐藏支持。托管您自己的VPN服务器对于安全性来说并不理想--如果您不想太担心安全性，那么只使用Google Cloud Print会更容易一些。

远程打印有多种其他不同的方式。例如，某些网络打印机可能能够接受电子邮件地址处的文档，并自动打印到达该地址的所有文档。有些可能与蓝牙或苹果的AirPrint配合使用，以无线方式接受打印作业。
