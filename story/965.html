与Windows 7兼容的防病毒软件列表

在Windows7发布后的几个小时内，我的收件箱就开始挤满了询问他们应该在Windows7上安装什么杀毒软件的读者。由于这似乎是一个如此重要的话题，我决定列出一个有效的软件包清单。

如果这是您第一次来这里，您应该查看我们列出的175篇Windows7调整、提示和使用方法文章

我们的建议

如果你想要一款运行良好且完全免费的软件，你应该试试免费的Microsoft Security Essentials防病毒解决方案，它不会降低你电脑的运行速度。

极客周：Microsoft Security Essentials是优秀版本。 Microsoft Security Essentials是免费的防病毒实用程序

注意：一般来说，在Windows Vista上运行的软件应该可以在Windows 7上正常运行，但安装低级系统组件(如防病毒、防火墙或类似软件)的软件除外。这些软件包通常设计为仅在特定版本的Windows上运行。

Windows防病毒检测

当您第一次安装Windows7时，您会收到一条弹出气球消息，告诉您需要在线查找杀毒程序…。注意气球上的扳手，它可以让你很容易地关闭气球。

一旦您安装了兼容的软件包，该消息将消失，您将在操作中心看到一条消息，告诉您“防病毒”已打开：

这些软件包还应该与Windows7更新机制挂钩，并在病毒定义过期时通知您。

现在，让我们来看看…的名单。我拍了一张Action Center屏幕的截图，证明Windows7可以正确安装和识别每个软件包，但我也对每个软件包进行了几次快速测试。

Avira AntiVir个人版

Avira是一个流行的免费防病毒解决方案，似乎在我们的论坛上很受欢迎，许多常客都支持它。这个软件工作正常，我在快速测试过程中没有发现任何问题。
