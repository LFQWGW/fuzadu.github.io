安排Windows 7 Media Center优化以获得更好的性能

Windows 7 Media Center包含优化功能，可优化媒体以与WMC配合使用。如果您有一个很大的库，则此过程可能需要很长时间，下面我们将介绍如何在您睡眠时安排该过程。

Windows 7 Media Center优化

优化是一项包含的功能，它将组织您的媒体文件，以便更流畅地使用Media Center。它执行优化任务，如重新索引您的媒体，以便更快地检索您的媒体。它还会重新启动Media Center服务以清除错误。

要访问优化功能，请启动WMC，然后从主菜单导航到Tasks(任务)，然后选择Settings(设置)。

从那里选择常规设置。

现在，在常规设置下，选择优化。

选中“执行优化”旁边的复选框，选择执行优化的时间，然后保存您的设置。

注意：您希望确保将其安排在不使用它的时间，因为在优化过程完成时，您将无法使用WMC或任何扩展器。

在此期间，您可能还希望安排Media Center的更新。

如果您是Windows 7 Media Center的发烧友并且经常使用它，计划优化将帮助它更快地访问您的媒体，提高稳定性，并提高整体性能。

有关详细信息，请查看Windows 7 Media Center上的入门极客指南
