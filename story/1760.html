使用Windows Media Player 12在Internet上流式播放音乐和视频

Windows 7附带的Windows Media Player 12中的一项新功能是能够通过Web将媒体流式传输到其他Windows 7计算机。“今天我们将看看如何设置它，以及您需要开始做些什么。”

注意：您需要在要使用的每台计算机上执行此过程。

您需要的是什么

两台运行Windows 7家庭高级版、专业版或旗舰版的计算机。您将从中流式传输媒体的主机或家庭计算机不能位于公用网络或域的一部分。 Windows Live ID。 在您的家庭路由器上启用UPnP或端口转发。 添加到Windows Media Player媒体库的媒体文件

Windows Live ID

联机注册Windows Live ID(如果您还没有)。有关Windows Live的链接，请参阅下面的链接。

配置Windows 7计算机

打开Windows Media Player并转到存储库部分。单击流，然后单击“允许家庭媒体的互联网访问”。

Internet Home Media Access弹出窗口将提示您将您的Windows Live ID链接到用户帐户。单击“链接在线ID”。

如果您尚未安装Windows Live ID登录助手，系统会将您带到Microsoft网站并提示您下载该助手。

完成Windows Live下载助手安装后，您将在“Link Online ID”窗口中看到Windows Live ID Online Provider。点击“链接在线ID”。

接下来，系统将提示您输入Windows Live ID和密码。输入您的Windows Live ID和密码，然后单击“登录”。

弹出窗口将通知您已成功允许Internet访问家庭媒体。

现在，您必须在第二台Windows 7计算机上重复完全相同的配置。在第二台计算机上完成相同配置后，您可能还需要配置家庭路由器以进行端口转发。

如果您的路由器支持UPnP，您可能不需要手动转发路由器上的任何端口。因此，这将是测试您的连接的好时机。去附近的热点，或者可能是邻居家，测试一下你是否可以播放你的媒体。如果不是，您将需要手动转发端口。您始终可以选择转发端口，以防万一。

注：我们在支持UPnP的Linksys WRT54GL路由器上进行了测试，发现仍然需要手动转发端口。

查找路由器上要转发的端口

打开Windows Media Player，并确保您处于“库”视图中。单击顶部菜单上的“流”，然后选择“允许家庭媒体的互联网访问”。

在“Internet Home Media Access”(Internet家庭媒体访问)窗口中，单击“Diagnose Connections”(诊断连接)。

将弹出“Internet Streaming Diagnostic Tool”(Internet流诊断工具)。单击底部附近的“端口转发信息”。

在“Port Forwarding Information”(端口转发信息)窗口中，您可以找到需要在路由器上转发的内部和外部端口号。内部端口号应始终为10245。外线号码会因您的计算机不同而有所不同。Microsoft还建议使用转发端口443。

配置路由器

接下来，您需要在家庭路由器上配置端口转发。我们将向您展示Linksys WRT54GL路由器的步骤，但是，不同路由器的端口转发步骤会有所不同。

在Linksys Configuration(Linksys配置)页面上，单击顶部的Administration(管理)选项卡，单击“Applications&Gaming(应用程序和游戏)”选项卡，然后单击其下方的“Port Range Forward(端口范围转发)”选项卡。

在“应用程序”下，键入一个名称。它可以是您选择的任何名称。在“开始”和“结束”框中，键入端口号。在IP地址列中输入家庭计算机的IP地址。单击“启用”下的复选框。对内部和外部端口号以及端口443执行此操作。完成后，单击“保存设置”按钮。

注意：强烈建议您使用静态IP地址配置您的家用计算机

当您准备好通过Internet播放您的媒体时，打开Windows Media Player并查找“其他媒体库”下面列出的您的主机和用户名。单击它，展开列表以查看您的媒体库。选择要播放的库和文件。

现在您可以通过Internet享受您的流媒体了。
