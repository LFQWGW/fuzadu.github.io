卸载、禁用或删除Windows 7 Media Center

虽然Windows 7 Media Center比以前的Windows版本有了很大改进，但您可能会出于不同的原因想要禁用它。在这里，我们来看看几种去除它的方法。

您可能出于多种原因想要禁用Windows 7 Media Center。也许你拥有一家企业，不想让它在机器上运行。或者，也许您根本不使用它，只是不想让它出现在您身边。

使用程序和功能关闭WMC

在所有版本的Windows7上清除它的最简单方法可能是打开控制面板，然后选择程序和功能。此方法类似于在Windows 7中禁用Internet Explorer 8。

在左侧面板上，单击打开或关闭Windows功能。

向下滚动到Media Feature，然后展开该文件夹。然后取消选中Windows Media Center…

您将收到一条验证消息，确认您要禁用它，请单击“是”。

则Windows Media Center旁边的框将为空…。单击确定。

正在禁用wmc，请稍候…

要完成该过程，需要重新启动。

从重启中恢复后，WMC图标将消失，并且将没有任何方法启动它。

重新启用WMC

如果您想重新启用它，只需返回并重新检查即可。

同样，您需要在配置时等待，但在配置完成后，不需要重新启动。

使用组策略禁用Media Center

注意：此过程使用Windows 7家庭版中不可用的组策略编辑器。

单击开始菜单，在搜索框中键入gpedit.msc，然后按Enter键。

现在导航到用户配置\管理模板\Windows组件\Windows Media Center。

双击Do Not Allow Windows Media Center to Run(不允许Windows Media Center运行)。然后选择Enabled(启用)旁边的单选按钮，单击OK(确定)并关闭组策略编辑器。

现在，如果用户尝试启动WMC，他们将收到以下消息。

结束 / 结尾 / 结论 / 推论

如果你不是Windows Media Center的粉丝，或者出于任何原因想要禁用它，这个过程很简单，有几种方法可以做到这一点。WMC不包含在Windows 7的Starter或Home Basic版本中。如果您是Windows 7 Media Center的新手，您可能需要查看我们的入门指南和设置实时电视。
