以下是适用于Windows 7用户的Windows 10的不同之处

与Windows8不同，Windows10给人的感觉实际上是为一台有键盘和鼠标的PC设计的。Windows7用户使用Windows10将更加自在，但仍有一些重大变化。

如果你是Windows7的用户，你可能会惊讶地发现，升级后发生了这么大的变化。谢天谢地，没有奇怪的热点需要学习。

Microsoft帐户集成

相关：Windows 10中需要Microsoft帐户的所有功能

安装Windows10时，系统首先会询问您是否要使用Microsoft帐户登录Windows系统。这类似于用苹果账户登录苹果Mac或iPhone，或者用谷歌账户登录Chromebook或Android设备。

如果你使用Microsoft帐户登录，许多桌面设置(包括你的墙纸)将在你的电脑之间同步。你将自动登录到Microsoft服务，如集成到Windows中的OneDrive客户端。使用一些新功能(如Windows应用商店)时，必须拥有Microsoft帐户。

如果你不想使用Microsoft帐户，也没问题-有一个小链接可以让你设置一个传统的本地Windows帐户。如果您愿意，您可以稍后轻松地将其转换为Microsoft帐户。

新的“开始”菜单

开始菜单看起来与在Windows 7上有很大不同，Windows 8开始屏幕上的实时地图磁贴在这里又回来了。但是，不要担心-如果你不喜欢，你可以移除所有的活瓷砖。只需右键单击它们并将其删除。“开始”菜单看起来有点不同，但它拥有所有你想要的常见功能-所有已安装应用程序的列表，以及关闭或重新启动电脑的电源选项。将鼠标移动到“开始”菜单的任意边缘，即可调整其大小。

通用应用程序和Windows应用商店

Windows 10自带的许多应用程序都是“通用应用程序”，是Windows 8的“Metro应用程序”或“Store应用程序”的继任者。与Windows8不同的是，这些应用程序实际上是在桌面的窗口中运行的，所以你可能会对使用它们感兴趣。

要获得更多此类应用程序，您需要打开Windows附带的应用商店应用程序，然后从Windows应用商店下载。没有办法通过从互联网下载这些类型的应用程序来“侧面加载”它们，尽管你可以自由地完全避免它们，并从网络上安装传统的Windows桌面应用程序。你还可以混合搭配应用商店中的传统Windows桌面应用程序和新应用程序。它们都将在您桌面上的窗口中运行。

设置应用程序与控制面板

相关：您需要了解的有关Windows 8和10中“重置这台电脑”的所有信息

“开始”菜单中的“设置”选项会将你直接带到新的“设置”应用，该应用由Windows 8上的“电脑设置”应用演变而来。这是一种更加用户友好的电脑配置方式。

但是，它仍然不包含所有设置。旧的Windows控制面板仍然包括在内。某些较旧的设置可能仅在控制面板中可用，而某些较新的设置可能仅在设置应用程序中可用。要快速访问控制面板和其他高级选项，您可以右键单击开始按钮或按Windows键+X。此菜单是Windows 8的有用保留项。

“刷新”和“重置”选项也实现了从Windows 8到Windows 10的飞跃。这些选项可以让您的计算机快速恢复到类似新的状态，而无需实际重新安装Windows。

您将无法在Windows 10家庭版系统上禁用Windows自动更新功能。您需要Windows 10专业版才能推迟更新。

任务栏上的Cortana和任务视图

Windows任务栏已稍有更改。在Windows 8中，微软从任务栏中删除了开始按钮，你在这里只看到程序的图标。在Windows10中，“开始”按钮不仅仅是回来了--还有一个“搜索网络和Windows”字段，可以启动微软的Cortana助手，还有一个“任务视图”按钮，可以提供所有打开的窗口和虚拟桌面功能的概览。

默认情况下，这两项功能都处于启用状态。如果要禁用它们，只需右键单击任务栏并选择隐藏“搜索”和“任务视图”选项即可。

EDGE取代Internet Explorer

Internet Explorer不再是默认浏览器，尽管它仍然可以供仍需要访问其较旧渲染引擎的企业使用。取而代之的是名为Edge的现代浏览器。微软的Edge浏览器应该更符合标准，性能更好。它也不再支持ActiveX控件，因此所有那些旧的Internet Explorer工具栏和浏览器插件将不再起作用。如果您一直在使用Internet Explorer，这是您将使用的浏览器。如果你使用的是Chrome或Firefox，你可以安装它，然后继续正常浏览。

台式机和安全性改进

Windows 8的许多其他桌面改进仍然存在，但如果你一直在使用Windows 7，你就不会看到这些改进。任务管理器进行了升级，因此更容易看到系统资源的使用情况，甚至可以在没有第三方软件的情况下管理启动程序。Windows资源管理器已重命名为文件资源管理器，现在有了功能区-即使您不喜欢该功能区，文件资源管理器也提供了许多有用的功能。例如，文件复制和移动对话框窗口有了很大改进，Windows可以在没有第三方软件的情况下挂载ISO光盘映像文件。

Windows 8还有许多安全方面的改进。Windows 10默认情况下包括Windows Defender-Windows Defender只是Microsoft Security Essentials的更名版本，因此所有Windows系统都有基线级别的防病毒保护。SmartScreen是一个信誉系统，它试图阻止有害的和未知的文件下载损害您的计算机。

相关：Windows 8中针对台式机用户的10项令人惊叹的改进

这些远不是Windows 10中唯一的重大改进，例如，你会发现通知中心，并在系统托盘中重新设计了电源、网络和声音图标。Windows 10包括用于录制和播放流媒体PC游戏的游戏DVR功能。微软已经做了许多低级别的调整，使Windows使用更少的磁盘空间，启动更快，并更好地保护免受攻击。

尽管有这些变化，Windows10比Windows8更容易掌握。它基于熟悉的桌面界面，包括开始菜单和桌面窗口。Windows10确实有“平板电脑模式”，但你必须手动启用--或者在使用平板电脑硬件时自动启用。在典型的个人电脑上，你不会被迫进入平板电脑模式。
