屏幕截图教程：Windows Live软件包测试版

微软发布了下一版本Windows Live软件包的测试版，其中包括适用于Vista和Windows 7的Movie Maker、Mail、Live Sync等工具。在这里，我们将看看测试版套件，看看你能期待什么。

Windows Live软件包是一套适用于Windows的优秀免费应用程序捆绑包。这些程序中的大多数，如Mail和Movie Maker，都作为功能包含在以前版本的Windows中。现在，这些应用程序已经从Windows中分离出来，理想情况下，这样它们就可以比Windows本身更频繁地更新。最新的版本还没有完全准备好进入黄金时间，但是如果你想开始使用它们，你可以今天下载新的测试版，然后自己试用新的工具。

注意：新的Windows Live软件包测试版只能在Vista和Windows 7上运行。*如果您使用的是XP，您仍然可以使用较旧版本的Live软件包。

安装和设置

下载并安装新的Live Essentials Beta版(链接如下)。选择您的语言，然后单击Download Now(立即下载)按钮。

如果你使用的是Live Mesh测试版，你会看到下面的屏幕，告诉你Live Mesh将被卸载，而Live Sync正在取而代之。

如果您以前安装了Mesh，您会注意到在安装过程中卸载Mesh时会出现提示。

现在选择您要安装的应用程序…。就像以前的版本一样，你不需要安装所有的东西，只需要安装你想要的程序。您将看到有一些新的程序，一些现有的程序将会更新。如果你不喜欢额外的工具栏，你可能想取消选择Bing Bar。

为了测试起见，我们继续安装了所有设备。在我们配备4 GB内存的Windows7Core2Duo计算机上，该过程花费了10-15分钟的…时间。您的里程数将根据网速和安装的应用程序数量而有所不同。

根据您选择安装的内容，可能需要重新启动。

重新启动计算机后，您将在Windows Live下的“开始”菜单中找到新的Live Beta应用程序。

Windows Live Mail测试版

新的Live Mail测试版实际上是一个相当不错的电子邮件管理程序，包含更多新功能，并且具有Office 2007中引入的熟悉的功能区界面。它允许您管理多个电子邮件帐户，如Hotmail、Gmail和Yahoo(前提是您订阅了Mail Plus)。

Windows Live Messenger测试版

新的Live Messenger Beta中一个有趣的功能是能够使用你的社交账户，如Myspace和Facebook。你不一定要连接社交服务，但如果你想，选择就在那里。

Messenger Companion(Messenger Companion)

这个新工具允许您在Internet Explorer中浏览网页时共享网页和讨论信息。

Windows Live影音制作测试版

Live Movie Maker的新测试版增加了更多的编辑功能、新的视觉效果以及将它们直接发布到Web上的方式。或者，你可以在Vista和Windows7家庭高级版或更高版本中刻录电影的DVD。

Windows Live Writer Beta版

Windows Live Writer也进行了更改，包括功能区界面，并且有更多工具可以帮助您轻松地将新文章发布到您的博客。有关WLW当前稳定版本的更多信息，请查看我们关于帮助将内容发布到您的WordPress博客的工具的文章。

Windows Live Sync测试版

Windows Live Sync旨在取代Live Mesh，后者从未离开测试阶段。它允许您在线将文件和文件夹同步到其他计算机，甚至可以在Windows和Mac之间同步(Mac版Live Sync目前仅支持英语)。

这是Windows Live软件包测试版中更令人兴奋的功能之一。当我们进行测试时，我们肯定会报道更多关于这方面的细节。

Windows Live照片库试用版

Live Photo Gallery也进行了全面检查，包括功能区界面和更多用于管理照片的选项。它包括自动调整、快速搜索和在Facebook、Flickr或YouTube等网站上共享照片的功能。

Windows Live家庭安全设置测试版

Live家庭安全设置作为新功能，有助于在浏览Internet时保护您的孩子。虽然它不是完美的，但它确实为你提供了一个额外的工具，帮助你的孩子在网上保护自己的孩子免受可疑内容的影响。

Outlook连接器

Live Essentials现在包括Outlook Connector，它允许您管理Outlook 2003、2007和2010中的Windows Live电子邮件、日历和联系人。以前它是单独下载的，不包含在套件中。您拥有的功能数量将取决于您运行的Outlook版本。您可以从此处找到有关Outlook Connector的更多信息。
