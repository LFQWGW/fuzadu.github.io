如何通过网络执行Time Machine备份

苹果的Time Machine通常备份到外置驱动器或无线备份到时间胶囊。但是，如果你有一台备用的Mac，你可以把它变成一台时光机服务器。你所有的其他Mac电脑都可以通过网络备份到它，就像它是一个时间胶囊一样。

这需要苹果的OS X Server软件。与微软的Windows Server不同，OS X Server是一款售价20美元的应用程序，你可以在任何Mac上安装。从技术上讲，您可以在没有OSX服务器的情况下做到这一点，但您不应该这样做。

支持的方法：OS X Server

如果你想这样做，可以花20美元从Mac App Store购买苹果的OS X Server。在OS X Server的众多功能中，它可以配置Mac作为时光机服务器。尽管名为OSX Server，但它并不是一个不同的操作系统--它只是一个可以安装在现有MacOSX系统之上的应用程序。它提供了图形界面和底层服务器软件，您可以对其进行配置。

如果您有一台闲置的Mac，或者，如果您有一台Mac台式机，您想要将其用作几台MacBook的集中式服务器，那么这是理想的选择。

在苹果电脑上安装OS X Server，并打开新安装的“Server”应用程序。在左侧的Services下选择“Time Machine”，并将右上角的滑块设置为“On”以启用Time Machine服务器。

系统将要求您输入要在Mac上存储备份的位置。您还可以将单个备份限制在一定的空间内，以防止一台Mac占用所有存储空间。

就这样了-如果你想的话，你就完蛋了。您还可以使用窗口底部的框添加其他备份目标，如果您有多个驱动器要提供作为备份位置，这将非常有用。例如，您可以选择多个插入Mac的外部驱动器，或者选择几个不同内部驱动器上的备份位置。

您还可以将对Time Machine服务器的访问权限限制为某些用户。默认情况下，本地网络上的任何Mac都可以备份到它。单击权限右侧的“编辑”按钮，您可以限制对特定用户帐户的访问。

在进行了几次备份之后，您可以单击此控制面板顶部的“Backups”选项卡，查看Time Machine服务器上的备份及其占用的空间大小。您还可以从此处删除备份，以管理服务器上的可用空间。

要将另一台Mac实际备份到您的Time Machine服务器，请在与您的服务器位于同一本地网络的另一台Mac上打开Time Machine界面。您将看到它在磁盘列表中显示为备份位置，就像时间胶囊设备所做的那样。

在要通过网络备份到Time Machine服务器的每台Mac上重复此过程。

像往常一样，苹果电脑只有在连接到插座并充电时才会备份到网络位置。如果您希望它使用电池电源充电，请单击Time Machine界面中的Options(选项)按钮，并激活“Back Up When With Batch Power”(使用电池电源时备份)复选框。

请记住，只有当您与Time Machine服务器位于相同的本地网络中时，这才会起作用。当您离开该网络时，您的Mac将不会进行备份--除了在其内部存储上创建本地备份之外。

不支持的方法：不要这样做！

警告：不要这样做！任何问题都可能导致您的备份损坏。例如，如果在Mac备份时网络连接断开，则Time Machine备份可能会损坏。

真的，这不值得--花20美元购买OSX服务器，如果你想这样做，可以使用官方支持的Time Machine服务器。这总比以损坏的备份告终要好。如果你不想花费20美元，只需用Time Machine将数据备份到连接的外部驱动器上，就可以跳过网络备份。

当然，我们是挖掘隐藏的安全选项和一起破解东西的铁杆粉丝。但是备份是极其重要的。不值得冒损坏备份文件的风险-即使它看起来在一段时间内运行良好，但您的备份最终可能会被损坏的风险总是存在的。OS X Server的服务器软件价格便宜，是解决这一问题的理想解决方案。

当然，你随时都可以买一个苹果时间胶囊。您可以通过网络备份到其内置存储，而不需要额外的Mac来充当服务器。
