如何在OS X中更改屏幕截图文件类型

在OS X上截图很容易，你甚至可以更改这些截图的目的地位置。但是你知道你也可以改变默认的截图格式吗？

默认情况下，当你在OSX上截屏时，它会保存为PNG格式。这没有什么错，但也许您更喜欢或需要使用JPEG格式的屏幕截图。对于单个屏幕截图，您可以在预览中打开图像，然后将其导出为该格式。

这行得通，但是非常不方便。如果您需要将屏幕截图一致地保存为JPEG格式，那么最好简单地更改OSX截图的方式。

要做到这一点，您所需要做的就是启动终端并执行一些命令行黑客操作。如果您不知道或忘记了，可以在应用程序>实用程序中找到终端。

键入以下命令以更改屏幕截图格式：

defaults write com.apple.screencapture type jpg

然后使用以下命令终止系统服务器，以使更改生效：

killall SystemUIServer

默认情况下，OSX不会显示文件格式扩展名，但如果你想确保你的更改被接受，那就截屏检查它的文件信息吧(Command+I)。

您也不局限于JPEG。您也可以选择使用其他文件格式，包括PNG、PDF、GIF和TIFF。例如，如果您想要自动将所有屏幕截图另存为PDF文件，只需将第一个终端命令修改为“PDF”即可：

defaults write com.apple.screencapture type pdf

如果您希望返回到原始文件格式(PNG)，则只需遵循此过程并在命令末尾放置“PNG”即可。

defaults write com.apple.screencapture type png

但是，不要忘记运行kill命令(KILLALL SystemUIServer)来重新启动系统服务器，否则您的更改不会立即生效。

虽然您不太可能经常或定期更改屏幕截图文件格式，但当某个工作或项目需要使用PNG以外的特定格式的屏幕截图时，它会在某些特殊情况下派上用场。
