如何在Ubuntu 16.04 LTS上安装和管理Snap包

Ubuntu 16.04 LTS推出了“Snap”包，这是一个很棒的安装应用的新方式。快照需要不同的终端命令-apt-get和dpkg只允许您以旧方式安装.deb包，而不是Snap。

快照(扩展名为“.snap”)更类似于容器。快照中的应用程序是独立的，包括它们运行所需的所有库，并且是沙箱形式的。它们会安装到自己的目录中，并且不会干扰您系统的其余部分。

并不是所有的应用程序都能以快照的形式使用，但是如果你遇到一个可以使用的应用程序，这里是如何安装它的。

搜索可用的快照包

要查看商店中所有可用软件包的列表，请打开终端并运行以下命令：

snap find

要按名称搜索特定包，只需将您的搜索词添加到Snap Find命令的末尾：

snap find name

要获得更完整的搜索-搜索包描述以及包名-只需通过grep过滤工具通过管道输出nsnap find命令，如下所示：

snap find | grep search

如何安装Snap软件包

要安装Snap软件包，请使用以下命令，按名称指定软件包。因为这会对系统进行更改，所以您必须在命令之前添加sudo，才能使用root权限运行它。

sudo snap install package-name

snap命令将下载并安装您指定的Snap包，并在终端窗口中显示进度。

您可以像启动任何其他应用程序一样启动您安装的应用程序。如果它是图形应用程序，它应该出现在桌面的应用程序菜单中。否则，您只需开始在终端输入应用程序的名称，然后按“Tab”键自动完成即可。然后可以按Enter键启动应用程序或运行安装的命令。

如何更新快照

要更新已安装的Snap软件包，请运行以下命令，并指定软件包的名称。如果有新版本的快照可用，将下载并安装该快照。

sudo snap refresh package-name

目前似乎还没有更新所有已安装快照的命令，但如果将来添加一个命令，我们也不会感到惊讶。

如何列出已安装的快照

要列出已安装的快照包，请运行以下命令

snap list

您也可以使用此命令搜索已安装的软件包-只需再次通过grep输出：

snap list | grep search

如何删除快照包

要从计算机中删除已安装的Snap软件包，请运行以下命令：

sudo snap remove package-name

查看最近的更改

运行以下命令以查看系统更改列表。这将显示您最近安装、刷新(更新)和删除的快照包的列表，以及执行这些操作的时间。

snap changes

查看更多运营信息

要查看更多快照命令操作，请使用以下命令查看快照命令手册。使用箭头和PAGE UP/DOWN键滚动浏览手册。完成后按“Q”键退出。

man snap

Ubuntu的开发人员可能会继续致力于Snap包格式和相关工具的开发，所以我们可能会在未来看到更多使用Snap包的命令行选项。

如果您对创建自己的.snap包感兴趣，请参考Ubuntu的Snap文档获取更多详细信息。
