使用Microsoft Web Apps存储、编辑和共享文档

微软新的Office 2010版本提供的更令人兴奋的功能之一是能够通过其Office Web Apps服务共享和编辑文档。今天，我们来看看它是如何与Office 2010配合使用的。

不久前，我们概述了Office Live Web Apps功能以及您可以期待的功能。在这里，我们将进一步了解Office2010的功能，以及这项新服务可以实现的功能。

在Office 2010中使用Microsoft Live Web Apps

在此示例中，我们将了解如何共享MS Office文档，并为在线协作做好准备。在共享文档之前，您可以单击“文件”选项卡，他们称之为“后台视图”。它包括“准备共享”功能，该功能允许您检查文档、检查可访问性和检查兼容性。

在文档检查器中，决定要分析的内容类型。

它为您提供结果，您可以单击以删除文档的不同方面。

保存到SkyDrive

单击左侧的“共享”选项卡，您可以选择在SkyDrive、SharePoint上共享文档、发布到博客或通过电子邮件发送。

当你将其共享到你的SkyDrive时，系统会提示你登录到你的Live帐户。

登录后，您可以选择SkyDrive上保存文档的文件夹。

建立连接需要一段时间，资源管理器将打开，显示您在SkyDrive文件夹中拥有的文件，您可以像平常一样将其保存在本地计算机上。

在保存文档时，您将在文档底部看到一个进度条，显示它正在上载到服务器。

Office 2010有一项名为“上载中心”的功能，您可以使用该功能跟踪您上载的文档。从这里，您可以管理上传到服务器的文档。

如果由于某种原因无法访问MS服务器，文档将在后台部分显示为挂起的上载，您可以尝试再次上载或取消更改。

联机访问和编辑文档

若要访问和编辑从Web上载的文档，你需要登录到Windows LIVE帐户并进入SkyDrive，然后选择要处理的文档。

在下一个屏幕中，您和其他合作者可以在文档上留下评论。

要与其他人共享文档，您需要确保他们有权访问存储文档的文件夹，然后向他们发送网址链接。

与桌面应用程序相比，Web Apps服务目前的功能有限，但它确实允许进行基本编辑，是共享和协作处理文档的好方法。

PowerPoint Web App允许您进行快速编辑和创建新的基本演示文稿。没有保存选项，因为对演示文稿所做的编辑会自动保存。

使用PowerPoint演示文稿，您可以开始幻灯片放映。如果您希望以只读模式将演示文稿共享给Web上的其他人，这将非常方便。

当前您只能读取、打印或打开Word Web Apps…。目前还不能在线编辑它们，但根据微软的说法，这一选择正在进行中。

结束 / 结尾 / 结论 / 推论

Office Web Apps仍处于技术预览版，Office2010还处于Beta测试阶段，但任何人都可以使用这两款应用，您可能希望现在就开始使用。有几种方式可以通过Web Apps和SharePoint在线共享和访问文档，这对业务非常有用。Web Apps的另一个很酷的地方是，它们可以与Internet Explorer以外的浏览器一起使用。根据该网站的说法，它正式支持Mac上的Internet Explorer7&8、Firefox3.5和Safari4，尽管我已经能够让它在Google Chrome上运行得相当好。如果您正在寻找在线或从移动设备存储、编辑和共享文档的简单方法，Web应用程序是一个令人兴奋的新功能，您现在就可以开始使用它了。
