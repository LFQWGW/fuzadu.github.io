Android的最佳文件管理器

虽然许多用户可以在智能手机的整个生命周期中不需要触摸他们的手机文件系统，但有时需要更先进的措施。例如，当您需要侧面加载应用程序或移动下载的文件时，您将需要一个文件管理器。以下是我们挑选的Android所能提供的最佳选择。

相关：如何在Android上管理文件和使用文件系统

然而，在我们开始之前，让我们先来谈谈这个列表中没有的一个：ES文件资源管理器。很多很多年来，当有人问Android上“最好的”文件管理器时，答案很简单，那就是ES。然而，近几个月来，开发人员将这款应用带向了一个不同的方向-许多用户并不欣赏这个方向。他们决定在应用程序中添加大量广告，其中许多实际上可以被认为是恶意的(虚假的病毒警告等)，所以我们不能再有意识地在非常、非常精挑细选的情况下推荐这款应用程序。

但是，如果您正在寻找ES文件资源管理器的替代品，那么您来对了地方。

最适合大多数用户：Solid Explorer

Solid Explorer已经有一段时间了，并且已经获得了相当强大的追随者。文件管理器的开发非常活跃，这总是令人高兴的-它已经充满了有用的功能，开发人员做得很好，不断有更多的功能到来(而不会让人感觉臃肿)。

Play Store中有两个版本的Solid Explorer：Solid Explorer File Manager和Solid Explorer File Manager Classic。后者本质上是一个遗留的应用程序，因为该应用程序的一些早期采用者更喜欢它，而不是较新的重新设计。我们的建议是针对较新的应用程序，而不是经典版本。

正如您对文件管理器的期望一样，Solid易于使用且简单明了。它提供对常用文件的快速访问，如照片、音乐、视频，甚至应用程序，甚至允许为特定位置设置自定义书签。如果你发现自己重复访问相同的位置，这真是天赐之物。

撇开Solid整洁的外观和直观的界面不谈，我个人最喜欢的功能是双窗格模式。启用此功能(这应该是默认设置)后，将手机翻转到横向模式将弹出两个实心窗格，这对于将文件快速复制/粘贴到新位置非常有用。因此，如果您下载新墙纸并希望将其放在Pictures文件夹中，这是最好也是最简单的方法。

Solid的引擎盖下还有一些其他功能。首先，它允许特定的文件夹被“隐藏”，因此它们只在隐藏的菜单下显示。这并不是真正意义上的安全保护，但可以很方便地防止窥探(不知道这个隐藏位置)访问您宁愿自己保留的文件。

Solid还完全支持带根的手机，这对于寻求高级功能的用户来说是一个重要的工具。对于那些对文件管理器外观挑剔的用户，您甚至可以自定义外观。这真的是全套服务。

Solid Explorer可以免费试用14天，但之后需要支付1.99美元的应用内购买费用。这是值得的。

快速任务的最佳选择：Android的内置文件管理器

相关：如何使用Android 6.0的内置文件管理器

好的，所以你不想花钱买一些东西，你要用一两次来访问你最近下载的文件。我明白了，这里也有一个解决方案：Android的内置文件管理器。

虽然我希望这很简单，但根据您运行的Android版本的不同，它会变得有点复杂。例如，Nougat应用程序抽屉中的“Downloads”按钮打开的实质上是一个完整的文件管理器。它提供对下载、视频、音频、最新文件甚至Google Drive的快速访问。这几乎涵盖了大多数用户的基础。

然而，如果你不在牛油糖上，事情就会变得有点…。没那么有用。首先，Downloads应用程序正好显示了这一点：Downloads。仅此而已。如果你想要更多的访问权限，那么你必须首先安装这个应用程序(仅限棉花糖用户)，它将在你的主屏幕上创建一个指向棉花糖文件管理器的快速链接。即便如此，它的功能也不像牛油果公司的文件管理器那样功能齐全。这将打开设备上本质上是内部存储的根目录。从那里，您可以导航到所需的位置。

还值得一提的是非库存设备上的内置文件管理器。例如，如果您有三星或LG设备，则您的手机会随每个制造商提供相应的文件管理器应用程序。对于90%的用户来说，这可能已经足够好了。跟上它！

就像所有其他清单一样，这并不是万无一失的。毫无疑问，会有一些用户拥有非常具体的用例，并因此想要特定的功能。这些用户可能会喜欢像Total Commander或FX文件资源管理器这样的东西-这两个都是非常好的文件管理器，但比我们通常想要推荐的“最好的”略多一些。对于大多数用户来说，以上两个选项应该涵盖了您需要的所有内容。
