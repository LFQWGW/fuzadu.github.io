忘记Windows Media Center：使用免费的基于Linux的Media Center系统

Windows 8不再附带Windows Media Center。要获得它，你需要从微软购买Pro Pack和Media Center Pack升级，总共花费110美元。可以考虑使用免费的、基于Linux的媒体中心系统。

一旦你支付了所有这些钱，你将只拥有旧版本的Windows Media Center，没有任何改进。不管怎样，微软最终可能会停止使用Windows Media Center，因为他们不再专注于它。

XBMC与MythTV之争

您可以下载许多不同的基于Linux的媒体中心发行版，但我们将重点介绍可能是最受欢迎和支持最好的两个：XBMCbuntu和Mythbuntu。

您真正的选择是在XBMC和MythTV之间。两者都有不同的优点，您更喜欢哪一种将取决于您如何使用媒体中心-是用作从传统电视连接录制电视的DVR，还是用作播放Internet流和本地媒体文件的盒子。

XBMC：XBMC最初名为“Xbox Media Center”。它旨在播放来自本地文件、网络共享或在线流媒体服务的视频。如果您使用媒体中心PC播放下载的文件或流式传输Netflix、Hulu、YouTube和其他在线资源，您将需要XBMC。 MythTV：MythTV是一种DVR/PVR。如果您有电视源(天线、电缆或卫星)和电视调谐卡，MythTV将允许您观看电视、安排录制时间以及稍后播放录制的电视节目。如果这是您使用媒体中心PC的方式，您将需要MythTV。

当然，您可以将XBMC转换为PVR，或者通过MythTV利用一些流媒体服务，但这两种方法都不能很好地发挥作用。每个包都有自己的优点，并针对不同类型的使用而设计。

您也可以使用基于Windows的媒体中心软件，而不是依赖专用的Linux系统-Windows Media Center不是Windows上的唯一选择。

XBMCbuntu

XMBCbuntu，以前称为XBMC Live，是一个包含XBMC软件和最小Ubuntu系统的软件包。该操作系统被设计为用作媒体中心，因此您不必安装Linux发行版并配置XBMC在其上工作。

XBMCbuntu是最受官方支持的基于XBMC的Linux发行版。可以从XBMC的主下载页面下载。下载ISO文件，将其刻录到光盘(或放在USB驱动器上)，然后从该文件引导。在安装XBMCbuntu之前，选择Try Ubuntu选项以尝试XBMCbuntu。

您需要以XBMC用户身份登录才能继续。(如果需要，可以在XBMCbuntu官方FAQ页面上找到更多说明。)

登录后，您可以启动XBMC软件并使用它，就像在Windows、Android、Mac或XBMC运行的任何其他平台上一样。

Mythbuntu

Mythbuntu是一个官方的Ubuntu衍生产品，专注于提供专用的MythTV系统。和XBMCbuntu一样，它提供了一个标准的XFCE桌面，没有所有典型的Ubuntu桌面软件--只有一个MythTV系统。Mythbuntu包括一个为MythTV定制的图形控制中心。它拥有您需要的一切，无论您是想建立一个独立的MythTV系统，还是想将一个新系统集成到现有的MythTV网络中。

与XBMCbuntu一样，在从其引导计算机之前，您首先需要下载Mythbuntu并将其刻录到光盘(或将其复制到USB驱动器)。启动光盘(或USB驱动器)并选择Try Ubuntu选项。您将被放到Mythbuntu桌面，在那里您可以轻松地启动Mythbuntu前端或将系统安装到您的硬盘上。

Mythbuntu需要更多的设置-在使用前端之前，您需要安装它，并使用设置实用程序在您的家庭影院PC上对其进行设置。

当然，软件解决方案不是唯一的选择。你可以用比标准Windows8系统上的Pro Pack和Windows Media Center更低的价格购买Roku或Apple TV等专用硬件。
