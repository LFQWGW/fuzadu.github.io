如何在Ubuntu中启用Caps Lock或Num Lock时收到通知

如果你不小心打开了Caps Lock太多次-我们都有过这样的经历-这里有一个解决方案。您可以向顶部面板添加指示器，该指示器显示Caps Lock、Num Lock和Scroll Lock键的状态，当其中一个键被按下时会通知您。

相关：如何在任何操作系统上禁用或重新分配Caps Lock键

当然，你可以禁用或重新分配Caps Lock键，但如果你偶尔使用，这是下一个最好的事情，你只需要安装一个名为Indicator Keylock的小应用程序。

首先，我们将添加包含Indicator Keylock程序的存储库。要执行此操作，请按Ctrl+Alt+T打开终端窗口。然后，在提示符下键入以下命令并按Enter键：

sudo add-apt-repository ppa:tsbarnes/indicator-keylock

出现提示时键入您的帐户密码，然后再次按Enter键。

当系统提示继续添加存储库时，按Enter键。

添加存储库后，键入以下命令并按Enter键以确保Indicator-Keylock包以及所有其他包都是最新的。

sudo apt update

现在，我们将安装指示器键锁程序。在提示符下键入以下命令，然后按Enter键。

sudo apt install indicator-keylock

如果出现提示，请输入您的密码。

当系统询问您是否要继续时，键入y并按Enter键。

安装完成后，或者在提示符下键入exit并按Enter键，或者单击终端窗口左上角的“X”按钮将其关闭。

要将指示器键锁程序添加到顶部面板，您必须注销并重新登录或重新启动。首先，您会注意到，即使在打开和关闭Num Lock时，您也没有在面板上看到任何指示器。但是，当您打开或关闭Num Lock时，您将看到一个通知框显示在屏幕右侧顶部面板的下方。

但是，当您启用Caps Lock时，您不仅会看到通知，还会在顶部面板上看到一个带有字母“A”的红色图标。启用Caps Lock时，该图标为红色。

面板上只显示一个图标，但您可以选择要显示的图标。单击面板上的A图标，然后从下拉菜单中选择一个图标。

该菜单通过将菜单上当前启用的锁定键的图标变为红色来指示哪些锁定键处于启用状态。例如，下图表明Num Lock键是当前唯一启用的Lock键。但是，请注意，即使启用了Num Lock键，面板上的图标也不是红色的。首选项中有一个设置，允许您指定哪个Lock键将面板上的图标变为红色，默认情况下，它最初设置为Caps Lock键。要更改此设置，请从下拉菜单中选择“首选项”。

还记得吗，最初，面板上没有图标？在面板上获得图标的唯一方法是启用Caps Lock。然后，您可以单击该图标并更改它或打开首选项。您可以选择始终在面板上显示图标，而不管是否启用了任何Lock键。要执行此操作，请选中“锁定键处于非活动状态时显示指示器”框。

首选项对话框上的Caps Lock、Num Lock和Scroll Lock选项控制启用哪个键会将面板上的指示器图标变为红色。例如，我希望面板上的图标在Num Lock键打开时变为红色，因为我的键盘没有该键的指示器。

如果您根本不想在面板上显示指示器，即使有任何Lock键处于打开状态，也可以取消选中“使用应用程序指示器”框。

单击“关闭”保存您的设置并关闭首选项对话框。

我已经在下拉列表和首选项对话框中选择了Num Lock。因此，指示器键锁现在设置为在面板上显示Num Lock图标，并且仅当Num Lock键打开时该图标才变为红色。如果Caps Lock(大写锁定)或Scroll Lock(滚动锁定)键处于打开状态，它们将在下拉菜单上变为红色，但不会将面板上的图标变为红色。

如果您取消选中首选项对话框上的“使用应用程序指示器”框，您可能想知道如何取回指示器，因为面板上根本没有图标可供单击，因此您可以打开首选项对话框。别担心。您仍然可以打开首选项对话框。只需单击Unity Launcher上的破折号按钮，或者您也可以按键盘上的超级键(通常是典型键盘上的Windows键)。

开始在屏幕顶部的搜索框中键入Indicator，直到您看到Indicator-LockKeys图标显示。单击该图标以打开首选项对话框。然后，您可以再次选中“使用应用程序指示器”框，以便在打开Caps Lock时显示指示器，并更改首选项对话框中的其他设置。

更改指示器键锁中的任何设置时，这些设置可能不会立即生效。在应用设置之前，可能需要切换受影响的Lock键几次。

Windows用户也可以利用类似的技巧。在Windows中，您可以让系统播放声音，也可以在按下这些键中的任何一个时收到任务栏通知。
