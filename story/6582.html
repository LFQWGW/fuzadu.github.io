Microsoft Word是字处理的全球标准。同时，它也是最让人抓狂的应用程序之一，这就是为什么这个极客学校系列都是关于学习如何在Word中格式化文档的。

Word 2013和小视角

微软远远不只是一个典型的古板的文字处理器。Word是最实惠、最接近您自己的印刷机的产品之一。事实上，有了Word，你就可以编写教科书，用图形创建完整的杂志和报纸版面，用索引写小说，等等。你可以在短短几个小时内完成，这是20年前可能需要整个编辑团队几天甚至几周才能完成的工作。

Microsoft Word完全消除了打字错误的加重(至少在理论上是这样)。不需要重新键入整个章节来添加或重新排列内容。相反，您可以在短短几秒钟内添加、移动甚至删除完整的句子、段落和章节！

当然，我们认为这种能力是理所当然的，但我们可以告诉你，这真的比使用打字机(更不用说活动打字机)要好-使用打字机犯错误意味着停止你正在做的事情，卷起滚筒以更好地暴露你的打字错误，然后要么用橡皮擦删除有问题的字符，要么小心地涂抹白色并耐心地吹干它。当然，然后你必须把滚筒卷回到你正在打字的那一行，进一步小心，以确保它们都完美地排列在一起。

如果你能想象一下你每天会犯多少个打字错误，那么你大概就能知道制作一份简单的文档需要多长时间。不用说，准确是值得的，除非你是一个真正优秀的打字员，否则打一篇文章或读书报告可能是一个漫长而艰巨的过程。忘记将图片添加到您的文档中。在家里做那种事情几乎是不可能的。哦，当然，你可以加上插图和照片，然后参考它们，但它不像我们已经习惯的剪贴式复制粘贴那样简单和优雅。

尽管如此，所有这些权力和控制都是伴随着相当陡峭的学习曲线而来的。掌握并流利地有效格式化醒目的文档可能是一件痛苦的事情。幸运的是，这就是我们的用武之地-如何使用Microsoft Word 2013格式化极客学校的文档。

家是文字之心所在

在深入到实际的文档格式化之前，我们将花一些时间来讨论“主页”选项卡。即使您在您的余生中从未接触过Word的其他部分(几乎不可能，但仍然是如此)，“主页”选项卡包含其最基本的功能，并且对于始终如一地很好地格式化您的文档至关重要。

在这里可以看到“主页”选项卡总共有五个部分：“剪贴板”、“字体”、“段落”、“样式”和“编辑”。

写字板 / 剪贴板 / 附有纸夹的笔记板

“剪贴板”功能非常基础；您现在应该知道它们：剪切、复制、粘贴。您最有可能使用右键单击菜单来执行许多剪切、复制和粘贴功能，或者使用键盘快捷键：分别为“CTRL+X”、“CTRL+C”、“CTRL+V”。

然而，打开“剪贴板”窗格，会发现一座功能的金矿，这些功能在格式化文档时实际上可以被证明是非常有用的。Word剪贴板收集您剪切或复制的所有内容以备日后使用。如果需要在整个文档中粘贴多个不同的文本和/或图像段落，此功能尤其有用。您只需将指针放在正确的插入点，打开“剪贴板”查看器，然后选择要粘贴的片段即可。

大多数Word用户应该非常熟悉“字体”部分和适用的对话框。即使您不是Word专家，您也会在每次创建文档时使用Word中的字体函数。每次使用粗体或斜体时，都是在使用字体函数。因此，熟悉“字体”部分和对话框是掌握Word格式花哨的绝佳方法。

段落 / 节 / 短评 / 段落符号

“段落”部分也很重要，它允许您设置关键的格式功能，如缩进、行间距和分页符。此外，通过调整段落控件，您可以使用边框、阴影以及打开或关闭段落标记。我们将在第二课中详细讨论这一点。

风格

样式是快速、轻松地管理整个文档的标题、标题和文本的一种很好的方式。无需逐个浏览文档并添加或更改标题，您只需应用样式，然后使用“样式”部分对其进行更改。在本系列的最后一课中，我们将更深入地探讨风格。

页面

你的页面是所有魔术发生的地方，也是你创作杰作的地方，因此，了解你周围的路是至关重要的。让我们通过打开“标尺”来深入讨论，然后解释如何设置制表符和页边距。

要打开标尺，我们将首先单击“View”选项卡，然后在“Show”部分中，选中“Ruler”旁边的复选框。请注意沿页面边缘显示的水平和垂直标尺。

如果您想使用另一种测量系统，可以从“文件”->“选项”->“高级”进行更改。

制表符

在标尺打开的情况下，我们可以介绍如何使用制表符和设置页边距。标尺用于显示，以显示制表位和页边距的位置。

制表符用于使用“Tab”键定位文本。这比手动设置间距效果更好，而且对于大多数字体来说，制表符是确保所有内容正确排列的最可靠方式。

Microsoft Word默认情况下将制表符的间隔设置为1/2英寸。当你点击“制表符”时，插入点会自动向右跳转(每制表符0.5英寸)。

您可以通过单击标尺来设置制表符，以指示要放置它们的位置。您将看到一条垂直的虚线，使您可以更精确地控制它们的去向。

您可以在文档的任何部分中设置选项卡，这意味着页面顶部可以具有与中间或底部不同的选项卡。基本上，如果需要或愿意，您可以在文档的每一行上使用不同的Tab键方案。

边距

您可以通过确保Word在“打印版面”中查看来查看页边距。

在本例中，我们看到左边距设置为2英寸，左边距设置为4英寸，从而获得2英寸的水平可打印区域。边距指示器是下箭头，而上箭头是悬挂式缩进，我们将在下一节介绍这一点。

在普通文档上，左右边距默认为1英寸和6.5英寸。这意味着在一张8.5x11的常规纸张上，不会出现打印的页边距为1英寸，从而为您提供了6.5英寸的水平可打印区域。

要移动边距和悬挂缩进，请将鼠标指针悬停在每个边距和悬挂缩进上，直到它变为箭头，然后将它们拖动到所需的大小。

如果你只是抓住左边的空白处，就会留下悬挂的缩进。

在这一点上，让我们更详细地简要讨论一下缩进。

缩进

缩进用于将段落定位为带有边距或位于表格中的列内。

你可以根据你所写的内容进一步调整你的页边距。例如，您可以创建“第一行缩进”。这更像是一种老式的风格，每段的第一行都会缩进。

这是一种更传统的设置段落格式的方法，允许您指示单行距文档中新段落的开始位置。今天，文本的格式通常是段落之间有双倍空格的块样式。

第二行或“悬挂缩进”将自动缩进第一行之后的每一行。缩进的一个混淆部分是您可以将它们移到页边距外，这是违反直觉的，除非您认为打印机可以在页边距外打印，并且仅受纸张宽度的限制。

当涉及到制表符、页边距和缩进时，没有太多需要掌握的东西。也就是说，了解它们是如何工作的是值得的，这样您就可以在文档中获得更精确的结果。它还可以让您更好地理解为什么文档看起来是这样的，或者更重要的是，为什么它看起来可能不是您想要的样子。

格式化标记

在我们继续之前，我们应该指出，您现在可能会注意到，在我们的一些屏幕截图中，有显示段落、空格、制表符和其他内容的格式化标记。要查看文档中的制表符和其他文本格式标记，请在“主页”选项卡上的“段落”部分选择？(段落)符号。

要选择显示哪些格式标记，可以在Word的“选项”中选择它们。要打开选项对话框，请首先单击“文件”选项卡，然后选择“选项”。最后，在“显示”下，您将看到您可以选择始终显示的格式选项。

例如，如果要关闭除段落和空格之外的所有格式标记，则只需选择这两个。然后，您可以关闭“段落”部分中的所有或个别格式标记。

格式化标记对于创建干净、格式一致的文档非常重要，它们不会显示在最终的打印文档中，而且您可以根据需要打开或关闭它们，因此要学会使用它们。

字体与字样

字体和字体将成为您日常文档格式化的常规部分，除非您对您编写的每个文档都使用一种字体感到满意。良好的字体使用是非常重要的，因为它可以让你更好地表达自己，让你的观点更好地被理解。因此，您至少需要了解它们是如何工作的最基本的知识，以及在哪里和为什么使用什么样的字体是合适的。

为了澄清起见，“字样”基本上是字母、数字和符号的集合在其整体上的外观。这里我们看到的是Times New Roman字体，无论您使用哪种字体，它都将具有相同的特征。换句话说，时间看起来就像时间，不管它是粗体、斜体还是您应用的任何格式。

字体可以理解为整个字体集合。例如，Times New Roman及其各种形式(粗体、斜体、粗体斜体)是一个“字体系列”。族中的每种变体(常规、粗体、斜体和粗体斜体)都是一种字体：

为简单起见，我们将把所有与类型相关的东西都称为字体，而不是用字样和字体来混淆您，而不是吹毛求疵。

模板

在结束本课之前，我们应该花点时间简要介绍一下模板，因为它们通常可以快速处理复杂的布局。

模板是预配置的文档，如简历或名片，可用于加快表单创建速度。几乎您能想到的任何东西都有模板。

Microsoft Word的目标有两个：(1)提供一组主题和样式，以便Word用户可以创建具有专业外观的文档；(2)通过提供可供用户选择的工具和预配置的对象集，使用户能够创建具有图形设计者质量的文档。

当您打开Microsoft Word或单击“文件”选项卡中的“新建”时，第一个屏幕会显示您可以使用的模板，这些模板要么已经包含在程序中，要么可以快速下载。如果您没有立即看到您想要的内容，请尝试“建议搜索”或使用搜索框。

右键单击任何模板，您都可以“预览”或“创建”该模板。您还可以固定首选模板，使其始终位于列表顶部。

如果模板存储在您的计算机本地，则创建模板将导致其打开，否则将下载该模板。请注意，某些模板(如下图中的礼券)是由第三方网站提供的，因此它们可能并不都是免费的。

如果您决定购买第三方模板，系统将为您提供有关如何操作的进一步说明。

选择模板后，它将作为新文档打开，您可以填写它并根据自己的喜好进行调整。我们在这里看到了“基本简历”的模板。

请注意Word将如何自动填写您的姓名，模板提供了如何使用它的说明。实际上，该模板实际上只不过是一个应用了主题(在第5课中讨论)的表(在第3课中讨论)。

填写完模板后，可以将其另存为新文档。您还可以获取模板，对其进行更改，然后将其另存为新模板。比方说，你想给我们的“基本简历”应用一种不同的风格。您只需打开模板，影响所需的更改，然后将其另存为新模板。

使用模板可以发现很多东西。最重要的是，您不必担心自己创建每个文档。需要一张快速名片或退休聚会邀请函吗？Word模板可以快速处理许多令人头疼的格式问题，让您有时间实际设计一些您会满意的东西！
