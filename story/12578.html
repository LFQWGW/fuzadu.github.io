家庭影院投影仪需要多大空间？

在投影仪上看电影和玩游戏都很棒。一旦你习惯了巨型显示器，即使是最大的电视也很难观看。但是在你买投影仪之前，你需要确保你有足够的空间放它。

投影仪本身不需要太大的回旋余地，但你需要确保它和屏幕之间有足够的空间来达到你想要的屏幕尺寸。只要稍微算算一下，就可以很容易地计算出你的起居室需要多深。

把它扔出去

当你开始研究投影仪时，你会看到一些规格和行业术语。最需要注意的是“投掷比”，也就是“投掷距离”。您将使用它来计算投影仪需要坐在离屏幕多远的地方。如果投掷距离是“1”，那么您需要将投影仪放在离屏幕一英尺的地方才能显示一英尺的对角线图像。如果投掷比小于1，您可以将投影仪放置在距离屏幕不到1英尺的地方，以生成1英尺的对角线图像。然后，如果投掷比大于1，则需要将投影仪放置在距离屏幕1英尺以上的位置才能生成1英尺的图像。

您将经常看到列出多个投掷比的投影仪。这意味着投影仪有一个变焦轮，因此您可以更改图像的大小。有多个投球比不会改变我们的数学，你只需要重新计算。

以下是确定投影仪需要离屏幕多远的计算方法：

Throw Ratio X Desired screen size (Inches or Centimeters) = Distance from screen (Inches or Centimeters)

作为一个例子，我们可以看看奔驰HT2150ST。比方说，我们想要150英寸的屏幕，并且需要知道投影仪应该放置在多远的地方。这个模型的投掷比在0.69到0.83之间，所以我们的数学是这样的：

0.69 (throw ratio) X 150 (desired screen size) = 103.5 (distance from screen)0.83 (throw ratio) X 150 (desired screen size) = 124.5 (distance from screen)

另一种衡量您需要多少空间的方法是查看投影仪的营销或型号名称。你会看到投影仪被宣传为“短投掷”、“超短投掷”，或者可能没有任何投掷区别。

什么是超短投影仪？

超短投掷投影仪的投掷比通常小于0.4。这意味着超短投影仪有一个广角镜头，可以让它们坐得尽可能靠近屏幕。这些设备非常适合家庭影院，因为您不必担心安装投影仪，也不必担心有人在投影仪和屏幕之间走动而挡住了部分图像。

我目前拥有一台ViewSonic PX800HD投影仪，投掷比为0.23。这意味着因为我把它放在离墙约40英寸的地方，所以我有一个175英寸的屏幕。它是光荣的。这是我们的计算结果，对这台投影仪进行了细分：

0.23 (throw ratio) X 175 (desired screen size) = 40.25 (distance from screen)

超短投影仪对于任何场地紧张的人来说都是很棒的，特别是那些可能不允许将投影仪安装在天花板上的租户。如果你想要一个巨大的便携式“监视器”，这也是很棒的：我在后院的电影之夜使用过我的投影仪，在婚礼和葬礼上放映照片，以及介于两者之间的一切。我不需要铺设一条很长的延长线，因为投影仪已经放在靠墙的地方了。

超短投射镜头是最昂贵的，所以在分辨率和亮度等其他因素相同的情况下，这些投影仪是最昂贵的。

什么是短投投影仪？

短投投影仪比它的超短投投影仪离屏幕更远一点。短投的投掷比通常在0.4到1之间，这意味着短投投影仪通常需要坐在离屏幕大约5到10英尺的地方。你可以在天花板上安装短投投影仪，也可以把它们放在起居室中间的小桌子上。

其中一个比较流行的短投投影仪是明基TH671ST。此投影仪有一个变焦轮，投掷比在0.69%到0.83%之间。要在您的家庭影院中获得150in的图像，此投影仪需要距离屏幕103英寸(8.6ft)到124.5英寸(10.38ft)。这在最小的公寓里是行不通的，但在大多数家庭中并不是不合理的。下面又是我们的方程式：

0.69 (throw ratio) X 150 (desired screen size) = 103.5 (distance from screen)0.83 (throw ratio) X 150 (desired screen size) = 124.5 (distance from screen)

我在使用短投投影仪时遇到的一个问题是确保一切都通电。投影仪、我的Xbox和其他小玩意儿都放在客厅中间一张咖啡桌的下部。我用沙发下面的一根长电源线接了一条配电板，然后把它插到了咖啡桌上。从那里，我把电源板和我的电子产品的所有电源线都放在一个线缆管理箱里，试图让一切看起来尽可能整洁。这有点管用，但我对我的超短投球更满意，因为我不需要太担心有线电视的管理。

短投投影仪介于超短投投影机和标准投影机之间，这意味着当你要付多少钱时，它们也处于中间状态。

什么是标准投影仪？

我们使用“标准”作为区别于短投或超短投投影仪的一种方式，但你可能看不到“标准”这个词列在营销中。标准投影仪的投掷比高于1，因此需要最大的空间来生成巨幅图像。

Optoma UHD60的投掷比在1.39到2.22之间，所以如果你想用它来拍摄150英寸的图像，你需要把它放在离屏幕208英寸(17英尺)到333英寸(27英尺)的地方。这需要一个很深的家庭影院，尽管在户外环境下可能完全没有问题。以下是我们的计算结果：

1.39 (throw ratio) X 150 (desired screen size) = 208 (distance from screen)2.22 (throw ratio) X 150 (desired screen size) = 333 (distance from screen)

不过请记住，除非你想遮挡部分图像，否则投影仪和屏幕之间17英尺范围内不能有任何其他东西。你可以把大多数标准的投影仪放在一张小桌子上，但如果你把它装在天花板上或放在墙上的架子上，你会更开心。

标准投影仪是生产成本最低的投影仪，因此也是最便宜的。请记住，您省下的一些钱将用于安装、超长HDMI电缆和其他必需品。

那么哪一种最适合你呢？

决定购买哪种类型的投影仪归根结底是两个因素：空间和预算。你的空间越小(也就是说，你想要的投掷距离越短)，你的投影仪就会越贵。但是，你也可以从更昂贵的短或超短投影仪中获得更多的用途，因为你几乎在任何地方都可以从它获得巨大的图像。

也要密切关注打折交易：我的ViewSonic PX800HD翻新后几乎是半价买的。它少了一根电源线，但我在投影仪上的积蓄让它变得物有所值。

无论你有多大的空间，你都可以用合适的投影仪得到一个巨大的屏幕！
