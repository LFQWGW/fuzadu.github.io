如何删除Win 7 Anti-Spyware 2011(虚假防病毒感染)

如果您的PC感染了Win 7 Anti-Spyware 2011恶意软件或类似软件，那么您来对了地方，因为我们将向您展示如何清除它，并将您的PC从这种阴险恶意软件(以及许多其他恶意软件)的可怕魔爪中解放出来

Win 7 Anti-Spyware 2011只是众多虚假防病毒应用程序之一，如AntiVirus Live、Advanced Virus Remover、Internet Security 2010、Security Tool和其他将您的计算机扣为人质直到您支付赎金的应用程序。他们告诉你，你的电脑感染了假病毒，并阻止你采取任何措施来删除它们。

这种特殊的病毒有很多名字，包括XP反间谍软件、Win 7反间谍软件、Win 7 Internet Security 2011、Win 7 Guard、Win 7 Security、Vista Internet Security 2011等等。它们都是相同的病毒，但是根据你的系统和你感染的是哪种病毒而重新命名。

现在是什么？

如果你不熟悉这个骗局，现在是时候来看看这个可怕的骗局的面孔了。如果您受到感染，请向下滚动到我们解释如何删除它的部分。

一旦电脑被感染，它就会显示这个看起来非常官方的窗口，假装扫描你的电脑并找到被感染的东西，但当然，这都是谎言。

真正疯狂的是，它会弹出一个看起来非常逼真的行动中心窗口，但实际上是病毒。

删除无赖假冒防病毒感染(一般指南)

您通常可以遵循几个步骤来消除大多数流氓防病毒感染，实际上是任何类型的大多数恶意软件或间谍软件感染。以下是快速步骤：

请尝试使用SUPERAntiSpyware的免费便携版本来删除病毒。 如果这不起作用，请使用联网功能重新启动您的电脑进入安全模式(在Windows开始加载之前使用F8键)。 请尝试使用SUPERAntiSpyware的免费便携版本来删除病毒。 重新启动您的PC并通过联网返回到安全模式。 如果这不起作用，并且安全模式被阻止，请尝试运行ComboFix。请注意，我还没有不得不求助于此，但我们的一些读者已经求助于此。 安装MalwareBytes并运行它，执行完整的系统扫描。(请参阅我们上一篇关于如何使用它的文章)。 再次重新启动您的电脑，并使用您的普通防病毒应用程序(我们建议使用Microsoft Security Essentials)运行完全扫描。 此时，您的PC通常是干净的。

这些都是通常有效的规则。请注意，有些恶意软件感染不仅会阻止安全模式，还会阻止您执行任何操作。我们很快会在另一篇文章中讨论这些内容，所以一定要订阅How-to Geek获取更新(页面顶部)。

正在删除Win 7 Anti-Spyware 2011

下载免费的MalwareBytes副本，将其复制到拇指驱动器，然后将其安装在受感染的PC上并运行扫描。在安全模式下执行此操作可能会更幸运。

如果病毒允许，您可能会更幸运地先安装MalwareBytes。在我的情况下，情况并非如此。当我第一次使用SUPERAntiSpyware进行扫描时，它检测到病毒并很好地删除了文件。

在这一点上，您应该有一个干净的系统。请确保安装Microsoft Security Essentials，不要再被这些病毒欺骗。

删除病毒后无法打开任何应用程序？

下一个问题是，一旦病毒被移除，你就无法打开任何东西--事实上，我仍然无法安装MalwareBytes。希望你有更好的运气。

为什么我打不开任何东西？因为病毒重写了注册表，强制所有应用程序打开病毒-这意味着您甚至无法打开注册表编辑器来修复问题。如果我正确地完成了扫描，这个问题本可以避免，但我在扫描完成之前中断了它。

在普通PC上，HKEY_CLASSES_ROOT下有一个注册表项，指定双击可执行文件(*.exe)时会发生什么-但在感染病毒的系统上，会用病毒可执行文件重写此值。这就是它阻止你打开任何东西的方式。

为了解决这个问题，我从另一台PC导出了一个干净的注册表文件，并对其做了一些额外的修改，问题解决了！您只需下载、解压缩.reg文件并将其复制到受感染的PC，然后双击该文件即可将信息添加到注册表中。
