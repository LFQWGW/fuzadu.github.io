在Windows Home Server中添加用户

Windows Home Server仍处于测试阶段，但当它正式发布时，我们将拥有有效管理您的家庭服务器所需的所有信息。以下是有关如何向您的家庭服务器添加新用户的快速教程。

Windows Home Server控制台是您进行大部分服务器管理的地方。在这里，您需要单击用户帐户，然后选择+添加。

在Add User Account(添加用户帐户)屏幕中，输入您的用户名和登录名。您可以在此确定是否允许他们远程访问服务器。

接下来，您可以扮演管理员，并选择您的新用户对共享文件夹的权限。读/写意味着用户可以查看和更改文件。只读意味着用户可以查看文件，不能进行更改。*没有访问权当然意味着此共享中的文件将不可用。*配置完用户设置后，单击完成。

Windows Home Server现在将添加您为其配置的用户和权限。/单击“完成”。

现在，您将在家庭服务器控制台中看到您的用户作为图标。您可以随时返回并进行适当的更改。
