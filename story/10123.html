如何手动和自动更新Plex媒体库

除了媒体本身，媒体服务器软件最重要的元素是它的最新程度-如果服务器不知道视频在那里，你就不能看视频。Plex Media Server提供了三种方法来使您的媒体收藏保持最新，这样您就可以随时了解可用的内容。

我为什么要这么做？

相关：如何设置Plex(并在任何设备上观看电影)

Plex Media Server体验包含三个组件。有媒体本身(就像您的电视节目)，有管理媒体的实际Plex Media Server，还有通过Apple TV、iPhone或其他设备访问服务器的Plex客户端应用程序。

由于您不是直接浏览包含视频的文件夹，因此Plex Media Server的数据库保持最新至关重要。例如，如果你外出出差，试图赶上你最喜欢的节目，但Plex Media Server还没有更新到包括最新的最新剧集，那么你就不走运了。在库更新之前，您将无法观看它们。

值得庆幸的是，Plex Media Server软件有多种方式可以确保您的媒体列表始终是最新的，包括手动更新和自动更新过程的多种方式。

如何手动更新库

最简单的方法是手动更新库。即使您遵循本指南后面的步骤并完全自动化更新过程，您也应该始终熟悉手动更新过程，因为这是强制立即更新的好方法。

要手动更新Plex库，请登录到Plex Media Server的Web控制面板。在主页上，从左侧导航窗格中选择一个库，如下所示，就像您的“电视节目”库一样。

在库中，单击右上角的圆形箭头图标。

这将触发对该库的手动更新，并且Plex Media Server将重新扫描分配给该库的目录。对要更新的所有其他库(例如，电影和音乐)重复此过程。

如何自动更新库

如果您需要立即强制更新，手动更新是很棒的，但是为了便于使用和真正流畅的用户体验，您确实想启用自动更新。自动更新不仅对你来说非常方便，而且如果你有其他人在你的房子里，它几乎是一种必需品。如果Plex始终是最新的，那么作为媒体服务器帝国的经理，您就不会被关于最喜欢的节目是否是最新的问题所困扰。

有几种方法可以自动化库更新，您可以在Plex Media Server的设置内的单个菜单中找到所有这些方法。单击Plex Web界面右上角的设置工具图标。

在设置菜单中，选择顶部导航栏中的“服务器”选项卡，然后从左侧导航面板中选择“库”。

要查看我们将突出显示的所有可用选项，您需要单击“显示高级”按钮以完全展开库选项。

高级选项可见时，您将在库设置菜单中看到以下条目：

这里有几个选项，您应该根据媒体的存储位置和更新需求来考虑每个选项。

自动更新：几乎所有人的最佳选择

顶部的选项“自动更新我的库”是理想的选项。几乎每个Plex用户都应该检查一下。只有当Plex用户的媒体存储在与Plex Media Server程序不同的计算机上时，自动库更新才不是可行的解决方案(因为自动检测文件夹通常不适用于网络共享上的文件夹)。

选中“检测到更改时运行部分扫描”是可以与自动库更新配对的附加选项。此选项将减少库更新时间和用于库更新的系统资源量。在功能强大的系统上这不是什么大事，但无论如何它都可以节省时间(如果您的Plex Media Server硬件功能不足，它会非常有用)。

计划更新：非常适合网络共享上的媒体

如果您发现自动库更新不适用于您的系统，您可以始终使用“定期更新我的库”设置来设置更新计划。此选项既适用于本地文件，也适用于位于网络共享(即，安装Plex Media Server的计算机以外的计算机)上的文件，因为它会以您指定的频率手动扫描整个目录结构。您可以指定低至每15分钟一次的增量更新频率，一直到一天一次。

自动垃圾：你可能不想要的家政服务

最后，有一个关于库清理的小考虑事项。默认情况下，选项“每次扫描后自动清空垃圾桶”处于选中状态。如果您有固定库(例如，您的所有媒体都位于固定内部驱动器上的特定服务器上)，最好将此选项保留为选中状态。这样，如果您从您的媒体服务器中删除了一堆内容，Plex将自动在您之后进行清理，并删除该媒体的所有条目。

但是，如果您使用的外置驱动器有时会从Plex Media Server计算机上移除，您在Plex Media Server上共享的媒体位于另一台计算机的网络共享位置(并非始终处于打开状态)，或者其他一些您希望保存在媒体库中的媒体并非始终在线的情况，请不要选中“清空垃圾桶”选项。如果您这样做了，每次Plex运行更新扫描并无法看到脱机介质时，它都会将其删除(只会在下次联机时转过身来重新扫描介质，下载所有元数据，依此类推)。

一旦您按照您想要的方式配置了自动更新，您就可以坐下来享受Plex Media Server的最佳优势之一：始终保持对整个媒体收藏的最新和集中管理。
