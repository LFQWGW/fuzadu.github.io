如何(以及为什么)开始使用Google Cloud Print

如果您可以随时从您的任何设备(台式机、手机、平板电脑)打印到您的任何打印机，那不是很棒吗？请继续阅读我们的Google Cloud Print，这是一种在您的所有设备上实现打印的绝佳方式，无论您的设备大小如何。

我为什么要这么做？

最好的工作流程和计算机调整让您的生活变得更轻松，设置Google Cloud Print当然也是如此。一旦您了解了系统并对其进行了配置，您将能够打印到您的所有设备，而不需要一堆烦人的中间人步骤。

换句话说，不会再有“好的，我会在我的Android手机上看这个文件，所以我会把它保存到我的手机上，复制到我的Dropbox账户，走到电脑前，从Dropbox那里拿到它，然后把它发送到我的家庭打印机上。”只会有“好的，我会把这个打印出来。”云打印系统会将它直接从您的手机(无论您在世界上的任何地方)传送到您选择的打印机。

长期以来，从台式电脑轻松打印一直是小菜一碟；谷歌云打印(Google Cloud Print)为您的电子设备稳定中的其他所有设备带来了同样的一键打印的便利性。

我需要什么？

要使用Google Cloud Print，您需要三个主要条件：

一个谷歌账号。 连接到打印机的启用云打印的打印机或能够运行Chrome的计算机。 能够打印到Cloud Print的设备(例如Android手机、iOS设备、笔记本电脑或平板电脑)。

将一切捆绑在一起的粘合剂是您的Google帐户和云打印服务器。绑定在一起的内容取决于您的个人设置。如果您有一台较新的支持Cloud Print的打印机(请查看此处的列表)，您将不需要计算机来充当打印服务器。

如果您没有启用云打印的打印机，则需要一台在您想要打印时打开并连接到打印机的计算机(如果您有一台始终在线的家庭服务器，现在最好将其配置为打印到您的家庭打印机，以便您可以使用它而不是桌面)。

最后一个组件是您希望配置为打印到云打印机的设备。此设备(或多个设备)很可能是移动的，因为毕竟，您的永久定位设备(如家庭办公计算机等)很可能已经配置为通过传统方法打印到您的家庭计算机。

如何配置我的打印机？

首先要做的是让实际的物理打印机在线并成为您的云打印网络的一部分。

如果您有一台支持Cloud Print的打印机，配置它的最佳方式是参考手册-查找Cloud Print入门部分或类似内容。设置应该很简单，只需在打印机的配置面板中插入几个项目即可。

如果您没有启用Cloud Print的打印机，则需要完成一些额外的(虽然不重要)步骤。为了完成它们，您需要使用的计算机1)当您想要打印时打开，2)安装了Google Chrome，并且3)可以远程打印到您想要打印到的打印机。

注意：在继续之前，我们强烈建议您进入计算机上的“设备和打印机”菜单，并删除所有旧打印机。

例如，我们安装了几台不再在我们的家庭办公室使用的幻影打印机。如果你不删除它们，它们最终会在你的Google Cloud打印账户中被列为无用的杂物。

单击Chrome界面右上角的菜单按钮，然后选择设置。在设置菜单中，向下滚动到底部，然后单击“显示高级选项”。当显示高级选项时，向下滚动直到您看到“Google Cloud Print”。

如果您没有在Chrome中登录您的Google帐户，该按钮将显示“Sign in to Google Cloud Print”(登录到Google Cloud Print)。如果您已经登录(例如，您已经启用了此Chrome安装的同步)，该按钮将显示为“添加打印机”。如有必要，请登录，然后单击“添加打印机”；此外，请花些时间确保“继续运行后台应用程序…”。在System部分下面选中，以确保您的Cloud Printer服务器即使在您不活跃地使用Chrome时也保持活动状态。

点击“Add Printers”(添加打印机)后，您将看到一个确认屏幕，指示您实际上不需要执行此步骤即可使用云打印机服务。没错，你可以严格使用Cloud Printer将PDF文件打印到你的Google账户上--这是一个很棒的功能，但这不是我们今天的目标。我们的目标是从我们的打印机获得物理打印输出。

继续，单击蓝色的“添加打印机”按钮继续。此步骤会自动将您计算机上可用的所有打印机添加到您的Google Cloud打印帐户。重要的是，只能在一台可以访问您的打印机的计算机上执行此操作-否则，您的帐户中将列出一堆重复的打印机。

添加完所有打印机后，您可以通过访问此链接来查看/管理它们：

在上面的屏幕截图中有两件事值得注意。第一，“共享”按钮，第二，“归我所有”标记，旁边是我们刚刚添加到系统中的所有打印机。云打印使人们可以像在Google Docs中共享文档一样轻松地共享打印机。无需设置某种复杂的联网安排，您只需授予您朋友的Google Cloud打印帐户使用您的打印机的权限即可。用户与您共享的打印机将显示在此处，但会标记为“Owner by[Friend‘s Name Here Here]”。

如何打印到我的云打印机？

既然您的打印机启用了云，我们就必须处理最重要的业务：将文档从您的设备传输到您的打印机。

如果您使用任何类型的Google App进行打印，那么生活将变得再简单不过了。无论你是在Android上使用Google Drive应用程序，还是在iOS上使用Gmail，还是刚刚从远程电脑登录到Google Docs的网络界面，都可以直接从Google Apps打印到Google Cloud Printer服务。例如，以下是我通过Android手机将女儿的家务清单从Google Drive发送到楼上打印机的屏幕截图--不需要特殊的附加软件或调整：

我所要做的就是打开文档，单击右上角的菜单，选择“打印”，选择打印机，然后在单击“打印”之前查看打印选项。当我走上楼梯去取回打印件时，它已经放在托盘里了。

如果您不是直接从Google App或Google Chrome实例打印，并且打开了帐户同步，那么您将需要某种辅助应用程序。有适用于Android、iOS、Windows和OSX的助手应用程序，它们将打印功能从Chrome/Google Apps扩展到系统上任何具有打印功能的应用程序(或者作为一种移动打印平台，您可以从设备打开文档并将其发送到云打印机)。

您可以查看Google Cloud打印支持的应用程序部分中的应用程序列表，然后获取适合您系统的应用程序。如果你好奇这些应用程序是如何工作的，你可以在这里查看我们的教程，演示如何在Android上使用第三方云打印应用程序。*更新：现在有了适用于Android的官方Google云打印应用程序。

一旦您完成了简单的配置过程，云打印系统的潜力就是巨大的。您可以与经常需要向您发送物理文档的朋友或同事共享一台打印机，您可以在父母家中设置共享照片打印机，这样您和您的兄弟姐妹就可以自动发送孙子孙女的照片，至少，您再也不用费力尝试将文件从您的移动设备传输到您的远程打印机上了。
