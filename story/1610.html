配置您的计算机以备份到Windows Home Server

Windows Home Server的一个很酷的功能是能够将网络上其他计算机的备份设置到服务器上。今天，我们来看看在您的网络上配置一台计算机以自动备份到WHS的过程。

备份到WHS

要备份网络中的计算机，请打开Windows Home Server控制台，然后选择计算机&备份。右键单击要备份的计算机，然后选择配置备份。

备份配置向导启动…

配置向导正在收集信息…，请稍候

选择要备份…的磁盘。请注意，如果您希望备份外部驱动器，也可以选择外部驱动器。请注意，磁盘需要格式化为NTFS，如果没有格式化，则不会显示在列表中。

现在选择要从备份中排除的文件夹。灰显的位置将自动排除，但您可能希望将其从排除列表中删除。如果要排除其他文件夹，请单击“添加”按钮。

现在浏览并选择要从备份中排除的文件夹。要减小备份的大小，您可以排除大型媒体文件和不重要的文档、图片和…。确保不排除最重要的文档、图片和其他数据。

完成从备份中排除文件夹后，单击“下一步”继续。

然后，您将看到“恭喜”屏幕，其中显示备份大小、将发生的备份时间和每日备份频率的摘要。

备份过程完成后，您将能够在WHS控制台中查看计算机是否已备份。

如果要更改备份发生的时间，请单击WHS控制台中的设置，然后单击备份。在“备份时间”下，您可以将开始时间和结束时间更改为适合您的日程安排。

要查看备份，只需右键单击计算机并选择查看备份即可。

它将打开查看备份窗口，您可以在其中验证和管理计算机的备份。

结束 / 结尾 / 结论 / 推论

这样您就可以开始备份了，知道您的计算机数据正在备份到服务器上，您就可以高枕无忧了。该过程相对简单，主要是确保备份不会不必要地包含不必要的文件(如大型多媒体文件)。您可以对备份进行更多的管理，我们将在未来的帖子…中详细介绍如何管理它们。所以敬请关注。请记住，您可以下载Windows Home Server的30天免费试用版，因此，如果您有一台额外的计算机可以专门用作服务器，您可能需要检查一下。
