在Windows上创建符号链接(也称为Symlink)的完整指南

Windows 10、8、7和Vista都支持指向系统上的文件或文件夹的符号链接(也称为符号链接)。可以使用命令提示符或名为Link Shell Extension的第三方工具创建它们。

什么是符号链接？

符号链接基本上是高级快捷方式。创建指向单个文件或文件夹的符号链接，该链接看起来与指向Windows的文件或文件夹相同-即使它只是指向文件或文件夹的链接。

例如，假设您有一个程序需要其位于C：\Program的文件。您确实希望将此目录存储在D：\Stuff，但是该程序要求其文件位于C：\Program。您可以将原始目录从C：\Program移动到D：\Stuff，然后在C：\Program上创建一个指向D：\Stuff的符号链接。当您重新启动该程序时，它将尝试访问其位于C：\Program的目录。Windows会自动将其重定向到D：\Stuff，一切都会像在C：\Program中一样工作。

此技巧可用于各种用途，包括将任何文件夹与Dropbox、Google Drive和OneDrive等程序同步。

有两种类型的符号链接：硬链接和软链接。软符号链接的工作方式类似于标准快捷方式。当您打开指向某个文件夹的软链接时，您将被重定向到存储文件的文件夹。但是，硬链接会使其看起来好像该文件或文件夹实际上存在于符号链接所在的位置，您的应用程序不会知道更多信息。这使得硬符号链接在大多数情况下更有用。

请注意，Windows实际上并不使用术语“硬链接”和“软链接”。相反，它使用术语“硬链接”和“符号链接”。在Windows文档中，“符号链接”等同于“软链接”。但是，mklink命令既可以创建硬链接(在Windows中称为“硬链接”)，也可以创建软链接(在Windows中称为“符号链接”)。

如何使用mklink创建符号链接

您可以以管理员身份在命令提示符窗口中使用mklink命令创建符号链接。要打开命令提示符，请在“开始”菜单中找到“命令提示符”快捷方式，右键单击它，然后选择“以管理员身份运行”。

在Windows10的创建者更新中，您可以使用普通的命令提示符窗口，而无需以管理员身份运行。但是，要在没有管理员命令提示符窗口的情况下执行此操作，您必须首先从设置>更新和安全>为开发人员启用开发人员模式。

在没有任何额外选项的情况下，mklink创建指向文件的符号链接。下面的命令在指向文件Target的链接上创建一个符号链接或“软”链接：

mklink Link Target

如果要创建指向目录的软链接，请使用/D。如下所示：

mklink /D Link Target

如果要创建指向文件的硬链接，请使用/H：

mklink /H Link Target

使用/J创建指向目录的硬链接，也称为目录连接：

mklink /J Link Target

因此，例如，如果要在C：\LinkToFolder创建指向C：\Users\Name\OriginalFolder的硬链接，请运行以下命令：

mklink /J C:\LinkToFolder C:\Users\Name\OriginalFolder

您需要用空格将路径括起来。例如，如果将文件夹命名为C：\link to Folder和C：\Users\Name\Original Folder，则应改用以下命令：

mklink /J "C:\Link To Folder" "C:\Users\Name\Original Folder"

如果您看到消息“您没有足够的权限执行此操作。”，则需要在运行命令之前以管理员身份启动命令提示符。

如何使用图形工具创建符号链接

如果您喜欢使用图形工具完成此操作，请下载Link Shell扩展。确保在工具本身之前下载适当的必备软件包-两者都链接到工具的下载页面。

安装后，找到要创建链接的文件或文件夹，右键单击，然后在菜单中选择“选取链接源”。

然后，您可以右键单击不同的文件夹，指向“放置为”菜单，然后选择“硬链接”创建指向文件的硬链接，选择“连接”创建指向目录的硬链接，或选择“符号链接”创建指向文件或目录的软链接。

如何删除符号链接

要删除符号链接，只需像删除任何其他文件或目录一样将其删除即可。只需小心删除链接本身，而不是它链接到的文件或目录。
