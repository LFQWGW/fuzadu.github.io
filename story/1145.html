适用于Ubuntu Linux的Avant Window Navigator

Avant window Navigator(AWN)是一个应用程序启动器和Dock，它将重新定义您的Linux体验。好的一面是它是高度可定制的，因此将非常适合你的Ubuntu主题。让我们看看如何在您的Ubuntu机器上安装和自定义AWN。

我们将通过开发者的PPA(Personal Package Archive)安装AWN的测试包。我用的是Ubuntu9.04(活泼的Jackalope)。

添加PPA存储库

1.进入PPA页面(在文章的末尾提到)，选择您的Ubuntu版本(在我的例子中，它很活泼(9.04)。

2.我们现在将Soures.list条目添加到软件源代码中，以便Ubuntu 9.04了解存储库。

3.要执行此操作，请转至“系统”>“管理”>“软件源”。

4.转到显示第三方软件的选项卡。

5.单击Add…。按钮，然后逐行添加以下行。

Deb http://ppa.launchpad.net/awn-testing/ppa/ubuntu Junty Main

deb-src http://ppa.launchpad.net/awn-testing/ppa/ubuntu Junty Main

最终结果应该如下所示(确保选中这两个复选框)。

暂时不要关闭此窗口。按照以下步骤添加与AWN关联的PPA密钥。

添加PPA密钥

在相同的AWN测试PPA页面(在帖子的末尾提到)，您将看到一行内容为签名密钥。

1.点击链接1024R/BF810CD5，进入公钥服务器页面。

2.按ALT+F2并键入Gedit打开Gedit。将代码复制到Gedit中的“--Begin PGP public key block--”下。

3.在桌面上将文件另存为PGPAWN。

4.现在我们将导入此PGP(PGPAWN)安全密钥。在Software Sources窗口中，选择“Authentication”选项卡，然后单击“Import Key File…”。找到我们保存在桌面上的PGPAWN文件并将其导入。

5.现在单击关闭按钮以关闭软件来源窗口。系统将提示您重新加载。

6.单击重新加载以更新软件源。

安装AWN-Window-Navigator和AWN-Extras

1.通过按ALT+F2并键入“gnome-Terminal”(不带引号)打开终端窗口。

2.在终端窗口中复制并粘贴以下行，以安装AWN-Navigator。

sudo apt-get install avant-Window-Navigator-Trunk

3.键入y进行安装。

类似地，我们将通过在终端窗口中粘贴以下行来安装AWN-Extras-applet。

sudo apt-get install awn-Extras-applets-trunk

启动Avant-Window-Navigator

您现在可以通过转到应用程序>附件>Avant Window Navigator来启动AWN。

配置Avant Window Navigator

单击第一个图标“AWN-Manager”以调出AWN管理器窗口。

一般偏好

1.在“常规”选项卡的“常规首选项”下，选中“登录时自动启动AWN”复选框。这将确保每次新的Ubuntu会话都会启动AWN。

2.我还会勾选“最大化窗口不覆盖条”的复选框，这样我就可以一直看到AWN了(您必须重新启动AWN才能使此设置生效)

主题AWN

Avant Window Navigator有一个不错的主题。但是，也可以选择添加更多主题。您可以从本文末尾提到的官方主题页面下载主题。

1.从主题页面下载主题。

2.打开AWN管理器，选择主题。

3.点击添加安装主题

单击应用以使用选定的主题。您可以选择最适合您桌面墙纸的主题。请随意尝试其他设置，看看您可以获得多大的创造力！
