你应该知道的七个有用的Chromebook诀窍

Chromebook与传统笔记本不同。虽然它们要简单得多，但它们仍然有您可能不知道的各种有用的功能。从访问远程电脑和打印，到擦除您的个人数据，恢复Chrome OS，再到安装桌面Linux，这些技巧将帮助您最大限度地利用Chromebook。

控制谁可以登录

相关：你能买到的最好的Chromebook，2017版

Chromebook的市场定位是“面向所有人”的笔记本电脑。默认情况下，任何拥有你笔记本电脑的人都可以拿起它，插入他们的谷歌账户，然后登录。他们将无法访问您的数据，但他们可以使用具有自己的Chrome设置的计算机。

如果要限制对Chromebook的访问，请通过单击系统托盘并选择齿轮图标打开设置菜单。从那里，向下滚动到“People”部分，然后单击Manage Other Users按钮。

在这里，您可以选择对其他用户限制该设备的程度，包括将除您之外的所有人锁在门外。这是你的Chromebook，你可以自私地使用它！

访问远程Windows、Mac和Linux桌面

您不能在Chromebook上运行Windows程序，但您可以访问远程Windows、Mac和Linux桌面。Chrome Web Store提供用于连接传统VNC服务器的VNC客户端，但Chrome实际上也有谷歌构建的远程桌面功能。您可以使用它从Chromebook访问您的台式PC，或者远程运行罕见的Windows应用程序。

要做到这一点，请在您的PC上安装Chrome中的Chrome远程桌面应用程序。然后，您可以激活“启用远程连接”选项，并使用Chrome远程桌面应用程序从Chromebook连接到您的PC。

这也不是Chrome OS独有的功能。您还可以使用Google Chrome从任何其他类型的PC远程访问Windows、Mac和Linux PC，无论您是否有Chromebook。

使用Google Cloud Print打印文档

如果你需要打印一些东西，你应该意识到，你不能直接把打印机插入Chromebook并打印到它们上。不过，您可以设置Google Cloud Print，并使用它从Chromebook远程打印到支持的打印机。

要将打印机添加到Chromebook，请跳到设置菜单并向下滚动，直到看到“显示高级设置”。点击那个。

向下滚动此菜单至“Google Cloud Print”部分，然后单击Manage按钮。

从这里，如果在网络上检测到新打印机，您可以添加新打印机-否则，将显示已经是Cloud Print一部分的设备。如果你曾经在另一台使用Chrome的PC上打印过网络打印机，那么它已经是你的Google Cloud打印的一部分了。很整洁，对吧？在Chromebook上设置打印机可能比市面上的任何其他设备都要简单。

Chrome OS还包括打印为PDF的功能，因此如果您愿意，您可以始终将文件另存为PDF，然后在另一台计算机上打印该PDF文件。

使用PowerWash擦除个人数据

Chrome OS包括一个“PowerWash”功能，功能类似于Windows 10的重置选项，执行出厂重置，将Chromebook恢复到原来的干净状态。当你打算把你的Chromebook给别人时，这是一个理想的选择，因为它会删除你所有的个人数据。可以把它想象成重新安装Windows或对平板电脑执行出厂重置。

您可以在设置屏幕上找到此选项。单击“显示高级设置”链接并向下滚动底部，在那里您将看到一个PowerWash按钮。

或者，如果你只想撤销所有的自定义设置，你可以使用“重置设置”，但可以将所有设置恢复为默认设置，而不必对Chromebook进行电源清洗。

查看本地文件

你的Chromebook不仅仅是一个网络浏览器。它还包括Files应用程序和本地文件查看器，允许您观看视频、播放音乐、阅读PDF和Microsoft Office文档、查看图像等。您可以下载所有类型的媒体文件，然后从Files应用程序打开它们。

从USB驱动器恢复Chrome操作系统

相关：如何出厂重置Chromebook(即使它无法引导)

Chromebook包括一个恢复模式，允许你在操作系统损坏时重新安装Chrome OS。也就是说，这不太可能发生，除非您在开发人员模式下胡乱操作。

要恢复您的Chrome操作系统，您需要创建恢复驱动器。你可以通过下载并运行谷歌的Chromebook Recovery Utility Windows、Mac或Linux版来做到这一点。

当你启动恢复实用程序时，它会提供一个简单的操作步骤--不过，你需要知道你的Chromebook的确切型号。你可以从恢复屏幕中找到Chromebook的型号，但也有一个链接可以从恢复实用程序的列表中选择型号。

一旦您选择了您的型号，您将在运行实用程序的PC中插入闪存驱动器或SD卡，然后从下拉菜单中选择它。最后一个警告将告诉您，该驱动器上的所有数据都将被擦除-只需单击“立即创建”即可继续操作。

此过程可能需要一些时间，具体取决于您的计算机、驱动器速度和互联网连接。按兵不动-实用程序会告诉你在整个过程中发生了什么。

要真正恢复Chrome OS，你需要按Escape+Refresh并按住电源按钮。老款Chromebook有专门的恢复按钮-你可以在谷歌网站上找到更多信息。从那里，只需插入驱动器并按照说明操作即可。

使用开发人员模式运行Desktop Linux

Chromebook允许您禁用其安全功能并启用开发人员模式。在开发者模式下，你可以随意修改Chrome OS，也可以启动其他操作系统，包括Ubuntu和其他传统的桌面Linux系统。你甚至可以与Chrome OS并排运行桌面Linux系统，通过热键在两者之间切换。
