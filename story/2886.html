如何最大化您的Linux笔记本电脑的电池寿命

笔记本电脑制造商花费大量时间调整他们的设备驱动程序以延长Windows电池续航时间。Linux通常不会得到同样的关注，Linux在相同硬件上的表现可能和Windows一样好，但电池续航时间不一定那么长。

多年来，Linux的电池使用率有了显著的提高。Linux内核变得更好了，当您使用笔记本电脑时，Linux发行版会自动调整许多设置。但是你仍然可以做一些事情来提高你的电池寿命。

基本电池节约提示

在做任何太复杂的事情之前，请调整与Windows笔记本或MacBook相同的设置，以最大限度地延长电池续航时间。

例如，告诉你的Linux笔记本电脑挂起-这就是Linux所说的睡眠模式-在你不使用它的时候更快。您可以在Linux桌面的设置中找到此选项。例如，进入系统设置>开机Ubuntu桌面。

屏幕亮度会显著影响电池寿命。显示器背光越亮，电池寿命就越差。如果你的笔记本电脑有改变屏幕亮度的热键，试试吧--它们也有望在Linux上运行。如果没有，您会在Linux桌面的设置中找到此选项。它可以在Ubuntu上的系统设置>亮度和锁定中找到。

您还可以告诉您的Linux桌面在屏幕处于非活动状态时更快地将其关闭。当笔记本电脑的屏幕关闭时，它将耗电较少。不要使用屏幕保护程序，因为屏幕保护程序只会让你的计算机做更多的工作，并使显示器保持打开状态，这只会浪费电力。

您还可以禁用不使用的硬件无线电。例如，如果您不使用蓝牙，您可以禁用它以获得更长的电池续航时间。转到系统设置>蓝牙，在Ubuntu桌面上禁用蓝牙。

如果你不使用Wi-Fi，你也可以通过禁用它来节省一点电量。在Ubuntu上，进入系统设置>网络并启用“飞机模式”以禁用Wi-Fi和其他无线收音机。

记住，你用笔记本电脑做什么也很重要。运行更重的软件和使用更多的CPU资源会导致您的笔记本电脑使用更多的电池电量。出于这个原因，您可能希望考虑更轻量级的桌面环境，比如基于Lxde的Lubuntu，而不是基于Unity的主Ubuntu桌面。

安装专有显卡驱动程序(如果需要)

如果您的笔记本电脑集成了英特尔显卡，恭喜您。您不需要担心图形驱动程序的电源管理问题。英特尔显卡并不是速度最快的，但它们拥有出色的开源驱动程序支持，并且“开箱即用”。

但是，如果您的笔记本电脑具有NVIDIA或AMD显卡，您可能需要做一些工作来降低功耗。

最糟糕的情况是一台配备NVIDIA Optimus或AMD可切换显卡的笔记本电脑。这样的笔记本电脑将有两个不同的GPU。例如，NVIDIA Optimus笔记本电脑将同时配备功能更强大、耗电量更大的NVIDIA GPU和功能较弱、电池友好的英特尔GPU。在适当支持英特尔显卡的Windows上，笔记本电脑设计为使用英特尔显卡，直到您玩游戏时才使用NVIDIA显卡。

当您在NVIDIA Optimus笔记本电脑上安装Linux发行版时，默认情况下，您的笔记本电脑将始终使用NVIDIA图形，从而耗尽您的电池。您需要安装NVIDIA的Linux驱动程序并设置Optimus(在Ubuntu上查找NVIDIA-PRIME软件包)才能正常工作。在某些笔记本电脑上，您还可以进入BIOS或UEFI固件设置屏幕并禁用您的独立GPU，以强制Linux仅使用板载显卡，而不进行任何额外的调整。

即使您没有双GPU、可切换显卡设置，您也可以从安装专有的NVIDIA或AMD显卡驱动程序中获益。它们可能允许访问标准开源驱动程序中不起作用的省电功能。

检查您的电池是否需要更换

如果您正在为电池续航时间而苦苦挣扎，那么您可能需要更换笔记本电脑的电池。随着时间的推移，所有电池都会变质，所能容纳的电量会逐渐低于出厂时的电量。

例如，在Ubuntu上，您可以从Dash打开Power Statistics应用程序。看看“笔记本电脑电池”部分。“充满时的能量”是电池充满电后目前能储存的电量。“能量(设计)”是指电池充满电后最初能储存的电量。

将“充满时的能量”除以“能量(设计)”，将结果乘以100，你会得到一个百分比。例如，在下面的屏幕截图中，我们将执行以下数学运算：

(44.8 / 54.3) * 100 = 82.5%

这意味着电池目前可容纳原始容量的82.5%。那还不算太糟。除非你刚买了一台新笔记本电脑，否则你不会达到100%的状态。但如果电池电量很低--比如低于50%--而且你的电池用完时间不多，你可能只需要更换电池。

如果您的Linux发行版上没有Power Statistics应用程序，您可以通过几个简单的终端命令获取此信息。

打开终端窗口并运行以下命令：

cat /sys/class/power_supply/BAT0/charge_full cat /sys/class/power_supply/BAT0/charge_full_design

将第一个数字除以第二个数字，再用倍数除以100，即可得到电池原始容量的百分比。例如，对于下面的屏幕截图，我们将执行以下数学运算：

(5901000 / 7150000) * 100 = 82.5%

这意味着电池目前的出厂容量是原来的82.5%。

高级节电实用程序

这就是所有容易摘到的果实。有许多低级别的调整可以稍微影响电池寿命，但它们通常需要做大量的工作，但回报不会太大。有各种各样的工具承诺帮助你延长电池续航时间，但与几年前相比，它们的用处已经不那么大了。Linux发行版会自动配置各种设置，以便在笔记本电脑上正常运行。

英特尔的开源PowerTOP实用程序将检查您的系统，查看各种节能功能的启用情况，甚至还会提供一些如何降低系统功耗的建议。它是一个命令行工具，所以您需要从那里运行它。您通常可以在Linux发行版的软件存储库中找到它。

例如，要在Ubuntu上安装和运行PowerTOP，您需要打开一个终端窗口并运行以下命令：

sudo apt install powertop sudo powertop --calibrate

如果你真的在苦苦挣扎，需要更长的电池续航时间，你可以安装TLP。它的设计是一套激进的电池续航时间调整方案。它也可以在Ubuntu的软件库中找到。我只需要安装它并重新启动系统-就是这样。TLP在引导时自动启动，并启用其默认的省电调整。

例如，TLP将更积极地挂起USB设备，停住硬盘驱动器的头，并限制CPU。如果你的Linux笔记本电脑已经有了稳定的电池续航时间，这些调整可能不是理想的调整，但如果你正在努力从系统中挤出更多的电池时间，这些调整可能会有帮助。

例如，要在Ubuntu上安装TLP，您需要运行：

sudo apt install tlp

然后，您可以重新启动系统，TLP将在每次引导时自动启动。要避免立即重新启动，您可以通过运行以下命令来启动它：

sudo tlp start

如果您对笔记本电脑的电池续航时间感到满意，那么您可能不应该使用TLP。但这是一个很好的最后选择，它胜过手动启用所有这些激进的调整。还有其他类似TLP的工具，但一次只能使用一种。他们在引擎盖下改变了大部分相同的设置。
