如何设置Plex(以及在任何设备上观看电影)

Plex Media Server是一种用户友好的方式，可以将您所有的电影、节目和其他媒体存储在一个位置-并使其可以从任何设备访问，无论您是在家里还是在旅途中。*如果您正在寻找一种不令人头疼的方式在任何地方观看电影，这就是它。

什么是Plex Media Server？

这是一个很常见的情况，太多的人发现自己处于这样的境地。你有媒体文件-音乐、电影、电视节目，甚至照片-但你没有一种简单的方式在你丰富多样的设备上轻松访问它们。

你会把你的蓝光刻录拷贝到你的笔记本电脑上吗？那你的手机呢--你会把文件转换成更小更兼容的文件吗？你会通过桌面和Chromecast在电视上播放节目吗？你会把你所有的照片上传到你的iCloud账户上，然后通过Apple TV发送到你的电视上吗？那你的音乐呢？如果你离家出差，想把电视节目流传输到你的iPad上，该怎么办呢？

即使对于拥有丰富媒体管理经验的严肃极客来说，让你的所有设备都能通话并访问同一集合也是一件非常痛苦的事情。不过，这并不一定非得如此。Plex是一个集中式媒体服务器系统，它几乎消除了您在管理大型个人媒体收藏时遇到的所有问题。

Plex模型很简单：您将所有媒体放在一台安装了Plex服务器软件的计算机上，然后在所有其他设备上安装Plex on。你可以将它安装在Windows、Linux或Mac电脑以及Android和iOS上，Xbox和PlayStation等视频游戏机上，你的Apple TV上，甚至还有内置Plex的智能电视。然后，您可以从这些设备中的任何一个远程访问您的整个媒体库，并毫不费力地观看它。

此外，因为Plex模型是服务器/客户端模型，所以您不必担心将文件复制到各种设备以及处理回放质量、视频分辨率和其他细节等问题。您的Plex服务器将对您的所有媒体内容进行动态代码转换，这样它就可以在您使用的任何设备上正常播放。想在后院用iPhone看电视节目吗？没问题。想在公婆家的笔记本电脑上看吗？也没问题。想要将您的新智能电视直接连接到您的媒体收藏中，而不需要任何额外的硬件吗？买一台合适的电视，也没问题。简而言之，运行Plex就像运行您自己的私有和精致的流媒体服务，在那里您是一个管理图书馆的人。

听起来很棒，是吧？您不仅可以将点播流传输到您的所有设备，还可以获得绝对漂亮的元数据和艺术随之而来。

让我们仔细看看您需要开始做些什么，然后开始为Plex准备您的媒体收藏，安装Plex，以及-每个人最喜欢的部分-访问您的Plex媒体收藏。

您需要的是什么

Plex是一款非常精美的产品，一旦你启动并运行它，它几乎是最令人头疼的。然而，最重要的是进入体验时，要清楚地了解你需要什么，以及你的Plex设置的所有部分是如何协同工作的。

主页是服务器所在的位置

首先，您需要一台存储所有文件并运行Plex Media Server的计算机。您可以在Windows、Linux、OS X上安装Plex，甚至可以在FreeNAS等专用服务器软件和Synology操作系统等NAS硬件上安装Plex(您可以在此处查看媒体服务器应用程序支持的所有平台)。但无论你选择哪种平台，它都必须是一台始终开机的计算机。如果当你访问它时，内容是离线的，那么为你所有的个人媒体需求提供一个全面的流媒体解决方案是没有意义的。

除了始终处于打开状态之外，您还会希望服务器计算机具有相当大的处理能力来处理上述代码转换。您希望同时查看的用户越多，您就越想要更好的硬件。Plex媒体服务器仍然可以在较旧的硬件上工作，但如果硬件不足，它会自动禁用代码转换，并且在非常旧或动力不足的硬件上播放会受到影响并出现卡顿。

所以，你会想要一个尽可能强大的CPU。Plex推荐至少使用Intel i3处理器(或同等处理器)或更好的处理器，内存至少为2 GB(RAM对Plex来说并不是特别重要)。您可以在这里阅读他们的硬件建议。

最后，您需要很大的硬盘空间-足以存储您拥有的所有电影、电视节目、音乐和照片。

如果你周围已经有一些你不用的硬件，一定要试一试。最糟糕的情况是，您发现文件播放效果不令人满意。最好的情况是，您发现旧硬件运行良好，并且无需购买任何新设备。

您可以在此处查看服务器软件的可用平台。

每台设备都有一个客户端

服务器只占Plex系统的一半。另一半是“客户端”应用程序，也就是你用来进行所有观看的应用程序。虽然你可以从服务器的基于网络的控制面板上播放媒体，但这就像在你的网络浏览器中观看Netflix一样-大多数人更喜欢坐在客厅里或在移动设备上观看。为此，您需要Plex客户端来访问服务器。

你可以找到适用于你能想到的几乎所有平台的Plex应用程序：Android、iOS、Windows phone、Apple TV、Roku等等。关于Plex，有一件事一直是人们长期困惑的根源，那就是它是否免费-而这种困惑在很大程度上取决于移动应用程序是否有“激活费”这一事实。

Plex媒体服务器软件一直是免费的。大多数客户端应用程序一直是免费的。*一些客户端应用程序象征性地收取几美元的一次性激活费(例如iOS Plex客户端应用程序的价格为4.99美元)。

Plex有两种方式来处理付费应用程序。如果您只需要一个应用程序，您可能希望只激活该应用程序。另一种选择是购买Plex Pass，这就像是一项订阅服务，让你们两人都可以访问所有付费应用程序，以及同步到移动设备进行离线访问和基于云的文件存储等好处。如果你需要跨越多个平台的许多应用程序，并且你想要高级功能，你可以考虑每月4.99美元的通行证订阅，或者149.99美元的终身通行证。

你可以在这里阅读更多关于哪些应用程序是付费的，哪些是免费的，以及免费的Plex会员和高级会员之间的区别。要检查平台可用性并下载适用于您平台的客户端应用程序，请查看此处的Plex下载页面。

既然我们已经了解了为Plex服务器选择设备的一般指导原则，以及如何获得客户端软件，那么让我们来看一下组织媒体的相当重要的一步。

第一步：整合和组织您的媒体

如果您的所有媒体内容都组织良好且位于同一位置，则Plex效果最佳。为此，您应该将所有媒体放在安装Plex服务器软件的同一设备上-无论是旧台式计算机、地下室的专用存储服务器还是NAS设备，您的所有媒体都应该放在上面。

此外，您希望将您的媒体组织到一个干净的文件夹结构中，这样可以将主要的媒体类型分开，便于您和Plex进行解析。

如果您已经投入了任何时间来组织您的媒体，特别是如果您过去使用过XBMC/KODI这样的媒体中心软件，那么您很有可能已经有了一个完美的(或近乎完美的)文件夹结构。

下面是一个简单的Plex最佳实践目录结构示例：

/Media/ /Movies/ /MovieName (Year)/ MovieName (Year).ext /Music/ /ArtistName - AlbumName/ Track# - TrackName.ext /TV Shows/ /ShowName/ /Season 01/ ShowName - s01e01.ext /Photos/ /Album Name/ Image.ext

在上面的目录结构中，您可以看到主要的媒体类别被分成不同的子文件夹(如电影和电视节目)，并且每种媒体类型都有非常简单的命名约定。电影放在以电影命名的文件夹中；最佳做法是将年份放在括号中，以减少混淆。音乐以直截了当的艺术家名称/专辑名称格式组织。电视节目按名称、季进行组织，剧集使用“sXXeXX”格式的这两个统计数据进行标记。照片甚至更简单--Plex只需从文件夹中读取相册名称并加载其中的图像即可。

虽然上面的示例覆盖了几乎99%的区域，但是您可能会发现在命名DVD、.ISO文件或其他不太常见的格式时需要一些额外的指导。如果您需要清理媒体的其他帮助，请查看此处的Plex媒体准备指南。(更新：该指南已从Plex的当前网站上消失，但我们提供的链接将把您带到该指南的存档版本。)

关于为Plex组织媒体的最后一点注意事项：如果您已经或正在使用另一个媒体中心软件(如XBMC)，请不要担心您的元数据。您可以安全地完全并行运行XBMC和Plex，而不会对元数据造成风险，因为这两个应用程序都不使用相同的元数据文件。

第二步：安装Plex Media Server

除了组织精美的媒体之外，安装的基础是Plex Media Server应用程序。对于我们今天的教程，我们将在Windows机器上安装它，但抛开细微差别不谈，安装过程在所有平台上几乎是相同的，因为大部分过程都是在基于Plex Web的控制面板中完成的。

转到下载页面，获取适用于您的平台的Plex Media Server副本。运行安装程序，放松一会儿。安装完成后，Plex将自动启动，并应为您加载Web控制面板。如果没有，请打开您的Web浏览器，然后导航到该计算机上的http://127.0.0.1:32400/。(或者，您也可以通过将127.0.0.1地址替换为您的计算机或NAS设备的本地网络IP地址，从网络上的另一台计算机访问它)。

接受用户协议后，系统将提示您登录到您的Plex帐户。如果您有一个以前在该平台上试验过的旧Plex帐户，请登录。否则，请单击“注册”链接并注册一个新帐户。

首次登录后，您将获得Plex Pass高级服务的所有功能概要。虽然我们碰巧认为这项高级服务相当不错，但我们还是不要操之过急--使用Plex一段时间来决定你是否想要投资它。关闭弹出窗口以返回到服务器设置。

第一步是命名您的服务器。默认情况下，服务器以安装它的计算机的网络名称命名。您可以将服务器名称更改为比说“爸爸办公室”更令人兴奋的名称，也可以保持不变。

接下来，是将媒体添加到我们的库中的时候了。选择“添加库”。

您可以在此添加许多不同的库类型：电影、电视节目、音乐、照片和家庭视频。

让我们从添加一些电影文件开始。选择“电影”，然后在出现的下拉菜单中，命名您的电影库并选择一种语言。对于大多数人来说，留下“电影”的默认名称就足够了。单击“下一步”。

现在是将Plex指向实际文件的时候了，选择“浏览媒体文件夹”，然后选择存放电影文件的文件夹。选择文件夹后，单击“添加库”完成该过程。

对电视节目、音乐和/或照片重复这些步骤。当然，ofo没有必要填充所有的库--如果您只将Plex用于电视节目，那么请继续并忽略所有其他库条目。

初始服务器设置的最后一步是允许远程访问您的媒体服务器，并将匿名数据发送到Plex。默认情况下，这两个选项都是选中的，我们建议您保持选中状态，除非您有迫切的理由要取消选中它们。选择“完成”。

此时，您将被踢入Plex服务器的基于Web的控制面板。根据您的机器工作速度的不同，您可能会看到“更新库…”或者，您可能已经有内容可以像这样显示。

请注意，在最初的几个小时甚至第一天，“最近添加的”部分并不是特别有用，因为它的所有内容都是最近添加到库中的。事情很快就会平静下来，最近添加的内容将在未来几天再次变得有用。

虽然你可以直接从Plex控制面板浏览所有的媒体库(甚至可以直接在Web浏览器中观看内容)，但它对签入你的媒体库更有用，而对真正享受你的内容则没什么用处。让我们看看如何通过远程客户端访问Plex。

第三步：从其他地方访问您的Plex媒体服务器

在这一点上，我们已经完成了所有的艰苦工作(甚至没有那么难)。一旦您启动并运行了实际的Plex媒体服务器，它就完全一帆风顺了。有多顺滑？由于您的整个图书馆存储在一台中央服务器上，因此您可以从任何计算机、智能手机或其他设备轻松地访问它，并获得完全相同的体验-相同的媒体、相同的元数据、相同的最近添加的列表、跟踪您看过哪些节目以及需要赶上哪些节目的相同数据库。

在你的手机/平板电脑/电视的应用程序商店中搜索该应用程序，安装它，然后启动它。同样，我们使用的是iOS应用程序，但它们本质上都是可互换的。单击“登录”并使用您在设置服务器时创建的相同凭据。

砰。您可以立即连接到您的媒体收藏。你看到我们iPhone上的屏幕是如何镜像到节目标题的吗，就是你刚才看到的媒体服务器控制面板的屏幕截图？
