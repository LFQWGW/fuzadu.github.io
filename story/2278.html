如何在Ubuntu中创建墙纸幻灯片

就像Windows7和OSX一样，Ubuntu能够创建幻灯片墙纸，这要归功于GNOME 2.28。以下是如何使用简单易用的GUI工具或肮脏的文本编辑器来控制您的墙纸幻灯片。

简单的方法

让我们首先向您展示使用名为Crebs(创建背景幻灯片)的GUI工具创建幻灯片墙纸的简单方法。Crebs在Ubuntu中很容易安装。要安装该软件，请打开终端并键入

sudo add-apt-repository ppa:crebs/ppa sudo apt-get update sudo apt-get install crebs

这将安装Crebs存储库，更新可用的软件包，然后安装Crebs。安装Crebs后，进入菜单并启动Crebs。

要创建墙纸幻灯片，只需使用“添加”按钮并选择要添加到墙纸的图像即可。您还可以拖放壁纸以更改它们的顺序。

获得所需的所有图像后，请更改下面关于幻灯片更改之间的时间量和希望过渡的时间长度的设置。按照您希望的方式设置所有设置后，在最底部键入名称，然后单击绿色复选标记以应用并保存桌面墙纸。

如果您想稍后再次应用该主题，可以在桌面上单击鼠标右键并选择“更改桌面背景”。如果你想与朋友分享墙纸幻灯片，你将不得不手动编辑XML，所以我建议只发送图片给他们，并允许他们在系统上重新创建主题。

手动方式

如果GUI不适合您，您也可以使用文本编辑器创建墙纸幻灯片。您只需创建包含以下部分的XML文件：

<starttime>部分只是说明幻灯片的开始时间。您可以让它在将来开始，也可以将其设置为过去的日期，它将立即开始。

您可以添加任意数量的<static>和<Transition>部分。<static>部分指向实际的墙纸文件以及显示每个图像的长度。<Transition>部分指定从一幅静态图像淡入下一幅静态图像所需的时间。遗憾的是，您不能将图像设置为随机显示。将XML文件保存在/usr/share/background/中，以使其可供系统上的所有用户使用。

结束 / 结尾 / 结论 / 推论

在任何运行GNOME桌面环境的Linux发行版中，使用这个简单的XML布局和一个充满图像的文件夹来创建您自己的墙纸幻灯片并不难。享受这一方便的技巧来自定义您的安装。
