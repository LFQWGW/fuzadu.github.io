全新Windows Live Mesh 2011入门

最近，微软对其基于Live Mesh/Sync云的服务犹豫不决。在介绍了Live Mesh公共测试版和一些重命名之后，它现在被称为Live Mesh 2011，我们将向您展示如何开始使用它。

Microsoft Live Mesh 2011允许您使用“云”在PC和Mac之间同步、存储和共享您的文件。如果你在测试Live Mesh Beta，你应该最近收到了微软的一封电子邮件，通知你Beta将于2011年3月31日停止工作。以下内容摘自他们的电子邮件。

请继续阅读，了解如何安装和使用最新版本的Live Mesh。

安装和设置Live Mesh 2011

Live Mesh包含在Windows Live Essentials 2011套件中(链接如下)，如果您使用的是测试版，则需要先卸载它。对于Live Essentials安装程序，需要指出的一件事是一个新的屏幕，它提供安装整个程序集，或者只安装您想要的程序。除非您出于某种原因想要安装所有内容，否则只需选择您想要安装的。

如果您正在使用测试版，请记住先卸载它，并且您需要在每台要同步的计算机上安装它。

安装完成后，您可以开始使用包括Live Mesh在内的新程序-您可能需要在安装后重新启动。

您可以在“开始”菜单的“所有程序\Windows Live”下找到它。

或者，您也可以在搜索框中键入Mesh，然后从Vista或Windows 7的结果中选择Windows Live Mesh。

第一次启动它时，系统将提示您使用您的Windows Live ID登录。如果您发现您经常使用它，您可能需要选择记住您的ID和密码并自动登录。

现在，您可以开始使用Live Mesh 2011在您的计算机和其他兼容设备之间同步数据。

在计算机之间同步文件和文件夹

首先，我们将看一看Windows计算机之间的数据同步--也支持Mac，但在撰写本文时我们还无法对其进行测试。启动Live Mesh并单击Sync a Folder超链接。

然后浏览到要同步的文件夹并单击同步按钮。

现在选择要与您的文件夹同步的计算机。

同步将启动…。您可以看到它在与您同步的每台计算机上运行。

然后，在该过程完成后，您将能够看到文件已成功同步。

Live Mesh在后台运行，并监视同步文件夹是否有任何更改。每当您将新文件或文件夹添加到同步目录时，两台计算机都将更新。您可以从任务栏上通知区域中的图标打开它。

默认情况下，网络位置无法与Live Mesh 2011同步，但它与Server 2003、2008和Windows Home Server版本1兼容。

与Live Mesh和SkyDrive同步文件夹

微软的一项经常被忽视的免费在线存储服务是SkyDrive。在新版本的Mesh中，它借用了5 GB的存储空间，你可以用来同步你的数据。

打开Live Mesh并选择同步文件夹，导航到要同步的文件夹，然后单击同步按钮。

这一次，您希望选择SkyDrive同步存储，然后单击确定，而不是单击另一台电脑。

在该过程完成后，您将看到文件正在同步的消息。

若要查看你同步到同步存储的文件夹和文件，请单击“SkyDrive同步存储”，并在出现提示时登录到你的Live帐户。

在这里，我们可以看到同步到SkyDrive的Excel Documents文件夹。单击文件夹以查看其内容。

现在，您可以访问并下载这些文件，或者直接在您的计算机上打开。不幸的是，我们找不到从SkyDrive的同步存储部分在MS Office Web Apps中打开它们的方法，这既不方便，又有点令人困惑。

你还将看到文件与之同步的其他计算机、同步状态以及SkyDrive同步存储中的空间量。

与其他计算机同步文件夹(&SkyDrive)

你实际上可以一次选择多台计算机和你的SkyDrive同步存储。例如，如果我们想要在其他计算机和SkyDrive同步存储之间同步Office Docs文件夹，请单击“选择设备”。

然后选择要同步文件夹的位置。在这里，我们将选择我们的上网本和SkyDrive同步存储。

远程连接到另一台计算机

Live Mesh 2011的另一个巧妙功能是能够远程连接到另一台计算机。例如，您可能想要连接到家里的工作计算机，反之亦然。

如果您尚未执行此操作，则需要在要远程访问的每台计算机上单击“允许远程连接到此计算机”。

例如，在我们的计算机列表中，并非所有计算机都设置为远程访问。

单击“连接到此计算机”，然后等待建立连接。

在这里，我们连接了我们的上网本-两者都运行Windows 7。现在，您可以像坐在另一台计算机前面一样访问您的另一台计算机。

从Live Mesh远程屏幕顶部的栏中，您可以控制另一台计算机的不同方面，如发送Ctrl+Alt+Del。

当您远程连接到另一台计算机，而另一人正在使用它时，他们将被锁定。

如果要显示桌面和正在执行的操作，请在“计算机名”上选择“显示我的操作”。

在计算机之间同步程序设置

您可能要做的另一件事是在计算机之间同步Internet Explorer和MS Office设置。在Live Mesh 2011中的“程序设置”下，单击要同步的每个程序旁边的“启用同步”。对要同步程序设置的每台计算机执行此操作。

对于我们的一台电脑，IE9有标准的MSN书签，我们打开了IE同步。

它很快就把另一台电脑上的书签带过来了。

这应该会让您开始使用Windows Live Mesh 2011，以便在计算机之间同步您的文件和文件夹。这项服务仍然相当笨拙，让人摸不着头脑，一些用户可能只想坚持使用Dropbox进行文件夹同步。

Live Mesh 2011可与Vista、Server2008、OSX10.5或更高版本、Server2003和Windows Home Server版本1…配合使用。不幸的是，他们遗漏了对XP的热爱。
