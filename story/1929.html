即使您的BIOS不允许，也可以从USB驱动器启动

您总是随身携带一个可靠的可引导USB闪存驱动器来解决计算机问题，但是如果PC的BIOS不允许您从USB引导怎么办？我们将向您展示如何制作可让您从USB驱动器启动的CD或软盘。

这个引导菜单和许多在USB驱动器变得便宜和普通之前创建的菜单一样，不包括从USB驱动器引导的选项。

一款名为plop Boot Manager的免费软件解决了这个问题，它提供了一个可以刻录到CD或软盘上的映像，使您可以引导到各种设备，包括USB驱动器。

把扑通的声音放到CD上

plop以zip文件的形式提供，其中包括各种文件。要将plop放到CD上，您需要该zip文件中的plpbt.iso或plpbtnomulul.iso。根据plop Boot Manager附带的自述文件，这两个磁盘映像都应该可以在大多数计算机上运行，不过如果有疑问，plpbtnomulul.iso应该可以在“所有地方”运行。

将plpbtnomulul.iso或plpbt.iso刻录到CD，然后跳到“Bootting Plot Boot Manager”部分。

将扑通声响放在软盘上

如果您的计算机已经足够老，仍然可以使用软盘驱动器，那么您需要将plpbt.img图像文件的内容放到软盘上，该图像文件位于plop的zip文件中。

为此，我们将使用一个名为RawWrite for Windows的免费软件实用程序。

我们没有那么幸运地安装了软盘驱动器，但是如果您安装了软盘驱动器，那么软盘驱动器下拉框中应该会列出软盘驱动器。选择您的软驱，然后单击“…”按钮并浏览到plpbt.img。按写入按钮将PLOP引导管理器写入软盘。

引导绘图引导管理器

要启动PLOP，您需要让CD或软盘驱动器以比硬盘驱动器更高的优先级启动。在许多情况下，特别是使用软盘时，这是默认的。

如果光驱或软驱未设置为先启动，则需要访问BIOS的启动菜单或设置菜单。执行此操作的具体步骤因BIOS而异-要获取该过程的详细描述，请搜索您的主板手册(如果您使用的是笔记本电脑，则搜索笔记本电脑手册)。

不过，通常情况下，当计算机启动时，屏幕上突出的位置会注意到一些重要的键盘笔划。在我们的示例中，它们位于屏幕底部。

按Escape键调出Boot Menu(引导菜单)。之前，我们刻录了一张带有PLOP Boot Manager的CD，因此我们将选择CD-ROM驱动器选项并按Enter键。

如果您的BIOS没有启动菜单，则需要访问设置菜单并更改启动顺序，以使软盘或CD-ROM驱动器具有比硬盘更高的优先级。通常，此设置位于设置菜单的“启动”或“高级”部分。

如果操作正确，PLOP Boot Manager将加载，并提供多个引导选项。

突出显示USB并按Enter键。

PLAP开始从USB驱动器加载。

尽管我们的BIOS没有选项，但我们现在使用USB驱动器启动，在我们的情况下，它包含一张Ubuntu Live CD！

这是一种让你的电脑从usb…启动的相当极客的方式。前提是您的计算机仍有软驱。当然，如果你的基本输入输出系统不能从通用串行总线启动，它可能有一个…。或者你真的需要更新一下。
