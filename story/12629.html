如何将Raspberg PI转变为Plex服务器

Plex Media Server非常适合存储和访问您的所有电影、电视节目和其他媒体。不幸的是，Plex Server硬件可能昂贵、耗电，或者两者兼而有之。要减少这两种账单，可以使用Plex服务器的树莓PI。

你需要知道的是

将Raspberry PI作为Plex Server运行确实有几个好处。它不会占用服务器或全尺寸PC那么大的空间。它还将使用更少的电力，即使在整天空闲的情况下也是如此。最重要的是，它比大多数其他能够作为服务器工作的硬件成本更低。

不过，也有一些缺点需要注意。Raspberry PI3的ARM处理器不支持转码。因此，当您设置您的视频时，您会希望选择MKV作为您的视频格式。这通常会绕过代码转换的需要。(几乎所有的Plex播放器都支持MKV，无需动态转码，但一些智能电视可能会出现问题。)

即使到那时，虽然你可以在本地观看标准蓝光质量的视频，但你可能无法远程观看这些视频。而且4K视频可能也不会播放得很好。另外，请记住这不是官方支持的，您需要手动更新服务器软件。

但是，一旦你考虑到这些潜在的缺陷，Raspberry PI确实可以成为一个称职的Plex媒体服务器。

开始

与使用专用的全功能PC或NVIDIA Shield作为Plex服务器相比，开始使用Raspberry PI的成本相对较低。您需要：

树莓皮3。 microSD卡。 供电，供电。 以太网线(如果可能)。 外部存储，如USB驱动器或NAS设备。 USB驱动器的供电USB集线器

或者，您可能想要考虑一个案例和一个Raspberry Pi的散热器。您需要一个监视器、键盘和鼠标才能使一切就位，但在此之后，您可以无头运行PI。

首先，您需要按照标准步骤设置Raspberry PI。最简单的做法是获取NOOBS副本以安装最新版本的Raspbian。

一旦安装了Raspbian，您将在终端中执行几乎所有其他操作。不过，您不需要熟悉终端；您可以复制并粘贴下面的命令。

配置软件

首先要做的是确保树莓PI完全是最新的。因此，请运行以下命令：

sudo apt-get update sudo apt-get upgrade

并在系统提示时提供超级用户密码。或者，您可以使用su并跳过所有sudo条目。

在完成所有更新之后，您将需要安装“apt-Transport-https”包。这允许来自HTTPS源的Plex Server。运行此命令：

sudo apt-get install apt-transport-https

下一步是添加dev2day网站(此软件的源代码)的加密密钥：

wget -O - https://dev2day.de/pms/dev2day-pms.gpg.key | apt-key add -

运行以下命令将dev2day存储库添加到软件包源列表：

echo "deb https://dev2day.de/pms/ stretch main" >> /etc/apt/sources.list.d/pms.list

现在您已经添加了dev2day存储库，是时候更新包列表了：

sudo apt-get update

现在，您已经准备好安装Plex Media Server了。只需键入以下命令并按Enter键：

sudo apt-get install plexmediaserver-installer

在Plex Server软件安装完成后，只有几个细节需要处理。

最终细节

为了避免任何权限问题，最好让Plex在PI用户下运行。键入以下命令并按Enter键：

sudo nano /etc/default/plexmediaserver.prev

您应该看到一条线，上面写着：

PLEX_MEDIA_SERVER_USER=plex

将单词“plex”更改为“pi”，然后使用Ctrl+X关闭该文件。选择保存并覆盖该文件。

为了实现可靠的远程访问，您需要静态IP。Raspberry PI重新启动后，打开终端并运行以下命令：

hostname -I

这是当前的IP地址，写下来或输入到文本编辑器中。

现在使用以下命令打开cmdline.txt文件：

sudo nano /boot/cmdline.txt

在文件底部，键入以下内容：

ip=TheIPYouWroteDown

使用您之前写下的IP。然后使用Ctrl+X关闭文件并保存。

现在使用以下命令重新启动Raspberry PI：

sudo reboot

您的Raspberry PI现在是一个正常运行的Plex服务器。您已经准备好在任何浏览器上访问plext.tv/web，以检测Plex Server并将其与您的帐户相关联。然后，您需要将Raspberry PI配置为识别外置驱动器、翻录DVD或蓝光，然后根据Plex准则命名您的媒体文件。

一旦一切就绪，您就可以在几乎任何设备上访问您的媒体收藏，从手机到游戏机。这是一个很棒的Plex Server选项，不会让银行破产。
