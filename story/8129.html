如何在Windows上从Time Machine备份还原文件

Windows通常无法读取Time Machine备份-事实上，它甚至不理解Mac在Time Machine驱动器上要求的HFS+文件系统格式。但你可以在你的Windows电脑上从你的Time Machine备份中恢复所有的文件。

这不会让你轻松恢复设置和其他应用程序，它们通常是特定于Mac的。不过，您可以从时光机备份中提取所有重要的个人文件。

将驱动器连接到您的个人电脑

相关：如何使用Time Machine备份Mac和恢复文件

无论哪种方式，第一步都是将Mac格式的Time Machine硬盘连接到您的Windows电脑上。希望您正在使用USB驱动器进行时光机备份-大多数Windows PC与迅雷不兼容。

当你将Mac格式的Time Machine硬盘连接到电脑上时，你看不到上面的文件。这是因为Windows无法理解驱动器的HFS+文件系统。你通常可以在Mac和Windows PC之间共享驱动器，因为Mac也理解常见的FAT32文件系统，但OS X坚持使用HFS+格式化Time Machine驱动器。

不要立即使用Windows文件系统格式化驱动器，否则将丢失其上的所有Time Machine备份。Windows无法读取，但您的所有文件仍在那里。

读取HFS+分区

相关：如何通过Boot Camp在Mac OS X和Windows之间共享文件

您需要能够理解HFS+文件系统的软件才能访问您的Time Machine备份文件。我们为此找到的唯一免费应用程序是HFSExplorer。不幸的是，它确实需要安装Java才能运行-我们建议您在使用完HFSExplorer后立即卸载Java，或者至少禁用Java浏览器插件以帮助保护您自己。安装甲骨文时也要注意甲骨文令人厌恶的安装程序垃圾软件。

如果你真的受不了Java，其他可能的解决方案可以包括Paragon的Windows版HFS+和微软Mediafour的MacDrive。这两个都是付费应用程序，您可能不想仅仅为了恢复一次文件而购买它们。但是，他们确实提供了时间限制的试用，这将适用于一次性恢复过程。

安装后打开HFSExplorer应用程序，点击File菜单，选择“Load file system from device”。它应该会自动为您检测合适的设备。如果没有，您可以从“检测到的设备”框中手动选择设备，直到其中一个设备正常工作。

从Time Machine备份恢复文件

在HFSExplorer中查看Mac格式的Time Machine驱动器的内容后，您将看到一个名为“Backups.backupdb”的文件夹。这是Time Machine Backups文件夹。

在它的下面，你会发现一个文件夹，里面有你的Mac的名字。这是包含来自该特定Mac的所有Time Machine备份的文件夹。在该文件夹下，您将看到以特定日期、日期和时间命名的文件夹和一个“最新”文件夹。

最新文件夹是您最新的Time Machine备份。除非您要还原旧的、已删除的文件或以前版本的文件，否则请转到最新的文件夹。

在最新的文件夹下，你可能会看到一个名为“Macintosh HD”的文件夹--这是Mac系统驱动器的机器备份时间。你可以从Mac系统恢复任何你想要的文件，但你会在Macintosh/HD/Users/Name下找到你的个人文件。

要将所有个人文件还原到Windows PC，请导航到此文件夹，选择该文件夹，然后单击解压缩按钮。HFSExplorer将从您的Time Machine驱动器中提取文件，并将它们复制到您的Windows分区。

您还可以提取单个文件或每个备份文件。例如，您可以在Time Machine备份中挖掘，以仅查找重要的文件，选择它们，然后单击Extract按钮将其解压缩。或者，您可以选择一个顶级文件夹-整个最新备份的“最新”备份或整个Time Machine备份中每个文件的“Backups.backupdb”文件夹。HFSExplorer会将目录和其中的所有内容复制到您的Windows PC上。然后，你可以使用普通的Windows工具来挖掘它们，恢复你想要的文件，并删除你不再需要的所有东西。

您可能不应该让HFSExplorer跟踪符号链接，尽管这可能会导致重复的文件。你以后随时可以把所有东西清理干净。

如果HFSExplorer由于某些原因不适用于你，你可以随时尝试上面的商业应用程序--它们的免费试用可能会让你这次不用花一分钱就能把文件从时光机硬盘上下载下来。

不，没有很好的时光机恢复界面-都是手动的。但是，即使您没有可用的Mac，您也可以获取所有这些重要的Time Machine备份文件。

如果你附近确实有一台Mac，你可以随时将Time Machine驱动器连接到Mac上，按住Option键，然后单击菜单栏上的Time Machine图标，然后选择“浏览其他备份磁盘”。然后，您可以从Windows Time Machine备份中提取重要文件，并将其复制到使用FAT32文件系统格式化的外部驱动器，这是Windows无法理解的。
