如何使用UniBeast在hackintosh上升级或安装Mac OS X Lion

它终于来了。最简单和直接的清理方式-使用USB拇指驱动器在Hackintosh上安装Mac OS X Lion。而且还有更多。继续读！

如果您还记得，我们已经向您展示了如何选择正确的硬件来构建hackintosh，如何在上面安装Mac OS X Snow Leopard，以及如何使用xMove升级到Lion。但是现在，有了一个新的和改进的指南，不是别人，正是Tonymacx86。我们过去一直在使用他的向导，而这一位是目前为止最好的一位。我们要感谢他对hackintoshing社区的贡献！

因此，按照本指南操作，您只需使用USB拇指驱动器就可以在您的hackintosh上安装MacOSX Lion。该指南名为UniBeast。遵循本指南，您将能够在您的hackintosh上全新安装Mac OS X Lion，或者升级您现有的Snow Leopard安装(这是您的个人偏好)。只有一件事需要记住。你需要访问运行MacOSX(雪豹或狮子，没关系)的系统，只需要USB驱动器的准备步骤。这个系统可以是真正的Mac，也可以是您现有的hackintosh。现在让我们开始看导游吧。由于指南本身是不言而喻的，我们将在这里跟随。你可以在tonymacx86的博客上阅读最初的指南。

对于这款产品，您需要一个运行Mac OS X的系统、Mac应用商店下载的Mac OS X Lion副本(作为应用程序下载)或Mac OS X Lion Recovery USB(可从苹果零售店和/或Apple在线商店购买)。你还需要一个U盘，最好是8 GB或更大。此外，请从tonymacx86网站的下载部分下载UniBeast包(您需要成为注册用户，所以请先注册，因为如果您遇到问题，可能还需要在论坛上发帖寻求帮助)。最后但并非最不重要的一点是，你需要耐心、宽容和热情来让事情运转起来！

首先，让我们准备好安装USB驱动器。确保手头有运行MacOSX的系统。连接USB拇指驱动器，然后从应用程序/实用程序打开Disk Utility，或者只需在Spotlight中键入Disk Utility即可。在左侧栏中选择您的USB驱动器，然后单击Partition选项卡。在“卷方案”下，选择“1分区”，并为其指定名称(例如，Installer)。确保格式设置为Mac OS Extended(日志)。接下来，单击选项按钮。选择主引导记录，然后按OK。现在您已经准备好将更改应用到USB，所以单击Apply，然后单击Partition。等待几秒钟，直到该过程完成。

现在是将安装文件复制到USB驱动器的时候了。如前所述，您可以使用Mac OS X Lion的下载副本(从Mac App Store下载)，也可以使用Lion USB拇指驱动器。无论采用哪种方式，您都应该拥有首选的安装模式，即下载的安装Mac OS X Lion App应该位于Mac的Applications文件夹中。或者，如果您使用的是Lion USB拇指驱动器，请确保在继续下一步之前将其连接到您的计算机上。那么让我们运行UniBeast。接受任何许可协议，并在到达Destination Select(目标选择)屏幕时选择您的USB拇指驱动器。

单击“下一步”后，您将看到一些选项。现在，如果您使用的是Mac App Store版本的OS X Lion，请选择“Mac App Store”选项。如果您使用的是Lion USB拇指驱动器，请选择“Apple Store”选项。但请记住，您只能选择一个选项。如果您要在笔记本电脑上安装Lion(使用正在准备的USB拇指驱动器)，请同时选择“笔记本电脑支持”选项。完成后，单击“下一步”，输入密码，然后喝杯咖啡。这将需要一段时间(15到20分钟不等)。

UniBeast安装完成后，您可以使用此USB拇指驱动器在Hackintosh上安装Lion，进行全新安装，或作为现有Snow Leopard的升级。如果您想要在hackintosh上与现有的Snow Leopard安装一起安装Lion，请确保首先使用磁盘实用程序创建一个空分区(或者如果您愿意，也可以只使用另一个空硬盘)。确保具有与《iBoot+Multibeast指南》中建议的相同的BIOS设置，此处对此进行了进一步说明。简而言之，将所有硬盘设置为SATA模式(AHCI)，启用HPET，使用少于4 GB的RAM(仅用于安装)，并确保没有额外的外围设备连接到计算机。

准备好安装了吗？我们走吧。确保您的启动顺序设置为首先从USB驱动器启动。连接USB驱动器后重新启动hackintosh。一旦您到达引导加载程序屏幕，您将看到您之前创建的安装程序分区(在这里，它被命名为“USB”)。

现在，还记得我们学到的关于引导标志的知识吗？您可能需要输入其中一些密码才能访问安装程序。幸运的是，tonymacx86已经定义了其中的几个，所以一定要使用适用于您的硬件配置的那些。

这是整个过程中唯一棘手的一步。一旦您进入安装屏幕，只需完成几个标准安装步骤，如选择语言和指定安装的目标分区/硬盘驱动器(如果您要进行全新安装)。如果您要更新现有的Snow Leopard安装，接下来的几个步骤(关于使用Disk Utility)可能不是必需的，一旦您到达Destination Select屏幕，分区将已经存在，只需选择该分区，然后开始安装。

如果是全新安装，请确保事先已准备好目标分区。如果没有，请先使用磁盘实用程序进行准备。在顶部的菜单栏上，单击实用程序>磁盘实用程序。Disk Utility加载后，选择您的硬盘，然后单击Partition(分区)选项卡。在Volume Scheme下，选择1 Partition，为其命名(Lion、Mac HD，您想要的任何名称)，并将分区格式设置为“Mac OS Extended(Journal)”。单击选项按钮，并确保选择了“GUID分区表”。如果不是，请选择它，然后按OK。完成后，在安装程序上按下一步，选择您刚刚创建的驱动器，然后立即安装Lion！

当Lion安装完成后，您需要重新启动您的hackintosh。再一次，您必须从您创建的USB引导。这一次，您的Lion分区将显示在引导加载程序屏幕上。使用箭头键选中它，然后按Enter键从它引导。您也可以考虑任何帮助您进入安装程序屏幕的引导标志。

一旦您到达桌面，祝贺您自己，您刚刚在PC上安装了Mac OS X Lion！

剩下要做的就是下载并运行MultiBeast来启用任何不起作用的设备，并安装引导加载程序(使用EasyBeast或UserDSDT)，这样您就不需要USB来引导您的hackintosh上的Lion了。多兽安装技巧和诀窍在“How-to Geek Guide to Hackintoshing”的第一部分和第二部分中有详细的解释，所以一定要看一看。
