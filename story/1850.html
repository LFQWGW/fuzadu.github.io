使用ORB在互联网上进行流媒体和电视直播

正在寻找一种方法来在互联网上流式传输您的媒体收藏吗？或者远程观看和录制电视？今天，我们将了解如何使用ORB实现所有这些功能，以及更多功能。

要求

Windows XP/Vista/7或基于英特尔的Mac，带OS X 10.5或更高版本。

1 GB RAM或更大

奔腾4 2.4 GHz或更高/AMD Athlon 3200+

宽带连接

用于流式传输和录制直播电视的电视调谐器(可选)

注：较慢的互联网连接可能会导致播放过程中出现卡顿。

安装和设置

在您的家庭计算机上下载并安装ORB。(下面的下载链接)您需要采用安装初始部分的默认值。

当我们到达安装的ORB帐户设置部分时，我们将必须输入信息并做出一些决定。选择您的语言，然后单击“下一步”。

我们需要创建用户帐户和密码。需要有效的电子邮件地址，因为我们稍后需要确认帐户。单击“下一步”。

现在您需要选择您的媒体来源。ORB将自动查找可能包含媒体文件的文件夹。

您可以单击(+)或(-)按钮添加或删除文件夹。要删除文件夹，请单击该文件夹以从列表中将其选中，然后单击减号(-)按钮。要添加文件夹，请单击加号(+)按钮并浏览该文件夹。

您可以添加本地文件夹以及来自联网计算机和USB连接存储的共享文件夹。

注意：运行ORB的主机和联网的计算机都需要运行才能远程访问共享网络文件夹。

选择完所有媒体文件后，请单击“下一步”。

ORB将继续索引您的媒体文件…

索引编制完成后，单击下一步。

ORB电视设置

注意：目前不支持将直播电视流式传输到Mac。

如果您的PC上连接了电视调谐器卡，您可以选择配置ORB来流式传输实况或录制的电视节目。单击Next(下一步)配置电视。或者，如果您不想为电视配置ORB，请选择跳过。

如果您有数字调谐卡，请键入您的邮政编码，然后单击获取列表以获取您的频道列表。从列表中选择电视提供商，然后单击下一步。如果没有，请单击跳过。

通过选中或取消选中每个通道的复选框，您可以选择或取消选择任何通道。选择自动扫描可让ORB找到更多频道或禁用没有接收的频道。完成后单击下一步。

下一步，如有必要，选择模拟提供商，然后单击Next(下一步)。

为机顶盒选择“是”或“否”，然后单击“下一步”。

就像我们对数字调谐器所做的那样，通过选中或取消选中每个频道的复选框来选择或取消选择任何频道。选择自动扫描可让ORB找到更多频道或禁用没有接收的频道。完成后单击下一步。

现在我们完成了设置。单击关闭。

远程访问您的媒体

媒体文件通过基于Web的界面访问。然而，在我们继续之前，我们需要确认我们的用户名和密码。检查您的收件箱中是否有来自ORB网络的电子邮件。单击附带的确认链接。系统将提示您输入在浏览器中选择的用户名和密码，然后单击Next。

您的帐户将得到确认。现在，我们准备好远程欣赏我们的媒体了。要开始，请将浏览器从远程计算机指向MyCast网站。(请参阅下面的链接)

输入您的凭据，然后单击登录。

登录后，您将看到MyCast主屏幕。默认情况下，您将看到一些“频道”，如电视节目指南、随机音频和照片、视频收藏夹和天气。

您可以添加、删除或自定义频道。要添加其他频道，请单击右上角的Add Channels(添加频道)…

…。并从下拉列表中选择。

要访问完整的媒体库，请单击左上角的打开应用程序，然后从其中一个选项中进行选择。

实况电视和录制电视

如果您有为ORB配置的电视调谐器卡，您将在电视/网络摄像头屏幕上看到您的节目指南。要观看或录制节目，请单击节目列表以调出详细信息框。然后单击红色按钮录制，或单击绿色按钮播放。

录制节目时，您会在节目指南的列表右上角看到一个脉动的红色图标。

如果要观看实况电视，系统可能会提示您选择媒体播放器，具体取决于您的浏览器和设置。

回放应该很快就会开始。

Windows Media Center用户注意事项

如果在您的电脑上运行Windows Media Center时尝试在ORB中流式传输实时电视，您将收到一条错误消息。单击停止MediaCenter按钮，然后重试。

音频信号 / 声音

在音频屏幕上，您可以找到按流派、艺术家和专辑编制索引的音乐文件。您可以通过单击一次，然后单击绿色的播放按钮，或只需双击来播放所选内容。

播放将在流格式的默认媒体播放器中开始。

录像 / 录像节目 / 录像带 / 影视图像

视频的工作原理与音频基本相同。单击所选内容并按绿色播放按钮，或双击视频标题。

视频播放将在流格式的默认媒体播放器中开始。

流格式

您可以在控制面板设置中更改默认流格式。要访问控制面板，请单击打开应用程序，然后选择控制面板。您也可以单击右上角的设置。

从下拉列表中选择“常规”，然后单击“流格式”选项卡。“您可以选择四个选项。”Flash、Windows Media、.SDP和.PLS。

创建播放列表

要创建播放列表，请将媒体标题拖放到右侧的播放列表工作区，或单击顶部菜单上的“添加到播放列表”。完成后单击保存。

共享您的媒体

ORB允许您通过Internet与朋友和家人共享媒体播放列表。有几种方法可以实现这一点。编译播放列表后，我们将从单击播放列表工作区底部的共享按钮开始。

系统将提示您选择共享播放列表的方法。您可以选择公开或私下共享播放列表。您可以通过链接、博客或在您的ORB公共配置文件上公开共享。*通过选择公共配置文件选项，ORB将自动为您创建一个具有类似http://public.orb.com/username的URL的配置文件页面，任何人都可以在互联网上轻松访问该页面。

私人共享选项允许您通过电子邮件邀请朋友，并要求收件人向ORB注册。

您还可以为播放列表指定自定义名称，或接受自动生成的标题。

完成后单击确定。

访问您的公共配置文件的用户将能够查看您的任何共享播放列表，并将其流式传输到他们的计算机或支持的设备。

便携式媒体设备和智能手机

ORB可以将媒体流传输到许多便携式设备和3G手机。iPhone和iPod Touch通过Safari浏览器支持流式音频。不过，视频和电视直播需要Orb Live iPhone App，Orb Live在应用商店的售价为9.99美元。

要将媒体流式传输到您的便携设备，请在移动浏览器中转到MyCast网站并登录。浏览您的媒体或播放列表。

选择并播放媒体。

播放将开始。

我们发现在Droid和iPhone上播放音乐效果都很好。然而，Droid上的视频播放还不尽如人意。视频看起来不错，但音频往往不同步。

系统托盘控制面板

默认情况下，ORB在启动时在系统托盘中运行。要访问系统托盘控制面板，请右键单击系统托盘中的ORB图标，然后选择控制面板。

使用您的ORB用户名和密码登录，然后单击OK。

在此，您可以添加或删除媒体源、添加管理帐户、更改密码等。如果您不想在启动时运行ORB，请单击常规图标。

取消选中系统启动时启动ORB旁边的复选框。

结束 / 结尾 / 结论 / 推论

这看起来可能需要很多步骤，但是让Orb启动并运行并不是非常困难。ORB可用于Windows和基于英特尔的Mac。它还支持流式传输到许多游戏机，如Wii、PS3和Xbox360。

如果您在多台计算机上运行Windows7，您可能需要查看我们关于如何使用Windows Media Player 12在互联网上流式播放音乐和视频的文章。
