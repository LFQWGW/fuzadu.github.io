如何在HP ProBook(或兼容笔记本电脑)上安装Mac OS X Lion

没有什么比构建一个hackintosh更令人满意的了，比如在非苹果的机器上安装MacOSX。虽然不像听起来那么容易，但最终的结果是值得努力的。使用特定的组件构建一台PC并在其上安装MacOSX可以为您节省数千美元，您可能会在购买一台真正的Mac时花费这些钱。现在，是时候踏入便携世界了。今天，我们将向您展示如何将HP ProBook(或任何兼容的Sandy Bridge笔记本电脑)变成95%的MacBook Pro！

为什么你应该(或不应该)做这件事？

让我们澄清一下这件事是否应该做。首先，我们都知道苹果生产的笔记本电脑非常棒。它的设计、建造质量和美学(更不用说发光的苹果了)都会让你垂涎三尺。其次，所有这些苹果笔记本电脑都捆绑了MacOSX，这(对一些人来说)是最友好、最无烦恼的操作系统。数字艺术家、音乐家、视频编辑，他们都喜欢Mac是有原因的。因此，结论是，如果硬件设计是你真正想要的，你应该买一台真正的Mac，我们完全没有阻止你这样做。但是如果你只关心操作系统(并且在口袋里省了几块钱)，你可以考虑试一试。但请记住，它的性能可能不如真正的Mac。结果各不相同，所以抱最好的希望，谨慎行事。

为什么选择HP ProBook？

如果您熟悉hackintoshing，您可能知道tonymacx86设计的hackintoshing方法是最“合法”的(因为它们不涉及盗版，而大多数其他方法都是基于盗版的)。如果您还记得，我们已经向您展示了如何使用tonymacx86方法在定制的计算机上安装Mac OS X Snow Leopard或安装Mac OS X Lion。最近，tonymacx86社区的一位名叫“BlueKing”的成员发现，惠普ProBook(尤其是4530S)是运行Mac OS X Lion或Snow Leopard最兼容的笔记本电脑之一(更不用说它也和MacBook Pro很像了)。正如你可能已经在hackintoshing basic帖子中读到的，这一切都是关于主板的，而HP ProBook似乎有一个可hackintoshing的主板。除此之外，它还必须是一个沙桥处理器。原因很简单，Sandy Bridge处理器有内置的图形支持，这意味着你可以在Mac OS X中稍微玩弄一下就获得完全的图形加速(QE/CI)，而且你不需要专用的图形卡。因此，如果您是现有的ProBook所有者，请查看您的计算机是否兼容。最后，您还需要检查一件事。如果你想在OS X上使用Wi-Fi，你的笔记本电脑应该有一个“Atheros”Wi-Fi模块。但是，如果你刚刚购买了ProBook，但没有所需的卡(你可能有Broadcom或Ra-link)，你可以从eBay上以相当便宜的价格购买Atheros卡。一定要订购正确的惠普品牌卡，部件号593127-001(只需在易趣上搜索惠普Atheros 593127就能买到)。那就把旧的拿出来，换成新的。既然你已经准备好了所有需要的物品，我们就可以出发了。

如果您有任何其他Sandy Bridge(酷睿i3或更高)笔记本电脑，您可以尝试此方法无论如何，只要确保搜索tonymacx86论坛为您的特定笔记本电脑，看看是否其他人已经尝试黑金toshing它。您还可以按照iBoot+MultiBeast方法中的说明在ProBook或任何其他笔记本电脑(而不是Lion)上安装Mac OS X Snow Leopard。以下步骤是对BlueKing的Lion安装指南的详细说明(对于雪豹安装，您也可以参考本指南)。

该程序

现在可能有两种情况。您可能已经在硬盘上安装了Windows(或任何其他操作系统)，或者硬盘上可能是空的。无论哪种方式，我们都希望它在GPT上格式化(因此，如果已经安装了Windows，您可能需要重新格式化它)。在继续之前备份您的数据。我们将使用UniBeast方法。首先准备UniBeast USB驱动器。我们有详细的指南，所以你一定要去看看。解释整件事需要时间。但简而言之，在Mac上购买OS X Lion应用程序，连接一个8 GB或更大容量的USB，将其格式化为Mac OS X Extended，从tonymacx86下载UniBeast，运行它，请务必在运行它时选择“笔记本电脑支持”，等等，瞧。

USB准备好后，将其连接到您的ProBook，然后打开它。请确保具有最佳设置，为此，最简单的方法是将您的BIOS恢复为默认设置。还要记下BIOS版本，因为我们稍后会用到它。从USB引导至安装程序(打开ProBook时按F9)，让其加载，选择语言，然后单击下一步。

然后在菜单栏上，单击实用程序>磁盘实用程序。现在，假设您想要一个双引导系统，我们将创建3个分区；一个用于Mac，一个用于存储数据，另一个用于Windows。如果您愿意，您可以创建更多(或更少)的分区。在磁盘实用程序中，单击分区选项卡。创建3个分区，分别将它们标记为Lion、Empty(或任何您想要的)和Windows，并相应地调整它们的大小(您可能创建“Windows”分区，也可能不创建“Windows”分区，这取决于您稍后是否要创建双引导系统)。单击“选项”按钮，单击“GUID分区表”，然后单击“确定”。确保将Windows分区和空分区格式化为MSDOS，并将Mac OS X分区格式化为Mac OS X Extended(日志)。单击“应用”，然后单击“分区”对驱动器进行分区。

此截图仅供参考，实际效果可能有所不同。这一张只是为了让你对它的样子有个大概的了解。您可能会注意到，“Windows”和“空”分区在这里没有标注，但是您已经明白了这一点。

完成分区后，退出磁盘实用程序。如果你想用Windows进行双启动，通常建议先安装Windows，然后再安装OS X。但既然你已经完成了安装程序的一半，反过来会需要更多的时间，所以我们会先安装OS X。继续使用安装程序。选择Mac分区作为目标(无论您将其命名为Lion还是Mac HD)，并让其安装。根据您的配置，大约需要15-30分钟。完成后，系统会通知您Mac OS X Lion已安装，并会提示您重新启动计算机。要执行此操作，请单击重新启动按钮。

当它重新启动时，您将需要再次从安装程序USB引导。这一次，你有两个选择。或者再次引导到安装程序，或者引导到您刚刚安装的OSX(这就是您需要做的)。使用箭头键选择名为Lion(或您命名的任何名称)的驱动器，然后按Enter键。几秒钟后您就会出现在桌面上。是的，您可能会在第一次启动时收到连接键盘的提示，这是一次性的过程。将USB键盘连接到笔记本电脑上，提示符将消失，然后您可以拔下键盘插头。

在你完全享受在笔记本电脑上使用MacOSX之前，还有一件事要做。由于这个hackintoshing ProBook项目是由tonymacx86论坛的一名成员发起的，他还专门为ProBook开发了一个特殊的安装程序。如果您还记得，在定制的PC上安装OS X之后运行MultiBeast是很重要的，这样您就可以确保所有设备(如声音)和其他功能(如睡眠)的运行。如果您一直在为除ProBook之外的任何Sandy Bridge笔记本电脑遵循此指南，那么您现在必须使用MultiBeast(请阅读如何使用它)。但在这种情况下，您需要的是专门设计的HP ProBook安装程序。首先，在tonymacx86论坛上注册(建议您在论坛上注册，这样如果您遇到任何问题都可以在那里寻求帮助)。注册后，请下载本文末尾附带的安装程序。我们没有授权重新分发它，所以您必须去注册并下载它。如果您要在不同的笔记本电脑上安装，请使用MultiBeast。

下一步是最简单的。下载安装程序并运行它。您需要做出几个选择。选择您的BIOS版本(您可能早些时候就已经记下来了，所以请参考它)。然后选择要安装的内容。选择您刚刚安装的OS X版本(如果是Snow Leopard，请在运行安装程序之前升级到10.6.8)。如果您使用的是i5或i7CPU，还要选择Ethernet Kext安装程序、Chimera(引导加载程序)和适当的SSDT。最后，最重要的是，选择合适的DSDT。首先，确定您的ProBook版本，无论是具有内置图形支持的版本(HD3000)、具有专用图形的版本(RADEON)，还是4730s。单击名称旁边的箭头以展开该类别下的项目。然后选择您之前记下的BIOS版本。另外，检查额外的工具，看看是否要安装其中的任何工具。下面的屏幕截图显示了所选内容在我的ProBook上的外观。

做出选择后，单击Install(安装)，输入密码，然后等待安装完成。完成后，您可以测试是否一切正常。请记住，如果需要，您可以随时再次运行安装程序。这意味着如果某些东西停止工作，您可以再次运行安装程序以重新启用它。根据大多数人的说法，HDMI输出、VGA输出、睡眠、音频、全显卡、Wi-Fi(如果你有Atheros卡)、网络摄像头，简而言之，一切都可以正常工作。tonymacx86社区现在正致力于将多点触控手势引入触摸板(尽管两个手指滑动/滚动可以完美地工作)。

现在，如果您对ProBook运行OS X的方式感到满意，您可以继续使用它。如果您希望使用Windows的双引导系统，请使用Windows DVD引导，并使用磁盘实用程序在我们指定的分区上安装Windows(您可以在Windows安装程序中将Windows分区格式化为NTFS，并在安装Windows后将空分区格式化为FAT32，以便Windows和OS X都可以在其上读/写数据)。一旦安装了Windows，你就会注意到你不能再引导到OSX。这是正常的，因为Windows已经取代了嵌合体引导加载程序。要解决此问题，请使用安装程序USB引导到OSX(从USB引导时选择Mac分区)。运行ProBook安装程序并安装Chimera(从要安装的项目列表中选择Chimera)。一旦完成，重新启动您的笔记本电脑，您现在就拥有了Chimera引导加载程序，它将为您提供引导到OSX或Windows的选项。这就对了，你有一台运行OS X和Windows的(非苹果)笔记本电脑，这两个世界都是最好的！

如果您在安装后有任何问题，请查看常见问题解答。此外，您可以前往tonymacx86论坛，在那里提出您的问题，您一定会得到答案。惠普ProBook有专门的论坛专区，所以快去看看吧。此外，避免在您的hackintosh笔记本电脑上使用默认的“软件更新”，否则它可能会破坏您的安装。
