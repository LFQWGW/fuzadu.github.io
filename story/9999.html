如何在Ubuntu 16.04上将Unity Desktop的启动程序移到屏幕底部

Ubuntu 16.04 LTS包含了期待已久的功能：你现在可以将Unity桌面的启动程序移到屏幕底部。它不再锁定在屏幕的左侧。然而，该选项需要终端命令或调整工具，因为Ubuntu的正常系统设置窗口中不提供该选项。

尽管有了这个新选项，Unity 7桌面环境仍然有一些限制。您只能将启动器栏放在屏幕的左侧或底部，而不能放在屏幕的右侧或顶部。但至少这是朝着正确方向迈出的一步。

选项一：运行终端命令

这只需要一个终端命令。不要担心-即使你以前从未使用过终端，你也可以通过快速复制和粘贴在几秒钟内完成这项工作。

首先，打开一个终端窗口。为此，请单击屏幕左上角的Ubuntu图标打开Dash，搜索“终端”，然后按Enter键。

将以下命令复制并粘贴-或在终端窗口中键入-然后按Enter键。*启动程序将立即出现在屏幕底部，因此您不必重新启动或登录并再次注销。

gsettings set com.canonical.Unity.Launcher launcher-position Bottom

Ubuntu会记住这个设置，所以您再也不用运行这个命令了。当您登录时，启动器将始终显示在屏幕底部。

要将启动器再次移回屏幕左侧，请改为运行以下命令：

gsettings set com.canonical.Unity.Launcher launcher-position Left

选项二：使用Unity调整工具

Unity调整工具是一个第三方应用程序，可以让你快速控制这个和许多其他隐藏的Unity桌面设置。Unity有相当多的选项可以配置，从图标动画到工作区行为，只有几个选项通常会在Ubuntu的用户界面中显示出来。

要安装此工具，请打开Ubuntu附带的软件应用程序，搜索“Unity调整工具”，然后安装它。

启动Unity调整工具应用程序，然后单击Unity下的“启动器”图标。

单击外观标题下位置右侧的“底部”。您也可以从此处将选项设置回“左侧”。启动器将立即切换到您选择的屏幕的任何一侧。

选项三：使用Dconf编辑器更改设置

如果您愿意，您可以使用Dconf Editor应用程序来更改此设置，而不是使用上面的gsettings终端命令。这样做没有什么真正的好处--只运行终端命令会更快。但这是一种选择。

要安装此工具，请打开软件应用程序，搜索“Dconf Editor”应用程序，然后安装它。

启动Dconf Editor应用程序，然后导航到Com>Canonical>Unity>Launcher。

点击右窗格中的“Launcher-Position”值，并将其设置为“Bottom”。您也可以从这里将其设置回“左”。

如果Ubuntu能提供一种简单的方式从默认的系统设置工具(可能在外观窗格下)控制这个选项，那就太好了，但是这个选项在发布时并没有添加到Ubuntu 16.04 Xenial Xerus中。它不太可能永远都不会被添加进来。但Ubuntu用户终于有了官方支持的方式，可以将启动器移动到屏幕底部，所以很难抱怨太多。
