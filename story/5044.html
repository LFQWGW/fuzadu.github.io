如何在Windows 8中使用Windows 7的备份工具

Windows 8有一个新的文件历史记录备份系统，可以取代Windows 7的备份工具。但是，Windows 8仍然包含Windows 7备份工具。它们对于创建完整系统映像备份特别有用。

这些工具被认为已弃用，可能不会出现在未来的Windows版本中。Microsoft希望您改用文件历史记录和刷新功能。

访问Windows 7备份工具

Windows7备份工具是隐藏的，不会出现在“备份”或类似短语的搜索中。

要访问它们，请按Windows键并搜索备份。选择Settings类别并打开Save Backup Copies with File History(使用文件历史记录保存文件的备份副本)窗口。

单击隐藏在“File History(文件历史记录)”窗口左下角的“Windows 7 File Recovery(Windows 7文件恢复)”选项。

您将看到熟悉的Windows7备份界面，现在命名为“Windows7文件恢复”。它的工作原理和你记忆中的Windows7一样，尽管微软建议你不要同时使用这两个功能。如果您已经启用了Windows 7备份计划，则无法启用文件历史记录。

要快速访问此窗口，您还可以搜索恢复，然后选择Windows 7文件恢复。

创建完整系统备份

与Windows 8备份工具不同，Windows 7文件恢复工具可用于创建完整的系统映像备份。系统映像备份是计算机当前状态的完整副本。从系统映像恢复会将所有文件、程序和设置恢复到创建映像时的状态。

要创建系统映像备份，请单击侧栏中的创建系统映像选项。

您可以将系统映像备份放在硬盘、多张DVD或网络位置上。它可能相当大，因为它是硬盘上所有文件的副本。

Windows说您不能从系统映像备份中恢复多个文件，但我们已经介绍了一种从系统映像备份中提取单个文件的方法。

恢复完整系统备份

要在将来还原完整系统备份，请打开PC设置屏幕。按Windows键+C，单击设置，然后选择更改PC设置。

选择“常规”类别并向下滚动，直到看到“高级启动”选项。单击立即重新启动按钮重新启动计算机，进入高级启动选项菜单。

在“高级选项”屏幕上，选择“疑难解答”->“高级选项”->“系统映像恢复”。您将能够选择一个系统映像并从中还原您的计算机。

如果您无法引导到Windows，则在多次尝试正常引导后，您的计算机应该会自动引导到高级启动选项屏幕。您还可以在启动时按住Shift键，从Windows 8安装光盘启动，或使用Windows 8系统修复光盘。

创建备份计划

如果您更喜欢Windows 7的备份方式和Windows 8的文件历史记录备份方式(请在此处阅读有关差异的更多信息)，您可以单击Windows 7文件恢复窗口中的Set Up Backup(设置备份)链接来创建Windows 7样式的备份计划。

该过程与在Windows 7上设置备份和还原功能相同。请注意，在启用Windows 7备份计划时无法启用文件历史记录。

Windows8的文件历史记录功能有一些限制，但这些限制是可以解决的。例如，虽然“文件历史记录”只能备份库中的文件，但您可以将计算机上的任何文件夹添加到库中以确保其已备份。使用Windows 7文件恢复的最令人信服的原因是能够创建完整系统备份映像。
