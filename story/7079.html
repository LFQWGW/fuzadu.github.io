其余设置

现在可以肯定地说，了解Windows8.1几乎完全依赖于开始屏幕环境。虽然我们在最后两节课中讲述了一些桌面方面的内容，但我们的操作假设是，尽管微软默认了传统的Windows用户，但Windows本身将继续在触控的基础上不断发展。

Windows8.1的隐私设置专门适用于Windows应用商店应用，这太糟糕了，因为实际上那里有一些我们希望在系统范围内实施的好东西。微软的做法令人鼓舞。你可以在特定的应用程序上禁用网络摄像头和麦克风，这很好，所以这是朝着更安全的在线状态迈出的很好的一步。

系统的网络设置仍然分散在启动环境和桌面之间。您可以在PC设置中执行的操作是有限的。为了真正进入并配置您的网络和连接，控制面板仍然是执行此操作的最佳位置。在很大程度上，我们将略过这一点，因为How-to Geek School最近广泛地介绍了Windows Networking。HomeGroups也是如此。

正如我们所提到的，其余设置相当简单。在花一些时间更新和恢复之前，我们将简单地跳过时间和语言。这节课对每个人都有一点帮助，所以让我们先来深入讨论一下隐私。

独处 / 隐私 / 秘密 / 隐居

我们喜欢隐私设置的存在，因为在一个全新的应用程序平台上，隐私设置清晰而有凝聚力是非常重要的。大多数时候，我们习惯于一个接一个地配置隐私设置。使用Windows 8.1，您可以在一个集中的位置处理此问题，从而使您的隐私更轻松、更有保障。

在这里，我们看到五个您可能可以快速掌握的简单领域。

重要的是要记住，这些设置是针对Windows应用商店应用的，正如我们将看到的，你不能对桌面应用执行任何操作。为此，如果您想要针对您的桌面环境调整隐私设置，则必须针对每个特定的应用程序进行调整。

全体的 / 普遍的 / 首席的 / 一般的

这里的常规设置是您可以启用或禁用的五个项目。

使用常规设置，你可以让你的应用程序访问和使用你的姓名、图片和其他帐户信息。老实说，很多应用程序在不考虑平台的情况下都会这样做，所以如果你有超级安全意识，你可以关闭这个功能，但它不太可能透露很多关于你的信息。

你可以关闭你的广告ID，这意味着你的应用程序中仍会看到广告，但它们不会根据你的Microsoft帐户进行个性化设置。

关闭SmartScreen筛选器是不明智的。SmartScreen筛选器根据数据库检查网站，如果网站是恶意的，则会将其阻止。要了解有关SmartScreen的更多信息，请查看我们的How-to Geek School课程。

如果您关闭了键入建议，则Windows不会尝试预测您正在键入的单词。这可以保持启用状态，除非您讨厌被这样的功能打扰。

最后，如果你使用一种以上的语言，那么你可以让网站在此基础上提供与当地相关的内容。在这些设置中，前两个设置对隐私最敏感。

地点 / 外景地 / 设置 / 外景拍摄场地

您现在应该熟悉基于位置的内容了。这是移动系统中的一个常见功能，允许平台向您的设备提供专门的本地内容。例如，如果你搜索玉米饼餐厅，你的设备就会知道向你显示附近的玉米饼餐厅。

Windows位置设置的好处是你可以完全关闭它，或者你可以根据每个应用程序打开或关闭它。

网络摄像头和麦克风

如果你有一台过去几年生产的笔记本电脑，那么大多数人肯定会有一个网络摄像头和一个麦克风。老实说，这些设置应该放在一页上，但是你一眼就明白了。如果你想禁止应用程序使用你的网络摄像头，那么关闭它，或者如果你想明确允许应用程序，你可以这样做。

同样，使用麦克风设置，您可以做同样的事情，完全关闭它，或允许/不允许特定的应用程序。

这非常简单明了。有这样的设置存在，而且它们位于中心位置，而且您不必单独管理每个应用程序，这很好。

其他设备

其他设备设置将显示允许您控制应用程序访问的任何设备。如果您的设备或系统上有任何此类设备，您可以在此处启用或禁用它们。

网状物 / 网状系统 / 人际网 / 网络

与早期相比，Windows网络已经走过了很长一段路，当时连拨入互联网服务提供商都可能是一件非常痛苦的事情。今天，通常只需连接到热点或无线路由器(甚至只需将您的计算机插入其中)并输入Wi-Fi密码即可。

除此之外，您还将了解网络的其他方面。很难简单地把连接到互联网看作是“联网”。事实上，将您的家用计算机联网仍然是共享文件的一种更可靠、更简单的方式。

是的，拇指驱动器和外置硬盘使文件从一台计算机传输到另一台计算机变得非常容易，但与简单地访问另一个房间(或另一个州)的计算机并在不离开椅子的情况下提取所需的文件相比，这需要付出更多的努力。

我们最近出版了一个关于Windows网络的完整系列，我们敦促您阅读这些文章，以全面掌握这些内容。然后，您可以用一个完整的系列来介绍如何保护您的Windows网络。

下面是对Windows8.1的网络PC设置的基本介绍。

连接

连接显示您当前可用的连接，并允许您单击每个连接来更改其安全设置、数据使用情况，并查看连接的属性，如SSID、MAC地址和IP地址。

这些设置，特别是“查找设备和内容”设置非常重要，您应该了解它们。在Windows8中，只需右键单击网络连接窗格中的连接并更改连接的网络可见性，但显然已将其纳入连接设置。

如果您想要快速管理连接设置，则可以使用连接设置，但为了真正管理您的网络连接，当然，您仍希望使用控制面板版本。

注意，您可能已经注意到WiFi连接下面有一个“管理已知网络”的链接，并想知道这是怎么回事。嗯，前段时间我们写了一篇关于如何在Windows8.1中忘记无线网络的文章。事实证明，这是一个费力的过程，但非常可行。值得庆幸的是，微软已经省去了忘记网络的所有劳动。最初，您只能很容易地忘记您所在范围内的无线网络，否则它们存储在注册表中，您可以从命令行删除它们，或者通过侵入注册表并在那里删除它们。

现在，您只需选择“管理已知网络”，即可看到您曾经访问过的所有无线网络。只需浏览列表并删除您不再需要的每一个即可。

这些改进表明微软正在倾听并做出改变，但正如我们所说，真正的权力仍然在控制面板中。

如果您想更多地了解您的网络连接，我们强烈建议您阅读我们关于共享文件和资源的How-to Geek School系列中有关Windows网络的更多信息。

飞机模式

任何在飞机上使用过手机或平板电脑的人都应该已经知道飞机模式是什么了。

此处的设置有些多余，因为从“网络”窗格可以更轻松地访问它们。

事实上，如果你点击“查看连接设置”，你会迅速进入PC设置，这表明如果你想禁用Wi-Fi或将你的设备设置为飞行模式，效率会低得多。

替代值 / 代理人 / 委托书 / 代用品

如果您不知道什么是代理，那么您不必担心这一部分。可以肯定地说，大多数Windows用户可能不使用代理服务器。今天，大多数代理服务器的目的是提供对Internet的受控访问，并且通常是匿名的。

例如，如果您直接连接到Internet，您的IP地址就会暴露出来，然后您在Internet上的活动就可以绑定到它。但是，如果您连接到代理服务器，您基本上可以用代理服务器的IP地址来掩蔽您的IP地址，这意味着如果许多用户连接到同一代理服务器，那么在技术上就不可能辨别谁在访问哪些内容。

Windows 8.1提供了代理设置的全新外观。如下所示，您可以将代理设置为自动检测。

如果关闭自动代理检测，则必须手动设置。注意，您可以将某些不想通过代理连接的网站列入白名单。如果您的代理服务器的请求被拒绝，或者您在访问站点的所有资源时遇到问题，这可能会很有用。

需要注意的是，如果您通过虚拟专用网络(VPN)连接到Internet，您的代理设置将不会被应用。如果您不知道VPN是什么，我们建议您查看我们关于Windows网络的How-to Geek系列中的这一课。

在我们结束之前，请注意旧的代理对话框仍然可以通过“连接”选项卡下的“Internet选项”控制面板使用。

此对话框也可以从桌面Internet Explorer中访问，从齿轮菜单中选择“Internet Options(互联网选项)”。

家庭组

我们将轻松介绍家庭组的设置，因为我们在过去的How-to Geek School系列中已经写了大量关于它们的文章。可以说，家庭组基本上是通过家庭网络共享资源的一种非常简单的方式。打开家庭组后，只需选择要共享的内容(文档、音乐、图片、视频、打印机)，即可完成。

请注意设置底部显示的密码。当你从另一台计算机连接到家庭组时，只需输入该密码，就可以访问共享计算机的资料。相反，您可以从连接的计算机共享内容，因此它确实像一个小小的家庭网络。

你可以随心所欲地设置密码，这样就不难记住了。您还可以通过将内容包含在库中来将内容添加到家庭组。我们在关于PC维护的How-to Geek系列中更多地讨论了库。

工作场所 / 车间

如果您希望从自己的个人设备访问工作场所资源，如内部网络位置和商业应用程序，您可以输入您的用户ID，然后单击“加入”。

这将完全取决于您的工作场所和IT管理员是否。如果他们对此一无所知，或者您不使用自己的设备，那么就没有必要担心这个设置。

时间和语言

您很有可能只需设置一次语言、地区、日期和时间，就再也不会弄乱它们了。但是，如果你经常出差，经常更改时区，或者去新的地区旅行，或者只是会说多种语言，那么了解这些设置是很好的。

让我们简单地分析一下每一个，并检查它们的控制面板等效项。

日期和时间

从此，您可以手动更改设备的日期和时间，也可以将其设置为自动打开。还有一些控件可以更改您的时区，并根据夏令时自动调整。

这看起来很像您可以使用日期和时间控制面板执行的操作，不过这里您可以选择将时钟添加到桌面时钟，以及选择不同的时间服务器。

在下面的PC设置中，您将看到您的日期和时间格式，以及更改它们的选项。

这与区域控制面板的用途相同，您可以在其上找到“格式”选项卡。

请注意，区域控制面板还有一个“Location”选项卡，其用途与下面讨论的区域和语言设置相同。

地区和语言

在“地区和语言电脑设置”中，您可以选择您所在的国家或地区，就像从地区控制面板上的“格式”选项卡中选择国家或地区一样。没有什么比这更重要的了。

您还可以从此处切换和/或添加语言。更改语言时，不会更改操作系统的语言。菜单、对话框、应用程序等内容，以及所有仍将以其编码语言显示的内容。如果你想让Windows以不同的语言运行，你实际上必须购买该语言的操作系统。

也就是说，您可以更改您的输入语言(键盘布局)，这意味着无论您键入什么，都将返回您正在使用的语言中的字符。

同样的事情也可以通过语言控制面板来完成。

更新和恢复

只要Windows Update存在，Windows就很容易保持更新，但备份和恢复通常是喜忧参半。在Windows8中，微软引入了名为“文件历史记录”的改进备份选项以及新的恢复选项。

它将所有这些和Windows Update整合到了PC设置中，同时保留了控制面板的版本，因此无论您是在触摸屏上还是在鼠标和键盘上，都有足够的选择。

Windows更新

Windows Update在Windows95中显示为一个单独的网站，您必须从Internet Explorer访问它。它现在是完全集成的，可以自动(推荐)下载/安装更新，而无需任何用户交互。

除了新的PC设置版本，您还可以获得久经考验的控制面板版本，它的操作和功能与Windows7中的操作和功能一样。

我们已经在前面的系列中详细介绍了Windows Update，如果您想了解有关保持计算机更新的所有信息，我们绝对建议您查看它。

文件历史记录

在Windows8之前，您只能使用备份实用程序来备份某些内容，这很麻烦，需要计划或手动运行。文件历史记录通过实施增量备份而不是定期备份改变了这一切。

这对您来说意味着，您可以将文件历史记录设置为定期备份媒体库，以便在您处理文件时，无论是音乐作品、文档还是图像，版本都会自动备份。

这样的结果意味着您有一个连续的文件版本记录，所以如果您犯了一些灾难性的错误，并且需要恢复文件的早期版本，文件历史记录可以让您轻松地做到这一点。我们已经涵盖了文件历史的操作之前的极客，所以如果你想知道所有关于它和它是如何工作的，我们建议你查看它！

痊愈 / 重新拥有 / 收回 / 回收修理

Windows8引入了一些新的恢复选项，其中包括“高级启动”选项，它将允许您恢复系统，就像您在启动较早的Windows系统时按F8键一样。

除了其他功能外，您还可以找到“刷新PC而不影响文件”和“删除所有内容并重新安装Windows”的选项。正如你在截图中看到的，不同之处在于，刷新会让你的照片、音乐、视频和其他个人文件保持不变，而“删除”基本上会擦除你的安装，并用出厂时更新的文件取代它。

虽然前两个选项几乎不需要解释，但“高级启动”选项可能需要一些更深入的检查。我们有一些关于Windows8.1恢复选项的很好的文章介绍给极客，我们建议您通读这些文章，这样您就可以知道如何在发生灾难时恢复系统。

如果您想了解有关家庭组、网络和恢复的更多信息，请花更多时间浏览这些电脑设置的其余部分，并阅读任何链接文章。
