什么是Windows RT，它与Windows 8有何不同？

Windows RT是Windows8的特别版。它运行在ARM上，在商店里你会发现它与Intel x86机器并驾齐驱，但你会惊讶地发现，Windows RT与你所知道的Windows有如此大的不同。

Windows RT是如此不同，以至于微软告诉Mozilla Windows RT“不再是Windows”。如果你想在商店里买一台Windows系统，你应该知道Windows RT和Windows8的其他版本之间的区别。

图片来源：猕猴桃Flickr

ARM与x86

Windows RT在开发中称为Windows on ARM，或WOA。它本质上是从Intelx86处理器到ARM处理器的Windows移植。x86处理器是当今标准笔记本电脑和台式机中的处理器，而绝大多数智能手机和平板电脑都使用ARM处理器。

因为它是Windows到不同体系结构的移植，所以它不支持遗留软件-也就是说，所有已经为Windows编写的软件。它之所以称为Windows RT，是因为它只支持为Windows Runtime或WinRT编写的应用程序(是的，“WinRT”指的是在这两种体系结构上运行的运行时，而“Windows RT”指的是只在ARM上运行的操作系统)。您可能知道Windows运行时应用程序是“Metro应用程序”。

图片来源：Flickr上的罗伯特·斯科布尔(Robert Scoble)

它有台式机，但…

Windows RT确实有桌面，就像其他版本的Windows 8一样。您可以启动桌面并使用Microsoft桌面应用程序，如控制面板、Windows资源管理器、Internet Explorer，甚至Microsoft Office-事实上，每个Windows RT系统都在桌面上附带了触控优化版本的Microsoft Office应用程序。不过，Windows Media Player不包括在内。

但是，您不能在Windows RT桌面上运行非Microsoft应用程序。您可能会认为这是因为桌面应用程序还没有针对ARM架构进行编译，开发人员只需要更新它们。这是真的，但微软不允许在Windows RT桌面上安装第三方应用程序。换句话说：Windows RT桌面被锁定，并且仅适用于Microsoft应用程序。您不能安装非Microsoft桌面应用程序。

第三方应用限制

Windows RT上的所有第三方应用程序都必须是Metro应用程序，并且Metro应用程序必须通过Windows应用商店并获得Microsoft的批准。这意味着Windows RT设备只能运行微软认可的应用程序，就像iPad只能运行苹果认可的应用程序一样。

这对浏览器的选择也有影响。在Windows RT设备上，Internet Explorer的Metro版本独占访问系统API，这意味着Mozilla、Google和其他公司不能为Windows RT开发自己的浏览器-Mozilla和Google都对此表示担忧。如果您使用Windows RT设备，您将使用Internet Explorer-就像您使用iPad一样，您将使用Safari。iPad的所有第三方浏览器都是Safari的外壳，就像所有Windows RT的第三方浏览器都是Internet Explorer的外壳一样。

图片来源：Flickr上的ASA Dotzler

Windows RT用于“设备”

Windows RT显然是受到iPad的启发，它的限制与苹果的类似并非巧合。微软将Windows RT电脑视为“设备”，而不是传统的PC。你在商店里买不到盒装的Windows RT--它只能预装在ARM系统上。最初，这些ARM系统将是少数几个Windows平板电脑，但没有什么能阻止Windows RT最终在笔记本电脑上运行，甚至是在配备ARM芯片组的台式电脑上运行。

Windows RT设备也将以其他方式锁定。微软规定Windows RT设备上的安全引导不是用户可配置的，因此您将无法删除Windows RT并安装Linux或其他操作系统。

图片来源：猕猴桃Flickr

缺少企业功能

Windows RT也缺乏使Windows如此成功的许多企业功能。Windows RT设备不能加入Active Directory或域，并且不支持组策略，因此不能通过现有的Windows基础结构轻松管理它们。

Windows RT还缺少各种其他功能，包括存储空间、BitLocker、远程桌面托管和加密文件系统。有关限制的完整列表，请查看微软宣布Windows8版本的帖子。

并非所有Windows平板电脑都运行Windows RT

许多Windows平板电脑--事实上，当Windows8首次亮相时，你会在商店里看到的大多数平板电脑--都不会运行Windows RT。他们将使用标准的x86英特尔处理器和Windows8的标准版本之一，这意味着他们将支持第三方桌面软件和所有标准的Windows功能。

如果你对Windows8平板电脑感兴趣，因为它支持传统的桌面软件，那么一定要买一台x86平板电脑，而不是ARM平板电脑。如果您正在寻找企业功能，您可能也希望远离Windows RT。

目前的传言显示，Windows RT的每一份可能会花费制造商大约85美元，这使得Windows RT比其他Windows8版本更贵-而且很可能使Windows RT平板电脑比iPad更贵。Windows RT平板电脑甚至可能比功能更全的Windows8x86平板电脑更贵。
