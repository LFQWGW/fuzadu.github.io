与您的Facebook好友在Docs.com上协作处理Office文档

是否希望与朋友和同事协作并共享Office文档、PowerPoint演示文稿等内容？在这里，我们将了解如何通过Docs.com与Facebook好友一起使用新的Office Web Apps。

最近，我们在SkyDrive上看到了新的Office Web Apps，我们对它所拥有的功能感到兴奋，但希望它能做得更多。标准Office Web Apps的一个主要问题是，几乎没有人每天使用Windows Live SkyDrive。这意味着如果你想与朋友在线分享文档，他们必须注册一个新账户，并弄清楚如何使用它。The Docs.com是一项新服务，消除了这种复杂性。它将Web Apps与Facebook集成在一起，让你可以与Facebook朋友协作处理文档。我们来看看如何与你的朋友和同事使用这一点。

开始

前往Docs.com站点，开始与您的Facebook好友共享Office文档。*单击右侧的登录链接，将其添加到您的Facebook。

如果您尚未登录到Facebook，请输入您的Facebook帐户信息，然后单击“登录”。

请注意，某些浏览器可能会以弹出窗口的形式阻止登录和上传窗口，因此您可能希望允许它们打开。

登录后，即可开始工作。单击添加文档链接可上载现有文档或创建新的Word、Excel或PowerPoint文件。

当您尝试上载文档时，文档可能会请求管理您的页面的权限。/单击“允许”继续。

现在，您可以上载新文档，并选择要查看和编辑的人员。您可以将文档设置为完全私有、与个人朋友共享或将其公开。

Web Apps本身的工作原理与SkyDrive上的Office Web Apps类似，后者具有相同的功能和限制。但我们注意到的一个奇怪之处是，在您的计算机上似乎没有在Office中打开Docs.com文档的选项。这意味着您需要在线完成所有编辑，或者下载副本进行单独编辑。

与SkyDrive上的Web应用程序一样，Docs.com作为Office查看器工作得很好，但编辑器可能不支持更高级的文档中的所有功能。

您可以从右侧侧边栏为您的文档添加标题，还可以与您的朋友共享文档并将其张贴在墙上。

添加好友来编辑或查看您的文档非常简单，因为当您键入以轻松选择好友时，您将看到IntelliSense样式的通知。

你可以发布关于文档的更新，也可以从侧边栏的WALL部分看到其他人对它的评论。

您可以从Docs.com页面顶部的选项卡轻松访问您自己的文档和您的朋友共享的文档。

或者，从首页上，你可以看到一些有特色的公共文档。这些可以帮助你获得伟大的想法，与你的朋友分享。

您可以在查看其他人的文档时对其进行点赞或评论，这是帮助您的朋友发现您感兴趣的公共文档的好方法。

Facebook上的文档

登录Docs.com后，您将在您的Facebook个人资料中看到一个新的可选选项卡。选择Docs链接可向您的朋友显示您的共享和公开文档。

您的朋友现在可以从您的墙上看到您的所有文档。

如前所述，当你分享了一个文档后，你可以在Facebook上发帖。这将在你的墙上发布一个漂亮的帖子，通知你的朋友，帮助他们找到你的文档。

现在，您可以通过“文档”选项卡上传文档并在Facebook上创建新文档，就像上传图片一样简单。

不幸的是，我们确实遇到了一些错误，尽管这是意料之中的，因为这是一项测试版服务。但总的来说，它运行得相当好，让我们可以很容易地在Facebook上与朋友和家人分享我们的文档。

结束 / 结尾 / 结论 / 推论

Office Web Apps是Office 2010的一个很好的补充，但通常很难将它们用于协作，因为很少有人拥有SkyDrive账户或知道如何使用它。Docs.com消除了这种复杂性，让你可以像在Facebook上共享照片和视频一样与朋友分享文档。现在，你可以使用熟悉的Office和Facebook界面，而不是学习一种新的共享文档的方式，世界各地的数百万人已经每天都在使用这种界面。

有关Web应用程序本身如何工作的更多信息，请查看我们的新Office Web应用程序屏幕截图之旅。它们在Docs.com上的工作方式几乎相同，因此您可以在此处查看每个Web应用程序中的更多功能。
