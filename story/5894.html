在Linux中挂载ISO映像

本页将解释如何在Linux中使用ISO映像文件。

必备模块

在使用此功能之前，您必须确保加载了循环模块。

sudo modprobe loop

挂载CD ISO映像

我们将使用常规挂载命令将ISO映像挂载到文件夹中，就像您使用常规驱动器所做的那样。不同之处在于，我们传递-o loop命令来指定循环模块，该模块可以处理ISO映像。

sudo mount filename.iso /media/iso -t iso9660 -o loop

当然，您应该确保提前创建了/media/iso文件夹。

装载DVD ISO映像

在挂载DVD的ISO映像时，您可能必须使用UDF类型而不是ISO。

sudo mount filename.iso /media/iso -t udf -o loop
