如何从iPad、iPhone或iPod Touch打印

如果无纸化办公还不适合你，你可以用iPad或iPhone打印。当然，你不能将打印机直接连接到你的iPad上，但有很多方法可以无线打印。

这并不意味着旧的有线USB打印机被排除在外-如果你有一台有线打印机，在连接到Windows PC或Mac后，你仍然可以从iPad或iPhone打印到它。

AirPrint

相关：如何从任何Mac或Windows PC启用AirPrint进行iOS打印

AirPrint是苹果批准的无线打印标准。它集成到苹果的iOS中，因此从iPad或iPhone很容易打印到支持AirPrint的打印机。

要使用AirPrint，您必须购买一台宣传为支持AirPrint的无线打印机。如果您对将旧打印机转换为启用AirPrint的打印机感兴趣，可以尝试将任何打印机转换为启用AirPrint的打印机。请记住，这并不是由苹果官方支持的，因此它可能不会完美地工作。

在亚马逊上快速搜索一下，就会发现支持AirPrint的打印机只需60美元就可以买到。购买其中一台打印机并完成其设置过程以连接到您的Wi-Fi网络。当您的iPad和AirPrint打印机连接到同一Wi-Fi网络时，您的iPad将自动检测打印机。当您在应用程序中选择打印选项时-例如Safari Web浏览器-您将能够从自动检测到的AirPrint打印机列表中选择并打印到这些打印机。不需要繁琐的设置过程、帐户或打印机驱动程序。

支持AirPrint的打印机不仅适用于苹果设备。它们还可以用作Wi-Fi打印机，因此您可以从Windows或Linux PC打印到它们。一些支持AirPrint的打印机也可能支持Google Cloud打印。

GoogleCloud打印

相关：如何(以及为什么)开始使用Google Cloud Print

Google Cloud Print是谷歌的无线打印解决方案。因为它需要一个谷歌账户，所以设置起来需要多一点工作，但它还有其他优势。使用Google Cloud Print，您可以通过互联网打印到您的打印机-这样，当您外出走动时，就可以打印到家里的打印机。谷歌云打印没有很好地集成到苹果的iOS中，但你可以从Chrome浏览器应用程序打印到支持谷歌云打印的打印机。

要使用Google Cloud Print，您首先需要在打印机上进行设置。如果您有一台支持Google Cloud打印的打印机，您可以直接在您的打印机上进行设置-您的打印机连接到Wi-Fi，与Google的服务器通话，并与您的Google帐户相关联。如果您使用的是较旧的打印机，则必须通过Windows PC或Mac进行设置。一旦你完成了所有的设置，你可以点击Chrome菜单按钮，点击打印，然后选择Google Cloud Print。

要在较旧的计算机上设置Google Cloud Print，请将其连接到较旧的计算机并安装Chrome浏览器。在Chrome中完成Google Cloud打印设置过程，将打印机与您的Google帐户相关联。现在，当电脑正在运行且Chrome处于打开状态时，您可以从iPad上的Chrome打印到打印机。Chrome应用程序将您的请求发送到Google的服务器，后者将请求发送到PC上的Chrome浏览器，然后再将请求发送到打印机。

如果您想要将其他类型的文档打印到支持Google Cloud打印的打印机，您可能需要尝试PrintCentral Pro。谷歌将这款付费应用程序放在与Google Cloud Print配合使用的官方应用程序列表上做广告。

制造商特定的打印应用程序

相关：无线打印说明：AirPrint、Google Cloud Print、iPrint、ePrint等

打印机制造商通常会提供自己的应用程序，这样购买无线打印机的人就可以很容易地从iPad、iPhone甚至Android设备上打印。如果您的无线打印机不支持AirPrint或Google Cloud Print，您可能希望尝试一下打印机制造商的应用程序。

打开应用商店，搜索您的打印机制造商的名称以找到合适的应用。你会发现像惠普ePrint，三星手机打印，佳能移动打印，爱普生iPrint和利盟移动打印等官方应用程序。甚至还有一些非官方的应用程序可供选择。这些应用程序通常允许您打开各种不同的文件-网页、照片、文档和其他内容-并在应用程序中打印它们。不同的应用程序有不同的功能，一些制造商的应用程序会比其他应用程序运行得更好。

就是这样，您现在可以从iPad打印网页、照片、电子邮件和其他文档。但是尽量不要打印太多--打印机墨水很贵。
