复杂度

与Chromebook一起生活：仅靠Chrome浏览器就能生存吗？

Chromebook正变得越来越受欢迎，仅2016年第一季度就售出了近200万台。但是Chromebook看起来还是有点吓人--只有Chrome浏览器你怎么生活呢？这对一台笔记本电脑来说真的足够了吗？

你怎么能只使用浏览器呢？

许多人在浏览器上花费了大部分计算机时间，而该浏览器通常是Google Chrome。对于已经花了大部分时间使用Chrome的人来说，Chromebook是一个有趣的前景。即使你大部分时间不使用浏览器，普通人在电脑上做的大部分事情都可以在浏览器中完成。

Chromebook是谷歌对未来的展望。Chrome OS辩称，我们今天认为理所当然的大部分计算机体验都是过时的、笨重的和不必要的。杀毒软件，本地安装的带有独立更新程序的应用程序，系统优化工具，充满设置的巨大控制面板(从Windows3.1开始)，兼容20年前的打印机的驱动程序，当你启动新的Windows笔记本电脑时运行的20个系统托盘程序，一个巨大的用户可见的文件系统，让你可以深入挖掘你的C：\WINDOWS\System32文件夹-所有这些都是不必要的。

对您来说，这听起来如何将取决于您如何使用您的计算机，以及您已经移动到“云”的程度。曾经有一段时间，一个拥有媒体库的极客在他们的PC上装了几个额外的硬盘，每个硬盘上都装满了电影和几季电视节目-可能是从未经授权的来源下载的，因为当时几乎没有合法的来源。现在，你只能从Netflix、Google Play、iTunes等网站流媒体电影和电视节目。为什么要费心下载、存储和备份所有这些文件呢？

曾经有一段时间，你需要通过POP3下载所有的电子邮件，并将它们存储在你的计算机上，以确保你定期备份电子邮件程序的数据，这样你就不会丢失电子邮件。现在，它们通常存储在网上，并在基于网络的客户端中访问。即使您使用本地电子邮件应用程序，您也可能正在使用IMAP访问您的电子邮件，IMAP会将您的电子邮件的主要副本存储在远程服务器上。

像Spotify、Rdio和Pandora这样的服务已经淘汰了大量的音乐收藏。Google Docs(甚至微软的Office Web应用程序)对于大多数普通用户来说已经足够好了，他们不需要Microsoft Office中的所有高级功能。微软甚至在较新版本的Windows中推出了更简单的“基于云”的应用程序，其新的现代界面。

其他Chromebook材料

Chromebook不仅仅是Chrome--它是Chrome OS。除了您可能已经熟悉的标准Chrome浏览器之外，Chrome OS还附带：

允许您使用您的Google帐户登录的登录屏幕。登录后，您的Chrome书签、扩展、应用程序和其他浏览器数据将与Chromebook同步。您可以允许任何人使用他们的Google帐户登录您的Chromebook，也可以将其限制为只允许特定的人使用。 带有可定制壁纸的桌面背景。 一个桌面任务栏，带有应用程序菜单(列出已安装的Chrome应用程序)、可固定的快捷方式和用于运行浏览器窗口的图标。 窗口管理功能，允许您在自己的专用窗口中打开应用程序，轻松调整窗口大小以占据屏幕的左右两侧，等等。 一个类似系统托盘的区域，为当前用户提供时钟、Wi-Fi菜单、电池指示灯和图标。您可以单击它快速访问设置和信息，如音量、Wi-Fi、蓝牙和电池状态。 Chrome设置页面上的新设置，允许您配置网络连接、连接到VPN、选择墙纸、调整触控板设置、控制可以登录的用户，以及将Chromebook重置为默认状态。 Files应用程序以及本地文件查看器，允许您查看图像、观看视频、播放音乐、打开文档、解压缩.zip文件等。 一些离线运行的“打包应用”。例如，它有一个使用你的网络摄像头的摄像头，一个计算器，以及一个教你关于Chromebook的帮助应用程序。Chrome OS的未来和更强大的软件可能会以更多打包应用的形式出现，就像这样。 能够运行Android应用程序。谷歌一直在放缓增加在Chrome OS设备上运行Play Store中完整的Android应用程序的选项，这极大地拓宽了该操作系统的功能范围。

Chrome本身已经比你想象的更强大了。看看这个被真正的网站使用的10个新浏览器功能的列表，看看“网络平台”到底有多强大。

Chromebook挑战

你可能认为Chromebook很酷，但那又如何呢？您可能认为需要脱机打印/使用笔记本电脑/查看本地文件/玩游戏/运行Photoshop。让我们来看看你在Chromebook上是如何做到这一点的。

打印：不幸的是，打印就是不会消失。那么，如何在Chromebook上打印呢？您可以使用Google Cloud Print。大多数新打印机都支持Google Cloud Print，因此您可以将它们连接到您的网络并轻松打印到它们。如果您的旧打印机连接到Windows或Mac计算机，则可以使用Google Cloud Print Connector功能使Google Cloud Print可以访问旧打印机。不过，不要试图把打印机直接插到Chromebook上--那是行不通的。 离线工作：Chromebook有一些离线支持。Gmail离线应用程序允许你使用你的Gmail账户，Google Calendar允许你离线使用日历，Google Docs允许你离线查看和编辑文档。亚马逊的Kindle Cloud Reader应用程序允许您离线阅读Kindle电子书。视频文件可以下载到Chromebook并在本地播放。你可以在Chrome Web Store的离线功能部分找到更多的离线应用--选择每天都在增加。 使用本地文件：Chrome OS提供了一个Files应用程序，可以让你离线下载和查看许多类型的常见文件。Files应用程序还可以让你访问你的Google Drive--购买Chromebook会给你额外的Google Drive空间(大多数在两年内提供100 GB的额外驱动器空间)。您可以将USB驱动器、外部硬盘驱动器、SD卡、数码相机和其他类型的存储设备连接到Chromebook，它们的文件将显示在此窗口中。 使用外围设备：Chrome OS支持各种外围设备，包括USB鼠标和键盘、蓝牙鼠标和键盘、具有各种连接的显示器、带典型音频插孔的耳麦、USB网络摄像头、智能手机和MP3播放器。如上所述，Chrome OS还支持USB驱动器、SD卡和数码相机等存储设备。你不能将打印机直接连接到Chromebook上，也不能使用外部CD刻录机或观看视频DVD(尽管Netflix可以工作！)。有关支持的文件类型和外围设备的更多信息，请访问Google的Chromebook帮助网站。 查找应用程序：Google提供了一个用于各种目的的Chrome应用程序列表。 玩游戏：根据你在电脑上玩游戏的方式，你可能很幸运，或者这可能是一个很大的减速带。如果您在浏览器中玩基于Web的游戏，Chrome OS允许您玩相同的基于Flash和HTML的游戏。Chrome网店里有一些流行的游戏，比如“愤怒的小鸟”和“剪断绳子”。但是，你不能安装Windows软件(或任何本地软件)，所以你不能玩电脑游戏。它们可以通过本地客户端移植到浏览器上，但很少有游戏做到这一点。 运行Windows软件：你不能在Chromebook上运行Windows软件。不过，你可以使用谷歌官方的Chrome远程桌面或Chrome Web Store中提供的VNC和Citrix应用程序来访问远程系统上运行的Windows应用程序。 使用功能强大的本地应用程序：如果您确实依赖Microsoft Office中的高级功能，或者需要Photoshop、视频编辑器或其他功能强大的桌面应用程序，那么您在Chromebook上找不到相应的应用程序。然而，让我们面对现实：大多数人实际上并不需要所有这些功能。

高级极客

如果你不是极客，你可以跳过这一部分。如果你是个极客，你会想要读这篇文章。

Chromebook实际上运行的是Linux。Chrome OS是一个运行在Linux上的精简环境--Chrome OS甚至使用了Gentoo Linux的Porage包管理系统。虽然Chromebook在出厂时处于锁定状态，以获得最大的安全性，但您可以启用“开发者模式”来对底层系统做任何您想做的事情。你可以将Ubuntu与Chrome OS一起安装，然后通过击键在Ubuntu和Chrome OS之间切换，从而获得一个功能强大的Linux环境，在Chrome OS旁边安装本地应用程序。您甚至可以用标准的Linux发行版取代Chrome OS。

这使得Chromebook成为比Windows RT设备(如微软的Surface RT)更强大、更开放的ARM设备，后者不允许安装桌面应用程序，并且有一个锁定的引导加载程序，阻止您安装Linux。

更重要的是，Chrome OS实际上有一些令人惊讶的强大软件可供使用-除了用于访问PC的Chrome远程桌面，Chrome网络商店甚至还提供了SSH客户端。您可以使用SSH和远程桌面应用程序访问从远程Linux终端到Chromebook上的Windows桌面的所有内容。

你自己试试吧

如果你对Chrome OS感兴趣，你可以在VirtualBox中使用Chrome OS。当然，当你坐下来看一台真正的Chromebook时，你不会真正获得同样的体验，就像虚拟机中的Windows与支持触控的物理机上的Windows不同--一方面，它更慢。

尽管Chromebook有所有优点，但作为第二台PC，Chromebook确实大放异彩。很多人95%的时间都可以使用Chromebook，但也有5%的时间你可能想玩Windows游戏，使用桌面应用程序，或者做其他事情。如果你能接受这些限制，你可能会放弃你的Windows PC，换一台Chromebook--但这仍然是相当激进的一步。
