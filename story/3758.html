如何在Windows中将目录列表打印或保存到文件

有时，您可能希望打印或保存目录中的文件列表。Windows没有一种简单的方式从它的界面来实现这一点，但这并不是很难实现。

打印目录列表可能不是您经常需要做的事情，但它偶尔会很有用。也许您只是想要一个快速列表，以便与另一个目录进行比较。也许你因为工作原因需要生成一份打印的清单。或者，您可能只想要一份已安装应用程序的保存列表。无论您出于什么原因，打印或保存目录列表都不太困难。我们将向您展示从命令提示符(或PowerShell)执行此操作的一种快捷方法，以及一个第三方工具，如果您必须经常执行此操作，该工具会使您的工作变得更容易。

使用Windows PowerShell打印目录列表

使用PowerShell打印或保存目录列表是一个简单、直接的过程。首先，您需要打开命令提示符，然后转到要打印其内容的目录。您可以通过以下两种方式之一完成此操作。

第一个(也是最简单的)是右键单击该文件夹，然后从上下文菜单中选择“在此处打开PowerShell窗口”命令。如果您已经打开了PowerShell窗口，也可以使用cd命令导航到该文件夹。

注意：无论您使用PowerShell还是命令提示符，此过程的工作方式都完全相同，因此请使用您觉得最舒服的任何一个。

在提示符下键入以下命令(将“filename.txt”替换为您希望为结果文件命名的任何名称)，然后按Enter键：

dir > filename.txt

无论您选择什么名称，Windows都会在同一目录中创建一个文件。

当您在记事本或您最喜欢的文本编辑器或文字处理器中打开该文件时，您将看到与在提示符下单独使用dir命令时所看到的目录清单相同的目录清单。

如果您只喜欢文件名本身的列表，可以使用/b开关修改前面的命令：

cmd /r dir /b > filename.txt

注意：此命令的cmd/r部分告诉PowerShell执行键入的命令，然后退出。如果您使用的是命令提示符，则不需要添加此命令的cmd/r部分，只需键入dir/b>filename.txt即可。

还有一个小小的奖金小费。如果您发现自己经常需要创建一个带有目录列表的文件，我们已经介绍了一个小技巧，它让您只需右键单击就可以将目录的文件列表复制到您的剪贴板上。这让事情变得更加方便，还允许您将生成的文件列表粘贴到您想要的任何类型的文档中。

使用第三方工具打印目录列表

如果您不介意安装第三方工具来完成工作，Directory List&Print允许您生成可以自定义、另存为文件或打印的目录列表，从而使事情变得更加简单。

免费版本的Directory List&Print可能会做您需要的一切，特别是如果您只是在寻找一种打印单个目录中的基本文件列表的方法。如果你需要更强大的功能，专业版($22)增加了包括大量元数据和Windows文件属性的能力，指定子目录的递归深度，提供额外的排序功能，等等。

Directory List&Print既有可安装的应用程序，也有便携应用程序，因此请选择适合您的应用程序。

使用这款应用相当简单。在“目录”选项卡上，选择要列出其内容的目录。您可以从分层文件夹视图或收藏夹列表中进行选择。

在“列”选项卡上，选择要在左侧列表中显示的列，然后单击“添加”按钮(右箭头)以添加这些列。使用左侧的向上和向下箭头调整列的位置。并随时单击“创建/更新文件列表”按钮，查看您的列表进展如何。

从“Display”和“Filter”选项卡中选择您想要的任何高级选项(我们会将这些选项的详细信息发送到应用程序的帮助文件)，然后在“Output”选项卡上，选择如何生成您的列表。您可以将其打印、复制到剪贴板或以多种常用格式保存。

Directory List&Print的另一个特别方便的功能是，您可以在文件夹的上下文菜单中添加一个选项，允许您在应用程序中快速打开该文件夹。为此，您必须首先以管理员身份运行目录列表和打印。右键单击.exe文件并选择“以管理员身份运行”选项。

应用程序加载后，打开“Setup”菜单，然后选择“Add to Directory Context Menu”选项。

现在，要生成一个快速列表，只需右键单击一个文件夹，然后从上下文菜单中选择“Open in Directory list+Print”命令即可。

还可以将目录从Windows资源管理器拖放到程序窗口上，以快速生成该目录的列表。
