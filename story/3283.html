Hackintoshing极客指南-第3部分：升级到Lion和双引导

如果您一直遵循我们的Hackintoshing指南，您可能知道基本知识，以及如何在您的定制PC上安装雪豹。在本指南中，我们将尝试将您的Snow Leopard Hackintosh升级到Mac OS X Lion，并解释一些有关双引导您的Hackintosh与Windows的事情。

我们从基础开始&选择兼容程度最高的硬件，然后向您展示了雪豹的安装。因此，如果雪豹(v10.6.7)非常适合你，你可以考虑升级到Mac OS X Lion。现在要注意的是，升级到Lion完全是你自己的选择。如果你感到满意，那就好好享受你的哈金托什吧。但是，如果您想要升级，请确保先执行备份。您可以使用备份/映像程序(如“SuperDuper”或“CCC”(复写克隆人))来映像您正在工作的Snow Leopard硬盘。这只是一个预防措施，所以如果出现任何问题，Lion无法安装在您的Hackintosh上，您可以很容易地恢复到以前的工作状态。您也可以使用最近发布的Tospacx86的rBoot救援CD。在你想象你的hackintosh之前，要确保你的一切都工作得很完美。

因此，在升级之前，您必须下载必要的内容。首先，访问tonymacx86的下载部分，下载xMove。您还需要最重要的下载-MacOSX Lion。它在Mac应用商店的售价为29美元，一旦购买，你就可以从你的Apple ID在任何一台Mac电脑上下载它。注意下载的大小，这是一个巨大的3.5 GB。因此，如果你的互联网连接速度很慢，你应该有很大的耐心。如果你不能从Mac应用商店下载，你可以使用苹果现在分发的Mac OS X Lion Recovery USB驱动器。它的价格是65美元，几乎和Mac应用商店下载的东西一样，只是它在一个拇指驱动器上。tonymacx86建议您使用Mac App Store下载，针对USB拇指驱动器的更新指南将会发布，我们将在其发布时对其进行介绍。除此之外，您可能需要MultiBeast，以启用可能因操作系统升级而禁用的设备。您可以考虑使用最新发布的MultiBeast 4.0(专门为Lion制作)。

您可能会对您的hackintosh上的Mac OS X Lion的兼容性有一些疑问。如果雪豹非常适合你，那么狮子座也可能非常适合你。但是，总有一种罕见的机会，某些东西不能工作，或者安装可能不成功。所以在你采取行动之前，先看一下论坛。看看拥有类似硬件的人是如何让它工作的。一旦你满意了，就把所有需要的东西都收集起来，然后开始吧。

从这里开始的步骤与tonymacx86“xMove+MultiBeast”指南相同，尽管更详细地说明了这一点。你也可以按照原来的导游走。Lifehacker也介绍了升级方法，一定要检查一下。

安装过程如下所示。我们将把安装文件复制到您的Mac硬盘，然后将这些文件移动到硬盘上的新分区，然后从该分区启动，以便在Snow Leopard上或在新分区上安装Mac OS X Lion。

所以，一旦你从Mac应用商店下载了Mac OS X Lion，它应该会自动运行，但如果它不能运行，你就必须手动操作。一旦安装程序打开，点击继续。此安装程序现在不会实际安装Lion，它只会复制安装文件。下一个屏幕将要求您选择要复制安装文件的硬盘。确保它是您的Mac硬盘(根驱动器，安装了Mac OS X Snow Leopard的驱动器)。如果看不到，请单击显示所有磁盘，选择该驱动器，然后单击安装。这几乎需要3-5分钟，之后，系统将提示您重新启动。

接下来，打开Disk Utility(从/Applications/Utilities，或简单地在Spotlight中写入磁盘实用程序，然后按Enter键)。选择安装了Snow Leopard的硬盘。选择硬盘，而不是Snow Leopard分区。选择后，单击分区选项卡，然后单击+按钮添加新分区。命名新分区安装程序，并使其略大于8 GB。与往常一样，分区格式应为Mac OS扩展(日志)。完成此操作后，请单击“分区”对驱动器进行分区。您的雪豹分区不会受到任何影响，只是会减少8 GB。

现在，您又多了一个升级用户忽略的选择。大多数情况下，用户在他们正在工作的雪豹上安装Lion。可以这样想，您现在有一个包含安装程序分区和Snow Leopard分区的硬盘驱动器。当您要引导到Lion安装程序时，您将把它安装在哪里？显然，在雪豹之上。这不会替换雪豹分区中的任何文件，但它只会被升级为Lion。但就像在某些情况下一样，Lion安装得很好，但无法引导。因为它是雪豹的升级版，所以没有办法回滚到雪豹。因为大多数用户事先没有克隆他们的雪豹硬盘，所以他们也失去了完全正常工作的雪豹硬盘。因此，他们不得不重新安装雪豹，将其调整到完美，并尝试再次升级到Lion，等等。然而，为了避免这种情况，您可以选择在一个全新的空分区上安装Lion。这样，您的Snow Leopard分区将保持完好无损。

因此，如果您对在空分区上安装感兴趣，请先不要退出磁盘实用程序。添加一个新分区，将其命名为Lion，给它至少25-30 GB的空间(如果需要，也可以更多)，然后创建该分区。完成后退出Disk Utility。

然而，如果你要走这条路，你需要注意两件事。首先，您必须使用MultiBeast来启用您的设备和外围设备，就像您在全新安装Snow Leopard之后所做的那样。其次，如果您计划删除Snow Leopard分区，请确保将Lion分区配置为启动盘(进入系统首选项>启动盘)。您也可以将Lion安装在单独的硬盘驱动器上，而不是空分区上。

创建安装程序分区后，运行xMove。请注意，当系统要求您选择要安装的目标磁盘时，请选择安装程序分区，而不是任何其他分区。

单击Continue，等待xMove安装程序开始工作，这需要一段时间。一旦完成，你就可以退出了。现在是安装Mac OS X Lion的时候了。

重新启动您的hackintosh，在变色龙提示符下，移到安装程序分区，然后从它引导。如果需要，您也可以选择使用一些引导标志进行引导。

当您看到Mac OS X Lion安装程序时，请回想一下Mac OS X Snow Leopard的安装程序。从现在开始一切都是一样的。选择一种语言，然后单击箭头按钮继续。

现在你有两个选择。在下一个屏幕上，您可以继续并单击Continue，选择要安装的Snow Leopard硬盘，然后在其上安装。这通常省去了安装kext之类的麻烦。但如果您想继续小心操作，最好在空分区上安装Lion，即进行全新安装。因此，您可以选择Lion分区(如果您在前面创建了它)，或者在此时使用磁盘实用程序为其创建一个新分区。

这里我们要进行全新安装，目标分区名为“MacHD”

做出选择后，点击Install(安装)按钮，等待15-20分钟。安装完成后，您的hackintosh将重新启动，您将进入引导加载程序提示。现在，如果您进行了全新安装，请从Lion分区引导，在进入桌面之前，您可能需要执行一些额外的步骤(用户帐户设置、区域设置等)。但是如果你安装在雪豹分区上，从它引导，你会直接进入桌面。您还可以使用任何允许您访问Lion安装程序的引导标志。恭喜您，您现在可以在PC上运行Mac OS X Lion了！

现在要做的就是启用所有禁用的设备。你知道你需要什么吗-多兽。请记住，在本指南的前一部分中，我们要求您记下您对MultiBeast的选择。如果您进行了全新安装，只需使用所有这些选项运行MultiBeast，您的非工作设备就会再次正常工作。如果您进行了升级，请仅对不起作用的设备使用选项。是的，还记得8 GB的安装程序分区吗？现在不再需要了。因此，打开Disk Utility，选择您的Mac硬盘。单击分区选项卡，选择安装程序分区，单击-按钮，然后单击删除以删除该分区。只需拖动分区滑块来填满未分配的空间，即可将可用空间分配给硬盘上的任何其他分区。单击Apply(应用)，然后单击Partition(分区)调整分区大小，并向其中添加未分配的8 GB。就这样。点击几下，你的狮子哈金托什就会在它的荣耀中咆哮！

双引导问题

如本指南第一部分所述，建议在Mac和Windows上使用单独的硬盘。但如果是笔记本电脑，您将只需要处理一个硬盘。下面是tonymacx86关于在单个硬盘上安装Mac OS X Snow Leopard和Windows的教程。只要你知道基本知识，它就非常简单和相同。

但如果你将Windows(Windows7)安装在单独的硬盘上，只需将其连接，并确保BIOS启动顺序中的第一个硬盘是Mac硬盘。基本上，这将允许变色龙(嵌合体)引导加载程序在您的计算机执行POST后立即加载。它将自动检测您的Windows硬盘及其所有分区。如果您的Windows驱动器上有其他分区，它们也会在引导加载程序中显示为“Windows NTFS”。您必须选择“系统保留”分区才能引导到Windows。只需使用箭头键选择它，然后按Enter，您就可以从它引导。就这么简单。
