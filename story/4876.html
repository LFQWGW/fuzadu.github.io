如何在Windows 8或Windows 10上播放DVD或蓝光

升级到Windows8或Windows10，你可能会惊讶地发现，你再也不能播放视频DVD或蓝光了。与以前的Windows版本不同，Windows8和Windows10不包含播放DVD的内置支持。

微软之所以选择不提供DVD支持，是因为很多新电脑--尤其是平板电脑和超极本--都没有配备DVD驱动器。Microsoft为附带DVD支持的每个Windows副本支付许可费。

注意：您仍然可以在Windows 8或10上使用数据DVD。这仅适用于视频DVD。

安装第三方DVD播放器

在Windows 8或Windows 10中播放DVD的最简单方法是安装第三方DVD播放器。我们推荐流行的VLC媒体播放器。它是免费的，在你安装之后，你就可以用VLC播放DVD了--没问题。蓝光则是另一回事，因为它们是受支持的，但由于DRM加密，许多蓝光不能播放。

要在VLC中播放DVD，请单击媒体菜单，然后选择打开光盘。

VLC远不是你唯一可以使用的媒体播放器--有很多免费的第三方媒体播放器集成了对DVD的支持。

使用授权的DVD播放器

如果您购买了一台配备DVD或蓝光驱动器的新Windows8或10计算机，则您的计算机制造商可能已在您的计算机中附带了DVD播放软件。它可能会配置为在您插入视频DVD时自动打开。如果没有，您可以尝试在开始屏幕输入DVD或蓝光来搜索已安装的应用程序，看看是否有名称中含有DVD(或蓝光)的应用程序。

要检查计算机上的所有软件，请使用所有应用程序屏幕。按Windows键访问开始屏幕，右键单击开始屏幕上的任意位置，然后选择所有应用程序。滚动浏览已安装的应用程序列表，查找DVD播放程序。

如果您使用的是Windows10，您可以单击“开始”菜单上的“所有应用程序”项来查看相同的应用程序列表。

购买Windows 8媒体中心包(仅限Windows 8)

Microsoft在Windows 8中不再包含Windows Media Center。包含DVD播放的Windows Media Center可以单独使用。如果您安装了Windows 8专业版，您可以购买Windows 8 Media Center Pack来激活计算机上的Windows Media Center和DVD播放。

微软将在2013年1月31日之前免费提供Windows 8媒体中心包-单击此处获取。

如果您的计算机上安装了Windows 8的非专业版基本版，您必须通过购买Windows 8专业版包升级到Windows 8专业版，然后才能获得Windows 8 Media Center。使用“向Windows 8添加功能”控制面板进行升级。要打开它，请按Windows键，键入Add Feature，点击或单击Settings类别，然后点击或单击Add Feature to Windows 8。

Windows Media Center不适用于Windows 8企业版。

虽然微软从Windows8中删除了这一功能可能看起来有点奇怪，但这是因为人们越来越关注流媒体，以及大量没有DVD驱动器的新电脑问世。

VLC的容易安装，以及电脑制造商将包括他们自己的DVD播放软件的现实，意味着Windows8缺乏对DVD的支持并不是真正的问题。
