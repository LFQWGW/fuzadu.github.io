使用Windows Live Movie Maker创建您自己的Windows DreamScene

对于一些用户来说，将Windows桌面墙纸更换为个人照片是他们在新安装后的首要任务之一。今天，我们将不再使用照片，而是看看如何使用Windows Live Movie Maker创建个人视频，然后将其设置为Windows DreamScene。

Windows DreamScene是一个仅在Vista旗舰版中可用的实用程序，允许用户将视频设置为桌面墙纸。最近，我们研究了如何使用VLC Media Player将视频设置为桌面墙纸，以及如何在任何版本的Windows7或Vista中启用Windows DreamScene。现在，我们将使用Windows Live Movie Maker为DreamScene提供个人体验。

就任 / 安顿 / 安装 / 就职

下载并安装Windows Live Movies Maker。该软件本身作为Windows Live软件包的一部分捆绑在一起。但是，您不必安装任何您可能不需要的程序。您可以在下面找到下载链接。

在“您即将完成”屏幕上，在单击“继续”之前，请确保取消选中任何不需要的选项的复选框。

打开Windows Live Movie Maker，然后通过选择“开始”选项卡上的“添加视频和照片”按钮或通过将剪辑拖放到情节提要来添加视频剪辑。

我们要做的第一件事是通过剪裁掉不需要的部分来编辑我们的视频。您可以通过拖动情节提要上的滑块、预览窗口下的滑块或使用“下一帧”/“上一帧”按钮一次拖动一帧来浏览视频。

您可以通过两种方式删除不需要的部分。拆分视频并删除部分，或裁切视频并将区域保留在裁切范围内。要将视频拆分成多个部分，请将预览滑块移动到视频剪辑中要拆分的位置，然后选择拆分。

选择不需要的部分，然后单击删除。

如果要“修剪”视频并将区域保留在修剪范围内，请选择功能区上的“修剪”按钮。

拖动预览窗口下方的滑块以选择要保留的位置。滑块外的区域将从影片中删除。里面的区域是电影中保留的部分。由于我们的DreamScene将重复循环，因此请尝试以一种尽可能无缝的方式修剪视频的开头和结尾，以使回放尽可能无缝。

音频信号 / 声音

如果要删除或减小背景音频，请选择编辑选项卡，然后选择视频音量按钮。将滑块一直向左移动以使所有声音静音。

如果您愿意，您可以在剪辑中添加音乐。单击主页选项卡上的添加音乐按钮，然后选择添加音乐。

注意：如果您要添加音乐，但不想从视频本身获取音频，请在添加音乐之前将音频静音。

您将在故事板上看到添加到项目中的音乐曲目。您可以通过选择功能区上的选项淡入淡出音乐。

启用DreamScene

现在，我们准备将视频设置为墙纸。首先需要安装一个名为DreamScene Activator的小应用程序。这个免费的便携实用程序可以在32位和64位版本的Windows 7中启用DreamScene。一旦启用了它，我们就可以将在Windows Live Movie Maker中创建的WMV文件输出为桌面墙纸。

下载并解压缩Windows 7 DreamScene Activator。您可以在下面找到下载链接。解压后，您需要以管理员身份运行应用程序。要执行此操作，请右键单击.exe并选择以管理员身份运行。

单击启用DreamScene按钮。这将重新启动Windows资源管理器。

启用DreamScene后，右键单击您的WMV文件并选择Set as Desktop Background(设置为桌面背景)。

尽情享受您的Windows DreamScene。

如果图标下的文本变暗，请在桌面上单击鼠标右键并选择个性化。

然后选择桌面背景。

选择填充作为图片位置，然后在退出前保存。在这些更改之后，您需要重新启动您的DreamScene。

现在您可以阅读您的图标并欣赏节目了。当你看够了你的梦幻场景，只需切换回你的旧墙纸即可。

结束 / 结尾 / 结论 / 推论

这真的是一种很酷的方式，既可以给你的windows桌面带来动作，又可以增加一些你自己的创造性触感。你不需要一段冗长或动感十足的视频。容易循环的短而简单的视频是理想的。你可能想要尝试一些东西，比如海浪在海滩上滚滚而来，当地的溪流，或者壁炉里的火。您也不需要使用Windows Live Movie Maker来创建视频。但是，您使用的视频文件必须是WMV或MPG。您是否创建了自己的DreamScene？在评论中让我们知道。
