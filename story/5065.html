您是否需要Windows 8专业版？

如果你想购买Windows8，你需要关注两个主要版本：Windows8(类似于以前版本的Windows家庭版)和Windows8专业版。

相关：您是否应该升级到Windows 10专业版？

更新：Windows10即将问世，我们提供了更新指南，让您知道是否需要Windows10专业版。

与以前的Windows版本相比，Windows 8的版本更少。这里我们将忽略另外两个：Windows8企业版面向较大的组织，而Windows RT面向基于ARM的设备，完全是另一种野兽。

升级到专业版

即使你购买了Windows 8标准版(或者你购买了一台装有Windows 8而不是Windows 8专业版的电脑)，你也可以随时升级到Windows 8专业版。

要做到这一点，请使用“向Windows8添加功能”控制面板(在Windows7中称为“Windows Anytime Upgrade”)。从此窗口购买Windows 8专业版软件包，无需重新安装Windows即可获得专业功能。你甚至不需要离开你的椅子。

Windows 8

你可以在商店里找到的大多数电脑上都预装了Windows8的标准版。虽然它不再被称为Windows的“家庭”版，但它与以前版本的Windows的家庭版相似。它包含大多数用户会使用的功能。

不管你喜不喜欢，新的Windows 8界面以及新的开始屏幕、魅力、现代应用程序和Windows应用商店都出现在所有版本的Windows 8中。Windows桌面，包括各种出色的新功能和安全改进，仍然存在。Internet Explorer10、集成杀毒软件和其他大多数你所期待的功能都集成到了Windows8的“核心”版本中。

Media Center功能

令人惊讶的是，Windows 8标准版不再包含某些媒体中心功能。您无法在Windows 8标准版上播放DVD或使用Windows Media Center应用程序。

要使用这些功能，您需要升级到Windows 8专业版，并从上面提到的向Windows 8添加功能窗口中购买Windows 8 Media Center Pack。如果您使用的是Windows 8专业版，您实际上可以在2013年1月31日之前免费获得Windows 8 Media Center Pack。

微软这样做是为了节省许可成本-许可DVD播放和媒体中心所需的编解码器需要花钱，当许多新电脑没有DVD驱动器，人们越来越多地从Netflix等在线服务流媒体视频时，这就不再那么有意义了。

这看起来可能是个问题，但实际上不是。如果你想播放DVD，你不需要付一分钱-只需安装VLC即可。有关在Windows 8上播放DVD的详细信息，请单击此处。

Windows Media Player仍然存在于Windows 8上(Windows RT除外)，您可以继续使用其他媒体播放应用程序。很少有人使用Windows Media Center-如果您想使用Windows Media Center本身，则只需购买Windows Media Center包即可。

如果您将Windows计算机用作媒体中心PC，则可以尝试使用XBMC之类的软件。你可能还想尝试使用Netflix、Hulu和其他服务的现代应用程序-它们的全屏界面可以在家里的电视上使用。

Windows8Pro

专业版的其余功能针对的是喜欢利用更高级功能的企业和“狂热”极客。在Windows8标准版上，这些功能中的许多都可以被免费的替代功能取代。

域加入和组策略：使用Windows服务器域和组策略的组织将需要Windows 8专业版。 远程桌面服务器：您可以从Windows8 PC连接到远程桌面服务器，但您需要Windows8专业版才能托管远程桌面服务器。但是，您可以在标准版的Windows8上使用第三方远程桌面服务，如TeamViewer或VNC。 BitLocker和加密文件系统：Windows的加密功能仅在Windows 8专业版上提供。如果你不相信这些功能，你可以在所有版本的Windows8上免费安装TrueCrypt。 Hyper-V：Windows Server中的Hyper-V技术可供Windows 8专业版用户用于运行虚拟机。其他所有人都可以免费下载VirtualBox或VMware Player。 虚拟硬盘引导：Windows 8专业版可以从VHD文件引导。如果这对您毫无意义，则不需要此功能。

我们还介绍了Windows专业版功能的更多免费替代方案。

不幸的是，Windows To Go是一项有用的功能，可以让你将Windows 8安装到USB驱动器上，并在任何电脑上启动它，但它只在Windows 8的企业版上提供-即使是专业版用户也不能使用它。

现在，您应该已经对是否需要Windows 8专业版有了一些了解。如果您是一家依赖域和组策略的企业，答案是明确的是肯定的。如果您不是一家企业，只有当您绝对需要Windows Media Center或其他“发烧友”功能(如BitLocker)时，您才需要它。
