参观Windows 8的20个内置应用程序及其功能

Windows8新的触摸式现代界面包含了相当多的应用程序。在您开始查看Windows应用商店以查找新应用之前，请先了解一下包含的应用及其功能。

这些应用程序有一些共同之处。它们都有一种强调内容的最小感觉，大多数都支持在开始屏幕上显示更新信息的实时磁贴，大多数都在推送微软的在线服务。

储备量 / 储备物 / 丰富 / 仓库

Windows应用商店是唯一可以获得现代应用程序的地方。它的工作原理类似于苹果的App Store或Google Play-搜索应用程序，然后单击“安装”按钮将其安装到您的计算机上。(使用搜索符或只需开始键入即可搜索应用程序。)

Windows应用商店还列出了一些桌面应用程序，但你不能安装桌面应用程序并获取它们的更新-Windows应用商店只会将你链接到应用程序开发人员的网站，在那里你可以像在Windows 7上一样下载和安装桌面应用程序。

Internet Explorer

Windows8的默认浏览器是现代版的Internet Explorer10。与老版本的Internet Explorer不同，它的速度相当快。它还经过了触摸优化，具有最小的界面，可以在您浏览时隐藏您的选项卡和导航栏。

Internet Explorer10缺少一些您可能期望的常见浏览器功能，包括对加载项和插件的支持。默认情况下，只允许某些网站使用Flash。

邮件、日历、人员和消息

邮件、日历、人员和消息应用程序紧密相连，以至于它们捆绑在Windows应用商店上的单个应用程序包中。您在一个应用中添加的帐户将与其他应用共享，因此您只需提供一次您的在线帐户详细信息。

Mail应用程序是一个简单的应用程序，用于检查您的电子邮件，它不像Microsoft Outlook那样试图提供大量的功能。您可以添加Hotmail、Outlook或Google电子邮件帐户。还可以选择添加使用IMAP、POP或Exchange ActiveSync(EAS)协议的任何帐户，因此您几乎可以添加任何电子邮件帐户。

日历应用程序显示来自在线服务的日历。它支持Hotmail、Outlook和Google日历，将它们合并到一个视图中。

People应用程序将您的所有联系人集中在一个地方。除了来自Hotmail、Outlook和Google帐户的联系人外，它还支持Facebook、Twitter和LinkedIn联系人。您的联系人的新社交动态也将出现在此处。

这款消息应用程序允许你与朋友聊天。它支持微软的Windows Live Messenger(以前称为MSN)和Facebook聊天。其他流行的服务，如Google Talk和AIM，在这款应用中不可用。

天气 / 气候 / 上风面 / 气象

Weather应用程序为您显示当前位置的天气，不过您也可以配置多个位置。该应用程序与实时磁贴功能配合良好，可以在您的主屏幕上显示当前天气。你还可以在应用程序中滚动查看详细的天气信息，包括每小时的天气预报、天气地图和历史天气数据。

新闻、金融和体育

新闻、金融和体育应用程序都显示不同类型的新闻。他们使用的数据来自微软的必应(Bing)。

新闻应用程序是一款触控优化的应用程序，允许您浏览和阅读最新新闻。它默认显示“必应日报”的新闻，但你也可以订阅其他新闻来源，如BBC、纽约时报和华尔街日报。

金融应用程序显示金融新闻以及市场表现图表。您还可以将您感兴趣的股票添加到观察列表中，并查看它们的表现。

体育应用程序显示体育新闻故事，以及比赛日程和比分。您可以观看您最喜欢的球队来获取最新消息。

旅行 / 沿旅行 / 经过 / 走动

必应旅游应用程序允许你浏览旅游目的地并查看更多信息。您可以查看有关某个位置的信息、查找该地区的航班和酒店、阅读有关该位置的导游以及浏览景点。

照片 / <非正式>摄影定名次

照片应用程序允许您查看多个位置的照片。您可以在硬盘或连接的设备上查看存储在图片库中的照片。你还可以在线查看存储在SkyDrive、Facebook或Flickr账户中的照片-尽管不支持其他服务，包括谷歌的Picasa。

音乐和视频

Xbox品牌的音乐和视频应用程序允许你在硬盘上播放音乐和视频文件，但它们也包括在线音乐和视频商店。

音乐应用程序播放你自己的音乐，并允许你从Xbox Music(前身为Zune)商店购买歌曲。这款应用还提供免费的、由广告支持的音乐流媒体，如Pandora、Spotify、Rdio和其他在线音乐服务。

视频应用程序允许你播放自己的视频，但你也可以从Xbox Video商店在线购买或租赁电影和电视节目。

小游戏

Xbox品牌的游戏应用程序允许您使用您的Xbox帐户登录。这里列出的游戏有Xbox成就、游戏邀请、轮到通知和其他与Xbox相关的社交功能。你不能在这里玩Xbox游戏机游戏，只能玩Xbox品牌的Windows8游戏。

<苏格兰>一堆 / 材料堆

必应(Bing)应用程序是一款简单的应用程序，用于搜索微软的必应(Bing)搜索引擎。如果你更喜欢谷歌，你可以安装Windows8版的谷歌搜索应用程序，或者只需在浏览器中进行搜索。

地图

地图应用程序使用必应地图。如果您有内置GPS的平板电脑，则可以使用它查看您在地图上的当前位置。你还可以搜索位置和获取方向，就像使用谷歌地图或MapQuest一样。

天驱

SkyDrive应用在线显示存储在你的SkyDrive帐户中的文件。你还可以使用此应用将文件上载到你的SkyDrive。

如果你想在桌面上使用SkyDrive，你需要下载并安装SkyDrive桌面应用-它不包括在Windows 8中。

读者 / 读本 / 选集 / 阅读器

阅读器应用程序是一个用于查看PDF和XPS文件的简单应用程序。Windows终于有了一个内置的PDF查看器，尽管没有桌面对应的查看器。

照相机 / 摄影机 / 厅 / 圆形建筑

相机应用程序允许您使用网络摄像头捕捉照片和视频。您可以使用PC内置的网络摄像头或连接外部网络摄像头。

Windows应用商店包括更多应用，但这些都是Windows 8上开箱即用的应用。这些应用可能会继续发展，增加更多功能-其中一些应用在过去几个月里取得了长足的进步。
