删除Windows磁盘清理中的所有内容安全吗？

Windows附带的磁盘清理工具可以快速擦除各种系统文件并释放磁盘空间。但有些东西--比如Windows10上的“Windows ESD安装文件”--可能不应该被删除。

在很大程度上，磁盘清理中的项目可以安全删除。但是，如果您的计算机运行不正常，删除其中的一些内容可能会阻止您卸载更新、回滚操作系统或只是解决问题，因此，如果您有足够的空间，可以很方便地保留它们。

磁盘清理-101

相关：释放Windows硬盘空间的7种方法

您可以从开始菜单启动磁盘清理-只需搜索“磁盘清理”即可。它将立即扫描它可以删除的文件，并向您显示一个列表。但是，这只显示了您可以使用当前用户帐户的权限删除的文件。

假设您拥有该计算机的管理员访问权限，您将希望单击“清理系统文件”来查看您可以删除的完整文件列表。

要删除一组文件，请选中它。要保留一组文件，请确保未选中该文件。您将在窗口顶部看到可以删除的最大数据量，以及在底部实际节省了多少空间。选择完数据后，单击“确定”，磁盘清理将删除您要删除的数据类型。

Windows ESD安装文件很重要

相关：您需要了解的有关Windows 8和10中“重置这台电脑”的所有信息

在Windows10上，这里有一个“Windows ESD安装文件”选项。删除它可以释放几GB的硬盘空间。“这可能是列表中最重要的选项，因为删除它可能会给您带来问题。

这些ESD文件通常用于将您的PC重置为出厂默认设置。如果你删除这些文件，你将有更多的磁盘空间，但你将没有重置电脑所需的文件。如果您想要重置Windows 10，您可能不需要下载Windows 10安装介质。

我们建议不要删除此内容，除非您迫切需要硬盘空间中的几千兆字节。如果您将来想要使用“重置您的电脑”功能，删除此功能将使您的生活变得更加困难。

磁盘清理的其他所有内容都可以删除

那么，所有其他选项都做些什么呢？我们进行了磁盘清理，并列出了一份清单。请注意，我们在运行Windows 10并安装周年纪念更新的PC上使用了磁盘清理。较早版本的Windows的选项可能会稍微少一些。某些选项可能仅当您的硬盘上有特定类型的系统文件时才会显示。

Windows Update清理：当您从Windows Update安装更新时，Windows会保留较旧版本的系统文件。这允许您稍后卸载更新。然而，如果你不打算卸载任何Windows更新，那就是浪费空间。“只要你的电脑工作正常，而且你不打算卸载任何更新，删除它就是安全的。” Windows Defender：根据磁盘清理工具，此选项删除“Windows Defender使用的非关键文件”。微软在任何地方都没有解释这些文件是什么，但它们很可能只是临时文件。您可以选择此选项来释放一些空间，Windows 10的内置杀毒软件将继续正常运行。 Windows升级日志文件：当您升级Windows时-例如，从Windows 7升级到10，或者从Windows 10的11月更新升级到Windows 10的周年纪念更新-Windows会创建日志文件。这些日志文件可以“帮助识别和排除发生的问题”。如果您没有任何与升级相关的问题，请随时删除这些内容。 下载的程序文件：此文件夹包含在Internet Explorer中查看某些网页时从Internet下载的ActiveX控件和Java小程序。请随意删除这些内容。如果您需要的话，当您访问需要它们的网站时，它们将被自动再次下载。 Internet临时文件：其中包含Internet Explorer和Microsoft Edge的“浏览器缓存”。高速缓存包括存储在硬盘上的零星网站，以便您将来可以更快地加载它们。您可以清除此选项以释放空间，但浏览器缓存最终会再次填满。还要注意的是，这只会影响微软的浏览器。其他浏览器，如Google Chrome和Mozilla Firefox，都有自己的浏览器缓存，你需要从Chrome或Firefox内部清除它们自己的缓存。只需记住：定期擦除浏览器缓存会减慢您的Web浏览速度。 系统错误内存转储文件：当Windows崩溃时-称为“死亡蓝屏”-系统会创建一个内存转储文件。此文件可以帮助准确识别出问题所在。但是，这些文件可能会占用大量空间。如果您不打算尝试对任何蓝屏死亡进行故障排除(或者您已经修复了它们)，您可以删除这些文件。 系统存档的Windows错误报告：当程序崩溃时，Windows会创建错误报告并将其发送给Microsoft。这些错误报告可以帮助您识别和修复问题。存档的错误报告已发送到Microsoft。您可以选择删除这些内容，但您将无法查看有关程序崩溃的详细报告。如果你不是在尝试解决问题，它们可能并不重要。 System Queued Windows Error Reporting：这与“System Archived Windows Error Reporting”相同，不同之处在于它包含尚未发送给Microsoft的排队错误报告。 Windows ESD安装文件：这一点很重要！如上所述，这些文件存储在您的PC上，用于将您的PC重置为出厂默认设置。您可以删除它们以释放空间，但如果您想要重置电脑，则需要创建并提供Windows安装介质。 交付优化文件：“Windows Update交付优化服务”是Windows 10的一部分，它使用您的计算机带宽将应用程序和Windows更新上传到其他计算机。此选项允许您删除不再需要的数据，但上载到其他电脑除外。 设备驱动程序包：Windows保留旧版本的设备驱动程序，无论它们是从Windows Update还是从其他地方安装的。此选项将删除那些旧的设备驱动程序版本，只保留最新的版本。如果您的PC及其设备似乎工作正常，则无法删除这些文件。

以前的Windows安装：*当您升级到新版本的Windows时，Windows会将旧的Windows系统文件保留10天。然后您可以在这10天内降级。10天后，Windows将删除这些文件以释放磁盘空间-但您可以立即从此处删除它们。在Windows10上，安装像周年更新或11月更新这样的大更新基本上等同于升级到全新的Windows版本。因此，如果您最近安装了周年更新，则此处的文件将允许您降级到11月份的更新。 回收站：选中此选项，磁盘清理工具在运行时也会清空计算机的回收站。 临时文件：程序经常将数据存储在临时文件夹中。选中此选项，磁盘清理将删除超过一周未修改的临时文件。这确保它应该只删除程序不使用的临时文件。 Windows临时安装文件：在安装新版本的Windows或重大更新时，Windows安装过程通常会使用这些文件。如果您没有安装Windows，可以删除它们以释放空间。 缩略图：Windows为图片、视频和文档文件创建缩略图，并将其存储在硬盘上，以便您再次查看该文件夹时可以快速显示它们。此选项将删除那些缓存的缩略图。如果您再次访问包含这些类型文件的文件夹，Windows将重新创建该文件夹的缩略图缓存。

我们在这里也看到了各种各样的其他选择。有些只出现在以前版本的Windows上，如Windows 7，有些只在您的计算机硬盘上有特定类型的文件时才会出现：

临时安装文件：程序有时会在您安装安装文件时创建安装文件，并且不会自动清理它们。此选项将删除不再用于任何用途的安装文件。 脱机网页：您可以保存网页，以便在Internet Explorer中“脱机”浏览。您的“脱机网页”是专门保存以供脱机使用的网页，选中此选项将删除它们。 调试转储文件：这些是在崩溃后创建的调试文件，用于帮助确定崩溃的原因。如果您不想解决问题，可以将其删除。 每用户存档错误报告：这些文件与“系统存档Windows错误报告”文件相同，但存储在用户帐户下，而不是存储在系统范围内。 每用户排队的Windows错误报告：这些文件与“系统排队的Windows错误报告”文件相同，但存储在用户帐户下，而不是存储在系统范围内。 旧Chkdsk文件：当硬盘上有文件系统损坏时，会运行chkdsk工具。如果您看到任何“旧chkdsk文件”，这些都是损坏文件的碎片。您可以安全地删除它们，除非您试图恢复重要的、不可替代的数据。 游戏统计文件：在Windows7上，这些文件包括你所包含的纸牌和扫雷等游戏的得分信息。删除它们，游戏将会忘记您的分数和其他统计数据。 安装日志文件：这些日志文件是在安装软件时创建的。如果出现问题，日志文件可以帮助识别问题。如果您不想对软件安装进行故障排除，则可以将其删除。 系统错误小型转储文件：这些文件是Windows在蓝屏崩溃时创建的较小内存转储文件。与较大的内存转储文件相比，它们占用的空间更少，但仍然可以提供识别问题的有用信息。如果您不想对系统问题进行故障排除，可以将其删除。 Windows升级丢弃的文件：这些是在Windows升级过程中未迁移到您的新电脑的系统文件。假设您的计算机工作正常，您可以删除它们以释放空间。

总体而言，只要您不打算回滚到设备驱动程序、卸载更新或解决系统问题，您就可以安全地删除磁盘清理中的几乎所有内容。但是你可能应该避开那些“Windows ESD安装文件”，除非你真的很想占用空间。
