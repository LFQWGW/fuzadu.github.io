如何轻松地在Linux启动时自动挂载分区

通常，让Ubuntu在启动时挂载分区需要摆弄令人困惑的“fstab”。当您打开计算机时，自动挂载分区的最简单方法是阅读本文。那么让我们开始吧！

我们将要安装的程序名为“Storage Device Manager”，它将使这一过程变得简单。在终端窗口中输入以下命令进行安装：

或在Ubuntu软件中心搜索

现在，从系统>管理>存储设备管理器启动它。运行后，从左侧面板中选择要在启动时挂载的分区(首先展开硬盘列表)。然后点击右侧的“助手”。

现在，您将看到选项窗口。只需选中“the file system is mount at boot time”，取消选中“mount file system in read-only mode”。

如果需要，您可以编辑其他选项，但要小心，它可能会损坏您的系统。完成后，单击“确定”按钮，然后单击“应用”。关闭程序并重新启动以查看效果。就这样!
