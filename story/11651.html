如何重命名Plex Media Center以便于识别

如果您在家中运行多个Plex Media Center客户端，则有时很难区分哪个是哪个。让我们来看一下不太明显的设置，您需要调整以给每个Plex客户端指定一个唯一的名称。

在很多情况下，任何给定的Plex机器的名称可能都无关紧要。但是，如果您的家庭网络上有多个Plex客户端，并且您想要利用Plex对从控制应用程序(如Plex for iOS或Plex Media Server web仪表板)发送媒体信息的内置支持，您很快就会遇到问题。

例如，最近为我们的整个房子配备了运行Rasplex的树莓PI单元，每台电视都连接着Rasplex。但是，当我们要从Plex Media Server仪表板投射视频时，我们会看到以下内容：

由于每个客户都有相同的标签，我们不知道我们是否会将我们选择的视频发送到卧室、客房、起居室或地下室的电影院。如果您处于类似的情况，那么本教程是为您准备的。与其玩视频选角彩票，不如花点时间重命名所有客户，这样他们就更容易识别了。

相关：如何将视频从Plex Media Server投射到Chromecast

令人惊讶的是，重命名Plex客户端并不是特别直观。您不能从服务器仪表板重命名它们；相反，您必须从每个单独的客户端对其重命名。

虽然客户端软件在每个平台上都略有不同(因此您需要查看设置菜单)，但我们可以向您展示如何在RasPlex上进行更改，让您大致了解一下。要做到这一点，只需打开“首选项”菜单，当使用默认的RasPlex皮肤时，只需在主菜单中按向左箭头键即可访问该菜单。

在Preferences(首选项)菜单中，选择“Change Preferences for System”(更改系统的首选项)一侧的齿轮图标。

在那里，您可以选择“服务”，您将看到一个“设备名称”条目。将该条目从通用的“RasPlex”条目更改为清晰的条目，该条目指示Plex客户端连接的位置(或连接到什么位置)，如“Living Room”或“Projector”。值得注意的是，此名称更改不仅会影响Plex服务器如何查看客户端软件，而且不会更改网络主机名或有关硬件的任何其他标识信息。

重新启动设备，更改应立即出现在您的Plex软件的铸造菜单中。在重新启动并打开我们的仪表板时，“客厅”已经被列为目的地之一，如下所示。“Actual Settings>Devices(实际设置>设备)”菜单中的列表更新时间稍长一些，但不用担心-它将在10-20分钟内刷新，无需重新启动服务器，并且不会影响投射功能。

一旦您更改了一个条目，即可浏览其余条目，您将拥有每个Plex客户端的唯一名称-当您尝试将电影投射到家中的不同屏幕时，无需再进行猜测。
