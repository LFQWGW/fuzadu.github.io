如何从Windows、Mac或引导盘擦除驱动器

无论你是放弃电脑还是处理USB驱动器，如果驱动器上有敏感的、未加密的数据，擦除它都是一个好主意。这将防止有人使用删除文件恢复工具从该驱动器恢复敏感数据。

您可能应该使用全磁盘加密，而不是像这样事后擦除磁盘。这将保护您的文件，无论您是否已删除它们。标准的驱动器擦除实用程序也不能与SSD一起正常工作，并且可能会降低其性能，而加密是可以保证工作的。

Windows 10(和Windows 8)

Windows10提供了一种内置方式，可以在您将计算机转给其他人时擦除系统驱动器。打开设置应用程序，导航至更新与安全>恢复，单击或轻触“重置此电脑”下的“开始”按钮，选择“删除所有内容”，然后选择“删除文件并清理驱动器”。

此功能是在Windows 8中添加的，因此将Windows 8或8.1电脑重置为出厂默认设置时，您将看到相同的选项。

Windows 10还包括一种内置方式，可以从操作系统中安全地擦除USB驱动器、SD卡或任何其他驱动器。此选项已添加到Windows 8的FORMAT命令中，因此它也可以在Windows 8和8.1上运行。

将“x：”替换为您要格式化的驱动器的驱动器号，请非常小心地选择正确的驱动器，否则将擦除另一个驱动器。“/p”开关告诉Windows要使用多少遍。例如，“/p：1”将在驱动器上执行一次遍历，覆盖每个扇区一次。您可以输入“/p：3”来执行三次测试，以此类推。对固态存储执行此操作可能会缩短驱动器的寿命，因此请尽量不要使用超过实际需要的通行证。理论上，你应该只需要一次通过，但为了安全，你可能想额外执行几次。

Windows 7(以及未安装操作系统的计算机)

Windows7不包含任何集成的磁盘擦除功能。如果你仍在使用Windows 7，你可以使用DBAN引导程序(也称为Darik‘s Boot and Nuke)启动电脑，并用它来擦除内部驱动器。然后，您可以在该驱动器上重新安装Windows，并使用已擦除的驱动器恢复出厂默认设置，或者在用垃圾数据覆盖该驱动器后将其处理掉-无论您想做什么。

DBAN是一个可引导的环境，因此您可以将其扔到USB驱动器上，或者将其刻录到光盘上，然后在甚至没有操作系统的PC上启动它，以确保PC驱动器被擦除。

要擦除USB驱动器、SD卡或其他驱动器，您可以使用橡皮擦之类的程序。如果您不想在终端中使用format命令，也可以在Windows10、8.1或8上使用此命令。安装橡皮擦后，您可以在Windows资源管理器中右键单击驱动器，指向“橡皮擦”，然后选择“擦除”将其擦除。

MacOSX

Mac OS X附带的磁盘实用程序工具可以安全地擦除驱动器。它适用于内部系统驱动器、外部USB硬盘驱动器、闪存驱动器、SD卡以及您可能想要安全擦除的任何其他驱动器。

要在Mac OS X中使用此工具擦除外部驱动器，请按Command+空格键打开Spotlight Search，键入“Disk Utility”，然后按Enter键。选择外置硬盘，点击“擦除”按钮，点击“安全选项”，就可以选择想要用垃圾数据覆盖的次数。然后单击“擦除”按钮，Disk Utility将擦除驱动器。

要擦除系统驱动器，您需要通过重新启动Mac并在引导时按住Command+R将其引导到恢复模式。

从恢复模式启动Disk Utility，选择您的系统驱动器，然后使用与上面相同的选项将其擦除。然后，您可以从恢复模式重新安装Mac OS X。

这在Linux上也是可能的。如果您有一台台式Linux PC，并且想要擦除全部内容，您可以随时启动DBAN并擦除它。但是您可以使用各种命令执行相同的操作，包括dd、shred和wipe命令。

如果您确实担心数据被恢复，您可以在事后使用删除文件恢复实用程序检查驱动器，并确认它找不到任何数据。当然，这不会像专门的法医工具那样有效。但是，如果你是这样的偏执狂，你可能应该直接销毁驱动器，这样以后就没有人可以使用它了。例如，军方就是这样处理包含非常敏感数据的硬盘的。
