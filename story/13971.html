如何关闭三星Galaxy Note 10或10 Plus的电源

现在三星发布了Galaxy Note 10和10 Plus，你会发现传统的关闭手机电源的方法不起作用了。相反，您必须使用新的组合键或更改其默认设置。这是怎么做的。

使用物理按钮关闭电源

在几乎所有其他Android智能手机或平板电脑上，你都可以通过长按电源按钮来关闭设备。此操作通常会启动电源菜单，允许您关闭电话电源、重新启动电话或执行更多操作。在三星的新旗舰上，这将打开Bixby虚拟助手。

在Galaxy Note 10和10 Plus中，三星加入了启动电源菜单的新组合键。与仅仅按住电源按钮(三星称之为“侧键”)不同，你需要同时按住电源按钮和音量降低按钮几秒钟。

当电源菜单出现在屏幕上时，您可以松开两个底部。现在，您只需轻触“关机”，您的三星Galaxy Note 10或10 Plus就会关闭。

更改侧键的按住操作

如上所述，三星允许您更改长按电源按钮的默认行为。如果你宁愿直接跳到电源菜单而不是启动Bixby，下面是你可以如何改变它。

首先，从屏幕顶部向下滑动，以便显示通知阴影。接下来，轻触右上角的齿轮图标以打开设置菜单。

现在，点击“高级功能”。该项目应该在选项列表的一半位置。

选择“侧键”。

现在，您可以更改双击电源按钮的默认行为，以及按下并按住按钮时发生的情况。您需要从“按住”部分选择“关闭电源菜单”。

或者，您也可以通过向下滑动通知阴影，点击电源图标，然后选择“侧键设置”来访问侧键菜单。然后，您可以选择“关闭电源菜单”作为按住操作。

从快速设置面板关闭电源

如果你不想或不能使用Galaxy Note 10或10 Plus的物理按钮来关闭手机，三星已经在通知阴影中偷偷提供了电源菜单的快捷方式。

要访问它，请从手机屏幕顶部向下滑动，以拉下通知阴影。从那里，点击齿轮按钮旁边右上角的电源图标。

现在将加载电源菜单，提供多个选项可供选择。轻触“关闭电源”以关闭您的设备。

使用Bixby关闭电源

最后一个选择是让三星的虚拟助手Bixby关闭三星Galaxy Note 10或10 Plus。

要开始，您需要启动Bixby。如果您没有更改长按电源按钮的默认行为，您可以这样做来调出助手。在那里，说“关掉手机”，你的手机就应该关机了。

您还可以通过Bixby Home滑动到最右侧的主屏幕来访问虚拟助手。在这里，你可以点击Bixby图标，然后告诉它，“关掉电话。”

重新打开您的Note 10或10 Plus电源

幸运的是，三星没有改变你重新打开Galaxy Note 10或10 Plus的方式。和几乎所有其他智能手机一样，你所需要做的就是按住电源按钮几秒钟。一旦三星的徽标出现在屏幕上，你就可以放手了。
