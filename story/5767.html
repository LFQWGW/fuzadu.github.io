将您的数据保存在本地：6个流行的Web应用程序的免费离线替代方案

网络应用风靡一时，但离线应用仍占有一席之地。无论您是想要更好的离线支持，还是只想将敏感数据保存在PC上，都有一款免费的桌面应用程序可以取代您的基于Web的工作效率应用程序。

我们已经研究了桌面应用程序的基于Web的替代方案，现在我们将采取相反的做法。以下是流行网络应用程序的一些坚实的、完全免费的离线桌面替代方案。

如果您将重要数据的唯一副本存储在本地，请务必执行定期备份。当你的硬盘不可避免地报废时，你不会想要失去一切。

电子邮件-Gmail、Outlook.com、Yahoo！邮费

离线电子邮件程序的开发已经放缓，但仍然有可靠的选择。这些程序将允许您完全脱机从台式计算机访问您的电子邮件。现代电子邮件帐户在默认情况下使用IMAP，您的电子邮件保持在线存储，但如果您更愿意将电子邮件历史记录存储在计算机上，您也可以将电子邮件客户端配置为使用POP3，并将电子邮件服务设置为在邮件下载后立即将其删除。

对于离线电子邮件，我们推荐Mozilla Thunderbird。它不再积极开发新功能，但它是一个可靠、稳定的电子邮件程序，拥有你可能需要的所有功能。附加功能可以通过附加组件添加，就像Firefox一样。

您可能还会对Windows Live Mail感兴趣，它仍然可以作为Microsoft的Windows软件包(以前称为Windows Live软件包)的一部分提供。它也不再被积极开发，但它是一个免费的电子邮件客户端，与微软早期的Outlook Express软件相比是一个很大的改进。当然，如果您有Microsoft Office，您也可以使用Microsoft Outlook作为电子邮件程序。

日历-谷歌日历、Outlook日历

仍然有几个可靠的程序可以让您完全脱机存储日历事件和任务。如果您使用Mozilla Thunderbird，请尝试Mozilla Lightning扩展。它是独立的Mozilla Sunbird日历应用程序的继任者，并为Thunderbird添加了日历和任务支持。Lightning集成到Thunderbird中，因此您可以将电子邮件、日历任务和待办事项集中在一个地方。

如果你正在寻找一些更漂亮的东西，让它永远留在你的桌面上，试试Rainlendar。它将日历事件和任务显示为非常可定制的桌面小部件。

Windows Live Mail还具有集成的日历功能。如果你正在使用它，看看它集成的日历功能。当然，Outlook也有日历功能，所以如果您是Microsoft Office用户，这可能是您可用的最强大的选择。

Office-Google Docs、Office Web Apps

Google Docs和现在的微软Office Web Apps因为提供免费的办公功能来存储你的文档而变得流行起来，这样你就可以从任何电脑或移动设备上访问它们。甚至连微软的Office也在上线，Office 2013默认将文档保存到Microsoft SkyDrive，而不是你的本地电脑。

仍然有功能强大、开发活跃的免费办公应用程序可以脱机使用。功能最强大的免费办公套件是LibreOffice，它包括文档、电子表格、演示文稿、绘图、数学甚至数据库功能。它是被称为OpenOffice的项目的一个更活跃的开发分支。此免费选项对查看和编辑Microsoft Office格式文档具有最佳兼容性。

如果你只想写快速文本文档，你可能想看看Abiword-它非常简单和轻量级。不要期待高级功能或与复杂的Microsoft Office文档兼容，但要期待轻量级、简单的文档编辑。

对于所有默认的SkyDrive和在线功能，即使是最新版本的Microsoft Office也可以脱机使用。Microsoft Office不是免费的，但在这里值得一提--只是因为已经有这么多人可以使用它了。

照片管理-Facebook照片、Flickr、Google+照片

基于网络的照片服务允许最容易的分享，并确保普通用户会备份他们的照片，因此它们已经变得越来越标准。然而，仍然有可靠的离线照片管理解决方案。

Google的Picasa桌面应用程序仍然是一个完全离线工作的高质量桌面照片管理应用程序。这是一个很好的选择，至少在谷歌决定在未来的春季大扫除中停止对桌面版Picasa的支持之前是这样。它可以从数码相机导入照片，执行基本的照片编辑，并帮助您组织照片。

微软还提供他们的照片库程序，作为Windows软件包桌面应用程序套件的一部分。

密码管理-LastPass

LastPass可能会在在线存储密码和其他私人数据之前对其进行本地加密，但有些人在没有在线同步的情况下会更舒服。如果您想要离线密码管理器，请使用KeePass。它是一个强大的、完全开源的密码管理器，可以通过加密锁定您的密码数据库-您每次使用它时都必须输入一个主密码。它没有在线同步功能，所以你必须自己弄清楚如何备份和移动你的数据库。

有移动应用程序，所以你也可以从你的智能手机访问你的KeePass数据库-不过，你必须手动将数据库复制到你的智能手机上。

即时消息-Google Hangout(Google Talk和GChat)、Facebook Messenger

即时通讯程序本质上是在线的--毕竟，这些信息必须通过互联网传输。但是，如果您使用Google Talk、Facebook Messenger等在线聊天应用程序，或者AIM或Yahoo Messenger等其他服务，消息将以明文形式在线传输。它们可能会被归档到在线服务中，因为Google Talk消息是用Gmail归档的。

如果您希望以更私密的方式通信，可以使用支持加密消息的桌面应用程序。Pidgin，连同Pidgin-OTR插件，允许您从桌面发送加密消息。您需要与安装了OTR插件的其他人进行通信，但如果两人都安装了适当的插件，则可以通过Internet发送加密消息。例如，在Google Talk上使用Pidgin-OTR，Google将不会将您的消息存档在Gmail中-它甚至无法理解它们。只有进行会话的计算机才能理解消息。这确保了您的聊天历史记录的内容-如果您选择保留聊天日志，您的聊天日志-在您的计算机上保持完全脱机-当然还有您对话伙伴的计算机。

可悲的现实是，离线桌面程序的开发支持正在消失。Mozilla已经结束了对Thunderbird新功能的开发，微软将不再致力于他们的Windows软件包套件，我们预计谷歌将在未来几年内取消对Picasa桌面版本的支持。甚至连Microsoft Office都在使用Office Web Apps在线移动，并默认保存到SkyDrive。最活跃的开发方向是网络应用程序或支持在线的桌面应用程序。但如果你想逆水行舟，这些应该能让你游上一段时间。
