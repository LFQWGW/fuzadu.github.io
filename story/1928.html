帮助在WordPress博客上发布内容的工具

现在你已经有了一个很好的博客，你想要对它做更多的事情并开始发布内容。下面我们来看看一些工具，这些工具可以让你直接发布到你的WordPress博客上。

使用WordPress写一篇新的博客文章很容易，就像我们在上一篇关于开始自己的WordPress博客的文章中看到的那样。网络编辑器为你提供了很多功能，如果你喜欢黑客攻击HTML，甚至可以让你编辑文章的源代码。虽然还有其他工具可以让你发布内容，但这里我们来看看如何使用专用的应用程序、浏览器插件，甚至通过电子邮件发布内容。

Windows Live Writer

Windows Live Writer(Windows Live软件包套件的一部分)是一款用于将内容发布到您的博客的伟大应用程序。这款适用于Microsoft的免费程序允许您将内容发布到各种博客服务，包括Bloger、TyPepad、LiveJournal，当然还有WordPress。您可以直接从其类似Word的编辑器中撰写博客帖子，并配有图片和高级格式。即使您处于脱机状态，您仍可以撰写帖子并保存它们，以便再次联机时使用。

有关安装Live Writer的更多信息，请查看我们关于如何在Windows7中安装Windows Live软件包的文章。

安装Live Writer后，打开它以添加您的博客。如果您已经为博客安装并配置了Live Writer，您也可以添加您的新博客。您只需在右上角单击您的博客名称，然后选择“添加博客帐户”。

选择“其他博客服务”将您的WordPress博客添加到Writer，然后单击“下一步”。

输入您的博客网址，以及您的用户名和密码。请勾选“记住我的密码”，这样您就不必每次写东西时都输入密码。

作者将分析您的博客并设置您的帐户。

在设置过程中，它可能会要求发布临时帖子。“这将允许您使用博客的真实主题预览博客帖子，这很有帮助，所以请单击”是“。

最后，添加您的博客名称，然后单击Finish。

现在，您可以使用富编辑器编写内容并将其添加到新的博客帖子中。

选择预览选项卡以查看您的帖子在博客…上的外观

或者，如果您是HTML极客，可以选择Source选项卡来编辑您的博客帖子的代码。

从窗口底部，您可以选择类别、插入标签，甚至可以将帖子安排在不同的日期发布。Live Writer与WordPress完全集成；使用桌面编辑器不会遗漏任何内容。

如果要编辑已发布的帖子，请单击打开按钮并选择该帖子。您可以选择和编辑任何帖子，包括通过Web界面或其他编辑器发布的帖子。

使用Live Writer向帖子添加多媒体内容

回到编辑选项卡，您可以从侧边栏添加图片、视频等内容。选择要插入的内容。

图片

如果插入图片，可以为其添加许多漂亮的边框和设计。

或者，您甚至可以从侧边栏的Effects选项卡添加艺术效果。

图片库

如果您想发布几张图片，比如您的一些度假照片，那么插入图片库可能是最好的选择。1选择侧边栏中的“插入照片库”，然后在图库中选择您想要的图片。

插入图库后，您可以从几种样式中选择来展示您的图片。

当您发布博客时，系统将要求您使用您的Windows Live ID登录，因为图库图片将存储在免费的SkyDrive存储服务中。

您的博客读者可以直接在您的博客上查看您的图片预览，然后可以通过链接在线查看每一张图片、下载图片或查看幻灯片放映。

录像 / 录像节目 / 录像带 / 影视图像

如果要将视频添加到博客帖子中，请如上所述从侧边栏选择视频。您可以选择已在线的视频，也可以从文件中选择新视频，然后直接从Windows Live Writer通过YouTube上传。

请注意，您必须使用您的YouTube帐户登录才能将视频上传到YouTube，因此如果您没有登录，当您单击Insert时，系统会提示您这样做。

Microsoft Office Word

Word 2007和2010还允许您将内容直接发布到您的博客。如果您已经键入了文档，并且认为它在您的博客上也会很好，那么这一功能尤其不错。请查看我们关于使用Word 2007作为博客工具通过Word 2007发布博客帖子的深入教程。

这在Word 2010中也适用，只是Office ORB已被新的后台视图取代。因此，在Word 2010中，若要开始新的博客帖子，请单击“文件\新建”，然后选择“博客帖子”。您可以像在Word 2007中那样添加博客设置并发布所需的内容。

或者，如果您已经编写了文档并想要发布它，请选择文件\共享(或在Word 2010的最终版本中保存并发送)，然后单击发布为博客帖子。*如果您尚未设置博客帐户，请按照Word 2007文章中所示进行设置。

通过电子邮件发布

我们大多数人每天都使用电子邮件，而且已经有了我们最喜欢的电子邮件应用或服务。无论是在台式机上还是在手机上，创建丰富的电子邮件和添加内容都很容易。WordPress可以让您生成一个独特的电子邮件地址，您可以使用该地址轻松地将内容和电子邮件发布到您的博客上。您只需编写电子邮件，主题为帖子的标题，并将其发送到这个唯一的地址。您的新帖子将在几分钟内发布。

若要激活此功能，请单击WordPress.com帐户顶部菜单栏中的“我的帐户”按钮，然后选择“我的博客”。

单击博客名称旁边的“通过电子邮件发布”下的“启用”按钮。

现在，您将有一封私人电子邮件可用于发布到您的博客。您发送到此电子邮件的任何内容都将作为新帖子发布。*如果您认为您的电子邮件可能已被泄露，请单击“重新生成”以获取新的发布电子邮件地址。

现在，任何电子邮件程序或WebApp都是博客帖子编辑器。我们可以随意使用丰富的格式或插入图片；这一切都很棒。这也是从移动设备向博客发帖的一种很好的方式。无论您是在手机上使用WebMail还是专用电子邮件客户端，您现在都可以从任何地方撰写博客。

移动应用

WordPress还提供直接从您的移动设备撰写博客的专用应用程序。*您可以在智能手机上撰写新帖子、编辑现有帖子和管理评论。*目前它们提供iPhone、Android和Blackberry应用程序。请单击下面的链接查看这些应用程序。

结束 / 结尾 / 结论 / 推论

无论您是想通过浏览器写作还是通过电子邮件将帖子发送到您的博客，WordPress都足够灵活，可以根据您的喜好进行操作。无论您发布什么内容，您都可以确保它看起来很专业，并且可以通过您的WordPress博客轻松访问。
