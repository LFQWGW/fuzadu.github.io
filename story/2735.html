你有没有想过你的操作系统是怎么得名的？

你有没有想过“XP”代表什么，或者“Ubuntu”是从哪里来的？一些操作系统的名字来自显而易见的地方，但其他操作系统需要一些解释。继续阅读，找出你最喜欢的操作系统的名字是从哪里来的。

我们已经收集了最流行和最知名的操作系统，以及一些不太知名的操作系统-如果你知道另一个操作系统的名字后面有一个有趣的故事，请务必在评论中告诉你的其他读者。

微软磁盘操作系统

微软盘操作系统(MS-DOS)最初是从西雅图消费品公司的Quick and Dirty Operating System(QDOS)获得许可的。MS-DOS没有图形用户界面(GUI)，取而代之的是用户通过命令提示符输入所有内容。DOS首字母缩写并不总是指MS-DOS。DOS是“磁盘操作系统”的通用术语，MS-DOS只是苹果DOS、AmigaDOS、FreeDOS等众多操作系统中的一个。

Windows 1.0-NT 4.0

Windows的名字来自于这样一个事实，即新的操作系统Windows1.0具有用户可以在GUI中与之交互的窗口。“Windows”名称一直延续到当前的操作系统，“Windows”后面的数字或名称是如何判断您的Windows版本的。

当微软开始在其操作系统的新技术(NT)分支上工作时，它坚持1.0-3.1的编号方案。Windows NT 4.0是第一个实现新NT内核的版本，并于1996年使用Windows 95用户界面发布。

Windows 95-ME

微软在很短的一段时间内开始给他们的Windows操作系统命名，用发布的年份来表示版本。Windows95、98和Millennium Edition都是在最初的Windows3.1内核基础上构建的。Millennium Edition于2000年发布，设计用于家用计算机，而Windows2000则设计用于商用计算机。

Windows 2000使用NT内核，是NT 4.0的直接后继者，后来被Windows XP取代。

Windows XP、Vista和Windows 7

Windows XP于2001年发布，XP代表体验。这种体验一直持续到2007年微软发布Windows Vista。根据微软的说法，vista之所以得名，是因为“归根结底，…。你想要达到的是你自己的个人Vista。“。这对微软来说是非常有哲理的，但最终他们在Windows7上采取了一种更简单的方法。

Windows7是自95年以来的第七个Windows版本。即使95不是Windows的第一个版本(如果你算1.0-3.1)，但Windows10就是不一样了。另外，人们可能会把它与MacOS10混淆。

系统1-7.5

苹果的系统操作系统是为麦金塔个人电脑开发的。操作系统一直遵循基本的版本编号方案，每个主要版本的增量为1点，而次要更新的增量为0.1点。

苹果麦金托什电脑于1984年发布，其名字来源于麦金托什苹果。

每个新版本都以增量更新和功能增强为特色，并支持较新的苹果电脑。System 1-4没有多任务支持，这对另一个苹果操作系统来说可能听起来很熟悉。

该操作系统的前七个版本被简单地称为“System 1-7”。直到7.6，“系统”一词被去掉，取而代之的是一个更有市场的“MacOS”名称，这一点一直是正确的。

Mac OS 8-10

MacOS8是在史蒂夫·乔布斯回来为苹果工作后不久于1997年问世的。MacOS8本应被称为7.7版，但由于一个法律漏洞，允许苹果关闭第三方Macintosh市场，因此被改为8版。

MacOS9现在被认为是最后一个“经典”MacOSS，是基于苹果内部代码的最终版本。

OS X(OS Ten)是Mac操作系统的第十个版本，也是第一个基于Berkeley软件分发(BSD)的Mac OS。底层操作系统是由史蒂夫·乔布斯在离开苹果期间创办的NeXTSTEP公司开发的，该公司于1997年被苹果收购。

OS X的每个次要版本(10.1、10.2等)。还有一个与之相关的大型猫科动物的名字。虽然有人猜测苹果的这些名字是从哪里来的，但这是一个相当大的巧合，该操作系统将与Shaye开发的现已贬值的Macintosh克隆系统同名，而且关于OS X名称和德国坦克的论点也相当令人信服。

<计算机>Linux 操作系

Linux本身的名字来源于Ari Lemmke，他负责运行最初的Linux内核上传到的FTP服务器。Linux内核的创建者Linus Torvalds想将内核命名为Freax，但是Ari给了他一个名为“Linux”的文件夹来上传他的内核。Linux的名字来自“Linus的Minix”，这是Linus试图为自己取代的操作系统。

<南非>人的本质 / 人性 / 同情心

Ubuntu(oo-boon-Too)来自非洲的祖鲁语和科萨语，大致意思是“人道待人”。

与许多Linux发行版一样，Ubuntu发行版对每个发行版都有两个名称，一个开发代码名和一个版本名。版本的数字名称基于发布的年份和月份，例如，版本10.04是在2010年4月发布的。每两年还会发布一个长期支持(LTS)版本，该版本带有LTS字母，并有一个用于修复错误的增量版本。目前长期支持的版本是Ubuntu 10.04.2 LTS。

代号以同一字母的形容词和动物的字母递增为基础。动物命名出现在开发的早期，通常反映了当前Ubuntu发行版的状态，例如，Warty Warthog是第一个表示其缺乏润色的发行版。自从微风獾(5.10)发布以来，一直是按字母顺序排列的。

Ubuntu也有衍生工具，它们要么具有特定的焦点、位置，要么在需要不同的发行版名称的操作系统中有很大的不同。一些官方衍生品是Kubuntu、xusbuntu、mythbuntu和eubuntu。所有官方衍生品都遵循与Ubuntu相同的命名方案。

还有一些Ubuntu定制，比如Linux Mint、Lspirire和Ubuntulite，它们都有自己的代码名和版本号。

Debian

Debian的名字来源于Debian发行版的创建者伊恩·默多克(Ian Murdock)。他以女友黛布拉·林恩(Debra Lynn)的名字命名这个发行版，将她的名字和他的名字合并为“Debian”。

每个稳定的Debian版本都有一个相关的版本号(1.0、2.0等)，但它也有一个基于1995年电影“玩具总动员”中角色的代号。最新的6.0版本是以玩具外星人“挤压”命名的。Debian的开发分支被永久命名为“SID”，来自电影中情绪不稳定的邻居。

Red Hat/Fedora/CentOS

红帽Linux之所以得名，是因为红帽在美国和法国都是自由和革命的象征。Red Hat有几种不同的风格，Red Hat Enterprise Linux(RHEL)、Fedora Project和CentOS

Red Hat Enterprise Linux不使用代码名，而只是使用类似于OSX的版本号。

软呢帽得名于红帽子的标志，标志中的影子人戴着一顶红色的软呢帽。Fedora 1-6被称为Fedora Core，而后来的版本从名字中去掉了“core”。

Fedora使用与上一版本相关的代码名。每个版本都与第一个版本相关，与其前一个版本具有“is-a”关系。例如，版本7的“月光男孩”和版本6的“佐德”一样是唱片公司，但它也是版本8的电影“狼人”。

CentOS是社区支持的RHEL版本。该名称来自Community Enterprise Operating System，是最流行的Web服务器操作系统之一。CentOS的版本号遵循RHEL版本编号，没有代码名。

白眉企鹅

Gentoo在将其Enoch Linux发行版重命名后得名于Daniel Robbins。为了增加特性和速度，当Daniel和其他开发人员开始使用EGCS编译器而不是GCC编译器时，Gentoo第一次被重命名。Gentoo是以游泳最快的企鹅--Gentoo企鹅命名的。

风格化的“g”标志类似于日本的银色Magatama，经常被发现在墓地作为祭祀神灵的物品。

SuSE

SuSE是一种德语发行版，最初代表“Software and system Entwickrun”，在英语中翻译为“Software and system Development”。自1998年以来，这个缩写词不再使用，公司的名称只有SUSE。

SUSE Linux最初的发行版是4.2版，而不是1.0版，参考了“银河漫游指南”中的“生命、宇宙和万物的终极问题的答案”。

SuSE的吉祥物是一只戴着面纱的变色龙，取名为“Geeko”，取自“壁虎”和“极客”两个词。

SuSE现在归Novell所有，免费的Linux发行版称为openSUSE。
