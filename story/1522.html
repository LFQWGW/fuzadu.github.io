如何将Office 2010文档上传到Web应用程序技术预览

Office 2010的主要新功能之一是能够使用Office Web Apps服务在线共享和编辑文档。今天，我们来看看如何将它们保存到您的SkyDrive，以便在Office Web Apps中使用。

注意：若要上载到Office Web Apps，您需要安装Office 2010 Beta版。

将文档上载到SkyDrive

在Windows Live中打开要上载到Web应用程序的文档，然后单击文件选项卡打开后台视图。在此示例中，我们将Word文档上载到Windows Live SkyDrive。

在后台打开的情况下，单击左列上的共享，然后单击保存到SkyDrive。

在Save to SkyDrive(保存到SkyDrive)下，单击Sign In(登录)按钮。

此时将打开一个登录对话框，让您输入您的Windows LiveID凭据。

登录后，将显示SkyDrive中的文件夹列表。突出显示要保存文档的文件夹，然后单击“另存为”按钮。在本例中，我们将此Word文档保存在SkyDrive的“Testing Docs”文件夹中。

连接到SkyDrive时，您将在Word底部的状态栏上看到正在联系的URL。

键入文件名，选择文件类型，然后单击保存按钮。

当文档保存到SkyDrive时，您将在进度条旁边看到消息“正在上传到服务器”。

访问SkyDrive上的文档

若要访问该文档，请转到SkyDrive和保存该文档的文件夹。

单击您刚刚上传的文档。

现在单击查看以打开文档。

下面是如何在Firefox的弹出屏幕中查看文档。在撰写本文时，Office Web Apps仍处于技术预览阶段，没有任何Word文档编辑功能，只有查看和下载功能。

结束 / 结尾 / 结论 / 推论

虽然Office Web Apps仍处于技术预览阶段，但这会让您对预期的情况有所了解。目前，您可以使用Web应用程序共享Word、Excel、PowerPoint和OneNote文档，每个应用程序都具有不同程度的编辑功能。如果你是早期采用者，目前正在测试Office2010Beta，这应该会让你开始上传文档，这样你就可以探索Web应用程序提供了什么。
