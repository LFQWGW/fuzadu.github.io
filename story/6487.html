为什么微软的滚动广告对Chromebook的评价是错误的

微软的Scroogled活动对Chromebook的评价是错误的。Chromebook绝对不是每个人都适合的，但它们也不是完全没用的。而且Chromebook与微软的愿景有更多的相似之处，而不是微软想要承认的。

如果Scrogled广告给人的感觉像是政治攻击性广告，那是因为它们基本上就是政治攻击性广告。他们由一位曾进行过政治竞选的人监督。对谷歌来说幸运的是，广告传达的主要信息是Chromebook的存在。

“当您没有联网时，它几乎就是一块砖头。”

相关：如何在Chromebook上脱机工作

在典当之星的广告中，微软辩称，当你没有连接到互联网时，Chromebook“几乎就是一块砖头”。这完全不是真的，因为Chromebook的离线支持比你想象的要多。电子邮件、日历、文档、PDF、电子书、视频、图像、音乐，甚至一些游戏--所有这些都可以在Chromebook上离线使用。Chromebook的离线功能令人惊讶。Chromebook正在获得完全离线工作的新型Chrome应用程序。

当然，Chromebook在网上更有用，但Windows笔记本也是如此。如果你和大多数人一样，你很有可能在你的Windows PC上使用相当多的Web应用程序，并且花很多时间在浏览器上。当Windows PC也没有连接到互联网时，对许多人来说，它“几乎就是一块砖头”。

“这不是一台真正的笔记本电脑。它没有Windows，也没有Office。“

Macbook和基于Linux的笔记本电脑“不是真正的笔记本电脑”吗？微软希望你这么认为，但他们并不垄断“真正的笔记本电脑”。Chromebook应该根据它们的实际功能进行评估。

相关：不再收取升级费用：使用Google Docs或Office Web Apps代替Microsoft Office

对微软来说，真正可怕的是：Chromebook确实有Office！微软提供了一个完全免费的基于Web的Office版本，称为Office Web Apps。这些都集成到了微软的SkyDrive网站上。Windows上的Office 2013甚至在默认情况下保存到SkyDrive，因此您的文档可能已经在那里可用。

微软在这里说出了他们的两个嘴巴。对于Google Docs的用户来说，他们认为Office Web Apps功能非常强大，应该优先考虑，因为它与Office文档格式有很好的兼容性。对于Chromebook的购买者，微软表示根本没有办法使用Office。

当然，与Google Docs不同，Office Web Apps有更多的限制，只有在连接到Web时才能使用。但Office Web Apps是存在的，微软认为它是一款功能非常强大的产品，提供了与Office文档格式惊人的兼容性。Chromebook确实有Office。它是Office的一个更有限的版本--但是Windows RT也有一个Office的有限版本。您实际需要使用Office做多少工作？如果您和大多数人一样，使用Office Web Apps可能就足够了。

“没有Wi-Fi，它根本做不了什么。”

值得一提的是，一些Chromebook带有3G移动网络接入。例如，三星销售的“Chromebook 3G”售价为330美元。它还包括Verizon的3G网络每月免费提供100MB的移动数据。如果你购买了一部支持3G移动数据的Chromebook，你仍然可以在Wi-Fi之外使用它。

“当你在线时，谷歌会跟踪你的行为，这样他们就可以卖广告了。”

谷歌跟踪你在Chromebook上做的事情，就像你在Windows上使用Chrome一样多。如果你真的担心谷歌跟踪你，你可能不想使用Chromebook或谷歌Chrome。即使你使用的是IE浏览器，谷歌分析和许多其他广告网络每天都在跟踪你。

如果微软如此关心你的在线隐私，他们很可能不会与Facebook合作，也不会将Facebook整合到他们的必应搜索引擎中。更具争议性的是，我们可能会指出，微软是第一家遵守美国国家安全局PRISM计划的公司。

不管你是否同意美国国家安全局的监控，重点是这个问题非常混乱。谷歌的广告跟踪真的是最需要担心的事情吗？还有很多其他的广告网络和社交媒体网站在网络上跟踪用户-更不用说像美国国家安全局这样的情报机构正在建立我们所有活动的大规模数据库。“Scroogled”是一个过于简单化的词。

谷歌也不是唯一一家想要向你展示广告的公司。微软甚至在其许多默认的Windows8应用程序中加入了广告。

“这不是…看起来的那样。这不是真正的笔记本电脑“

微软还特别批评Chromebook，因为“它不是看起来的那样”，“不是真正的笔记本”。微软在这里对Windows RT应用了双重标准。

相关：什么是Windows RT，它与Windows 8有何不同？

Windows RT看起来与Windows 8一模一样--事实上，Windows RT甚至配有Windows桌面。然而，Windows RT并不是它看起来的那样--它不是一个“真正的”Windows版本，你不能在它上面安装任何Windows桌面程序。您只能使用Windows应用商店中的新“Windows 8风格”程序。

微软甚至取消了iTunes的名字，称其为一款在传统PC上可用但在Chromebook上不可用的应用程序。然而，iTunes不能在微软的Windows RT上使用，而且很可能永远也不会使用。如果Chromebook上缺少iTunes是个问题，为什么Windows RT不是问题呢？

相关：为什么连微软都认为Windows桌面是一场噩梦的8个原因

这里真正的问题是，微软的愿景与谷歌如此接近。微软展望了云计算的未来和更有限的设备，并看到Windows桌面的终结在望。这就是他们推动Windows RT、Surface RT和Surface 2的原因。微软并不真正相信Windows桌面的长期未来。这使得微软和Windows RT与Google和Chrome OS处于更加均衡的水平。

典当行会买Chromebook吗？

相关：你应该买Chromebook吗？

很多典当行都不会买Chromebook。但许多典当行根本不会购买笔记本电脑，因为转售价值如此之低。也许广告中的女人应该让她的母亲把Chromebook归还，并要求退还她的钱。这是一个比试图典当礼物去好莱坞更好的主意-当然，她是一名女演员，这一切都不是真的。她可能已经在好莱坞了。

对微软来说，不幸的是，他们在这里所做的一切只是让Chromebook在更多人的心目中成为竞争对手。你可能不想要Chromebook，但不要让微软的政治攻击性广告告诉你如何思考。你自己想想吧。
