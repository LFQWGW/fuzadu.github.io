如何将您的Android手机降级到以前的版本

当新的Android版本推出设备时，并不是每个人都欣赏这些变化。如果你想换回来，有时可以把你的Android设备降级到以前的版本。这就是你怎么做的。

在我们开始之前，提到风险是很重要的。通常不支持将Android手机降级，这不是一个容易的过程，而且几乎肯定会导致设备上的数据丢失。请确保在开始之前备份手机。

此外，这一过程可能会使你的设备保修无效，或者可能会让你的手机变成电子垃圾。因此，我们不建议您继续，除非您有修改设备固件的经验并完全了解涉及的风险。

在闪烁你的手机之前

安卓手机降级有多容易取决于制造商。一些制造商让用户非常容易地将固件映像“闪存”到他们的设备上，而另一些制造商则几乎不可能做到这一点。

刷新的典型过程要求您首先解锁引导加载程序。并不是所有的Android手机都允许这样做，所以你可以自己检查一下你的设备制造商是否提供了一种官方的方法来解锁引导加载程序，或者你是否需要找一个替代的方法。

请记住，解锁引导加载程序会擦除设备的内部存储。确保您的手机已备份，否则您的数据将永远丢失。

相关：如何解锁Android手机的Bootloader，官方方式

启动加载程序解锁后，您可以将您的设备刷新到较旧版本的Android。请注意，解锁引导加载程序可能会使您的保修失效。

到目前为止，最容易闪存的设备是Google Pixel系列的手机。谷歌为Pixel设备提供工厂映像，针对Android 9和10有不同的版本。你需要安装Android SDK-带有Fastboot和Android Debug Bridge(ADB)工具-才能刷新Google Pixel设备。

然而，对于其他设备制造商来说，这是一个喜忧参半的问题。除非制造商提供，否则你需要找到适合你的设备的固件镜像。XDA-Developers论坛是为你的特定Android手机找到合适的降级固件镜像的好地方，但也要在制造商的网站上搜索官方固件镜像。

使用Fastboot方法可以刷新非谷歌设备，尽管三星等其他Android设备制造商确实可以使用第三方工具。

使用Fastboot和ADB手动刷新

给你的Android设备降级的唯一“官方”方法是快速启动(Fastboot)方法。虽然不是每个制造商都鼓励这样做，但你可以从谷歌获得Fastboot工具供你使用，而且应该可以与几乎所有Android制造商的官方固件镜像一起使用。

要继续操作，您只需解锁引导加载程序，并具有适合您的设备和运营商的固件映像。*在开始之前，请确保您的Android设备上启用了USB调试。

相关：如何访问开发者选项并在Android上启用USB调试

如果您已经找到合适的固件镜像，您的引导加载程序已解锁，并且您已经安装了Android SDK，请将您的Android手机连接到您的PC。虽然以下步骤适用于Windows10用户，但ADB和Fastboot命令在MacOS和Linux上的工作方式应该类似。

打开文件资源管理器，转到Android SDK安装的保存位置。确保您需要刷新的Android图像文件也在这里。这些文件通常以ZIP文件的形式出现，其中包含多个IMG文件。将内容解压到此位置。

在文件夹中，按住键盘上的Shift键，右键单击窗口内部，然后单击“在此处打开PowerShell窗口”按钮。在MacOS或Linux上，使用终端应用程序导航到此文件夹。

在Windows PowerShell窗口中键入ADB Devices，以确保检测到您的Android设备。如果是，请键入ADB reboot bootloader*重新启动您的设备并引导到Android Bootloader菜单。

如果ADB命令不起作用，请键入.\adb。

大多数制造商的固件都带有一个“flash-all”脚本，可以将所有相关的图像文件刷新到您的设备上。设备进入引导加载程序模式后，键入flash-all开始刷新过程。

如果flash-all脚本不起作用，您需要手动刷新文件夹中的各个项目。在Windows PowerShell(或终端)中键入以下命令，并在每个步骤之间按Enter键：

*fast boot flash bootloader<bootloader name>.img。 *fast boot reboot-bootloader。 *fast boot flash Radio<Radio file name>.img。 *fast boot reboot-bootloader。 *fast boot flash-w update<镜像文件名>.zip

此过程完成后，您的电话应该会重新启动。如果刷新过程成功，您的设备现在将运行降级的固件映像。

使用第三方闪存工具

多亏了志愿开发人员的努力，第三方闪存工具已经面世。这些特定于制造商的固件刷新工具为您提供了一种无需使用快速引导方法即可刷新固件的替代方法。

如果您的设备没有第三方工具，最好使用上面列出的快速引导方法。

三星设备可以利用Odin对其手机进行降级。这是官方Fastboot方法之外最知名的闪存工具之一。

ODIN界面不是最适合初学者使用的，但它可以让你在一些三星设备上闪存，包括最新解锁的Galaxy手机。

请记住，一些蜂窝运营商会锁定通过其商店销售的设备上的引导加载程序。这些网络运营商可能不愿允许用户解锁他们的手机，除非手机完全付费。

相关：如何使用ODIN手动更新您的三星手机

其他制造商还可以使用其他工具。XDA Developers的这篇文章列出了摩托罗拉设备的各种闪存工具，包括推荐的RSD Lite。华为设备用户可以改用华为恢复更新程序(HuRuUpdater)。

我们再怎么强调这一点也不为过：这些工具不能保证有效，而且它们也没有得到制造商的批准。无论您使用的是第三方工具还是快速启动方法，设备被烧毁的风险都是相同的。

同样，在开始之前，请确保在尝试刷新设备之前已经备份了手机，因为在刷新过程中手机将被擦除。
