Windows10 20H1更新的新特性将于2020年春季发布

微软已经在着手Windows10的20H1更新。预计在2020年5月左右的某个时候，这是Windows10 2004版。它比Windows10 2019年11月的更新大得多，但感觉仍然像是一系列有用的改进。

这篇文章是关于2020年2月11发布的Windows Insider 19041.84版本中包含的最新功能的最新消息。我们最初是在2019年8月28日发布的，在微软的整个开发过程中一直在更新。

更好地控制可选更新

Windows Update会自动安装许多更新，但有些更新是可选的。现在，有了一个新的屏幕，在一个地方显示所有这些更新。

硬件驱动程序更新，像20H1本身这样的大功能更新，以及像C和D更新这样的每月非安全质量更新都会出现在这里。

要在更新到20H1后找到此屏幕，请转到设置>更新和安全>Windows更新>查看可选更新。然后，您可以选择可能要安装的更新。

Windows Update仍将自动安装许多硬件驱动程序更新，但有时可能不会自动安装其他更新。在过去，您必须仔细查看设备管理器并选择要更新的特定设备。现在，所有可选的硬件驱动程序更新都将显示在此屏幕上。微软说，“如果你有问题，那些可选的驱动程序中的一个可能会有帮助。”

全新的Cortana体验(打字)

微软正在宣传一种“全新的基于聊天的用户界面”的“新Cortana体验”。现在，您可以向Cortana键入查询，而不是大声说出来。您与Cortana的对话历史将显示为一个聊天窗口，因此您只需从任务栏打开Cortana即可查看最近查询的结果。

Cortana面板现在也是一个更普通的窗口。你可以像其他窗口一样，通过拖动标题栏来调整它的大小，并在桌面上来回移动它。此外，它还支持Windows10的浅色和深色主题。

微软表示，除了新的设计之外，它还“用新的语音和语言模型更新了Cortana”，并“显著提高了”语音助手的性能。而且，很快，微软表示，即使你的Windows操作系统被设置为使用Cortana不支持的显示语言，你也可以在它支持的任何语言中使用Cortana。

微软已经暂时删除了Cortana的一些功能，如笑话、计时器和必应即时回答。它目前专注于一项基于工作效率的功能。

用于重新安装Windows的云下载

Windows10有一个新的“云下载”选项，您可以在将PC重置为默认Windows系统时使用。当你进入“设置”>“更新与安全”>“恢复”并选择重置电脑并删除所有内容时，你现在就可以告诉Windows使用“云下载”了。Windows将下载最新版本的Windows 10并将其安装在您的系统上，而不是从本地系统上的文件重新安装Windows 10。

这将节省以后更新的时间。以前，要做到这一点，唯一的方法是在“重置”系统之前更新Windows10，或者创建新的Windows10安装介质。Windows10的重置功能变得更加强大。

Windows Update的带宽限制

现在，“设置”应用程序可以让你更好地控制下载Windows更新所使用的带宽。在当前版本的Windows中，您可以将带宽限制设置为带宽的百分比。Windows1020H1将允许您以Mbps为单位设置精确的“绝对带宽”限制，以便更精确地限制下载的更新。此选项以前在组策略中可用，但现在所有人都可以在“设置”中使用。

要在任何版本的Windows 10上查找更新带宽限制选项，请转至设置>更新和安全>交付优化>高级选项。

带有Linux内核的WSL 2

新的Windows Linux子系统(WSL)可能是Windows1020H1中最重要的功能。这是WSL版本2，它比第一个版本更强大。WSL2使用真正的Linux内核在Windows10上提供更强大、功能更全的Linux环境。

微软将构建自己的Linux内核，并将其与WSL2一起发布，该Linux内核将通过Windows Update进行更新。您还可以构建自己的Linux内核并让Windows10使用它。不过，您不必考虑这些-WSL2具有与WSL1相同的用户体验，并且无需任何额外配置即可“正常工作”。

WSL2承诺“显著提高文件系统性能”和“完全系统调用兼容性”。这种兼容性意味着支持像Docker这样的技术，这些技术不能在最初的WSL1上运行。

除此之外，微软还增加了对ARM64设备的支持-换句话说，WSL现在可以在ARM PC上的Windows上运行。WSL发行说明说，只有“如果你的CPU/固件支持虚拟化”，这才会起作用。

此外，还提供了更多配置选项。例如，您可以在Linux发行版的/etc/wsl.conf文件中设置默认用户帐户。

更快的Windows搜索体验

Windows 10的2019年5月更新修复了开始菜单搜索。微软通过利用旧的Windows搜索索引器做到了这一点，该索引器在后台运行，扫描你电脑的文件来创建搜索数据库。

微软询问内部人士为什么要关闭搜索索引器，并收到了三个需要改进的主要方面：“过多的磁盘和CPU使用率，普遍的性能问题，以及索引器的低感知价值。”微软表示，它现在正在检测高峰使用时间，以便更好地优化索引器运行的时间。例如，当游戏模式开启、节电模式开启、低功耗模式开启、CPU使用率约为80%、磁盘使用率高于70%或电池电量低于50%时，它将不会运行。

默认情况下，Windows搜索在开发人员PC上也会变得更快。Windows搜索索引器现在将“默认排除常见的开发人员文件夹，如.git、.hg、.svn、.Nuget等。”编译和同步代码时，性能将会提高。

任务管理器中的磁盘类型

Windows 10的任务管理器现在可以显示您的磁盘类型-固态硬盘或硬盘。这样可以更容易地查看计算机中的硬件，并且如果系统中有多个磁盘，它还可以帮助您区分哪个磁盘是哪个磁盘。

此信息显示在Performance(性能)选项卡上。打开任务管理器(Ctrl+Shift+Esc)，点击[更多详细信息]即可找到。

任务管理器中的GPU温度

这并不是任务管理器获得的唯一新功能。任务管理器的性能选项卡也将显示您的GPU温度。假设您的显卡带有足够新的驱动程序-它必须支持WDDM2.4驱动程序模型-您也可以在性能选项卡下的GPU状态页面中找到此信息。这也仅适用于专用显卡，不适用于集成或板载GPU。

这只是任务管理器中最新的GPU监控功能。以前的更新添加了一些功能，如每进程GPU使用率、总体GPU使用率显示、图形驱动程序版本信息、图形内存使用率和硬件详细信息。

全新的“平板电脑体验”

Windows10有经典的桌面模式和Windows8风格的平板模式，默认情况下甚至会隐藏任务栏图标。这对许多人来说并不理想，所以20H1增加了一种新的介于两者之间的“平板电脑体验”。

如果您使用的是带触摸屏的2合1 PC，并且没有连接键盘或鼠标，它可以使传统桌面界面更易于使用。例如，任务栏图标将进一步分开，文件资源管理器将针对触摸进行优化，并且您可以使用桌面上的窗口。

微软表示，这不是平板电脑模式的替代品，但当你拆下键盘或翻转键盘时，可转换电脑将不再自动进入平板电脑模式。取而代之的是，他们将进入这种全新的触控优化体验。微软正在放弃2合1设备的平板电脑模式，使经典的Windows桌面在触摸屏上更容易使用。

Xbox游戏栏中的FPS

Windows10的新游戏栏已经更像是一个全屏覆盖，功能丰富，包括快速音量控制、性能图表，甚至Spotify集成。现在，有了FPS计数器和成就覆盖，它变得更好了。

在玩游戏时按Windows+G打开游戏栏，您将看到一个实时的FPS计数器，无需安装FRAPS等第三方应用程序或激活特定于游戏的选项。

Windows让你“让你的设备无密码”

Microsoft new允许您在设置>帐户>登录页面上添加一个新选项，从而“让您的设备无密码”。这听起来很棒，很有未来感，但实际上这只是一个新的设置，要求你电脑上的每个人都要使用PIN或其他Windows Hello登录方法，比如脸或指纹解锁。

安全模式现在也适用于PIN登录。如果您已将Windows Hello设置为使用PIN登录，则在启动到安全模式后，您将能够使用该PIN登录到您的电脑。以前，安全模式会让您输入帐户密码。

重命名虚拟桌面

Windows10的虚拟桌面在按键盘上的Windows+Tab或单击任务栏上的任务视图图标时出现的任务视图界面中可用，它们的可配置性越来越强。

您现在可以重命名它们，而不是使用默认名称“Desktop 1”、“Desktop 2”等等。只需单击任务视图界面顶部的每个虚拟桌面的名称，然后键入新名称即可。

正如微软指出的那样，你甚至可以在名字中使用表情符号。只需按Windows+。(句号)打开表情符号选取器并输入表情符号。在Windows10中，几乎所有文本字段都可以使用此表情符号面板。

改进的网络状态信息

“设置”>“网络和Internet”>“状态”中的网络状态页已重新设计。现在，它会在页面顶部显示您可用的所有网络接口。例如，如果您的PC同时安装了Wi-Fi和以太网，则此处将同时显示Wi-Fi和以太网。

微软表示，这个新界面将“提供更多有关设备连接的信息，将几个页面结合在一起，让你清楚地看到你是如何连接到互联网的。”

Windows还将在此页面上显示每个界面的数据使用情况，因此您不必转到“设置”中的其他位置来查看您正在使用的数据量。

对网络摄像头的内置支持

Windows 10正在获得对基于IP的摄像机的内置支持，这些摄像机通过您的本地网络发送其视频源。传统上，您需要第三方软件才能在Windows 10上查看这些摄像头视频源。

有了此更新，您将能够通过转到“设置”>“设备”>“蓝牙和其他设备”>“添加蓝牙或其他设备”来添加基于网络的摄像头。如果您的本地网络上有受支持的摄像头，Windows 10会找到它，您只需单击一下即可将其添加到您的系统中。

添加后，您可以使用内置摄像头应用程序(和其他摄像头应用程序)访问网络摄像头。目前，Windows 10只支持使用ONVIF配置文件S的符合标准的摄像头。

更好地控制登录时重新启动应用程序

重启电脑后，Windows10会自动重新打开许多应用程序，包括Google Chrome。现在有了一个新的选项，可以更轻松地禁用此功能。

要查找此选项，请转到“设置”>“帐户”>“登录选项”。在“重新启动应用程序”下，切换“在我注销时自动保存可重新启动的应用程序，并在我登录后重新启动它们”(如果您想要关闭此功能)。

此前，该选项被隐藏起来，并与“使用我的登录信息自动完成我的设备”选项合并在一起，该选项显示为“使用我的登录信息自动完成设备设置，并在更新或重启后重新打开我的应用程序”(Use My Sign-in Info to Automatic Finish My Device)。现在，这是两个独立的选项。

该功能现在也能更好地工作。它现在重启了“大多数”UWP应用程序以及传统的Windows桌面应用程序。

磁盘清理不会删除您的下载文件夹

Microsoft正在从经典磁盘清理应用程序中删除Downloads文件夹。此选项已随2018年10月的更新添加到磁盘清理中。批评人士说，意外删除Downloads文件夹中的所有文件太容易了，特别是如果你是一名经验丰富的用户，没有意识到该选项已添加到磁盘清理中。

使用此更新时，Downloads文件夹将从磁盘清理中消失。您仍然可以使用磁盘清理来清空回收站、删除临时文件、删除旧的Windows安装以及其他所有操作-但短期下载选项消失了。

清理下载文件夹的选项在Storage Sense中继续存在，可在设置>系统>存储>配置Storage Sense或立即运行它。它只是从经典的磁盘清理界面中消失了。

画图和写字板现在是可选功能

微软已将MS Paint和写字板变成“可选功能”，默认情况下仍会安装MS Paint和WordPad，但可以将其卸载以释放一点空间.

进入“设置”>“应用程序”>“应用程序和功能”>“可选功能”，你会看到画图和写字板以及其他可选功能，比如Windows Media Player。

Microsoft Paint占用11.6MB，而WordPad占用9.11MB，因此删除它们不会释放太多空间。微软原本打算从Windows上移除Paint，并通过商店分发，但它放弃了这些计划，甚至用新功能更新了Paint。

Windows 10设置应用程序中的标题

微软在设置应用程序中尝试横幅已经有一段时间了，它又回到了20H1 Insider版本中。新横幅显示在“设置”窗口的主屏幕顶部，显示您的图片、名称和用于联机管理您的Microsoft帐户的链接。它提供指向您的OneDrive和Windows Update设置以及有关其状态的信息的快速链接。

值得庆幸的是，微软这次没有包括微软奖励(前身为必应奖励)的广告。

在搜索主页中进行快速搜索

当你点击任务栏上的搜索框打开“搜索主页”面板时，你会在底部看到新的“快速搜索”，这样你就可以一键访问天气、热门新闻和新电影等内容。

蓝牙SWIFT对的改进

微软正在改进Windows10更快的蓝牙配对体验，它以前被称为Quick Pair，现在显然被称为Swift Pair。

当您附近的受支持设备处于配对模式时，您将看到一条通知，提示您完成配对。这是添加到Windows 10 2018年4月更新中的。现在，它得到了进一步的精简。整个配对过程都是通过Windows10的通知执行的，不需要打开设置应用程序，而且会少显示一条通知。如果您不想配对设备，可以使用取消按钮关闭通知，如果可能，通知会显示有关设备名称和类型的更多信息。

这仍然只适用于支持的设备，如微软的Surface键盘和鼠标，但有望在未来应用到更多的设备上，使蓝牙配对过程对更多的PC用户来说更快。

文本光标指示器

现在，您可以调整Windows10文本光标指示器的大小和颜色--这条小线显示您在应用程序中键入的位置。

要查找此选项，请转到设置>轻松访问>文本光标。启用新的“文本光标指示器”，选择大小，并选择您容易看到的颜色。您可以选择所需的任何自定义颜色。

如果您对此选项感兴趣，您可能还希望调整鼠标光标的大小和颜色。微软在Windows10的2019年5月更新中增加了这一选项。

用眼睛拖放

Windows10有一个眼睛控制功能，可以与某些特定的眼睛跟踪设备配合使用。这是一个辅助功能，让你只需移动眼睛就可以控制你的电脑。在20H1，眼球跟踪将变得更加强大。现在，您只需移动眼睛即可执行鼠标拖放操作。

语言设置改进

Windows10的“设置”>“时间与语言”>“语言”中的语言设置页面已经过重新组织，使其更易于使用和理解。例如，它现在会在屏幕顶部显示Windows、应用程序和网站的默认选定语言、键盘、语音和区域设置。

此更新也包含了对非英语语言的更好支持。SwiftKey触摸键盘的“智能打字”功能现在支持39种不同的语言。这意味着更有用的自动更正和键盘文本预测。改进的文本预测甚至在您为硬件键盘启用文本预测时也有效。

听写也变得更好了。Microsoft现在支持在使用听写时使用更多语言-要使用听写，请在任何文本字段中键入时按Windows+H。

Microsoft在东亚Microsoft输入法编辑器(IME)方面做了大量工作。有一个新的日本输入法，并改进了中国和韩国的输入法。

令人惊讶的变化是，微软在8月份宣布将不再将记事本移至商店。现在，它将通过应用商店自动更新，使Microsoft能够更频繁地更新记事本，而不是每六个月更新一次。您也可以卸载记事本。

记事本仍将默认安装，因此不会有太多变化。微软一直在更新记事本，增加了UNIX行尾支持和集成的必应搜索等新功能。微软甚至想更频繁地更新记事本。

无论如何，这是最初宣布的计划。微软在12月份改变了主意，将记事本从商店中移除。记事本什么都没有改变--目前是这样。

在路上：您的手机应用程序中的呼叫

如果你的手机运行Android 7或更新版本的Android，Windows 10的Your Phone应用程序将允许你从PC拨打和接听电话。

微软一直在20H1 Insider版本中测试这一功能，但表示将在所有运行Windows10 19H1(2019年5月更新)或更新版本的PC上测试这一功能。即使您不升级到Windows10版本2004，您也会获得此功能。

已在此处：文件资源管理器中的联机文件搜索

这一功能最早出现在Windows10的20H1更新中，但它已经作为2019年11月更新的一部分提供。

在这两个版本的Windows 10中，文件资源管理器都有新的搜索体验。当您在搜索框中键入内容时，您将看到一个包含建议文件列表的下拉菜单。它还将在线搜索你的OneDrive帐户中的文件，而不仅仅是你本地电脑上的文件。

您仍然可以通过按Enter键访问更强大、更经典的搜索体验。例如，这将允许您搜索非索引位置。
