如果升级到Windows 8专业版，您将获得7项功能

Windows8.1邀请您“通过新版本的Windows获得更多功能”。如果你升级到专业版，你将获得更多6个功能-如果你额外付费，还会获得第七个功能。

Windows7的专业版与此类似。BitLocker、组策略和域加入等功能也需要Windows7专业版或旗舰版。(请注意，BitLocker需要Windows 7旗舰版)。

BitLocker加密

相关：如何在Windows上设置BitLocker加密

Microsoft的BitLocker加密功能仅在Windows 8专业版和企业版上可用。如果要使用BitLocker Drive Encryption对整个系统驱动器进行加密，使用BitLocker To Go对USB闪存驱动器进行加密，或者创建加密的容器文件，则需要Windows专业版。

对于使用任何版本Windows的人来说，TrueCrypt曾经是BitLocker的一个很好的替代品。TrueCrypt项目现在已经正式关闭，他们的网站建议你使用BitLocker，所以这件事还悬而未决。使用TrueCrypt可能仍然安全--这是一个有争议的话题。

新的Windows 8.1设备发货时启用了设备加密。这是一个不同的功能，只有当您使用将加密密钥备份到Microsoft服务器的Microsoft帐户登录时，此功能才有效。

组策略

相关：使用组策略编辑器调整您的PC

只有Windows专业版才有权访问组策略编辑器。这个高级工具可以用来调整许多通常无法更改的Windows设置。它还提供对许多其他设置的访问，这些设置通常需要更复杂的注册表攻击。例如，如果要禁用锁屏并让Windows 8.1直接转到登录屏幕，则需要在组策略编辑器中更改组策略编辑器中的设置或使用注册表攻击。

系统管理员通常使用此功能来锁定大型网络上的PC，但您也可以使用它来调整您自己的PC。大多数Windows用户不需要组策略编辑器，但调试者会很高兴拥有它。组策略中的许多可用设置都可以通过注册表黑客进行更改-但不是所有设置都可以。

远程桌面托管

相关：如何通过Internet访问Windows远程桌面

所有版本的Windows都包括用于远程连接到Windows计算机的远程桌面客户端软件。但是，只有Windows专业版包含远程桌面服务器软件，用于托管远程桌面服务器并接受传入连接。

如果您要远程连接到计算机的桌面，这会很有用。但是，第三方远程桌面程序是免费提供的。像TeamViewer这样的免费远程桌面选项更易于设置，以便通过Internet进行访问。微软的远程桌面软件需要端口转发，而TeamViewer使用简单的帐户系统，不涉及摆弄路由器。

Windows域加入

相关：什么是Windows域，它对我的PC有什么影响？

只有运行Windows专业版的计算机才能加入Windows域。域用于工作场所、教育机构和其它大型网络。如果你有一台工作或学校发给你的笔记本电脑，它可能是域名的一部分-否则你家里就没有域名。

在Windows专业版提供的所有功能中，限制此功能的原因最为明显。想要使用Windows域的企业必须在他们的个人电脑上支付更高的Windows专业版费用-他们不能只使用更便宜的“核心”版本。

超V

相关：如何在Windows 8或10中安装或启用Hyper-V虚拟化

在Windows8中，Windows现在包括一个名为Hyper-V的内置虚拟化工具。您可以使用Hyper-V创建虚拟机并在其上安装操作系统，就像使用VirtualBox或VMware一样。它们都集成到了操作系统中。Windows还提供了一个图形工具，称为Hyper-V管理器，用于管理这些虚拟机。默认情况下未启用这些功能，因此您必须从“打开或关闭Windows功能”控制面板安装它们。

这个功能可能对某些人有用，但是想要使用虚拟机的典型Windows用户使用免费的VirtualBox就可以了。

从VHD启动

相关：如何在不重新分区的情况下双引导Windows 7和8(使用VHD)

专业版的Windows可以从虚拟硬盘(VHD)文件启动。对于想要尝试其他操作系统的Windows用户来说，这是一个方便的功能。您可以将另一个Windows实例安装到Windows驱动器上的VHD文件中。然后可以将该VHD文件添加到您的引导菜单中，您可以直接引导到它。

这允许您安装其他版本的Windows并在本地启动它们，而无需处理驱动器分区的麻烦。其他版本的Windows也可以创建和附加VHD文件，但只有专业版可以从中启动。

Windows Media Center-需要额外支付9.99美元

相关：Windows 8或10上Windows Media Center的5个替代方案

Windows Media Center不是Windows专业版的一部分。如果您使用的是Windows专业版，则必须通过“向Windows添加功能”对话框额外支付9.99美元。微软发现，使用Windows Media Center和Windows Media Player的DVD播放功能的人很少，因此他们删除了这些功能，以节省许可费。如果你想要这些功能，你必须付费--这包括许可费。

9.99美元的费用虽然不方便，但不是什么大不了的事。重要的是，出于某种原因，你不能只花9.99美元就能获得Windows8.1或Windows8核心版的Windows Media Center。相反，你必须升级到Windows专业版，然后再支付媒体中心的费用-如果你是一个不想要这些其他功能的家庭用户，仅购买Windows Media Center就需要总共139.99美元。尽管价格不菲，但Windows Media Center自Windows7以来一直没有更新过，而Windows7家庭高级版是免费的。

其他媒体中心程序-如Plex和XBMC-可以免费使用。如果你喜欢Media Center，你可能想坚持使用Windows7，而不是支付高昂的费用。

Windows8的企业版提供了更多的功能，但没有微软的批量许可是无法获得这些功能的。
