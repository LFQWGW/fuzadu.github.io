使用WUBI安装程序在Windows上轻松安装Ubuntu Linux

您可能正在寻找一种方法来尝试Ubuntu Linux，但是不喜欢创建分区、使用慢速活动CD的想法，或者没有足够的资源来运行虚拟机。今天，我们来看看如何使用Wu bi安装程序在您的计算机上轻松运行Ubuntu。

WUBI是一个官方支持的Ubuntu安装程序，允许Windows用户轻松入门Linux领域。使用WUBI安装Ubuntu的过程与在Windows中安装任何其他软件程序的过程相似。它为您省去了创建另一个分区或创建VM的麻烦。五笔现已问世数年，从8.04《哈代苍鹭》开始，正式版就收录在Ubuntu Live CD上。我们将了解如何从Ubuntu Live CD安装WUBI，并单独下载WUBI.exe并安装Ubuntu。

从Ubuntu CD安装带有WUBI的Ubuntu

在第一种方法中，我们已经下载了Ubuntu Live ISO并将其刻录到CD上。在Windows中，弹出Ubuntu 9.10安装盘并运行wubi.exe。

在Ubuntu菜单屏幕上，单击Install Inside Windows按钮。

在Ubuntu安装程序屏幕上，您可以选择语言、安装驱动器、安装大小、用户名和密码，然后在完成安装后单击Install。

在…开始安装时，请稍等片刻

当它完成时，需要重新启动，但您可以立即执行，也可以等到以后再执行。

从WUbi.exe安装Ubuntu

如果你还没有Ubuntu光盘，另一个选择是下载WUBI安装程序并开始安装过程。这进一步简化了过程，因为您不需要下载ISO并将其刻录到光盘。使用此方法需要指出的一点是，您将获得想要安装的桌面环境类型的额外选择。在本例中，我们选择了Kubuntu环境。

现在，当安装进行时，它将为所选的桌面环境下载适当的ISO。Kubuntu使用的KDE环境与Ubuntu中使用的Gnome不同。主要区别在于KDE的图形更加华丽，Windows用户可能更容易适应。

无论您使用WUBI安装Ubuntu时采用哪种方式，都会将所有内容安装到C：\Ubuntu目录下。

引导进入Ubuntu

第一次重新启动后，您可以在Windows引导管理器屏幕上选择Windows7或Ubuntu。

第一次引导到Ubuntu时，请等待几分钟，等待安装完成。您将看到一系列屏幕，然后显示主安装屏幕，显示进度，屏幕显示Ubuntu9.10提供的功能。

您的系统将再次重新启动，您将再次拥有Windows7或Ubuntu可供选择。

当您从引导加载程序中选择Ubuntu时，它将进入GRUB引导加载程序，您可以在其中选择Ubuntu、恢复模式或返回到Windows7引导管理器。

在Ubuntu登录屏幕上，您会注意到用户名和密码将是您在上述步骤中提供的用户名和密码。

这真的是所有的事情，…。现在，您可以轻松地引导到Ubuntu，并让您的极客开始使用这个流行的Linux版本。

完全安装Ubuntu后，您可以进入高级系统设置\启动和恢复…来选择默认引导系统。然后在Windows或Ubuntu之间进行选择。要了解有关此设置的更多信息，请查看极客的文章：在Windows7/Vista双引导设置中轻松设置默认操作系统(Easy Set Default OS in a Windows7/Vista Dual Boot setup)。
