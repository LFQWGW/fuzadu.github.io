1月支持结束后如何更换Windows软件包2012

Microsoft将于2017年1月10日终止对Windows软件包2012套件的支持。如果你使用套件的任何组件应用-Movie Maker、照片库、OneDrive、家庭安全设置、邮件或Live Writer-以下是你需要了解的内容。

Windows Essentials 2012自发布以来一直是一款受欢迎的应用程序套件，今天仍有数量惊人的人在使用其中一些组件应用程序。2017年1月10日，微软将结束对该套件的官方支持。当然，你仍然可以使用它，但应用程序将不再接收任何类型的更新，包括安全更新。您也将无法再下载安装程序软件。如果您仍在使用Windows软件包2012，请继续阅读，了解官方支持终止对您意味着什么，以及您可以在哪里寻找替代产品。

您可以继续使用Windows软件包2012

对Windows软件包2012的官方支持将终止，但这并不意味着您必须停止使用它。如果您已经安装了它，您可以像往常一样继续使用它。请注意，将来不会有任何更新，包括安全更新。对于Windows Live Mail的用户来说，没有安全更新将是最重要的。对于套件中的其他应用程序来说，这就不那么重要了。

微软不再提供Windows软件包2012的安装程序供下载。网络上到处都有它的副本，但我们通常不建议安装来自你不知道或不信任的第三方来源，所以我们不会在这里链接到他们。不管怎么说，你可能会选择下面的其中一种更好的选择。

您无需更换家庭安全设置和OneDrive

那么，如果您想用最新的同等应用程序替换Windows软件包应用程序，该怎么办呢？我们将从简单的内容开始：家庭安全设置应用程序和OneDrive的所有功能都已内置到Windows 8和10中，因此，如果您使用这两个应用程序中的任何一个，您就可以开始使用了。事实上，如果你在使用这两款软件中的任何一款，你甚至都不会获得随套件一起安装家庭安全设置应用的选项。

如果您使用的是Windows7，则内置了家长控制功能。它们没有家庭安全设置应用程序提供的功能齐全，但它们应该能满足你的大部分需求。

OneDrive现在也内置于Windows 8和10中。如果你使用的是Windows 7，则需要下载OneDrive应用，但它是全新的，因为Windows软件包2012中提供了OneDrive应用，并且会不断更新。

Windows Live Mail的最佳替代方案

Windows Live Mail可能是您要更换的Windows软件包2012中最重要的组件。虽然您还可以继续使用，如果您愿意的话，我们不推荐您使用。在电子邮件客户端中，拥有可用的新安全更新非常重要。

公平地说，如今大多数人已经转向使用基于网络的电子邮件服务，如Gmail或Outlook.com。就不断更新的功能、垃圾邮件防护和更高的安全性而言，这些可能是您最好的选择。如果你喜欢桌面客户端，如果你不需要额外的功能(如基于规则的排序)，Windows8和10内置的Windows Mail应用程序实际上是一个相当可靠的选择。

如果您已经拥有包含Outlook的Microsoft Office副本，则应该尝试该选项。它的功能可能比电子邮件客户端需要的功能多，但从美学上讲，它仍然感觉非常像Windows Live Mail。

如果你想看看第三方的选择，我们推荐看看em Client，Mailbird和Thunderbird。这三个都是免费的-或者都有免费版本-并且已经存在了足够长的时间来开发完整的功能集。

Windows图片库的最佳替代方案

相关：了解Windows 7：使用实时照片库管理照片

长期以来，照片库一直是组织、查看和编辑照片的首选。虽然它不会得到更多的功能更新，但你可以继续使用Windows软件包2012的版本，因为安全更新在你的图像查看器中不会有太大影响。

如果你渴望更现代的东西，那么Windows8和10内置的Photos应用程序是一个不错的选择。它提供了查看、组织和对你的照片进行温和编辑的功能。如果你想获得更多的功能和轻松的分享能力，你可能还想看看在线服务，比如Google Photos、Prime Photos(面向Amazon Prime用户)和Flickr。这三款产品都提供了大量的在线存储、自动和手动组织工具，以及不同程度的图像编辑功能。

Windows Movie Maker的最佳替代品

电影制作人是个怪兽。Windows XP和Vista附带了一个非常流行的版本。当Windows7问世时，微软将这款应用程序从操作系统中分离出来，并发布了一个新版本，作为Windows软件包套件的一部分。虽然新版本没有那么强大，但它仍然在功能和易用性之间取得了很好的平衡，这一点在今天仍然受到许多人的赞赏。

好消息是，Windows软件包2012提供的当前版本在Windows7、8和10上仍然运行得很好。无论如何，这款应用程序已经好几年没有真正更新了，所以停止支持对任何人来说都不会有太大影响。可能更好的消息是，微软计划在不久的将来向Windows Store发布新版Movie Maker。我们猜测，新版本将仅限于Windows10用户，但除此之外，我们并不真正了解新版本的功能或发布时间的任何细节。

同时，如果您正在寻找比当前版本的Movie Maker更现代一点的东西，并且不想等待新版本，我们推荐Ezvid。它是免费的，和Movie Maker一样，它在易用性和功能之间取得了很好的平衡。如果你准备转到更高级的东西上--但仍然是免费的--“达芬奇解决方案”(DaVinci Resolve)是非常棒的，只需0美元的低成本。

Windows Live Writer的最佳替代品

Live Writer是你喜欢…的应用程序之一。或者从未听说过。这是一款博客发布应用程序，提供了一个令人愉快和功能丰富的界面。它具有所见即所得的编辑功能，并链接到多个博客平台，包括WordPress、Blogger、LiveJournal等。如果你在多个博客上工作，你也可以很容易地切换。

这里的好消息是，2015年，微软发布了名为Open Live Writer的Live Writer开源分支，你可以从Open Live Writer网站或Windows Store免费下载和使用。就像Live Writer一样，Open Live Writer可以使用很多流行的博客平台，包括WordPress、Blogger、TypePad、Moveable Type和DasBlog。它正在积极开发，并定期发布新功能。

有了合适的替代方案，您就不会为Live Essentials的死亡而悲痛了--事实上，您可能会使用更好的东西。
