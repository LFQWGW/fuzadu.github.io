如何与朋友共享您的Plex媒体库

你在你的Plex Media Server上投入了大量精力来收集和策划这么多优秀的电影和电视节目，如果你能与你的朋友分享所有这些内容，那不是很棒吗？只需做一些小调整，你就可以--我们会告诉你怎么做。

你为什么要这么做

如果你和我们有任何相似之处，你可能已经花了很多时间建立你的媒体中心，精心策划你真正喜欢的电影和节目，配上精心挑选的扇形艺术。当Plex Media Server让分享变得如此容易的时候，没有与你的朋友分享这样的内容是一种耻辱。

只需很少的工作，您就可以将您的Plex Media Server配置为与您的朋友共享内容(而您的朋友反过来也可以使用此教程与您共享他们的内容)。现在不是像“天哪，你看过XYZ秀”这样的对话了吗？这是我在Agees…看过的最好的科幻电影。“。然后解释在哪个频道和什么时候看，你可以告诉你的朋友在你的媒体服务器上查看新节目，告诉你他们的想法。

然而，在我们开始学习本教程之前，有一种情况是不适合共享您的Plex Media Server的：如果您在家中连接的速度又慢又粗糙。如果由于上传连接速度慢，当您外出时访问自己的Plex Media Center已经够难的了，那么与一个或多个朋友共享相同的连接只会导致挫折感。除此之外，分享你的个人图书馆是与朋友分享你最喜欢的媒体的一种有趣的方式。

让我们来看看您需要开始做些什么，以及一些次要的注意事项。

您需要什么来共享库

要按照本教程进行操作，您只需要做几件事。首先也是最重要的是，您需要启动并运行一台Plex Media Server。如果您找到这篇文章，我们可以假定您已经启动并运行了一台服务器，但有些读者可能需要回溯并阅读我们的Plex设置指南。

其次，您需要确保可以从家庭网络外部访问您的Plex Media Server。虽然设置Plex几乎是不可能的，因此您无法在家庭网络中访问它，但有时您需要进行一些故障排除，以确保您出门在外时可以访问它。如果远程访问对您不起作用，那么它肯定也不会对您的朋友起作用。

最后，与您共享媒体中心的每个朋友都需要一个免费的Plex帐户。请注意，他们不需要自己的Plex Media Server。如果他们有自己的Plex Media Server，并且可以转过身来与您共享他们的服务器，那就太棒了。如果他们没有Plex Media Server，没什么大不了的：他们的主库只会默认为您与他们共享的库。他们可以在这里注册他们的免费账户。

如何共享您的图书馆

一旦您检查了服务器外设置的先决条件，可以从您的家庭网络外部访问，并且您的朋友有Plex帐户-其余的就非常简单了。要开始使用，请登录您的Plex帐户并查找右上角的配置文件图标。点击它并选择“用户”。

在出现的菜单的左侧，选择“朋友”。

在“朋友”菜单中，当您继续使用系统时，您将在其中管理好友请求和邀请，您将在右上角的导航栏中看到一个条目“邀请朋友”。现在就点击它。

在弹出的邀请框中，输入您的朋友用来注册Plex的电子邮件地址。如果他们还没有注册Plex，您可以将其发送到他们的主要电子邮件地址(当他们通过电子邮件接受您的邀请时，系统会提示他们创建一个Plex帐户)。输入他们的电子邮件后，单击“下一步”。

在下一个屏幕中，邀请向导将显示您可用的Plex Media服务器。对于每台服务器，您可以选择是要共享库的所有库，还是要共享库的一部分。如果您取消选中“共享”部分中的“所有库”，您将能够单独选择要共享的库。除非您有迫切的理由不共享库，否则最简单的方法就是让它们全部共享。您可能会注意到服务器下面的“限制”选项卡。免费用户唯一可用的限制切换是共享/不共享“频道”功能。默认情况下，此共享是关闭的，我们建议您保持原样；如果您和您的朋友都订阅了同一频道，例如bbc世界新闻频道，则您对该频道的设置将覆盖他们的设置。所有其他限制设置(如根据评级限制内容)仅限于具有Plex Premium帐户的用户。

一旦您选择了要共享的库，请单击“邀请”。

此时，中央Plex系统将向您朋友的帐户发送一封电子邮件。他们需要打开该电子邮件并单击邀请链接以确认您的邀请。

如何访问共享库

当有人与您共享库时，您会收到电子邮件通知和Plex仪表板中的通知。让我们看两个场景，一个是与您共享的Plex库是您唯一的库，另一个是与您共享的Plex库与您自己的库并存。

在本教程中，我们与我们的朋友“J”共享了我们的Plex Media Server，他自己没有Plex库。这是我们的邀请函在J的仪表盘上的样子。

要接受请求，您需要首先单击右上角的配置文件图标，就像我们从我们的帐户发送邀请时所做的那样。单击配置文件图标，然后选择“用户”。两个图标上的下标“1”表示邀请已经到达。

单击左侧菜单中的“朋友”选项卡，然后单击邀请旁边的绿色复选标记接受邀请。

一旦邀请被接受，如果您返回到Plex主屏幕，您将看到J的库默认为我们的库，并且他以前空的仪表板填充了来自我们服务器的媒体。

在这一点上，我们的朋友J拥有我们与他共享的所有图书馆的完全查看权限，并且可以轻松地观看他想看的任何内容。Plex系统将跟踪他看了什么，跟踪他在观看时停在哪里，并为他提供我们从Plex Media Server获得的所有其他好处。

但是，如果有人与您共享Plex Media Server，而您已经拥有自己的媒体，该怎么办呢？您只需通过切换在想要访问的服务器之间切换即可。登录到您的Plex Media Server控制面板或使用任何Plex Media Server应用程序(如Plex for iOS)时，您只需单击服务器选择菜单(如下所示，位于Web控制面板的左上角)，然后从其他可用的服务器中进行选择。

在上面的屏幕截图中，您可以看到我们的服务器(不是很有创意地将默认名称命名为“plexmediaserver_1”)和我们朋友的服务器“发明”。要观看共享内容，我们只需点击一下即可在两台服务器之间切换。

虽然在两台服务器之间切换是很棒的，如果您有心情进行一些漫无目的的浏览，但这甚至不是一个必要的步骤。Plex搜索功能将扫描您自己的库和与您共享的所有库。例如，如果我们搜索“war”一词，您可以看到第一个搜索结果位于我们的服务器上，但接下来的三个结果位于我们朋友的服务器上：

建立和运行Plex共享并不需要太多的努力，但是一旦你这样做了，你不仅可以与你的朋友分享你最喜欢的电影和节目，反过来，你也可以享受他们最喜欢的电影和节目。
