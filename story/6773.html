自定义您的网络共享设置

本课的目标是解释Windows中可用的所有网络共享设置及其功能。这听起来很简单，不是吗？但不幸的是，事实并非如此。

正如您将看到的，有很多可用的共享设置。有些很容易弄清楚，而另一些就不那么容易了。此外，Windows8.x引入了一个新的设置，该设置在任何地方都没有文档记录。为了理解它和它的作用，我们必须进行大量的实验。但是我们确实掌握了它的诀窍，我们可以给您解释一下，这样您就可以正确地进行设置了。

我们在本课中介绍的另一个主题是如何更改分配给活动网络连接的位置。正如您将看到的，只需一次简单的更改，Windows就可以重新配置所有可用的网络共享设置。因此，了解您应该在何时以及如何更改网络位置非常重要。

在本课结束时，您将了解如何配置网络配置文件和网络共享设置，以便只打开需要在网络上使用的功能。

我们开始工作吧!

在Windows中的何处查找网络共享设置

Windows 7和Windows 8.x具有相同的网络共享设置。它们都在同一个地方，但是它们的顺序不同，它们的分组也是不同的。

要找到它们，请转到“控制面板>网络和Internet”，然后转到“网络和共享中心”。在设置网络连接和网络共享时，此窗口非常重要。在这里，您可以找到用于更改网络适配器设置的选项、指向所有网络共享设置的链接，以及用于设置新连接或解决问题的向导。

要访问您的网络共享设置，请单击或点击左栏上的“更改高级共享设置”链接。

现在您将看到一个列表，其中包含在Windows中找到的所有可用的网络共享设置，并按网络位置分组。

更改默认共享设置

如前所述，根据您使用的Windows版本的不同，这些设置会以不同的顺序显示。

Windows 7

在Windows7中，所有设置都分为两类：“家庭或工作”和“公共”。这将根据Windows7中可用的三个网络位置将所有网络共享设置分成组(要了解有关网络位置的更多信息，请阅读第2课)。

您将看到一行“当前配置文件”，告诉您哪个网络位置被分配给活动的网络连接。该配置文件中的设置将应用于您的活动网络连接，而不是其他网络连接。

如果单击每个配置文件类型旁边的箭头，则可以展开每个组，您将看到它们包括相同的网络共享设置，但“家庭组连接”除外，该设置仅适用于“家庭或工作”网络位置。

您还将看到，对于不同的位置，每个设置都有不同的值。这很棒，因为它允许Windows根据您连接的网络快速调整您的网络共享设置。但是，对于您的家庭网络，您可能需要自定义默认值。

Windows 8.x

Windows8.x让事情有点混乱，因为它们将所有网络共享设置分成三个部分：“私有”、“来宾或公共”和“所有网络”。

“私有”组包括三个设置：“网络发现”、“文件和打印机共享”和“家庭组连接”。这些设置仅适用于设置为专用的网络连接。

“来宾和公共”组仅包括两个设置：“网络发现”和“文件和打印机共享”。它们仅适用于设置为“公共”的网络连接。

“所有网络”组包括四个设置：“公用文件夹共享”、“媒体流”、“文件共享连接”和“密码保护共享”。这些设置的问题在于它们适用于所有网络连接(“私有”和“公共”)。

为什么这会是个问题？因为如果您打开“公用文件夹共享”，您也会为“公用”网络连接打开它，这可能存在安全风险。另外，想象一下在关闭所有网络连接的密码保护共享时可能会遇到的问题。

这就是为什么在将安全性放在首位的同时注意和自定义这些设置非常重要的原因。

Windows网络共享设置

在开始配置所有网络共享设置之前，最好先了解每个设置的作用。让我们根据它们在Windows 8.x中的顺序逐一来看一下它们：

网络发现

此设置使Windows搜索网络上的其他计算机和设备，并在网络上广播您的计算机，以便其他人看到它。应该打开“网络发现”，以便您的计算机可以访问网络上的其他计算机，并能够与它们共享内容。

在Windows8.x中，您还会发现一个子设置，上面写着“打开网络连接设备的自动设置”。不幸的是，微软在任何地方都没有记录这个设置，我们花了很多时间和实验才弄清楚它是怎么做的。如果启用此功能，Windows似乎能够检测连接到网络的设备(如连接到路由器或另一台PC的外部硬盘)，然后使用它们提供各种服务，包括在这些设备上进行“文件历史记录”备份。

当此设置关闭时，Windows将不会检测到此类设备，并且将无法使用它们执行“文件历史记录”备份或提供其他类型的服务。

文件和打印机共享

此设置允许Windows共享文件和打印机，以及访问网络上其他计算机共享的文件和打印机。

必须打开此设置才能执行任何类型的网络共享。

家庭组连接

“家庭组连接”用于在Windows中打开或关闭家庭组功能。如果您有多台安装了Windows 7或Windows 8.x的计算机，则应打开此功能，并使用它来轻松实现网络共享。若要了解有关家庭组及其功能的更多信息，请返回到第2课。如果您想了解如何使用此功能进行共享，请阅读第5课。

当您关闭此设置时，Windows将只允许通过使用用户帐户和密码进行网络共享。其他计算机上的用户必须使用与您共享文件夹和设备的用户帐户进行身份验证。

公用文件夹共享

“公用文件夹共享”是一个老式的概念，用于与同一台计算机和网络上的其他人共享文件夹。

我们将在下一课中更详细地解释这一点。在您阅读之前，请记住，当启用这种共享时，您放在“C：\Users\Public”文件夹中的任何内容都对网络上的其他设备和计算机公开可用。

媒体流

此设置允许您指定希望网络上的人员和设备访问计算机上的图片、音乐和视频的位置。

基本上，您的标准库(图片、视频等)。可用于媒体流，您可以使用Windows Media Player访问它们。如果您的网络中有一个Xbox控制台，您可以很容易地将您的库流式传输到它。

Windows7和Windows8.x之间有一个重要的区别：Windows7默认关闭此设置，而Windows8.x则打开它，您需要自定义它的工作方式。

文件共享连接

没有多少人知道文件共享连接是由Windows加密的。默认情况下，Windows使用128位加密，因此您的数据传输不会容易被其他人嗅探。

这种加密适用于所有现代操作系统，您不需要更改默认设置，除非您的网络中有一些非常旧的计算机使用过时的操作系统。只有到那时，您才应该考虑将128位加密更改为40位或56位加密。

受密码保护的共享

“密码保护共享”对于您的网络安全很重要，但它也可能构成一个烦人的因素。

默认情况下，只有属于同一家庭组的计算机和设备才能查看您共享的内容，而无需身份验证。使用不同操作系统的计算机或设备只有在使用允许访问您共享的内容的用户帐户和密码进行身份验证时，才能访问您共享的内容。

如果您的网络主要包括Windows计算机和设备，并且它们使用的是Windows7或Windows8.x等最新版本，这是可以接受的。如果您有Linux PC或Mac OS计算机，则此设置可能会很麻烦，并且您可能更喜欢禁用密码保护共享。

我应该更改哪些网络共享设置？

为了帮助您确保创建尽可能最佳的设置，我们想分享一些建议：

在Windows7和Windows8.x中，使用“公共”网络连接的默认值。这意味着您将同时关闭网络发现以及文件和打印机共享。 对于“私有”(Windows 7中的“家庭或工作”)网络连接，请打开以下设置：“网络发现”、“网络连接设备的自动设置”(仅在Windows 8中可用)、“文件和打印机共享”和“允许Windows管理家庭组连接”。 如果您不打算使用公用文件夹进行共享(在做出此决定之前，不要忘记阅读第4课)，最好对所有网络位置关闭此功能。 对于所有网络位置，如果您不打算使用“媒体流”，也应该将其关闭。 我们建议您保留默认的128位加密，以帮助保护文件共享连接，并打开“密码保护共享”，除非这是一件很麻烦的事情，并且只有少数受信任的人可以访问您家庭网络中的计算机。

如果您为活动网络连接选择了错误的位置，请不要更改其网络共享设置。在连接到新网络时，您将面临各种问题。解决此问题的最佳方法是更改错误分配的位置，详细信息将在下一节中介绍。

如何在Windows 7中更改网络位置

在Windows7中，您可以从“网络和共享中心”更改分配的网络位置。

在窗口中间，您将看到一个部分，上面写着“查看您的活动网络”。

此部分显示每个网络适配器或网卡的活动连接。如果您安装了虚拟化软件(如Virtual Box或VMWare Player)，您将看到虚拟化软件安装了其他网络适配器。对于那些虚拟适配器，最好不要更改分配的网络位置/配置文件。

要更改活动网络连接的位置，请单击当前分配的位置。此时将显示“Set Network Location”(设置网络位置)窗口，您可以在其中选择不同的位置。

选择最适合您的位置。您将被告知此更改及其对您的意义。

单击“关闭”完成。

您的网络共享设置现在已相应调整，并更改为为新活动位置设置的默认值。

请注意，如果您选择“Home”位置，系统还会要求您创建一个家庭组并与其共享内容，即使您的网络上可能已经存在一个家庭组。最好关闭向导并按照第5课中说明进行操作，以便正确设置家庭组。如何在Windows 8.x中更改网络位置

在Windows8.1中更改网络位置的过程与在Windows7中有很大不同。首先，请转到“PC设置”。要做到这一点，一种方法是转到开始屏幕，通过从屏幕右侧轻弹或按键盘上的“Windows+C”来调出咒语。单击或轻触“设置”，然后单击“更改电脑设置”。

在PC设置中，进入“网络”。在“连接”部分，您将看到活动的网络连接。如果您使用的是台式PC，并且有有线网络连接，则您的连接将命名为“Network”，并显示在“Ethernet”下。

如果您使用的是装有Windows 8.1的笔记本电脑或平板电脑，您将在名为“Wi-Fi”的部分中看到您的无线网络的名称。

单击或轻触您的网络连接，将显示其属性。

显示的信息取决于您正在使用的网卡和您的连接类型(有线或无线)。但是，对于所有连接，您将看到一个名为“查找设备和内容”的部分，其中包含一个开关。如果要为活动网络连接分配“专用”位置，请将此开关设置为“开”。

如果要指定“公共”位置，请将此开关设置为“关闭”。您的网络共享设置将相应调整，并更改为为新活动位置设置的默认值。
