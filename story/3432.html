Windows家庭服务器

我已经测试Windows Home Server(WHS)大约6个月了。我想我应该写一些关于微软这个新产品的东西。基本上，WHS是一台电脑，它会放在你的壁橱里，将多媒体内容传送到你家周围的各种电脑和其他极客设备上。您可以创建用户帐户、流式播放音乐和视频，以及创建到家庭服务器的备份。您的孩子可以在他们的计算机上看电影，而您在听您最喜欢的音乐，反之亦然。保存文件副本就像将它们放在网络驱动器上一样。

还有一个自动备份功能，可以将您家庭网络中的所有PC增量备份到服务器。添加更多硬盘空间非常容易。您只需连接一个额外的外置USB硬盘即可。WHS会将其添加到服务器的总体空间中。记住这一点的一件事是，动力成为整体的一部分。这意味着，它将成为整个服务器空间的一部分，而不是单个驱动器。

启动Windows Home Server Beta 2时，始终会出现Windows Server 2003闪屏。这是因为WHS是基于Sever 2003技术构建的。我不确定您是否会在最终版本中得到这个。但是，它不像Server2003那么复杂，所以不要太害怕。使用WHS无需担心Active Directory。您将能够在家中管理服务器，而不必成为IT极客！

在接下来的几周里，我将在网站上添加更多的如何操作。当Windows Home Server正式向公众开放时，您将有机会获得快速有效地开始使用它所需的信息！
