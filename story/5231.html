通过在Gmail中启用实验室功能来拉皮条

我们最近研究了如何使用多收件箱实验室功能更轻松地管理Gmail中的多个收件箱。这是一个非标准功能，而且它远不是您唯一可用的功能。事实上，有许多隐藏的功能可以帮助你从Gmail中获得更多。

在深入研究Gmail Labs以了解可用的功能之前，重要的是要注意到，正如Google指出的那样，Labs的功能是“疯狂的试验性东西”。你可能会发现一切都运行得很完美，但它们很有可能会破坏你的收件箱，或者把你锁在收件箱之外。

谢天谢地，这里有一个安全网--相当于在安全模式下启动Windows。如果出了问题，你只需要访问https://mail.google.com/mail/u/0/?labs=0就可以联系到你的收件箱，暂时禁用所有实验室功能，让你有机会在设置中正确禁用它们。

注意到这一安全措施后，您就可以浏览可用的内容了。有些实验功能很简单，只不过是一个启用或禁用的新按钮，但也有一些稍微复杂一些，需要您配置的设置。

支持实验

在Gmail中，单击页面右上角的齿轮图标，选择设置，然后单击实验室。Labs的功能列表并不太长，所以尽管有搜索功能，您也可以滚动浏览列表。启用单个实验室功能只需选择适当的启用选项，然后单击页面底部的保存更改即可。

许多实验室都非常节省时间。启用“标记为已读按钮”，当您想要将选定的邮件标记为已读时，您可以与浏览“更多”菜单所需的额外点击吻别。如果您想一下子标记几封已读的邮件，只需单击收件箱顶部新添加的按钮即可。

删除电子邮件时，您通常会返回到收件箱，但这可以通过启用自动前进进行更改。启用实验室功能后，请转到设置的常规部分，然后您可以选择是进入下一次对话还是预览对话。

如果你喜欢让Gmail变得更像桌面电子邮件客户端的想法，启用Preview Pane实验室。在收件箱的右上角，单击出现的新按钮右侧的箭头，然后选择垂直拆分(将预览窗格放在电子邮件列表的右侧)或水平拆分(将预览窗格放在下面)。

访问Google功能

还有一些实验室可以让与其他谷歌服务的互动变得更容易。如果您使用Google Calendar，请启用“Google Calendar Gadget”，然后您可以选择用即将到来的事件替换页面左下角的聊天面板。还有一个快速添加选项，让您无需离开收件箱即可创建日历事件。

Google Docs用户有几个实验室可以使用。启用“创建文档”，您可以随时通过按G和W来创建新文档。若要更轻松地处理您审阅的Google文档，请启用“邮件中的Google文档预览”，这样您就可以查看文件，而无需离开电子邮件。

与“谷歌日历小工具”(Google Calendar Gadget)类似，“谷歌文档小工具”(Google Docs Gadget)在收件箱文件夹下面的区域增加了一个新的面板，你可以用它来快速搜索和浏览你的谷歌文档(Google Docs)文件。

自定义Gmail

实验室不仅仅是关于新功能--它们还可以用来定制Gmail的外观。“自定义键盘快捷键”的功能完全符合您的预期--它使您可以定义自己的键盘快捷键，这样您就可以更快地完成工作。

启用此实验后，设置中会出现一个新的键盘快捷键部分。在这里，您不仅可以覆盖默认快捷键，还可以定义两个快捷键来执行相同的操作。

如果您启用了Google Calendar或Google Docs Labs，您可能会发现访问聊天有点尴尬。这就是“右侧聊天”实验室的用武之地--它将聊天窗格向右移动，更好地利用了屏幕空间。

您可以启用的小工具还有很多，但目前，我们只需再添加一个。“未读邮件图标”实验室只在Firefox和Chrome中提供，它会改变Gmail收件箱中的未读邮件图标，以指示收件箱中有多少未读邮件-当你在另一个选项卡中工作时，可以方便地监控电子邮件。
