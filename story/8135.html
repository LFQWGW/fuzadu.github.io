最终升级到Windows 10的10个理由

发布四个月后，Windows10运行良好，尽管有一些小问题，但我们认为是时候升级了。如果你还没有的话，下面是你可以考虑迈出这一步的十个原因。

Windows8的两个截然不同的界面-新奇的开始屏幕和令人尊敬的桌面-造成了巨大的混乱，除了最坚定的用户之外，所有人都能够对它感到平静。

在许多情况下，用户只是简单地推迟升级或恢复到Windows 7，毕竟Windows 8真的没有提供任何令人信服的优势，比起它的老款、但广受欢迎的同类产品，除了可能略有改善的桌面体验之外，Windows 8确实没有提供任何令人信服的优势。

在这一点上，Windows8注定要与Vista一起在历史上占有一席之地(这并不像与Windows ME混为一谈那么糟糕)。

Windows 10在回归Windows 7精神的同时，融合了Windows 8最好的功能，令人耳目一新，显然最终将成为微软的最后一款操作系统。因此，不用再费劲了，下面是我们认为您最终应该升级的十个原因。

开始屏幕已死，开始屏幕万岁！

无论出于何种目的，Windows8进军开始屏幕的尝试都是失败的。尽管如此，与旧的Windows开始功能(开始菜单)相比，开始屏幕至少是与老旧的老功能相比有所突破。

Windows10的解决方案是将开始屏幕和开始菜单结合起来。这是一个相当不错的折衷方案。它保留了Windows 8的现代风格，同时恢复了开始菜单的基本功能和熟悉性。

新的“开始”菜单色彩丰富、大小可调，并且最终可配置。最棒的是，你不会陷入一种或另一种模式，如果你使用触摸屏，或者只是更喜欢旧的Windows 8风格，你仍然可以在全屏或平板模式下使用新的开始菜单。

微软新的开始菜单是否会像我们都认为的那样大受欢迎，还有待观察。使用Windows的大众可能会接受它，但要赢得每个人的心还需要时间。

控制面板更少，设置更多

控制面板，自Windows95以来一直保持不变，也在发生变化，这真的是一场革命，而不是演变。控制面板是旧Windows的一个古板部分，多年来在其自身重要性的重压下变得笨拙起来。

新的“设置”绝对是一股新鲜空气。它们与Windows8中的“PC设置”没有太大不同，只是它们更加突出，并且承担了更多控制面板的职责。

一旦微软完成了这次大修，您应该可以使用新的设置完全控制计算机。现在说实际的控制面板将在很短的时间内消亡可能还为时过早，但这肯定是不祥之兆。

通知：更“迷人”的空间利用

到目前为止，哪个操作系统，无论是移动操作系统还是台式机操作系统，都没有通知中心功能？Windows就是这样的人，但随着Windows10的推出，这一点发生了变化。

新的操作中心取代了备受厌恶的“魅力”(是的，魅力终于消失了)，还提供了访问系统设置的快捷方式。

诚然，Windows上的通知中心可能看起来相当平庸，但这是系统第一次真正将通知合并到一个单一的位置，这意味着你不会错过更多的消息和重要的警报。

虚拟桌面

Windows10也将是首个运行多个桌面工作区的Windows版本，或者更广为人知的是虚拟桌面。

Linux发行版、OSX，几乎所有拥有重要桌面操作系统的人都拥有虚拟桌面。所以，这在某种程度上是令人兴奋的，“这不仅仅是关于时间的”。

科塔纳

Cortana被标榜为Windows虚拟个人助理。也许这是那些让人瞠目结舌的事情之一，因为Android在这方面已经做得很好了，有了“OK Google”，苹果的Siri已经覆盖了iPhone和iPad上的东西，但至少能够只用你的声音与你的台式电脑互动的想法，长期以来一直让PC用户摸不着头脑。

Cortana最初出现在Windows Phone平台上，但现在微软正在将其迁移到其旗舰操作系统上。“我们已经介绍了如何使用和配置Cortana，如果你想亲自尝试的话。”

航空公司回来了(一点点)

有报道称，微软将带回Aero，这一消息受到了各地Windows7用户的庆祝。老实说，Windows10在透明效果方面有了明显的提升，但现在就断言Aero是否会完全回归还为时过早。

尽管如此，至少部分透明胶片的回归表明，微软终于开始以更有意义的方式与其用户基础建立联系。

Windows 8引入的最大问题之一是，微软基本上删除了人们在Windows 7中实际喜欢和使用的大多数东西-开始菜单、开始按钮、玻璃般的透明效果-因此，Windows 10现在似乎已经部分接受了Windows 7的根源，这一事实是有希望的。

压缩和较小的安装

这是除了最严肃的Windows追随者之外，所有人都可能错过的事情之一。

Windows 10不仅安装大小更小，而且还采用了更高效的压缩算法，这意味着你的文件会更小(1.5-2.6 GB，取决于你的版本)，这样你就可以在同样的空间里存储更多的文件。

微软表示，到目前为止，它已经能够将Windows10的安装大小削减4 GB到12 GB，而且随着今天的现代处理器性能不断增强，感觉现在是用户开始使用压缩的绝佳时机。

由于您的计算机可能配备得非常好，可以处理开销压缩所需的一点点额外CPU，因此您最终会有更多的存储空间来支付您的费用，谁不想要他们的硬盘上有更多的空间呢？

Direct X12意味着游戏玩家会喜欢它

Direct X 12是微软历史悠久的游戏API自可能推出Direct X以来最重大的改进。

Direct X 12的主要目标是减少驱动程序开销，这意味着玩家可能会在他们现有的硬件上体验到更好的性能，这是另一回事-如果您的显卡是相当新的，但不是尖端的新显卡，您仍然可以使用Direct X 12，也就是说，您可能不必升级计算机中最昂贵的组件之一。

最棒的是，Direct X 12将默认与Windows 10一起安装，这是所有迹象中最强烈的迹象，表明微软正在尽其所能改变人们对Windows 8游戏糟糕透顶或是一场“灾难”的看法。

对于现有的Windows用户来说，这将是一个免费升级

尽管最近我们对Windows10升级应用程序和任务栏图标感到恼火，但Windows10将对现有Windows用户免费升级这一事实对微软来说是一大进步。

微软必须在尽可能多的Windows电脑上安装Windows10，无论是过去的、现在的还是将来的。如果像一些人预测的那样，这将是最后一个大型的主流Windows版本，那么我们可以假设，在未来的某个时候，几乎每个人都会使用它。

微软似乎开始意识到这一点

其他正在发生的事情也越来越受到关注，比如新的Edge浏览器，旨在取代老化的、备受诟病的Internet Explorer；提高了旧硬件上的性能；以及新的、改进了的Windows Store和应用程序体验。

此外，一旦微软完善了Continuum功能，这意味着你将能够从键盘/鼠标设置无缝过渡到触摸设置，甚至可以像使用电脑一样使用Windows Phone。因此，如果你使用Surface这样的混合设备，那么当你把它从底座或键盘上拆下来时，它会自动切换到全屏平板模式，反之亦然。

微软进一步计划推出Windows Phone，它将能够与键盘、鼠标和显示器对接，使其成为一款超便携但功能强大的电脑。
