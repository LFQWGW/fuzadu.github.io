如何从命令提示符的当前目录打开资源管理器窗口

有各种各样的方法可以从文件资源管理器打开命令提示符窗口-您甚至可以直接在文件资源管理器地址栏中运行命令-但您知道从命令提示符打开文件资源管理器窗口也一样容易吗？

相关：在Windows 10中打开命令提示符的10种方法

假设您处于命令提示符下，正在处理文件和文件夹，并且希望在资源管理器中访问它们。无需手动导航，只需一条简单的命令即可到达。点击Windows+X并从Power Users菜单中选择“Command Prompt”，打开命令提示符窗口。

注意：如果您在高级用户菜单上看到的是PowerShell而不是命令提示符，这是在Windows 10的创建者更新中产生的一个开关。如果您愿意，可以很容易地切换回在高级用户菜单上显示命令提示符，或者您也可以尝试一下PowerShell。您可以在PowerShell中执行命令提示符中可以执行的几乎所有操作，另外还有许多其他有用的功能。

相关：如何将命令提示符放回Windows+X Power Users菜单

我们将使用start命令，所以让我们从一个简单的命令开始，在命令提示符下打开当前文件夹的文件资源管理器窗口。在提示符下，只需键入以下内容：

start .

句点在命令提示符中用作当前文件夹的简写，因此这将在文件资源管理器中打开当前文件夹。

您还可以使用双句点打开当前文件夹的父文件夹。例如，假设在命令提示符中，您当前正在查看名为“Reports”的文件夹，该文件夹位于名为“Project A”的目录中。您可以键入以下命令在文件资源管理器中打开“Project A”文件夹，而无需在命令提示符下离开“Reports”文件夹：

start ..

只需一个命令，“Project A”文件夹就会在文件资源管理器中打开。

您还可以在双句点速记的基础上，在末尾添加相对路径。让我们继续我们示例，假设在“Project A”文件夹中也有一个名为“Sales”的文件夹。当您仍在“Reports”文件夹中时，您可以键入以下命令在文件资源管理器中打开“Sales”文件夹，而不会离开命令提示符中的“Reports”文件夹。

start ..\Sales

当然，您也可以键入完整路径以打开PC上的任何文件夹：

start c:\windows\system32

您还可以将该命令与任何内置Windows环境变量或更新的shell：Operator样式一起使用。因此，例如，您可以键入以下命令来打开当前用户的AppData文件夹：

start %APPDATA%

或使用类似下面的命令打开Windows启动文件夹：

start shell:startup

因此，如果您在命令提示符下输入，并且希望切换到使用文件资源管理器来执行某些任务，只需记住简陋的start命令。这也是给你那些不太精明的朋友留下深刻印象的好方法。当然，start命令也用于运行程序，该功能还有许多附加开关可用。如果您对这些内容很好奇，只需键入start/？在命令提示符下获取开关及其使用方式的完整列表。
