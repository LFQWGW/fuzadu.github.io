如何用巧克力将Linux风格的apt-get安装带到Windows

Linux用户喜欢从命令行安装软件，原因不难理解。您不需要下载、运行可执行文件，也不需要躲避更改默认搜索引擎的各种尝试，您只需键入想要安装的内容，然后按“Enter”即可。

如果你想要Windows上的这种宁静，只需看看Windows的软件包管理器Chocolatey就行了。这个免费的工具可以让你安装数以千计的应用程序，只需一个快速命令(巧克力安装)，然后用另一个命令(巧克力升级)更新它们。如果您是那种安装大量自由软件的人，这可以为您节省大量时间。

您可以在两个位置使用巧克力：命令提示符和PowerShell。下面介绍如何在这两个环境中安装此包管理器，但您实际上只需要其中一个。挑任何你觉得舒服的。

从命令提示符下安装和使用巧克力

Chocolatey只有在您以管理员身份运行时才能在命令提示符下运行。最简单的方法是右键单击“开始”菜单中的“命令提示符”，然后单击“以管理员身份运行”。

要安装巧克力，首先需要运行以下命令，该命令将为您下载并运行安装程序：

@"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"

这个命令可能会改变，所以如果你有问题，请检查官方的巧克力说明。

安装巧克力之后，您可以立即使用它(尽管在某些情况下，您可能需要关闭并重新打开管理命令提示符)。安装软件很简单：只需键入choco install，后跟程序名即可。在这里，我们运行的是choco install filezilla客户端，以安装广受欢迎的Filezilla FTP客户端。

您也不必限制自己一次只能安装一个应用程序。例如，使用以下命令将一次性安装所有列出的应用程序：

choco install firefox thunderbird vlc libreoffice

当然，您需要一些方法来了解存在哪些包以及它们是如何命名的。您可以使用以下语法搜索以查看是否提供给定的应用程序：

choco search key_word

或者，该网站提供了您可以安装的所有软件包的完整列表，并提供了自己的搜索工具。

如果要将特定软件包升级到最新版本，请使用命令“choco Upgrade”，后跟软件包名称。如果你想一下子升级你用巧克力安装的所有东西，巧克力升级所有的都会做到这一点。如果你想深入挖掘，还有更多的子命令。只需键入“choco-？”即可查看完整的列表。

在PowerShell中安装和使用巧克力

相关：5个Cmdlet，帮助您开始使用PowerShell

如果你是PowerShell的忠实粉丝，你也可以在那里用巧克力。这意味着，一旦您开始使用PowerShell并掌握诀窍，您就可以自动执行各种操作。

您需要打开管理员PowerShell才能开始。最简单的方法是右键单击“开始”按钮，然后单击“Windows PowerShell(管理)”。

巧克力本身需要从计算机外部运行脚本的权限。因此，官方安装说明建议您在安装巧克力之前更改ExecutionPolicy。首先，在PowerShell提示符下键入以下命令，然后按Enter键：

Get-ExecutionPolicy

如果您看到“受限”结果，那么您需要运行以下命令来设置您的执行策略，以便巧克力安装能够正常工作：

Set-ExecutionPolicy AllSigned

命令Set-ExecutionPolicy Bypass也可以工作，尽管这是一个不太安全的选项：它允许运行任何脚本，无论它是否由开发人员签名。

设置执行策略后，可以使用以下命令运行安装程序：

iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
