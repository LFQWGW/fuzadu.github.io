Windows 7 Media Center操作指南极客指南

如果您已从XP升级到Windows 7，这可能是您第一次访问Media Center。在这里，我们创建了一份指南，介绍使用Windows 7 Media Center的最佳提示、技巧和教程。

注：Windows 7 Media Center仅在Windows家庭高级版和更高版本中可用。

入门和看电视

如果您是Windows7 Media Center的新手，您会想要看看我们的初学者指南，其中向您展示了如何设置WMC以查看直播电视、录制、流视频、Netflix等。

如果您的PC有调谐器卡，设置为观看直播电视似乎是一项艰巨的任务。但实际上，Windows通过一个简单易用的向导驱动界面来引导您完成整个过程，并为您找到所有频道，这让它变得很容易。有关更多信息，请查看我们关于如何在WMC中设置直播电视的文章。

现在您已经设置了Live TV，可以开始调整体验了。如果你喜欢像DVR一样控制电视，你可能想要通过注册表调整来增加直播电视暂停缓冲区。这将避免在你去做某事时暂停直播电视的烦扰。…。然后再回来，节目又开始播放了。

然后，您可能想要自定义频道指南。您可以从添加电视频道徽标开始，然后向收视指南添加颜色编码。

最后，如果您喜欢在WMC中观看大量直播电视，您可能希望它始终以直播电视模式启动。这将允许您避免几个菜单导航步骤，并将您自动带到您想要的位置。您还可以使用其他启动定制，这些定制将在电影或音乐库等其他领域启动。

观看流媒体视频

如果您的计算机上没有安装电视调谐卡，您仍然可以免费观看主要网络(如Showtime、CBS和其他网络)的在线内容(广告有限)。当互联网电视功能第一次推出时，选择并不是那么好，但在撰写本文时，已经添加了更多的内容。

你不仅可以观看当前的流媒体内容，还可以提供很酷的经典节目，如“星际迷航”、“夏威夷5.0”、“暮光之城”等等。有关更多信息，请参阅我们关于在Windows7Media Center中如何在没有调谐卡的情况下观看电视的文章。

由于WMC应该是您所有媒体的中心中心，您可以添加观看YouTube视频的功能。这样，您就可以舒适地坐在沙发上或从PC上观看YouTube，而无需打开浏览器会话。

如果这对您来说还不够，请查看我们关于使用TunerFree MCE或Second RunTV添加更多流媒体频道的文章。

录制和编辑电视

Media Center允许的更酷的功能之一是能够像使用DVR一样使用您的PC并录制实况电视。虽然这真的只是一个简单的点击录制按钮的过程，但您也可以设置计划来录制实况广播，并使用Windows Live Movie Maker编辑您的电视录制。

还有一些先进的工具，比如远程土豆，可以让你远程安排和流式播放录制的电视节目。

删除商业广告

虽然本机控件可以很好地观看直播电视，但您可能希望能够跳过商业广告。你也可以尝试一些其他的插件，让你有更多的控制，比如用MCEBuddy转换视频和删除商业广告。

另一种帮助处理商业广告的方法是使用简单的注册表黑客来增加实况电视和录制电视的跳过和重放间隔。

听音乐

他们称它为媒体中心是有原因的--它不仅仅局限于电影和电视。大多数人不会考虑使用WMC来管理他们的音乐收藏，但是如果你想把你的多媒体放在一个中心位置，你可以从抓取你的CD收藏开始。

然后，您可以通过创建播放列表来组织音乐库。

收听地方广播和在线广播

如果你是一个仍然喜欢他们的收音机的人，你可以使用WMC收听本地和在线的流媒体广播。如果您有受支持的调谐器卡和连接的调频天线，您可以收听本地调频广播…。甚至可以创建工作站预设。

或者，如果您想收听流媒体广播，请查看如何添加XM Radio帐户，或者您可以使用RadioTime插件免费收听来自世界各地的100，000多个广播电台。

管理图片

如果您的收藏中有很多图片，您可以使用WMC图片库来帮助管理和显示它们。它提供了创建图片幻灯片的功能，这样您就可以轻松地与朋友和家人共享这些图片。

为了更好地控制您的照片，您可以使用WMC来裁剪、编辑和打印照片。

自定义Windows 7 Media Center

现在您已经知道如何使用wmc播放音乐、电影、图片和电视…。您可能希望了解一下如何对其进行自定义。您可以做的一件事是添加背景图像和主题，以获得更美观的UI。

将背景和主题添加到WMC的用户界面后，您可以使用Media Center Studio自定义开始菜单。

若要进一步自定义“开始”菜单，可以通过隐藏从不使用的菜单条来消除混乱。

将您的手机用作Windows 7 Media Center的遥控器

如果你没有调谐器卡的遥控器，你可以做的一件很酷的事情是用你的Android手机作为Gmote的遥控器。

如果你没有Android手机，你也可以用iPhone或iPod Touch作为遥控器。你不仅可以使用这些手机应用程序控制WMC，还可以控制你的整个PC。

其他Windows 7 Media Center提示与技巧

如果您的孩子可以访问Media Center，您可能需要设置家长控制功能。这允许您阻止您不想让您的孩子看到的内容。您可以选择他们可以或不能观看或收听的内容。如果他们试图访问不适当的媒体，他们将不得不输入只有您知道的代码。

如果你花太多时间看电视和电影，你可能会忘记做重要的事情。如果你倾向于沉浸在一部电影中，忘记了你需要做的事情，你可以设置在观看节目时弹出的事件提醒。

如果您在尝试完成任务时，Media Center决定获取节目列表的更新，您会感到恼火，您可以将更新安排在特定的时间进行。

如果您是Windows Home Server用户，使用它设置Windows Media Center连接器是一项很酷的功能。这将允许您从服务器访问媒体内容，并对其存储、运行状况、备份等进行标记。
