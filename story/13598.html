菜单键是用来做什么的？(以及如何重新映射)

你知道你的键盘有菜单键吗？在全尺寸键盘上，它位于右Ctrl键的左侧。此键可打开上下文菜单，但您可以对其进行调整以使其更有用。

你键盘上的菜单键在哪里？

在全尺寸键盘上，菜单键位于空格键右侧的右Windows键和右Ctrl键之间。菜单键有时也称为“应用程序键”。

一些较小的键盘(例如笔记本电脑键盘)省略了Menu键以节省空间。其他较小的键盘省略了向右的Windows键，而将Menu键保留在向右的Alt和Ctrl键之间。

无论哪种方式，如果您的键盘有菜单键，它将位于您的右Ctrl键的左侧。它上面没有印有“菜单”的字样--它有一张看起来像菜单的小图片。此图片未标准化，在不同的键盘上看起来会不同。有时它显示一个小指针在菜单上方徘徊，有时它看起来像一个风格化的菜单-一个里面有几条水平线的正方形或矩形。

菜单键是用来做什么的？

菜单键可打开当前应用程序的上下文菜单。这基本上就像在应用程序中右键单击您的选择。

尝试一下-在查看此网页时按下Menu键，您将看到Web浏览器的上下文菜单，就像您在页面上单击了鼠标右键一样。

如果您没有鼠标或没有带鼠标右键的鼠标，则此键非常有用。它可以在许多不同的应用程序中工作。如果您在文件资源管理器中选择一个文件或文件夹并按下Menu键，您将看到一个上下文菜单，就像您右键单击了该文件一样。

此键使您可以只使用键盘而不使用鼠标来使用上下文菜单。按Menu键，使用箭头键选择一个选项，然后按Enter将其激活。使用键盘快捷键选择文本或其他元素，然后按Menu键激活上下文菜单选项-所有这些都不需要您的手离开键盘。

微软现在正在讨论将此密钥转换为Office密钥，以匹配Windows密钥。大多数个人电脑用户可能不会碰这个键，这就解释了为什么微软提出了改变它的想法。它有点像Sys RQ，Scroll Lock和Pause Break键。

Shift+F10也可以起到菜单键的作用

如果你的键盘没有菜单键，但是你想用键盘快捷键打开上下文菜单，不要害怕。在大多数应用程序中，可以按Shift+F10组合键打开上下文菜单。它基本上和菜单键是一样的。

然而，这并不是在每个应用程序中都有效-它取决于应用程序。如果您正在使用的应用程序没有任何反应，请尝试Ctrl+Shift+F10。

如何重新映射菜单键

菜单键不像Windows键那么烦人，如果你不小心按下它，Windows键可能会让你退出游戏和其他全屏应用程序。但是，您可能希望更改菜单键的行为，使其执行更有用的操作。毕竟，如果你不使用按键，那是最好的键盘空间。

我们喜欢SharpKeys，因为它可以快速地将一个键重新映射到另一个键。您可以在Windows注册表中重新映射键，但这要复杂得多。SharpKeys提供了一个方便的图形界面，可以为您自定义基础注册表值。

安装并启动SharpKeys后，单击“添加”按钮添加新的重新映射。

在左窗格中选择“特殊：应用程序(E0_5D)”。您也可以单击“键入键”并按下菜单键-正如我们上面提到的，它有时被称为“应用程序键”，就像这里所说的那样。

在右窗格中，选择要将菜单键重新映射到的任何键。例如，您可以选择“Web：Back”，该键将在您的Web浏览器和任何其他支持该键的应用程序中充当Back键。

完成后，单击“确定”。

单击“写入注册表”将更改写入Windows注册表。您现在必须关闭“快捷键”窗口，然后重新启动PC或注销并重新登录。您的更改将在您下次登录时生效。

如果要更改键的功能或撤消更改，请再次打开“快捷键”，选择您的规则，然后使用“编辑”或“删除”按钮修改或删除它。将更改写入注册表，然后注销并重新登录。

您也可以使用SharpKeys重新指定其他关键点-例如，可以使Caps Lock或Windows键作为其他关键点使用。

如何使用自动热键自定义菜单键

要进行更高级的自定义，我们建议使用AutoHotkey。您可以使用AutoHotkey编写一个快速的小脚本，当您按下它时，该脚本将侦听菜单键并执行其他操作。在AutoHotkey中，此键称为“AppsKey”。

例如，AutoHotkey脚本中的以下行将禁用菜单键(“AppsKey”)并使其不执行任何操作(“Return”)：

AppsKey::Return

自动热键脚本中的以下代码将侦听菜单键，并在您按下它时启动Microsoft Word：

AppsKey:: Run WINWORD return

菜单键很有可能在某一时刻消失。然而，由于键盘上仍然有滚动锁键，菜单键很有可能在未来几十年内仍然存在。
