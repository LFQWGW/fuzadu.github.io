如何扎根您的Nexus手机或平板电脑

使用Nexus Root Toolkit可以快速启动Nexus设备，无论您的是Nexus 7、Galaxy Nexus，甚至是Nexus S。启动允许您使用功能强大的应用程序，这些应用程序在Android的默认沙箱中不起作用。

我们以前已经介绍过在Android上扎根，提供的方法并不适用于很多用户--很难给出适用于所有Android设备的指令。此方法应该可以完美地工作，但仅适用于Nexus设备。

此过程将擦除Nexus设备中的数据，因此您必须在此过程中备份和恢复数据。在此过程中，您还将解锁Nexus设备的引导加载程序，从而允许安装自定义ROM。

安装Nexus根工具包

您可以在这里下载由WugFresh开发的Nexus Root Toolkit。下载.exe文件后运行该文件进行安装。Nexus Root Toolkit将在您安装后自动启动。选择您正在使用的设备型号和Android版本。

如果您不完全确定正在使用的设备，请打开设备上的设置屏幕，然后选择关于平板电脑或关于手机。

驱动程序安装

确保您的计算机具有正确的驱动程序，以便它可以与您的Nexus通信，这可能是根操作中最复杂的部分之一。Nexus Root Toolkit包括自动驱动程序配置-如果失败，它可以引导您完成手动设置驱动程序的过程。

您需要在Nexus上启用USB调试才能解锁和启动它。打开Settings(设置)屏幕，选择Developer Options(开发人员选项)，将Developer Options(开发人员选项)屏幕顶部的滑块设置为ON，并启用USB Debug(USB调试)复选框。

单击Nexus Root Toolkit窗口中的Full Driver Installation Guide-Automatic+Manual(完整驱动程序安装指南-自动+手动)按钮开始。

假设您使用的是Windows7，请尝试自动驱动程序配置按钮-理想情况下，这将为您解决所有问题。

出现提示时，使用随附的USB电缆将Nexus连接到您的计算机。Windows安装完驱动程序后，单击确定。

如果这不起作用，您必须按照驱动程序设置窗口中的步骤手动配置驱动程序。按照说明按顺序完成每个步骤-他们会带您完成整个过程。就我个人而言，我必须完成手动驱动程序设置，安装三星驱动程序(Nexus7是华硕设备，但三星驱动程序对我有效)。

后退 / 倒车 / 向后倒 / 备份

此过程将擦除您的设备，因此您可能需要先备份它。单击“备份”按钮备份您的设备。

单击“创建Android备份文件”按钮以创建包含您的应用程序和数据的备份文件。点击设备屏幕上显示的“备份”按钮以继续。

您还应该使用备份数据/媒体选项来备份设备的媒体文件。

解锁根(&R)

备份Nexus后，使用解锁按钮解锁您的设备。解锁过程会擦除您的设备，因此在继续之前请确保您已备份！

Nexus上会提示您解锁引导加载程序。在您同意之后，您的引导加载程序将被解锁，并且您的Nexus将被擦除并重置为其出厂状态。在继续之前，您必须在Nexus上完成设置过程，并再次启用USB调试模式。

完成后，选择不要刷新CWM，除非您还想将ClockworkMod刷新到您的设备，然后单击Root按钮。

Root进程完成后，在Nexus上打开SuperSU应用程序并更新SU二进制文件。

启动BusyBox应用程序，授予其root权限，然后点击Install按钮。您现在已成功扎根！

从备份恢复

现在，您可能希望从之前进行的备份中恢复Nexus。单击恢复按钮。

使用Restore Android Backup File和Restore Data/Media按钮，选择您之前创建的备份文件。

当设备上出现完整还原屏幕时，点击还原我的数据按钮。同意后，您的应用程序和数据将恢复。
