如何在Windows 8或10中更改无线网络的优先级

Windows会自动将您连接到您过去连接过的无线网络。如果您靠近以前连接过的多个网络，Windows会使用优先级系统来选择Wi-Fi网络。

Windows 7是一个图形化工具，用于管理您的无线网络优先级。不过，在Windows 8和10上，您只能从命令提示符进行管理。

如何查看您的无线网络的优先级

Windows 10在设置窗口中显示您的无线网络的优先级。要查看列表，请转到设置>网络和互联网>Wi-Fi>管理已知网络。

此列表按优先级顺序显示您保存的无线网络。Windows将首先连接到顶部的无线网络(如果可用)，然后向下移动。如果您可以将无线网络拖放到此处，您就可以对列表进行重新排序。但Windows不允许您这样做。

如果您不希望Windows将来将您连接到无线网络，您可以单击此处并选择“忘记”。除非您选择连接到该网络，否则Windows不会自动将您连接到该网络。

如何正确更改无线网络的优先级

要手动更改优先级，您需要使用管理员命令提示符。要打开命令提示符，请右键单击开始按钮，然后选择“命令提示符(管理员)”。

运行以下命令，按优先级顺序查看已保存的Wi-Fi网络列表。这将显示与您在设置屏幕中看到的列表相同的列表：

netsh wlan show profiles

这里您需要注意两件事：接口的名称和您要优先处理的无线网络的名称。

例如，这里的接口名称是“Wi-Fi”，我们选择的优先网络是“Remora”。

要确定无线网络的优先级，请运行以下命令，将“wifi-name”替换为Wi-Fi网络的名称，将“interface-name”替换为Wi-Fi接口的名称，并将“#”替换为要放置Wi-Fi网络的优先级号。

netsh wlan set profileorder name="wifi-name" interface="interface-name" priority=#

例如，要获取接口Wi-Fi上的网络备忘录并将其设置为列表中的第一优先级，我们将运行以下命令：

netsh wlan set profileorder name="Remora" interface="Wi-Fi" priority=1

您可以再次运行netsh wlan show profile命令，您将看到您选择优先处理的网络首先出现在列表中。设置应用程序中的顺序也将更改。
