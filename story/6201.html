如何从您的电话远程访问您的计算机

在几乎每个计算机用户的生活中，都会有这样一个时刻，他们需要PC…中的文件。而且电脑不在附近。幸运的是，有一种简单的方法可以直接从手机或平板电脑远程访问电脑，让找回丢失的文件变得轻而易举。

虽然有很多其他选择可以实现这一点，但我们将把重点限制在几个最简单、最方便的选择上。为什么要更努力地工作，而不是做更少的事情？这是你最好的选择。

选项一：Chrome远程桌面(Windows、Mac、Linux、Android、iPhone)

啊，不错的老Chrome远程桌面。自从几年前首次发布以来，这一直是我个人远程访问的首选。它使用起来又快又容易，设置起来完全没有痛苦，几乎可以在任何设备上运行，从Windows、Mac和Linux到Android和iOS。

当然，它也有它的警告，比如你必须是Chrome用户。虽然Chrome的用户很多，但我知道也有相当多的用户不喜欢谷歌的浏览器，这没什么--下面我们会为你们讨论一个很好的选择。但如果你使用Chrome，这可能是你最简单的选择。

如何设置Chrome远程桌面(测试版)

Chrome Remote Desktop最初的迭代是作为Chrome网络商店中可安装的Chrome应用程序进行的，但由于Google正在逐步淘汰Chrome应用程序，以换取渐进式的Web应用程序，我们将从如何使用服务的全新Web应用程序版本设置远程桌面开始。

注意：在撰写本文时，web上的Chrome远程桌面仍处于测试阶段，因此可能会有一些漏洞。因此，我们暂时将设置远程桌面Chrome应用程序的教程留在下面。

要开始使用，请访问新的Chrome远程桌面测试版网站。单击“Set Up Remote Access”(设置远程访问)框下角的“Download”(下载)箭头。

这将弹出一个对话框来安装新的Remote Desktop扩展(不要与旧的Remote Desktop应用程序混淆，这是另一回事)。单击Add Extension(添加扩展名)按钮。

下载和安装需要几秒钟时间，之后您将输入您的计算机名称。点击下一个。

从这里选择您的登录PIN。让它变得更难！完成后，单击“开始”。

在Windows上，你应该会看到一个弹出框，询问你是否介意让这一功能对你的设备进行更改。点击‘是’，您就都设置好了。

如何使用旧版Chrome应用程序设置Chrome远程桌面

就像我之前说的，谷歌将在接下来的几个月里逐步淘汰这个版本的远程桌面(以及所有其他Chrome应用程序)，但现在它仍然可以使用，所以我们会一直保留这个版本，直到它不能使用为止。

首先，你需要在浏览器上安装Chrome远程桌面应用程序。它可以在Chrome网络商店买到，安装过程只需几秒钟。

安装后，您将从Chrome的应用程序菜单启动该应用程序-它应该是书签栏中的第一个链接。在那里，只需查找Chrome远程桌面链接即可。

第一次启动它时，您必须通过安装一个小实用程序来启用到计算机的远程连接。这个过程非常简单，Chrome会引导你完成整个过程。

下载并安装该实用程序后，Chrome Remote Desktop将提示您输入PIN。当然，要做一些你容易记住，但其他人很难理解的事情！(所以，不要使用123456或其他值。你知道得更清楚！)

输入PIN后，需要几秒钟才能启用远程连接。一旦完成，您的计算机-无论其名称是什么-将显示在列表中。在这个屏幕截图中，你可以看到我的笔记本电脑和我的台式机。

从那里，您可以从您登录的任何其他计算机或移动设备轻松访问您连接的任何计算机。你唯一要记住的是，Chrome远程桌面软件必须在你需要之前先安装好-你不能远程操作！

如何使用Chrome远程桌面连接到您的PC

现在一切都准备好了，你需要为你的手机(Android或iOS)下载Chrome远程桌面应用程序。启动它就可以开始了-我在这里使用的是Google Pixel XL，但无论你使用的是哪种设备，过程都应该大致相同。

一旦你启动这款应用程序，它应该会显示一个列表，如果你的电脑。点击您需要连接的设备。

它将在此处请求您的PIN码，并提供选项以始终允许在没有PIN码…的情况下从该设备连接。老实说我不建议你这么做。如果你的手机出了什么问题，你毕竟不会希望任何潜在的坏人同时访问你的手机和电脑。

输入PIN后，点击“连接”。

砰。连接将是即时的。此时，您可以轻松地执行所需操作-将手指用作鼠标，然后轻触即可单击。这几乎是说得通的。

完成后，只需点击底部的“停止共享”按钮即可。连接将被终止。简单的农民。

选项二：TeamViewer(Windows、Mac、Linux、Android、iPhone)

就像我之前说的，并不是每个人都使用Chrome。如果你不使用Chrome，那么你就不能很好地使用Chrome远程桌面，不是吗？

相关：如何锁定TeamViewer以实现更安全的远程访问

对于其他所有人，都有TeamViewer。和Chrome Remote Desktop一样，它也是一应俱全的--Windows、Mac、Android、iOS等等。虽然安装和使用并不是那么难，但我现在要警告你，Chrome Remote Desktop要简单得多。而且TeamViewer的复杂性意味着它需要更多的工作来确保安全-如果您要使用它，这是您绝对需要做的事情。如果你不这样做，你基本上就是让你的房子的门没有上锁，这是不好的。

如何设置TeamViewer

首先，访问TeamViewer网站并下载该程序的最新版本。主屏幕上有一个绿色的大盒子，很难让人看不见。

在安装过程中，您需要选择安装类型和用例。TeamViewer对个人使用是免费的，所以如果您只是在您的个人计算机上执行此操作，请使用该选项。不过，如果您将其用于公司用途，请在这里诚实。

安装只需要几分钟，您就可以开始了。

默认情况下，TeamViewer将为您提供远程ID和PIN，但这仅在您实际站在计算机前面时才有用-这里的想法是您可以将其提供给其他人，以便他们可以远程访问您的计算机。但是，如果您外出走动并且需要远程访问您自己的系统，这不会有太大的好处。

为此，您需要设置一个TeamViewer帐户并将您的计算机连接到该帐户。要进行设置，请单击右侧小窗口中的“注册”按钮。当然，如果您已经有TeamViewer帐户，只需登录即可。

一旦您的帐户全部设置好并可以使用，您还将为这台特定的计算机分配密码。再说一次，做一些容易记住但很难弄明白的事情。现在也是调整这些安全设置的好时机。

如何使用TeamViewer连接到您的PC

要访问您的PC，请在您的Android或iOS设备上安装TeamViewer的移动应用程序，然后启动它。*点击底部的“Computers”按钮，然后登录到您刚刚创建的帐户。

之后，点击“我的计算机”，将显示当前附加到您的TeamViewer帐户的所有计算机的列表。

点击您要连接的设备。远程连接需要一分钟时间才能建立，但在此之后，您就可以开始滚动了。

界面的底部(同样是在电话上)将显示您可以执行的操作的快速列表：关闭、键盘、操作(如键盘快捷键)、Windows、TeamViewer设置和隐藏。这些都是非常有用的设置，不仅仅是简单的点击式使用，也是TeamViewer在Chrome远程桌面上占据优势的一个领域。

完成操作后，只需单击X按钮(或“上一步”)即可关闭连接。

如何使用TeamViewer来回传输文件

但是等等，还有更多！如果您只是想要抓取几个文件，这里还有另一个选择：您可以使用TeamViewer的文件传输系统。

应用程序登录到您的TeamViewer帐户后，点击底部的“Files”选项，然后点击“Remote Files”。

登录后，点击“我的计算机”按钮，然后选择您需要访问的计算机。

从这里开始，非常简单：在文件系统中导航，然后点击您想要传输的文件旁边的复选框。选择文件后，点击底部的“我的文件”按钮，然后点击顶部的小纸质图标，将文件传输到所需位置。

完成后，只需轻触“上一步”按钮即可断开连接。真的就是这样。

有无数其他的远程访问选择，但这是最好的跨平台选项中的两个，无论您拥有哪种计算机或电话，都应该可以正常工作。

虽然我承认我使用Chrome远程桌面来满足我所有的远程需求(这通常是很少见的)，但我承认TeamViewer显然是这里更强大的选择。文件传输选项执行出色，易于使用。只要确保，如果您想要利用TeamViewer的功能，您就可以采取必要的步骤来保护它。
