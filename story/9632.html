如何在Ubuntu上获取最新的NVIDIA、AMD或英特尔显卡驱动程序

多亏了Steam for Linux，支持Linux的游戏比以往任何时候都多。但是，就像在Windows上一样，这些游戏中的许多都需要最新的图形驱动程序，以实现最佳性能和最少的错误。最新版本的Ubuntu可能包括更新的驱动程序，但不一定是最新的驱动程序。

你可以自己安装最新的驱动程序，但要小心：这样做可能会遇到问题。Ubuntu会为每个版本打包并测试某些版本的图形驱动程序，并且不会执行重大更新。为了获得最大的稳定性-尤其是如果你不关心游戏-坚持使用Ubuntu提供的图形驱动程序。即使您直接从制造商下载最新的图形驱动程序，也可能会遇到错误。

您使用的是32位版本的Ubuntu还是64位版本的Ubuntu？

相关：如何检查您的Linux系统是32位还是64位

在继续之前，您需要知道您使用的是32位版本的Ubuntu还是64位版本的Ubuntu。要在Ubuntu的默认Unity桌面上进行检查，请单击屏幕右上角的设备，然后选择“关于这台计算机”。您将看到此信息显示在“OS类型”的右侧。您也可以在航站楼检查这个。

在继续之前，您还应该从Update Manager安装任何可用的Ubuntu更新。您将在“关于此计算机”窗口中看到一个“安装更新”按钮，该按钮将为您解决此问题。

官方的Graphics Drivers Team PPA-这是个人软件包存档的缩写-旨在长期解决这个问题。它将提供更新的图形驱动程序。游戏玩家将能够启用这个可选的存储库，并获得更新的图形堆栈，而不需要搜索它，普通的Ubuntu用户将能够忽略它，继续使用当前版本的Ubuntu附带的稳定版本。

无论如何，这是我们的长期目标。目前，这款PPA还在测试中。它目前也只提供NVIDIA驱动程序，所以如果你有AMD或英特尔显卡硬件，你就必须到其他地方去找你的驱动程序。

要将此PPA添加到您的系统，请打开终端窗口并运行以下命令：

sudo add-apt-repository ppa:graphics-drivers/ppa

完成后，运行以下命令下载最新的软件包列表：

sudo apt-get update

更新的NVIDIA图形驱动程序现在可以安装。您可以查看PPA的描述页面，查看可用的最新版本，或者键入以下命令并按“Tab”键查看列表：

sudo apt-get install nvidia-

例如，目前361版的NVIDIA图形驱动程序是可用的最新版本。您需要运行以下命令来安装它：

sudo apt-get install nvidia-361

也可以使用NVIDIA自己的安装程序直接从NVIDIA下载和安装图形驱动程序。此工具设计为独立于发行版，在任何Linux发行版上编译和安装最新的驱动程序。每个图形驱动程序版本都有一个自述文件，提供安装说明和更多信息。

但是，如果可能的话，最好坚持使用专门为您的Linux发行版构建的包。我们推荐PPA，除非由于某些原因对您不起作用。

AMD

AMD的Catalyst驱动程序-现在被称为Radeon Crimson，但仍然只是旧的fglrx驱动程序-是在AMD硬件上实现最佳Linux游戏性能所必需的，AMD正在为未来开发一个新的开源驱动程序架构，但它还不能与旧的fglrx驱动程序竞争。

不幸的是，目前似乎没有可用的最新版本的PPA。他们最终应该是上面提到的图形驱动团队PPA的一部分，但目前还不是。

您必须安装并使用AMD提供的官方fglrx驱动程序包。访问AMD的Linux下载中心，下载适用于您的图形处理器的驱动程序。

安装这些驱动程序的确切方法取决于您使用的Ubuntu版本。AMD目前为Ubuntu15.04、Ubuntu14.04LTS和Ubuntu12.04LTS提供了预先构建的.deb包。如果您使用的是旧版本的Ubuntu，或者是新版本的Ubuntu(AMD还不支持Ubuntu 15.10)，那么您需要在下载页面上选择通用的“Linux”选项，并使用AMD的工具自己编译和构建软件包。AMD的官方Linux图形驱动程序安装程序注意到，它将指导您完成驱动程序的下载、安装和任何可能发生的问题的故障排除的过程。

英特尔提供了“Intel Graphics Installer for Linux”，它将在Ubuntu上下载并安装最新的开源英特尔图形堆栈软件，而在撰写本文时，最新版本是支持Ubuntu 15.10的Intel Graphics Installer for Linux 1.2.1。英特尔的图形安装程序不支持任何其他版本的Ubuntu，包括Ubuntu 14.04 LTS。它可能总是只支持最新版本的Ubuntu。这种东西也不含PPA。

谢天谢地，它至少是一个图形工具。下载并安装软件包，您就可以从应用程序菜单启动“Intel Graphics Installer”。它将从英特尔下载最新的图形软件包并为您安装。

在此之后，您很有可能需要重新启动。至少，在新驱动程序运行之前，您必须注销并重新启动图形X服务器。重新启动是确保您的系统使用新的图形驱动程序和库的最快方式。

如果您使用软件存储库安装驱动程序，则在向PPA添加新版本时，您将在Update Manager中收到更新。如果您使用的是制造商提供的安装程序，您将不会自动收到更新-您将来必须重新下载并重新运行该安装程序才能获得最新版本。
