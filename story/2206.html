如何在VMware工作站上安装Windows Home Server Beta“Vail”

如果您是Windows Home Server的爱好者，您可能想测试一下代号为Vail的新版本。您可能没有额外的盒子来测试它，所以这里我们来看看如何在VMware Workstation上安装它。

开始

您需要以下项目才能开始使用。

具有有效许可证的VMware Workstation，或者您可以免费使用30天试用期。 WHS Vail Beta ISO。 可实现硬件虚拟化的计算机。 主机上有足够的硬盘空间来创建160 GB的虚拟硬盘，这是最低系统要求。 主机上至少有2 GB或更多RAM-分配给Vail的最小RAM为1 GB

在本文中，我们使用了一台配备酷睿i3处理器和6 GB RAM的计算机，运行Windows7旗舰版x64和VMware Workstation 7.1。

硬件虚拟化

开始之前，您需要确保您的计算机能够进行硬件虚拟化。如果不是，您可能需要检查您的BIOS设置以启用它。还要检查并安装适用于您的计算机的任何BIOS更新。一个优秀的实用程序可以快速发现它是否能够进行硬件虚拟化，这是安全的。不需要安装…。只需下载文件并运行即可。

安装VMware

如果您还没有，请下载并安装VMware Workstation(下面的链接)。如果您已经安装了免费的VMware Player，系统将提示您先卸载该播放器。不要担心丢失你在Player中创建的任何虚拟机，因为工作站也附带一个版本的Player。

需要重新启动才能完成VMware Player的卸载。

当您从重新启动返回或刚刚开始安装时，请通过向导接受典型安装的默认值。在此过程中，您还将被要求输入您的许可证号或继续免费试用。

同样，需要重新启动才能完成安装。

创建新虚拟机

安装后，您将在任务栏中看到VMware Workstation图标。从那里启动它，然后单击New Virtual Machine。

当New Virtual Machine Wizard启动时，单击Custom(Advanced)Configuration(自定义(高级)配置)。

在向导的下一步中，只需保留默认值并选择Next。

现在选择我将稍后安装操作系统。这将通过首先配置正确的VMware设置，而不是让其执行无人参与安装，从而简化安装过程。

对于来宾操作系统，请选择Windows Server 2008 x64。

为您的VM指定一个有意义的名称。如果您最终安装了许多不同的VM，为它们指定唯一的名称有助于保持它们的一致性。

在本文中，我们没有更改处理器设置，而是保留默认设置。不过，这可能是您稍后想要尝试的东西。

现在决定要为Vail VM分配的内存量。请记住，它将使用主机内存，因此如果您不想在VM运行时大幅降低其他任务的速度，您可以将其设置为1 GB的最低要求。在我们的系统上，我们有一些空闲的内存，所以我们将其提升到2 GB。

对于网络类型，选择使用桥接网络以获得最佳性能。这将为您的Vail VM提供其自己的IP地址，并将充当您网络中的一台单独的计算机。

对于SCSI控制器类型，您只需将其保留为默认推荐的LSI Logic SA即可。同样，您可能希望在另一个时间尝试这些设置，但是现在您可以离开它。

现在您需要为Vail创建一个新的虚拟磁盘。

选择要为scsi…的磁盘类型

现在您需要选择虚拟硬盘的大小。最低要求是160 GB，因此您需要确保主机上有足够的磁盘空间。

然后您可以指定磁盘文件，我们只是将其保留为默认值。

您将获得创建Server VM时选择的设置的概述。仔细检查一下，如果有些地方看起来不对劲，您可以在这一点上返回并进行更改。

在Workstation主屏幕中，在Devices下双击CD/DVD，以便我们可以安装ISO映像。由于Vail采用ISO格式，因此不需要将其刻录到VM的光盘上。您可能希望在以后将其安装到实际计算机上时将其烧毁。

选择Use ISO image file(使用ISO映像文件)并浏览到其位置。

现在打开机器电源，开始Vail的安装过程。

像往常一样完成安装。如果您以前没有安装过，请不要担心，按照向导操作，该过程非常简单。虚拟机将重新启动几次，这是正常的，所需的时间会有所不同。在我们的机器上花了大约30分钟。

安装过程完成后，单击我已完成安装按钮。

在安装过程中，它没有占用Corei3桌面上的太多资源，但一旦您开始使用Vail，它将使用您为其分配的主机资源。

既然Vail已经安装在VMware Workstation上，您就可以登录并开始使用它了。在按CTRL+ALT+DELETE屏幕上，从工具栏中选择VM\Send Ctrl+Alt+Del。

现在您可以登录到新的虚拟Vail Server。

VMware工具

现在，您将需要安装VMware工具来增强虚拟机体验。从工具栏转到VM\Install VMware Tools。

当AutoPlay屏幕出现时，单击Run setup.exe安装工具。

现在运行VMware Tools向导并接受默认设置进行安装。

需要再次重新启动才能完成VMware Tools的安装。

这就对了！您现在可以虚拟测试Windows Home Server的测试版。如果您想测试客户端，您可以创建另一个虚拟机并连接它。您还可以通过RDP访问虚拟服务器，以便测试新功能。如果您已经在主机上安装了WHS版本1连接器软件，您将无法安装连接器软件。

VMware播放器

还记得在本文开始时我们提到过，您在免费VMware Player中安装的任何虚拟机都可以吗？如果安装成功，没有任何错误，这是正确的。您可以从开始菜单\所有程序\VMware文件夹访问VMware Player。

在我们的测试中，我们之前设置的所有虚拟机都工作正常，没有任何问题。

结束 / 结尾 / 结论 / 推论

这就是一切！这个过程相当简单，因为一切都是由向导驱动的。您只需确保按照正确的顺序执行步骤并选择正确的选项即可。现在，您可以尽情地测试Vail，而不必担心有额外的机器来运行它。请记住，Vail还在Beta中，在这一点上，您肯定不想去掉版本1而用Vail取而代之。如果你没有工作站的许可证，他们会提供30天的免费试用，这样你至少可以玩一个月的Vail。

如果您确实有一台额外的计算机在收集灰尘，并且希望将其转换为Windows Home Server，请查看我们关于如何安装和设置WHS版本1的指南。
