Windows、Linux和Mac的最佳自由文本编辑器

我们都使用文本编辑器做笔记、保存网址、编写代码以及其他用途。每个操作系统都有一个默认的基本文本编辑器，但我们中的大多数人都安装了自己的增强型文本编辑器，以获得更多功能。

在本文中，我们收集了许多用于不同目的的不同文本编辑器的链接。您可以使用文本编辑器进行基本的文本编辑和做笔记、编写编程代码、生成LaTeX文档、写一本书，以及许多其他用途。

记事本和写字板更换

您是否正在寻找Windows中默认记事本之外的更多功能？您更愿意在Linux中使用图形文本编辑器，而不是内置的vi吗？对于有用的文本编辑器，有很多选择。

有些采用选项卡式界面，比如Jarte(基于写字板文字处理引擎，很容易与WordWeb集成)、EditPad Lite(也具有自动备份功能)和Notesab Light(还可以计算输入程序的数学表达式的值)。Jarte、EditPad Lite和Noteab Light都只在Windows上可用。Jarte也可以作为可移植程序使用。

通常，Vi是Linux操作系统中的默认文本编辑器，它是一个键盘密集型程序，没有图形用户界面(GUI)。TED记事本是适用于Windows的一个很好的文本编辑器，它有312个文本处理功能、创新特性和省时工具的热键，它也可以作为可移植程序使用。Emacs还可用于Windows和Linux，并且可以自定义。它还包括文件比较实用程序和文件管理器。您还可以将组织模式添加到Emacs中，这是一个个人信息管理和大纲工具。如果您更喜欢带有GUI的文本编辑器，Vim和Gedit都是不错的选择，并且都是可用的。Vim本质上是Vi的图形版本。有关在Vi或Vim中编辑文本文件的帮助，请参阅我们的初学者指南。

GetDiz是Windows的记事本替代品，允许您从Windows资源管理器中快速编辑许多文本文件，并增强了处理DIZ和NFO文件的功能。它还可以正确显示ASCII图片。另一个适用于Windows的ASCII文本格式化工具是TextMorph，它还可以将文本与HTML相互转换，并清理电子邮件(删除所有的“>”符号等)，以及按单词或多个段落进行搜索和替换。

程序员文本编辑器

有许多为程序员提供有用功能的文本编辑器。大多数都支持多种编程语言的语法突出显示、多文档编辑，并且可以通过插件进行扩展。有些还允许通过FTP编辑远程文件。

PSPad不仅支持语法突出显示，还支持大多数流行编程语言的匹配括号突出显示。它还有一个十六进制编辑器、宏录制器和一个差分工具。PSPad也很容易与免费版本的Topstyle CSS编辑器集成。Notepad++还支持括号突出显示和宏录制。它还支持语法折叠，并且可以使用附带的插件管理器通过插件进行高度自定义。PSPad和记事本++仅适用于Windows。

跨平台(Windows、Linux和Mac OS X)编辑器jEdit支持200多种编程语言的语法高亮显示和自动缩进，以及差异实用程序、FTP浏览器和块选择。它还可以使用插件和宏进行扩展，并且通过内置的插件管理器功能可以使用数百个插件和宏。

Programmer‘s Notepad for Windows支持使用方案(内置和用户定义)突出显示语法，代码折叠和大纲显示，具有多级拆分视图的选项卡式界面，以及导出为HTML(使用CSS)和RTF的能力。

如果您喜欢Linux中的Vi编辑器，但更喜欢作为程序员文本编辑器的图形编辑器，那么Editra和Komodo Edit是不错的选择。它们都提供了Vi仿真，以及对许多编程语言中的语法突出显示和代码折叠的支持。Editra有一个选项卡式界面，允许块(取消)注释和(取消)缩进，并且可以使用内置的插件下载/安装程序进行扩展。Komodo Edit支持后台语法检查，并包含带有shell命令集成、宏和代码片段的工具箱。Editra和Komodo Edit都可用于Windows、Linux和Mac OS X。

其他选项包括以下选项：

Crimson Editor-一个非常小的Windows编辑器，包含一个目录树视图窗口。 Geany-用于Windows、Linux和Mac OS X的小型快速IDE，支持代码折叠、代码导航、构建系统和插件界面。 Notepad2-Windows版的快速、轻量级文本编辑器，类似记事本，具有语法突出显示功能，可作为便携程序运行

Microsoft Word替换

还有一些免费程序可以替代Microsoft Word。它们可以用作文本编辑器，但它们比简单的文本编辑器具有更多的格式化功能。您可以添加图像和表格、更改字体和颜色以及插入超链接。

AbiWord可以在Windows和Linux上运行，可以读写OpenOffice.org文档、Microsoft Word文档、WordPerfect文档、Rich Text格式文档和HTML网页。它具有高级文档布局选项，如表格、项目符号、编号列表、图像、样式、脚注和尾注。它甚至有一个像Microsoft Word一样的邮件合并实用程序。您可以使用多种插件来扩展AbiWord，这些插件可以在安装AbiWord时选择。还提供便携版本，您可以从USB闪存驱动器运行。

Angel Writer是一款适用于Windows的小型富文本编辑器，具有很高的性能，让您可以轻松创建令人印象深刻的文档。

极简主义文本编辑器

如果您在写作时被文本编辑器和文字处理器中过多的功能分散了注意力，那么您可能想尝试一下所谓的“极简主义”文本编辑器。它们是“朴实无华”的编辑器，要么不提供任何格式化功能，要么不提供现代文字处理器的许多其他功能，甚至是第三方文本编辑器，或者这些功能在你需要之前都是隐藏的。没有所有花哨的面孔在你面前，你可以集中精力写作。下面是我们找到的一些极简主义文本编辑器的列表。

Dark Room-适用于Windows，需要.NET Framework 2.0，并且可以作为可移植程序使用。 JDarkRoom-适用于Windows、Linux和Mac OS X。 Q10-可用于Windows，也可作为便携程序使用。 文案-可用于Windows，也可作为便携程序使用。 WriteMonkey-可用于Windows，也可作为便携程序使用。 Bookwrite-适用于Windows和Linux。 脚本-可用于Linux。 FocusWriter-适用于Windows、Linux和Mac OS X，也可作为Windows的便携程序使用

你甚至可以从微软免费下载Word5.5，并在Windows的DOSBox下运行。

如果你想要一个简单的文本编辑器，能够从设定的字数开始倒计时，可以试试yEdit2for Windows。如果您必须编写一定数量的单词，yEdit2可以让您更轻松。

安全文本编辑器

您还可以将文本编辑器用作存储私人信息的安全位置。有几种文本编辑器要么包含加密功能，要么专门为安全存储文本而设计。上面的程序员文本编辑器部分提到了Notepad++，它允许您使用SecurePad插件添加加密功能，该插件可以通过插件管理器获得。SecurePad将加密当前文档或整个文档中的选定文本。

Steganos LockNote是一种在文件中安全存储信息块的小而简单的方法。例如，如果您购买了一个仅供下载的程序，则可以使用LockNote将该程序附带的产品密钥或序列号存储在同一文件夹中，以便您始终知道在哪里可以找到它。

CryptNote-可用于Windows，也可作为便携程序使用。 CryptoTE-可用于Windows和Linux，也可作为Windows的便携程序使用。 NotepadCrypt-作为便携程序提供给Windows。 Xint-适用于Windows。 f0dder的fSekrit-可用于Windows，也可作为便携程序使用

LaTeX编辑器

你写了很多科学论文、文件或书吗？如果是这样的话，有几个文本编辑器允许您通过图形界面轻松使用TeX/LaTeX(文档标记语言和文档准备系统)来创建数学内容和结构化文档，如学术文章、论文和书籍。

LaTeX编辑器(LED)-可用于Windows，也可作为便携程序使用。 LyX-适用于Windows、Linux和Mac OS X。 WinEdt-适用于Windows。 TeXstudio-适用于Windows、Linux和Mac OS X，在Windows和Mac OS X上作为便携程序提供。 Texaker-适用于Windows、Linux和Mac OS X

小说写作编辑

甚至还有一个用来写小说的编辑器，叫做yWriter5，可以在Windows和Linux上使用。它将你的小说分成章节和场景，帮助你跟踪你的工作。但是，yWriter5不会为您建议情节构思、人物名称或为您编写小说的任何部分。写作的创造性任务仍然取决于你，yWriter5只是让它变得更容易。

另一个值得一提的文本编辑器是Linux中的Nano，它是一个易于使用的文本编辑器，可以直接在命令行上运行。Nano默认安装在Ubuntu和许多其他Linux发行版中，并且比Vi或emacs更容易学习。
