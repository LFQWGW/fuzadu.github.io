如何在Windows上启用和保护远程桌面

虽然有很多替代方案，但微软的远程桌面是访问其他计算机的一个完全可行的选择，但它必须得到适当的保护。*在建议的安全措施到位后，远程桌面是极客可以使用的强大工具，可以让你避免安装第三方应用程序来实现这类功能。

本指南及其附带的屏幕截图是针对Windows 8.1或Windows 10制作的。但是，只要您使用的是以下Windows版本之一，就应该能够遵循本指南：

Windows 10专业版。 Windows 8.1专业版。 Windows 8.1版企业版。 Windows 8/企业版。 Windows 8专业版。 Windows 7专业版。 Windows 7企业版。 Windows 7旗舰版。 Windows和Vista商务版。 Windows®Vista旗舰版。 Windows®Vista企业版。 Windows®XP专业版

启用远程桌面

首先，我们需要启用远程桌面，并选择哪些用户可以远程访问计算机。按Windows键+R调出运行提示符，然后键入“sysdm.cpl”。

进入同一菜单的另一种方法是在开始菜单中键入“这台电脑”，右键单击“这台电脑”，然后转到“属性”：

无论采用哪种方式，都会调出此菜单，您需要在其中单击Remote选项卡：

选择“允许远程连接到此计算机”及其下面的选项“仅允许来自运行具有网络级身份验证的远程桌面的计算机的连接”。

没有必要要求网络级身份验证，但这样做可以保护您的计算机免受中间人攻击，从而使您的计算机更加安全。即使像Windows XP一样古老的系统也可以连接到具有网络级身份验证的主机，因此没有理由不使用它。

启用远程桌面时，您可能会收到有关电源选项的警告：

如果是这样，请确保单击电源选项的链接并配置您的计算机，使其不会进入睡眠或休眠状态。如果您需要帮助，请参阅我们关于管理电源设置的文章。

接下来，单击“选择用户”。

Administrators组中的任何帐户都已拥有访问权限。如果您需要向任何其他用户授予远程桌面访问权限，只需单击“添加”并键入用户名即可。

单击“检查名称”以验证输入的用户名是否正确，然后单击“确定”。-在“系统属性”窗口上也单击“确定”。

保护远程桌面安全

您的计算机当前可以通过远程桌面连接(如果您在路由器后面，则只能在您的本地网络上)，但是我们需要配置更多设置才能实现最高安全性。

首先，让我们来解决一个明显的问题。你授予远程桌面访问的所有用户都需要有强密码。有很多机器人不断地扫描互联网，寻找运行远程桌面的易受攻击的电脑，所以不要低估强密码的重要性。他们使用8个以上的字符(建议使用12+)，包括数字、小写和大写字母以及特殊字符。

转到“开始”菜单或打开运行提示符(Windows键+R)，然后键入“secpol.msc”以打开“本地安全策略”菜单。

在那里，展开“本地策略”并单击“用户权限分配”。

双击右侧列出的“允许通过远程桌面服务登录”策略。

我们建议删除此窗口中已列出的两个组：Administrators和Remote Desktop Users。*之后，单击“Add User or Group”并手动添加要授予远程桌面访问权限的用户。“这不是必要的步骤，但它为您提供了更多权限来控制哪些帐户可以使用远程桌面。”“如果将来您出于某种原因创建了一个新的管理员帐户，但忘记了为其设置强密码，您的计算机就会向世界各地的黑客开放，如果您从未费心删除该帐户。”

通过在运行提示符或开始菜单中键入“gpedit.msc”，关闭“本地安全策略”窗口并打开“本地组策略编辑器”。

当本地组策略编辑器打开时，展开“计算机策略”>“管理模板”>“Windows组件”>“远程桌面服务”>“远程桌面会话主机”，然后单击“安全”。

双击此菜单中的任何设置以更改其值。我们建议更改的设置包括：

设置客户端连接加密级别-将此设置为高级别，以便使用128位加密保护您的远程桌面会话。

要求安全的RPC通信-将此选项设置为已启用。

要求对远程(RDP)连接使用特定的安全层-将其设置为SSL(TLS 1.0)。

要求使用网络级身份验证对远程连接进行用户身份验证-将此选项设置为“已启用”。

一旦进行了这些更改，您就可以关闭本地组策略编辑器。我们拥有的最后一个安全建议是更改远程桌面侦听的默认端口。*这是一个可选步骤，通过模糊做法被认为是安全的，但事实是，更改默认端口号会极大地减少您的计算机将收到的恶意连接尝试的数量。*您的密码和安全设置需要使远程桌面无论侦听哪个端口都不会受到攻击，但如果可以，我们不妨减少连接尝试的数量。

通过模糊性实现安全性：更改默认RDP端口

默认情况下，远程桌面侦听端口3389。如果要选择一个小于65535的五位数字作为自定义远程桌面端口号，请记住该数字，然后通过在运行提示符或开始菜单中键入“regedit”来打开注册表编辑器。

注册表编辑器打开后，展开HKEY_LOCAL_MACHINE>System>CurrentControlSet>Control>Terminal Server>WinStations>RDP-TCP>，然后双击右侧窗口中的“PortNumber”。

在PortNumber注册表项打开的情况下，选择窗口右侧的“Decimal”，然后在左侧的“Value Data”下键入您的五位数字。

单击“确定”，然后关闭注册表编辑器。

由于我们更改了远程桌面使用的默认端口，因此需要将Windows防火墙配置为接受该端口上的传入连接。若要转到开始屏幕，请搜索“Windows防火墙”并单击它。

Windows防火墙打开后，点击窗口左侧的“高级设置”，然后右键点击“入站规则”，选择“新建规则”。

将弹出“新建入站规则向导”，选择“端口”并单击“下一步”。*在下一个屏幕上，确保选择了TCP，然后输入您之前选择的端口号，然后单击“下一步”。-再单击“下一步”两次，因为接下来几页上的默认值就可以了。*在最后一页上，为这个新规则选择一个名称，例如“自定义RDP端口”，然后单击“完成”。

最后几步

现在应该可以在本地网络上访问您的计算机，只需指定计算机的IP地址或名称，然后在两种情况下都使用冒号和端口号，如下所示：

要从网络外部访问您的计算机，您很可能需要转发路由器上的端口。然后，应该可以从任何具有远程桌面客户端的设备远程访问您的PC。

如果您想知道如何跟踪谁在登录您的PC(以及从哪里登录)，您可以打开事件查看器查看。

打开事件查看器后，展开应用程序和服务日志>Microsoft>Windows>TerminalServices-LocalSessionManger，然后单击操作。

单击右窗格中的任何事件可查看登录信息。
