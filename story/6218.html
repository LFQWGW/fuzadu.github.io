10个伟大的免费图标包，让您的Android手机成为主题

Android允许你定制自己的主屏幕，添加小工具，安排快捷方式和文件夹，选择背景，甚至完全取代附带的启动程序。你也可以安装图标包来为你的应用程序图标设置主题。

默认情况下，第三方启动程序使用标准的应用程序图标，但它们不一定要这样做。您可以安装第三方启动器将用来代替标准应用程序图标的图标包。

如何使用图标包

相关：什么是自定义Android启动器，以及为什么您可能想要使用一个

要使用图标包，您需要使用支持图标包的第三方启动器，如Nova、Apex、ADW、Go Launcher、Holo Launcher或Action Launcher Pro。

一旦您使用了第三方启动器，您就可以安装图标包并进入您的启动器的设置。您会发现一个选项，允许您在已安装的图标包之间进行选择。这些图标包中的许多还包括墙纸，您可以按正常方式进行设置。

MIUI 5图标

这个图标包提供了1900多个免费图标，这些图标与中国小米科技开发的MIUI ROM使用的图标相似。大的图标列表是一个很大的优势--这个包将给你的大多数应用程序图标一个非常圆滑、一致的外观。

DCikonZ主题

DCikonZ是一个免费的图标主题，包含多达4000多个图标，外观一致。这个图标主题之所以脱颖而出，不仅仅是因为它很大，而且还因为它朝着自己的方向前进，避免了许多图标包使用的超简单、扁平的外观。

全息图标

Holo Icons用简单、外观一致的图标取代了许多应用程序图标，这些图标与谷歌的Holo风格相匹配。如果你是Android的全息外观的粉丝，那就试一试吧。它甚至调整了谷歌自己应用程序中的许多图标，使它们看起来更一致。

方形图标包

正方形图标包把你的图标变成简单的正方形。甚至连谷歌Chrome也变成了球形，而不是正方形。这使得每个图标的大小一致，并提供独特的外观。这里的图标看起来几乎有点像Windows Phone和Windows8.1上提供的小尺寸磁贴。

免费版没有付费版提供的图标那么多，但它确实为许多流行的应用程序提供了图标。

完整协调的 / 圆形的 / 全面的 / 丰满的

想要圆形图标吗？试试圆形图标主题，它提供简单的圆形图标。开发人员说，他们的灵感来自Mozilla的Firefox操作系统上使用的始终如一的圆形图标。

崩溃的图标包

Crumbled Icon Pack应用了一种效果，使图标看起来就像在崩溃一样。Crumred Icon Pack不是为单个图标设置主题，而是为设备上的每个应用程序图标添加效果。这意味着你所有的应用程序图标都将是有主题的，并且是一致的。

精致的图标包

你的Android主屏幕是不是太五颜六色了？Dainty Icon Pack为1200多个应用程序提供简单的灰白色图标。在一个简单的背景下这将是最理想的。这里的灰白对比度可能有点低，但其他方面都很光滑。

单工图标

单面图标提供了更多的对比度，带有灰色的黑底图标。这个图标包可以简化繁忙的主屏幕，允许照片壁纸通过。

最小图标集

MIN尝试尽可能地最小化，为570多个应用程序提供简单的白色图标。这将是一个理想的简单的墙纸与应用程序名称隐藏在您的启动器，提供一个平静的，最小的主屏幕。对于它无法识别的应用程序，它会将应用程序的部分图标圈在一个白色圆圈中。

优雅 / 雅致 / 简明

优雅则完全朝另一个方向发展，提供融合了更多细节和渐变的图标，而不是追求极简主义。它的1200多个图标为那些不喜欢最简、扁平外观的人提供了另一个很好的选择。

图标包设计者通常必须创建并包含他们自己的图标来替换与特定应用程序相关联的图标，所以你可能会发现你的一些应用程序图标并没有被大多数这些主题所取代。当然，没有图标包的标准Android手机也不会有一致的图标。

即使你的应用程序抽屉中的所有图标都没有主题，如果你使用广泛使用的应用程序，你主屏幕上的少数几个应用程序图标也会是主题的。
