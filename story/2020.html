使用Windows Media Player Plus向WMP添加新功能

您使用Windows Media Player 11还是Windows Media Player 12作为默认媒体播放器？今天，我们将向您展示如何使用Windows Media Player Plus第三方插件添加一些方便的新功能和增强功能。

安装和设置

下载并安装Media Player Plus！(链接如下)。在开始之前，您需要关闭Windows Media Player，否则您将收到下面的消息。

下次打开Media Player时，您将看到Media Player Plus设置窗口。默认情况下，某些设置将处于启用状态，例如键入时查找功能。

使用Media Player Plus！

键入时查找允许您从Media Player中的任何位置开始键入搜索词，而不必在搜索框中。搜索词将自动填写搜索框并显示结果。

您还将在库窗格中看到禁用组标题。

此设置将在连续列表中显示库项目，类似于Windows Media Player 10的功能。

在用户界面下，您可以在标题栏中显示当前播放的艺术家和标题。

这在默认情况下是启用的。

上下文菜单页面允许您启用上下文菜单增强功能。

文件菜单增强功能允许您将Windows上下文菜单添加到媒体库窗格和/或列表窗格上的Media Player。右键单击一个标题，选择File，您将看到Windows上下文菜单。

右键单击标题并选择Tag Editor Plus。

Tag Editor Plus允许您快速编辑媒体标签。

“高级”选项卡显示媒体播放器通常不会显示的许多标签。只有带有记事本和铅笔图标的标记才是可编辑的。

还原插件页面允许您配置在Media Player崩溃后应自动还原哪些插件。

启动时恢复媒体页面允许您配置Media Player继续播放最后一个播放列表、曲目，甚至在应用程序关闭时它是正在播放还是暂停。因此，如果您在歌曲中途关闭，则下次打开Media Player时，它将从该点开始播放。

您还可以将Media Player设置为从您停止的位置倒带一定的秒数。如果您正在看电影，这一点尤其有用。

还可以选择将您当前播放的歌曲发送到Windows Live Messenger。

通过转到“工具”、“插件属性”并选择“Windows Media Player Plus”，您可以随时访问这些设置。

Windows Media Plus是WMP11和WMP12的一个不错的免费插件，它为Windows Media Player带来了很多额外的功能。如果你在Windows7中使用Media Player 11或WMP 12作为你的主播放器，你可能想试一试。
