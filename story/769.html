如何在Windows中创建快捷方式图标以创建系统还原点

Windows的系统还原功能没有像以前那样获得好评，尽管它仍然是一个令人难以置信的有用功能。从我们自己论坛上的反馈来看，它几乎每天都能让人们免于一定的破坏。唯一的问题是手动创建新的还原点需要太多的步骤。难道我们不能为它做一个快捷方式图标吗？事实证明，是的，有几种方法可以做到这一点。

Windows 7用户：创建VBScript快捷方式图标，该图标创建带说明的还原点

在Windows7中使其成为创建还原点的快捷方式图标的最简单方法是将该快捷方式指向一个小小的自定义VBScript。遗憾的是，在Windows 8和10中对系统还原调用方式的更改会阻止此脚本在这两个版本中工作。

您可以自己创建脚本。首先，我们复制以下文本：

If WScript.Arguments.Count = 0 Then     Set objShell = CreateObject("Shell.Application")     objShell.ShellExecute "wscript.exe", WScript.ScriptFullName & " Run", , "runas", 1 Else     GetObject("winmgmts:\\.\root\default:Systemrestore").CreateRestorePoint "description", 0, 100 End If

接下来，打开记事本。将复制的脚本粘贴到空白记事本文档中，然后使用所需名称保存脚本。只需确保使用文件扩展名.vbs而不是记事本的默认.txt，这样Windows就会知道您保存的不是纯文本文件，而是VBScript文件。如果您有兴趣了解更多信息，您可以使用VBScript做一些非常酷的事情。

如果您不想创建自己的脚本文件，也可以下载我们为您创建的脚本文件。下载实际上包含两个不同的脚本。“CreateRestorePoint.vbs”脚本将提示您键入还原点的描述，就像上面的脚本一样-这在还原时非常有用。但是，还有一个“CreateRestorePointSilent.vbs”脚本，它将在没有提示的情况下简单地创建还原点。

CreateRestorePoint脚本

无论您是自己创建文件还是下载文件，当Windows询问您是否可以对您的PC进行更改时，您只需双击该文件并回答“是”即可。在“脚本”窗口中，只需键入创建还原点的原因说明，然后单击“确定”即可。Windows将创建还原点，而不会进一步提示您。

如果要验证还原点是否已创建，可以通过在开始菜单中搜索“创建还原点”来打开系统还原。在打开的系统属性窗口中，单击“System Restore”(系统还原)并逐步执行向导以查看可用的还原点。请注意，有时在运行脚本后需要几分钟时间才能创建还原点并显示为可用选项。

一旦验证了脚本文件是否正常工作，您可能希望将其保存在某个位置，并创建一个可以自定义的快捷方式。只需右键单击并将文件拖动到您想要的快捷方式的位置，然后在出现提示时选择“在此处创建快捷方式”。然后，您可以更改快捷键的任何属性，包括指定用于启动快捷键的键盘快捷键。

Windows 7、8和10用户：创建可立即创建还原点的常规快捷方式图标

如果您使用的是Windows 8或10，或者如果您使用的是Windows 7，并且不喜欢使用VBScript，则还可以使用System Restore的命令行和参数创建创建脚本的常规快捷方式。这样做的唯一真正缺点是您不能为还原点添加描述。

如果您使用的是Windows7，则此快捷方式的工作方式与您预期的一样，每次使用时都会创建一个新的还原点。

如果您使用的是Windows8或Windows10，对系统还原的更改意味着该快捷方式带有一个警告。在Windows 8和10中，如果在过去24小时内创建了自动还原点，则系统还原不会使用此快捷方式中的命令创建新的还原点。如果您的任何硬盘存在不到24小时的还原点，则需要先删除这些还原点，然后才能创建新的还原点。您可以打开“系统属性”对话框的“系统保护”选项卡，单击“配置”按钮，然后在弹出的窗口中单击“删除”按钮。

当然，如果您已经打开了“系统还原”选项卡，则用于创建新手动还原点的按钮就在那里，所以您也可以直接使用该按钮。如果没有低于24小时的还原点，则快捷方式将正常工作。因此，您可以在更改设置之前始终使用它，并且至少知道您将拥有一个不超过一天的还原点。

要创建快捷方式，请右键单击要放置该快捷方式的任何位置，然后在上下文菜单中选择新建>快捷方式。

在“创建快捷方式”窗口中，在“键入此快捷方式的名称”框中复制并粘贴以下命令，然后单击“下一步”。

cmd.exe /k "wmic.exe /Namespace:\\root\default Path SystemRestore Call CreateRestorePoint "My Shortcut Restore Point", 100, 7"

您粘贴的命令将启动Windows Management Instrumentation Command(WMIC)工具，并告诉它创建还原点。在下一个屏幕上，键入新快捷方式的名称，然后单击“完成”。

新的快捷键将以命令提示符图标为特色。右键单击它，然后选择属性。

在“属性”窗口的“快捷方式”选项卡上，单击“高级”。

在“高级属性”窗口中，选择“以管理员身份运行”选项，然后单击“确定”。这将使您不必在每次使用快捷方式时都以管理员身份运行该快捷方式。

更改有关快捷键的任何其他内容(如指定快捷键或不同的图标)，然后单击“确定”关闭快捷键的属性窗口。现在，您只需在想要创建还原点时双击新快捷方式即可。

虽然这两种创建快捷方式的技术都需要一些设置时间，但是拥有该快捷方式是有用的，并且可以节省您的时间，并且在将来会让您头疼不已。您可以立即创建还原点，而不是搜索系统还原工具并费力地通过几个屏幕和六次单击。既然您可能应该在已有的基础上更频繁地创建它们，为什么不让它变得简单一些呢？
