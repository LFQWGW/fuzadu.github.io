如何从命令行测试网速

想要快速检查当前Internet连接的速度吗？使用speedtest-cli，您可以在任何操作系统上在命令提示符下运行测试。

我们已经向您展示了如何使用Web浏览器测试您的互联网连接速度，但有几个原因您可能需要改用命令行工具。可能您通过ssh远程连接到服务器，并希望在该机器上测试速度。也许您已经习惯了使用命令来排除网络故障，并且想要在工具包中添加一个速度测试。或者您可能只是认为命令行很酷。没有任何评判。

无论您的原因是什么，运行速度测试都很简单，只需输入speedtest命令，然后在安装了speedtest-cli后按Enter键即可。以下是如何在所有三大操作系统上安装它。

在Ubuntu(和其他Linux发行版)上安装speedtest-cli

在Ubuntu(和其他基于Debian的发行版)上安装speedtest-cli再简单不过了。只需运行一条命令：

sudo apt install speedtest-cli

该命令在其他Linux发行版上可能会有所不同，因此请查看发行版的包管理器，并查找名为speedtest-cli的包。

使用Homebrew在MacOS上安装speedtest-cli

相关：如何使用Homebrew for OS X安装软件包

在Mac上安装speedtest-cli最简单的方法是使用Homebrew，这是Mac的开源软件包管理器。如果您没有安装HomeBREW，请查看我们的安装指南(如果您经常使用命令行，则绝对应该安装HomeBREW)。

设置HomeBREW后，您可以使用一个简单的命令安装speedtest-cli：

brew install speedtest_cli

请注意包名称中的下划线：由于某些原因，Homebrew中的名称与Linux包管理器中的名称不同。

在Windows上安装speedtest-cli

相关：如何在Windows上安装Python

这稍微复杂一些，但完全可行。首先，你必须在Windows上安装Python，这基本上意味着下载正确的安装程序，并确保在系统路径中都添加了python和npip文件。

完成此操作后，只需运行一条命令即可安装speedtest-cli：

pip install speedtest-cli

您现在可以在Windows上使用speedtest-cli。请注意，您可以使用pip在Linux和MacOS系统上安装speedtest-cli，但为了一致性起见，使用系统范围的包管理器会更简单。
