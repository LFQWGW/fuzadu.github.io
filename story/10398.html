如何选择Mac休眠(或“进入待机”)的时间

当你让Mac闲置几分钟，或者当你合上笔记本电脑的盖子时，Mac电脑会自动进入低电量睡眠模式。一些Mac电脑还支持“待机模式”，这与Windows PC上的休眠模式基本相同。以下是如何配置你的Mac电脑何时休眠。

默认情况下，2013年或以后生产的Mac电脑在睡眠三个小时后将进入待机模式。较老的Mac电脑将在大约70分钟的睡眠后进入待机模式。在待机模式下，Mac会将内存中的内容保存到磁盘并关机以节省电量，但当你再次使用它时，它需要更长的时间才能使用。

STANDBY被设计为完全在后台工作，这就是为什么没有图形选项来配置它的原因。与Windows PC不同的是，在Windows PC上，可以在关机菜单中获得“休眠”选项，而在你的苹果菜单中，除了“睡眠”选项之外，没有办法获得一个图形“待机”选项。

Mac只有在特定情况下才会进入待机模式。MacBooks-即便携式Mac而不是台式Mac-只有在电池供电的情况下才会进入待机模式。如果他们插上电源就永远不会进入待机模式。如果MacBooks连接了外部设备，包括以太网、USB、迅雷、SD卡、显示器，甚至无线蓝牙设备，它们也永远不会进入待机模式。

如果台式Mac电脑连接了外部设备，它们将进入待机模式，但如果Mac上安装了USB驱动器或SD卡等外部介质，它们将永远不会进入待机模式。

如何更改待机延迟

如果你想让你的Mac在任何可能的情况下休眠而不是睡眠，你可以将待机延迟设置为较短的时间。如果您想减少休眠频率，可以将待机延迟设置为更长的时间。

待机延迟是一个隐藏选项，在“节能程序配置”窗口中不可用。要检查它，您需要打开一个终端窗口。您可以通过按Command+Space组合键打开Spotlight搜索对话框，键入“Terminal”，然后按Enter键来打开终端窗口。或者，打开查找器窗口，进入应用程序>实用程序>终端。

运行以下命令以检查您的待机延迟时间：

pmset -g | grep standby

如果您在“待机”旁边看到一个“1”，则表明您的Mac支持待机并且已启用。

你在“待机延迟”旁边看到的数字是你的Mac在睡眠后进入待机模式之前等待的时间，以秒为单位。因此，例如，下面屏幕截图中的待机延迟值10800表示10800秒。那是180分钟，或者说3小时。

要更改待机延迟，请运行以下命令，将#替换为所需的待机延迟(以秒为单位)：

sudo pmset -a standbydelay #

例如，如果您想让您的Mac在进入休眠模式后60分钟内进入待机状态，请运行以下命令：

sudo pmset -a standbydelay 3600

在密码提示符下输入您的密码。您的更改将立即生效。

要让你的Mac进入待机模式，你可以合上盖子，按下电源按钮，或者在苹果菜单中选择“睡眠”选项。

要准确控制Mac在进入休眠模式之前等待的时间，请转到系统偏好设置>节能程序，然后调整“关闭显示时间”滑块。这一设置可以控制你的Mac何时进入睡眠模式，并在此之后一定时间进入待机(或休眠)模式。

如何完全禁用待机模式

如果您愿意，您可以完全禁用待机模式。这不一定是个好主意，因为这意味着当你让便携式MacBook进入睡眠模式时，它的电池会逐渐耗尽，而不是进入非常低的电量模式待机模式。但这是你的选择。

如果您想要完全禁用待机模式，请通过在终端窗口中运行以下命令将“Standby”值设置为“0”：

sudo pmset -a standby 0

要稍后重新启用待机，请运行以下命令：

sudo pmset -a standby 1

如果您的Mac在您指定的确切延迟后没有进入待机模式，请务必检查是否有什么东西完全阻止它进入待机模式。例如，只要让SD卡一直插入和挂载，就会阻止任何Mac进入休眠状态。
