如何在平板电脑和智能手机上使用Microsoft Office

微软因为没有为iPad、其他平板电脑和智能手机提供微软Office而被称为晚宴。事实是，微软确实为移动设备制造了相当多不同版本的Office，尽管它们不是Office的完整版。

对于不同版本的Windows，有几个不同版本的Microsoft Office，然后您可以使用Office Mobile、Office 365和Office Online。每一种都有不同的特点，甚至不同的支付结构。

这里我们将从微软自己的平板电脑和智能手机平台开始，因为它们拥有功能最齐全的Office版本。作为一家“设备和服务”公司，微软显然是在试图利用他们的Microsoft Office服务来推动他们的设备。

Windows 8和Windows RT-主要是完整版Microsoft Office

在Windows 8平板电脑上使用Office非常简单。你可以购买完整版的Microsoft Office，并在你的Windows 8平板电脑上使用，就像在Windows 8笔记本电脑上使用一样。Office仅在桌面模式下运行，因此您必须使用平板电脑的桌面才能利用它。它不是触摸屏的理想界面，但它是你在台式PC上使用的Microsoft Office的完整版。

大多数Windows8平板电脑实际上并没有安装Microsoft Office，所以你必须单独付费。一些Windows8平板电脑DO-8英寸平板电脑将包括Office，而一些10英寸平板电脑，如华硕T101，出于某种原因也包括Office。其他平板电脑，比如微软自己的Surface Pro，都没有预装Office。

Windows RT平板电脑确实包含免费版本的Office，但它有点有限。例如，根本不支持宏。也就是说，Windows RT设备附带的Office版本与完整版Office for Windows8几乎相同，而且是免费提供的。

Windows Phone-Office Mobile

Windows Phone自带免费版本的Office，名为Office Mobile。与iPhone和Android手机上的Office Mobile版本不同，Windows Phone上的Office Mobile完全免费使用，不需要Office 365订阅。

当然，Office Mobile是针对移动电话的Office的简化版本。它的设计目的是让您可以随时随地查看Office文档和执行一些基本编辑，而不是作为一个完整的工作效率解决方案发挥作用。

iPhone和Android-Office 365

相关：Office 365和Office 2016有什么不同？

微软现在为iPhone和Android提供“Office Mobile for Office 365 Subscribers”应用程序，你可以从苹果的App Store或Google Play安装这些应用程序。Office 365是Microsoft针对Microsoft Office的订阅服务。你每年支付100美元，就可以在你的Windows和Mac PC上下载最新的Office桌面版本。您还可以获得额外的OneDrive(以前称为SkyDrive)存储空间，并能够使用适用于iPhone和Android的Office 365应用程序。通过这些应用程序，您可以创建和编辑存储在OneDrive存储中的文档

你确实需要订阅才能使用这些应用程序，所以你不能只购买一次应用程序就永远使用它们。它们也远不是微软Office的完整版--它们是简化的智能手机应用程序，而不是严肃的生产力解决方案。

iPad和Android平板电脑-Office 365或Office Online

Office Mobile for Office365应用程序显然是为智能手机设计的。这些相当简单的应用程序并不是您在台式计算机上熟悉的功能丰富的Microsoft Office版本。没有办法在iPad或Android平板电脑上原生使用功能齐全的Office版本，您可以尝试在iPad和Android平板电脑上使用Office Mobile for Office 365订户应用程序，然后您将能够使用简单的移动界面。但它肯定不能替代Windows或MacOSX上的Office。

相关：不再收取升级费用：使用Google Docs或Office Web Apps代替Microsoft Office

过去，微软还建议在平板电脑的网络浏览器中使用Office online，以使用更像桌面的界面。Office Online(前身为Office Web Apps)是OneDrive(前身为SkyDrive)中提供的基于浏览器的Microsoft Office版本。它没有完整桌面版的Office功能强大，但它有类似的界面，而且它的界面可能更适合平板电脑。

更好的是，Office online在任何平台上都是完全免费的，所以你可以不花一分钱-即使是在你的Windows PC上-只要你可以在Web浏览器中使用简化版本的Office即可。

现实地说，大多数寻找可靠工作效率解决方案的平板电脑用户可能会更好地使用其他办公套件，比如苹果的iWork for iPad，谷歌的QuickOffice，或者其他本地支持平板电脑的软件。

远程桌面解决方案

相关：如何从手机远程访问您的计算机

您还可以研究远程桌面解决方案。您可以在Windows PC或Mac上安装Microsoft Office，然后使用远程桌面软件从平板电脑或智能手机访问桌面。想要在iPad上提供Office的公司可以考虑设置一个员工可以连接的远程桌面服务器。它没有可靠的本地办公应用程序好，但它是在iPad、Android平板电脑或任何其他设备上获得Microsoft Office完整版的一种方式。

说到其他笔记本电脑平台，微软还提供了Microsoft Office for Mac OS X的完整桌面版，它是Office 365订阅服务的一部分，你也可以购买盒装版本。在Chromebook、Linux电脑上-是的，甚至是Windows电脑和Mac电脑-你可以免费使用Office Online在你的网络浏览器中使用Microsoft Office。
