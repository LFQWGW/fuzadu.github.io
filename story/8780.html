屏幕截图浏览：Windows10附带的29个新的通用应用程序

Windows10不仅仅是一个改进的桌面环境。它包括许多“通用应用程序”，它们经常取代现有的桌面应用程序。与Windows8不同的是，这些应用可以在桌面的窗口中运行，所以你可能真的想要使用它们。

这些应用程序比Windows 8附带的应用程序成熟得多。Windows 8附带的一些应用程序-如Reader PDF阅读器和Windows Scan扫描仪应用程序-默认情况下不再安装，但可以从Windows应用商店安装

3D生成器

相关：3D打印机什么时候值得购买供家庭使用？

这款应用程序专为创建、导入、建模、保存甚至打印3D模型而设计。微软在Windows8.1中添加了3D打印支持，并提供了新的3D打印机驱动程序模型。虽然3D打印机还没有普及，但这款应用程序仍然可以用来处理和打印3D模型。

闹钟和闹钟

如果你曾经使用过智能手机，那么闹钟应用程序应该会立刻让你耳熟能详。它允许您设置闹钟，使用世界时钟，并集成了计时器和秒表功能。不用说，这款通用的应用程序在智能手机或更小的平板电脑上可能最有用，但它仍然可以在桌面上使用并包含在桌面上。

计算器 / 计算者

Windows10还包括一个通用计算器应用程序，以取代旧的桌面计算器。它包括标准、科学和程序员布局，以及各种单位转换功能。现在它可以在桌面上较小的窗口中运行，作为传统Windows计算器的替代品实际上很有用。

日历 / 历法 / 日程表 / 名单

新的日历应用程序在Windows8.1的基础上有了很大改进。除了Outlook.com日历和Exchange日历之外，你还可以将谷歌和苹果iCloud日历连接到日历应用程序上。有日视图、工作日视图、周视图、月视图和今天视图可用于查看您的事件。它还可以与实时磁贴一起使用，让您可以快速查看即将到来的活动。

照相机 / 摄影机 / 厅 / 圆形建筑

相机应用程序使用设备的摄像头-后置摄像头或前置网络摄像头-来录制视频和照片。它就像智能手机或平板电脑上的相机应用程序一样，这使得它在台式机上并不是最舒适的。然而，你可以用它在任何装有内置摄像头的Windows笔记本电脑或平板电脑上拍照和录制视频，这在过去需要第三方软件。

接触支架

这是作为一个现代应用程序实现的，但它只是一个支持界面。除了信息和指向各种支持页面的链接外，此应用程序还包括与Microsoft支持代表进行实时文本聊天的功能。

科塔纳

相关：为什么我对Windows10中的Cortana感到兴奋

Cortana是微软的个人助理技术，它是作为一个通用的应用程序来实现的，所以它可以很容易地从Windows应用商店自动升级，而不需要更新操作系统。当然，您通常只需使用任务栏快捷方式或“嘿Cortana”语音命令启动Cortana即可。

获取Office

相关：为什么Windows 10提供两个不同版本的Microsoft Office

微软在大多数Windows10电脑上都没有安装Office。取而代之的是，在开始菜单中有一个“获取Office”应用程序，它可以打开一个快速链接，指向微软的Office订阅服务Office365的相关信息。订阅此服务(或使用免费试用版)，即可获得适用于Windows的Microsoft Office桌面应用程序。

还有一些“通用”的Office应用程序，它们在台式PC上运行得很好。然而，微软现在将它们贴上了“移动”应用的标签，它们只能在10.1英寸或更小的设备上免费使用。微软正在推动大多数Windows用户使用标准的Office桌面应用程序，这些应用程序需要订阅或至少一次性购买。

获取Skype

Windows8.1包括一款Skype现代应用程序，但微软在Windows10发布前一个月无礼地将其取消。取而代之的是，微软包括了一个“获取Skype”应用程序，它提供了一种快速下载旧Skype应用程序的方式，该应用程序曾被命名为“Skype for Windows桌面”，以区别于全屏应用程序。

微软显然将在Windows10的未来更新中添加基于Skype的内置即时通讯应用，但这些应用目前还没有推出。

开始

入门应用程序包括视频、教程和其他各种有用的提示和技巧，帮助您熟悉Windows10及其附带的应用程序。微软从Windows 8中吸取了教训，Windows 8没有包括任何形式的帮助和提示，让新用户快速上手，而且入门应用程序看起来相当全面，也很有帮助。

Groove音乐

Groove Music的前身是Xbox Music，它本身的前身是Zune Music。它仍然需要“Xbox配置文件”才能运行。这是微软的音乐应用程序和服务，旨在与Spotify、Apple Music、Google Play Music、Rdio和类似服务竞争。

此应用程序允许您播放位于计算机上的音乐以及存储在OneDrive中的音乐文件。它还允许你购买“Groove Music Pass”，它的功能类似于Apple Music或Spotify订阅，让你可以每月付费访问数百万首歌曲。

信件 / 邮包 / 邮政 / <计算机>电子邮件

邮件应用程序也有了很大改进。它链接到你在电子日历应用程序中使用的同一个账户，这意味着你可以在这里添加Outlook.com和Echange账户，以及谷歌和苹果iCloud电子邮件账户。它还支持Yahoo！邮件，任何IMAP账户，甚至POP账户--这是Windows8上的Mail应用程序所不能做到的。滑动、拖放和其他各种功能的工作方式完全符合你的预期。这是一种轻量级、简单的收发电子邮件的方式，并且提供了活动磁贴。

地图

相关：如何在Windows 10的地图应用程序中获取脱机地图

基于必应地图，内含的地图应用程序是微软与谷歌地图和苹果地图等应用程序的主要竞争对手。它包括方向、收藏夹、附近位置的目录、3D城市视图和其他典型功能。

在桌面地图应用程序中不同寻常的是，它还支持离线地图，即使在没有互联网连接的情况下，也可以让你搜索地点、获取方向，并查看附近的地图。您可以通过打开主设置应用程序并使用系统>脱机地图选项来下载脱机地图。

微软边缘

相关：Windows 10上Microsoft Edge的11个提示和诀窍

Microsoft Edge是Windows 10的新网络浏览器，旨在取代Internet Explorer，尽管Internet Explorer仍未作为选项包括在内。关于Microsoft Edge的文章已经很多了，而且它似乎是基于快速的Web渲染引擎和一个漂亮、简单的界面。然而，它也可以在烤箱中使用更多的时间。例如，微软计划在未来的更新中增加对浏览器扩展的支持。

微软纸牌收藏

在Windows8中，微软移除了纸牌和其他传统的Windows游戏，如扫雷游戏，但在商店里提供了微软纸牌收藏应用程序。在Windows 10中，现在默认安装纸牌收藏。

这只是Windows Store游戏新时代的一个样本，它集成了Xbox，广告，每月1.5美元或每年10美元的费用，可以玩纸牌游戏，而不需要每隔几个游戏就看一次全屏视频广告。微软因在传统上内置在Windows中的免费游戏中免费玩的风格而面临批评，不收取额外费用。

如果你喜欢这种风格的游戏，你可以从商店下载Microsoft扫雷游戏，这是一款类似于传统扫雷游戏的重新想象游戏，现在包括“冒险”模式。这两款游戏都提供日常挑战和各种其他功能。你也可以在默认情况下安装“糖果粉碎传奇”，这是另一款“免费玩”的游戏，它会让你赚得盆满钵满。

货币 / 钱 / 财产 / 财富

Money应用程序实际上是MSN Money，提供来自MSN的财经新闻。除了财经新闻，你还可以跟踪股票、货币和市场波动，并使用抵押贷款计算器。如果你愿意，Live Tiles可以让你在开始菜单上查看财经更新。

影视

Groove Music曾被称为Xbox Music，影视也曾被称为Xbox Video。

如果说Groove Music是为了与Apple Music竞争，那么电影和电视就是为了与iTunes Store的电影和电视节目购买部分竞争。你可以从Windows应用商店购买或租赁单个电影和电视节目，并在此应用程序中观看。然而，没有订阅服务可以观看Netflix或Hulu风格的自助式视频。

您还可以添加自己下载的视频并在此应用程序中观看，该应用程序可以作为基本的视频播放器使用。Windows 10中仍然包含Windows Media Player。

新闻 / 新闻广播 / 新闻人物 / 新闻事件

新闻应用程序基本上是微软的MSN新闻网站的APP形式。除了显示像谷歌新闻这样的热门新闻外，你还可以定制自己的兴趣爱好和查看当地新闻。Windows8上的新闻应用程序允许你在应用程序中添加任何要阅读的RSS订阅源，但这一功能似乎不再可用-除非它被埋在应用程序中的其他地方。

奥尼诺特

相关：OneNote现在是免费的：微软的笔记应用值得使用吗？

OneNote是Windows8上为数不多的“Metro应用程序”中闪闪发光的明珠之一，它表明，实际上可以使用微软的新应用程序框架创建功能强大、有用的应用程序。

OneNote最初只是Microsoft Office的一部分，是微软的笔记应用程序。虽然您也可以使用OneNote的桌面应用程序版本，但包含的应用程序非常方便、易于使用且完全免费。它对手写笔有很好的支持。但缺点之一是，如果你不需要它所有的高级功能，使用起来可能会更复杂。

居住 / 使充满 / 使住满人 / 居住于

人物应用程序使用的帐户与您在日历和邮件应用程序中配置的帐户相同，因此它可以方便地访问您的联系人目录-无论这些联系人存储在Outlook.com、Exchange、iCloud还是Gmail中。

电话伴侣

Phone Companion应用程序将引导您配置您的智能手机或平板电脑-Windows、Android、iPhone或iPad-以在Windows 10上运行。它实际上是为了鼓励您安装微软的各种应用程序，以便您的手机也可以使用您PC上可能使用的服务。例如，告诉应用程序您有Android手机或iPhone，建议您安装OneDrive、Skype、OneNote、Outlook和Microsoft Office for Android应用程序。

这表明了微软的新战略。它希望它的服务可以在你的所有设备上随时随地使用，即使它们运行的是Android或iOS。

照片 / <非正式>摄影定名次

相关：控制智能手机的自动照片上传

照片应用提供了一个图片库，其中显示位于您的本地计算机上的图像以及来自OneDrive的照片。适用于Windows Phone、Android和iPhone的OneDrive应用程序可以自动将你从手机拍摄的照片上传到微软的云存储服务，这样就可以将我们的所有照片集中在一个地方。

与Windows8首发的Photos应用不同，这款应用不能访问Facebook、Flickr和Google Photos等其他服务。它仅限于你电脑上的OneDrive和照片。

这也是您在Windows 10上打开图像文件时出现的默认图像查看器。

设置

相关：如何在Windows 10或Windows 8.x中访问旧控制面板

设置应用程序是作为通用应用程序实现的，它包含大量设置。大多数Windows10用户可能大部分时间都可以使用此界面进行基本设置。

它仍然不能完全取代仍然存在的控制面板。Microsoft可能会继续向设置应用程序添加设置，直到不再需要控制面板。

体育

体育应用程序类似于Money和News应用程序。它是MSN Sports的一个界面，显示与体育相关的新闻文章，并提供其他功能，如比赛比分。就像金钱和新闻一样，体育可以在活动的瓷砖上显示新闻。

储备量 / 储备物 / 丰富 / 仓库

商店应用程序是您获得新的通用应用程序的地方。这是iPad上的Windows10版本的App Store或Android设备上的Google Play。它会自动检查、下载和安装包含的通用应用程序以及您从应用商店安装的任何应用程序的更新版本。

语音记录器

另一款在智能手机上更舒适的应用程序是语音记录器，当你打开它时，它只会显示一个带有麦克风按钮的白色屏幕。单击或轻触麦克风图标以开始录制，然后在完成后单击或轻触停止按钮。使用此应用程序可以使用PC的麦克风快速录制您的语音和其他声音。

天气 / 气候 / 上风面 / 气象

这是Windows8中最具吸引力的应用之一，目前仍在Windows10中。它提供你选择的地点的天气预报，以及历史天气数据和来自MSN的天气相关新闻。添加多个地点，并在方便的界面中查看当地天气。天气应用程序提供了一个特别方便的活动磁贴，它将在您的开始菜单中显示当地天气。

Windows反馈

使用此应用程序可以向微软的Windows团队提供反馈。您可以报告错误、请求特性，并对其他人报告的问题进行投票。从理论上讲，微软的Windows团队可以查看反馈，看看最常报告的错误是什么，以及最想要的功能是什么。

Xbox

相关：如何使用Windows10的游戏DVR和游戏条录制PC游戏

Xbox应用程序包括相当多的功能。它不仅为显而易见的活动馈送提供了关于你的Xbox Live好友及其成就的信息，而且它还集成了Xbox聊天功能。Windows商店中的许多游戏-甚至包括微软纸牌收藏-都有内置的Xbox成就。

除了这里所有与Xbox相关的功能之外，这里还是你可以控制Windows10内置的Game DVR视频捕获功能的地方。这款应用程序还可以让你在Xbox One上流式播放游戏，并在你的PC上玩这些游戏。

微软也希望在未来允许PC到Xbox-One游戏流媒体。

这些只是发布时包含的应用程序。Microsoft将继续更新它们，新版本将定期从Windows应用商店自动下载到您的Windows 10 PC。
