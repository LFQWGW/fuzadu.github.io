如何使用Homebrew for OS X安装软件包

Homebrew是MacOSX上最流行的软件包管理器。Homebrew Cask扩展了Homebrew，支持快速安装Mac应用程序，如Google Chrome、VLC等。不再拖放应用程序！

这是安装Mac终端实用程序和图形应用程序的一种简单方式。这有点像Windows上的巧克力或OneGet，或者像Linux附带的包管理器，甚至还可以安装Mac App Store中没有的许多有用的应用。

基础

Homebrew是一个软件包管理器，设计用于在MacOSX上安装UNIX工具和其他开源应用程序。它将快速下载并安装它们，并从源代码编译它们。Homebrew Cask扩展了Homebrew，支持安装二进制应用程序-这是一种通常从DMG文件拖到应用程序文件夹中的应用程序。

安装Homebrew和Homebrew Cask

首先，您需要安装Xcode的命令行工具。在现代的MacOSX系统上，您只需在终端窗口中运行以下命令就可以安装它们。如果你愿意，你也可以安装苹果公司的完整Xcode应用程序-但这会占用你Mac上更多的空间，而且没有必要。

xcode-select --install

接下来，安装自制软件。您只需打开终端窗口，复制粘贴以下命令，然后按Enter键即可：

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

这个脚本告诉您它将执行什么操作。按Enter键，然后提供您的密码以安装它。默认情况下，它会安装HomeBREW，这样您就可以使用BREW命令，而无需键入sudo命令并提供密码。

完成后运行以下命令，以确保HomeBREW已安装并且工作正常：

brew doctor

更新：不再需要以下命令。Homebrew Cask现在自动安装为Homebrew本身的一部分。

完成后，运行以下命令安装Homebrew Cask。它还使用Homebrew安装Cask：

brew install caskroom/cask/brew-cask

使用Homebrew Cask安装图形应用程序

现在您可以开始安装您想要的图形应用程序了。这涉及到一些非常简单的命令。要搜索一个，请使用以下命令：

brew cask search name

要安装应用程序，请运行以下命令。Homebrew Cask将自动下载、解压缩应用程序，并将其安装到您的应用程序文件夹中。

brew cask install name

要使用Homebrew Cask卸载应用程序，请运行以下命令：

brew cask uninstall name

使用Homebrew安装开源实用程序

Homebrew命令是底层软件包管理器，用于安装您可能需要的所有UNIX和开源实用程序。这是在MacOSX上安装它们的最简单方式，就像在Linux上一样。就像自制木桶一样，它使用简单的命令。

要搜索实用程序，请执行以下操作：

brew search name

要下载并安装该软件包，请执行以下操作：

brew install name

要稍后从系统中删除该软件包，请执行以下操作：

brew remove name

有关使用这些命令的更多详细信息，请阅读其官方网站上的Homebrew Cask使用指南或Homebrew BREW命令手册。并不是您正在寻找的每一个图形应用程序或Unix实用程序都将可用，但它们中的大多数很可能都将可用。

不幸的是，Homebrew Cask没有图形用户界面。这令人遗憾，因为虽然我们极客喜欢简单的终端实用程序，但许多人可以从MacOSX上轻松安装软件中受益。他们可以避免下载DMG文件和四处点击。而且，因为MacOSX现在是Windows风格的安装程序垃圾软件的发源地，所以Homebrew Cask是一种绕过这一问题的方法。
