复杂度

与Android和iPad相比，Windows 8应用商店应用堆栈情况如何？

所以你想买一台平板电脑--你可以在iPad、Android平板电脑和现在的Windows 8或Windows RT平板电脑之间进行选择。Windows平板电脑通常是最贵的。使用平板电脑时，软件可用性至关重要。

现在Windows应用商店的应用程序已经达到7万个，而且Windows 8发布已经超过6个月了，让我们来看看Windows应用商店，看看它与其他平板电脑平台的竞争到底有多强。

录像 / 录像节目 / 录像带 / 影视图像

视频播放应用程序在Windows应用商店中的表现实际上相当不错。它包含Netflix和Hulu Plus的应用程序-这两个服务是最受欢迎的流媒体视频服务。Windows8附带的视频应用程序允许你从微软的视频市场租赁或购买视频。

Windows平板电脑在这方面做得相当不错，但它们仍然无法与其他平台相提并论。没有亚马逊即时视频应用程序(只在iPad和Kindle Fire上提供)、官方YouTube应用程序或索尼Crackle服务应用程序。如果你订阅了HBO Go、DirecTV或Comcast Xfinity等服务，你将无法使用现代应用程序播放他们的视频。

音乐 / 乐符 / 音乐作品 / 和谐悦耳的声音

内置的Xbox Music应用程序在Windows8中提供免费的音乐流媒体，这很有用。然而，如果你已经投资了另一个音乐服务，你可能会失望。Windows应用商店中不提供Spotify、Rdio、MOG和Rhapsody等流媒体音乐服务。Pandora只能作为非官方应用程序使用。

你会发现一些流行的服务，如Slacker和TuneIn Radio，所以它不是完全错过--但Windows8的第三方音乐服务选择相当差。

您可以尝试通过Internet Explorer应用程序在浏览器中收听这些服务，但当浏览器进入后台时，音乐将停止播放。如果你想在Windows8或Windows RT平板电脑上收听Spotify，你可以在桌面上的浏览器中收听。

雷丁 （英格兰南部城市）

Windows商店实际上有很多电子书服务可供选择，亚马逊Kindle、Nook和Kobo都有官方应用。如果你想用平板电脑阅读电子书，Windows8可以做到。

如果你想要其他服务，商店可能会有一些限制。在iOS和Android上广受欢迎的Flipboard新闻阅读器还没有应用程序。目前还没有针对Pocket和Instapaper这样的稍后阅读应用服务的官方应用。当然，也有RSS阅读器和稍后阅读应用-它们可能不会与你现有的任何服务集成，也不会像现在这样精益求精。

小游戏

Windows应用商店确实有一些简单的平板电脑游戏可供选择。根据你玩游戏的次数，你可能会对可用的浪费时间的游戏感到满意，其中包括像“愤怒的小鸟”和“割断绳子”这样的流行游戏。

不幸的是，即使是这里的选择也是有限的。虽然iOS和Android已经发布了两部续集“割断绳子”，但只有第一部在Windows商店有售。说到《愤怒的小鸟》，Windows商店里只有《愤怒的小鸟空间》和《愤怒的小鸟星球大战》。然而，它们每个5美元，而在其他平台上是免费的。

如果你想要一台平板电脑来玩游戏，这甚至不是一场比赛-iPad遥遥领先。例如，“童话故事”广受好评的“行尸走肉”游戏适用于Windows桌面、Mac、Xbox、PS3、PlayStation Vita和iPad。它甚至还没有在Android上发布，所以它不太可能以触控优化的形式出现在Windows8上。许多大牌游戏只在iOS上发布。许多其他游戏在Android上发布的时间要晚得多。在触控优化的平板游戏方面，Windows Store远远落后于Android的Google Play。

生产力 / 生产率 / 繁殖力 / 生产能力

Windows RT设备自带Microsoft Office，但它只在桌面上工作-它不是一个触控优化的界面。Windows8设备不附带Office，不过当然可以购买和安装。

您可能会认为，考虑到Windows平台作为生产力平台的历史，Windows应用商店在生产力应用程序方面会很强大。然而，这与事实相差不远。

虽然微软为OneNote提供了一款现代应用程序，但在新的环境中，所有其他办公应用程序都不见了。Windows应用商店也没有多大帮助。搜索“office”会出现Office Depot的应用程序和各种桌面办公应用程序。在Google Play上进行同样的搜索，仅在第一页就会出现七个不同的平板电脑优化的办公效率应用程序。

如果你想使用完全触控优化的办公应用程序，iPad或Android平板电脑会好得多--至少在微软发布其Office应用程序的Windows Store版本之前是这样。

Windows 8和RT体验提供了一些独特的优势，例如可以完全使用Microsoft Office，也可以同时使用多个应用程序。然而，在新的“现代”界面中提供的生产力应用程序寥寥无几。

社交应用

那么，社交网络在Windows8上的表现如何呢？People应用程序提供了与Twitter和Facebook的一些集成，但让我们将重点放在完整的应用程序上。

Twitter在Windows应用商店提供了一款官方应用程序。然而，在那之后，选择就变得越来越少--没有官方的Facebook、LinkedIn、Google+、Pinterest、Instagram或Tumblr应用程序。

Windows应用商店对自己的情况没有帮助，因为对这些应用程序(以及其他常见应用程序)的搜索会带来许多质量不佳的应用程序，而在其他平台上进行的相同搜索会带来更高质量的应用程序。这会给人一种Windows应用商店到处都是垃圾的感觉。

你当然可以在浏览器中使用社交网络的网站，但你会放弃动态磁贴的优势，放弃在浏览时在侧边栏中使用社交网络的能力(因为一次只能运行一个浏览器应用程序)，以及推送通知，所以体验就不一样了。

其他服务

谷歌服务显然在谷歌自己的平台Android上表现良好，但谷歌的应用程序在iPad上也得到了很好的表现。

然而，谷歌和Windows8在一起玩得并不好。有一款官方的谷歌搜索应用程序和一款谷歌Chrome浏览器(只能在Windows8上运行，不能在Windows RT上运行)，但仅此而已。你可以在Windows8附带的基本邮件应用程序中阅读你的Gmail，但你甚至不能再在日历应用程序中查看你的谷歌日历了。谷歌地图、谷歌文档、谷歌硬盘、谷歌日历-如果你选择了Windows平板电脑，这些都是你将在浏览器中使用的服务，而不是专用的应用程序。

许多其他服务也在行动中缺失，包括广受欢迎的铸币厂网站的一款应用程序，它可以帮助你跟踪你的钱。这也不全是坏事--例如，Dropbox提供了一款Windows8应用程序，这样你就可以继续使用你的Dropbox账户，而不会投资于SkyDrive。

触摸式图像编辑器在Windows8上的表现不如Windows8，也远不及iPad甚至Android平板电脑上的应用程序强大。

结论

在7万多个应用程序和6个多月过去了，微软要把Windows8变成一个可信的平板电脑平台，还有很多工作要做。它只是不成熟--许多流行而重要的应用程序是不可用的，而且许多可用的应用程序没有其他平板电脑平台上可用的选项那么成熟。

应用程序的数量并不是最重要的事情-如果它们是人们想要的高质量应用程序，7万就绰绰有余了-而是关于哪些应用程序是可用的。一旦你开始查看Windows应用商店，很明显，将Windows PC用作平板电脑将涉及许多权衡。

当然，你可能会喜欢平板电脑上的Windows，并且可以接受有限的软件。您可能更喜欢Windows平板电脑界面(提供真正的多任务处理)和以前称为Metro的界面设计。目前，平板电脑上的Windows与台式机上的Linux有很多共同之处。无论你是购买Windows平板电脑还是在台式机上安装Linux，你都应该知道你正在放弃大量可用的软件。

有一件事是肯定的-Windows应用商店需要更强大的产品。毫不奇怪，当Windows RT设备和iPad一样贵(或者更贵)，而且几乎没有可用的应用程序时，它们的销量并不是很好。
