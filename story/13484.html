如何将Android文件无线传输到Linux桌面

你有安卓手机和Linux台式机吗？您可以无线传输文件、从PC发送文本以及从计算机控制电话。这就像Windows10的Linux版手机应用程序！

Android与Linux的集成

KDE Connect是一款圆滑且功能丰富的软件，可以将Android手机直接集成到KDE桌面环境中。

它提供了许多功能，如手机和计算机之间的双向通知、双向无线文件传输以及从计算机发送短信文本。KDE连接是KDE桌面环境的组成部分。

GSConnect是为GNOME桌面环境开发的软件版本。它是作为GNOME扩展构建的，GNOME用户必须安装GSConnect。

要解决Android方面的问题，KDE和GNOME用户都必须安装和使用KDE Connect Android应用程序。

你要与Android手机配对的电脑不一定要使用Wi-Fi。可以使用以太网电缆将其插入网络。它必须与您的手机在同一网络上，但这是唯一的要求。

安装步骤

安装GSConnect很简单，但是必须按照正确的顺序执行这些步骤。

在您的Android手机上安装KDE Connect应用程序。 如果您使用的是Google Chrome，请安装Chrome-gnome-shell Linux软件。 配置Chrome或Firefox以与GNOME外壳集成。 使用Chrome或Firefox安装GSConnect GNOME扩展。 将您的Android手机与您的KDE或GNOME桌面环境配对。

KDE用户只需完成第一步和最后一步。使用Firefox作为浏览器的GNOME用户不需要执行第二步。

安装Android App

在Android手机上，打开Play Store并搜索“KDE Connect”。找到应用程序后，单击绿色的“安装”按钮。

安装完成后，您将能够在应用程序启动程序中找到“KDE连接”图标。

如果您使用Google Chrome作为浏览器，则需要安装Chrome-GNOME-Shell集成软件。如果您使用的是Firefox，则不需要执行此操作，因此请跳过此步骤，直接转到标题为“配置Firefox以管理GNOME扩展”的部分。

如果您使用的是Ubuntu或其他基于Debian的发行版，请使用aapt-get命令将此软件包安装到您的系统上。在其他Linux发行版上，请改用您的Linux发行版的包管理工具。

sudo apt-get install chrome-gnome-shell

配置Chrome以管理GNOME扩展

打开Google Chrome并浏览到Chrome网络商店。搜索“GNOME Shell集成”。

当你看到GNOME Shell集成扩展时，点击蓝色的“添加到Chrome”按钮。

系统将要求您确认是否要将该扩展添加到Chrome。单击“添加扩展”按钮。

当扩展添加到Chrome后，您将在Chrome工具栏的右上角看到GNOME Footprint图标。

您现在应该跳到下面的安装GSConnect GNOME桌面扩展部分。

配置Firefox以管理GNOME扩展

打开Firefox，浏览到Firefox Add-ONS网站，搜索“GNOME Shell集成”。

当您看到GNOME Shell集成扩展时，单击蓝色的“add to Firefox”按钮。

系统将要求您确认是否要将该扩展添加到Firefox。单击蓝色的“添加”按钮。

将扩展添加到Firefox后，您将在Firefox工具栏的右上角看到GNOME Footprint图标。

我们现在可以添加GSConnect GNOME桌面扩展。

安装GSConnect GNOME桌面扩展

在Google Chrome或Firefox中，单击工具栏右上角的GNOME Footprint图标。搜索“GSConnect”，并在出现“GSConnect”条目时单击它。

单击ON/OFF(开/关)按钮，显示蓝色的“ON”部分。这将下载、安装并激活GSConnect GNOME扩展。

您现在可以关闭浏览器了。

验证移动设备是否在系统菜单中

打开GNOME系统菜单。您应该会在菜单中看到一个标题为“Mobile Devices”的新条目。如果看不到此新菜单项，请重复上述步骤。

将您的Android手机和您的计算机配对

单击移动设备菜单项。菜单将展开。1单击“移动设置”菜单项。

将出现一个窗口，其中标题为您的计算机的名称。如果您的Android手机上正在运行KDE Connect应用程序，您将看到它列在此窗口中。

在Android手机上，如果KDE Connect应用程序未运行，请启动该应用程序。当该应用程序打开时，您应该会看到您的计算机的名称被列为可用设备。在本例中，它是“Howogleek”。

点击您的计算机的名称。应用程序会告诉你设备没有配对。按蓝色的“请求配对”按钮。

配对请求对话框将出现在您的计算机上。它会告诉您请求连接的Android手机的名称。在本例中，它是荣誉。

单击“Accept”(接受)按钮接受配对请求。

您的KDE Connect应用程序将显示一组现在可用的功能。

您的Android手机将在您的计算机上显示为已连接。

如果你点击Android手机的名称，对话框会向你显示一组选项和设置，你可以使用这些选项和设置来微调电脑和手机之间的互通。

GNOME系统菜单中的Mobile Devices菜单项将被显示已配对的Android手机名称的菜单项所取代。单击该菜单项将显示其中具有新功能的子菜单。

您的两台设备现在已配对。

使用KDE要简单得多

KDE连接是KDE不可或缺的一部分。由于两者之间的顺利集成，与GNOME进程相比，安装过程非常轻松。

在KDE中打开系统菜单。单击“应用程序”图标。

单击“设置”菜单项。

单击“系统设置”菜单项。

将出现系统设置对话框。

向下滚动，直到您可以看到KDE连接菜单项。1单击“KDE连接”菜单项。

在Android手机上，如果KDE Connect应用程序未运行，请启动该应用程序。当应用程序打开时，您应该会看到您的计算机名称被列为可用设备。

点击您的计算机的名称。应用程序会告诉你设备没有配对。按蓝色的“请求配对”按钮。

在您的计算机上接受配对请求。您的Android手机现在将作为配对设备在系统设置窗口中列出。

单击您的电话名称。系统设置对话框将列出与您的配对设备相关的设置和功能。

您的Android手机和您的计算机现在已配对。

完成Android设置

KDE Connect应用程序需要您的许可才能允许其某些插件运行。点击每个插件以确保其具有所需的权限。

例如，轻触“通知同步”插件将把您带到通知访问屏幕。在列表中找到“KDE连接”，然后将切换开关切换到打开位置。

Android会要求您确认是否希望授予KDE Connect权限。点击“允许”按钮。

KDE连接滑块按钮现在将变为蓝色，表示已授予权限。

继续授予您要使用的插件的权限。当解决了每个需要权限的插件时，会将其从列表中删除。

要返回到任何插件的设置，请点击三点菜单图标。

然后点击“插件设置”菜单项。

所有的插件都将被列出，并可以从这里进行管理。

Android存储位置

Filesystem exposed插件可以让您的计算机访问Android手机中的位置。这允许将文件从您的计算机传输到您的手机。“如果您只想将文件从Android手机发送到您的计算机，则无需费心使用此设置。

要设置存储位置，请点击“文件系统公开”项。

在文件系统暴露设置屏幕上，单击蓝色加号(+)。

轻触“添加存储位置”菜单中的“单击以选择”选项。

您的Android手机可能会提供多个存储位置可供选择。用来研究本文的Android手机只提供了一个Downloads文件夹。点击您要使用的位置，然后点击“选择”。

点击显示名称菜单项并提供存储位置的名称。在我们的示例中，这是“下载”。点击“确定”。

下载现在将出现在文件系统暴露设置屏幕上。

将文件传输到您的计算机

点击KDE连接中的“发送文件”选项。

KDE连接将打开默认存储位置，该位置之前设置为下载。我们在这个地方只有一个文件。要将其传输到配对的计算机，请点击该文件。

您将在计算机上看到一个转接通知弹出窗口，通知您转接已成功完成。

传输多个文件

您可以一次传输多个文件，并且可以从手机上的不同位置进行传输。

如果要从默认存储位置以外的位置传输文件，请点击“汉堡包”菜单。

此时会出现一个侧面板，允许您浏览Android上的存储空间。

如果你想发送图片，你可以点击“图片”图标。如果你想传输音频文件，你需要点击“音频”图标。

假设我们已经窃听了图像。您可以通过依次点击每个文件来选择多个文件。选定的文件上会出现蓝色框中的白色勾号。

要传输文件，请点击“打开”一词。

默认情况下，传输的文件到达您的计算机的Downloads目录中。

KDE文件管理器集成-Dolphin

一旦你的Android手机与电脑配对，KDE文件管理器Dolphin就会与KDE Connect应用程序集成。

如果你启动Dolphin，你会看到你的Android手机列在设备下面。

单击您的手机名称以查看在KDE Connect应用程序中设置的默认存储位置。拖放此目录中的文件会将它们传输到您的Android手机。

GNOME文件管理器集成-Nautilus

要用GNOME实现同样的功能，还有更多的工作要做。不过，结果也一样好。

首先，我们需要安装Nautilus和KDE集成提供程序。请关闭任何Nautilus窗口，然后在终端窗口中键入以下命令，然后按Enter键。

sudo apt install python-nautilus gir1.2-nautilus-3.0 sshfs

打开系统菜单，单击Android手机的名称，然后单击“挂载”菜单选项。

打开Nautilus文件管理器窗口。您将看到Android手机的条目。它不会按名称列出-这是KDE用户的额外福利-它将按它在Wi-Fi网络上的IP地址列出。

在本例中，它是192.168.4.24。

通过选择代表Android手机的IP地址，您可以浏览到手机的默认存储位置，并通过将文件放入该目录来传输文件。
