Boxee入门

Boxee是一款免费的Media PC应用程序，可以在Windows、Mac和Ubuntu Linux上运行。使用Boxee，您可以将在线视频、音乐和图片与您自己的本地媒体和社交网络集成在一起。今天我们要仔细看看Boxee和它的一些功能。

注意：本教程使用的是Windows7。您在Mac或Ubuntu Linux版本上的体验可能略有不同。

硬件要求

运行在1.0 GHz或更高频率的基于x86(英特尔/AMD处理器)的系统

512MB系统内存(RAM)或更多

支持OpenGL 1.4、Direct X 9.0的显卡

软件需求

Mac OS X 10.4+(基于英特尔的处理器)

Ubuntu Linux 9.04+

Windows XP/Vista/7(Vista或7为64位)

安装Boxee

在下载和安装Boxee之前，您需要注册一个免费帐户。(请参阅下面的链接)

一旦您的帐户经过注册和验证，您就可以登录并下载应用程序了。安装非常简单，…。只需接受默认值即可。

Boxee将以全屏模式打开，系统会提示您使用用户名和密码登录。在您登录之前，您可能想花点时间单击“指南”图标，了解一下在Boxee中导航的知识。

一些基本的键盘导航如下所示。使用箭头键向右、向左、向上和向下移动。按“Enter”进行选择，按反斜杠“\”在全屏和窗口模式之间切换，按“Esc”返回上一个屏幕。播放时，音量由加号和减号(+/-)键控制，您可以使用空格键播放/暂停，使用箭头键跳过。

Boxee还可以与任何红外遥控器配合使用。如果您有iPhone或iPod Touch，您可以下载软件将其作为Boxee遥控器使用。

如果您使用的是鼠标和键盘，请将鼠标悬停在用户名和密码框上以输入您的登录凭据。如果使用遥控器，请单击确定按钮并使用屏幕键盘输入凭证。完成后，单击“完成”。

当您准备好登录时，请输入您的凭据，然后单击“登录”。

首次登录时，系统将提示您校准屏幕。如果您选择“跳过”，您可以稍后在“设置”>“外观”>“屏幕”下校准屏幕。

当Boxee打开时，您将看到主屏幕。左边是您的提要。这将是来自Boxee上的朋友以及Facebook和Twitter等社交网络的任何推荐内容。不过，当您第一次登录时，它将主要是来自Boxee工作人员的信息。您将在中心有“特色”内容，在右侧有您的队列。您还可以在顶部看到菜单。

弹出式菜单

可以通过按“Esc”键或回到遥控器来访问弹出菜单。根据您在Boxee中的位置，您可能需要点击几次才能“退出”到弹出菜单。从弹出菜单中，您可以轻松访问任何资源、设置和收藏夹。

行列 / <古>辫子 / <计算机>队列 / 长队

队列是您希望观看的电视节目、电影或互联网视频的播放列表。当您找到想要观看的产品时，选择它，然后单击“添加到队列”。

所选项目将添加到您的队列中，并且可以随时从菜单访问。

电视节目图书馆

电视放映库可以包含来自本地硬盘的文件或来自Web的流式内容。Boxee从各种在线位置(如Hulu和电视网站)提取内容。点击节目，看看目前有哪些特定的剧集。

要搜索您最喜欢的节目，请单击左侧的黄色箭头，或者使用键盘或遥控器导航到左侧。在搜索框中输入您的选择。

我的应用程序

默认情况下，“我的应用程序”部分包括最受欢迎的应用程序列表，如Netflix、Pandora、YouTube等。您可以从“我的应用程序”中删除应用程序，也可以从应用程序库中添加新的应用程序。

要访问所有可用的应用程序，请单击左箭头按钮，或单击左侧的黄色箭头，然后选择“App Library”。

从库中选择一个应用程序，然后单击它打开…

…。然后选择“添加到我的应用程序”。或者，如果您不想将该应用程序添加到您的“我的应用程序”中，也可以单击“开始”来播放该应用程序。

音乐、图片和电影

Boxee将扫描您的PC以查找电影、图片和音乐。您可以通过单击“Scan Media Folders…”选择扫描特定文件夹

…。或从弹出菜单中选择设置>媒体，然后浏览您的媒体。

结束 / 结尾 / 结论 / 推论

Boxee是将您的本地媒体与在线流媒体内容相结合的一种很好的方式。它可以作为应用程序在您的家用PC上运行，也可以作为独立的媒体PC运行。不过，还应该注意的是，您对在线内容的访问权限将因您所在的国家而异。
