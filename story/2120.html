在桌面上脱机编辑Office Web Apps文档

是否要在Office 2010或2007中而不是在Office Web Apps中编辑您的联机文档？以下是您如何脱机编辑您的文档，同时仍可利用Web Apps提供的联机协作。

脱机编辑Office Web Apps文档

Office Web Apps是在浏览器中预览和编辑文档的好方法。新的Online Office具有25 GB的在线免费存储空间，是与其他人共享和协作处理您的文档的好方法。但是，即使您不喜欢联机编辑文档，您仍然可以使用Web应用程序。您只需联机查看文档，在应用程序中打开文件菜单，然后选择在Word中打开。

或者，如果您已经在浏览器中编辑文档，然后意识到您需要比Web应用程序提供的更高级的Office功能，则可以单击Web应用程序功能区中的在Word中打开。

单击“确定”确认您信任该文档并要打开它。

然后，相应的桌面Office应用程序(在我们的示例中为Word)将打开并下载新文件。如果您使用的是Office 2010，您会注意到它显示它正在联系服务器获取信息；Office 2007的打开时间似乎比平时长。

如果这是第一次在桌面上的Office中编辑Office Web Apps文档，系统可能会要求您先登录，然后才能看到该文件。请输入您在Office Web Apps中使用的Live ID以继续。

现在，您将在桌面上看到您的文档在Word中打开，或在其他Office应用程序中打开，具体取决于您打开的文档。*默认情况下，受保护的视图将阻止您编辑文档，因为它源自联机位置。/单击“启用编辑”可根据需要编辑您的文档。

现在，您可以在桌面上在Word中完全编辑您的文档，并使用任何可用的功能，即使Web应用程序不支持这些功能。*这在Office 2010和2007中都很有效，这很不错。*您可能会注意到，保存图标现在右上角有箭头。*这表明当您保存文档时，它实际上会保存到您的SkyDrive，您可以在其中联机查看更改和预览文件。*此外，您与之共享文档的任何人现在都将看到更新版本，即使您脱机更新了文档。

注：根据您的Internet连接速度，保存对文档所做的更改可能会比平时花费更长的时间。

如果要保存文件的本地副本，请在Office 2007中打开“文件”菜单或单击ORB，然后选择“另存为”。现在，您可以将文件保存到桌面上的任何位置。

你可能会遇到的问题

在我们的测试中，我们确实发现了一些脱机编辑的问题。首先，Office Web Apps只支持IE、Firefox和Safari。*尽管这些应用程序在Chrome中运行良好，但当我们尝试打开文件进行脱机编辑时，我们被告知不支持我们的浏览器。*如果您使用其他Office套件(如OpenOffice.org)，可能会遇到类似的问题。

某些Web应用程序(如Excel和OneNote)允许您与其他人同时编辑文档。但是，当我们尝试在计算机上的Office应用程序中打开文件时，Web应用程序警告我们不能同时脱机和联机编辑文档。如果您想要协作，则必须在浏览器中工作。

如果您遇到这些问题，请浏览到SkyDrive上的文档，单击“更多”，然后选择“下载”。如果要将您所做的更改保存回Office Online，只需在Office中照常保存文档，然后再次将其上载到SkyDrive即可。

结束 / 结尾 / 结论 / 推论

作为微软的在线协作套件，Office Web Apps是一个很有前途的开端，但与桌面上功能齐全的Office相比，Web应用仍然是缺乏的。尽管如此，由于您仍然可以轻松地脱机编辑文档，然后将更改保存回SkyDrive，因此它们仍然非常有用。我希望我们很快就能解决一些问题，并支持更多浏览器(如Chrome)，但在此之前，无论您使用什么软件，这些信息都应该可以帮助您与Web应用进行协作。
