使用Windows Live Movie Maker将照片和家庭视频转换为电影

您是否在寻找一种简单的方法来拍摄数码照片和视频，并将其转换为电影或幻灯片？今天我们将详细了解如何使用Windows Live Movie Maker。

就任 / 安顿 / 安装 / 就职

Windows Live Movie Maker捆绑在Windows Live软件包套件中(链接如下)。但是，您不必安装任何您可能不需要的程序。

注意“您即将完成”屏幕。在单击“继续”之前，请确保取消选中复选框以设置您的搜索提供商和主页。

添加图片和视频

打开Windows Live影音制作。只需将视频或照片拖放到情节提要区域，即可添加视频或照片。您还可以单击情节提要区域或主页选项卡上的添加视频和照片按钮来浏览视频和照片。Windows Live Movie Maker支持大多数视频、图像和音频文件类型。

选择您的文件并添加，单击“打开”将它们添加到Windows Live Movie Maker。默认情况下，wlmm不允许您从网络位置…添加文件。因此，如果您要添加的文件位于网络驱动器上，请查看我们关于如何向Windows Live Movie Maker添加网络支持的文章。

规划 / 布局 / 设计 / 安排

所有添加的剪辑都将显示在右侧的情节提要区域中，而当前选定的剪辑将显示在左侧的预览窗口中。

您可以通过单击并拖动中间的分隔线来调整这两个区域的大小。

通过单击缩略图大小图标，使情节提要上的剪辑变大或变小。右下角的滑块用于调整缩放时间刻度。

预览您的电影

通过单击空格键或按预览窗口下的播放按钮，您可以随时回放影片并预览其在预览窗口中的外观。

您还可以在情节提要中手动移动预览栏滑块，以便在视频进行时查看剪辑。

调整情节提要上的剪辑

您可以单击并拖动情节提要上的剪辑以更改照片和视频的显示顺序。

添加音乐

没有什么比音乐更能让电影栩栩如生了。选择添加音乐会将您的音乐添加到电影的开头。选择“在当前点添加音乐”以将其包含在影片中的预览栏滑块的当前位置，然后浏览您的音乐剪辑。WLMM支持许多常见的音频文件，如WAV、MP3、M4A、WMA、AIFF和ASF。

音乐剪辑将出现在故事板上的视频/照片剪辑的上方。

您可以通过单击音乐剪辑并将其拖动到情节提要上的其他位置来更改音乐剪辑的位置。

添加标题、标题和制作者

要向电影添加标题屏幕，请单击“开始”选项卡上的“标题”按钮。

在预览屏幕的文本框中直接键入您的书目。标题将放置在情节提要上预览滑块的位置。但是，您可以通过单击标题并将其拖动到情节提要的其他区域来更改位置。

在“格式”选项卡上，有一些文本设置。您可以更改字体、颜色、大小、对齐方式、颜色和透明度。“调整”组允许您更改背景颜色、编辑文本以及设置字幕在影片中显示的时间长度。

“格式”选项卡上的“效果”组允许您选择标题屏幕的效果。通过将光标悬停在每个选项上，您将获得每个效果在预览窗口中的实时预览。单击以应用任何效果。

对于标题，请选择您希望使用情节提要上的预览滑块显示标题的位置，然后单击“开始”选项卡上的“标题”按钮。

就像标题一样，您可以直接在预览屏幕的文本框中键入标题，然后可以使用上面的“字体和段落”、“调整”和“效果”组进行任何调整。

字幕的作用与片头和字幕相同，不同之处在于它们会自动放在电影的末尾。

跃迁

转到功能区上的动画选项卡以应用过渡。从情节提要中选择剪辑，然后将鼠标悬停在其中一个过渡上以在预览窗口中查看它。单击过渡以将其应用于剪辑。可以将变换单独应用于剪辑，或在单击的同时按住Ctrl按钮以选择要对其应用相同变换的多个剪辑。

平移和缩放效果也位于“动画”选项卡上，但只能应用于照片。与变换类似，您可以将它们单独应用于剪辑，或在单击的同时按住Ctrl按钮以选择要应用相同平移和缩放效果的多个剪辑。

应用后，可以调整过渡以及平移和缩放效果的持续时间。

您还可以单击下拉菜单以查看其他过渡或效果。

视觉效果

与平移、缩放和过渡类似，您可以将各种视觉效果应用于单个或多个剪辑。

编辑视频和音乐

注意：这实际上不会编辑您导入到Windows Live Movie Maker项目中的原始视频，而只是编辑它在WLMM项目中的显示方式。

“主页”(Home)选项卡上有一些非常基本的编辑工具。“向左旋转”和“向右旋转”按钮将调整任何可能方向不正确的剪辑。

“适应音乐”按钮将自动调整照片的持续时间(如果您的项目中有)，以适应电影中音乐的长度。

音频混音允许您更改音量级别

您还可以从“编辑”选项卡执行一些稍微高级的编辑。选择情节提要上的视频剪辑，然后单击“裁切”工具以编辑或删除视频剪辑的一部分。

接下来，单击并拖动预览窗口中的滑块以选择要保留的位置。例如，滑块外部的区域是从影片中修剪的区域。里面的区域是电影中保留的部分。也可以在功能区上手动调整起点和终点。

完成后，单击“保存修剪”(Save Trim)。

您还可以拆分视频剪辑。将预览滑块移动到视频剪辑中要拆分的位置，然后选择拆分。

您的视频将被分割成单独的部分。现在，您可以应用不同的效果或将它们移动到情节提要上的不同位置。

编辑音乐剪辑

选择情节提要上的音乐剪辑，然后选择功能区上的选项选项卡。您可以通过左右移动滑块来调整音乐音量。

您还可以选择在电影开始和结束时淡入或淡出音乐剪辑。从淡入和淡出下拉列表中，选择无、慢、中或快。

要调整音频剪辑的声音，请单击编辑选项卡，选择视频音量按钮，然后调整滑块。将其完全向左移动，以使视频剪辑中的任何背景噪音静音。

自动电影

如您所见，Windows Live Movie Maker允许您添加效果、过渡、字幕等。如果你不想自己做这些事情，AutoMovie会自动添加字幕、字幕、项目之间的交叉淡入淡出过渡、照片的平移和缩放效果，以及根据音乐调整你的项目。只需选择“主页”选项卡上的“自动电影”按钮即可。你可以在几分钟内从零开始看电影。

上传到YouTube

您可以直接从Windows Live Movie Maker在YouTube上共享视频。单击“主页”(Home)选项卡上“共享”(Sharing)组中的YouTube图标。

系统将提示您输入YouTube用户名和密码。

填写有关您的电影的详细信息，然后单击“发布”。

这部电影在上传到YouTube之前将被转换为WMV。

YouTube转换完成后，您的新电影即已上线并可供观看。

将电影另存为视频文件

选择左上角的图标，然后选择保存电影。当您将鼠标悬停在每个选项上时，您将看到视频的输出显示大小、纵横比和每分钟估计的文件大小。

所有这些设置都会将您的电影输出为WMV文件。(遗憾的是，唯一的选择是将影片另存为WMV文件。)。唯一的区别是如何根据预设的公共设置对它们进行编码。刻录到DVD选项也会输出WMV文件，但会打开Windows DVD Maker并引导您完成创建和刻录DVD的过程。

如果选择刻录到DVD选项，请在WMV文件转换完成后关闭此窗口，Windows DVD Maker将提示您开始。

当你的电影结束后，就是放松和享受的时候了。

结束 / 结尾 / 结论 / 推论

Windows Live Movie Maker让普通人可以很容易地从那里快速制作出好看的电影和幻灯片，并拥有自己的图片和视频。但是，以前版本(以前称为Windows Movie Maker)的长期用户可能会对Windows Live Movie Maker在早期版本中缺少的一些功能感到失望。

要了解有关将新项目刻录到DVD的详细信息，请查看我们关于如何使用Windows DVD Maker创建和制作DVD的文章。
