如何在Windows 7、8或10中创建系统映像备份

Windows中的内置备份实用程序非常可靠。让我们来看看如何在不需要第三方实用程序的情况下创建PC的完整备份映像。

打开系统备份映像工具。在Windows 10中，进入控制面板>备份和还原(Windows 7)>创建系统映像。 选择要保存备份映像的位置。 选择要备份的驱动器。 开始备份。 (可选)制作一张系统修复光盘，您可以使用该光盘启动计算机并还原备份映像。

正常的备份程序，如CrashPlan或Windows的内置文件历史记录功能，实质上是将文件复制到另一个位置。另一方面，完整的系统映像备份就像是整个硬盘的完整快照。系统映像备份的优势在于，如果硬盘驱动器崩溃，您可以更换它，恢复映像，然后将系统恢复到捕获映像时的位置。无需重新安装Windows或您的应用程序。

系统映像备份的最大缺点-除了耗时稍长-是您无法将备份恢复到另一台PC。您正在创建完整Windows安装的映像，由于Windows是专门为您的硬件设置的，因此它不能在另一台PC上按原样工作。这就好比试图将您的硬盘插入另一台PC，然后期望一切都能正常加载。不过，考虑到这一点，映像备份仍然非常方便。

像Macrium Reflect或Acronis True Image这样的第三方应用-至少是付费版本-确实提供了一些你在Windows系统映像备份工具中找不到的高级功能。例如，两者都支持增量备份、受密码保护的映像以及浏览单个文件的备份的功能。但免费是免费的，如果你不需要额外的功能，Windows工具提供了一种可靠的方式来执行系统的完全备份。

第一步：打开系统映像备份

在Windows 7中查找系统映像备份工具的过程与在Windows 8和10中不同，因此我们将向您展示如何在所有版本中查找该工具，然后解释如何创建和使用系统映像。

Windows 10中的开放式系统映像备份

在Windows10中，点击开始，键入“备份”，然后选择该条目。

在“Backup and Restore(Windows 7)”(备份和还原(Windows 7))窗口中，单击“Create a System Image”(创建系统映像)链接。

Windows 8中的开放式系统映像备份

在Windows8中，点击开始，键入“文件历史记录”，然后选择“文件历史记录”条目。

在“File History”(文件历史记录)窗口中，单击“System Image Backup”(系统映像备份)链接。

Windows 7中的开放式系统映像备份

单击“开始”，单击“入门”项右侧的箭头，然后单击“备份文件”。

在“Backup and Restore”(备份和还原)窗口中，单击“Create a System Image”(创建系统映像)链接。

第二步：创建系统映像备份

打开系统映像工具后，在Windows7、8或10中创建系统映像的步骤是相同的。

当您第一次打开该工具时，它将扫描您的系统以查找外部驱动器。然后，您可以决定要将图像保存到何处。它可以连接到外部驱动器、多张DVD或网络位置。选择要保存备份的位置，然后单击“下一步”。

默认情况下，该工具仅备份系统驱动器。如果需要，您可以包括其他驱动器，但请记住，这会增加最终映像的大小。通常，我们希望为每个驱动器创建单独的映像备份。

在确认屏幕上，请注意图像可能占用的空间量。如果有任何地方看起来不对劲，你仍然可以回去进行调整。如果一切正常，请单击“开始备份”按钮。

当该工具创建图像时，您将看到一个进度指示器。

这可能需要一段时间。在此示例中，我们正在备份一个包含约319 GB数据的驱动器。当备份到通过USB连接到我们PC的外部硬盘时，大约需要2.5小时。根据您的PC和要备份到的存储类型，您的时间会有所不同。

第三步：制作系统修复光盘

相关：如何在Windows 8或10中创建和使用恢复驱动器或系统修复光盘

备份完成后，Windows会为您提供创建系统修复光盘的选项。如果您需要更换硬盘而无法启动Windows，您可以使用此光盘启动您的电脑并从映像备份中还原。我们强烈建议您继续制作光盘，然后为其添加标签并将其存储在安全位置。

选择要用于创建光盘的驱动器，然后单击“创建光盘”按钮。

当需要恢复映像时，您可以从恢复光盘启动PC以访问许多恢复工具，包括“系统映像恢复”。

创建映像备份可能需要一些时间，所以最好是在几个小时内-甚至是晚上都不需要计算机的时候创建。如果您需要使用该备份时出现需要，请务必查看我们关于在Windows中还原映像备份的完整指南。
