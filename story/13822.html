每个人都在订阅电脑游戏：他们值得吗？

每个电视网络都在制作自己的流媒体应用，每个主要的游戏发行商都想要自己的订阅。艺电、微软和育碧已经推出了他们的订阅服务--但是他们中的任何一个都值得吗？

PC游戏：财富的尴尬

有几种PC游戏订阅服务。有些提供自助餐，有些则是流媒体服务，你甚至不需要一台游戏PC。

在本文中，我们只关注前者。目前，这份名单包括微软面向PC的Xbox Game Pass的EA Origin Access，以及将于2019年9月推出的Ubisoft的UPlay Plus。

我们不包括英伟达的GeForce Now或即将推出的谷歌体育场。这些服务专注于将游戏从远程服务器流式传输到您家中的硬件。

订阅游戏库的想法并不是特别新鲜。艺电针对个人电脑的服务自2016年开始存在，游戏机的Xbox Game Pass于2017年推出。尽管如此，微软和育碧直到2019年才赶上艺电在个人电脑领域的领先地位。其他拥有桌面启动器的游戏制造商，如动视暴雪(Activision Blizzard)和史诗游戏(Eic Games)，尚未效仿。

它们是怎么工作的？

其基本思想是为您的Windows桌面下载一个包含该服务的应用程序。这些应用中的大多数都包括游戏启动器、游戏商店、公告和聊天等社交功能。一旦你下载了这款应用并登录到你的账户，你就可以从该服务下载游戏了。它类似于蒸汽或史诗游戏商店，不同的是你不需要为单个游戏付费。

服务项目

以下是三大主流游戏订阅服务提供的所有选项：

EA Origin Access Basic：作为我们正在关注的最古老的服务，它于2016年上线，并已成为一项多层服务。第一层是Origin Access Basic，每月5美元，或者你可以支付30美元一年的费用。Basic可以让你访问EA所谓的“保险库”--这是一大批PC游戏宝库，其中包括(在撰写本文时)200多款游戏，如“战地5”、“战地1”、“星球大战前线2”和“疯狂19”。此外，Origin会员在Origin游戏商店购买的游戏可以享受10%的折扣。 EA Origin Access Premier：这个级别的费用是每月15美元或每年100美元，在撰写本文时只增加了额外的10个游戏。不过，您也可以提前访问即将推出的游戏的完整版。相比之下，Basic对提前访问的图书有10小时的时间限制。如果新的EA游戏对你很重要，那么Premium是更好的选择。它也不会强迫你为像国歌这样的游戏付费，国歌在发布时看起来很有趣和令人兴奋，然后，嗯，就有了问题。 适用于PC的Xbox Game Pass：随着Xbox用户快乐地使用了两年的Game Pass，微软终于在2019年6月发布了这项服务，想起了这位PC游戏玩家。每月10美元，它提供100多场游戏的访问。微软承诺将把它所有的第一方游戏都包括在Game Pass中，就像它对控制台版本所做的那样。和艺电的服务一样，Xbox提供大量老游戏，以及一些新游戏，如Metro Exodus和即将推出的Gears 5。此外，Xbox还表示，会员可以享受“独家会员折扣和优惠”。 Xbox Game Pass旗舰版：这项服务为订户提供Xbox Live Gold(大多数Xbox用户的必备)和PC和游戏机的Game Pass。对于任何在家拥有游戏PC和Xbox的人来说，这都是一笔绝佳的交易，因为微软新推出的第一方游戏是跨平台兼容的。这意味着您可以开始在控制台上播放，将您的进度传递到PC，然后返回控制台。 Uplay Plus：这是最新的服务-它将于2019年9月3日首次亮相。Uplay Plus每月收费15美元，可以访问108款游戏，包括“分区2”、“彩虹六号围城”、“刺客信条：奥德赛”(以及更老的“刺客信条”)，以及从“远哭2”开始的“遥远的哭泣”系列。这是迄今为止我们所覆盖的最昂贵的服务。然而，订阅费不仅包括基础游戏，还包括大多数游戏的额外内容和扩展。EA Origin Access和Xbox Game Pass都没有涵盖任何这方面的内容。如果Uplay Plus提供一个价格更低、“没有DLC”的版本，那会很好，但这可能会在未来出现。

他们值得吗？

现在，这是最难的部分。因为市面上有如此多的游戏风格，我们不能说这些服务中的任何一个都是一刀切的选择。因此，我们发明了一些理论配置文件来比较和对比不同游戏玩家的需求。

对于这份榜单，我们假设玩家平均每年购买大约三个AAA级游戏，这还不包括他可能会在销售中挑选的游戏。诚然，这是我们最好的猜测，因为目前关于购买游戏习惯的信息并不多。尽管如此，每年大约180美元的新发行似乎是合适的。

以下是为特定类型的游戏玩家提供的最佳服务：

买便宜货的人：如果你按广告价格购买我们所涵盖的所有服务中的一个选项，就会花掉你330美元。这比每年购买五款游戏略多一点，但你可以从种类繁多的产品中获益。 品牌粉丝：如果你只关注Xbox、育碧(Ubisoft)或艺电(EA)，那么购买你特定品牌的服务是不费吹灰之力的。一年的个人电脑Xbox游戏通行证将花费你大约120美元的最低服务。Origin Access Basic为30美元，Premier为100美元。Uplay Plus每年的费用为180美元。 完成者：那些对完成战役更感兴趣的游戏玩家可能不会发现订阅所有三项服务都是有价值的。更好的选择是选择带有Origin Access Basic的Xbox Game Pass，或者选择Uplay Plus本身。这两个数字中的任何一个都相当于每年购买近三个AAA级冠军头衔。完成者的缺点是-除了Uplay Plus的订阅者-额外的内容将花费更高。 跨平台：如果你有一台Xbox和一台PC，你应该考虑Xbox Game Pass-特别是如果你是微软的第一方游戏的粉丝，比如战争机器系列和(今年晚些时候将推出的)HALO。 阴谋者：“不管你选择哪项服务，好处是你可以随时取消。所以，如果EA推出了一款你渴望尝试的游戏，注册Origin Access Premier，玩这款游戏，然后在完成后取消服务。它很容易做到按月订阅，对于那些不想承诺一整年的人来说是理想的选择。

你得不到的是

虽然这些产品很棒，但它们甚至不能涵盖PC游戏玩家感兴趣的所有即将推出的游戏。瑞德项目的赛博朋克2077在这些服务中都无法使用，该工作室的即时经典游戏“巫师3”也不能使用。“史诗”中的游戏和动视暴雪的任何新内容都将受到限制。

底线

再说一次，很难说这些订阅中的哪一个(如果有的话)是值得的--这在很大程度上取决于你的游戏品味和风格。这就是为什么我们使用上面的基本配置文件来考虑各种选项，并从几个角度来看待价值。另一件需要考虑的事情是，一旦你停止支付订阅费，你就失去了访问游戏的机会。

如果你是一个没有太多时间玩的游戏玩家，你最好直接买几个游戏，然后以你自己的速度玩。这样，你就没有定时器或月度订阅悬在你的头顶上了。

尽管如此，对于那些喜欢玩很多游戏，但在发布时不愿全价购买AAA游戏的玩家来说，这些服务值得一看。
