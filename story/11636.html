如何禁用Norton的通知和捆绑软件

与大多数其他防病毒工具一样，Norton也相当具有侵入性。它会在您的Web浏览器中安装一个工具栏，即使您下载安全文件也会弹出通知，并向您显示各种特别优惠和报告。但是您可以将Norton设置为仅在出现实际问题时通知您。

以下步骤是使用Norton Security Deluxe执行的，但其他版本的Norton防病毒软件的流程应该类似。要了解有关我们推荐哪些防病毒程序的更多信息，请查看本指南。

去掉Norton的浏览器扩展

相关：不要使用杀毒软件的浏览器扩展：它们实际上会降低您的安全性

当您安装Google Chrome、Mozilla Firefox和Internet Explorer时，Norton会自动安装“Norton Security Toolbar”和“Norton Identity Safe”浏览器扩展。

诺顿安全工具栏不仅为你的浏览器增加了一个工具栏，还可以改变页面，以显示搜索结果是否“安全”。您不需要此功能-您的Web浏览器和搜索引擎已经尝试阻止危险网站。无论您是否使用浏览器扩展，防病毒软件都会自动扫描下载内容。Norton Identity Safe扩展提供了密码管理器，但我们建议使用其他密码管理器。

我们一般不建议使用防病毒浏览器扩展，因为它们经常会出现问题。我们建议删除它们。

在Google Chrome中，单击菜单并选择更多工具>扩展。单击Norton Identity Safe和Norton Security Toolbar扩展右侧的垃圾桶图标以将其删除。

在Mozilla Firefox中，单击菜单按钮并选择“Add-ons”。单击Norton Security Toolbar右侧的“Disable”(禁用)按钮以加载项。Norton似乎没有为Firefox提供Norton Identity Safe插件，所以您只需要删除工具栏即可。

在Internet Explorer中，单击菜单按钮并选择“管理加载项”。选择工具栏和扩展下的Norton Toolbar和Norton Identity Protection加载项，然后单击窗口底部的“Disable”(禁用)按钮。

稍后会提示您启用Norton的浏览器扩展，但您可以单击弹出窗口中的“不要再问我”链接，Norton将来不会再询问您。

禁用报告、后台任务通知和播发

诺顿的其余设置位于其界面中。要打开它，请在通知区域找到Norton图标-它看起来像一个黄色的圆圈，里面有一个复选标记-并双击它。图标可能隐藏在系统托盘图标左侧的向上箭头后面。

单击Norton Security窗口右上角的“Settings”链接。

你会在这里看到一个“静音模式”复选框，你可以点击它，让诺顿迅速安静下来。然而，静默模式并不是永久的解决方案。您一次只能启用静默模式一天，之后它会自动禁用。

要查找诺顿的多个通知设置，请单击此处的“管理设置”。

Norton将每隔30天自动显示一份报告，其中包含有关其在过去30天内执行的操作的信息。要禁用报告通知，请将“30天报告”滑块设置为“OFF”。诺顿仍将生成报告；只是不会提醒您。

您仍然可以通过单击Norton界面中的“30天报告”按钮手动查看每个30天报告。此按钮仅在Norton生成报告后才会出现，因此如果您刚刚安装了Norton，则不会看到该按钮。

诺顿在后台自动执行一些任务，包括自动防病毒扫描和系统清理任务。默认情况下，Norton在运行这些后台任务时会显示通知。

如果您不想看到这些通知，请在此处将“Norton Task Notification”设置为“Off”。

默认情况下，Norton会自动显示其他Norton产品的“特别优惠”。这些基本上只是诺顿其他产品的广告。

要禁用这些特惠通知，请在管理设置屏幕上向下滚动，并将“特惠通知”设置为“关闭”。

隐藏安全下载通知

每当您从各种Web浏览器、电子邮件客户端、下载管理器和对等程序下载文件时，Norton都会自动检查信誉服务。如果一切正常，您将看到一条消息，上面写着“[File Name]is security”。

这些弹出窗口完全没有必要。您可以禁用它们，让Norton仅在出现实际问题时提醒您。

若要在下载安全文件时禁用通知，请单击“设置”>“防火墙”>“入侵和浏览器防护”，并将“下载洞察通知”设置为“仅限风险”。

阻止反垃圾邮件弹出窗口

Norton包括与Microsoft Outlook等桌面电子邮件客户端集成的反垃圾邮件功能。此功能包括一个为您提供信息和反馈弹出窗口的“欢迎屏幕”。如果您只想让反垃圾邮件功能在后台静默运行，您可以禁用这些功能。

要执行此操作，请在Norton的界面中单击设置>反垃圾邮件。单击“客户端集成”选项卡，并将“欢迎屏幕”和“反馈”都设置为“关闭”。

在您更改这些设置后，Norton应该会为您让路，在后台自动保护您的电脑。如果Norton检测到问题，您应该只会看到通知。
