如何将树莓PI变成Google云打印服务器

Google Cloud Print是将您的打印机连接到云并享受随时随地打印访问的绝佳方式，但有一个问题。如果您没有最新的云打印就绪打印机，则需要打开计算机才能启用远程访问。请继续阅读，我们将为这项任务配置一个微小的、耗能的树莓PI。

我为什么要这么做？

相关：如何在Windows中启用本地Google云打印和打印机共享目前，在您的家中有两种方式可以访问Google云打印：您可以购买一台支持Cloud Print的打印机，该打印机直接链接到您的Google帐户和您的Cloud Print服务，或者您可以使用PC(可以访问您希望添加到Cloud Print的打印机)作为云打印服务器。

第一种情况是理想的，因为打印机本身直接连接到云，不需要中介。然而，除此之外，您的目标应该是确保中介浪费的资源尽可能少。让台式计算机24/7全天候运行的唯一目的就是充当云打印服务器，对于一项需要很少马力的作业来说，这是一大堆资源。

为了减少浪费的资源，我们选择将一个小型、低功耗的Raspberry PI设备变成一个超轻量级的Cloud Print Server。这个设置最好的部分是，树莓派可以继续扮演其他角色。例如，我们的Raspberry Pi Cloud Print Server也是与我们的Raspberry PI天气指示器功能相同的设备。完成这两项工作所需的资源如此之少(假脱机偶尔打印作业，并运行一个简单的脚本来检查天气和切换LED)，因此没有理由不堆叠任务并从我们的Raspberry Pi购买中获得更多。以下是一些可以与Raspberry Pi云打印服务器轻松堆叠的项目：

用树莓PI构建LED指示器(用于电子邮件、天气或任何东西)。 如何将树莓PI改造成低功耗的网络存储设备。 如何将树莓PI变成永远在线的BitTorrent Box。 如何在您的树莓PI上安装用于轻量级Usenet下载的NZBGet

安装Chromium

我们的Raspberry Pi as Cloud Print Server模型的秘诀是开源浏览器Chromium。将云打印功能添加到PC的官方方法之一是使用谷歌的Chrome网络浏览器作为打印服务器。不幸的是，尽管有针对相当多Linux发行版的官方Chrome版本，但它只支持x86/x64架构，而不支持支持Raspberry PI和Rasbian的基于ARM的架构。这就是Chromium的用武之地，因为我们仍然可以访问Chromium中的相关功能，我们需要这些功能来将我们的Raspberry PI链接到Google的云打印机系统。

单击“添加打印机”。您将被踢到授权页面，如下所示：

填写您用于管理云打印机的帐户的Google帐户凭据。确保选中“保持登录”，因为这将是我们不定期交互的独立打印服务器。

授权您的帐户后，您将看到“添加打印机”按钮。Raspberry PI有权访问的任何打印机(无论是本地打印机还是网络打印机)都将添加到您的Google Cloud Print帐户中。如果这些打印机以前是通过其他方式添加的，您需要访问您的云打印管理页面以删除较旧的条目。

单击Add按钮后，您将看到一个确认页面，指示已添加打印机，您已准备好开始打印。现在是试印的绝佳时机：
