如何创建测试实验室，以便您可以关注极客学校(而不会损坏您的PC)

我们的极客学校文章可能会变得相当复杂，没有理由在你自己的台式PC上做一大堆疯狂的事情。相反，您可以像我们一样只使用VirtualBox为您的所有测试创建虚拟机。下面是如何做到这一点的。

您甚至可以将多个虚拟机联网在一起，因此，只要您的PC中有相当数量的RAM，您就可以完全测试一个场景，而不会影响PC的设置。“这是一种更安全的学习方式！”

在我们开始之前，有一些我们不需要的东西：

可在此处获得的VirtualBox副本。 Windows 7 ISO，您可以使用以下方法之一从Microsoft合法获取。 可在此处获得的Windows Server 2008 R2 ISO。

设置您的实验室

我们需要做的第一件事是安装VirtualBox，由于这是下一次、下一次完成类型的安装，我们相信您将能够在任何时间内完成它。

安装完成后，我们就可以创建第一台虚拟机了。

由于我们将同时拥有客户端和服务器，因此我们将其命名为Windows 7(客户端)，并从下拉列表中选择Windows 7操作系统，然后单击Next。

接下来，您需要为虚拟机分配相当大的内存，最小内存为512MB，但是由于我的PC中有相当多的内存，所以我决定给虚拟机分配2 GB内存。不过，安全的选择可能是1 GB。

此时，我们仍然需要向VM添加硬盘，因此将单选按钮切换到“Create a Virtual Hard Drive Now”，然后单击Create。

虚拟硬盘的类型很多，但VDI选项正好适合我们的目的。

从硬盘的角度来看，固定大小的磁盘可以提供稍好的性能，坦率地说，我们需要我们能获得的所有性能。因此，将收音机翻转到固定大小选项，然后单击下一步。

Windows7的最小推荐空间是25 GB，但如果你有额外的空间，增加到30 GB不会有什么坏处。

一旦VirtualBox创建完我们的驱动器，我们就可以开始工作了。

不过，我们仍然需要配置一些设置，所以在您太兴奋之前，请单击机器菜单并选择设置…。。选择。

当设置对话框打开时，切换到存储部分，然后单击右侧的空IDE控制器。

现在再往右看，单击CD的小图标，在这里您将选择将虚拟CD\DVD插入驱动器。

最后，我们要转到网络部分，将网络设置从NAT更改为内部网络。这一点非常重要。如果你不这样做，你不能仅仅为你的虚拟机创建一个相互对话的网络。

一旦你做完了所有的事情，你就可以继续启动机器了。

一旦启动，它将与任何其他Windows7安装一样。现在我们已经有了一个可以工作的虚拟机，您将需要继续设置一个或两个以上的客户端(运行Windows 7)，也许还需要设置一个运行Windows Server 2008 R2的虚拟机，以供我们即将发表的有关Windows Server的极客学校文章使用。

配置 / 构造 / 布局 / 外形

按Windows+R键盘组合在虚拟机上打开一个运行框，然后键入ncpa.cpl并按Enter键。

右键单击您的网络适配器并打开属性。

然后选择Internet协议版本4(TCP/IPv4)并单击属性按钮。

现在更改单选按钮选项以允许静态IP地址分配，然后在单击OK应用设置之前，将您的IP设置为192.168.1.1，将子网掩码设置为255.255.255.0。

注意：对于您引入网络的每台虚拟机，您需要将IP地址中的最后一位加1。这意味着您的第一台虚拟机将是192.168.1.1，您的第二台虚拟机将是192.168.1.2。所有虚拟机的子网掩码必须保持相同。

您现在应该能够从任何其他虚拟机ping任何虚拟机。
