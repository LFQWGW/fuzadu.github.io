如何在Mac上引导Linux Live USB驱动器

你认为你可以简单地把一个标准的Linux live USB驱动器插入你的Mac，然后从它引导吗？再想一想。您需要不遗余力地创建一个活动的Linux USB驱动器，该驱动器将在Mac上引导。

这可能相当令人头疼，但我们已经找到了一个图形实用程序，它可以使这一点变得很容易。您将能够在Mac上快速启动Ubuntu、Linux Mint、Kali Linux和其他主流Linux发行版。

问题

相关：如何为每个操作系统创建可引导USB驱动器和SD卡

苹果使得从USB驱动器启动非MacOSX操作系统变得困难。虽然你可以将外部CD/DVD驱动器连接到Mac上，并从标准的Linux live CD和USB引导，但简单地将通用USB安装程序和uNetboot等标准工具创建的Linux live USB驱动器连接到Mac上是行不通的。

有几种方法可以绕过这一点。例如，Ubuntu提供了一些艰苦的指令，包括转换USB驱动器的文件系统和使其分区可引导，但一些人报告这些指令对他们不起作用。Ubuntu建议只烧录一张光盘是有原因的。

如果您将其安装在Mac上，Refind应该可以让您启动这些USB驱动器。但是你不必在你的Mac上安装这个替代的UEFI引导管理器。下面的解决方案应该允许您创建Linux live USB驱动器，它可以在现代Mac上启动，而不需要任何额外的摆弄或任何额外的操作-插入、重启和运行。

使用Mac Linux USB Loader

相关：如何使用Mac的磁盘实用程序对驱动器进行分区、擦除、修复、恢复和复制

SevenBits的一个名为“Mac Linux USB Loader”的工具对我们来说工作得很好。只需点击几下，这款Mac应用程序就可以在Mac OS X中创建包含您喜欢的Linux发行版的USB驱动器。然后，您可以重新引导和引导这些USB驱动器，以便从实时系统使用Linux发行版。

注意：在运行Mac Linux USB Loader应用程序之前，请确保将其移动到您的应用程序文件夹中。这将避免稍后出现缺少“Enterprise Source”错误。

首先，将USB驱动器插入Mac，然后打开Disk Utility应用程序。检查USB驱动器是否使用MS-DOS(FAT)分区格式化。如果不是，请删除该分区并创建FAT分区-而不是exFAT分区。

接下来，打开您下载的Mac Linux USB Loader应用程序。如果您已经下载了Linux ISO文件，请选择“Create Live USB”选项。如果没有，请选择“Distribution Downloader”选项，轻松下载Linux发行版ISO，以便与此工具一起使用。

选择您下载的Linux发行版的ISO文件，然后选择一个连接的USB驱动器来安装Linux系统。

选择适当的选项，然后单击“开始安装”继续。Mac Linux USB Loader将创建一个可引导的USB驱动器，该驱动器将在您的Mac上工作，并引导至该Linux发行版，而不会出现任何问题或黑客攻击。

在启动驱动器之前，您可能需要在此处更改一些其他选项。例如，您可以在驱动器上设置“持久性”，USB驱动器的一部分将保留给您的文件和设置。这只适用于基于Ubuntu的发行版。

单击主屏幕上的“持久性管理器”，选择您的驱动器，选择应该为持久性数据保留多少驱动器，然后单击“创建持久性”以启用此功能。

引导驱动器

相关：如何在Mac上安装和双引导Linux

要真正启动驱动器，请重新启动Mac，并在启动时按住Option键。您将看到出现引导选项菜单。选择连接的USB驱动器。Mac将从连接的USB驱动器引导Linux系统。

如果您的Mac刚启动到登录屏幕，您看不到启动选项菜单，请再次重新启动您的Mac，并在启动过程的早期按住Option键。

此解决方案将允许您在Mac上启动常见的Linux USB驱动器。您可以在不修改系统的情况下正常引导和使用它们。

在尝试将Linux系统安装到Mac的内部驱动器之前，请谨慎操作。这是一个更复杂的过程。
