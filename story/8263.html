再见Microsoft Security Essentials：Microsoft现在建议您使用第三方防病毒软件

Microsoft Security Essentials(Windows 8上的Windows Defender)一度高居榜首。多年来，它在测试结果中有所下滑，但微软辩称，这些测试没有意义。现在，微软建议Windows用户改用第三方杀毒软件。

我们从微软的一次采访中得到了这一启示。微软官方网站仍然将MSE标榜为提供“全面的恶意软件防护”，没有任何迹象表明他们不再推荐使用它。微软与其用户沟通不畅。

更新：微软现在发布了一份声明，称“我们相信微软的反恶意软件产品，并向我们的客户、我们的朋友和我们的家人强烈推荐这些产品。”不幸的是，他们的声明没有直接解决霍莉·斯图尔特的评论或MSE考试成绩不断恶化的历史。考虑到MSE的糟糕分数，我们听到的所有关于它让现实世界中的人失望的故事，以及微软前后不一致的沟通，我们仍然觉得我们不能再推荐MSE了。

开门红

微软安全基础曾一度高居榜首。2009年，AV-Compatives.org给它打了非常高的分，并称它是性能最好的免费杀毒软件。

MSE对像我们这样的Windows极客非常有吸引力，他们很快就理解了它。它收到了非常好的恶意软件检测分数，速度极快，而且是免费的。它不仅是免费提供的-它不会打扰你，并试图向你追加销售付费的杀毒解决方案，如AVG和Avast！做。MSE是一股新鲜的空气-无论是在它的界面上，还是在其快速的性能上。它的测试结果显示它是领先的，所以它是当时最好的杀毒软件。

多年来，我们一直推荐使用MSE作为免费的杀毒软件。它默认包含在Windows8中，名为“Windows Defender”。这是Windows8最大的安全改进之一--您有一个杀毒软件，所以每个Windows用户都有保护。如果Windows用户最终不需要寻找第三方杀毒软件，那就太好了。

不断下滑的分数和借口

在过去的几年里，Microsoft Security Essentials在恶意软件检测评分测试中有所下滑。在AV-TEST的2011年年度审查中，微软安全精华(Microsoft Security Essentials)在其测试的所有产品中排在保护最后一位。2012年10月，Microsoft Security Essentials得分太低，以至于失去了AV-TEST认证。2013年6月，MSE从AV-TEST获得零保护分数-这是可能的最低分数。它在最近的其他测试中也是最后一名，包括丹尼斯技术实验室的一项测试。

下图显示了2013年7月和8月AV-TEST图表底部的MSE。当谈到恶意软件防护时，它的测试低于所有其他测试的防病毒程序。

当时，微软辩称，这些测试不能代表现实世界。他们说，他们专注于试图阻止现实世界的威胁，而不是在检测罕见恶意软件是一个重要因素的测试中竞争。他们争辩说，避免假阳性是一个重要的目标，现实世界的经历比随意的测试结果更重要。

像我们这里的How-to极客这样的极客相信他们，相信他们的话。当然，我们在个人电脑上使用Microsoft Security Essentials已经有很多年了。我们没有遇到任何恶意软件，即使在使用其他防病毒程序进行扫描以获得第二个意见之后也是如此。我们喜欢Microsoft Security Essentials，因为它如此轻巧、不具侵入性，而且不会试图向我们推销充满我们不需要的系统实用程序的付费安全套件。我们喜欢Windows8用户不需要任何额外的防病毒保护的想法，从Windows用户的生活中消除了另一个复杂的系统工具。

Microsoft已停止尝试

Microsoft Security Essentials网站承诺“全面的恶意软件防护”和“屡获殊荣的防护”，因此，如果用户相信微软致力于使MSE成为一个有能力的防病毒解决方案，那是情有可原的。但微软现在表示，MSE只是用户不应该依赖的基本保护。

在接受丹尼斯保护实验室的采访时，微软恶意软件防护中心的高级项目经理霍莉·斯图尔特(Holly Stewart)表示，Microsoft Security Essentials只是一个“基线”，旨在“始终处于”防病毒测试的底部。她说，微软将MSE视为第一层保护，并建议Windows用户改用第三方杀毒软件。

根据霍莉·斯图尔特的说法，微软“在几年前有了一个顿悟，回到2011年，我们意识到我们有一个更伟大的使命，那就是保护所有微软客户。”她说，微软将其信息传递给其他防病毒制造商，帮助他们改进产品。“我们过去有一部分时间用来预测检测结果，”但现在这些人被指示专注于新出现的威胁，并与其他反病毒公司分享这些信息。

她接着说：“我们正在向我们的合作伙伴提供所有这些数据和信息，这样他们至少可以做得和我们一样好。自然的发展是，我们将永远在这些测试中垫底。老实说，如果我们的工作做得正确，那就会发生。“

然而，她认为“基线并不等于坏”，并说它们提供了高质量的杀毒软件。但是微软自己也建议用户不要使用MSE，所以很难把它当回事。这不是一般人应该使用的产品--它比没有杀毒软件要好，但也不是我们应该推荐的产品。微软告诉反病毒测试公司不要向普通用户推荐MSE，并告诉普通用户MSE在他们的网站上为他们提供了“全面的恶意软件防护”，这是在伤害其用户。微软需要挑选一条信息并坚持下去。

如果你是个极客，你大概可以勉强应付MSE

现在，如果你和我们一样是极客，MSE和Windows Defender是非常有用的。如果您有良好的安全实践，并且知道自己在做什么，那么使用这个轻量级选项可能就可以很好地进行管理。但普通的Windows用户并不总是遵循正确的安全做法，他们应该使用在测试中表现良好的强大的杀毒软件-正如微软自己现在建议的那样。

如果你是个极客，你可能不应该向你的朋友推荐MSE，也不应该把它安装在你父母的电脑上。是的，这是个遗憾--MSE的轻量级和无麻烦的特性造就了一个很棒的界面和一台速度更快的计算机。但杀毒软件的核心是检测引擎，微软似乎在这方面认输了。
