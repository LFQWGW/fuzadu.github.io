如何在Linux或OS X上安装Microsoft PowerShell

PowerShell现在是开源的，可以在Linux和Mac上使用。你可以从微软下载64位版本的Ubuntu 16.04、Ubuntu 14.04、CentOS 7、Red Hat Enterprise Linux 7和Mac OS X 10.11的官方软件包。

从Microsoft下载软件包

请访问GitHub上的PowerShell项目的Release页面以找到这些软件包。下载适合您的操作系统的版本：

Ubuntu16.04：下载以“16.04.1_amd64.deb”结尾的包。 Ubuntu 14.04：下载以“14.04.1_amd64.deb”结尾的包。 CentOS 7和Red Hat Enterprise Linux 7：下载以“el7.centos.x86_64.rpm”结尾的包。 Mac：下载以“.pkg”结尾的包。

如何在Linux上安装PowerShell

下载软件包后，在Linux桌面上启动一个终端窗口。您现在需要安装软件包的依赖项和软件包本身。

在Ubuntu 16.04上，运行以下命令：

sudo apt-get install libunwind8 libicu55 sudo dpkg -i /path/to/powershell.deb

因此，如果您将软件包“powershell_6.0.0-alpha.9-1ubuntu1.16.04.1_amd64.deb”下载到主文件夹中的Downloads文件夹，则需要运行以下命令：

sudo apt-get install libunwind8 libicu55 sudo dpkg -i ~/Downloads/powershell_6.0.0-alpha.9-1ubuntu1.16.04.1_amd64.deb

请注意，您可以使用制表符完成来加快此过程。例如，如果文件位于Downloads文件夹中，则需要键入~/downloads/powershell，然后按Tab键。如果Bash是该目录中唯一以“powershell”开头的文件，则Bash将自动完成该文件名。

在Ubuntu 14.04上，运行以下命令：

sudo apt-get install libunwind8 libicu52 sudo dpkg -i /path/to/powershell.deb

在CentOS 7上，运行以下命令：

sudo yum install /path/to/powershell.rpm

如果一切正常，PowerShell现在应该安装在您的系统上。

如何在Mac上安装PowerShell

要在Mac上安装PowerShell，只需双击下载的.pkg文件。它将启动一个软件包安装程序，并像安装任何其他应用程序一样安装PowerShell。

目前，该软件包似乎没有签名，所以你必须绕过GateKeeper才能安装。要执行此操作，请右键单击或按住Ctrl键单击文件.pkg，选择“打开”，然后同意运行安装程序。

如何在Linux或Mac上启动PowerShell

相关：5个Cmdlet，帮助您开始使用PowerShell

打开终端并运行“powershell”命令以访问PowerShell shell环境。这在Linux和Mac上都有效-无论您使用的是哪一种。

您将看到以“ps”开头的PowerShell提示符，您可以像在Windows上一样运行PowerShell cmdlet。

要退出PowerShell提示，只需键入“exit”，然后按Enter或关闭终端窗口。

有关更多详细信息，请访问PowerShell项目的GitHub页面。您可以在那里下载源代码、报告问题和查看更多官方文档。
