如何在Windows中启用原生Google云打印和打印机共享

我们已经向您展示了如何在您的移动设备上启用Cloud Print，甚至使用第三方工具将其添加到Windows。请继续阅读，我们将向您展示如何将云打印功能添加为Windows服务，并从您的任何Windows计算机添加本地打印。

我为什么要这么做？

云打印是从任何互联网连接位置打印到任何启用云的打印机的绝佳方式；它完全打破了强制您将文件传输到直接连接到打印机的网络上的计算机的模式。

除了你从云打印中获得的所有好处(我们将在这里更深入地研究)之外，多亏了谷歌的新版本，你现在有两种很棒的方式来访问云打印。

相关：购买合适打印机的入门指南(How-to Geek Guide to Buy the Right Printer)

首先，Google Cloud Print Driver为Windows增加了随时随地打印的功能，无需1)从Chrome打印或2)依赖第三方助手应用程序。由于新的云打印驱动程序像普通本地打印机一样将云打印直接集成到Windows中，因此任何可以访问系统打印机的应用程序都可以访问您的云打印打印机。

其次，Google Cloud Print Service将您现有的所有打印机直接链接到Google Cloud Print。您可以在服务级别集成您的打印机和Cloud Print，而不是依赖Chrome扩展来充当打印服务器。

我需要什么？

对于本教程，您需要一些东西：

一个Google帐户。 一台Windows计算机。 一份谷歌Chrome浏览器。 一份谷歌云打印服务协议副本(如果您希望将现有打印机链接到您的云打印网络)。 一份Google Cloud打印驱动程序副本(如果您要将Google Cloud Print添加到默认Windows打印机列表中)

现在，为了避免任何挫折，让我们非常清楚地说明这一区别：如果您想要将现有打印机集成到您的Google Cloud Print帐户中，您需要安装该服务(我们将在教程的第一部分中介绍)，但是如果您想要在您的计算机上安装Google Cloud Printer作为打印机(以便您可以打印到现有的Cloud Printers)，您需要安装驱动程序。将打印机链接到云的服务；将Google Cloud Print添加到默认打印机列表的驱动程序。

安装Google云打印服务

虽然设置Google Cloud Print Service并不是一件很痛苦的事，但对于Google产品来说，这是非常不直观的。要安装打印服务，您需要首先访问此处的Google云打印服务下载页面。尽管如果你不使用Chrome，它会拒绝你下载应用程序，但实际上你可以用任何浏览器直接下载。问题出现在以后，当安装应用程序真正调用Google Chrome时(现在，它是否真的需要任何Chrome组件才能运行，或者Google只是想让你使用他们的浏览器，目前还不清楚)。

单击“接受并安装”下载初始安装应用程序后，Windows UAC将提示您授权安装。确认安装并让其完成安装。
