IT：如何在Windows Server 2008 R2中设置DHCP

继续学习IT基础知识系列，今天我们将向您展示如何在Windows Server 2008上设置DHCP，而不是在路由器上使用它。

注意：这是我们正在进行的IT管理基础系列教学的一部分，可能不适用于所有人。

本指南假设您已经在一台机器上启动并运行了Server2008-如果没有，您应该查看我们的指南，了解如何安装它。在继续之前，您还应该在服务器上设置静态IP地址。

要开始，请启动Server Manager，右键单击Roles，然后选择Add Roles。

系统将提示您正常的“开始前”屏幕，单击“下一步”后，您可以选择DHCP服务器。

接下来，您需要选择要将DHCP协议绑定到的网络连接。

输入您的DNS服务器的IP地址，在本例中是同一台计算机-但请注意不要输入环回地址(127.0.0.1)，因为这将是您的客户端进行名称解析的地址。

再次单击“下一步”跳过WINS设置，这将引导您创建DHCP作用域，您可以在其中单击“添加”按钮。

现在您需要：

为您的作用域指定一个名称。 输入您希望客户端可以使用的第一个地址。 输入您希望客户端使用的最后一个地址。 输入子网掩码(通常为255.255.255.0)。 输入默认网关的IP地址(通常路由器IP为.1)

一旦您点击了OK，您就可以点击Next 4次，进入确认屏幕，在这里您最终可以点击Install。

一旦安装完成，您的DHCP就可以正常工作了，您可以立即开始管理您的DHCP服务器。
