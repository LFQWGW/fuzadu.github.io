如何使用Windows的文件历史记录备份数据

文件历史记录是Windows 10的主要备份工具，最初是在Windows 8中引入的。尽管名为文件历史记录，但它不仅仅是恢复以前版本文件的一种方式-它是一个功能齐全的备份工具。

设置文件历史记录后，您只需将一个外部驱动器连接到您的计算机，Windows就会自动将您的文件备份到其中。保持连接状态，Windows将按计划自动备份。

如何启用文件历史记录

相关：如何使用Windows 10的所有备份和恢复工具

与其他复杂的备份工具不同，文件历史记录设计为快速且易于启用。要启用它，请先将外置硬盘连接到您的计算机。接下来，从“开始”菜单打开“设置”应用程序。导航至更新和安全>备份。

单击或轻触Back Up Using File History(使用文件历史记录备份)下的“Add a Drive”(添加驱动器)选项，以添加文件历史记录将备份到的外部驱动器。它将列出外部驱动器，并为您提供备份到它们的选项。

您也可以使用控制面板来完成此操作，但我们将在这里介绍新的设置界面。如果您想改用控制面板(例如，如果您仍在使用Windows 8)，请打开控制面板并导航到系统和安全>文件历史记录。

选择一个驱动器，Windows将使用它来记录文件历史记录。“自动备份我的文件”选项将出现并自动打开。每当您将驱动器连接到计算机时，Windows都会自动将您的文件备份到驱动器。

如何配置文件历史记录

选择“更多选项”以配置“文件历史记录”的备份频率、保留这些备份副本的时间，以及-最重要的-它备份哪些文件。

默认情况下，文件历史记录每小时自动备份您的文件，但您也可以在此处选择不同的时间。可选择10分钟、15分钟、20分钟、30分钟、1小时、3小时、6小时、12小时一次，或每天一次。

它通常会永远保留您的备份，但当它们变成1个月、3个月、6个月、9个月、1年或2岁时，您可以让它删除它们。您还可以让File History根据需要自动删除备份，以便在File History驱动器上腾出空间。

默认情况下，文件历史记录将设置为备份用户帐户主文件夹中的重要文件夹。这包括桌面、文档、下载、音乐、图片、视频文件夹。它还包括许多程序存储应用程序数据的漫游文件夹、OneDrive文件夹和其他文件夹。

您可以在此窗口中查看完整的文件夹列表，并添加更多文件夹。选择“添加文件夹”，您就可以选择计算机上的任何文件夹进行备份。您也可以在这里选择一个文件夹，然后使用“删除”按钮来阻止Windows备份它。

注意：在Windows 8中，您没有从文件历史记录添加文件夹的选项-相反，您必须将文件夹添加到库中才能将其包括在内。

您还会发现一个“排除这些文件夹”部分，它允许您从备份中排除特定子文件夹。例如，您可以让Windows自动备份Documents文件夹中的每个文件夹，但忽略一个特定的文件夹。在Windows8中，您可以在“文件历史记录”窗口的左侧找到此信息。

要开始备份到其他驱动器，请使用“停止使用驱动器”按钮。这允许您停止备份到当前驱动器，并开始备份到新驱动器。不会删除备份，但Windows Server只能配置为一次备份到一个驱动器。

此处的“查看高级设置”链接会将您带到控制面板，其中提供了另一个可用于配置文件历史记录的界面。单击此处的“高级设置”，您会发现更多选项，包括在事件查看器中查看最近的错误、清除旧版本的文件，以及允许属于您的家庭组的其他计算机备份到您的驱动器。

如何从备份还原文件

要从外部驱动器还原文件，请打开设置应用程序，选择“更新和安全”，选择“备份”，选择“更多选项”，向下滚动到窗口底部，然后选择“从当前备份还原文件”。

您也可以打开控制面板，选择“系统和安全”，选择“文件历史记录”，然后单击“恢复个人文件”。

(如果您有在另一台计算机上创建的文件历史记录或备份，只需在新计算机上设置文件历史记录，然后选择包含旧文件历史记录备份的驱动器。然后，它们将出现在还原文件界面中，以便您可以还原文件，就像在当前计算机上创建备份时一样。)

此界面将允许您查看备份和恢复文件。浏览可用文件并选择一个或多个文件或文件夹。您可以通过右键单击它们来预览它们，或者选择它们并单击绿色按钮将它们还原到您的计算机上。

要选择时间段，请单击窗口一侧的箭头按钮或窗格。您还会被告知有多少个不同的备份时间段可用。例如，在下面的屏幕截图中，窗口顶部的“第2个，共3个”表示有三个可用备份，我们正在查看第二个备份。有一个较旧的备份版本可用，以及一个较新的备份版本。

如何从文件资源管理器中还原文件

您还可以从文件资源管理器快速还原文件的以前版本。打开文件资源管理器，右键单击要还原的文件，然后单击“还原以前的版本”。您也可以单击“属性”，然后选择“以前版本”选项卡。

此处将提供文件历史记录中该文件的任何可用早期版本。您还可以预览它们，将一个版本还原到其原始位置，或将以前的版本还原到您计算机上的其他位置。

您还可以查看特定文件夹中的以前版本和已删除的文件。为此，请导航到文件资源管理器中的文件夹，单击窗口顶部功能区栏上的“主页”选项卡，然后单击“历史记录”。

您将看到一个可以还原的文件列表，这些文件曾经位于该文件夹中。这与您正常还原文件时使用的界面相同，但文件资源管理器允许您从特定文件夹中启动以加快速度。

文件历史记录是一个非常简单和有用的备份选项，而且它是完全免费的。如果您想要操作系统状态的完整备份副本，Windows 10还允许您创建系统映像备份。这不是备份文件的理想解决方案-即使您进行系统映像备份，您也应该使用文件历史记录创建更频繁的重要文件备份-但一些极客可能会发现，如果他们正在玩注册表或其他系统文件，它可能会很有用。
