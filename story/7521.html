如何在Windows中查看最近打开的文件列表

无论您出于备份原因需要检查您的计算机上最近打开了哪些文件，还是想查看其他人在您的计算机上打开了哪些文件，OSFV工具都会派上用场。今天，我们将讨论该工具是什么，以及如何使用它来查看最近在Windows中打开了哪些文件。

什么是打开保存文件视图？

Open Save Files View(打开保存文件视图)是另一个很棒的NirSoft实用程序，从现在起我们将称之为OSFV。它访问两个注册表项(OpenSavePidlMRU和OpenSaveMRU)，这两个注册表项允许它显示您用Windows打开的所有文件的综合列表。

该程序将显示有关每个文件的几位信息，包括：

文件的名称。 文件类型(文件扩展名)。 它们被打开的顺序。 打开任何给定格式的最后一个文件的时间。 创建和修改的时间和日期。 文件的大小。 与文件属性相关的任何信息

此程序可在32位或64位Windows上运行。

正在下载程序

一旦准备好开始，您将需要从NirSoft网站下载OSFV实用程序。只需滚动到页面底部，根据您的操作系统下载32位或64位版本的程序即可。

下载程序后，让我们在桌面上创建一个名为OSFV的文件夹，并将文件从zip文件夹解压到此文件夹中。

使用OSFV

成功解压文件后，您只需双击“OpenSaveFilesView.exe”并等待程序加载即可。

现在你可能在想你还能做些什么。说实话，除了查看最近访问的文件、与它们相关的信息以及导出您收集的信息的副本之外，您可以使用此实用程序执行的操作并不多。

查看最近打开的文档

比方说，你抓到某人在你的电脑上，他们声称他们只是在使用互联网，但你不相信他们。当它们消失后，您可以运行此实用程序并检查最近访问的文件。

首先，单击两次显示“开放时间”的列，使箭头指向下方。您将看到按最近打开的文件列出的文件。

正如您在上图中看到的，最近访问的文件包括三个图像文件，一个文本文件、一个Word文档和一个Flash视频文件。

文件修改的证据

现在假设，例如，您想要向某人证明您最近没有修改过任何文件。您可以从单击“File Modified Time”列开始，直到箭头像以前一样朝下。

正如您在此图像中看到的，四个文件被列为最近修改过的文件。如果要向某人证明.tif文件和第一个Word文档未被修改，可以按住“Ctrl”按钮并选择这些文件。我们还假设您想要显示FLV文件最近被访问过。只需按照相同的过程操作，直到您选择了三个文件。

现在您已经选择了文件，单击保存按钮，如上图所示。弹出保存对话框后，选择保存位置并为文本文档指定名称。对于本例，我们将其称为“证据”。请注意，默认保存位置与应用程序所在的文件夹相同。

现在让我们打开文件，看看我们证据的详细情况。导航到该文件并双击它。

请注意，每个文档都有与其关联的详细信息的完整列表。

语言选项

Open Save Files View还提供其他四种语言版本：荷兰语、德语、希腊语和俄语。要安装语言，只需单击要下载zip文件的语言的链接。然后解压“openavefilesview_lng.ini”文件并将其移动到程序的安装文件夹中。在本例中，我们要将其移动到的文件夹是桌面上的“OSFV”。

完成此操作后，您可以再次运行Open Save Files View程序，您将看到所选语言的程序。在这种情况下，它是荷兰语。

要删除语言包并返回默认英语，只需将语言文件与可执行文件一起从Open Save Files View文件夹中删除，然后重新启动程序即可。
