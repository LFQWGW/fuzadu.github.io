还在Windows XP上吗？手动更新或被蠕虫感染

微软刚刚用一个关键更新修补了Windows XP中的一个远程代码执行漏洞-在它离开主流支持五年多之后。但是，Windows Update不会自动安装它。您必须从微软网站手动下载并安装。

正如Microsoft安全响应中心解释的那样，此修补程序修复了Windows XP、Windows Server 2003、Windows 7和Windows Server 2008中远程桌面服务中的一个“易受攻击的”漏洞：

在微软终止主流支持五年多之后，微软出人意料地为Windows XP(和Windows Server2003)发布了一个关键的安全补丁。这就是这个虫子有多大。

然而，有一个很大的问题：Windows Update不会自动安装在Windows XP上。正如微软发布的CVE-2019-0708公告所解释的那样：

这些补丁程序命名为KB4500331，可在微软的Update Catalog网站上获得。如果您仍在使用Windows XP或Windows Server 2003，则应立即下载并安装这些修补程序。

此错误不影响Windows 10和Windows 8系统。Windows 7和Windows Server 2008系统将通过Windows Update收到修补程序。如果您运行的是不受支持的Windows版本，则只需手动安装这些补丁程序。如果是，Microsoft建议您升级到受支持的Windows版本。
