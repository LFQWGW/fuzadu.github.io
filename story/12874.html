如何脱机使用Google Docs

Google Docs很棒，但因为它通常需要互联网连接才能使用，所以在离线的情况下完成任务可能很有挑战性。如果你使用的是Google Chrome，名为Google Docs Offline的官方扩展改变了这一点。

相关：Google Docs的10个提示和诀窍

注：离线使用Google Docs需要谷歌官方Chrome扩展，因此只能在Google Chrome中使用。它适用于文档、工作表和幻灯片，但不适用于表单。

如何脱机使用Google Docs

首先，你需要脱机安装Chrome扩展Google Docs。安装后，你会在每个主要的谷歌应用程序中发现一个新的设置，可以让你设置离线使用的东西。当你在一个应用程序中启用该设置时，它会在所有支持的谷歌应用程序中自动启用，所以你不需要在每个应用程序中都启用它。

在我们的示例中，我们将使用Google Docs，但它在幻灯片和工作表中的工作方式是相同的。在应用程序中，点击左上角的汉堡图标，然后再次点击“设置”。

在设置窗口中，将“OFFLINE”开关切换到ON位置，然后单击“OK”。

这样做可以启用所有Google Drive应用程序(文档、工作表和幻灯片)的脱机模式。

为了在本地节省空间，Google Docs只将最近访问过的文件保存在本地，以供脱机使用。您必须手动启用它，方法是单击特定文档旁边的三个点图标，然后切换到“可脱机使用”以从任何位置访问您的文件。

任何脱机可用的文件都会在文档、幻灯片或工作表主页的左下角显示灰色复选标记。

现在，当您在脱机模式下打开文件时，文档顶部会出现闪电图标，表示您是在脱机状态下打开文件。

您现在可以在不连接到Internet的情况下创建、打开和编辑任何文件。下次您的计算机连接到网络时，您所做的所有更改都会同步到Google的服务器。
