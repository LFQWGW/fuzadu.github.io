Windows10的免费升级服务已经结束：现在怎么办？

免费的Windows 10升级优惠终于结束了，微软将停止用误导性的升级弹出窗口骚扰Windows 7和8.1用户。但Windows7和8.1还没有结束。它们都是微软在未来几年将正式支持的可靠操作系统。

如果你确实升级到了Windows10，你仍然不需要支付任何费用。你可以继续免费使用Windows 10，甚至可以在相同的硬件上重新安装。事实上，即使错过了升级良机，还是没有办法免费升级到Windows 10。

Windows 7和8.1版本将在2020和2023年前获得安全更新

让我们来解决一些问题：这不像Windows XP的最后期限，在那里微软将停止支持Windows XP。Windows 7和Windows 8.1仍然受到微软安全更新的支持，而且在未来几年内都会如此。

Windows 7的安全更新将持续到2020年1月14日，而Windows 8.1将在2023年1月10日之前收到安全更新。您始终可以在Microsoft的Windows生命周期概况网页上找到这些支持日期。

因此，如果你的PC仍在使用Windows7或8.1，那也没问题。它们仍然受到官方的支持，尤其是Windows7在企业中被广泛使用。让Windows Update保持启用状态，您将一直获得安全更新，直到2020或2023年。

微软正在移除GWX和那些令人讨厌的升级服务

相关：立即升级还是今夜升级：微软如何大刀阔斧地向每个人推介Windows10

那种咄咄逼人的“获得Windows10”升级优惠，也被称为“GWX”，终于要消失了。升级提示应该已经消失。如果你在接下来的几天里确实看到了剩余的升级优惠弹出窗口，点击它实际上不会做任何事情，因为免费升级不再可用。

微软宣布，将推出Windows 7和8.1的升级，删除负责生成这些升级提示的GWX程序。继续从Windows Update安装更新，GWX工具将从您的计算机中删除。

Windows Update也将失去升级优惠。Windows 10将不再是Windows 7和8.1的Windows Update中的“推荐更新”，因此您可以启用自动更新，并安全地从Windows Update安装一切。

软件开发人员不会很快放弃Windows 7

尽管微软采取了咄咄逼人的升级策略，但仍有很大比例的个人电脑--尤其是商用个人电脑--仍在使用Windows 7。约49%的个人电脑在使用Windows 7，而Windows 10仅有约719%。Windows7不会很快消失。许多软件开发商将继续支持Windows 7和8.1，所以不用太担心不兼容问题。

当然，也有一些例外。微软自己也想鼓励人们升级，因为有这么多新的微软程序只能在Windows 10上运行。例如，你只能使用Xbox应用程序将Xbox One上的游戏流式传输到你的PC上，而Xbox应用程序只能在Windows 10上使用。微软正在将许多Xbox One游戏移植到Windows 10上，但它们只能通过Windows Store提供，而且只能在Windows 10上使用。但即使是微软也没有将Microsoft Office等重要的生产力应用程序限制在Windows 10用户中。最新版本的Microsoft Office继续支持Windows 7。

开发人员可以为Windows应用商店创建“通用Windows平台”应用程序，这些应用程序只能在Windows10上运行，但开发人员并不急于这样做。即使大多数Windows10用户仍然主要依赖桌面应用程序。

此外，游戏玩家要注意，Windows 10确实独家有DirectX 12，而Windows 7和8.1只有DirectX 11，如果游戏或游戏引擎使用DirectX 12，则需要Windows 10才能玩。但许多游戏将提供DirectX 12作为几个选项之一，就像许多游戏为Windows XP用户提供DirectX 9支持，以及为Windows 7用户提供DirectX 11支持一样。新的跨平台Vulkan图形API也提供了许多与DirectX 12相同的功能，并将在Windows 7和8.1上运行-因此，一些开发人员将使用Vulkan而不是仅限Windows的DirectX 12。

Windows 10升级者仍然可以免费重新安装和使用它

任何利用升级优惠的电脑都有权继续免费使用Windows 10。微软将不会开始收取Windows 10的订阅费，无论你是利用了免费升级优惠，还是你的电脑附带了Windows 10。微软正在推出一项新服务，允许企业以月费购买Windows 10企业版许可证，但仅此而已。

相关：如何以简单的方式进行Windows 10的全新安装

如果您曾经将PC升级到Windows10，并且需要重新安装，这很容易。您只需访问微软的获取Windows10页面并创建Windows10安装程序USB驱动器或DVD即可。在你的电脑上安装Windows10，它会自动签入微软的服务器并激活你的电脑。这是因为它有“数字许可”(以前称为“数字授权”)，使您的计算机有权继续免费使用Windows10。

即使您利用了升级优惠，然后立即降级回Windows 7或8.1，您的PC仍有资格随时升级。

还有一种方法可以免费获得Windows 10

相关：您仍然可以免费升级到Windows 10的所有方式

如果你有一台电脑在优惠结束前没有获得数字许可证，仍然没有办法免费升级到Windows 10-目前是这样。微软正在为使用辅助技术的人提供特别的Windows 10升级优惠。然而，该工具实际上并不检查你是否在使用辅助技术。微软正在使用荣誉制度。

Assistive Technologies优惠将于2018年1月16日到期。但是，在此之后，通过在安装过程中输入Windows 7、8或8.1密钥，您仍然可以免费获得Windows 10。我们不确定这个把戏什么时候会停止工作，因为微软并没有公开承认它的存在。

Windows 10现在售价120美元

按照官方说法，Windows 10的升级现在技术上来说，家庭版的价格是120美元，专业版的价格是200美元。然而，很少有人会真正支付这笔费用。如果你想升级，最好买一台装有Windows10的新电脑。这就是大多数人未来发展的方式，除了一些小的例外，比如个人电脑制造商。

是的，有一天Windows 7将成为下一个不再接收安全更新的Windows XP。但那个日期离现在还有三年多的时间。到那时，你很有可能已经购买了一台装有改进版Windows 10的新电脑，或者如果你真的不想使用Windows 10，你也可以切换到另一个操作系统。
