如何在OS X中默认显示展开的打印和保存对话框

默认情况下，Windows OS X会向您显示简单的小型打印和保存对话框。您可以单击右侧的箭头将其展开以获得更多选项，也可以使用终端中的一个简单命令在默认情况下显示展开的对话框。

让我们向你展示一下我们的意思。以下是“保存”对话框的简化版本，它没有提供太多选项。它允许您选择文件名、添加标签、选择位置，最后选择格式。

但是，单击“导出为”字段旁边的箭头按钮将展开对话框，这样您就可以在Mac的系统驱动器中导航、创建新文件夹以及显示或隐藏文件的扩展名

类似地，下面是Safari中显示的简单打印对话框。它提供了更改份数、打印哪些页面的选项，以及将文档打印为PDF的选项，但其他选项不多。

下面是展开的打印对话框，其中包括进一步的纸张大小、方向和缩放等选项。

这两个对话框的扩展版本都有一些非常有用的功能，你们中的一些人可能希望在每次打印或保存时利用这些功能。您可以使用一个简单的命令行技巧使展开的版本在默认情况下始终显示(或直到您将它们改回)，而不必总是展开它们。

要进行这些更改，请首先打开终端，该终端可在应用程序>实用程序中找到。

在终端打开的情况下，键入以下命令(或直接复制并粘贴)以更改打印对话框上的默认视图，并在完成后按“Enter”。

defaults write -g PMPrintingExpandedStateForPrint -bool TRUE

在航站楼里，它会看起来像这样。

要使此命令生效，您必须重新启动计算机。因此，如果您也想影响保存对话框的更改，您应该在重新启动之前插入以下命令以节省时间。完成后不要忘记按“Enter”键。

defaults write -g NSNavPanelExpandedStateForSaveMode -boolean TRUE

同样，在终端中，它看起来与以下内容类似。

现在，继续并重新启动。

如果要撤消此更改，则需要输入前面的每个命令，只在末尾使用FALSE标志，而不是TRUE。

例如，要恢复到旧的打印对话框，请输入以下内容。

defaults write -g PMPrintingExpandedStateForPrint -bool FALSE

若要恢复到旧的“保存”对话框，请使用此命令。

defaults write -g NSNavPanelExpandedStateForSaveMode -boolean FALSE

同样，您需要重新启动才能使这些更改生效。
