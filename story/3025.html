如何使用Ember媒体管理器来组织您的媒体收藏

如果你所有的媒体都贴错了标签，组织不善，那么媒体中心就没有那么有趣了。请继续阅读，了解如何使用Ember Media Manager来塑造您的媒体，并让您的收藏闪耀光芒。

为什么我需要媒体管理器？

也许你从来没有听说过媒体经理，或者如果你听说过，你会把它写成是那些痴迷于此的人需要修修补补的东西。媒体管理器在确保您的媒体中心显示的内容是准确的、经过定制的调谐，并最终展示您的媒体收藏有多么棒方面具有无可估量的价值。

那么什么是媒体经理呢？媒体管理器只是一个应用程序，它对您的媒体进行分类，并将图像和元数据写入存储媒体的目录，以便媒体中心应用程序可以访问该数据，以便显示电影的正确信息(评级、评论、演员列表等)。以及电影的媒体(盒画、电影海报、扇画等)。

如果您的媒体中心已经内置了媒体抓取功能，您为什么还想要一个呢？大多数媒体中心都内置了某种刮板-刮板是一种小脚本，可以在互联网电影数据库等在线数据库中进行梳理，以寻找匹配的媒体。不幸的是，媒体刮板从像样的到差劲的，用你的HTPC遥控器或媒体中心键盘纠正他们的错误是乏味的。此外，几乎每个媒体中心都将其采集的数据存储在本地。你知道这什么意思吗?。如果您侵占了媒体中心的地盘，并且必须重新安装，则所有数据都必须重新擦除。在房子的其他地方再安装一个媒体中心吗？如果您的媒体中心软件甚至支持数据，那么是时候再次重新包装或冒险导出和导入数据了。在大型媒体收藏集上进行抓取可能需要几个小时。除了等待媒体集合抓取之外，您还必须重新检查并更正错误。这很乏味，你没有理由这么做。

当您使用媒体管理器时，所有这些信息都与电影和电视节目一起存储。如果您将一个新的媒体中心添加到您的家庭网络中，那么该媒体中心需要做的就是从您的媒体共享中读取数据并加载它，而不是花上几个小时重新擦除您的媒体。

在下面的指南中，我们将使用Ember Media Manager，这是一个流行的开源媒体管理工具。emember Media Manager最初是为XBMC媒体中心创建的，但是您可以修改您的设置来为其他流行的媒体中心应用程序抓取数据。但是，在本教程中，我们将使用它为XBMC配置介质。

安装和配置Ember媒体管理器

如果您知道要键入的正确设置，安装和配置Ember Media Manager(EMM)是轻而易举的事。在下面的屏幕截图中，我们将向您介绍如何配置EMM以获得最佳的媒体中心体验。请记住，我们正在为XBMC进行设置，您需要调整设置以适应您的媒体中心访问电影信息、缩略图和粉丝图片的方式。

您要做的第一件事是跳过主Ember Media Manager网站上的安装助手工具。这是一个小的216KB的安装程序，可以检查你的计算机操作系统版本，然后为你下载最新的版本。它已经坏了一段时间了(因为Ember Media Manager的官方开发已经暂停)。相反，请访问SourceForge上的Ember Media Manager下载目录，并从当前版本(撰写本文时为1.2)获取合适的版本。

一旦您有了适当的文件，只需将内容提取到您想要存储Ember Media Manager的任何位置，打开该目录，然后第一次运行它。不需要真正的安装-您可以称之为便携，但由于它与您的媒体收藏如此紧密地捆绑在一起，除非您将其放在介质驱动器上随身携带，否则原则上它是便携的。

撇开语言选择不谈，您将遇到的第一个实际提示是请求电影文件的位置。单击Add按钮，命名您的源，输入源路径，并设置源选项。SCAN递归地指示EMM向下挖掘文件夹目录，并在子文件夹中查找电影。由于我们将所有电影都保存在一个简单的每个文件夹一部电影的配置中，因此未选中此选项。对于XBMC中的大多数缩略图和图稿功能，仅检测每个文件夹中的一个影片至关重要。如果您当前将所有电影都混杂在一个大文件夹中，并且没有分类到单独的子文件夹中，我们建议您在继续操作之前先进行快速清理。最后，应该选中使用文件夹名称作为初始列表，以便EMM在初始抓取之前使用文件夹名称提取电影名称。

注意：我们强烈、强烈、强烈地建议您复制几个电影目录作为测试。我们从集合中选择了几部电影，并专门将它们放在一个标记为/EMM Test/的文件夹中。通过这种方式，您可以熟悉EMM，而无需冒险对整个媒体收藏进行全面且难以修复的更改。一旦您对EMM充满信心，您就可以将源文件切换回主媒体文件夹。一旦您为您的电影设置了测试文件夹，请对您的电视节目执行同样的操作。

一旦您设置了电影源，EMM将提示您选择它应该如何保存海报、影迷图片和NFO文件。如果您正在为XBMC设置内容，则需要检查folder.jpg、Movie.tbn、post.jpg、fan art.jpg、<Movie>-fan art.jpg和<Movie>.nfo。其中一些，如folder.jpg和<movie>.tbn是半冗余的，但它们是有目的的。Folder.jpg将在XBMC的文件浏览器和其他文件浏览器(如Windows资源管理器)都能识别的缩略图中放入一个缩略图，并且<Movie>.tbn是XBMC中电影封面缩略图的默认格式。将它们都放入可以更轻松地在多个系统上进行浏览，并减少在XBMC中的加载时间。NFO文件是存储媒体元数据的简单文本文件。emember Media Manager将在每个电影文件夹中创建一个包含所有抓取的电影数据的文件。这样，以后XBMC再也不会去找它了。它将从电影目录中拉出您的自定义数据。魔法！

设置电影文件夹后，您将对电视节目重复此过程。如果你没有电视节目，就跳过这个。如果你这样做了，请听从我们早先的警告！制作一个测试文件夹！您将以相同的方式选择信号源，并在设置菜单中选中以下选项：

对于本轮，您将检查folder.jgp、fan art.jpg、season-all.tbn、<ep>.tbn，然后在季节部分seasonXX.tbn、folder.jpg和fan art.jpg中。

如果您对XBMC缩略图系统的结构感兴趣，可以在XBMC wiki中查看缩略图系统的完整指南。

选择完信号源后，单击下一步完成并启动Ember媒体管理器。您将看到完整的界面(暂时为空)。在左下角，当EMM扫描您的文件夹时，您将看到电影和电视节目标题旋转而过。最初的扫描不是擦伤。它只是EMM获取文件名，并调出它在文件夹中找到的任何线索。例如，我们的测试文件夹中有17部电影，根据文件夹中存储的内容和电影文件本身，EMM提取了从标题到粉丝图片、收视率等各种信息。这完全取决于您以前是否使用其他媒体管理器抓取过文件，或者您下载的文件是否已经附加了额外的文件。

如果没有媒体信息，也不要感到惊讶。对于处女系列或你手工撕开的系列，这很可能是你找到它的方式。不过别担心！在本教程结束时，您将完全更新并获得出色的媒体收藏。

使用Ember媒体管理器刮掉您的媒体收藏

此时，您已经设置了应用程序，进行了初始扫描，现在需要开始填写有关媒体文件的附加信息。

让我们以“圣诞节前的噩梦”为例开始吧。它位于EMM中，但绝对没有数据；它只是一个原始的电影文件。这不是很有趣，这让我们的媒体中心完全没有什么有趣的东西可以加载。让我们对这部单独的电影做一个简单的剪辑，向您展示这个过程是如何进行的。请注意，你不必逐个刮掉每一部电影！emember媒体管理器做了很好的批量抓取工作。但是，如果您是媒体组织和剪贴这一行业的新手，我们希望您能看到循序渐进的过程。选择一部电影并右击它，我们将在圣诞节前使用“噩梦”，然后选择(重新)刮电影。

对于几乎每一部电影，除了一部奇怪的外国电影或鲜为人知的独立电影外，EMM几乎应该会立即做出回应。它在几秒钟内就发现了圣诞节前的噩梦。

选择匹配项，然后单击确定。Emm会再刮一会儿，然后给你看电影海报的展示。有些电影只有一张海报，有些有几十张。拉下尽可能大的图像，你永远不会知道明天的媒体中心和高清电视的分辨率会有多高--这只是一点点面向未来的东西。

在电影海报选择之后，就是扇子艺术了。可以说，“粉丝艺术”的范围很广，从流行场景的电影剧照，制片厂发布的宣传壁纸，粉丝定制的图像，甚至是完全原创的粉丝创作的艺术品，应有尽有。大多数电影都有相当多的画面可供选择。如果您拿不定主意，EMM有一个内置的缓存功能。如果您勾选了多个扇图，只有您选择的蓝色扇图将是主要的扇图，所有其他的将与电影一起本地缓存在/ExtraThumbs/类别中，以便于将来访问。

当你挑选完电影海报和影迷插图后，Ember电影管理器会把你踢到你正在制作的电影的编辑电影仪表板。在那里你可以回顾它所做的所有改变，包括演员名单，剧情总结，评级，类型标签，以及你已经获得的海报/粉丝艺术/额外的拇指。

如果一切正常，请单击角落中的“确定”。原本只是简单地写着“这部电影没有信息可用”的灰色空框现在是一个彩色的电影摘要，上面有电影海报、粉丝图片、评级(包括批评评级和父母警告评级)以及指示电影分辨率和音质的图标。就像你看到的一样？花点时间右键单击您刚刚刮过的电影，然后选择Lock。这是Ember Media Manager中的一项新功能，允许您锁定您满意的条目，这样以后的抓取就不会意外地替换您手工挑选的插图。锁定项目的条目背景为浅蓝色。

最初的电影刮得太好了，让我们把一切都刮掉吧！emember Media Manager使您可以超级轻松地刮掉整个收藏。单击位于Movies选项卡右侧菜单栏上的擦除媒体图标。将出现一个下拉菜单。在此菜单中，您可以选择要擦除的介质。因为我们有一个新的收藏(除了我们刚刚刮的那部电影！)。没有电影数据，我们将擦除所有电影，需要EMM在电影匹配时提示我们，并下载所有项目。在很大程度上，EMM在辨别哪部电影是哪部电影方面做得很好，但它偶尔会有一个问题，比如正在讨论的《变形金刚》电影是1984年版还是2007年版。

大规模刮除有一个不利之处。嗯，只是为你挑选最受欢迎的电影封面和粉丝艺术。如果你是一个定制狂热分子，你会想要手动刮掉每部电影。我们想要达成妥协；我们让EMM来做大量的清理工作，然后我们查看列表，看看是否有我们想要更改的封面或电影海报。在大多数情况下，默认选择对我们来说都是可以的，让EMM来做繁重的工作节省了相当多的时间。

当你看完电影后，就该做你的电视节目了。电视节目的制作方式几乎完全相同，只是有一些细微的不同-过程如此相似，我们将跳过逐步的屏幕截图，只给你一个不同之处的简要介绍。

“电视节目”选项卡位于主界面中“电影”选项卡的正旁。您只需点击它，然后浏览您的电视节目列表。电影和电视节目界面的最大不同之处在于，你不能批量刮掉你的电视节目。在制作电视节目的过程中，还有几个额外的步骤，比如选择节目的语言，抓取第三季的缩略图，当然还有第三季/第二集的剪贴。因此，您需要右键单击并(重新)刮掉每个电视节目。您可以高亮显示多个节目，然后选择(重新)刮除所有节目，但当完成每个节目时，系统仍会提示您选择列表中的下一个节目。

为一部有多季和数百集的大剧拼凑可能需要几分钟-例如，《吸血鬼杀手巴菲》花了整整10分钟的拼凑。与EMM只需要抓取几个图像的电影不同，电视节目抓取涉及到多季的封面和粉丝艺术、剧集摘要、剧集屏幕抓取和额外信息。谢天谢地，因为您正在抓取数据并将其与您的媒体一起存储，所以您只需执行一次操作！就像电影一样，如果你取消了一个电视节目，你对结果真的很满意，不要忘了右击并锁定条目，这样它在未来就不会意外地被重新强奸。

如果您对Ember Media Manager的电影和电视节目抓取功能的测试进行得很顺利，请继续并将测试目录中的所有文件夹复制回各自的媒体目录中。然后进入菜单栏，单击Edit->Settings，在电影和电视节目下的Files and Sources部分，将源位置从测试目录更改为主目录。我们知道这是额外的一两步，但是在测试目录中使用像EMM这样的强大工具学习诀窍要比把整个媒体目录搞得一团糟要好得多。一旦您更改了源目录，您就可以扫描您的媒体收藏的其余部分，以使一切保持最新状态，并享受您组织精美的收藏！

在XBMC中查看更新的介质

如果你不能在电视上欣赏，那么所有这些工作都没有多大价值。启动XBMC并更新您的库。XBMC应该重新扫描您的所有目录，并在搜索本地数据时使用本地缓存的数据，然后再将刮板发送到Internet上以填补空白。多亏了你得心应手的工作，应该不会有任何空白需要填补了。

在上面的屏幕截图中，我们可以看到在Media Info视图下XBMC的默认皮肤(Confluence)中显示的圣诞节前噩梦列表。我们在Ember Media Manager中看到的所有信息和伟大的扇子艺术都在现场。如果您单击菜单按钮，您可以查看其他信息，如演员列表。

在这里的截图中，我们可以看到“吸血鬼猎人巴菲”的所有几季都是在海报包装视图中展示的。我们可以深入到各个季节以获取更多信息和屏幕截图。

同样，我们在Ember Media Manager中仔细收集和检查的所有信息都存在于XBMC中。每集电视节目都有一个屏幕帽、一个摘要和其他信息，如编码类型、显示比例等。
