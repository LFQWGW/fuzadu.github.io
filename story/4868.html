如何在Windows 8中禁用Charm Bar和Switcher热角

有几个程序可以阻止你在Windows8中将鼠标移到屏幕角落时，应用切换器和咒语不会出现，但你可以通过这个快速的注册表攻击自己来做到这一点。

你也可以通过安装Classic Shell这样的应用程序来隐藏符号栏和切换器，它还会添加一个开始菜单，让你直接登录到桌面。

编辑注册表

首先，按Windows键，键入regedit，然后按Enter键打开注册表编辑器。出现User Account Control(用户帐户控制)提示时，单击Yes(是)。

在注册表编辑器中导航到以下注册表项：

右键单击ImmersiveShell密钥，指向新建，然后选择密钥。

将关键点命名为EdgeUI并按Enter键。

右键单击EdgeUI键，指向New，然后选择DWORD值。

将值命名为DisableTLcorner，然后按Enter键。

以相同的方式创建另一个DWORD值，并将其命名为DisableCharmsHint。

双击DisableTLcorner值，键入1，然后按Enter键。这会禁用左上角的热角，默认情况下会激活切换器。

双击DisableCharmsHint值，键入1，然后按Enter键。这会禁用右上角和右下角的热角，默认情况下会显示魅力栏。

若要撤消更改并在将来恢复热点，请右键单击您创建的DisableTLcorner和DisableCharmsHint值并将其删除。

请注意，这只会禁用热角本身。如果您将鼠标移动到其中一个角落，然后将其沿屏幕边缘移动到屏幕中央，则会出现切换器或符号栏。

要在禁用热角后快速访问这些功能，请按Windows键+C打开字符栏，或按Windows键+Tab打开切换器。
