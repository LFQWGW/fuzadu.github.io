Windows Media Center媒体浏览器入门

如果您是Windows Media Center用户，您会非常想要查看Media Browser。Windows Media Center的媒体浏览器插件可获取您的数字媒体文件，并将其显示在视觉上吸引人的、用户友好的界面中，并附带图像和元数据。

要求

Windows 7或Vista。 Microsoft.Net 3.5框架

准备您的媒体文件

要使Media Browser能够自动下载媒体库的图像和元数据，您的文件必须命名正确。例如，如果您有电影《蝙蝠侠开始》的MP4文件，则需要将其命名为Batman Begins.mp4。它不能是Batmanegins.mp4或BATMAN-BEGINS..mp4。否则，媒体浏览器不太可能显示图像和元数据。如果你发现你的一些文件没有拉出封面图片或元数据，那么在IMBD.com这样的网站上仔细检查媒体的官方标题。

电视节目文件

电视节目文件的处理方式略有不同。收藏中的每一部电视连续剧都必须有一个主文件夹，其中包含剧集的名称和每一季的各个子文件夹。

以下是文件夹结构和支持的命名约定的示例。

电视剧\南方公园\第1季第1集第1集第4集。 电视剧\南方公园\第一季\南方公园1×01-1.mp4集。 电视剧\南方公园\第一季第101集-第1.mp4集

注：即使节目只有一季，你也需要总是有第一季的文件夹。如果你有几季的特定节目，但没有碰巧有第一季，只需创建一个空白的第一季文件夹。如果没有Season 1文件夹，其他季节将无法正常显示。

安装和配置

采用默认值下载并运行最新的Media Browser.msi文件。(下面的下载链接)到达最终窗口时，选中“配置初始设置”框，然后单击“完成”。

您可能会看到一个弹出窗口，通知您媒体浏览器的文件夹权限设置不正确。点击“是”。

添加您的媒体

浏览器配置工具应该已自动打开。如果没有，您可以通过开始>所有程序>媒体浏览器>媒体浏览器配置向导将其打开。

要开始添加媒体文件，请单击“添加”。

浏览包含媒体文件的文件夹，然后单击“确定”。在这里，我们要添加一个包含一组电影文件的文件夹。

您可以向每个媒体库添加多个文件夹。例如，如果您的电影文件存储在4个或5个不同的文件夹中，您可以将它们全部添加到Media Browser中的单个库下。*要添加其他文件夹，请单击当前添加的文件夹下右侧的“添加”按钮。

左边的“添加”按钮将添加一个额外的媒体库，例如用于电视节目的媒体库。完成后，关闭Media Browser Configuration Tool(媒体浏览器配置工具)。

打开Windows Media Center。您将在主界面上看到Media Browser磁贴。单击以打开它。

最初打开媒体浏览器时，系统将提示您运行初始配置。单击“确定”。

您将看到一些常规配置选项。重要的选项是元数据。如果要提取媒体的图像和其他元数据，请选中此选项(默认情况下)。

完成后，单击“继续”，然后单击“确定”重新启动媒体浏览器。

当您重新进入媒体浏览器时，您将看到下面列出了您的媒体类别，并在主显示中显示了最近添加的文件。单击媒体库以查看文件。

单击顶部的“查看”，查看一些可供选择的不同显示选项。下面你可以看到“细节”。这将在左侧列表中显示您的视频。当您将鼠标悬停在标题上时，摘要和封面艺术将显示在右侧。

“封面流”以从右到左的格式显示您的书目，并配以镜面封面艺术。

“拇指条”在底部的条带中显示您的片头，上面有摘要、图像和电影数据。

构形

可以通过媒体浏览器配置工具更改设置和选项，也可以通过单击媒体浏览器主页右下角的“扳手”直接在媒体浏览器中更改设置和选项。某些设置可能仅在一个或另一个位置可用。有些将在两个地方都可以买到。

插件和主题

Media Browser以各种插件和主题为特色，可以添加可选的定制和流畅的视觉吸引力。要安装插件或主题，请打开媒体浏览器配置工具。单击“Plug-ins”选项卡，然后单击“More Plug-ins…”纽扣。

注意：单击顶部的“高级”将显示几个附加的配置选项卡。

浏览左侧的插件列表。当您找到您喜欢的内容时，选择它，然后单击“安装”。

安装完成后，您将看到它们列在“已安装的插件”下面。

要激活任何已安装的主题，请单击“显示”选项卡。从Visual Theme下拉列表中选择它。完成后关闭Media Browser Configuration Tool(媒体浏览器配置工具)。

一些主题，例如下面显示的“钻石”主题，包括可选的视图和设置，可以通过屏幕顶部的配置按钮进行访问。

单击该电影将显示附加图像和信息，如概要、运行时间、IMDB评级…

…。甚至还有演员和角色的名字。

剩下的就是当你准备好观看的时候点击“播放”。

结束 / 结尾 / 结论 / 推论

Media Browser是一个非常棒的插件，它为Windows Media Center带来了完全不同的媒体管理和美学水平。还有许多其他定制和配置我们没有在这里介绍，例如添加电影预告片、音乐支持和集成录制的电视。媒体浏览器可以在Windows7和Vista上运行。也支持扩展器，但可能需要其他配置。
