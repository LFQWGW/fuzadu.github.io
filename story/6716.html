如何启用和使用Internet Explorer 11的企业模式

企业模式是Internet Explorer中的一项新功能，允许企业在IE的现代版本中使用旧的Web应用程序。此功能旨在帮助出于某种原因仍需要Internet Explorer 8的企业升级到更现代、更安全的浏览器。

此功能将作为Windows 8.1 Update 1的一部分出现在Internet Explorer 11中，并且还将作为Windows 7、Windows RT、Windows Server 2008 R2 SP1和Windows Server 2012 R2上IE 11的更新提供。

它是如何工作的

Internet Explorer的企业模式是Internet Explorer 11中的一种特殊兼容模式。在企业模式下加载的网站呈现的效果与Internet Explore 8中的一样。一些企业已经对旧版本的Internet Explorer进行了标准化，并可能使用不能与现代版本的Internet Explorer一起运行的内部Web应用程序。微软希望鼓励这些企业升级到现代版本的Windows和Internet Explorer，而不是坚持在Windows XP上使用Internet Explorer 8，后者的漫长生命即将结束。

网站不能要求在此模式下呈现，并且它不会作为普通用户的选项出现在正常菜单中。微软希望企业提供他们自己的网站列表，这些网站将在企业模式下自动加载。它也可以作为菜单选项使用，可以打开和关闭，但默认情况下此菜单选项是隐藏的，必须在组策略编辑器中启用。

如果您的所有Web应用程序在现代版本的Internet Explorer中都运行良好，则此功能不适合您。然而，如果你坚持使用Internet Explorer8，不能放弃它，这个功能的设计就是为了让你可以升级。

使用组策略启用企业模式

相关：Windows中的“组策略”是什么？

如果您需要企业模式，则很有可能您使用的是专业版或企业版的Windows，并且可以访问组策略编辑器。不能在标准版的Windows 8.1或家庭版的Windows 7上启用企业模式。

要启动本地组策略编辑器，请按Windows键+R，在运行对话框中键入gpedit.msc，然后按Enter。

导航至用户配置>管理模板>Windows组件>Internet Explorer。

向下滚动并找到“工具”菜单选项中的“允许用户打开并使用企业模式”。双击它，将其设置为启用，用户将能够手动启用企业模式。

系统管理员还可以启用使用企业模式IE网站列表选项。您需要创建要在企业模式下呈现的网站列表，并将其保存到本地计算机或网站上的文件中，然后在此处输入该文件的地址。Internet Explorer将加载、读取该列表，并在企业模式下呈现该列表上的所有网站。

激活企业模式

启用企业模式后，只需在Internet Explorer中轻触Alt键，单击“工具”菜单，然后选择“企业模式”即可切换当前网站的企业模式。如果您已经设置了将在企业模式下自动打开的网站列表，您甚至不需要执行此操作-它将全部自动发生。

如果您刚刚启用了组策略设置，则必须关闭并重新打开Internet Explorer，此选项才会出现在“工具”菜单中。

这项功能需要专业版或企业版的Windows就不足为奇了。它被命名为企业模式，面向的是网站过时的企业，而不是典型的家庭用户。
