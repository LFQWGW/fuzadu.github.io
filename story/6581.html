什么是最好的杀毒软件？我如何选择？

有大量的杀毒程序可供选择，那么如何找到最好的呢？你用的是电脑自带的还是你朋友推荐的？你怎么知道它是否好吃呢？

防病毒测试组织每月测试防病毒产品，根据其有效性、性能和可用性对其进行评级。在选择杀毒软件时，你不必盲目行事；你可以在知情的情况下做出决定。

免费防病毒产品与付费防病毒产品

在选择防病毒产品时，您需要做出的首要决定之一是是否要花钱。大多数防病毒程序都是付费解决方案，需要花费一些钱。然而，也有很多好的、免费的杀毒产品。你实际上不需要花一分钱就可以得到坚实的保护。

免费防病毒产品通常具有与商业防病毒产品相同的防病毒检测功能，因此它们也会检测和阻止恶意软件。付费防病毒产品有额外的功能，更像是一个完整的安全套件。您将获得额外的防火墙、防盗跟踪、反垃圾邮件功能、网络钓鱼过滤器等功能。这些功能并不是真的必需的--你并不真的需要一个完整的安全套件，只需要一个防病毒程序即可。

免费的产品确实会让你唠叨不休。他们可能会偶尔显示优惠，鼓励您升级到付费版本。这就是他们赚钱的方式；免费版起到了付费版广告的作用。这就是你要做的权衡--你会有点唠叨，但你不需要支付每年的订阅费。

坚持使用知名的防病毒产品

我们建议坚持使用信誉良好的公司生产的知名防病毒产品。原因之一是，防病毒产品将受到审查，并将出现在防病毒测试中，因此您可以了解它的表现有多好。如果您使用一个非常晦涩难懂的程序，您将找不到很多针对它的测试，并且您也不会真正了解它的工作情况。

更危险的是，一些恶意软件将自己伪装成反病毒程序。“假杀毒”恶意软件可能使用诸如AntiVirus Live、Advanced Virus Remover和Internet Security 2010之类的名称。这些形式的恶意软件伪装成合法的防病毒产品，但实际上是恶意的，在后台做着令人讨厌的事情。选择未知的杀毒软件会让你面临被感染的风险。请坚持使用知名的、信誉良好的杀毒软件。

检查测试

相关：在线查找最新防病毒测试结果的4个地方

我们已经覆盖了各种不同的网站，您可以在这些网站上找到防病毒测试结果。其中最重要的是AV测试。AV-TEST定期对防病毒产品进行测试，根据其保护、性能和可用性对其进行评级。这将告诉您他们在阻止提供给他们的恶意软件样本方面有多成功，他们在这样做的过程中表现得有多好，以及他们的界面使用起来有多容易。

前往反病毒测试站点，查看他们最近的测试报告，看看防病毒产品的表现如何。如果您已经在使用杀毒软件，测试可以向您显示其性能有多好。如果您正在寻找新的，您可以看到最有效的防病毒产品，并从那里开始。

没有唯一最好的杀毒软件。即使我们现在可以选出一个最好的杀毒软件，测试排名也会有所不同，所以现在最好的可能不是下个月最好的。

Windows Defender(也称为Windows Defender)。Microsoft安全要点

如果你在反病毒测试网站上查看微软的免费防病毒产品--在Windows 8上作为Windows Defender提供，或者可以下载，并作为更早版本的Windows的Microsoft Security Essentials提供--你会发现一些令人惊讶的东西。Windows Defender或Microsoft Security Essentials显示在列表的最底部，检测评级为零。它被认为是一个“基线”，所以这里的每一款其他产品都被评为比微软的免费杀毒软件好多少。根据AV-TEST的结果，这只是另一种说法，即微软的杀毒产品是测试过的效率最低的产品。在它成为“基线”之前，它一直位于图表的底部。

这里发生了什么事？嗯，事情很复杂-微软的免费杀毒软件曾经在测试中表现得非常好，并获得了极客们的很多喜爱。多年来，它的排名有所下降，微软发表了一些相互矛盾的声明，一名微软员工在接受采访时建议你使用第三方杀毒软件，另一名微软发言人表示，微软事后仍推荐自己的杀毒产品。

事实是，如果你是一个知道你在做什么的极客或知识渊博的计算机用户，你可以用微软的免费杀毒软件勉强度日。然而，如果你想为自己提供尽可能最可靠的保护，或者如果你正在为不太了解电脑的人安装防病毒程序，你可能会想要避免使用微软的产品，而使用更有效的产品。

如果您使用的是Windows 7或Windows Vista，请记住这些版本的Windows附带的Windows Defender不是防病毒产品。这是一个反间谍软件程序，不能保护您免受恶意软件的攻击。Windows Defender仅是Windows 8上的防病毒产品，不适用于早期版本的Windows。

不要忘记基本的计算机安全实践

相关：基本计算机安全：如何保护自己免受病毒、黑客和窃贼的侵害

选择防病毒产品并不是保护您的计算机安全的结束；它只是一个开始。无论你的杀毒软件有多好，没有一个杀毒软件是十全十美的，偶尔也会有恶意软件溜走。您需要练习基本的计算机安全实践，这样才能保证计算机的安全，最大限度地减少计算机暴露于恶意软件的风险，并降低防病毒软件感染病毒的风险。

以上所有信息仅适用于Windows。在Mac上不需要杀毒软件，在Linux上也不需要。在Android上，反恶意软件应用程序绝对不是必需的。当然，这假设你没有从可疑的网站下载盗版Mac、Linux或Android软件--如果你下载了，那么所有的赌注都落空了，你很容易受到感染。
