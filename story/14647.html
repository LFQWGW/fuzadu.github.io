如何取消您的Xbox游戏通行证订阅

Xbox游戏通行证是一种按月付费玩100多场游戏的好方法。如果库变得无聊和/或您想要放弃该服务，本指南将向您展示如何取消Xbox游戏通行证订阅。

微软于2017年6月推出Xbox Game Pass。只需每月9.99美元，Xbox用户就可以访问一个包含100多个游戏的旋转资料库。如果你想把这些游戏保存在你的数字图书馆里，该服务还会对它们提供折扣。此外，用户可以在Windows PC上访问Xbox Play Anywhere游戏。

两年后，微软在Windows10上推出了类似的独立自助服务，每月收费9.99美元。该公司还推出了Xbox Game Pass旗舰版，将两者与Xbox Live Gold结合在一起，每月14.99美元。

也就是说，本指南向您展示了如何取消这些订阅。归根结底，这一切都是通过微软的网站完成的，但你可以在控制台上取消Xbox Game Pass、Xbox Game Pass Ultimate和Xbox Live Gold，而不需要访问PC。取消PC的Xbox游戏通行证需要一台计算机。

使用PC取消您的Xbox游戏通行证订阅

首先，打开任何浏览器并导航到您的Microsoft帐户的服务和订阅页面。如果需要，请登录您的帐户。

接下来，找到您的Xbox游戏通行证订阅，然后单击Xbox徽标下的“管理”链接。

在我们的示例中，我们将取消Xbox Game Pass Ultimate。同样，如果您没有这个具体的计划，您将在列表中看到Xbox Game Pass和/或Xbox Game Pass for PC。您还将看到Xbox Live Gold订阅。

在接下来的页面上，单击“取消”链接。

使用Xbox取消您的Xbox游戏通行证订阅

在Xbox主屏幕上，使用控制器突出显示您的配置文件图标，然后按“A”按钮打开指南。接下来，浏览菜单的选项卡并选择齿轮图标。这将加载“系统”选项卡。

向下导航，突出显示“设置”，然后按“A”按钮。

在接下来的屏幕上，突出显示“帐户”，然后导航到右侧以选择“订阅”。按“A”键继续。

突出显示您的订阅，然后按“A”按钮继续。

在“Payment and Billing”(付款和计费)下，突出显示“View and Manage Subscription”(查看和管理订阅)，然后按“A”按钮。

Microsoft Edge在Microsoft.com上加载您的帐户。使用控制器查找要取消的订阅。突出显示Xbox徽标下的“Manage”链接，然后按控制器上的“A”按钮。

使用控制器移动屏幕上的光标并突出显示“取消”。按“A”键完成。
