如何使用和定制Google Chrome Web Apps

谷歌今天宣布了他们的新Chrome网络商店，有大量的网站和游戏，可以作为应用程序安装在你的浏览器上，在每台PC上同步，并根据你想要的方式进行定制。这就是一切的运作方式。

注：本指南实际上并不是针对专家极客的，尽管非常欢迎您在评论中留下您的想法。

什么是Chrome Web Apps？

新的Chrome Web Apps实际上只不过是普通的网站，针对Chrome进行了优化，然后用一个漂亮的图标包装起来，安装在你的浏览器中。其中一些网站，特别是基于网络的游戏，也可以通过Chrome网络商店支付少量费用购买，尽管大多数服务是免费的。

一旦你安装或购买了其中一个网络应用程序，Chrome将使用Chrome浏览器在每台电脑上保持应用程序同步，这就像是将你的整个操作系统存储在网络上。这是一个有趣的概念，所以请继续阅读，了解更多关于它是如何工作的。

使用Google Chrome Web Apps

一旦你前往Chrome网络商店，找到了你想要安装的应用程序，比如这个纽约时报的应用程序，你只需点击安装按钮-如果这是一个非免费的游戏，系统会通过谷歌结账过程点击几下就会提示你购买。

几秒钟后，您将在New Tab页面上看到一个闪亮的新图标，所有这些应用程序都位于该页面。您可以通过打开一个新选项卡，并确保Apps部分已展开，随时返回到您的应用程序列表。

一旦你启动了其中一款应用程序，你就会立即注意到一个不同之处--这款纽约时报阅读器经过了优化，显然是为在Chrome上很好地工作而设计的，它具有大量的定制功能，以及丰富的新闻导航体验。并不是所有的Web应用程序都能达到这样的质量水平，但希望如此。

如果你留意的话，你会注意到“纽约时报”的Chrome版只不过是一个URL--这意味着你可以很容易地在任何支持HTML5的浏览器中浏览www.nytimes.com/chrome/，并获得类似的体验。Chrome Web Store中的大多数应用都是这种情况，因为它们只是基于常规的网络技术，如HTML5和Flash。

自定义Chrome Web应用程序

通过自定义应用程序的打开方式，您可以将应用程序提升到一个新的水平--有固定的标签、单独的窗口，甚至还有全屏模式。让我们来看一下第一个，别针标签。只需转到您已安装的任何应用程序，右键单击它，然后从菜单中选择“作为固定标签打开”即可。一旦你点击启动这款应用程序，它就会在一个固定的标签中启动，一旦你关闭它，它就会与浏览器一起重新打开。

有些应用程序在全屏模式下运行效果更好，比如Write Space应用程序-您可以右键单击它，然后选择“Open Full Screen”(打开全屏)…

…。一旦你点击图标，你就会立即进入全屏视图。在这里很难看到文本，但它是一个无干扰的文本编辑器，可以离线存储您的数据。注：在Windows上使用F11键可以退出全屏模式，在OS X上可以使用Cmd+Shift+F组合键。

如果你用鼠标右键点击许多应用程序，你可以选择选项来定制应用程序的工作方式--一些应用程序有Chrome风格的选项页面，而另一些应用程序只是把你带到网站的常规设置页面。

如果您使用“作为窗口打开”选项…

当您从现在开始使用该应用程序时，它将在一个专用窗口中打开，不会出现任何地址栏或任何其他杂乱的屏幕。外面冷!

唯一真正令人沮丧的问题是…。您不能拖动和重新排列应用程序。

创建应用程序快捷方式

如果您想在桌面上放置应用程序的快捷方式，只需右键单击并选择“创建快捷方式”即可。

这将调出您可能熟悉的常规创建应用程序快捷方式窗口，但它将调到该应用程序。

这样做的好处是桌面上的图标质量通常会更好。

跨计算机同步Google Chrome Web Apps

要确保您的Web应用程序在您使用的每台PC上都可用，请进入“工具”->“选项”->“个人资料”，然后确保已启用同步。如果您已经启用了它，请确保单击Customize…

请确保选择了要在此屏幕上同步的“应用程序”。您还需要确保任何其他PC都有相同的设置-此设置对话框不会跨PC同步。
