Windows 10常见问题解答：您需要知道的一切

现在Windows10可供公众下载和安装，人们对新版本的Windows比以往任何时候都有更多的疑问。我们汇总了我们在How-to Geek上最常遇到的问题，并对它们进行了汇编，以帮助您快速了解Windows10。

每周，我们的Ask@howgeek.com电子邮件收件箱都会收到数百个问题，我们还会回答几十个来自朋友和家人的问题，他们知道我们在一家科技刊物工作。各行各业的人都对Windows10非常好奇。此外，因为它对全球数百万用户是免费升级的，所以人们对从升级过程到操作系统变化的一切都非常感兴趣。

对于所有好奇的读者、邻居和正在搜索有关Windows10更多信息的人们，为了方便您，我们在这里四舍五入地列出了我们遇到的最常见的问题。

Windows 10真的是免费的吗？

在过去的一年里，人们对Windows10的定价(或没有定价)有很大的困惑。如果你感到困惑，不要感到羞愧，在Windows10的开发和测试版测试期间，微软自己不止几次改变了关于升级和定价时间表的故事。

相关：不，Windows10不需要订阅：以下是微软的赚钱计划

对于很多人来说，Windows10真的是免费的，就像啤酒一样免费。如果您当前运行的是任何正版(非盗版)版本的Windows 7或Windows 8.1，您将免费升级到同等版本的Windows 10。Windows 7家庭版/基本/高级版用户和Windows 8.1用户将免费升级到Windows 10家庭版。Windows 7专业版/旗舰版用户和Windows 8.1专业版用户将升级到Windows 10专业版。

你会注意到，我们没有说Windows 8；如果你有一台Windows 8机器，你首先需要免费升级到Windows 8.1，然后才能升级到Windows 10。目前绝大多数Windows用户都属于上面的保护伞。你的Windows 7台式机，你的Windows 8.1笔记本电脑，只要拥有合法的Windows许可证，就有资格升级。还有一点值得注意的是，Windows 10免费升级密钥与硬件捆绑在一起。您不能升级一台计算机并回滚并保留密钥，以便在另一台计算机上执行全新安装。

有一个小问题：升级只有在Windows10推出的第一年是免费的。Windows10于2015年7月29日正式发布，升级对符合条件的用户将保持免费，直到2016年7月28日。

如果你正在建造一台新的个人电脑，需要一个全新的Windows许可证，你可以花119美元买Windows10(或者花199美元买Windows10专业版)。但实际上，考虑到你可以为Windows7购买更便宜的密钥并进行升级(或者直接从旧笔记本电脑或电脑的底部找来)，购买任一口味的全价Windows10密钥都是一笔糟糕的交易。例如，你可以在车库大甩卖上买一台陈旧得可怕(甚至可能坏了)的Windows笔记本电脑，几乎不花一分钱，然后用这把钥匙升级。

即使你不想冒险使用别人可能录制下来供自己使用的旧笔记本电脑上的密钥，购买旧版本的Windows并升级仍然会更便宜。例如，你可以花131美元买一个全新的Windows8.1专业版，然后把它升级到Windows10专业版(在这个过程中，你可以节省大约70美元)。

简而言之，即使Windows10不是免费的(几乎对所有人都是)，它仍然相当经济，因为你可以使用旧的(更便宜的)Windows7和Windows8密钥来执行免费升级到Windows10。

如何获得Windows 10？

因此，Windows10对几乎所有人来说都是免费的，就像啤酒一样。但是你到底是怎么拿到复印件的呢？这些年来，情况发生了很大的变化(与去旧电脑商店购买装有新操作系统的收缩包装盒的日子相比，变化很大)。

幸运的是，在您的计算机上安装Windows10(无论您是在执行升级还是全新安装)都是一件简单的事情。你从微软(Microsoft)下载Windows 10安装程序工具，运行安装程序，然后在安装向导中完成安装向导，并在这里或那里重新启动，然后砰的一声，你就运行了Windows 10。这真的是一个令人惊讶的简单过程，我们对升级的流畅性印象深刻。您可以从运行Windows7或8.1转到运行Windows10，速度几乎与您的连接下载更新的速度一样快。

如果您是那种想要清楚地了解您到底需要做什么以及此过程涉及哪些步骤的人，请务必查看我们的文章如何从Windows 7或8升级到Windows 10(现在)，以查看详细列出的升级步骤。

我的计算机可以运行Windows 10吗？

这个问题我们已经被问过很多次了，我们不会撒谎，我们不是这个问题的忠实粉丝，因为很难给出一个好的答案。与其他有具体答案的问题不同，比如“我能把Windows7家庭版升级到Windows10吗？”这个问题只有在我们有关于所讨论的计算机的详细信息和其中的硬件的详细信息的情况下才有具体的答案。

如果你的电脑当前运行的是令你满意的Windows7或Windows8，特别是在较新的硬件上，那么它很有可能运行得很好。另一方面，如果你使用的硬件来自你的旧Windows XP机器，而且它几乎没有以令人满意的水平运行Windows 7，那么现实情况是，一路跳到Windows 10可能会产生一种你不会特别满意的缓慢的用户体验。

虽然我们不能就您的具体硬件给出具体答案，但我们可以建议您阅读此处所需的Windows 10系统，并检查您的计算机和硬件与Get Windows 10系统托盘应用程序的兼容性(该应用程序不仅会报告您是否有足够的内存等，而且还会报告您的打印机和其他设备是否兼容)。

在离开这个特定的主题之前，我们可以说的是，我们对Windows10在旧硬件上运行得如此之好感到惊喜，所以不要因为你的电脑寿命太长就排除升级的可能性。

我必须升级吗？

今年早些时候，当微软向数百万用户推出“获取Windows10”托盘应用程序时，许多或多或少完全不知道Windows10的人突然意识到了这一点，并非常好奇这个图标(以及当你点击图标时弹出的信息应用程序)是什么意思。

我们收到的关于Get Windows10应用程序最常见的问题是，不假思索地问，“我如何摆脱Get Windows10应用程序？”因为人们讨厌那些他们不想要的恼人的东西。第二个最常见的问题是“我必须升级吗？”

答案肯定是否定的。你不必升级到Windows 10。虽然微软正在积极推广Windows 10(毕竟对大多数人来说都是免费的)，但除非你愿意，否则你不需要升级。

我应该升级吗？

那些不关心是否必须升级的人通常在考虑是否应该升级。那么，您是否应该升级到Windows 10？除非有一些令人信服的理由不这样做(比如你不能为你的工作所依赖的硬件获得驱动程序)，否则没有什么理由不升级到Windows 10。这是从Windows 7到Windows 8的一次巨大飞跃。它是对Windows 8的一个极好的更新(而且，从各个方面来说，这都是对强制使用的Windows 8无桌面磁贴系统的一种改进)。

不过，你应该马上升级吗？如果你在这篇文章发表的时候正在阅读这篇文章，在Windows10正式发布日期仅仅几周之后，你可能就不应该这么做。我们已经更新了几台机器，没有任何问题，这是我们非常注重Windows的技术写作工作的一部分，但如果你正在考虑升级你的第一台机器，至少等几个月再解决任何问题将是非常明智的。该窗口为制造商更新驱动程序、要修补的Windows 10中不可预见的问题，以及让您正确备份所有文件并真正为过渡做好准备提供了时间。

归根结底，顽固地持有Windows7太久，以至于错过了Windows10的升级期，这将是非常愚蠢的。无论您等待一个月还是六个月，Windows10都是一次可靠而值得的升级。

我必须升级才能执行干净安装吗？

这是升级过程中最令人困惑和恼人的事情之一：如果你想做一个完全干净的安装(很多人都这样做)，你不能只运行安装程序，然后给它一个合法的Windows7或8密钥。实际上，您必须先将计算机从Windows 7或8升级到Windows 10，然后再次运行安装程序才能执行全新安装。

相关：如何以简单的方式进行Windows 10的全新安装

虽然这看起来令人费解(坦率地说确实如此)，但背后是有原因的。当您执行升级时，您的硬件配置将向Microsoft注册，并充当系统确认其为合法Windows计算机的“指纹”(如果您愿意)。之后，当您执行全新安装时，该指纹将验证您的Windows10计算机的身份。

这很麻烦，但你只需要做一次，这台机器就会永久注册，可以在Windows10上使用。

如果我讨厌Windows 10，我可以降级回Windows 7或8吗？

您完全可以从Windows 10降级回以前版本的Windows 7或Windows 8.1。你甚至可以完全擦除你的机器，然后用旧钥匙重新安装你的旧版本的Windows。但是，这是一个非常大的挑战，但是，你只能在前30天内这样做。

30天后，两件事发生了。首先，PC上存储的回滚文件将被删除(因此，如果不完全重新安装以前版本的Windows，则不可能降级)。其次，以前的Windows密钥被用作批准您过渡到Windows10的某种形式的金票，这对用户来说不太显眼。

相关：如何卸载Windows 10并降级到Windows 7或8.1

如果您在第一个月回滚计算机，则会恢复整个旧Windows安装。如果你在30天后回滚，你就不会再回滚那么多了，因为你只是发布了你的Windows10许可证，释放了你的旧密钥，让你的旧密钥用于你的旧版本的Windows。您仍然需要完全重新安装Windows，并使用旧密钥恢复旧安装并运行。

现在有趣的是，升级到Windows10，即使您回滚到以前的Windows版本，也可以确保您获得链接到该许可证密钥的Windows10的永久副本。如果您决定在两年后进行升级，至少根据本文撰写时微软的说法，您将可以免费下载Windows10。

我的旧应用程序和外围设备还能工作吗？

比如“我的电脑会运行Windows10吗？”这个是主观的。好消息是，我们还没有遇到一个不起作用的应用程序(包括一些非常旧但很有用的应用程序，我们从Windows XP开始就一直坚持使用)。坏消息是，每个案例都是不同的，也许你非常喜欢的、在Windows7上完美工作的照片打印甚至没有Windows8驱动程序，更不用说支持Windows10了。

我们听说过很多关于Windows10自动更新的问题。你听到的传言绝对是真的：在家庭版，或者只是普通的Windows10中，更新在发布时会自动下载并应用。您可以通过修改Internet连接类型来欺骗Windows，但我们不建议故意延迟更新。

如果你运行的是Windows10专业版，你可以将更新延迟一段时间，但一旦延迟启动，它们仍然会自动应用(除非你利用组编辑器技巧或注册表黑客来禁用它们)。

Windows 10是否向Microsoft报告？

多年来，Windows一直以各种形式向微软汇报。最明显、最持久的报告是基本的Windows身份验证过程。还有一个和以前一样老的错误报告服务，当你的程序崩溃或事情崩溃时，它会打电话回家，这样微软就可以在表面上防止未来出现这样的问题。

Windows 10更进一步，超越了验证安装和报告软件问题的简单性，将在线体验与本地计算机体验更紧密地结合在一起，确保Windows 10与微软和微软属性(如Bing)的沟通程度高于任何以前的Windows版本。

简而言之，是的，Windows10真的和微软的老东家聊得天花乱坠。最重要的是，如果你愿意挖掘，几乎所有的隐私设置都可以根据你的意愿进行调整。我们强烈建议阅读我们的文章，深入了解Windows10隐私设置以获取更多信息。

我真的需要为纸牌和DVD播放付费吗？

在微软搞乱之前，你不会真正意识到有多少人真的喜欢旧纸牌。回到Windows8，微软抛弃了与Windows一起存在了20多年的老式纸牌和扫雷应用程序，只是取而代之的是Windows应用商店的Xbox集成版本。由于Windows8的使用率较低，这一变化或多或少没有引起人们的注意，因为世界上大多数人仍然在使用Windows7，开心地玩着免费游戏。
