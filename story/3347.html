如何使用其他选项增强您喜爱的文本编辑器

您最喜欢的文本编辑器是否缺少一些功能？也许您想轻松获得字数统计或更改选定文本的大小写。有一种简单的方法可以使用这些特性和其他特性来增强您的编辑器。

Dolphin Text Editor Menu是一个免费程序，可以为任何可以键入句子或段落的程序添加功能，例如记事本、Word、OpenOffice、写字板，甚至是Visual Studio等编程环境。

双击您下载的.exe文件(请参阅本文底部的链接)以安装Dolphin Text Editor菜单。按照安装向导中的屏幕说明操作，如果显示“用户帐户控制”对话框，请单击“用户帐户控制”对话框中的“是”(具体取决于您的“用户帐户控制”设置)。

安装向导上的Select Additional Tasks(选择其他任务)屏幕允许您选择是否要创建桌面图标，并允许您在[Windows]启动时自动运行Dolphin Text Editor菜单。

安装向导上的最后一个屏幕允许您选择启动Dolphin Text Editor菜单和查看版本历史记录。默认情况下，这些选项处于选中状态。我们建议将启动Dolphin Text Editor菜单保留为选中状态，以便在您单击“完成”关闭安装向导时自动启动程序。

在“Dolphin Text Editor”菜单启动之前，将显示“Choose a Language”对话框。单击所需的语言，然后单击确定。

将显示快速入门指南，提供使用Dolphin Text Editor菜单的基本步骤，包括激活菜单的默认组合键(Ctrl+NumPad0)。

注意：要初始使用Dolphin Text Editor菜单，请确保Num Lock处于打开状态。这是使用默认组合键所必需的。

单击X按钮关闭快速入门指南对话框。

在文本编辑器或字处理器中打开文档。在本例中，我们将使用记事本。

我们突出显示了本文中键入到记事本中的第一段。按Ctrl+NumPad0访问Dolphin Text Editor菜单。要获取突出显示文本的字数统计，请从主弹出菜单中选择其他，然后从子菜单中选择字数统计。

字数统计对话框显示有关字数、字符数、段落数、行数的信息以及有关选定文本的其他信息。

您还可以轻松更改选定文本的大小写。选择所需文本后，按Ctrl+NumPad0，然后从子菜单中选择更改大小写，然后选择所需的大小写选项。

我们将所选文本更改为标题大小写。

您可以在Dolphin Text Editor菜单中设置各种选项。要访问这些选项，请右键单击系统托盘中的海豚图标，然后从弹出菜单中选择选项。

注意：您可以轻松地将Dolphin Text Editor菜单图标移动到系统托盘的可见区域，以使其更易于访问。?？?

此时将显示选项对话框。您可以选择为Dolphin Text Editor菜单选择要忽略的程序，并且弹出菜单在所选程序中将不可用。

在“忽略的程序”对话框中，选择不希望“Dolphin Text Editor”菜单可用的程序。做出选择后，单击对话框右上角的X按钮将其关闭。

关闭“忽略的程序”对话框时，不会返回到“选项”对话框。如果要更改更多选项，必须如前所述重新打开它。

如果Dolphin Text Editor菜单上有您不使用的选项，则可以将其从菜单中删除，只显示您将使用的选项。要执行此操作，请单击“选项”对话框中的“隐藏/显示菜单项”。

在“选择菜单项”对话框中，选中不想在菜单上显示的每个项目的复选框，使该框中没有复选标记。自定义完菜单后，单击右上角的X按钮关闭对话框。

Dolphin Text Editor菜单可以作为USB闪存驱动器或外部驱动器上的便携程序随身携带。要使其便携，请右键单击系统托盘中的Dolphin图标，然后从弹出菜单中选择Install to a Memory Stick(安装到记忆棒)。

在浏览文件夹对话框中，导航到要安装Dolphin Text Editor菜单的USB闪存驱动器或外部驱动器，然后将其选中。如果要将程序安装在特定目录中，请导航到该驱动器上的所需目录。单击OK。

用不了多久，Dolphin Text Editor菜单就会安装在USB闪存驱动器或外部驱动器上自己的目录中。双击DolphinTextEditorMenu.exe文件以运行该程序。它会添加到系统托盘中，并且可以像安装的版本一样使用。
