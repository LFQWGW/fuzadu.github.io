将图像和元数据添加到Windows 7 Media Center电影库

如果您使用Windows7Media Center中的电影库来访问和查看您的电影收藏，您就会知道该界面有些不尽如人意。今天，我们将向您展示如何使用一个名为YAMMM的程序在电影库中添加电影封面艺术和元数据。

为了使Windows Media Center影片库能够显示元数据和封面图片，每个影片文件必须存储在一个同名的单独文件夹中，其中XML文件用于显示元数据，folder.jpg文件用于显示封面图片。YAMMM自动执行从Internet将folder.jpg和.xml文件下载到电影文件夹的过程。

设置您的文件夹

如果您目前将电影存储在一个大文件夹中，并且担心必须为每个文件手动创建单独的文件夹，请高枕无忧。File2Folder Script 1.0将自动执行此过程(请参阅下面的下载链接)。如果您已经将电影文件存储在各个文件夹中，或者不介意手动创建它们，您可以跳到安装和配置YAMMM一节。

要自动创建各个影片文件夹的过程，请下载File2Folder Script 1.0并解压缩filtofolder.bat文件。然后，您需要将filtofolder.bat复制到存储所有电影文件的文件夹中。注意：要使此脚本正常运行，应在您的本地计算机上执行此过程。建议不要在网络文件夹中运行它。

双击filefolder.bat以运行该进程。这将打开命令窗口并移动您的文件。该过程完成后，按任意键关闭命令窗口。

现在，您应该将每个电影都包含在它自己的单独文件夹中。

安装和配置YAMMM

现在是下载和安装YAMMM的时候了(下面的下载链接)。要开始安装，只需双击下载的.msi文件并按照安装向导操作即可。

在安装过程中，将打开YammmConfig工具。您需要执行的第一步是定义您的影片库位置。在库选项卡上，单击“Include a Folder…”纽扣。

浏览以选择您的电影文件夹，然后单击“确定”。如果您的电影库中有多个文件夹，请重复此过程以添加其他每个文件夹。完成后，单击“保存”。

由于YAMMM作为服务运行，如果您希望从网络位置添加文件夹，则需要在Network Credentials部分下提供联网计算机的用户名和密码。如果联网的计算机没有密码，您需要设置密码。完成后单击“保存”。

在设置选项卡上，选中“重命名电影文件夹…”框和“重命名电影文件…”盒。这将自动重命名您的电影文件以包括日期。完成后单击“保存”。

接下来，单击元数据选项卡。在“创建以下元信息文件”下，选中Windows 7 Media Center。同时选中Users下的All User Accounts(所有用户帐户)旁边的复选框。单击“保存”。

单击Artwork选项卡。选择其中一个调整大小选项并将质量降低到低于100可以减少打开影片库时的延迟时间。单击“保存”。完成所有配置后，单击“退出”。

单击“关闭”以完成安装。

YAMMM现在将继续提取电影文件的元数据和图像。如果您查看各个电影文件夹，您会注意到电影文件已被重命名以包含日期，并且YAMMM已经下载了一个dvdid.xml文件和一个folder.jpg文件，以便为Movie Library提供元数据和图像。如果您选择调整图像大小，除了folder.jpg之外，您还将看到folder.Original.jpg文件。

现在，当您在Media Center中浏览电影库时，您的电影文件将与漂亮的电影封面一起显示。

通过在电影库中添加元数据，您现在可以按标题、类型、年份、父母评级、类型和添加的日期对电影收藏进行排序。

如果您点击您选择的任何一部电影，您将获得该电影的详细概要。

现在剩下的就是点击“播放”享受一场电影了。

当您向电影文件夹添加更多电影时，YAMMM将自动提取每个文件夹的元数据和封面图片。有关配置影片库的更多帮助，请查看我们上一篇关于设置文件夹并将其添加到Windows Media Center影片库的文章。
