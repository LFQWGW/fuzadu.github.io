如何使用Ubuntu One在线共享文件

Ubuntu One，Ubuntu内置的云文件存储服务，允许你将文件在网上公开或私下与他人共享。你可以直接从Ubuntu的文件浏览器在互联网上共享文件。

Ubuntu one有两种文件共享方式：发布(Publish)和共享(Share)，前者让任何知道文件地址的人都可以在网上公开使用该文件，后者则与Ubuntu One的其他用户共享一个文件夹。

开始

您需要启用Ubuntu one来共享文件，如果您还没有这样做的话。要这样做，只需点击Ubuntu底座上的U形Ubuntu One图标即可。你也可以从仪表盘上启动Ubuntu One。

Ubuntu One使用Ubuntu的单点登录服务。如果您已经拥有Launchpad帐户或任何其他Ubuntu单点登录帐户，您可以使用现有帐户登录。如果您还没有帐户，请单击“我还没有帐户-给我注册”按钮，从应用程序中创建一个帐户。

启用后，您可以将文件放在您主目录的Ubuntu one文件夹中，以便与您的Ubuntu one帐户在线同步。您还可以使用Ubuntu one应用程序指定想要同步的不同文件夹-或者在Ubuntu的文件浏览器中右键单击某个文件夹，指向Ubuntu one，然后选择Synchronize this Folder。

联机发布文件

Ubuntu One可以在线公开提供文件。您将收到一个特殊的短URL或网址，您可以将其发送给其他人。任何知道地址的人都可以在不注册或登录的情况下查看文件，因此这是共享文件的一种便捷方式-但以这种方式共享敏感数据可能不是一个好主意。

要从Ubuntu的Nautilus文件管理器在线发布文件，请右键单击该文件，指向Ubuntu one子菜单，然后选择Publish。你必须选择一个已经与Ubuntu one同步的文件--如果它没有被同步，你可以把它移到你的Ubuntu one文件夹中。

发布文件后，再次右键单击它，然后在Ubuntu one子菜单中选择copy Web Link选项。

Ubuntu会将文件的网址复制到您的剪贴板上。您可以将其粘贴到Web浏览器的地址栏中，以便在线查看或将链接发送给其他任何人。

您还可以从Web浏览器公开共享文件。登录Ubuntu One网站上的Ubuntu One帐户，或单击Ubuntu One应用程序中的Go to the web for public and Private Sharing Options链接。

单击要联机发布的文件右侧的“更多”链接。点击“发布文件”链接，Ubuntu One将显示一个您可以与他人共享的URL。您可以通过单击“停止发布”链接从此处停止共享文件。

私下共享文件

您还可以与一个或多个特定的电子邮件地址共享文件夹。与您共享该文件夹的每个人都将收到一封通知电子邮件。如果他们还没有Ubuntu one账户，他们将被要求注册。共享文件夹出现在Ubuntu One文件夹中的Shared With Me文件夹下，当每个人在线登录时，共享文件夹将同步到他们的计算机。您可以在只读模式下共享文件夹，也可以允许用户修改它们。

要共享文件夹，请右键单击已与Ubuntu one同步的文件夹，指向Ubuntu one子菜单，然后选择共享。

您将看到地址簿中的电子邮件地址列表。如果要与不在地址簿中的电子邮件地址共享文件夹，可以从此窗口将电子邮件地址添加到地址簿。

选择您要与之共享文件夹的一个或多个人，然后单击“共享”继续。要选择多个人，请在按住Ctrl键的同时单击每个人。

您也可以点击Ubuntu One网站上的共享此文件夹链接来共享文件夹。

输入电子邮件地址，指定您的选项，然后单击共享此文件夹按钮以共享该文件夹。

您可以停止从Ubuntu One网站共享文件夹。
