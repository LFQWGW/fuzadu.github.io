什么是LAN唤醒，我如何启用它？

科技经常带来荒谬的便利，比如可以在几英里外打开电脑，而不需要按下电源按钮。局域网唤醒已经有一段时间了，所以让我们看看它是如何工作的，以及我们如何启用它。

什么是LAN唤醒？

LAN唤醒(有时缩写为WOL)是远程将计算机从极低功耗模式唤醒的行业标准协议。随着时间的推移，“低功耗模式”的定义发生了一些变化，但我们可以将其理解为计算机“关闭”并可以使用电源的意思。该协议还允许附加的无线局域网唤醒功能。

如果您计划出于任何原因远程访问您的计算机，这将非常有用：它允许您保留对文件和程序的访问权限，同时将PC保持在低功耗状态以节省电力(当然还有金钱)。任何使用VNC或TeamViewer等程序，或者保持文件服务器或游戏服务器程序可用的人，为了方便起见，可能都应该启用该选项。

局域网唤醒取决于两件事：您的主板和网卡。您的主板必须连接到ATX兼容电源，就像过去十年左右的大多数计算机一样。您的以太网卡或无线网卡也必须支持此功能。因为它是通过BIOS或您的网卡固件设置的，所以您不需要特定的软件来启用它。现在对局域网唤醒的支持相当普遍，即使没有宣传它是一种功能，所以如果你有一台在过去十年左右制造的计算机，你就可以使用了。

对于那些自己建造钻机的人来说，在购买以太网卡时要小心。虽然主板上的大多数内置网卡不需要此步骤，但离散网卡通常需要连接到主板的3针电缆才能支持LAN唤醒。在购买之前先在网上做个调查，这样以后你就不会失望了。

魔术数据包：局域网唤醒的工作原理

启用局域网唤醒功能的计算机实质上是等待包含网卡MAC地址的“魔术数据包”到达。这些神奇的数据包是由专为任何平台设计的专业软件发送的，但也可以通过路由器和基于互联网的网站发送。用于wol魔术数据包的典型端口是udp 7和9。因为您的计算机正在主动侦听数据包，所以一些电源正在为您的网卡供电，这将导致您笔记本电脑的电池耗尽得更快，所以当您需要勉强维持一些额外电量时，经常出差的人应该注意关闭此端口。

魔术数据包通常通过整个网络发送，其中包含子网信息、网络广播地址和目标计算机网卡(无论是以太网还是无线网卡)的MAC地址。上图显示了对Magic Packet使用的数据包嗅探工具的结果，这让人质疑它们在不安全的网络和互联网上使用时到底有多安全。在安全的网络上，或对于基本的家庭使用，应该没有任何实际的理由担心。许多主板制造商通常将软件与局域网唤醒功能一起实施，以提供无麻烦或在很大程度上无配置的使用场景。

如何在系统上启用局域网唤醒功能

要开始使用局域网唤醒，您必须在几个地方启用它-通常是在您的BIOS和Windows中。让我们从BIOS开始。

在BIOS中

大多数较旧的计算机和许多现代计算机都将其局域网唤醒设置隐藏在BIOS中。要进入BIOS，您需要在启动计算机时按某个键-通常是Delete、Escape、F2或其他键(您的启动屏幕会给出按哪个键才能进入安装程序的说明)。*一旦进入，请在电源管理或高级选项或类似的选项下进行检查。

在此HP计算机的BIOS上，该设置位于“电源故障后恢复”选项附近。有些则不那么明显：在我的华硕主板(下图)上，局域网唤醒选项被埋在菜单系统的两层深处，位于“Power On by PCIe/PCI”下，因为内置网络控制器在PCI控制器后面-只有在描述文本中才能看到这是正确的选项。

关键是，找到相关选项并不总是容易或明显的，因为BIOS菜单系统差异如此之大。如果你有问题，检查一下你的电脑或主板手册，或者在谷歌上快速搜索一下。请记住，大多数制造商在线提供PDF版本的文档。

在Windows中

您还需要在操作系统中启用局域网唤醒功能。下面是它在Windows中的运行方式。打开“开始”菜单，键入“设备管理器”。打开设备管理器，展开“网络适配器”部分。右键单击您的网卡并转到属性，然后单击高级选项卡。

向下滚动列表找到“Wake on Magic Packet”，并将值更改为“Enabled”。您可以不使用其他“唤醒”设置。(注意：我们的一个测试平台没有这个选项，但是局域网唤醒在本指南中的其他设置被正确启用的情况下仍然工作良好-所以如果没有的话，不要担心。)

现在单击电源管理选项卡，并确保启用了“允许此设备唤醒计算机”和“只允许魔术包唤醒计算机”框。完成后，单击“确定”。

在MacOS中

打开您的系统首选项，然后选择节能程序。你应该会看到“网络访问唤醒”或类似的东西。这将启用LAN唤醒。

在Linux中

Ubuntu有一个很棒的工具，可以检查你的机器是否支持局域网唤醒(Wake-on-LAN)，并可以启用它。打开终端，使用以下命令安装Eethtool：

sudo apt-get install ethtool

您可以通过运行以下命令检查兼容性：

sudo ethtool eth0

如果您的默认接口是其他接口，请将其替换为Eeth0接口。

查找“支持唤醒”部分。只要列出的其中一个字母是g-g，你就可以使用魔法包进行局域网唤醒。要启用此选项，请使用以下命令。

sudo ethtool -s eth0 wol g

这个应该可以解决了。您可以运行该命令来检查并查看它现在是否已启用。寻找“唤醒”部分。你现在应该看到的是一辆破车，而不是一辆卡车。

如何使用局域网唤醒魔术数据包唤醒您的计算机

要发送局域网唤醒请求，您可以选择丰富的选项。

Depicus有一系列出色的轻量级工具来完成这项工作，包括用于Windows的基于GUI的工具和用于Windows和MacOS的基于命令行的工具。此外，Wiki.tcl.tk还有一个很棒的跨平台的轻量级脚本来处理请求。

DD-WRT有很好的wol支持，所以如果你不想下载软件来做这件事，你真的没有必要这样做。或者，如果你外出走动，你可以用你的Android设备来唤醒你的电脑。

此外，许多应用程序都支持其中的局域网唤醒(Wake-on-LAN)功能。例如，如果你试图用远程桌面程序从远处访问你的电脑，你可以用TeamViewer内置的“唤醒”按钮唤醒沉睡的电脑，该按钮使用局域网唤醒(Wake-on-LAN)。

您可能需要调整该程序中的其他设置才能使其工作，因此请参阅该程序的手册以获取有关局域网唤醒的更多信息。

此外，根据程序的不同，仅当您从现有网络上的计算机发送魔术数据包时，局域网唤醒功能才会起作用。如果你的程序没有自动处理局域网唤醒的网络连接，你需要设置你的路由器转发7号和9号UDP端口，特别是你连接到的PC的MAC地址。如果您不确定如何做到这一点，请查看我们关于路由器转发端口的指南。您可能还想重新设置动态DNS地址，这样就不需要每次都检查远程计算机的IP地址。
