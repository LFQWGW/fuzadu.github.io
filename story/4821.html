Windows中实时文件系统与母版光盘格式的比较

在Windows上刻录CD或DVD时，系统将询问您是要使用Live File System格式还是使用母版光盘格式。每种方式都有自己的优点和缺点。

Windows7将其称为“类似于USB闪存驱动器”或“带有CD/DVD播放器”。但是，不可重写光盘究竟如何才能像USB闪存驱动器那样工作呢？

光盘刻录基础知识

标准可写CD或DVD只能写入一次。当您将数据写入光盘的某个区域时，该数据将永远存在于光盘上。您不能擦除此数据，除非物理销毁光盘本身。

可重写光盘的工作方式有所不同，允许您将光盘“重置”回其原始状态并再次刻录到该光盘。

母版光盘格式

主盘格式是大多数人都熟悉的格式，因为它存在的时间要长得多。使用母版光盘格式时，一张光盘只能刻录一次。如果您正在用文件装满光盘或将ISO映像刻录到光盘上，这是理想的选择。

但是，无论您要刻录多少文件，单次刻录限制都适用。例如，如果您使用母版光盘格式并将50 MB的文件刻录到光盘上，则不能稍后返回并添加更多文件。一旦以母版格式刻录不可重写光盘，其状态为最终状态。您本可以使用的数百兆字节都丢失了-一次烧录就是极限。

但是，母盘格式更兼容。您可以在早于Windows XP的Windows版本以及其他类型的设备(如DVD播放机和CD播放机)上使用母版光盘。这些设备通常不支持实时文件系统光盘。

当将母版光盘格式与可重写光盘一起使用时，您必须使用擦除整个光盘的“擦除”操作来删除文件。您不能简单地删除光盘中的单个文件来恢复空间。

实时文件系统

实时文件系统的工作方式有所不同。使用实时文件系统格式化光盘后，您可以多次刻录到光盘，而不是只刻录到光盘一次。例如，您可以将光盘插入光盘驱动器，并定期向其中添加文件。每个文件都将在您添加时刻录到光盘。对于母版光盘，您添加的文件会进入某种临时区域-在您单击刻录按钮之前，它们不会刻录到光盘上。

当您要在另一台计算机上使用光盘时，可以通过弹出光盘来关闭会话。这会将一些数据写入光盘，因此您应该尽可能少地关闭会话。

您可以稍后打开一个新会话并将更多文件刻录到光盘，从而创建一个新会话。但是，如果您使用的是不可重写光盘，则仍然只能向光盘的每个区域写入一次。例如，如果您将一个50 MB的文件刻录到光盘上，然后将其删除并将另一个50 MB的文件刻录到光盘上，则光盘上使用的总空间仍为100 MB。您刻录到光盘的原始50 MB仍然存在，尽管它已被标记为已删除，并且在您使用光盘时不会显示。

如果您将可重写光盘与Live File System配合使用，已删除文件使用的空间将立即被擦除，空间将被回收。这是可重写光盘的一大优势-您可以像写入USB闪存驱动器一样写入和删除文件，而不必在每次要擦除某些文件时执行繁琐的全磁盘擦除操作。

但是，实时文件系统不像母版光盘格式那样兼容。它可以在运行Windows XP和更新版本的Windows的计算机上运行，但许多其他类型的设备不能使用Live File System光盘。

归根结底，没有一个正确的选择-有一个更兼容的选项和更方便的选项。您选择的选项应取决于您希望如何将文件刻录到光盘，以及要将生成的光盘与哪些设备一起使用。
