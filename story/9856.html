如何从iOS打印到Google云打印机

苹果自己的AirPrint打印机深度集成到iPhone和iPad中。你在整个操作系统中看到的“打印”选项只会打印到支持AirPrint的打印机。“你仍然可以打印到支持Google Cloud打印的打印机，但这需要一点额外的努力。

许多现代无线打印机同时支持AirPrint和Google Cloud Print，所以如果您靠近它们，它们可能就能正常工作。但谷歌云打印(Google Cloud Print)允许你连接较旧的打印机，并通过互联网打印，这是苹果的AirPrint所不提供的。

从Google自己的应用程序打印到Google Cloud Print

谷歌自己的应用程序通常都内置了谷歌云打印(Google Cloud Print)。例如，假设您正在使用Chrome应用程序查看网页，在Gmail应用程序中阅读电子邮件，或者在Docs、Sheet或幻灯片应用程序中查看Google Drive文档。你可以打开应用程序的菜单，然后选择“打印”选项来打印文档。此选项位于不同应用程序中的不同位置。例如，在Google Chrome中，你需要打开菜单，点击看起来像一个带箭头的正方形的“共享”按钮，然后点击第二排图标上的“打印”操作。在谷歌自己的应用程序中，如果你愿意，你可以选择打印到谷歌云打印，而不是苹果的AirPrint。

Google Cloud Print将提供与您的Google帐户关联的打印机，因此请确保您使用与您在Google Cloud Print中关联打印机的相同Google帐户登录该应用。如果您使用的是其他Google帐户，您可以访问Google Cloud Print网站，并与其他Google帐户共享打印机。

使用PrintCentral Pro从任何应用程序打印

相关：无线打印说明：AirPrint、Google Cloud Print、iPrint、ePrint等

不幸的是，谷歌没有为iOS提供官方的Google Cloud打印应用程序。它只将Google Cloud Print构建到自己的应用程序中。如果你想从另一款应用程序打印到支持Google Cloud打印的打印机-或者打印一种谷歌应用程序都不支持的文档类型-你将需要第三方打印应用程序。

谷歌官方推荐使用PrintCentral Pro应用程序在iOS上打印。有两个版本可供选择：一个用于iPhone、iPod Touch和Apple Watch，另一个用于iPad。iPhone版售价5美元，iPad版售价8美元。

虽然还有其他声称支持Google Cloud Print的打印应用程序，但它们也是付费应用程序，谷歌还没有正式推荐。这就是我们推荐PrintCentral Pro的原因。

这款应用程序填补了iOS上谷歌云打印(Google Cloud Print)和其他类型打印机的不足。一旦安装完毕，就可以打开该应用程序，看到附近的打印机列表。PrintCentral Pro支持多种其他打印机，包括大多数Wi-Fi打印机。在列出可用打印机的屏幕上点击“+”按钮，然后选择“Google Cloud Print”，然后使用您的Google帐户登录以访问您拥有的任何Google Cloud打印打印机。

虽然您可以打开PrintCentral Pro应用程序并加载网页、图像和其他文档进行打印，但iOS的共享功能可以提供很大帮助。“这允许您将几乎任何应用程序中的文档”共享“到PrintCentral Pro，然后PrintCentral Pro可以在支持Google Cloud Print的打印机上打印它们。

要做到这一点，请打开您想要打印的应用程序，然后打开您要打印的文档。*点击“共享”按钮，该按钮看起来像一个带有向上箭头的正方形。当您看到共享表并启用“PCL Pro”操作时，轻触图标底行右侧的“更多”按钮。然后，您将在第二排图标中看到一个“PCL Pro”图标。点击此图标可将当前文档的副本发送到PrintCentral Pro应用程序。你会得到正常的界面，允许你把它打印到任何可用的智能打印机上-包括谷歌云打印打印机。

虽然你可以通过AirPrint打印机获得最完整的打印体验，但iOS 8中添加的共享扩展使PrintCentral Pro-和Google Cloud Print-使用起来更加方便。这款应用程序基本上可以扩展iOS上的任何应用程序，支持打印到Google Cloud Print或其他类型的打印机，只要该应用程序包含共享按钮即可。

如果你只需要打印网页、电子邮件或文档，你也可以使用谷歌自己的应用程序免费打印。
