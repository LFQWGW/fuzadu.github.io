复杂度

组织桌面的最佳应用程序启动器和坞站

你的桌面是不是杂乱无章，什么都找不到？你的“开始”菜单是不是很长，必须滚动才能看到有哪些程序？如果是这样的话，您可能需要一个应用程序启动器来组织您的桌面，让您的生活变得更轻松。

我们已经以不同的形式创建了许多有用的应用程序启动器的列表。您可以从停靠程序、便携应用程序启动器、开始菜单和任务栏替换以及面向键盘的启动器中进行选择。

停靠应用程序启动器

坞站是图形应用程序启动器，可增强和组织您的桌面。它们通常是非常可定制和可扩展的。

船坞

RocketDock是一个位于桌面边缘的Windows应用程序启动器，或称Dock。它模仿MacOSX的启动工具栏，包含启动程序、打开文件和文件夹的快捷方式。您还可以使用docklet扩展码头的功能，并且码头的外观是可自定义的。

我们之前已经向您展示了如何安装、使用和扩展RocketDock，以及如何使RocketDock可移植。

对象船坞

ObjectDock是Windows的另一个Dock，类似于RocketDock。它允许您在一个吸引人的动画码头上组织快捷方式、程序和运行任务。您还可以将额外的功能作为小部件添加到Dock中，例如天气小部件、时钟、日历和电池状态小部件。通过导入快速启动快捷方式和固定的任务栏项目快速设置驳接。您的停靠可以放置在屏幕的任何边缘。

还有一个付费版本的ObjectDock(19.95美元)，它允许你创建任意多的码头，并为你的码头添加选项卡。它还提供类似Aero-Peek的功能，以改进任务切换，并允许您访问坞站上的系统托盘图标。

RK启动器

RK Launcher是另一个适用于Windows的免费停靠工具，它在屏幕边缘提供了一个视觉上令人愉悦的栏，您可以轻松地向其中添加程序、文件和文件夹的快捷方式。底座可以放置在屏幕的任何边缘或某个角落。您可以使用主题和自定义图标完全自定义外观，并使用docklet添加功能。它有很多功能，能够将程序最小化到Dock上，这使得RK Launcher成为一个很好的任务栏替代品。

XWindows坞站

XWindows Dock是一个免费的Windows停靠程序，模拟Mac启动器工具栏。它是完全可定制的，包含反射、透明、阴影、模糊等图形效果。他们的网站声称，他们只使用最新的技术，“你可以得到最强大、最稳定、最快的Windows平台Dock”。新的插件管理器还提供了一个新的堆栈容器，类似于RocketDock可用的Stacks docklet，具有扇形/网格视图。

滑船坞

Sliderdock是一个适用于Windows的免费停靠程序，与我们到目前为止向您展示的停靠程序不同。它允许您通过拖放将程序快捷方式、文件和文件夹轻松添加到每个圆形停靠站或环形。您可以有多个图标环。在每个环中旋转鼠标滚轮会旋转图标，从而提供对图标的快速访问。您还可以使用键盘快捷键访问码头上的图标。Sliderdock是完全可定制的，从戒指和图标的外观到Dock的行为。

圆形码头

Circle Dock是Windows下的另一个免费的圆形停靠程序，但与Sliderdock不同。当您激活Circle Dock时，即使您的鼠标位于屏幕边缘，它也可以显示在您的鼠标光标的右侧。可以使用鼠标滚轮或箭头键访问停靠在屏幕之外的任何图标。您可以跨子级别添加无限制的文件夹、快捷方式、链接和管理。码头上的背景和图标是完全可自定义的。Circle Dock支持多显示器和虚拟桌面，便于携带。要运行它，只需解压缩文件并运行.exe文件。

Winstep Nexus坞站

Winstep Nexus Dock是一个免费的、完全可定制的Windows Dock程序，它提供实时的图标反射和许多其他令人眼花缭乱的效果。Nexus Dock的拖放功能使您可以轻松管理应用程序、文件、打印机等。使用拖放功能可在坞站内外移动、复制和重新排列对象。将文档拖放到坞站上的应用程序对象上，以自动将其加载到相应的程序中。使用Nexus Dock的虚拟文件系统支持将控制面板和我的电脑等项目拖动到您的坞站。拖到码头上的文档、图片和视频文件以缩略图形式显示，便于识别。通过将喜爱的.ico、.png或.tif文件拖放到驳接上的对象上，更改驳接上图标的外观。Nexus Dock还可以替代任务栏，在Dock上显示最小化的正在运行的程序和系统托盘。

有一个付费(24.95美元起)的Nexus Dock旗舰版，它有额外的功能，比如多个Dock，能够将快捷方式分组到嵌套子Dock、选项卡式Dock，以及复制、删除和禁用和启用现有Dock的功能。

WinLaunch

WinLaunch是一个免费的、便携的Windows应用程序启动程序，取自Mac OS X Lion。它开始最小化，最初使用Shift+Tab键盘快捷键激活。激活时，Windows桌面图标隐藏，Windows背景模糊，在启动器上显示图标。可以像在iOS中一样对图标进行分组；将一个图标拖放到另一个图标上可创建可重命名的组，如下图所示。您还可以使用“抖动模式”移动和删除项目以及创建新文件夹，该模式可通过在项目上按住鼠标来激活。通过按“F”键将图标添加到启动器，将启动器缩小为可移动窗口，您可以在其中拖动快捷方式、文件或文件夹。

便携式应用程序启动器

以下是便携的应用程序启动器列表。它们对于启动USB闪存驱动器上的便携应用程序很有用，或者如果您不想在PC上安装另一个软件程序。

PortableApps.com

PortableApps.com是一款非常受欢迎的Windows便携软件解决方案，您可以在任何便携存储设备(USB闪存驱动器、iPod、存储卡、便携式硬盘等)、本地存储设备和云中随身携带您喜欢的软件。它是一个完全开源的免费平台，并提供了许多有用的可移植程序。它本质上是一个便携桌面，您可以保存所有便携程序的所有设置和首选项。它带有自己的应用程序启动器，可以方便地访问您的程序。

CodySafe

CodySafe是PortableApps.com的替代品，它为Windows提供了免费的应用程序启动程序，并提供了对便携程序的访问。您可以从CodySafe站点下载便携程序，但也鼓励您从PortableApps.com、PortableFreeware.com和PenDriveApps.com等站点获取程序。您可以对便携应用程序进行分组和分类，并使用皮肤、主题和声音自定义CodySafe。

CodySafe还提供付费版本(19.90美元、29.90美元和89.90美元起)，提供越来越多的附加功能，如子组和子类别、访问Apps Depot、密码保护和硬件加密。

有关更多信息，请参阅我们关于CodySafe的文章。

便携式应用程序启动器

便携应用程序启动器(PAL)是一个免费的Windows启动器，可以将快捷方式组织到组和类别中。它可以从系统托盘或用户定义的热键访问。您还可以使用样式自定义菜单。PortableApps.com格式的便携程序(PAF)可以自动安装到PAL上。除了提供启动应用程序的便捷菜单外，您还可以将文档添加到特殊的Docs屏幕，以及Extras屏幕上的工具和实用程序，例如备注、日历，以及轻松访问便携免费软件集合。

开胃小吃 / 开胃物

开胃是一个免费的，开源的，便携的Windows应用程序启动器，它允许你在一个可定制的，可调整大小的码头上组织你的程序，这个码头可以水平或垂直显示。将图标调整为三种不同大小中的任意一种，并添加您自己的自定义图标。通过将图标拖放到码头上来组织图标，并将图标分组到菜单中。使用蒙皮自定义开胃菜码头。

P开始

PStart是一个简单的Windows系统任务栏应用程序启动器。它专为运行便携应用程序而设计，但也可以在您的本地硬盘驱动器上作为额外的开始菜单使用，以提供快速的程序搜索功能。使用PStart打开重要的文档和文件夹，以及便携程序。

安装PStart时，您可以选择将PStart安装到本地硬盘或可移动设备。PStart在作为便携应用程序安装时使用相关路径。如果在将USB闪存驱动器插入不同的计算机时为其分配了不同的驱动器号，则您的便携式应用程序、文件和文件夹仍可正常打开。

还有一个用于快速查找文件的“搜索”选项卡和一个用于存储您不想忘记的信息的“备注”选项卡。

套房

ASuite是另一个免费的、便携的Windows应用程序启动器，类似于PStart。它在列表选项卡上以树状结构显示您的程序快捷方式、文件、文件夹和网页链接。就像PStart一样，它也是为在可移动介质上运行而设计的，比如USB闪存驱动器。ASuite使用相对路径(如PStart)，因此可以在任何Windows计算机上打开您的程序、文件和文件夹，而不会出现问题。该程序需要安装，但可以安装到任何本地或可移动的驱动器上。

注意：在Windows 7中使用ASuite时，我们建议您将程序安装到“C：\Program Files”以外的位置。ASuite需要在设置时写入设置，如果将其安装到您没有完全写入权限的位置，则会出现错误。

安装托盘菜单

SE-TrayMenu替代了Windows7中缺失的快速启动工具栏(它也可以在XP、Vista和Windows8中使用)。使用SE-TrayMenu，可以通过Windows系统托盘中的可定制弹出菜单快速访问最常用的应用程序和系统命令。使用拖放将程序、文档、文件夹和Internet链接快速添加到菜单。菜单也是完全可定制的。

SE-TrayMenu既可以安装，也可以作为便携程序使用。

有关更多信息，请参阅我们关于SE-TrayMenu的文章。

便携开始菜单

便携式开始菜单是一个简单而免费的Windows应用程序启动器，类似于可以安装在USB闪存驱动器或本地硬盘驱动器上的开始菜单。在一个简单的菜单系统中组织您的程序，并使用系统托盘图标启动它们。当您关闭USB闪存驱动器上的便携开始菜单时，正在运行的应用程序也可以自动关闭。便携式开始菜单还允许您自动安装和卸载TrueCrypt容器。

开始菜单、任务栏和桌面Windows应用程序启动器

以下程序是应用程序启动程序，可取代或增强Windows“开始”菜单、任务栏或桌面。我们还列出了Windows7桌面的Gadget形式的启动程序。

Jumplist-Launcher

Jumplist-Launcher是一个免费的Windows程序启动器，允许您合并任务栏上的应用程序，将多个应用程序合并到一个跳转列表中。它不需要安装，因此您可以直接在本地硬盘或USB闪存驱动器上运行。您可以在一个跳转列表中添加多达60个自定义组中的程序，并将快捷方式、文件和文件夹拖放到Jumplist-Launcher设置对话框中。

有关更多信息，请参阅我们关于Jumplist-Launcher的文章。

7个堆栈

7Stacks是一款Windows版的免费应用程序启动器，可模拟Mac OS X中的Stacks功能。一旦您安装了7Stacks，桌面上就会添加一个图标，让您可以轻松地在桌面上创建新的堆栈作为快捷方式。然后，您可以将最多10个不同的堆栈固定到任务栏。如果您不想将堆栈固定在任务栏上，您可以使用菜单模式，并将指向堆栈的快捷方式保留在桌面上。您可以从特殊文件夹(如我的文档)或硬盘上的普通文件夹创建堆栈。

注意：在Windows7中，堆栈固定在任务栏上。在Windows XP和Vista中，堆栈固定在快速启动工具栏上。

有关信息，请参阅我们关于7Stacks的文章。

8启动启动器

8Start Launcher是一个免费的、可自定义的Windows应用程序启动器，允许您轻松地将快捷方式、URL收藏夹、文件、文件夹和应用程序链接组织到组和类别中。可以通过系统托盘、使用热键或单击鼠标中键来访问启动器。它是便携的，可以使用相对路径，作为USB闪存驱动器上便携程序的应用程序启动器非常有用。启动器的外观可以使用外观进行自定义，您可以使用自定义图片文件(.jpg、.png、.ico、.bmp、.gif)作为按钮图标。

ViPad

ViPad是适用于Windows的免费应用程序启动器和桌面组织工具，允许您将喜欢的程序快捷方式、网站链接、系统工具快捷方式、文件、文件夹等集中在一个位置。您甚至可以将项目分类到个性化选项卡中。使用拖放将项目放到启动器上并重新排列项目。

Windows 7应用启动器小工具

Windows 7 App Launcher Gadget提供了一个非常小的应用程序启动器，以小工具的形式显示在您的桌面上。您可以将程序快捷方式、文件和文件夹直接拖放到小工具上。您还可以将Firefox、Opera和IE中的收藏夹添加到小工具中，以便快速访问网站。

先行窗口导航器

多基

Docky是一款Linux平台上的Dock应用程序，可以更快、更轻松地打开常用应用程序和管理窗口。它类似于avant window Navigator，并且完全集成到GNOME桌面中。除了作为应用程序启动器，Docky还可以管理您正在运行的应用程序，并托管各种docklet，包括CPU监视器、天气预报和时钟。应用程序可以与Docky集成，将额外的项目添加到上下文菜单中，或者修改图标以显示更多信息。

键盘应用程序启动器

下面的应用程序启动程序是为那些喜欢使用键盘而不是鼠标的人准备的。它们使启动应用程序和打开文件变得又快又容易。

查找并运行机器人(FARR)

Find and Run Robot(FARR)是为键盘狂人提供的免费应用程序启动器。它使用自适应的“实时搜索”功能，让您只需键入即可快速查找计算机上的程序和文档。使用自定义热键显示FARR窗口，然后只需开始键入您要查找的应用程序、文件或文件夹的第一个字母，结果就会立即显示。您还可以使用Farr运行网络搜索、发送电子邮件、操作文件等。插件、附加组件和扩展也可用于Farr。

Launchy

Launchy是一款适用于Windows、Linux和Mac的免费实用程序，旨在帮助您只需按几下键即可启动文档、文件、文件夹和书签。它还会在Windows的“开始”菜单中为程序编制索引，以便快速访问您最喜欢的程序。Launchy将作为一个小窗口打开，您可以在其中键入您的搜索词。当您键入时，结果将显示在窗口下方。有定制Launchy外观的皮肤和扩展其功能的插件。只有在Windows中使用Launchy时，插件才可用。

Windows 7应用启动器(7APL)

Windows 7应用程序启动器(7APL)允许您使用热键或Windows 7跳转列表功能在Windows中一次启动多个应用程序。您可以创建包含要一次启动的所有应用程序的配置文件，并对每个配置文件应用一个热键。要从跳转列表运行配置文件，请将7APL.exe文件固定到任务栏。

7APL不需要安装。只需解压缩.zip文件并双击.exe文件即可。

有关更多信息，请参阅我们关于Windows7应用启动程序的文章。

沿路做记号 / 开拓 / 在树皮上刻路标 / 公开宣布

Executor是Windows的多用途启动器，允许您从一个中心位置快速运行程序和搜索任何内容。它就像是Windows运行对话框的一个更高级、更可定制的版本。它使用关键字操作，每个关键字都可以分配一个热键，因此Executor也可以像许多流行的热键管理器一样运行。可以自定义Executor的布局、外观和行为。

按键启动

Key Launch是一个应用程序启动器，允许您忽略开始菜单和桌面快捷方式，使用键盘启动程序。定义别名以使用缩写启动程序。例如，您可以为“Microsoft Word”定义“w”，这样您只需按Ctrl+空格键，然后按“w”即可打开Word。

随从 / <史>助手

Famulus是一个简单且便携的Windows文件和应用程序启动器。按住数字键盘上的‘*’键不到一秒钟可调出文本提示。在提示符中键入预定义的自定义命令，然后按Enter键以运行关联的文件、文件夹、应用程序或网站。为便于访问，前面输入的5个命令被存储起来。命令以“@”符号开头，以直接运行URL、文件路径或系统命令。

注意：如果您使用的笔记本电脑没有数字键盘，您可以在设置中更改激活密钥。

ControlPad应用程序启动器

ControlPad允许您将数字键盘用作Windows的命令执行系统。配置任何数字代码(或关键字)以执行程序、打开文档、打开网页或向操作系统发送一系列击键。按住数字键盘上的‘*’键大约一秒钟可进入一个窗口，您可以在其中输入数字代码或关键字。然后，按数字键盘上的‘/’键将代码或关键字分配给程序、文件、文件夹等。

注：如果您使用的笔记本电脑没有数字键盘，有一种特殊的笔记本电脑模式，它使用F12(按住键)而不是‘*’键作为激活控制板的键。

如果所有这些应用程序启动器选项都不够，您还可以使用快速启动工具栏在Windows中创建功能强大的应用程序启动器。要在Windows 7中执行此操作，您需要将快速启动工具栏添加回任务栏。您还可以安装一个免费程序，该程序允许您对快速启动栏上的快捷方式进行分组，并添加标题、分隔符和子菜单。
