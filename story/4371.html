如何使用第三方工具在Windows Home版本中获取专业版功能

某些功能最强大的Windows功能仅在Windows专业版或企业版中可用。不过，您不必升级到Windows专业版即可使用这些强大的功能-请使用这些免费的替代功能。

这些功能包括远程访问桌面、加密硬盘、在窗口中运行Windows XP、更改组策略中的高级设置、使用Windows Media Center、从U盘运行操作系统等。

远程桌面服务器

Windows家庭版可以连接到远程桌面会话，但不能充当远程桌面服务器。如果您想通过网络或Internet远程访问您的家庭Windows桌面，您可以使用VNC。VNC的工作方式类似于远程桌面-在您的家庭计算机上安装VNC服务器后，您可以在另一台计算机上安装VNC客户端并远程访问您的家庭计算机。VNC客户端可用于所有平台-Windows、Mac、Linux，甚至Android和iOS。

UltraVNC是一个很好的开源解决方案-它既包括服务器应用程序，也包括客户端应用程序。VNC还有其他不那么极客的替代方案，比如TeamViewer。

阅读更多信息：使用TeamViewer远程帮助计算机用户，并使用TeamViewer for Android和iPhone访问路上的台式机

BitLocker驱动器加密

BitLocker是一种全磁盘加密功能，可对硬盘上的数据进行加密。在引导时，数据通常使用密码进行解密。假设你关闭了电脑电源，没有你的密码或密钥，人们就不能访问你的数据--如果没有你的密码或密钥，你的文件看起来就像是胡言乱语。不幸的是，BitLocker只在Windows7旗舰版和Windows8专业版中可用-如果你有Windows7专业版，你甚至不能使用BitLocker。

相反，您可以使用TrueCrypt来加密整个硬盘-在Windows加载之前，您必须在引导时键入密码或提供密钥。TrueCrypt还可以用于创建加密容器和加密文件，而无需加密整个硬盘。

阅读更多：使用TrueCrypt加密您的Windows系统驱动器和TrueCrypt入门指南

Windows XP模式

Windows XP模式在Windows 7上提供了虚拟化的Windows XP环境。对于在新版本的Windows上运行无法正常运行的旧应用程序，该模式非常有用。从本质上讲，Windows XP模式是一个预打包的Windows XP虚拟机，它使用微软的Virtual PC。

如果你想使用Windows XP模式(而且你手头有一张旧的Windows XP光盘)，你不必升级--你可以安装免费的VirtualBox或VMware播放器，然后在虚拟机上安装Windows XP。虚拟机的功能类似于Windows XP模式，允许您在桌面的窗口中运行旧软件。

这也适用于已取消Windows XP模式的Windows 8。

阅读更多信息：为Windows 7家庭版和Vista创建XP模式

组策略设置

Windows的专业版本包括组策略编辑器，它可以轻松更改Windows中的一些更高级的设置。系统管理员经常使用它来调整Windows PC大型网络的设置-然而，即使在您的家用计算机上，您也可能会发现它很有用。本地组策略编辑器提供了一种更改各种设置的简单方法，您可能会遇到Web上的网页告诉您更改组策略中的某个设置。

但是，在大多数情况下，您可以在Windows注册表中更改相同的设置-尽管它可能不太用户友好。如果您遇到要更改的组策略设置，请快速进行Web搜索并查找您可以更改的相应注册表项。

如果您还可以访问运行Windows专业版计算机，则可以确定组策略设置修改哪个注册表值并自行更改。

阅读更多内容：改进Windows的20个最佳注册表黑客，以及如何查看组策略对象修改了哪些注册表设置

备份到网络(Windows 7)

Windows 7附带的Windows备份功能不允许您备份到网络位置，除非您安装了Windows专业版。如果您使用的是Windows 7家庭版，您可以使用其他备份解决方案。

微软的SyncToy是一款流行的免费备份工具，可在Windows 7上运行。您还可以创建计划任务，运行SyncToy进行自动备份。如果你正在寻找另一种选择，开源的FreeFileSync是另一个运行良好的可靠应用程序。

阅读更多内容：使用SyncToy 2.1在计算机和驱动器之间同步文件夹，并安排SyncToy在Windows 7中使用Task Scheduler自动运行

Windows Media Center(Windows 8)

在Windows 8中，Windows Media Center已从Windows家庭版中删除。您可以将Windows升级到专业版，如果您依赖Windows Media Center，也可以尝试第三方媒体中心解决方案。一个非常流行的是XBMC，尽管您可能也对Plex感兴趣。

阅读更多信息：如何使用XBMC在整个房间内同步您的媒体

Windows To Go(Windows 8)

Windows To Go是Windows 8中的一项全新功能。它允许Windows 8安装在USB驱动器上，并在任何计算机上运行-只需将USB驱动器插入任何计算机，重新启动计算机，您就可以使用Windows 8环境了。不幸的是，该功能仅在Windows8企业版中可用--即使是专业版用户也无法使用。

如果你想在U盘上安装操作系统，并随身携带，在任何你喜欢的电脑上运行，你可以使用像Ubuntu这样的Linux发行版。使用UNetboot将Ubuntu或您喜欢的任何其他Linux发行版安装到U盘上-您将拥有自己的个人浏览器(Mozilla Firefox和Google Chrome都在Linux上运行)和桌面，您可以随身携带并在任何计算机上运行。

阅读更多信息：创建可引导的Ubuntu USB闪存驱动器
