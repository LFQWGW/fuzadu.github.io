如何在Windows中使用键盘或桌面快捷键打开或关闭Wi-Fi

一些笔记本电脑配有“Wi-Fi”功能键或开关，可以快速启用或禁用Wi-Fi。不过，如果你的电脑没有这些，你可以用Windows内置的工具制作一个。

您需要从创建桌面或开始菜单快捷方式开始。完成此操作后，如果您愿意，可以使用键盘快捷键调用它。

第一步：找到您的Wi-Fi连接名称

首先，您需要检查Wi-Fi连接的名称。您将需要它来编写启用和禁用Wi-Fi连接的命令。

进入控制面板>网络和互联网>网络和共享中心>更改适配器设置。

记下要禁用的Wi-Fi连接的名称。在下面的屏幕截图中，连接的名称是“Wi-Fi”。

第二步：创建桌面快捷方式

现在您知道了连接的名称，可以创建所需的桌面快捷方式了。右键单击Windows桌面，然后选择新建>快捷方式以创建快捷方式。

将以下行复制并粘贴到“Type the location of the item”框中：

netsh interface set interface name="Wi-Fi" admin = disabled

将命令中的Wi-Fi替换为您的Wi-Fi连接的名称。

将快捷方式命名为“Disable Wi-Fi”或类似的名称，然后点击“Finish”。

现在，我们将创建启用Wi-Fi快捷方式。右键单击桌面，然后选择新建>快捷方式以创建新的快捷方式。

将以下命令复制并粘贴到“键入项目的位置”框中，将Wi-Fi替换为Wi-Fi连接的名称。

netsh interface set interface name="Wi-Fi" admin = enabled

将快捷方式命名为“启用Wi-Fi”或类似名称，然后单击“完成”。

第三步：使桌面快捷方式以管理员身份运行

我们正在使用的netsh命令需要管理员访问权限才能对您的系统进行此更改。我们需要使这些快捷方式以管理员身份运行。

要执行此操作，请右键单击其中一个快捷键，然后选择“属性”。

单击快捷方式选项卡上的“高级”按钮，启用“以管理员身份运行”选项，然后单击两次“确定”。

对第二个快捷方式重复此过程，使两个快捷方式都以管理员身份运行。

第四步(可选)：分配键盘快捷键

如果您愿意，现在可以指定键盘快捷键。右键单击其中一个快捷方式，然后选择“属性”。

在“快捷键”选项卡上，单击“快捷键”框，然后按要分配给快捷键的键或组合键。之后单击“确定”。

例如，如果要指定Ctrl+Alt+F1键盘快捷键，请单击“快捷键”框，然后按Ctrl+Alt+F1。

对第二个快捷键重复此过程，为其指定所需的任何键盘快捷键。例如，您可以指定Ctrl+Alt+F1禁用Wi-Fi，分配Ctrl+Alt+F2启用Wi-Fi。

请注意，只有当应用程序快捷键存储在桌面或开始菜单中时，这些键盘快捷键才会起作用。如果将它们放在任何其他文件夹中，当您按键盘上的快捷键时，指定的快捷键将不起任何作用。

如果愿意，您可以通过将以下行复制粘贴到文件资源管理器的地址栏，将快捷方式添加到开始菜单：

%appdata%\Microsoft\Windows\Start Menu\Programs

复制-将快捷方式粘贴到出现的文件夹中。它们将出现在您的“开始”菜单中，您可以在那里单击它们，键盘快捷键也将起作用。如果您愿意，您可以将它们从桌面上删除。

如何使用快捷键

现在，您可以通过双击桌面上的快捷键或按指定给该快捷键的任意快捷键来禁用或启用Wi-Fi。

此方法的一个“陷阱”是，每次运行快捷方式时，您都会看到一个“用户帐户控制”弹出窗口，询问您是否要执行该操作。您只需单击“是”，您的更改将立即生效。虽然有一些方法可以在不提示您的情况下以管理员身份运行命令，但这些方法存在潜在的安全问题，因此我们不推荐使用这些方法。

其他快速禁用Wi-Fi的方法

相关：飞机模式是做什么的，真的有必要吗？

还有其他方法可以快速启用和禁用Wi-Fi。在Windows 10上，您可以使用飞行模式，该模式将禁用Wi-Fi、蓝牙和任何其他无线无线电。只需单击任务栏右侧、时钟右侧的通知图标，然后单击操作中心底部的“飞机模式”磁贴即可。再次单击此磁贴可禁用飞机模式并重新启用Wi-Fi。

你也可以进入“设置”>“网络和互联网”>“飞行模式”，你会发现一个快速启用和禁用Wi-Fi的开关。

当然，敲击键盘上的几个键比这两个键都更快、更方便。但很高兴知道他们就在那里。
