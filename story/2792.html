如何安装Windows Server 2008 R2

Windows Server 2008 R2是微软Windows Server操作系统的最新版本。微软尽最大努力使每项任务尽可能简单，Server2008R2就是这一目标付诸实施的一个闪亮例子。我们将带您完成基本安装，并向您展示它有多简单。

基本安装

从安装盘启动后，您将进入语言和首选项屏幕。选择这些选项后，您将进入操作系统选择页面。根据您的许可证和服务器的用途，有多种选项可供选择。出于我们今天的目的，我们将选择企业版(完全安装)。

我们可以阅读标准的最终用户许可协议。

由于我们正在进行基本安装而不是升级，因此我们选择自定义(高级)选项。

我们有一个空白的24 GB磁盘，所以我们将把它安装在那里。如果要使用可用驱动器空间创建分区或重新格式化驱动器，请选择驱动器选项(高级)。

Windows的安装将需要一段时间，并会重新启动几次。

安装完成后，系统会提示我们在登录之前更改密码。

Windows要求您使用强密码，密码长度为七个字符，其中至少包含以下四个字母中的三个：大写字母、小写字母、数字或符号。您需要确保现在将其写在某个地方，因为如果您稍后忘记了它，整个安装将不得不重新完成。

登录后会立即弹出初始配置任务窗口。您也可以在命令提示符下键入Oobe.exe以到达此处。

我们首先要更正的一件事是时间，因此请选择“设置时区”。请确保先设置时区，因为日期和时间会在之后移动。

接下来，我们要选择Configure Networking(配置网络)。网络中安装的第一台服务器需要是域控制器，由于它们需要静态IP，我们现在需要设置一个。双击Local Area Connection，弹出信息框后，单击Properties。

单击网络框中的Internet协议版本4，然后单击属性。将单选按钮更改为使用以下IP地址：，然后输入特定服务器和网络IP地址的设置。完成后，单击“确定”保存这些设置。

完事了！

这就是进行Windows Server2008R2基础安装的全部内容。我们使用的是VMware，但如果您正在使用任何虚拟化软件进行安装，我们建议您在此时拍摄快照，因为我们尚未为此服务器设置任何角色或功能，以后只需克隆一个基本安装即可节省时间，就像我们刚才所做的那样。在这里，您可以为该服务器指定目录或证书服务等角色，但这些主题将在单独的文章中讨论。
