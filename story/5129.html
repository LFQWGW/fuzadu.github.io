Microsoft正在关闭Windows Live Messenger：这对您意味着什么

Windows Live Messenger(前身为MSN Messenger)将于2013年3月15日关闭。*微软正在用Skype取代它，并将Windows Live ID(现在称为Microsoft帐户)迁移到Skype。

如果您是Windows Live Messenger用户，则时间会发生变化。*您将需要使用Skype应用程序进行聊天-第三方即时消息客户端和旧版本的Windows Live Messenger将不再起作用。

再见Windows Live Messenger和第三方客户端

从3月15日起，您将不能再使用Windows Live Messenger桌面应用程序登录Messenger。*如果您尝试登录，系统将提示您从计算机下载Skype并卸载Messenger。

如果您使用其他应用程序(如Pidgin或Trillian)在Windows Live Messenger上聊天，这些应用程序可能也会停止运行。然而，将您的客户端更改为使用Skype而不是Windows Live Messenger将不会那么简单。*Skype Plug-in for Pidgin仅允许您在Skype已经打开并在后台运行的情况下与您的Skype好友聊天。*Trillian的Skype支持更具整合性，但仍需要安装Skype，目前无法工作，因为Skype协议发生了变化。

如果你想在Skype上聊天，目前唯一真正的选择就是官方Skype客户端。*幸运的是，Skype可用于Windows 8、传统的Windows桌面、Mac OS X、Linux、iOS、Android和Windows Phone，因此并不缺乏选择。然而，根据我们的经验，一些客户端-比如Linux的Skype和Android的Skype-没有那么精致，而且存在相当多的漏洞。

迁移到Skype

如果要继续使用Windows Live Messenger帐户并与联系人聊天，您需要将Messenger帐户迁移到Skype。*此迁移可以采用两种方式之一，具体取决于您是否已经拥有要与其合并帐户的Skype帐户。

首先，确保您安装了最新版本的Skype。*您可以从Skype网站下载，或使用Skype中的帮助>检查更新选项。*如果您已经登录Skype，请使用Skype>注销选项注销并返回登录屏幕。

从登录屏幕中，选择使用Microsoft帐户登录的选项。

使用您的Microsoft帐户登录(以前称为您的Windows Live ID：与您用于登录Windows Live Messenger的帐户相同)。

您将看到一个屏幕，询问您是否已经拥有Skype帐户。*如果没有，请单击我是Skype新手按钮，您的Microsoft帐户将链接到一个新的Skype帐户。*如果您这样做了，请单击我有一个Skype帐户选项，您将能够将您的Microsoft帐户与您现有的Skype帐户合并。

无论您选择哪个选项，您都会在Skype联系人列表中看到您的Windows Live Messenger联系人。假设每个人都切换了，你将能够继续在Skype内与你的联系人聊天。

虽然Skype主要专注于音频和视频聊天，但它也可以用于文本聊天。*您还可以将Skype界面设置为更小的好友列表-只需单击视图菜单，然后选择压缩视图即可。

Windows Live Messenger在三月份不会完全关闭。*它仍将在中国大陆继续发挥作用。
