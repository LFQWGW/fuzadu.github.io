如何在Windows 7 Media Center中翻录音乐CD

如果您是Media Center用户，您已经知道它可以播放和管理您的数字音乐收藏。但是，你知道你也可以在Windows7Media Center中翻录一张音乐CD，并将其自动添加到你的音乐库中吗？

在Windows 7 Media Center中翻录CD

将CD放入光驱。在Windows Media Center中，打开音乐库并选择CD。

如果您以前没有在Windows 7中使用Windows Media Center或Windows Media Player翻录CD，系统将提示您选择是否要添加版权保护。单击“下一步”。

默认情况下，您的CD将被翻录为.wma格式。Windows Media Center的翻录设置将从Windows Media Player中提取。因此，要更改翻录设置，我们需要在Media Player中执行此操作。单击完成。

在Windows Media Player中，单击菜单栏中的工具，然后选择选项。如果您是Windows Media Player 12的新手，请查看我们的初学者指南，了解如何使用WMP 12管理您的音乐。

选择翻录音乐选项卡，然后从格式下拉列表中选择您的输出格式。您还可以通过移动音频质量下的滑块来选择音频质量(比特率)。完成后，单击确定。

现在，您可以开始翻录CD了。单击翻录光盘。

单击是确认要翻录CD。

您可以在转换每个曲目时跟踪进度。

CD制作完毕后，您就可以随时开始在Windows 7 Media Center中欣赏音乐了。
