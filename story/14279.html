如何在Windows 10上播放不支持的视频格式

像Movies&TV和Windows Media Player这样的Windows应用程序只允许播放某些类型的视频。如果您正在努力播放不受支持的视频格式，则需要改用第三方视频播放器或编解码器或转换文件。

如果您不确定Windows是否支持您的视频文件格式，请先尝试。打开影视应用程序或Windows Media Player，然后尝试打开文件。如果出现“Format is not support”(不支持格式)错误，则需要尝试以下方法之一。

如果你找不到电影和电视应用程序，那就去找电影和电视吧。这是该应用程序在某些市场(如英国和澳大利亚)的替代名称。如果未安装，请从Microsoft Store下载电影和电视。

使用第三方视频播放器

电影和电视应用程序只支持一些常见的格式，如MOV、AVI和MP4。Windows Media Player支持许多其他文件类型，但微软似乎正在将用户推离预装的应用程序。

由于影视应用程序并不支持所有视频文件格式，而且Windows Media Player已有十年未更新，因此在Windows 10上播放不受支持的视频文件的最佳方式是使用第三方视频播放器。

媒体播放器

我们推荐VLC Media Player作为大多数Windows 10用户的最佳选择。它能够播放几乎所有的视频和音频文件格式，使其成为默认播放器的强大替代品。

如果默认的电影和电视应用程序无法播放您的视频文件、实况流或DVD，您可以在几乎所有情况下都假设VLC会播放这些文件、实况流或DVD。它还提供了数百种不同的定制选项。

相关：媒体播放器的瑞士军刀VLC中隐藏的10个有用功能

若要在VLC中打开视频文件，请单击顶部菜单中的“媒体”>“打开文件”。

多用途车辆

MPV是Windows用户的一种功能强大的替代视频播放器。它是跨平台的，因此可以选择Linux、MacOS和Android。

与无穷无尽的菜单和定制VLC不同，MPV简单明了，没有菜单，只有基本的播放选项可用。它也是便携的，这意味着你可以从USB闪存驱动器运行它。

要在MPV中打开视频文件并开始播放，只需将文件拖入打开的MPV界面即可。

PotPlayer

PotPlayer媒体播放器是最适合高级用户使用的视频播放器之一。它提供了大量的定制，支持高分辨率视频的硬件加速，并包括一个内置的视频编辑器，用于在播放过程中编辑视频文件。

默认界面全部为黑色，但您可以使用各种内置主题和配色方案对其进行自定义。

要在PotPlayer中打开媒体文件，请在打开的PotPlayer界面内单击鼠标右键或单击左上角的“PotPlayer”按钮。从那里，单击“打开文件”选择您的视频文件。

更改默认视频播放器

如果要使用第三方播放器作为默认媒体播放器，则需要在Windows 10设置中更改此设置。

您可以通过右键单击任务栏中的Windows开始菜单按钮，然后单击“设置”按钮来访问“设置”菜单。从那里，选择Apps>Default Apps。

在“默认应用程序”菜单中，单击现有的视频播放器。如果安装，电影和电视应用程序很可能是默认的。

从下拉列表中选择您的第三方媒体播放器。选择后，Windows将使用您选择的第三方播放器打开它检测到的任何视频文件。

安装其他视频编解码器

编解码器是一种将视频文件“解码”成适当的图像和音频的软件。如果您的电脑缺少适用于您的视频文件格式的正确编解码器，则无法加载视频。您可以下载第三方视频编解码器并将其安装到您的PC上以解决此问题。

不过，要当心。信誉不佳的网站，包括非法流媒体网站，确实包含可能损害您PC的编解码器下载的恶意广告。

要避免这种风险，请下载K-Lite Codec Pack，这是最流行的视频编解码器捆绑包之一。它添加了对大量媒体文件格式的支持，包括FLV和WebM。

虽然我们建议首先使用第三方播放器，但安装K-Lite将为Windows Media Player和其他一些第三方视频播放器添加额外的视频格式支持。然而，这种方法不适用于默认的电影和电视应用程序。

要开始安装，请下载您选择的K-Lite编解码器包变体并运行安装程序，该安装程序将默认为具有预设设置的“正常”安装模式。如果您想要自定义，请选择“高级”。

单击“下一步”进入下一个安装阶段。

您需要选择首选的视频播放器。这将默认为K-Lite的默认选项。将其更改为您的首选视频播放器。

如果您正在使用电影和电视应用程序，并选择此作为您的首选视频播放器，安装程序会通知您这将不起作用。

如果您正在使用VLC，您将收到类似的警告。VLC包含自己的一组视频编解码器，因此这不是必需的。

请选择K-Lite Included Media Player Classic、较旧的Windows Media Player或第三方播放器。

根据需要确认其他安装设置，然后单击“下一步”继续。

如果您在上一个屏幕上选择了“将MPC-HC安装为辅助播放器”选项，则会出现接下来的几个阶段。

确认K-Lite的Media Player Classic Player的设置，然后单击“下一步”继续下一阶段。

确认K-Lite Media Player经典设置的下一阶段。此阶段是预先设置的，因此除非您想更改K-Lite Media Player Classic的硬件加速选项，否则请按“下一步”继续。

在下一阶段设置字幕和字幕的语言选项。从下拉菜单中选择您的主要语言、第二语言和第三语言。

准备好后，单击“下一步”继续。

在下一个菜单中选择您的首选音频配置。这也是预先设置的，因此如果您愿意，请使用默认设置并单击“下一步”继续。

如果询问，请在下一阶段通过单击“拒绝”按钮拒绝任何其他附加软件选项。

最后，仔细检查您的K-Lite安装选项，然后单击“安装”按钮开始。

安装完成后，单击“完成”按钮。

您选择的媒体播放器现在应该允许您开始播放包中包含的一些附加视频文件格式。

转换为另一种视频格式

如果您设置使用电影和电视应用程序，您唯一的选择是将不受支持的视频文件转换为默认Windows播放器可以打开的格式。

有几种方法可以做到这一点。现有的在线网站可以自动将视频文件转换为其他格式。例如，搜索“FLV to MP4”将为您提供一个潜在的在线转换网站列表，但这些网站可能存在风险，不推荐使用。

最佳选择(尤其是对于安装了VLC的用户)是使用VLC转换视频文件。

VLC有一个内置的转换菜单，可供希望将视频和音频文件转换为Windows支持的格式(如MOV、AVI、MP4等)的用户使用。

相关：如何使用VLC转换视频或音频文件

一旦VLC将您的文件转换为Windows友好的格式，您就可以在电影和电视应用程序中打开它们进行播放。
