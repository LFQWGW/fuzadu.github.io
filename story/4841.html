Ubuntu 12.10，Quantal Quetzal中的8个新功能

Ubuntu 12.10已经发布，你现在可以下载了。从与网络应用和在线服务的更好集成到Unity的改进，有相当多的变化-尽管没有一个是巨大的或开创性的。

下一次的新功能清单可能会更令人兴奋，马克·沙特尔沃斯(Mark Shuttleworth)承诺将秘密开发新的“Tada！”将在Ubuntu13.04发布之前发布的特性。

Web应用集成

Ubuntu12.10包含Web应用集成，旨在让Gmail等Web应用成为Ubuntu桌面的头等公民。访问Firefox或Chromium支持的网站，您将看到安装Web应用程序的提示。例如，你可以使用这个功能将Gmail固定在你的启动器上--你会得到一个专门的Gmail窗口，并在Ubuntu的消息菜单中看到新的电子邮件通知。

请访问我们关于Ubuntu的web应用集成的帖子，了解更多该功能如何在不同网站上使用的示例。

在线账户控制面板允许您在一个地方向Ubuntu添加各种在线账户。Ubuntu桌面上的应用程序可以与这些帐户集成。例如，如果你在这里添加了一个Flickr账户，你就可以在你的Flickr账户中搜索照片，并在Ubuntu的仪表盘上预览它们。添加一个Twitter帐户，它将出现在Gwibber中，或者添加一个AIM帐户，它将出现在同理心即时消息程序中。

仪表板预览

你现在可以右击Ubuntu仪表盘中的任何东西来查看全屏预览。例如，如果您右键单击一个应用程序图标，您将看到有关该应用程序的信息，以及一个屏幕截图和一个卸载按钮。右键单击图像或文档，您将看到其预览。

轻松实现全磁盘加密

Ubuntu12.10提供了简单的全磁盘加密设置。只需在安装Ubuntu时启用Encrypt the new Ubuntu install for security选项即可。以前，这需要使用备用安装光盘。

与主文件夹加密不同，此功能使用密码加密Ubuntu分区上的所有文件-您必须在引导时输入密码才能使用系统。对于笔记本电脑和其他存储敏感数据的系统来说，这是一个有用的功能，但会降低性能。

统一统一

Ubuntu的Unity桌面现在是统一的。以前，有两个版本的Unity可用-三维加速的默认版本名为Unity，另一个版本名为Unity 2D的备用版本用于没有三维加速的系统。虽然它们看起来很相似，但实际上它们是不同的桌面。

Unity 2D现在已删除，所有系统默认使用Unity。在不支持硬件加速3D图形的系统上，Gallium3D LLVMtube用于在软件中渲染Unity的3D效果。

安全引导支持

Ubuntu12.10包括一个签名的GRUB2引导加载程序，允许它安装在使用UEFI安全引导功能的电脑上--换句话说，安装在Windows8电脑上。在这些系统上安装Ubuntu不需要向UEFI固件添加任何密钥，也不需要禁用安全引导。

有关什么是安全引导的更多信息，请阅读HTG解释：Windows8的安全引导功能是如何工作的&它对Linux意味着什么。

集成到软件源中的驱动程序

这不是一个很大的新功能，但是如果您正在寻找驱动程序控制面板来为您的硬件安装封闭源代码的驱动程序，那么它不再是一个单独的应用程序。您会发现附加驱动程序控制面板集成到了Software Sources应用程序中。

软件更新

当然，Ubuntu12.10还包含系统上所有软件的常用更新-Linux内核3.5.4、GNOME 3.6、Firefox和Thunderbird 16、LibreOffice 3.6.2等等。

一个不是最新版本的应用程序是Nautilus文件管理器-Ubuntu仍然在使用Nautilus 3.4，而不是Nautilus 3.6。GNOME项目对Nautilus3.6进行了一些有争议的更改，删除了各种功能，包括桌面图标。

亚马逊搜索结果

当您执行搜索时，Amazon搜索结果会出现在仪表盘中-因此，如果您正在搜索终端以启动终端应用程序，您将看到允许您在Amazon.com上购买电影“The Terminal”的结果。

如果您不喜欢此功能，我们提供了禁用亚马逊搜索结果的指南。
