Chrome 77的新特性，现已推出

Chrome 77将于9月10日登陆稳定频道，为Chrome带来轻松定制。不用再下载主题了：只需在New Tab页面上点击几下，就可以将您最喜欢的颜色应用到Chrome的浏览器工具栏上。

将选项卡发送到您的其他设备

更新：Chrome77可以让你在设备之间发送标签。这以前被称为“Send Tab to Self”功能，并且隐藏在一个标志后面。现在，您只需右键单击某个选项卡，即可在您的设备之间发送该选项卡。假设你在另一台设备上的Chrome浏览器中登录了同一个Google账户，那么在这里可以选择这个选项。

新建选项卡页上的背景和主题控件

更新：在Chrome 77的最终版本中，默认情况下没有启用此功能，但您可以使用隐藏标志来启用它。

老版本的Chrome允许你定制你的新标签页，但是Chrome77提供了更多的选项。要查找这些选项，请单击新选项卡页面右下角的“自定义”。

背景部分允许您选择漂亮的背景图像或上传您自己的背景图像。有一个新的“每天刷新”选项，每天都会自动改变你的新标签页背景。

“快捷方式”部分允许您选择在“新建选项卡”页上显示哪些图标：您选择的图标还是访问量最大的网站。您还可以切换“隐藏快捷方式”，在此页面上根本看不到任何网站快捷方式。

颜色和主题部分允许您创建自定义浏览器主题。在此选择两种颜色的组合，或使用颜色选择器选择您自己的颜色。它们将用于新选项卡页面背景和Chrome浏览器工具栏本身。以前，这需要从Chrome Web Store安装一个主题。

隐藏电动汽车指示灯

Chrome中的“扩展验证”(EV)指示器正在移动。目前，一些网站有特殊的扩展验证证书，Chrome和其他浏览器的显示方式不同。例如，当您访问paypal.com时，您将看到一个“PayPal，Inc.”。在Chrome的Omnibox中，锁定网站地址左侧的文本。

在Chrome77中，这些额外的文字会消失，你只会看到一个锁定图标。您仍然可以看到向其颁发证书的公司名称，但您必须先单击锁定图标。

正如Google的Devon O‘Brien解释的那样，“Chrome Security UX团队已经确定电动汽车用户界面并没有像预期的那样保护用户。”人们似乎没有注意到这个功能，或者如果它存在的话，做出不同的决定。它甚至可以呈现令人困惑和误导的公司名称。早在2018年6月，苹果就已经对Safari进行了这一改变。

一个新的欢迎屏幕

如果你在一个新系统上安装Chrome，或者只是在Chrome中创建一个新的用户账户，当你第一次打开浏览器时，你会看到一个新的欢迎屏幕，邀请你“让Chrome成为你自己的”。这只是在你第一次打开Chrome时替换了新的选项卡页面，所以如果你愿意，你可以忽略它。你甚至可以随时访问它--只需将Chrome：//欢迎/插入Chrome的Omnibox即可。

浏览向导，Chrome将邀请您将快捷方式固定到New Tab页面，为其选择背景，并将Chrome设置为您的默认浏览器。

联系Picker获取网站

谷歌一直在为Chrome添加新的网络功能。这些不是给你用的--它们是给做网站的网络开发人员用的。在这些功能出现在网页上之前，你永远不会注意到它们中的许多。它们使网络更快、更强大、更安全。

新的Contact Picker API就是一个让网络功能更强大的例子。它可以让网站查询你的联系人，Chrome可以让你从设备的地址簿中选择联系人提供给网站。它目前处于“原创试验”阶段，这意味着只有少数几个经过批准的网站可以对其进行测试。

这听起来只适用于手机，但也可以在台式机上使用。例如，联系人选取器可以从Windows10的联系人应用程序甚至您的Google联系人中拉出联系人。

它甚至比让Android或iPhone应用程序访问你的联系人更私密。Chrome会看到你的所有联系人，并给你一个列表，但只有你选择的联系人会被移交给网站。该网站无法查看您的所有联系人。

谷歌助手在更多Chromebook上

谷歌助手似乎将推出更多Chromebook。两年前，随着Pixelbook的发布，谷歌助手首次出现在Chrome OS上。谷歌一直在测试在额外的Chromebook上启用助手。事实上，有一个隐藏的标志可以在Chrome76的所有Chromebook上启用一个实验性的Google助手功能。

在Chrome 77中，此标志现在默认情况下处于打开状态。谷歌助手应该可以在更多的Chromebook上运行。然而，正如9to5Google指出的那样，这一标志似乎并不能真正在市面上的所有Chromebook上启用Google Assistant。

无论哪种方式，谷歌助手都将在更多搭载Chrome OS 77的Chrome OS设备上默认启用。

技术人员

谷歌在每一次Chrome更新上的大部分工作都是看不见的。它只会让Chrome和你使用的网页变得更好。

例如，谷歌在Chrome中内置了一个新的“Enter Key提示”属性。这样，当用户与键盘交互时，网站可以自定义触摸键盘的Enter键上显示的操作。例如，如果您正在键入一条消息，您可能会看到一个“发送”键。如果您正在执行在线搜索，您可能会看到一个“搜索”键。

关于Chrome77的所有技术细节--包括PaymentRequest对于网络开发人员的行为方式有何不同，谷歌将Chrome的引用标头限制在最大4KB，等等--都可以在谷歌的Chromium博客上找到。

Chrome将在更新可用时自动安装，并提示您重新启动浏览器。为了加快速度，您可以通过单击菜单>帮助>关于Google Chrome手动检查更新。
