如何在Mac上禁用垃圾桶和屏幕截图声音效果

把文件移到垃圾桶，清空垃圾桶，或者截取Mac桌面的屏幕截图，Mac就会播放声音。如果你厌倦了那些皱巴巴的纸张和相机快门的声音，你可以通过更改一个选项来禁用它们。

此选项不一定是您所期望的。你可能认为禁用垃圾声音的选项会在Finder的首选项窗口中，但事实并非如此。

如何在Mac上禁用声音效果

此选项在系统首选项窗口中可用。通过单击工具栏上的Apple菜单并选择“系统首选项”来打开它。

单击系统首选项窗口第二行上的“Sound”图标。

取消选中Sound Effects选项卡下的“Play User Interface Sound Effects”选项。

这个窗口中没有清楚的解释-即使你将鼠标悬停在选项上-但禁用该选项将禁用Mac上的垃圾和屏幕截图声音。

如何使用命令禁用声音效果

下面的命令执行与更改上面的选项完全相同的操作。只有当您想要自动执行此操作时，才有必要这样做。例如，一些人可能想要将此更改添加到一个脚本中，该脚本可以同时更改新Mac上的多个设置。

要禁用音效，请在终端窗口中运行以下命令：

defaults write com.apple.systemsound "com.apple.sound.uiaudio.enabled" -int 0

要撤消更改并重新启用声音效果，请运行以下命令：

defaults write com.apple.systemsound "com.apple.sound.uiaudio.enabled" -int 1

如果您使用的是其他人的Mac，并且不想更改设置，您可以通过将Mac设为静音来暂时禁用音效。将音量设置为“静音”，当您执行这些操作时，Mac将不会播放任何声音。
