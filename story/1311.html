初学者极客：在Windows7和Vista中设置默认程序

如果你曾经安装过一个程序，突然发现某些文件类型是用它打开的，而不是以前的，这可能会让你感到困惑和恼人。今天，我们来看看您如何控制哪些应用程序正在打开您计算机上的某些文件类型。

问题

您终于了解了Windows7，并在Windows Media Player上收听了您最喜欢的歌曲，一切都很棒。然后，您决定安装iTunes，因为您刚刚收到了一台iPod作为礼物。安装后，你会发现你的音乐文件夹里的图标看起来不一样，默认情况下它们是用iTunes打开的，而不是用wmp…打开的。发生了什么？

当您安装iTunes时，您可能没有注意到，除非您取消选中该选项，否则它会自动安装为所有音频文件的默认播放器。对于所有不同的程序和文件类型，这种情况可能会经常发生。使用Windows7和Vista中的设置默认程序功能，您可以控制哪些应用程序与每种文件类型相关联。

设置默认程序

要访问设置默认程序功能，请单击Start(开始)，然后单击Default Programs(默认程序)。

单击Set Your Default Programs(设置默认程序)。

浏览每个程序，并决定是否要将其设置为默认程序打开它能够打开的所有文件类型。

或者，您可以选择要将每个程序与哪些文件类型相关联。与本例一样，我们选择Foobar2000作为特定类型音频文件的默认程序，而VLC和WMP则打开其他类型的音频文件。

您还可以进入并将文件类型与不同的程序相关联，如本例中的VLC正在打开WMV文件，我们希望将其更改为Windows Media Player。您可以突出显示文件类型，然后单击更改程序按钮或简单地双击它。

从“推荐的程序”或“其他程序”下的列表中选择一个程序，或者，如果未列出特定的应用程序，您也可以浏览到该应用程序的位置。请注意，Always Use the Selected Program to Open该类file(始终使用所选程序打开此类文件)框被选中并呈灰色显示，因为我们要通过设置的默认程序进入Open With屏幕。Windows假定更改是永久性的，因为我们走了一条来这里的路线，并将其灰显。

如果您右键单击文件并通过上下文菜单…转到打开方式

该框不会灰显，您可以决定是否始终打开该类型的程序。

在Set Default Programs(设置默认程序)中，您还可以更改插入介质或设备(如闪存驱动器)时的自动播放设置。例如，您可以将DVD设置为始终在Media Player中打开，将音乐CD设置为每次都会询问您。您也可以通过取消选中对所有介质和设备使用自动播放来将其关闭。

如果您将其设置为在您放入时询问您，比方说dvd…。“自动播放”将打开，您可以选择用什么来打开它。您也可以通过单击“控制面板”中的“查看更多自动播放选项”来访问上述设置。

另一个用于控制文件关联的很酷的实用程序是我们前面介绍过的默认程序编辑器。这将为您提供我们刚刚看过的所有选项以及更多选项。

这应该可以帮助您开始决定要为不同的文件类型提供哪些程序。在本教程中，我们重点介绍了音乐和电影文件，但它对任何文件类型都是一样的。
