如何从Chrome Web Store外部安装扩展

试着从Chrome Web Store之外安装一个扩展，Chrome会告诉你扩展“只能从Chrome Web Store添加”。但是，此消息是不正确的-您仍然可以从其他地方安装扩展。

此限制是为了防止恶意网站安装错误的扩展、应用程序和用户脚本。您应该只安装来自您信任的合法网站的扩展-例如，LastPass网站。

手动安装扩展模块

要手动安装扩展，请单击“扳手”菜单，指向“工具”，然后选择“扩展”以打开“扩展”页。

如果您看到该消息，则说明Chrome已经将扩展、应用程序或用户脚本下载到您的计算机上。你可以在Chrome的默认下载文件夹中找到它。扩展名和应用程序的文件扩展名为.crx，而用户脚本的文件扩展名为.user.js。

如果它没有下载到您的计算机上，请右键单击页面上的扩展安装链接，然后使用另存为选项将其保存到您的计算机上。

将crx(或user.js)文件拖放到扩展页面上进行安装。

系统将提示您确认该扩展的安装，就像您是从Chrome Web Store安装的一样。

始终允许来自Chrome Web Store外部的扩展

如果您经常从Web Store外部安装扩展，则可以通过添加命令行标志来允许从任何网站安装扩展。

要添加命令行选项，您必须编辑Chrome快捷方式的属性。要在Windows7上访问这些浏览器，假设您从任务栏启动Chrome，右键单击任务栏上的Chrome图标，在出现的菜单中右键单击Google Chrome，然后选择属性。

如果从开始菜单或桌面启动Chrome，请右键单击开始菜单或桌面上的快捷方式。

更改此设置后，关闭所有Chrome窗口并从您修改的快捷方式启动Chrome。您可能需要在关闭所有Chrome窗口后等待片刻，以确保Chrome不再在后台运行

尝试从网页安装扩展，您将看到熟悉的安装提示-单击继续以安装扩展。

更改Chrome策略

Chrome包含专为系统管理员设计的策略设置。如果您在业务中使用Chrome，并且希望允许从一两个特定网站安装扩展，您可以修改Chrome的策略。此设置在Windows的注册表以及Mac和Linux的Chrome首选项文件中更改。
