如何在Windows 10中设置普通帐户的时间限制

Windows10中的家长控制相当可靠，但要使用它们，你必须用微软账户设置整个家庭，还必须为你的孩子创建特定的孩子账户。如果您更喜欢使用常规本地帐户，您仍然可以设置任何非管理员用户可以使用计算机的时间限制。

在Windows10中，家长控制提供了一些很好的功能来监控孩子的账户。它们可以让你限制网页浏览，孩子们可以使用哪些应用程序，以及孩子们可以使用电脑的次数。缺点是，要使用这些控件，这个家庭的所有成员都必须拥有微软账户。你还必须为孩子设置儿童账户，这可能会施加一些你可能不想要的限制。好消息是，您仍然可以使用本地帐户并施加一些相同的限制。如果您习惯于使用组策略，则将用户限制为只能运行指定的程序并不困难。您可以在路由器级别过滤网站。而且，正如我们将在这里讨论的那样，您甚至可以对本地用户帐户施加时间限制。

相关：如何在Windows 10中添加和监控孩子的帐户

您将在命令提示符处为用户设置时间限制。要打开命令提示符，请右键单击[开始]菜单(或按Windows+X)，选择“命令提示符(管理员)”，然后单击“是”以允许其以管理权限运行。

用于设置用户时间限制的命令遵循以下语法：

net user /time:,

下面是该命令的每个部分的工作方式：

将<username>替换为要限制的用户帐户的名称。 将<day>替换为要设置限制的日期。您可以拼写日期的全名，也可以使用缩写SU、M、T、W、Th、F、Sa。 将<time>替换为使用12小时(凌晨3点、下午1点等)的时间范围。或24小时营业(03：00、13：00等)。格式。您只能以一小时为增量使用时间，因此不要在时间中添加任何分钟。

例如，假设您想要限制一个名为Simon的用户帐户仅在星期六上午8：00到下午4：00使用计算机。您可以使用以下命令：

net user simon /time:Sa,8am-4pm

您还可以通过用连字符分隔日期来指定具有相同时间限制的日期范围。因此，要限制用户仅在工作日从下午4：00到晚上8：00使用计算机，您可以使用以下命令：

net user simon /time:M-F,4pm-8pm

此外，您可以用分号将多个日期/时间限制串在一起。让我们将我们之前施加的这两个时间限制一起放在同一个命令中：

net user simon /time:Sa,8am-4pm;M-F,4pm-8pm

使用相同的格式，您还可以在同一天指定多个时间范围。例如，此命令将用户限制为在所有工作日的上午6：00至8：00和下午4：00至晚上10：00：

net user simon /time:M-F,6am-8am;M-F,4pm-10pm

您还可以通过指定所有时间来删除对用户的限制：

net user simon /time:all

最后，如果您使用该命令但将时间留空(在time：part之后不输入任何内容)，用户将永远无法登录。如果您想要临时锁定帐户，这可能会很有用，但请注意不要不小心将其留空。此外，如果您需要立即显示您为用户设置的时间，您只需键入net user命令，后跟帐户名即可：

net user simon

就是这样。您不必使用家长控制或Microsoft帐户来为用户设置时间限制。您只需在命令提示符下花费几分钟即可。
