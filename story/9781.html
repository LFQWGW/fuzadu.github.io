如何让“Windows LIVE游戏”游戏在Windows10上播放

许多旧电脑游戏似乎都能在Windows 10上正常运行，但使用微软失败的Windows LIVE游戏(GFWL)平台的游戏也不例外。他们会在Windows10上给你一个错误。不过，你可以从许多游戏中完全删除GFWL，或者只是对它进行故障排除，让它正常工作。

许多游戏已经放弃了GFWL，允许您赎回非GFWL副本或安装删除GFWL的补丁。但并不是所有人都有。例如，Rockstar的“侠盗猎车手4”和“侠盗猎车手：来自自由之城”的剧集仍然使用GFWL，贝塞斯达的“Fallout 3”也是如此。这些游戏定期在Steam上进行促销，所以玩家在未来几年都会偶然发现GFWL。

获取游戏的非GFWL副本

许多游戏已经从微软的PC游戏平台迁移到Steam。如果你过去购买了这款游戏-无论你是购买了实体零售版、数字下载版，甚至是微软Windows游戏市场(Microsoft‘s Games for Windows Marketplace)的拷贝-你通常都可以将受GFWL污染的旧版转换成可以正常运行的现代版。

以下游戏允许您在Steam上兑换它们，如果您有零售密钥或GFWL本身提供的密钥。这些都是大牌、大预算的游戏，这不是偶然的。发行商和开发者没有费心将GFWL从许多表现不佳的旧游戏中删除。

蝙蝠侠阿甘精神病院。 蝙蝠侠阿卡姆之城。 BioShock 2。 黑暗灵魂：准备死亡版。 丧尸围城2。 死亡崛起2：私下记录。 土3。 生化危机5。 超级街霸4街机版

如果您有其中一个游戏的代码，您可以在Steam上兑换它。安装并启动Steam。点击“游戏”菜单，选择“在Steam上激活一款产品”，输入游戏的产品密钥，即可在Steam上兑换。Steam安装的版本将是没有GFWL的最新版本。

将GFWL踢出游戏

一些更大、更受欢迎的游戏都有第三方工具，可以有效地破解游戏中的Windows Live。这些工具并不是为多人游戏中的盗版或作弊而设计的-事实上，如果游戏确实包含多人游戏，它们就会禁用对多人游戏的访问。它们只是为了消除GFWL的麻烦。这样的修改并不适用于每一款游戏-只是特别受欢迎的游戏。

后果3：Nexus Mods的Windows Live游戏禁用程序将禁用GFWL。Fallout Script Extender修改工具Fallout Script Extender Modeling Tool也禁用了GFWL。 侠盗猎车手4：XLiveLess修改将从游戏中移除GFWL，并确保保存游戏功能正常。它还禁止访问多人游戏功能。 侠盗飞车：自由之城的第三集：XLiveLess的功能也来自自由之城。 Halo 2：XLiveLess for Halo 2承诺将GFWL从微软的第二款Halo游戏中删除，微软不再支持甚至销售这款游戏。 红色派别：游击队：XLiveLess也将从这款游戏的原始版本中删除GFWL。Steam上提供的此游戏的最新版本不再有GFWL，但此游戏的旧产品密钥无法在Steam上激活。

您可能需要像7-Zip这样的文件解压程序来解压这些下载。请按照下载的自述文件中的说明来了解如何安装您下载的任何修改，并确保其正常工作。

更新您的GFWL客户端软件

如果你在现代版本的Windows上玩一款需要GFWL的游戏，而且没有其他版本或裂缝可以帮助你禁用它们-或者如果你无论如何都想使用GFWL-你可以让GFWL起作用。尽管Windows10断言Windows LIVE游戏与Windows的现代版本不兼容，但它可以工作。

问题是，需要GFWL的游戏都有自己的GFWL安装程序。当你在现代版本的Windows上安装这些游戏之一时，它会安装一个无法正常运行的旧版本GFWL。而不是尝试自我更新，GFWL将无法正常工作，而不会给你任何错误的迹象，游戏甚至不会启动或提供错误信息。

要解决此问题，请从微软网站下载最新版本的Windows LIVE游戏并进行安装。安装最新版本后，启用GFWL的游戏应该会启动并运行。当然，它们可能不会完美工作。例如，在Windows 10上的一款特定游戏中，我们不得不使用键盘来导航GFWL界面，因为鼠标不能正常工作。“Tab”和“Enter”键是导航界面所必需的。

创建本地配置文件

通过在GFWL中创建本地配置文件(也就是脱机配置文件)而不是在线配置文件，可以避免可能干扰游戏的在线连接和同步问题。这对大多数游戏都有效，尽管如果你想使用多人游戏功能，你必须创建一个在线个人资料。如果你在42个支持GFWL和Xbox服务的国家之一，这也将允许你玩支持GFWL的游戏。如果是这样，您就会知道，因为GFWL会向您显示一条错误消息。

要做到这一点，在任何启用GFWL的游戏中，通过按键盘上的“Home”按钮打开GFWL界面，然后选择“Create New Profile”。要在Create Gamer Profile屏幕上向下滚动，请单击“Create a Local Profile”，然后输入您要使用的详细信息。

如果在玩游戏时执行此操作，您将丢失所有保存的文件。如果您已经在使用这些配置文件之一，则这些保存文件不会与您的在线配置文件相关联，因此您必须切换回另一个配置文件才能重新访问您的保存。这最好在第一次设置GFWL时完成。

解决在线连接问题

如果你在连接GFWL游戏中的其他玩家时遇到问题，那么你可能会遇到一些麻烦。微软不再积极维护这些东西，即使你可以连接，也不能保证你会有良好的体验，所以请记住这一点。我们遇到了很多问题。

但是，您可以通过更改防火墙设置来使在线连接正常工作。微软推荐三件事：

在您的无线路由器上启用UPnP。“这允许GFWL自动转发连接到其他播放器所需的端口。UPnP是一个安全问题，但当你玩完游戏时，你总是可以禁用它。 如果您不想启用UPnP，请为入站和出站流量打开以下端口。您必须在运行的任何高级防火墙软件中允许这些端口。但是，对于大多数人来说，您只需要转发路由器上的这些端口。我们建议只使用UPnP，但以下是GFWL需要的端口：TCP端口3074、UDP端口88和UDP端口3074。 允许GFWL客户端通过您的防火墙。如果您使用的是防火墙软件，请确保允许C：\Program Files(X86)\Microsoft Games for Windows Live\Client中的FGFWLClient.exe程序与入站和出站连接通信。

这是一个很好的起点，应该可以解决大多数问题-至少是大多数可以解决的问题。微软还提供了一份排除GFWL连接问题的长篇指南。如果您需要更多帮助，请参考官方指南。

有关详细信息，请查看PC游戏维基网站上的Windows游戏列表-LIVE游戏列表。它提供了支持GFWL的游戏及其支持状态的详尽列表。

如果你遇到了其他问题，PC游戏维基网站也提供了一篇很好的文章，介绍了如何解决更多可能会咬你的晦涩难懂的GFWL错误。
