如何在Windows 10上安装Windows Media Center

微软从Windows10中删除了Windows Media Center，目前还没有正式的方法将其恢复。虽然有像Kodi这样可以播放和录制直播电视的很好的替代品，但社区已经让Windows Media Center在Windows10上发挥了作用。

这不是官方的把戏。就微软而言，如果你想要Windows Media Center，你可以继续使用Windows7或8.1，尽管这变得越来越困难。微软对支持Windows Media Center不再感兴趣。

第一步：下载非官方Windows Media Center安装程序

我们认为有必要重申这一警告：这不是微软官方支持的。此过程涉及从非官方来源下载修改后的Windows Media Center应用程序，因此如果您对此感到不舒服，这可能不适合您。“我们自己尝试了一下，没有遇到任何问题，该文件在多个恶意软件扫描仪上显示为干净，其他大型网站也报道过此应用程序。但这就是我们能说的。

如果你准备好试一试，请前往我的数字生活论坛上的这个帖子。正常情况下，您需要注册才能看到最新的下载链接，但以下是截至2016年6月的最新下载链接：

Windows Media Center(64位)。 Windows Media Center(32位)

根据您使用的是32位还是64位版本的Windows10，下载合适的安装程序。只需单击下载页面上的“通过浏览器下载”链接即可。

第二步：安装Windows Media Center

下载的存档是一个.7z文件，因此您还需要下载并安装7-Zip才能打开它。

完成后，您可以在文件资源管理器中右键单击下载的.7z文件，然后选择7-Zip>Extract Here。

您将获得一个WMC文件夹。随附的自述文件建议将此文件夹复制到没有空格的短路径，以避免出现问题。例如，您可以将其直接放入C：\驱动器中。

打开文件夹，右键单击“_TestRights.cmd”文件，然后选择“以管理员身份运行”。将打开命令提示符窗口，您可以将其关闭。

然后，您可以右键单击“Insteller.cmd”文件并选择“以管理员身份运行”。

您将在命令提示符窗口中看到安装进度。在看到“按任意键退出”消息之前，不要关闭窗口。

如果出现问题，您可能需要再次尝试运行_TestRights.cmd文件并重新启动，然后才能再次运行Insteller.cmd文件。

如果您以前安装过此软件包-或者如果您从Windows 7或8.1升级，并且以前安装了Windows Media Center-您可能需要右键单击“Uninstaller.cmd”文件，然后选择“以管理员身份运行”以删除Windows Media Center的任何剩余部分，然后才能正常安装。如果您想要完全卸载Windows Media Center，这也是您需要运行的文件。

第三步：运行Windows Media Center

成功安装Windows Media Center后，它将显示在您的“开始”菜单中，作为您可以启动的普通应用程序。它应该可以正常运行，就像在Windows7和8.1上一样。

救命啊，我还有另一个问题！

如果您遇到其他问题，请打开Workarounds.txt文件以了解详细信息。此文件包含人们遇到的问题和已知有效的修复方法的列表。

例如，如果你在播放某些类型的媒体时遇到“解码器错误”，它建议你安装Shark007编解码器包。它还提供有关Windows Media Center查找电视调谐器卡和设置直播电视问题的说明。

虽然Windows Media Center目前可以正常工作，但将来对Windows 10的更改可能会破坏它。

例如，我们看到有报道称，Windows 10 11月的更新-Build 1511-如果人们特意安装的话，会自动卸载Windows 7版本的纸牌和其他旧的Windows桌面游戏。如果将来的Windows10更新卸载了Windows Media Center，我们不会感到惊讶。如果发生这种情况，社区将有望再次找到解决办法。
