使用户帐户控制(UAC)停止在Windows 7或Vista中遮挡屏幕

在Windows7或Vista中，当用户帐户控制窗口出现时，屏幕会变暗，这是非常恼人的。他们称它为“安全桌面”，但我认为它令人厌恶。

请注意，在继续操作之前，这会降低系统的安全性。

Windows 7让这一切变得简单

如果您运行的是Windows7，您只需进入控制面板中的UAC设置(或在搜索框中键入UAC)，然后向下拖动滑块，直到您看到“请勿使桌面变暗”。

非那样做不行!

Windows Vista商用版/旗舰版用户

要进入此配置屏幕，请在开始菜单搜索框中键入SECURITY。您应该会看到“本地安全策略”作为首要搜索项。

在本地安全策略窗口中，向下浏览至本地策略\安全选项

在窗口的右侧，向下滚动到靠近底部的位置，找到标题为“User Account Control：Switch to the Secure Desktop When Promting for Elevation(用户帐户控制：提示提升时切换到安全桌面)”的项目。双击将其打开，然后将其更改为已禁用：

此时应禁用安全桌面。

Windows Vista家庭用户

对于Windows Home用户，您需要通过开始菜单搜索框打开regedit。向下浏览到此注册表项：

在右窗格中单击鼠标右键并创建名为PromptOnSecureDesktop的新32位DWORD值，将该值设置为0。

可下载注册表调整。 只需下载、解压并双击DisableSecureDesktop.reg文件即可将信息输入注册表。还有一个包含的EnableSecureDesktop.reg文件可以将事情恢复到原来的状态。

安全顾虑

从大量的评论中可以看出，这篇文章是有争议的。确实，禁用安全功能总是会降低系统的安全性，在进行这样的更改之前，您应该认真考虑后果。
