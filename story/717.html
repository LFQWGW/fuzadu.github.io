如何监控计算机的CPU温度

有两组用户担心他们的计算机温度：Overclockers…。几乎所有拥有强大笔记本电脑的人。那些东西会把你烤熟的！那么，您有没有想过您的CPU的运行温度到底是多少？

有相当多的Windows程序可以用来监控温度。这里有两个我们最喜欢的选择。

对于基本CPU温度监控：核心温度

计算机中需要测量的最重要的温度是处理器或CPU。Core Temp是一款运行在系统托盘中的简单、轻量级的应用程序，它可以监控CPU的温度，而不会把它和其他东西搞得乱七八糟。它提供了几个不同的选项，这样你就可以根据你的口味定制它，甚至可以与其他程序一起使用，比如Raineter。

从其主页下载Core Temp并将其安装在您的计算机上。请非常小心地取消选中安装的第三页上的捆绑软件！默认情况下，我未选中此选项，但其他用户已注意到，默认情况下他们选中了此选项。

当您运行它时，它将以图标或一系列图标的形式出现在您的系统托盘中，显示您的CPU温度。如果您的CPU有多个核心(就像大多数现代CPU一样)，它将显示多个图标-每个核心一个图标。

右键单击图标以显示或隐藏主窗口。它将为您提供一系列关于CPU的信息，包括每个内核的型号、速度和温度。

特别要注意“TJ。MAX“值-这是制造商已将您的CPU额定为可运行的最高温度(以摄氏度为单位)。如果你的CPU接近那个温度，就会被认为过热。(通常情况下，最好将其调低至少10到20度-即使这样，如果您离它很近，通常也意味着出了问题，除非您的CPU超频了。)

对于大多数现代CPU，Core Temp应该能够检测到TJ。您的特定处理器的最大值，但您应该在线查看您的特定处理器并仔细检查。每个处理器都有一点不同，并且有一个精确的TJ。最大值非常重要，因为它可以确保您的CPU获得正确的温度读数。

转到选项>设置以配置Core Temp的一些更有用的功能。以下是我们建议查看的一些设置：

General>Start Core Temp with Windows：您可以打开或关闭此功能；这取决于您。打开它可以让你随时监控你的体温，而不必记得启动它。但如果你只是偶尔需要这款应用，关闭它也没问题。 Display>Start Core Temp Minimized：如果您打开了“Start Core Temp with Windows”，您可能会想要打开此选项。 Display(显示)>Hide Taskbar Button(隐藏任务栏按钮)：同样，如果您要让它一直运行，最好打开它，这样它就不会浪费任务栏上的空间。 通知区域>通知区域图标：这允许您自定义核心临时在通知区域(或系统托盘，通常称为通知区域)中的显示方式。你可以选择只显示应用程序的图标，或者显示CPU的温度-我建议使用“最高温度”(而不是“所有内核”，它会显示多个图标)。您还可以在此自定义字体和颜色。

如果图标只出现在弹出任务栏中，而您想要始终看到它，只需单击并将其拖动到任务栏即可。

如果您决定在通知区域中显示温度，您可能需要在Core Temp设置的常规选项卡中更改温度轮询间隔。默认情况下，它被设置为1000毫秒，但如果闪烁的数字惹恼了你，你可以把它调高一些。请记住，设置得越高，如果CPU过热，Core Temp通知您的时间就越长。

核心温度可以做的比这多得多-例如，你可以前往选项>过热保护，当电脑达到最高安全温度时让它提醒你-但这些基本知识应该就是你关注CPU温度所需的全部。

对于整个系统的高级监控功能：HWMonitor

通常，您的CPU温度将是需要监控的最重要的温度。但是，如果您想要查看整个系统(主板、CPU、显卡和硬盘)的温度，HWMonitor可以为您提供更多信息。

从HWMonitor主页下载最新版本-我推荐ZIP版本，它不需要安装，不过如果您愿意，也可以下载完整的安装版本。启动它，您将看到一个包含温度、风扇转速和其他值的表格。

要找到您的CPU温度，请向下滚动到您的CPU条目-例如，我的是“Intel Core i7 4930K”-并查看列表中的“Core#”温度。

(请注意，“核心温度”与“CPU温度”不同，对于某些PC，“CPU温度”会出现在“主板”部分下。通常，您会想要监控核心温度。有关更多信息，请参阅我们下面关于AMD温度的说明。)

您也可以随意查看系统中其他组件的温度。使用HWMonitor可以做的事情不多，但它是一个很好的程序。

关于AMD处理器温度的一个注记

AMD处理器的温度监控长期以来一直困扰着电脑爱好者，与大多数英特尔处理器不同的是，AMD机器会报告两个温度：“CPU温度”和“核心温度”。

“CPU温度”是中央处理器插座内的实际温度传感器，而“核心温度”则根本不是温度。这是一个以摄氏度为单位的任意刻度，在某种程度上是为了模仿温度传感器。

您的BIOS通常会显示CPU温度，这可能与显示核心温度的程序(如Core Temp)有所不同。有些程序，如HWMonitor，两者都显示。

CPU温度在低水平时更准确，但在高水平时就不那么准确了。当您的CPU变热时，核心温度会更准确-这是温度值真正重要的时候。所以，在几乎所有的情况下，你都会想要注意核心温度。当您的系统空闲时，它可能会显示不可能的低温(例如15摄氏度)，但一旦事物稍微升温，它将显示一个更准确、更有用的值。

如果你得不到读数怎么办(或者温度看起来真的不对劲)

在某些情况下，您可能会发现上述程序中的一个不太起作用。也许它与另一个温度监测程序不匹配，也许它低得离谱，或者你根本无法获得温度。

可能发生这种情况的原因有很多，但这里有几点需要检查：

你在找合适的传感器吗？如果两个程序不一致，有可能--特别是在AMD机器上--一个程序报告“核心温度”，另一个程序报告“CPU温度”。确保你是在比较苹果和苹果。正如我们上面提到的，核心温度通常是您想要监控的。 确保您的软件程序是最新的。例如，如果您使用的是旧版本的Core Temp，它可能不支持您的CPU，在这种情况下，它不会提供准确的温度(或者甚至可能根本不提供温度)。下载最新版本，查看是否解决了问题。如果您的CPU非常新，则可能需要等待程序的更新。 你的电脑有多老了？如果它的历史超过几年，则可能不受Core Temp等程序的支持。

我们可以写一本关于监控CPU温度的书，但是为了便于理解，我们就到此为止。希望您可以大致估计一下您的CPU冷却得有多好。

监测你的体温很好，每个人都应该偶尔检查一下。但是如果你的电脑经常过热，可能有更深层次的原因需要你去调查。打开任务管理器，看看是否有任何进程在使用你的CPU，然后停止它们(或者找出它们失控的原因)。或者确保你没有阻塞电脑上的任何通风口，特别是如果它是笔记本电脑的话。用压缩空气吹灭通风口，以确保它们不会充满灰尘和污垢。电脑越旧越脏，风扇就越难工作来降低温度-这意味着电脑很热，风扇的声音也很大。
