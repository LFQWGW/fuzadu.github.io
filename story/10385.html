如何让Windows为您排除PC故障

Windows包括各种旨在快速诊断和自动解决各种计算机问题的“疑难解答”。“疑难解答”不能解决所有问题，但如果您遇到计算机问题，它们是一个很好的起点。

故障排除程序内置于Windows 10、8和7的控制面板中，因此几乎所有Windows用户都可以使用它们。在Windows10的创建者更新中，现在可以通过设置应用程序获得大多数故障诊断程序。

Windows 10

如果你已经安装了Windows10的创建者更新，你可以在设置中找到这些。导航到设置>更新和安全>故障排除。

截至创建者更新，此处提供以下疑难解答：蓝屏、蓝牙、硬件和设备、家庭组、传入连接、Internet连接、键盘、网络适配器、打印机、播放音频、电源、程序兼容性疑难解答、录制音频、搜索和索引、共享文件夹、语音、视频播放、Windows应用商店应用和Windows Update。

如果您的电脑出现问题，相关的疑难解答程序可能会为您找到并解决问题。

选择要运行的疑难解答程序，然后单击“运行疑难解答程序”。许多故障排除程序将自动运行并修复他们发现的问题，而一些故障排除程序会建议各种修复程序，您可以选择是否应用这些修复程序。

设置界面不会列出所有可用的故障排除程序。例如，它省略了后台智能传送服务、Windows Media Player DVD、Windows Media Player媒体库和Windows Media Player设置疑难解答。

如果您需要它们，它们仍然可用-它们只是隐藏在控制面板中。要查找它们，请打开控制面板，在其搜索框中键入“Troubleshoot”，然后单击“Troubleshooting”图标。

单击故障排除窗格左侧的“全部查看”，您将看到可用故障排除程序的完整列表。

Windows 7和8

您可以在Windows 7和Windows 8的控制面板中找到这些工具。如果您使用的是Windows 10周年纪念日更新版或更早版本的Windows 10，还需要使用控制面板。

导航到“控制面板”>“系统和安全”>“排除常见计算机问题故障”。在Windows7上，请改为单击“查找并修复问题”。

您将看到您可能需要的最常见故障排除程序的列表。

这些并不是唯一可用的故障排除程序。单击侧边栏中的“View All”可查看故障排除程序的完整列表。以下是您可能会找到的故障排除程序列表，尽管并非所有版本的Windows都包含相同的故障排除程序：

后台智能传输服务：查找并修复后台智能传输服务的问题，Windows Update和其他一些服务使用该服务进行后台下载。 硬件和设备：检查您的计算机是否存在硬件设备问题。如果硬件设备-尤其是最近安装的硬件设备-无法正常工作，此故障排除程序可以查找并修复硬件检测和驱动程序问题。 家庭组：查找家庭组网络和文件共享设置的问题。 传入连接：检查Windows防火墙是否阻止了您需要的传入连接，并帮助您解除阻止。 Internet连接：检测并修复您的Internet连接和加载网站的问题。 Internet Explorer性能：识别可能会降低Internet Explorer速度的问题并修复这些问题。 Internet Explorer安全：识别可能导致Internet Explorer中的安全和隐私问题的设置，并对其进行修复。 网络适配器：查找并修复Wi-Fi适配器或其他网络适配器的问题。 播放音频：扫描可能导致声音无法正常播放的问题。 电源：识别问题并修复电源设置问题，以延长计算机的电池寿命。 打印机：检查并修复打印机和打印问题。

程序兼容性疑难解答：帮助您选择运行为旧版Windows设计的程序的最佳兼容性设置。 录音：扫描可能会阻止麦克风录音工作的问题。 搜索和索引：修复Windows搜索和索引器的问题。 共享文件夹：确定可能会阻止共享网络文件夹运行的问题。 系统维护：查找并修复损坏的快捷键，并执行和系统维护任务，包括检查时钟是否正确。 视频播放：检测可能导致视频无法正常播放的问题并进行修复。 Windows Media Player DVD：修复了可能会阻止DVD在Windows Media Player中播放的问题。 Windows Media Player媒体库：修复Windows Media Player媒体库的问题。 Windows Media Player设置：修复Windows Media Player设置的问题。 Windows应用商店应用：修复可能阻止Windows应用商店应用-换句话说，Windows 10的新通用Windows平台应用-正常工作的问题。 Windows Update：识别并修复可能导致Windows Update完全无法工作或无法安装某些更新的问题。

要运行故障排除程序，只需在故障排除窗格中单击它。要快速找到相关的故障排除程序，您可以从故障排除窗口执行搜索。

单击故障诊断程序后将启动它。只需单击“下一步”即可开始故障排除。

大多数故障排除程序将自动运行，查找问题并修复发现的任何问题。要防止故障诊断程序自动更改您的系统，请单击故障诊断程序窗口左下角的“高级”链接，然后取消选中“自动应用修复”选项。在故障排除程序对您的系统进行任何更改之前，系统将提示您更多信息。

虽然大多数故障排除程序会自动运行，但有些故障排除程序会为您提供需要单击的选项。例如，程序兼容性疑难解答程序将引导您选择工作不正常的程序并更改其兼容性设置。传入连接疑难解答程序将询问您正在尝试执行什么操作，以便它知道要排除哪种类型的传入连接故障。

仅此而已。并不是你遇到的每一个问题都有一个故障排除程序，而且确实存在的故障排除程序也不一定能够解决所有的问题。但是，当您遇到某些问题时，故障排除程序是一个很好的起点。
